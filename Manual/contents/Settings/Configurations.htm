Конфигурации

В связи с требованиями кроссплатформенной разработки и необходимостью того, чтобы один продукт был доступен и настраивался для различных клиентов, устройств или платформ, GameMaker Studio 2 включает конфигурации для упрощения работы. Используя эту систему, вы можете быстро переключаться между различными заставками, иконками и включенными файлами и в любой момент протестировать или экспортировать только те из них, которые подходят для целевой платформы или клиента, и все это на основе одного и того же базового кода.

Чтобы иметь возможность справляться с различными требованиями клиентов и платформ, вы можете использовать редактор конфигурации, который доступен из меню Extras в браузере активов:

Все игры, созданные в GameMaker Studio 2, начинаются с конфигурации "по умолчанию", которая указана в редакторе при первом открытии. его:

Для настройки конфигураций рекомендуется сначала подготовить эту конфигурацию "по умолчанию", включив в нее все настраиваемые активы проекта. проекта. Для этого просто пройдитесь по следующему списку опций и установите в них то, что требуется по умолчанию:

  • Параметры расширения: Установите различные поля "Copy to..." на необходимые платформы для конфигурации. Любые "Прокси" файлы, созданные здесь, будут иметь свою "Целевую" платформу также будет связана с конфигурацией.
  • Редактор спрайтов: Установите группу текстур, в которую будут добавлены спрайты.
  • Редактор наборов плиток: Установите группу текстур, в которую будут добавлены наборы плиток.
  • Редактор шрифтов: Установите группу текстур, в которую нужно добавить шрифты.
  • Менеджер групп текстур: Здесь можно задать любые свойства для каждой конфигурации, включая родительскую группу, к которой принадлежит группа текстур.
  • Менеджер аудиогрупп: Здесь можно настраивать звуки, которые назначаются различным аудиогруппам...
  • Включенные файлы: Установите в поле "Копировать в..." необходимые платформы для конфигурации.

После того как вы установили конфигурацию по умолчанию, вы можете добавить новую конфигурацию в редактор конфигурации, щелкнув правой кнопкой мыши на папке конфигураций и выбрав Add Config, или нажав кнопку "Create" в нижней части окна. Новая конфигурация будет показана как "дочерняя" от конфигурации по умолчанию - все пользовательские конфигурации являются дочерними и вы можете дать ей соответствующее имя.

Чтобы отредактировать конфигурацию, необходимо либо дважды щелкнуть на ней , либо воспользоваться правой кнопкой мыши , выбрав пункт меню "SetConfig", который выделит конфигурацию в редакторе, после чего можно приступать к ее настройке. Это делается простым перебором перечисленных выше опций и изменением их на желаемые параметры для вновь созданной конфигурации. GameMaker Studio 2 автоматически сохранит эти значения для конфигурации, и вы сможете переключаться между конфигурациями в любое время, просто используя опцию меню RMB или дважды щелкнув на .

Дальнейшие конфигурации могут быть добавлены либо как дочерние элементы конфигурации по умолчанию, либо как дочерние элементы созданных вами пользовательских конфигураций, в зависимости от того, какая конфигурация была выделена в редакторе при ее создании. Стоит отметить, что вы можете использовать любую заданную конфигурацию с любой заданной целевой платформой при компиляции или тестировании ваших проектов. Для этого необходимо выбрать нужную конфигурацию в разделе Config в Target Manager:

Вы также можете открыть редактор конфигурации в любое время из диспетчера целей, нажав на кнопку меню в правом верхнем углу (как показано на рисунке выше). выделено на рисунке выше).

После того, как вы зададите конфигурацию для использования таким образом, при любой компиляции с целью тестирования или создания конечного пакета активов будет использоваться выбранная конфигурация. Для получения дополнительной информации об окне Цели см. раздел о компиляции.
