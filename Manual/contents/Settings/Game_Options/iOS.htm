Варианты игр дляiOS

В этом разделе описаны различные доступные вам опции, которые определяют, как будут компилироваться ваши игровые проекты для iOS. Различными разделами являются:

Общий

Первая часть раздела "Общие сведения" посвящена настройкам сборки для вашей игры. Для начала вы устанавливаете ориентации , в которых в которых будет работать ваша игра для наилучшего восприятия пользователем. Доступны следующие варианты:

  • Портрет
  • Перевернутый портрет
  • Ландшафт
  • Перевернутый ландшафт

Затем вы можете перейти в раздел " Поддержка устройств ", где доступны следующие опции для настройки типа устройства:

  • iPhone или iPod Touch: iPhone 6 и выше или устройства iPod Touch
  • iPad: устройства iPad
  • Оба: и iPhone/iPod, и iPad

Второе, что нужно сделать, это заполнить информацию о продукте вашей игры, включая ее отображаемое имя, идентификатор комплекта и информацию о версии. Обратите внимание, что идентификатор пакета должен быть в правильном обратном формате URL для корректной работы финальной сборки игры, например: com.[Company].[GameName]. Здесь вы также можете указать идентификатор команды, который вы хотите использовать для подписи финального приложения, которое GameMaker Studio 2 создает для вас. Заданный здесь идентификатор будет заменять идентификатор команды, который вы указали в настройкахiOS.

Далее можно установить или снять флажок Отложить вывод индикатора дома. Если эта опция отмечена, домашняя панель всегда будет отображаться при низком уровне альфа, и первый взмах пробуждает панель, а второй взмах переносит вас домой. Если опция не отмечена, то бар не будет отображаться в игре, но первый свайп по краю приведет вас домой. По умолчанию опция не отмечена.

Наконец, у вас есть возможность сгенерировать Podfile и Podfile.lock. Эти файлы необходимы, если вы добавляете CocoaPods в расширение для iOS с помощью редактора расширений. Файлы содержат специальные макросы, необходимые GameMaker Studio 2 для настройки всех перечисленных CocoaPodsпри создании игры. Нажав на кнопки, вы создадите файлы с макросами, а также откроет проводник файлов, если вы захотите отредактировать их вручную. Также есть кнопка, которая приведет вас к следующему руководству:

  • iOS и tvOS: использование какао-подборок

Графика

Здесь вы можете изменить следующие детали, связанные с тем, как будет отображаться ваша игра, причем доступны следующие опции:

  • Интерполировать цвета между пикселями: Включает линейную интерполяцию, которая, по сути, "сглаживает" пиксели. Для четкой пиксельной графики этот параметр должен быть выключена, но если у вас есть красивые альфа-смешения и сглаженные края графики, ее лучше оставить включенной. По умолчанию эта функция выключена.
  • Половина текстур iPad 1: iPad 1 имеет гораздо меньше текстурной памяти, чем более современные устройства, поэтому вы можете включить эту опцию, чтобы уменьшить размер текстур наполовину, если вы хотите ориентироваться на это устройство или добиться максимальной совместимости.
  • Масштабирование: Здесь вы можете выбрать масштабирование холста рисунка с сохранением соотношения сторон экрана устройства - добавление "прокладок" по краям для соответствия экрану - или выбрать растяжение холста рисунка для соответствия экрана устройства.

Наконец, есть возможность задать размер страницы текстуры. По умолчанию (и наиболее совместимый) размер - 2048x2048, но вы можете выбрать любой размер от 256x256 до 8192x8192. На сайте также есть кнопка Preview , которая генерирует страницы текстур для этой платформы и открывает окно, чтобы вы могли увидеть, как они выглядят. Это может быть очень полезно, если вы хотите увидеть, как структурированы текстурные страницы и чтобы избежать создания текстурных страниц большего (или меньшего) размера, чем необходимо. Более подробную информацию о текстурных страницах можно найти здесь.

ПРИМЕЧАНИЕ: Помните, что чем больше размер страницы текстуры, тем меньше совместимость вашей игры с различными браузерами и устройствами.

Иконки

В этом разделе можно добавить различные значки, необходимые для различных устройств iPhone и iPad и различных разделов магазина приложений iOS. iOS App Store. Эти значки должны быть созданы как 24-битные изображения *.png в размере, указанном для каждого из них.

Стоит отметить, что в GameMaker Studio 2 есть инструмент Project Image Generator, который можно использовать для автоматического создания всех изображений, необходимых для различных целевых платформ, на которые компилируется ваша игра. для которых компилируется ваша игра. Если вы используете этот инструмент, вам следует пересмотреть созданные изображения, чтобы убедиться, что они соответствуют вашим требованиям. Инструмент генерирует как значки, так и изображения (см. раздел ниже).

ПРИМЕЧАНИЕ: При импорте старого проекта может появиться следующее предупреждение:Это происходит потому, что текущая целевая версия iOS с момента загрузки проекта изменился формат необходимых значков. Вам придется заново создать все иконки, так как они были заменены на стандартные иконки GameMaker Studio 2 (вы можете использовать вышеупомянутый инструмент инструмент Project Image Generator, чтобы сделать это быстро).

Изображения

В этом разделе можно добавить отдельную графику в качестве стартовых экранов (изображение, которое будет отображаться на короткое время, пока для каждой из ориентаций устройства, а также цвет границы, который будет использоваться в качестве фона для экранов запуска, не заполняющих весь экран. Эти изображения должны быть созданы как отдельные 24bit *.png без прозрачности и соответствующего размера. Обратите внимание, что это не считается заставкой, и у Apple есть довольно строгие рекомендации по ее настройке и использованию. Дополнительную информацию см. на странице разработчика " Руководство по человеческому интерфейсу ".

Вы также можете задать способ отображения экрана запуска из следующих вариантов:

  • Сохранить соотношение сторон: Это позволит сохранить соотношение сторон изображения по сравнению с экраном устройства, добавляя "подложку" по краям, где это необходимо.
  • Crop To Fit: Это позволит сохранить изображение в соотношении 1:1 с экраном устройства и обрезать (или подрезать) края по мере необходимости.
  • Растянуть: Это растягивает изображение, чтобы заполнить экран устройства (и может исказить отображаемое изображение).

Социальная

Этот раздел предназначен для включения различных социальных функций в вашей игре для iOS. Можно выбрать следующие опции:

  • Facebook: Если вам необходимы функции Facebook, вам необходимо установить этот флажок, а затем указать Facebook App ID и Facebook App Name. Обратите внимание, что доступ ко всем функциям Facebook осуществляется через расширение расширение, и когда вы впервые установите флажок Использовать Facebook, вам будет предложено загрузить и установить расширение Facebook из каталога Рынок.
  • Включите Push-уведомления: Для того чтобы ваша игра могла использовать Push-уведомления, они должны быть сначала включены здесь, иначе они не будут работать.
  • Включить вход с помощью Apple: Эта опция предназначена для использования расширения Apple Sign In Extension на iOS. Вы можете получить это расширение из Игры YoYo Торговая площадка, а само расширение содержит полные инструкции по использованию. Если вы не используете это расширение, то вам не следует отмечать эту опцию.
