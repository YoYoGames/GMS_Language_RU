Девять ломтиков

Девятикратная нарезка (также известная как 9-кратная нарезка) - это техника, используемая для масштабирования прямоугольных изображений с целью сохранения деталей, благодаря чему они сохраняют свою первоначальную форму после масштабирования.

Это достигается путем разделения изображения на девять отдельных частей или "срезов". Угловые фрагменты перемещаются во время масштабирования и рисуются в том же масштабе, затем краевые и центральные фрагменты растягиваются или повторяются (в зависимости от настроек), чтобы заполнить пространство.

Четыре направляющие размещаются на спрайте, чтобы разделить изображение на девять фрагментов. Следующий рисунок демонстрирует, как направляющие остаются на месте после того, как спрайт масштабируется с помощью Nine Slice:

Режим Tile Mode фрагмента определяет, как этот фрагмент будет отображаться при масштабировании спрайта. Его можно установить только для краевых и центральных фрагментов (так как угловые угловые фрагменты не масштабируются).

Ниже показан каждый режим тайла при применении к центральному срезу:Если для спрайта включена функция Nine Slicing, он будет использоваться для изменения размера спрайта везде, где он появляется, и будет работать во всех существующих местах, где использовался спрайт, без каких-либо других изменений.

ПРИМЕЧАНИЕ: Nine Slicing не будет работать с функциями, которые рисуют только часть спрайта или искажают его, такими как draw_sprite_part() и draw_sprite_pos().

В сочетании с последовательностями эта функция может использоваться для построения пользовательских интерфейсов, а также, например, для создания прямоугольных стен в игровом уровне. Память текстур также можно сохранить с помощью Девять нарезок, так как меньшее изображение может быть использовано для покрытия большей области на экране.

Девятикратную нарезку можно включить для спрайтов в IDE. Для получения дополнительной информации см. раздел Редактор спрайтов.
