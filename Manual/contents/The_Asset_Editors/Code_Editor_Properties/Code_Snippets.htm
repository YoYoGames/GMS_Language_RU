Фрагменты кода

Очень удобным инструментом, который вы можете использовать при редактировании кода, является использование фрагментов кода. При использовании редактора сценариев вы можете нажать F4, чтобы открыть всплывающее окно фрагментов кода, которое позволяет вам выбрать один из часто используемых методов кода из списка:

В этом окне вы можете выбрать мышью фрагмент, который нужно использовать, или нажать соответствующую горячую клавишу, перечисленную справа. Это добавит фрагмент в ваш код для редактирования.

При желании вы можете определить собственные фрагменты кода. Перед этим вам необходимо создать файл с именем "snippets.txt" в одном из следующих каталогов:

  • Windows: %ProgramData%\GameMakerStudio2\User\
  • macOS: /Users/Shared/GameMakerStudio2/User/

Эта папка не изменяется при любом обновлении GameMaker Studio 2, поэтому отредактированный вами файл останется нетронутым (но любые правки файла базовых фрагментов в каталоге установки будут отменены).

После создания файла в каталоге User вы можете редактировать его с помощью любого текстового редактора, следуя следующим правилам:

  • Каждый сниппет должен быть на отдельной строке (не должно быть пустых строк)
  • Каждый фрагмент должен начинаться с горячей клавиши, за которой следует "-", затем название фрагмента (то, что отображается в меню), за которым следует двоеточие ":", например:

I - Создание экземпляра:

После двоеточия вы добавляете фрагмент кода

I - Создание экземпляра:instance_create_layer(x, y, |layer|, object);

Код, который вы добавляете, также должен соответствовать определенному формату:

  • Весь код находится в одной строке
  • Вы используете "$" для новой строки.
  • Вы помещаете текст, который нужно выделить первым, между символами "|" (курсор выделит этот текст для редактирования, когда фрагмент будет добавлен).

Смотрите примеры, которые уже находятся в базовом файле фрагментов, чтобы увидеть, как он устанавливается в соответствии с правилами, описанными выше. Базовый файл можно найти в каталоге установки GameMaker Studio 2:

  • \GameMaker Studio 2\TextEditor\snippets.txt
