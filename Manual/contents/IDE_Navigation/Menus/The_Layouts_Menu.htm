Меню разметок

Это меню связано с сохранением и загрузкой различных разметок IDE. Вы можете настроить IDE по своему усмотрению, при этом некоторые элементы (например, браузер ресурсов, или окна вывода) расположены иначе, чем по умолчанию. Это позволяет вам настроить GameMaker Studio 2 в соответствии с вашим рабочим процессом. С помощью опций этого меню вы можете сохранить разметку текущего проекта в виде файла *.xml, который затем можно загрузить в любой проект, чтобы установить одинаковые разметки. Опции этого меню следующие:

  • Загрузить разметку - откроется проводник файлов, в котором можно выбрать ранее сохраненный файл разметки *.xml для загрузки.
  • Сохранить разметку - Откроется проводник файлов, где вы можете назвать и сохранить текущую разметку IDE в виде файла *.xml.
  • Сбросить разметку - сброс текущей раскладки IDE к настройкам по умолчанию.
