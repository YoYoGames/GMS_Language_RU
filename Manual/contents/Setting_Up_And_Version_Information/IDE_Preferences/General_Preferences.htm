Общие предпочтения

Общие настройки используются для определения общего поведения IDE GameMaker Studio 2. Самые общие из них перечислены на главной странице, а многие другие разделены на несколько подкатегорий, которые описаны далее на этой странице.

  • Язык IDE: Эта опция позволяет выбрать язык для IDE из списка предлагаемых вариантов. По умолчанию это английский язык.
  • Скин IDE: Здесь вы можете выбрать один из различных скинов, которые может использовать IDE. По умолчанию GameMaker Studio 2 поставляется с двумя скинами, светлым и темным, и при начальной настройке используется темный скин.
  • Максимальная частота кадров редактора GameMaker Studio 2: Этот параметр позволяет установить максимальную частоту кадров, с которой будет обновляться IDE GameMaker Studio 2, что позволяет установить частоту кадров в соответствии со скоростью обновления вашего монитора. По умолчанию значение равно 60.
  • Аудиоустройство по умолчанию: Здесь вы можете выбрать аудиоустройство по умолчанию, которое будет использовать IDE GameMaker Studio 2. Вы можете обновить список, нажав кнопку Refresh Button .
  • Автоматическая реакция на подтверждение выхода: Эта опция позволяет установить, следует ли показывать подтверждающее сообщение при выходе из GameMaker Studio 2. По умолчанию установлено значение "показать сообщение", но его можно установить так, чтобы пропустить сообщение, как если бы вы нажали кнопку сообщение, как если бы вы нажали кнопку OK.
  • Automatic Response to The System Font Changed Dialog: Эта опция позволяет установить, следует ли показывать подтверждающее сообщение, когда системный шрифт был добавлен, удален или отредактирован. По умолчанию установлено значение "показать сообщение", но можно настроить пропуск сообщения, как если бы вы нажали кнопку OK.
  • Включить уведомления: Установка этого флажка позволит IDE показывать уведомления о новых исполнениях и т.д.. По умолчанию эта опция включена.
  • Тайм-аут уведомления (мс): Устанавливает продолжительность времени (в миллисекундах), в течение которого будет отображаться уведомление. По умолчанию - 6000.
  • Выход при выходе: Если эта опция отмечена, то IDE будет автоматически выходить из пользовательской сессии при закрытии GameMaker Studio 2. Эта опция предназначена в первую очередь для пользователей образовательных учреждений, поскольку, если она установлена, это означает, что при каждом закрытии программы пользователь будет выходить из сеанса и освобождать место в сеансе для других пользователей. пользователь будет выходить из сеанса, освобождая место в сеансе для других пользователей. По умолчанию эта функция выключена.
  • Включить переопределение DPI: Эта опция позволяет переопределить настройки DPI по умолчанию на свои собственные. Вы должны сначала включить опцию, а затем установить либо значение Raw DPi Override (по умолчанию 96), либо процентное соотношение к собственному DPi (по умолчанию 100%). После изменения вам будет предложено перезапустить IDE, что может потребовать сохранения любой работы, выполняемой в текущем открытом проекте. Обратите внимание, что минимальный процент DPi составляет 50%, а максимальный - 500%, и если вы навести курсор на значок информации, то можно увидеть рекомендуемое максимальное значение для используемого монитора:

    Обратите внимание, что если вы превысите рекомендуемое значение DPi для монитора, вам будет показано уведомление, позволяющее сбросить настройки. уведомление, позволяющее сбросить DPi.
  • Показывать экран приветствия при запуске: Этот параметр включен по умолчанию и просто указывает GameMaker Studio 2 показывать короткую заставку при запуске программы. Если снять этот флажок, заставка не будет отображаться.
  • Разрешить закрытие пустых вкладок с помощью сочетания клавиш: Вы можете закрыть все открытые вкладки с помощью сочетания клавиш / + W, но если вы отключите эту опцию, сочетание клавиш перестанет работать. По умолчанию эта опция включена.
  • Открытие новых ресурсов для редактирования при перетаскивании файлов в IDE: Определенные ресурсы (например, спрайты или звуки) можно перетащить на основную рабочую область в IDE GameMaker Studio 2, и они будут обнаружены и добавлены как новый ресурс в Asset Browser. По умолчанию соответствующий редактор ресурсов также будет открыт для редактирования, когда это произойдет, но если снять флажок с этой опции, то редакторы не будут открыты, и вам придется открывать их вручную.
  • Открывать новые ресурсы для редактирования при создании: Если этот флажок установлен, то каждый раз, когда вы создаете новый ресурс в браузере активов, соответствующий редактор ресурсов будет открываться автоматически. Этот параметр установлен по умолчанию, и снятие флажка означает, что вам придется вручную открывать все вновь созданные ресурсы.
  • Автоматически перезагружать измененные файлы: Если этот флажок установлен, то IDE будет автоматически сканировать и проверять файлы на предмет того, не были ли они изменены вне GameMaker Studio 2, и перезагружать их, если это так.
  • Отключите анимацию переходов IDE: В IDE GameMaker Studio 2 есть несколько анимированных переходов для таких вещей, как открытие окон и т.д... однако они не всем приходятся по вкусу, и вы можете ощутить прирост производительности, если на старых или маломощных машинах, если они отключены. Вы можете отключить их здесь, по умолчанию они включены.
  • Opt-in To Feedback Surveys: Когда эта опция отмечена, вы сообщаете GameMaker Studio 2, что хотите участвовать в опросах обратной связи, то есть время от времени во время работы будет появляться сообщение об области, над которой вы работаете. и запрос на оценку и отзывы. По умолчанию эта опция включена.
