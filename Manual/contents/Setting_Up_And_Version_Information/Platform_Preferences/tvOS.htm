ПредпочтенияtvOS

В настройках tvOS есть следующие параметры:

  • Автоматический ответ на предупреждение об ограничении постоянного хранилища tvOS: игры tvOS имеют ограниченное локальное хранилище для файлов, и при компиляции для этой цели по умолчанию будет показано предупреждение, напоминающее об этом факте. Вы можете изменить настройку здесь, чтобы отключить это предупреждение, выбрав "OK". По умолчанию установлено значение "Показать сообщение".
  • Подавить сборку и запуск: По умолчанию при создании проекта tvOS и последующей сборке конечного исполняемого пакета GameMaker Studio 2 будет собирать пакет, а затем пытаться запустить его на всех подключенных устройствах. Установка этого параметра отменить сборку приложения и его запуск на устройстве и просто создать проект XCode на главном компьютере Mac.
  • Путь установкиmacOS: Путь установки файлов проекта на компьютере Mac для сборки.
  • Идентификатор команды по умолчанию: Здесь вы можете добавить свой идентификатор команды по умолчанию, присвоенный вам компанией Apple. Этот идентификатор команды будет использоваться, когда файлы вашей игры будут отправлены в Xcode для создания приложения и позволит Xcode сгенерировать необходимые сертификаты подписи. Обратите внимание, что этот параметр будет применяться по умолчанию ко всем играм, созданным для tvOS, но его можно изменить для каждого проекта в разделе Общие параметры tvOS. tvOS.htm">Параметры игры.
