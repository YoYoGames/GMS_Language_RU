ПредпочтенияWindows

В настройках Windows есть следующие параметры:

  • РасположениеVisual Studio: Эта опция предназначена для указания GameMaker Studio 2 на установку Microsoft Visual Studio на компьютере сборщика.
  • Тип упаковки по умолчанию: При создании конечного исполняемого пакета для Windows вы можете выбрать между созданием инсталлятора (с помощью NSIS) и сжатым файлом ZIP . в котором будет содержаться игра и все дополнительные файлы. По умолчанию при компиляции вам будет показан диалог выбора между этими двумя вариантами, но вы можете установить это предпочтение, чтобы всегда использовать один или другой вариант. Обратите внимание, что когда эта опция установлена в Show Dialog, в реальном диалоговом окне, появляющемся при компиляции, включение опции "Remember Packaging Option" установит это предпочтение на выбранный вами вариант.
