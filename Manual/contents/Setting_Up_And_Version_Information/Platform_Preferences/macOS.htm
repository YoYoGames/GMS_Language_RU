ПредпочтенияmacOS

В настройках macOS есть следующие параметры:

  • Идентификатор команды по умолчанию: Здесь вы можете добавить свой идентификатор команды по умолчанию, присвоенный вам компанией Apple. Этот идентификатор команды будет использоваться, когда файлы вашей игры будут отправлены в Xcode для создания приложения и позволит Xcode генерировать необходимые сертификаты подписи. Обратите внимание, что этот параметр будет применяться по умолчанию ко всем играм, созданным для macOS, но его можно изменить для каждого проекта в разделе Общие параметры macOS. Параметры игры.
  • Автоматический выбор упаковки: Здесь вы можете установить предпочтение по умолчанию для создания исполняемого файла для цели macOS. Опция "Показать диалог" всегда будет показывать диалог перед созданием исполняемого файла, предлагая вам выбрать между DMG или Zip, но вы можете установить любой из этих вариантов в Предпочтениях, чтобы пропустить это диалоговое окно. Обратите внимание, что включение опции "Запомнить вариант пакета" в диалоге также установит этот параметр на выбранный тип пакета.
