Установка высоты звука

Это действие можно использовать для установки высоты тона звука. Вы предоставляете звуковой актив из браузера активов, а затем задаете желаемую высоту тона для изменения, и все последующие экземпляры этого звука, которые вы воспроизводите, будут иметь измененную высоту тона. Это действие не изменит высоту тона воспроизводимого в данный момент звука, поэтому его следует использовать до применения действия Play Audio для этого звука.

Значение высоты тона - это множитель высоты тона, то есть вводимое значение умножает текущую высоту тона на данное значение, поэтому значение по умолчанию 1 не изменяет высоту тона, а значение меньше 1 понижает высоту тона, а больше 1 повышает высоту тона. Для этой функции лучше использовать небольшие приращения, так как значение меньше 0 или больше 5 может быть не слышно. Стоит отметить, что общее допустимое изменение высоты тона ограничено (1/256) - 256 октавами, поэтому любое значение больше или меньше этого не будет зарегистрировано. зарегистрировано.

ПРИМЕЧАНИЕ: Приведенное выше значение зажатого диапазона - это то, к чему GameMaker Studio 2 пытается привязать диапазон, но это значение не гарантируется на всех целевых платформах. iOS, например, привязывается к (1/256) - 8, поэтому вам может потребоваться экспериментировать на каждой целевой платформе. и иметь разные версии звукового ресурса, каждая из которых предварительно сдвинута, если вам нужны более высокие или более низкие значения октавы.

Синтаксис действия:

Аргументы:

Аргумент Описание
Звук Звуковой ресурс для установки высоты тона
шаг Высота тона для установки звука (по умолчанию 1)

Пример:

Приведенный выше код блока действий сначала получает высоту тона для заданного звука и сохраняет ее во временной локальную переменную. Затем проверяется нажатие клавиши, и если оно обнаружено, то значение высоты тона проверяется на то, не меньше ли оно 2. Если да, то звук останавливается, к переменной высоты тона добавляется 0,1, и затем эта переменная используется для установки высоты тона звука снова перед воспроизведением. высоты тона звука перед его воспроизведением.
