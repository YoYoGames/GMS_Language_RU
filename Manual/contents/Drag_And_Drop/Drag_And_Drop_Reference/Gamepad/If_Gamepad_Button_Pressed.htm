При нажатии кнопки геймпада

С помощью этого действия вы можете опросить геймпад, чтобы узнать, нажата ли в данный момент какая-либо кнопка. Действие будет возвращать true на каждом шаге, когда кнопка геймпада нажата (и удерживается), и false в противном случае, хотя если вы нажмете на модификатор вы можете проверить, не нажата ли кнопка, т.е. действие будет возвращать true, если кнопка не нажата, и false, если нажата. Если вам нужно проверить нажатие только одной кнопки, то используйте действие if gamepad button down.

Если проверяемая кнопка является аналоговой, то нажатие не будет обнаружено, пока значение кнопки не превысит порог, установленный с помощью действия Set Gamepad Button Threshold.

Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже:
