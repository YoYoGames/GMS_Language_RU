Pop Stack

С помощью этого действия вы можете извлечь значение из верхней части структуры данных стека. Вы передаете переменную, которая содержит индекс стека (возвращается действием Create Stack), а затем указываете целевую переменную, которая должна содержать возвращаемую переменную (которая может быть помечена как временная локальная переменная).
