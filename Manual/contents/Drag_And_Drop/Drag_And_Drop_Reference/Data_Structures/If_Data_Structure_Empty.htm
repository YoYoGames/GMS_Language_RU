Если структура данных пуста

С помощью этого действия вы можете проверить, пуста ли заданная структура данных или нет. Сначала вам нужно выбрать тип структуры данных для проверки (либо список, либо карта, либо стек - никаких сеток, так как они не могут быть пустыми), а затем предоставить переменную переменную, содержащую индекс структуры данных (который возвращается действиями Create List, Create Map и Create Stack). Действие вернет true, если структура данных пуста, и false, если нет. Обратите внимание, что в действии можно поставить флаг "not", чтобы проверить, содержит ли структура данных данные, проверив, не пуста ли она.

Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже:
