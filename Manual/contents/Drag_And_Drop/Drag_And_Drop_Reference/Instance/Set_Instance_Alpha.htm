Set Instance Alpha

Этот блок действий устанавливает значение image_alpha для экземпляра. Альфа изображения - это значение "прозрачности" для отрисовки любого спрайта, назначенного данному экземпляру, и имеет значение от 0 до 1, где 0 - полностью прозрачный, а 1 - полностью непрозрачный. полностью прозрачным, а 1 - полностью непрозрачным. Обратите внимание, что если спрайт имеет прозрачные или полупрозрачные пиксели, то их прозрачность будет масштабироваться на эту величину, поэтому если в спрайте есть несколько 50% прозрачных пикселей, а альфа экземпляра установлена на 0,5, то конечная альфа будет равна 25%. для прозрачных пикселей будет равна 25%. Вы можете установить это значение как относительную величину

Обратите внимание, что для того, чтобы изменения в этой переменной были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker Studio 2 по умолчанию нарисует спрайт), либо быть нарисованным с помощью действия Draw Self .
