Set Sprite

Этот блок действий устанавливает sprite_index экземпляра на другой спрайт из браузера активов. Вы выбираете спрайт, на который нужно перейти, а затем указываете изображение (кадр анимации), которое нужно показать. Обратите внимание, что если вы не хотите, чтобы спрайт анимировался после установки кадра, вам нужно использовать действие Set Animation Speed и установить его на 0, или если вы хотите, чтобы спрайт продолжал анимировать, то установите значение для использования встроенной переменной image_index вместо этого.

Изменение спрайта также изменит маску столкновений экземпляра, если вы не указали отдельный mask_index в редакторе объектов. Обратите внимание, что для того, чтобы изменения в этом действии должны либо не иметь события рисования (и тогда GameMaker Studio 2 по умолчанию нарисует спрайт), либо быть нарисованы с помощью действия Draw Self.
