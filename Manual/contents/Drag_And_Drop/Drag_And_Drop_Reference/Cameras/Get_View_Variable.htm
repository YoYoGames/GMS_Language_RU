Получить переменную представления

Это действие позволяет получить значение определенных встроенных переменных, связанных с данным портом представления. Вы выбираете переменную порта представления для получения, затем указываете порт представления для цели (от 0 до 7) и, наконец, целевую переменную для хранения возвращаемого значения (она может быть отмечена как временная локальная переменная). (она может быть помечена как временная локальная переменная). Порт просмотра - это область экрана, которая будет использоваться для отрисовки данного вида камеры (как установлено в редакторе комнаты), и различные переменные, которые вы можете получить таким образом. приведены в таблице ниже:

Переменная Описание
Camera Возвращаемое значение будет уникальным значением ID камеры, назначенной порту (см. здесь для получения дополнительной информации).
Visibility Это вернет значение true или false, где true означает, что порт представления виден (т.е. он рисуется на экране), а false означает, что ничего не рисуется.
View Port X Coordinate Это X-положение порта просмотра в игровом окне.
View Port Y Coordinate Это положение по оси Y порта просмотра в игровом окне.
View Port Width Это ширина (в пикселях) порта просмотра.
View Port Height Это высота (в пикселях) порта просмотра.
View Port Surface ID Это вернет уникальное значение ID для поверхности, назначенной порту просмотра, или -1, если поверхность не была назначена.
