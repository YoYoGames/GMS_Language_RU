Draw Sprite Transformed

Эта функция рисует заданный спрайт аналогично действию Draw Sprite, но с дополнительными опциями для изменения масштаба, смешивания, вращения и рамки рисуемого спрайта. Изменение этих значений не изменяет ресурс никаким образом (только то, как он рисуется), и вы можете использовать любую из доступных переменных свойств спрайта вместо прямых значений для всех аргументов в функции. Изображение ниже иллюстрирует как различные значения влияют на отрисовку спрайта:

ПРИМЕЧАНИЕ: Смешивание цветов рекомендуется только для HTML5-цели, когда включен WebGL, хотя вы можете установить цвет смешивания, если он не включен, и он будет смешивать спрайт как обычно. Однако при таком смешивании создается дубликат спрайта, который затем сохраняется в кэше и используется при необходимости. Это далеко не оптимально, и если вы используете многократное изменение цвета, это замедлит производительность вашей игры, если вы не активируете WebGL. если вы не хотите использовать WebGL, то вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это в случае необходимости, используя функцию sprite_set_cache_size().

ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах.

Аргументы:

Аргумент Описание
Спрайт Спрайт для рисования
Рама Кадр спрайта для рисования (используйте встроенную переменную image_index для текущего кадра)
X Позиция x для рисования в пределах комнаты
Y Позиция y для рисования в пределах комнаты
Масштаб X Масштабный коэффициент для оси X (используйте встроенную переменную image_xscale для текущего масштаба X).
Масштаб Y Масштабный коэффициент для оси Y (используйте встроенную переменную image_yscale для текущего масштаба Y).
Вращение Угол рисования для спрайта (от 0 до 360, вращение против часовой стрелки, где 0 - вправо - используйте встроенную переменную image_angle для текущего вращения).
Цвет Цвет для смешивания со спрайтом (по умолчанию белый, а для получения текущего цвета можно использовать встроенную переменную image_blend ).
