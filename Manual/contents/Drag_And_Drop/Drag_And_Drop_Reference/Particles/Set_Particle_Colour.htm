Цвет частиц набора

Это действие задает цвет или цвета, которые будут использоваться при создании спрайта и в течение его жизни. Сначала вы указываете значение ID типа частицы (оно должно совпадать со значением, полученным при создании частицы с помощью действия Create Particle Type ), а затем вы можете задать цвета, которые будут использоваться.

Вы можете выбрать три цвета, и спрайт будет смешиваться между ними в течение времени жизни частицы. Например, если частица имеет красный, зеленый и желтый цвета и время жизни 60 игровых кадров, то когда она создается (кадр 0), она будет красным цветом, затем на половине своего времени жизни (кадр 30) она будет переходить в зеленый, а в конце жизни (кадр 60) - в желтый. Если вы используете пользовательский спрайт вместо формы частицы, любой цвет, кроме белого, будет сливаться с используемым спрайтом.

ПРИМЕЧАНИЕ: Альфа-компонент цвета не используется, только RGB-компоненты из палитры цветов. Для установки альфа-компонента используйте действие Set Particle Alpha.
