Set Particle Sprite

Это действие можно использовать для установки спрайта, который частица может использовать при рисовании. Вы предоставляете уникальное значение ID типа, для которого нужно установить спрайт (полученное при создании типа с помощью действия Create Particle Type) и ресурс спрайта для использования. Если спрайт имеет вложенные изображения, то частица будет анимировать кадры со скоростью, установленной в редакторе изображений для спрайта. Также обратите внимание, что если вы зададите значение цвета для частицы (с помощью действия Set Particle Colour ), то этот цвет будет смешиваться с используемым спрайтом, подобно тому, как установка смешивания изображения для экземпляра приводит к смешиванию цвета со спрайтом экземпляра (для отсутствия смешивания используйте белый цвет).

ПРИМЕЧАНИЕ: Если вы используете это действие, вы не должны использовать действие Set Particle Shape, так как вы можете иметь только спрайт или форму, но не оба.
