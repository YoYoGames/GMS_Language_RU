При нажатии клавиши

Это действие проверяет первоначальное нажатие на клавишу клавиатуры. Оно возвращает true на одном игровом шаге (кадре), когда клавиша нажата изначально, и false во всех остальных случаях. Если вы включите модификатор not, это действие будет обратным и будет проверять, не была ли нажата клавиша, возвращая false на том игровом шаге (кадре), на котором она была нажата, и true во всех остальных случаях. Если вам нужно проверить, удерживается ли клавиша нажатой, а не только начальное событие нажатия, то используйте действие If Key Down.

Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже:

Если проверка истинна при первоначальном нажатии, воспроизводится звук, затем, пока клавиша удерживается нажатой, смешивание спрайта изменяется на красное. Если клавиша не удерживается, проверка выполняется при отпускании клавиши, чтобы сбросить цвет смешивания на белый.
