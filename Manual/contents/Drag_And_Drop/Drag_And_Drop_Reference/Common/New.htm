Новый

Это действие используется толькопри создании структур с помощью функции, которая была помечена как функция-конструктор (см. раздел "Объявление новой функции " для получения дополнительной информации). Это действие принимает метод функции для ранее определенной функции, а также ряд аргументов, которые при необходимости можно увеличить, используя значок слева. Эти аргументы должны соответствовать входным данным, требуемым функцией, и будут использоваться для заполнения создаваемой структуры. Структура будет возвращена в целевую переменную, которая может быть помечена как временная локальная переменной или нет.

Аргументы:

Аргумент Описание
Сценарий Имя сценария или определяемой пользователем функции для вызова.
Аргумент0 ... Аргумент3 Различные аргументы (значения), которые должны быть переданы скрипту или функции (неиспользуемые аргументы можно оставить пустыми)
Цель Имя переменной, которая будет использоваться для возвращаемых значений (можно оставить пустым).
