Выполнить код

Это действие просто добавляет блок редактора кода в список DnD™, куда вы можете добавить любой допустимый GML-код, необходимый для объекта. Обратите внимание, что весь код в действии будет выполнен до того, как будет выполнено следующее до того, как будет выполнено следующее действие в списке. Чтобы было легче отследить, что делают эти действия, вы можете назвать их с помощью идентификатора JSDoc @description, как показано на рисунке ниже. на рисунке ниже:
