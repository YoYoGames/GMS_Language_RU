Если оценка

Это действие используется для проверки значения переменной экземпляра score с помощью определенного выражения. Вы указываете тип выражения для проверки и значение для проверки текущего значения, а оператор "if" будет всегда будет возвращать либо true, либо false в зависимости от используемых выражений и значений. Доступными выражениями являются:

  • Equals to - Переменная и значение в точности равны.
  • Меньше - Переменная меньше значения
  • Больше чем - Переменная больше, чем значение
  • Less than or Equal to - Переменная меньше или равна значению
  • Greater than or Equal to - Переменная больше или равна значению

Если вы отметите аргумент Not, то вышеприведенные выражения будут отрицательными, например, "equals to" станет " not equals to",таким образом, вы будете проверять, не равно ли значение оценки заданному значению.

ВАЖНОЕ ПРИМЕЧАНИЕ: Из-за проблем с точностью плавающей точки, проверка на точное равенство двух значений может вернуть ложное значение, так как одно из них может быть равно 1, а другое - 1.00000000000001. другое может быть 1.00000000000001. Этого можно избежать, используя действие Decimal to Integer перед проверкой или используя выражения "больше чем" или "меньше чем".
