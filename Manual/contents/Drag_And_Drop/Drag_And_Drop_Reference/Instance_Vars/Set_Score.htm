Зачет по сету

С помощью этого действия вы можете создать или установить переменную экземпляра для оценки. Вы указываете значение, которое нужно либо установить в переменную score, либо добавить к переменной score (если вы отметите флажок relative, то значение будет добавлено к текущему значению "score"). значение). Обратите внимание, что в отличие от обычных переменных экземпляра имя переменной "score" хранится внутри, поэтому для получения значения score необходимо использовать действие Get Score.
