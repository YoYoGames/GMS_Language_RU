Окно инспектора

Окно Inspector - это окно, в котором отображаются все свойства, связанные с выбранным элементом в IDE. В окне отображается информация об инспектируемом элементе, а также все редактируемые свойства этого элемента и позволяет изменять их по мере необходимости, обновляя инспектируемый актив в соответствии с ними.

ВАЖНО! В настоящее время окно инспектора доступно только для проверки свойств последовательности, но в будущих обновлениях оно будет расширено для включения других активов.

Вы можете открыть несколько окон инспекторов в любое время, и каждое из них может быть связано с определенным элементом в IDE. Чтобы при каждом щелчке на элементе все открытые инспекторы не менялись, показывая новый выбор, необходимо сначала заблокировать окно инспектора. Для этого нужно открыть инспектор, выбрать элемент IDE для проверки, а затем нажать кнопку блокировки на инспекторе, чтобы заблокировать его на элементе (показано на рисунке ниже). После того как вы заблокировали инспектор, вы можете открыть другой, а затем повторить процесс.

Обратите внимание, что заблокированное окно инспектора не блокирует редактирование каких-либо значений для проверяемого элемента. Оно просто блокирует инспектор на элемент, и вы можете продолжать редактировать его в любое время, даже если он не находится в фокусе основной IDE.
