Обзор языка GameMaker

В этом разделе руководства содержится вся информация, необходимая для понимания и использования программы GameMaker Studio 2 язык сценариев GML. Функции времени выполнения на этом языке могут быть использованы для создания ваших игр и добавляются в объекты из редактор объектовобычно когда вы решили создать новый проект. GameMaker Language Project хотя он также может быть использован вместе с DnD™(Drag and Drop). Ниже показано типичное изображение объекта с открытым редактором кода на событии:

Каждое событие имеет свою собственную вкладку в редакторе, и вы можете добавлять, редактировать или удалять код из них в любое время (для получения дополнительной информации о событиях см. События объекта). Сам код должен иметь базовую структуру и может содержать индексы ресурсов, переменные, функции, выражения, ключевые слова... и т.д... все это объясняется в следующих разделах. Если вы новичок в программировании или переходите с DnD™, рекомендуется начать со страницы Базовая структура кода а затем прочитать все остальные страницы этого раздела, проверяя код каждой из них в рамках GameMaker Studio 2 в самой GameMaker Studio 2.
