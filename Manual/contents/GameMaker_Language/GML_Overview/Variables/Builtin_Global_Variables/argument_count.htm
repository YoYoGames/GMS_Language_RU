счётчик аргументов

Эта переменная, доступная только для чтения, возвращает количество "аргументов", передаваемых функции скрипта или методу. Обычно используется в сочетании с массивом аргументов (argument[0...15]), чтобы позволить варьировать входные аргументы для данной функции.

ПРИМЕЧАНИЕ: Это больше не является обязательным требованием и предоставляется больше для поддержки наследия, чем для чего-либо еще. Все определяемые пользователем функции будут принимать переменные аргументы, и любой аргумент, переданный в функцию, который не является частью именованных входа, по умолчанию будет инициализирован как undefined, что можно проверить с помощью функции is_undefined().

Синтаксис:

argument_count;

Возвращается:

Целое число

Пример:

function my_array()
{
var i, arg;
for (i = 0; i < 5; i += 1;)
{
if argument_count > i
{
arg[i] = argument[i]
}
else
{
arg[i] = -1;
}
}
}

Приведенный выше код использует переменную argument_count вместе с массивом argument для инициализации переменных и является примером того, как разрешить пользовательской функции принимать переменное количество аргументов.
