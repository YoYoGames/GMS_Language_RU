здоровье

Эта переменная является глобальной и используется для хранения числового значения, которое обычно используется для здоровья игрока. Эта переменная предназначена только для поддержки старых проектов предыдущих версий программы GameMaker и должна не использовать в новых проектах поскольку в будущем она может быть устаревшей.

Синтаксис:

здоровье;

Возвращается:

Real (значение с плавающей точкой одинарной точности)

Пример

if (health <= 0)
{
global.state = "Game Over";
instance_destroy();
}

Приведенный выше код проверяет переменную health, и если она меньше или равна 0, устанавливается глобальная переменная и экземпляр уничтожается.
