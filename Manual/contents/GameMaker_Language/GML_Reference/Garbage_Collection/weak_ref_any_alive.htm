weak_ref_create

С помощью этой функции вы можете проверить слабые ссылки на различные структуры, чтобы узнать, живы они еще или нет. Вы передаете массив слабых ссылок для проверки (каждая слабая ссылка должна быть создана с помощью функции должна быть создана с помощью функции weak_ref_create()), и функция вернет true, если любой структуры все еще или false, если это не так и они были собраны в мусор. Обратите внимание, что если вы предоставите массив, в котором любое из значений не является слабой ссылкой, функция вернет undefined.

Обратите внимание, что функция также принимает два необязательных аргумента, первый из которых позволяет указать начальный индекс массива для проверки, а также аргумент, задающий количество позиций (длину) от этого индекса для проверки. Проверяться будут только индексы массива в пределах этих параметров будут проверены.

Синтаксис:

weak_ref_any_alive(weak_ref, [index], [length]);

Аргумент Описание
слабый_реф Слабая ссылка на структуру, которую вы хотите проверить.
[индекс]. ВАРИАНТ! Индекс в массиве, с которого следует начать проверку
[длина] ВАРИАНТ! Количество позиций от заданного индекса для проверки

Возвращается:

Булево (или неопределенное)

Пример:

if weak_ref_any_alive(inventory_ref_array)
{
instance_destroy(obj_Inventory_Control);
}

Приведенный выше код проверяет массив слабых ссылок, и если какая-либо из них еще жива, то экземпляр уничтожается.
