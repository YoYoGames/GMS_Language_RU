gc_is_enabled

С помощью этой функции вы можете проверить, включен ли сборщик мусора или нет. Функция вернет true, если он включен, и false в противном случае.

Синтаксис:

gc_is_enabled();

Возвращается:

Bool

Пример:

if (!gc_is_enabled())
{
gc_enable(true);
}

Приведенный выше код проверяет, включен ли сборщик мусора, и если нет, то включает его.
