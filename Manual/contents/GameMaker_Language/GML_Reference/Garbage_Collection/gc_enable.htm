gc_enable

С помощью этой функции вы можете включить или отключить сборщик мусора. Вызов функции с аргументом true включает его, а с помощью false отключает (не рекомендуется). По умолчанию он включен.

Синтаксис:

gc_enable(enable);

Аргумент Описание
включить включить(true) или отключить(false) сборщик мусора.

Возвращается:

Н/Д

Пример:

if (global.debug == true)
{
gc_enable(false);
}

Приведенный выше код отключает сборку мусора, если заданная глобальная переменная равна true.
