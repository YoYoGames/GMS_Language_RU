steam_get_user_persona_name

Эта функция может быть использована для получения имени пользователя (экранного имени) для любого определенного ID пользователя. Вы указываете ID пользователя, после чего функция возвращает уникальное значение async ID, которое может быть использовано для правильной проверки деталей в событииSteam Async. Это событие будет иметь async_load DS map, которая будет заполнена следующими парами ключ/значение:

  • " id" - Идентификатор async, возвращаемый функцией, которая вызвала событие
  • " event_type" - Строка " user_persona_name"
  • " steamid""- Значение Steam ID, используемое для получения имени пользователя.
  • "persona_name" - Имя пользователя для данного Steam ID

Синтаксис:

steam_get_user_persona_name(steamID);

Аргумент Описание
steamID Уникальный идентификатор Steam для пользователя.

Возвращается:

Реально

Пример:

request = steam_get_user_persona_name(global.UGC_UserID);

Приведенный выше код запросит имя пользователя ID пользователя, хранящегося в глобальной переменной "UGC_UserID", сохраняя возвращаемое значение в переменной для разбора в Async Event.
