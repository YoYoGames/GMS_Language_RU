steam_get_user_steam_id

Вы можете использовать эту функцию для возврата уникального идентификатора пользователя Steam, который в настоящее время вошел в клиент Steam. Это уникальное 64-битное значение, которое можно использовать для идентификации отдельных пользователей, а не их экранного имени пользователя (его можно узнать с помощью функции найти с помощью функции steam_get_persona_name()). Поскольку это int64, вам может понадобиться привести его к этому значению при использовании в последующих функциях с помощью функции int64() функция.

Синтаксис:

steam_get_user_steam_id();

Возвращается:

Целое число (int64)

Пример:

if steam_initialised()
{
global.u_id = steam_get_user_steam_id();
}

Приведенный выше код установит глобальную переменную на уникальный Steam ID текущего пользователя, если API клиента Steam правильно инициализирован.

Назад: Общий API Steam
