steam_get_persona_name

Вы можете использовать эту функцию для возврата имени пользователя, который в настоящее время вошел в клиент Steam. Это видимое экранное имя, а не уникальный идентификатор пользователя (его можно узнать с помощью функции steam_get_user_steam_id()).

Синтаксис:

steam_get_persona_name();

Возвращается:

Строка

Пример:

if steam_initialised()
{
global.p_name = steam_get_persona_name();
}

Приведенный выше код установит глобальную переменную на текущее имя экрана пользователя, если API клиента Steam правильно инициализирован.

Назад: Общий API Steam
Далее: steam_get_user_persona_name
