steam_available_languages

Эта функция используется для получения списка всех доступных языков для Steam. Возвращаемое значение - это просто список языков, разделенных запятыми.

Синтаксис:

steam_available_languages();

Возвращается:

Строка

Пример:

язык = steam_available_languages();

Вышеприведенный пример получает доступные языки для Steam в виде строки и сохраняет ее в переменной.

Назад: Общий API Steam
