Общий паровой API

Следующий набор функций предназначен для проверки доступности определенных аспектов API клиента или сервера Steam и, соответственно, возвращает true или false в зависимости от факторов. например, занят ли сервер, вошел ли пользователь в систему и т.д.. Это означает, что эти функции следует использовать перед любым другим вызовом функций Steam API, чтобы убедиться, что клиент/сервер настроен правильно и взаимодействует с вашей игрой:

  • steam_initialised
  • steam_stats_ready
  • steam_get_app_id
  • steam_get_user_account_id
  • steam_get_user_steam_id
  • steam_get_persona_name
  • steam_get_user_persona_name
  • steam_is_user_logged_on
  • steam_current_game_language
  • steam_available_languages
