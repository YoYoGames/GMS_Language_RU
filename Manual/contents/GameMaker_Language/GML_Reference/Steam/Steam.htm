Пар

Этот раздел предназначен для тех пользователей, которые получили доступ к API Steam для публикации вашей игры на этой платформе. Чтобы иметь возможность использовать эти функции, вы должны быть приняты в Steam ранее, либо через издателя, либо через через систему самостоятельной публикации, а также правильно настроить параметрыSteam Preferences и Steam Game Options.

Существует огромное количество различных функций, связанных с API Steam, поэтому мы разделили их на следующие разделы, чтобы было легче ориентироваться:

  • Общий паровой API
  • Наложение пара
  • Таблицы лидеров
  • Достижения и статистика
  • Облакопара
  • DLC (загружаемый контент)
  • UGC (User Generated Content)
