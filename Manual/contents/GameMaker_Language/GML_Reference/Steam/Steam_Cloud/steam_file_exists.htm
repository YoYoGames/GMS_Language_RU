steam_file_exists

С помощью этой функции вы можете проверить, существует ли файл из Steam Cloud или нет, с возвращаемым значением true, если он существует, или false в противном случае.

Синтаксис:

steam_file_exists(filename);

Аргумент Описание
имя файла Имя файла для проверки.

Возвращается:

Булево

Пример:

if steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt");
}

Приведенный выше код проверяет, существует ли файл на Steam Cloud, и если да, то открывает его и считывает содержимое в переменную "save_str".
