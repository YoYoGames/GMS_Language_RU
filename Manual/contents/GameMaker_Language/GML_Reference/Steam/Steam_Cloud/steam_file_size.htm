размер_файла

С помощью этой функции вы можете проверить размер файла, хранящегося в Steam Cloud. Возвращаемое вещественное число - это размер файла в байтах.

Синтаксис:

steam_file_size(filename);

Аргумент Описание
имя файла Имя файла, размер которого необходимо проверить.

Возвращается:

Реально

Пример:

file_bytes = steam_file_size("Save.txt");

Приведенный выше код сохраняет размер файла из Steam Cloud в переменной "file_bytes".
