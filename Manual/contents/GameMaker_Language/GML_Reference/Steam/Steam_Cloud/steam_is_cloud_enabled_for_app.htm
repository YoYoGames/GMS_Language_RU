steam_is_cloud_enabled_for_app

С помощью этой функции вы можете проверить, включен ли сервис Steam Cloud для вашей игры. Она вернет true, если это так, и false в противном случае. Обратите внимание, что это не означает, что вы автоматически можете использовать функции Cloud поскольку пользователь может отключить синхронизацию с облаком в своем клиенте Steam. Проверить это можно с помощью функции steam_is_cloud_enabled_for_account(), но даже если она отключена для пользователя (и включена для игры), функции все равно будут работать для хранения и получения данных из локальной копии всех файлов, они просто не будут загружать их в облако по окончании игры или синхронизировать при запуске игры.

Синтаксис:

steam_is_cloud_enabled_for_app();

Возвращается:

Булево

Пример:

if steam_is_cloud_enabled_for_app()
{
quota = steam_get_quota_total();
}

Приведенный выше код проверяет, включено ли облако steam для игры, и если да, то получает размер квоты хранения и сохраняет его в переменной.
