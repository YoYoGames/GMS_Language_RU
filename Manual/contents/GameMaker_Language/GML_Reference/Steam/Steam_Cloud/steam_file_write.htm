steam_file_write

Вы можете использовать эту функцию для записи данных в файл, который затем будет синхронизирован со Steam Cloud, когда пользователь выйдет из игры. если файл не существует, эта функция создаст его для вас, а если он уже существует, она перезапишет все данные, которые уже хранятся в файле, новой строкой данных. Функция вернет значение 0 в случае неудачи по какой-либо причине и значение больше 0 в случае успеха.

Синтаксис:

steam_file_write(filename, data, size);

Аргумент Описание
имя файла Имя файла для записи.
данные Данные для записи (строка).
размер размер записываемых данных.

Возвращается:

Реально

Пример:

var fname="SaveData.txt";
var data = string(global.Level) + "|" + string(global.Points) + "|" + string(global.HP);
var len = string_length(data);
steam_file_write_file(fname, data, len);

Приведенный выше код подготовит ряд локальных переменных, а затем использует их для записи в файл (или создания файла), который затем будет синхронизирован с Steam Cloud.
