steam_file_read

Эта функция считывает содержимое заданного файла в строку, которая впоследствии может быть разобрана в вашей игре.

Синтаксис:

steam_file_read(filename);

Аргумент Описание
имя файла Имя файла для чтения.

Возвращается:

Строка

Пример:

if steam_file_exists("Save.txt")
{
save_str = steam_file_read("Save.txt");
}

Приведенный выше код проверяет, существует ли файл на Steam Cloud, и если да, то открывает его и считывает содержимое в переменную "save_str".
