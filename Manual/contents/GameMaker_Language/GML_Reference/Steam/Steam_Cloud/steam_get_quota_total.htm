steam_get_quota_total

При использовании Steam Cloud для хранения и синхронизации файлов вы должны установить квоту пространства, которое потребуется вашей игре. Эта квота устанавливается для каждой игры, поддерживающей облачное хранилище, по принципу "каждый пользователь - каждой игре", поэтому, например, если квота для игры X составляет 1 мегабайт, то каждая учетная запись Steam, владеющая игрой X, может хранить в облаке не более 1 мегабайта данных, связанных с этой игрой. Любые другие игры с поддержкой Cloud, которыми владеет пользователь (например, игра Y), не будут затронуты данными, хранящимися в Game X. По умолчанию квота для новых игр Steamworksсоставляет один гигабайт, но вы можете изменить это значение в панели управления Steamworksдля вашей игры.

ПРИМЕЧАНИЕ: Как только квота будет исчерпана, запись файлов завершится неудачей. Если вы считаете, что квота может быть исчерпана пользователем вашей игры, вам следует создать код, чтобы справиться с этим, так как, ничего не делая, вы оставляете пользователей в ситуации, когда они не смогут исправить ситуацию, что приведет к плохому восприятию вашей игры.

Синтаксис:

steam_get_quota_total();

Возвращается:

Реально

Пример:

if steam_is_cloud_enabled_for_app()
{
quota = steam_get_quota_total();
}

Приведенный выше код проверяет, включено ли облако steam для игры, и если да, то получает размер квоты хранения и сохраняет его в переменной.
