Облакопара

Облако Steam Cloud обеспечивает простую и прозрачную систему удаленного хранения файлов для вашей игры. Все файлы, записанные на диск с помощью облачных функций GameMaker Studio 2 (перечисленных ниже), будут реплицированы на серверы Steam после выхода игры. Если пользователь сменит компьютер, файлы будут загружены на новый компьютер перед запуском игры, то есть игра сможет получить доступ к файлам, прочитав их с помощью соответствующих функций Steam. Клиент Steam выполняет работу обеспечивая синхронизацию файлов на всех компьютерах, к которым обращается пользователь.

ПРИМЕЧАНИЕ: По умолчанию облако не включено для игры на Steamworks. его необходимо включить предварительно на вкладке "Облако" в админке игры Steamworks, где необходимо установить квоту байтов и файлов. В следующий раз, когда вы опубликуете свою игры конфигурацию Steamworks, облачное хранилище будет готово к использованию.

Для доступа к Steam Cloud из GameMaker Studio 2можно использовать следующие функции

  • steam_is_cloud_enabled_for_app
  • steam_is_cloud_enabled_for_account
  • steam_get_quota_total
  • steam_get_quota_free
  • steam_file_exists
  • размер_файла
  • steam_file_persisted
  • steam_file_write
  • steam_file_write_file
  • steam_file_read
  • steam_file_share
  • steam_file_delete
