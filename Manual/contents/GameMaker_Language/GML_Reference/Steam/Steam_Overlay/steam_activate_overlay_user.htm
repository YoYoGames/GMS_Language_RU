steam_activate_overlay_user

Эта функция откроет в оверлее Steam один из выбранных диалогов, относящихся к указанному идентификатору пользователя. Доступны следующие возможные диалоговые строки:

  • " steamid" - открывает веб-браузер сообщества Steam на страницу пользователя или группы
  • " chat"- открыть окно чата для указанного пользователя

Обратите внимание, что идентификаторы Steam могут быть большими числами, поэтому вам может понадобиться привести значение вашего идентификатора к виду int64() перед тем, как передать его в функцию.

Синтаксис:

steam_activate_overlay_user(dialog_name, steamid);

Аргумент Описание
имя_диалога Диалог, на котором нужно открыть наложение (см. ниже).
steamid Идентификатор пользователя Steam или идентификатор группы для использования.

Возвращается:

Н/Д

Пример:

var key = keyboard_lastkey;
switch (key)
{
case vk_f1: steam_activate_overlay_user(" steamid",global.GameGroupID); break;
case vk_f2: steam_activate_overlay_user("chat", global.FriendID); break;
}

Приведенный выше код опрашивает последнюю нажатую клавишу клавиатуры, и если это функциональная клавиша 1 или функциональная клавиша 2, он откроет оверлей Steam, чтобы либо увидеть группу Steam, сохраненную в глобальной переменной "GamegroupID", либо откроет окно чата, чтобы чата с пользователем, хранящимся в глобальной переменной "FriendID".
