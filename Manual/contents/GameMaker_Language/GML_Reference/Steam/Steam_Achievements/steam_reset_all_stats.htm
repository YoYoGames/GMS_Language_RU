steam_reset_all_stats

С помощью этой функции вы можете сбросить все статистические данные для текущего пользователя до значений по умолчанию (как определено в панели управления Steamworksдля вашей игры). Рекомендуется использовать эту функцию только в качестве отладочного инструмента при разработке вашей игры.

Синтаксис:

steam_reset_all_stats();

Возвращается:

Н/Д

Пример:

ini_open("Save.ini");
if global.Version != ini_read_real("Data", "Version", 0)
{
ini_write_real("Data", "Version", global.Version);
steam_reset_all_stats ();
}
ini_close();

Приведенный выше код проверяет сохраненное значение в ini-файле на соответствие значению глобальной переменной, и если они отличаются, то сбрасывает статистику игры.
