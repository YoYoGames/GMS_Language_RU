steam_get_stat_float

С помощью этой функции вы можете получить значение определенной статистики с плавающей точкой. Статистика должна быть предварительно определена на странице учетных записей панели управления Steamworksдля вашей игры, а строка, передаваемая в функцию, должна совпадать со строкой, используемой в качестве API Name на панели управления.

Синтаксис:

steam_get_stat_float(stat_name);

Аргумент Описание
имя_статьи Имя статистики, которую нужно получить (строка).

Возвращается:

Реально

Пример:

var dist_pc = (dist / dist_max) * 100;
if steam_get_stat_float("Travelled") < dist_pc
{
steam_set_stat_int("Travelled", dist_pc);
}

Приведенный выше код вычисляет процент на основе переменной пройденного расстояния "dist" и максимального расстояния, которое вы можете проехать "dist_max". Затем он опрашивает текущее значение статистики "Travelled" и, если оно меньше чем рассчитанное значение, он снова устанавливает статистику.
