steam_ugc_create_query_user

Эта функция может быть использована для запроса базы данных пользовательского контента. Функция автоматически использует идентификатор по умолчанию для приложения, пользователя и предполагает, что запрос выполняется потребителем (а не создателем). Функция требует, чтобы вы использовали следующие константы для типа данных для запроса, типа элемента для запроса и порядка, в котором будут отсортированы возвращаемые элементы, а также номер страницы - обратите внимание, что запрос вернет максимум 50 элементов.

Функция возвращает уникальное значение хэндла запроса, которое должно быть сохранено в переменной для использования в других функциях запроса. Обратите внимание, что эта функция только подготавливает запрос, но не отправляет его - для этого вы должны вызвать функцию steam_ugc_send_query.htm">steam_ugc_send_query() - и вы можете использовать другие функции steam_ugc_query_* () для уточнения поискового запроса перед его отправкой.

Синтаксис:

steam_ugc_create_query_user(list_type, match_type, sort_order, page);
