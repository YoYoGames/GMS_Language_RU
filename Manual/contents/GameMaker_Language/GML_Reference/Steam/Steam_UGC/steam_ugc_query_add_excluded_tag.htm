steam_ugc_query_add_excluded_tag

Эта функция может быть использована для дальнейшей фильтрации любого заданного запроса пользовательского контента, в частности, для исключения заданного пользовательского контента из запроса. Ручка запроса - это значение, возвращенное при создании запроса (например, с помощью, steam_ugc_create_query_user()) а второй аргумент - это строка, которую вы хотите использовать в качестве тега для исключения. Функция вернет true, если фильтр запроса был установлен правильно, и false в противном случае.

Синтаксис:

steam_ugc_query_add_excluded_tag(ugc_query_handle , tag_name);

Аргумент Описание
ugc_query_handle Ручка запроса, которую следует использовать.
имя_тега Имя тега для исключения (строка).

Возвращается:

Булево

Пример:

var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1);
steam_ugc_query_add_excluded_tag(query_handle, "симулятор ходьбы");
steam_ugc_query_set_return_long_description(query_handle, true);
steam_ugc_query_set_allow_cached_response(query_handle, true);
query_ID = steam_ugc_send_query(query_handle);

Приведенный выше код создает запрос запроса и сохраняет его хэндл в локальной переменной для дальнейшего использования в остальных функциях, которые определяют запрос перед отправкой запроса.
