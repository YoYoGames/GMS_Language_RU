steam_user_owns_dlc

Если для вашей игры созданы DLC, вы можете использовать эту функцию, чтобы проверить, купил ли пользователь их, прежде чем получить доступ к любым связанным с ними файлам. Функция возвращает true, если игрок владеет содержимым, и false, если но обратите внимание, что даже если пользователь владеет DLC, он может не иметь его установленным, поэтому вам следует использовать дополнительную функцию steam_user_installed_dlc() чтобы проверить, установлено ли оно перед использованием. Обратите внимание, что идентификаторы Steam могут быть большими числами, поэтому вам может понадобиться привести значение идентификатора к виду int64() перед передачей в функцию.

Синтаксис:

steam_user_owns_dlc(dlc_id);

Аргумент Описание
dlc_id Уникальный идентификатор для проверяемого DLC (целое число int64).

Возвращается:

Булево

Пример:

global.Level_Max = 100;
if steam_user_owns_dlc(10354)
{
if steam_user_installed_dlc(10354)
{
global.Level_max = 200;
}
}

Приведенный выше код проверит, купил ли пользователь и установил DLC с идентификатором 10354, и если да, то установит глобальную переменную на другое значение.
