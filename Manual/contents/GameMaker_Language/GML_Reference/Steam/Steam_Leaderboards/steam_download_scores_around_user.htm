steam_download_scores_around_user

Эта функция используется для извлечения записей в таблице лидеров относительно записи текущего пользователя. Параметр range_start - это количество записей, которые нужно получить до записи текущего пользователя, а параметр range_end - это количество записей после записи текущего пользователя. записей после записи текущего пользователя, и запись текущего пользователя всегда включается в результаты. Например, если текущий пользователь находится на 5 месте в таблице лидеров, то при установке начального диапазона в -2 и конечного диапазона в 2, то в результате будет получено 5 записей: с 3 по 7. Если в таблице лидеров недостаточно записей до или после записи пользователя, Steam скорректирует диапазон, чтобы попытаться вернуть запрашиваемое количество записей. Например, если пользователь находится на первом месте в таблице лидеров, start установлено значение -2, а end установлено значение 2, Steam вернет первые 5 записей в таблице лидеров.

Синтаксис:

steam_download_scores_around_user(lb_name, range_start, range_end);

Аргумент Описание
lb_name Название таблицы лидеров, с которой вы загружаете результаты (строка).
начало диапазона Стартовая позиция в таблице лидеров.
конец диапазона Конечная позиция в таблице лидеров.

Возвращается:

Реально

Пример:

score_get = steam_download_scores_around_user("Game Scores", -4, 5);

Это событие отправит запрос на сервер Steam для получения диапазона из 10 оценок из таблицы лидеров "Game Scores", центрированной на игроке, и сохранит async id запроса в переменной "score_get". Затем этот запрос будет обрабатываться в Асинхронное событиеSteam как показано в расширенном примере для steam_download_scores().
