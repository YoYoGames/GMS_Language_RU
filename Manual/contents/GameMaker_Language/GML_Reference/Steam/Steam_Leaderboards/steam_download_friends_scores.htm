steam_download_friends_scores

С помощью этой функции вы можете получить только те оценки в таблице лидеров, которые принадлежат тем людям, которые отмечены как "друзья" в клиенте Steam . Так, если в вашей таблице лидеров 200 записей, и 50 из них - ваши друзья, эта функция получит только эти 50 результатов. Имя таблицы лидеров - это строка, которая была задана при создании таблицы лидеров с помощью функции steam_create_leaderboard(), а функция вернет значение, которое затем можно использовать для идентификации обратного вызова в Steam Async Event, или вернет -1 в случае неудачи.

Синтаксис:

steam_download_friends_scores(lb_name);

Аргумент Описание
lb_name Название таблицы лидеров, с которой вы загружаете результаты (строка).

Возвращается:

Реально

Пример:

score_get = steam_download_friends_scores("Game Scores");

Это отправит запрос на сервер Steam для получения оценок друзей пользователя из данной таблицы лидеров и сохранит async id запроса в переменной "score_get". Затем этот запрос будет обработан в событииSteam Async Event, как показано в расширенном примере steam_download_scores().
