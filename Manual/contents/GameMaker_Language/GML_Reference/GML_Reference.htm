Справочник поязыку GameMaker

Этот раздел руководства представляет собой справочное руководство по GameMaker Studio 2 Язык(GML). Здесь описаны все доступные функции с указанием необходимых аргументов и примерами кода, демонстрирующими их использование. Для получения информации о том, как использовать GML, пожалуйста, обратитесь к разделу ОбзорGML раздел руководства.

ПРИМЕЧАНИЕ: Везде, где в руководстве в качестве возвращаемого значения функции указано "N/A", это означает, что функция не предназначена для возврата какого-либо значения в рамках своей работы. Если вы попытаетесь проверить возвращаемое значение из этих функций, то вы вы получите значение undefined.
