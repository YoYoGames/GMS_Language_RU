дата_действительного_времени

С помощью этой функции вы можете проверить значение времени даты, чтобы узнать, является ли оно действительным (возвращается true) или нет (возвращается false). Обратите внимание, что эта функция будет считать действительным только то значение времени после 1/1/1970, а все, что раньше, вернет false. возвращает false, поэтому самое раннее, что вы можете проверить, это:

date_valid_datetime(1970, 01, 01, 0, 0, 0, 0);

Синтаксис:

date_valid_datetime(year, month, day, hour, minute, second);

Аргумент Описание
год Год для проверки.
месяц Месяц для проверки.
день День проверки.
час Час для проверки.
минута Минута проверки.
второй Второй для проверки.

Возвращается:

Булево

Пример:

if date_valid_datetime(2011, 9, 15, 10, 3, 30)
{
mydatetime = date_create_datetime(2011, 9, 15, 10, 3, 30);
}

Это установит "mydatetime" в 15 сентября 2011 года, 10:03.30, если это допустимое значение (а это так).
