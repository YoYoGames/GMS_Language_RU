дата_установить_временную зону

Устанавливает базовый часовой пояс, который будет использоваться для всех остальных функций даты и времени. Этот часовой пояс может быть либо локальным (установленным системой), либо UTC, и вы можете использовать одну из следующих констант, чтобы определить, какой из них используется (по умолчанию это местное время):

Постоянная Описание
часовой пояс_местный использовать местный часовой пояс, установленный системой
часовой пояс_utc использовать всемирное координированное время

Синтаксис:

date_set_timezone(timezone);

Аргумент Описание
часовой пояс Часовой пояс, который будет использоваться для базового времени.

Возвращается:

Н/Д

Пример:

if (date_get_timezone() != timezone_utc)
{
date_set_timezone(timezone_utc);
}

Этот код проверяет базовую настройку часового пояса для игры и, если она не UTC, то изменяет ее.
