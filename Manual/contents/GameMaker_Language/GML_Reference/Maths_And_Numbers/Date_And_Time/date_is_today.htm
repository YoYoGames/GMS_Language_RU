дата_сегодня

Эта функция возвращает true, если заданное значение времени является днем, на который производится проверка (т.е. сегодня), и false в противном случае. Это может быть удобной функцией для таких вещей, как пасхальные яйца в ваших играх, или для разблокировки сезонного контента. Обратите внимание, что на эту функцию влияет установленный часовой пояс (по умолчанию это местное время), который можно изменить с помощью функции date_set_timezone() функция.

Синтаксис:

date_is_today(date);

Аргумент Описание
дата Используемое время даты.

Возвращается:

Булево

Пример:

if date_is_today(global.Halloween)
{
global.Max_Levels = 200;
}

Приведенный выше код проверит время, хранящееся в глобальной переменной "Halloween", чтобы узнать, сегодня ли это, и если да, то установит другую глобальную переменную на новое значение.
