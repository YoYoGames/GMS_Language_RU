дата_високосного_года

Эта функция возвращает true, если компонент year заданного значения datetime является високосным годом, и false в противном случае. Эта функция может быть удобна для таких вещей, как пасхальные яйца в ваших играх или для разблокировки специального содержимого.

Синтаксис:

дата_високосного_года(дата);

Аргумент Описание
дата Используемое время даты.

Возвращается:

Булево

Пример:

if date_leap_year(date_current_datetime())
{
if !global.ExtraContent
{
global.ExtraContent = true;
}
}

Приведенный выше код проверяет текущую дату, чтобы узнать, является ли год високосным или нет. Если это так, то устанавливается глобальная переменная.
