дата_get_timezone

Получает базовый часовой пояс, используемый для всех остальных функций даты и времени. Этот часовой пояс может быть либо локальным (установленным системой), либо UTC, и функция возвращает одну из следующих констант:

Постоянная Описание
часовой пояс_местный использовать местный часовой пояс, установленный системой
часовой пояс_utc использовать всемирное координированное время

Синтаксис:

date_get_timezone();

Возвращается:

Постоянная

Пример:

if (date_get_timezone() != timezone_utc)
{
date_set_timezone(timezone_utc);
}

Этот код проверяет базовую настройку часового пояса для игры и, если она не UTC, то изменяет ее.
