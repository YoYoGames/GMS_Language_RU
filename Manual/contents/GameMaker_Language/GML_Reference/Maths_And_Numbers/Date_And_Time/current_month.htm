текущий_месяц

Эта переменная, доступная только для чтения, возвращает текущий месяц в виде числового значения, где 1 - январь, а 12 - декабрь.

Синтаксис:

текущий_месяц;

Возвращается:

Реально

Пример:

draw_text((32, 32, "Сегодня " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +".");

Приведенный выше код выводит день, месяц и год.
