random_set_seed

Для генерации случайного числа GameMaker Studio 2 начинает работу со случайным начальным числом. С помощью этой функции вы можете установить это число в известное значение и таким образом "заставить" исход всех последующих случайных событий быть одинаковым при каждом запуске программы. запуска. Например, эту функцию можно использовать в сочетании с функцией random_get_seed() для создания процедурно генерируемого контента и сохранения результатов без огромных сохранений (вы сохраняете только семя, больше ничего не нужно). больше ничего не нужно). Если вам нужны действительно случайные результаты для всего, вы должны использовать функцию randomise() функцию.

ПРИМЕЧАНИЕ: Хотя этот посевной материал дает стабильные результаты на каждой целевой платформе, результаты на разных платформах могут отличаться из-за различий в работе каждой цели.

Синтаксис:

random_set_seed(val);

Аргумент Описание
val Семя закладывается.

Возвращается:

Н/Д

Пример:

if debug
{
random_set_seed(1);
}

Приведенный выше код устанавливает случайное зерно в 1, только если переменная "debug" истинна.
