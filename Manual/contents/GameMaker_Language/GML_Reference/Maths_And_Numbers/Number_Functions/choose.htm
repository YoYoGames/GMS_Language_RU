выбрать

Иногда для случайного выбора нужно указать не числа, а что-то другое, или нужные числа не расположены в реальном порядке или в заданном диапазоне. В этих случаях вы можете использовать функцию choose() для генерации случайного результата. Например, допустим. вы хотите создать объект со случайным спрайтом в начале, тогда вы можете использовать эту функцию для установки индекса спрайта на один из множества заданных спрайтов. Обратите внимание, что вы можете использовать столько аргументов, сколько вам нужно (обратите внимание, что большее количество аргументов будет означать, что функция будет разбираться медленнее).

ПРИМЕЧАНИЕ: Эта функция будет возвращать одно и то же значение при каждом новом запуске игры, поскольку GameMaker Studio 2 каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры.

Синтаксис:

choose(val0, val1, val2... max_val);

Аргумент Описание
val0... max_val Входное значение, которое может быть строкой, целым числом, переменной или константой.

Возвращается:

Один из заданных аргументов

Пример:

sprite_index = choose(spr_Cactus, spr_Flower, spr_Tree, spr_Shrub);
hp = choose(5, 8, 15, 32, 40);
name = choose("John", "Steven", "Graham", "Jack", "Emily", "Tina", "Jill", "Хелен");

Приведенный выше код использует select для установки ряда свойств экземпляра.
