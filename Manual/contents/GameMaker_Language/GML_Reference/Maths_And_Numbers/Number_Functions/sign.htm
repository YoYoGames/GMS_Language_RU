подписать

Эта функция возвращает, является ли число положительным, отрицательным или ни тем, ни другим, и возвращает 1, -1, 0 соответственно. Например - sign(458) вернет 1, sign(-5) вернет -1 и sign(0) вернет 0.

Синтаксис:

sign(n);

Аргумент Описание
n Число, знак которого нужно получить.

Возвращается:

Целое число

Пример:

y += sign(y - mouse_y);

Приведенный выше код добавит 1, -1 или 0 к y в зависимости от результата y - mouse_y.
