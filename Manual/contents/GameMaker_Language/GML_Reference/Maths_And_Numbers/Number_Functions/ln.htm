ln

Натуральный логарифм ln(n) - это количество времени, необходимое для достижения определенного уровня непрерывного роста, где n - достигнутый уровень. Таким образом, если мы хотим узнать, сколько единиц времени нам нужно для достижения роста в 20 единиц, мы используем ln(20), который возвращает 2,99 единиц времени для достижения такого роста.

Синтаксис:

ln(n);

Аргумент Описание
n Входное значение.

Возвращается:

Реально

Пример:

alarm[0] = ln(age) * room_speed;

Приведенный выше код использует натуральный логарифм значения, хранящегося в переменной "возраст", для установки сигнала тревоги.
