медиана

Эта функция возвращает медиану входных значений, то есть среднее значение. Если количество аргументов четное, возвращается меньшее из двух средних значений. Функция может иметь столько аргументов, сколько требуется (обратите внимание, что большее количество аргументов будет означать, что функция будет медленнее разбираться), которые все должны быть реальными значениями. Это означает, что, например, median(43, 12, 25, 19, 6) вернет 19, так как это среднее значение между всеми остальными.

Синтаксис:

median(val1, val2, ... max_val);

Аргумент Описание
val0 ... max_val Значения для сравнения.

Возвращается:

Реально

Пример:

x = median( 0, x, ширина комнаты );
y = median( 0, y, высота комнаты );

Это не позволит игроку выйти из любой стороны комнаты, используя медиану в качестве фиксатора. Если игрок, например, движется влево от границы комнаты, его x будет меньше 0. Это означает, что среднее число первой из медиан будет равно 0. будет равно 0, поэтому игрок переместится в точку (0,y).
