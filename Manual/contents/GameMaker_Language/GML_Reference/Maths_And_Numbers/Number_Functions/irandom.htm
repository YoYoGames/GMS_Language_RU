irandom

Эта очень полезная функция возвращает только целые числа (целые числа). Так, например, чтобы получить случайное число от 0 до 9, вы можете использовать irandom(9), и она вернет число от 0 до 9 включительно. Обратите внимание, что можно использовать и плавающие числа, но при этом верхнее значение после точки будет исключено, поэтому irandom(9.7) вернет значение только от 0 до 9. Верхняя граница функции составляет $7fffffffffffffLL, поэтому при использовании очень больших чисел следует соблюдать осторожность.

ПРИМЕЧАНИЕ: Эта функция будет возвращать одно и то же значение при каждом новом запуске игры, поскольку GameMaker Studio 2 каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры.

Синтаксис:

irandom(n);

Аргумент Описание
n Верхний диапазон, из которого будет выбрано случайное число.

Возвращается:

Целое число

Пример:

if (irandom(9) == 1)
{
score += 100;
}

Это даст один шанс из десяти (поскольку 0 включено) добавить 100 к счету.
