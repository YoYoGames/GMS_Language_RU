угловая разница

Эта функция возвращает наименьшую разницу между двумя углами в виде значения от -180 до 180 градусов (где положительный угол - против часовой стрелки, а отрицательный - по часовой).

Синтаксис:

разность углов(ang1, ang2)

Аргумент Описание
анг1 Первый угол, который можно использовать.
угол2 Второй угол для использования.

Возвращается:

Реально

Пример:

var pd = point_direction(x, y, mouse_x, mouse_y);
var dd = angle_difference(image_angle, pd);
image_angle -= min(abs(dd), 10) * sign(dd);

Приведенный выше код получит угол направления от экземпляра к курсору мыши, затем получит разницу между этим углом и текущим значением. угол_изображения, используя это значение для медленного поворота в сторону мыши.
