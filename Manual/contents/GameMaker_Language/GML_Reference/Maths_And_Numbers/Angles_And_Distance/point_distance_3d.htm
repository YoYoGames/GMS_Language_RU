точка_расстояния_3d

Эта функция принимает заданные компоненты вектора и возвращает длину (расстояние) вектора. Она работает точно так же, как точка_расстояние() но с добавлением факторизации значения z (глубина) для использования в трехмерном пространстве.

Синтаксис:

point_distance_3d(x1, y1, z1, x2, y2, z2);

Аргумент Описание
x1 Координата x первого компонента вектора
y1 Координата y первого компонента вектора
z1 Координата z первого компонента вектора
x2 Координата x второй компоненты вектора
y2 Координата y второй компоненты вектора
z2 Координата z второй составляющей вектора

Возвращается:

Реально

Пример:

var inst, ex, ey, ez;
inst = instance_nearest(x, y, enemy);
if inst
{
ex = inst.x;
ey = inst.y;
ez = inst.z;
if point_distance_3d(x, y, z, ex, ey, ez) < 200
{
instance_create_layer(x, y, "Bullets", obj_Missile)
}
}

Приведенный выше код получит координаты x, y и z ближайшего врага, а затем использует их для проверки расстояния (длины) вектора, образованного им и координатами игрока. Если значение меньше 200, объект игрока создаст экземпляр obj_Missile".
