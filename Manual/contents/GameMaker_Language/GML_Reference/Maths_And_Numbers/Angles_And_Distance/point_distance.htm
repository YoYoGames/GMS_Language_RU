расстояние между точками

Эта функция возвращает длину вектора, образованного заданными компонентами [x1,y1] и [x2,y2]. Например, на изображении ниже, если мы хотим получить расстояние между позицией корабля игрока и позицией врага, чтобы мы могли рассчитать, находится ли он достаточно близко, чтобы стрелять в него, то мы используем эту функцию (точный код). враг находится достаточно близко, чтобы стрелять в него, то мы используем эту функцию (точный код приведен в примере под изображением):

Синтаксис:

point_distance(x1, y1, x2, y2);

Аргумент Описание
x1 Координата x первого компонента вектора
y1 Координата y первого компонента вектора
x2 Координата x второй компоненты вектора
y2 Координата y второй компоненты вектора

Возвращается:

Реально

Пример:

var ex, ey;
ex = instance_nearest(x, y, enemy).x;
ey = instance_nearest(x, y, enemy).y;
if (point_distance(x, y, ex, ey) < 200)
{
instance_create_layer(x, y, "Bullets", obj_Missile)
}

Приведенный выше код получит координаты x и y ближайшего врага, а затем использует их для проверки расстояния (длины) вектора, образованного им и координатами игрока. Если значение меньше 200, объект игрока создаст экземпляр "obj_Missile".
