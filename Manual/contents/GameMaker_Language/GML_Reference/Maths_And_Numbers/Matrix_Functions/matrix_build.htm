matrix_build

Эта функция может быть использована для создания вашей собственной матрицы и возвращает значение индекса для этой новой матрицы, которое должно быть сохранено в переменной для дальнейшего использования. Сама матрица создается как массив из 16 значений, где первые 4 элемента являются строкой 1, вторые 4 элемента являются строкой 2 и т.д... матрицы 4x4.

ПРИМЕЧАНИЕ: Когда вы строите новую матрицу таким образом, порядок работы - YXZ.

Синтаксис:

matrix_build(x, y, z, xrotation, yrotation, zrotation, xscale, yscale, zscale);

Аргумент Описание
x Компонент x вектора перевода.
y Компонент y вектора перевода.
z Компонент z вектора перевода.
xrotation Угол поворота вокруг оси x (в градусах °).
yrotation Угол поворота вокруг оси y (в градусах °).
хзотация Угол поворота вокруг оси z (в градусах °).
xscale Величина масштаба x.
yscale Величина масштаба y.
zscale Величина масштаба z.

Возвращается:

Индекс матрицы (матрица 4x4)

Пример:

t_matrix = matrix_build(x, y, 0, 0, 90, 0, 1, 2, 1);

Приведенный выше код построит новое преобразование матрицы и сохранит индекс результирующей матрицы в переменной "t_matrix".
