matrix_stack_is_empty

Эта функция может использоваться для проверки того, пуст ли стек матрицы (возвращается true) или нет (возвращается false).

Синтаксис:

matrix_stack_is_empty();

Возвращается:

Булево

Пример:

if !matrix_stack_is_empty()
{
matrix_stack_clear();
}

Приведенный выше код проверяет стек матрицы на то, пуст ли он, и если нет, то очищает его.
