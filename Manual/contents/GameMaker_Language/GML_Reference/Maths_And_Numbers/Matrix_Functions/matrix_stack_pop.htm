matrix_stack_pop

Эта функция выводит текущую верхнюю матрицу из стека.

Синтаксис:

matrix_stack_pop();

Возвращается:

Н/Д

Пример:

var m1 = matrix_build(66, 145, 0, 0, 0, 0, 1, 1, 1);
var m2 = matrix_build(0, 0, 0, 0, 0, 0, image_angle * 6, 1, 1, 1, 1) ;
matrix_stack_push(m1);
matrix_stack_push(m2);
matrix_set(matrix_world, matrix_stack_top());
draw_sprite(tyre, 0, 0, 0, 0);
matrix_stack_pop();
matrix_stack_pop();

Приведенный выше код создаст две новые матрицы, а затем поместит их в стек матриц. Затем матрица мира устанавливается на вершину стека, рисуется спрайт, а затем трансформации выталкиваются из стека.
