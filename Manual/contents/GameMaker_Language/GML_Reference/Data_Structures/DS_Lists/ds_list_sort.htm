ds_list_sort

С помощью этой функции можно отсортировать все значения в списке по возрастанию или убыванию. Если список содержит строки, они будут отсортированы в алфавитном порядке на основе английского алфавита из 26 букв.

Синтаксис:

ds_list_sort(id, ascend);

Аргумент Описание
id Идентификатор списка для сортировки.
подняться Должен ли порядок значений быть возрастающим(true) или убывающим(false).

Возвращается:

Н/Д

Пример:

if newgame
{
ds_list_sort(name_list, true);
}

Приведенный выше код отсортирует список, проиндексированный в переменной "name_list", если переменная "newgame" отмечена как true.
