ds_list_read

С помощью этой функции можно воссоздать сохраненный список DS (тот, который ранее был записан в виде строки с помощью функции ds_list_write()). Сначала вы должны создать новый список DS для чтения строки, и если список DS уже существует и в нем хранится информация, то она будет очищена перед чтением. Эта функция имеет жизненно важное значение при создании механизмов сохранения/загрузки для вашей игры.Обратите внимание, что если структура данных была создана с помощью предыдущих версий программы GameMaker вам следует добавить необязательный аргумент "legacy", установив его в true, поскольку формат строк в этой версии изменился.

Синтаксис:

ds_list_read(id, str [, legacy]);

Аргумент Описание
id Идентификатор структуры данных для чтения.
str Строка для чтения.
наследие(необязательно) Может быть как истинным, так и ложным или полностью отсутствовать.

Возвращается:

Н/Д

Пример:

list = ds_list_create();
ini_open("save.ini");
var str = ini_read_string("Lists", "0", "");
if str != ""
{
ds_list_read(list, str);
}
ini_close();

Приведенный выше код создает список и сохраняет индекс в переменной "list". Затем он открывает ini-файл и считывает из него строку, проверяя, что сначала строка не возвращается пустой. Затем эта строка считывается в только что созданный список ds_list.
