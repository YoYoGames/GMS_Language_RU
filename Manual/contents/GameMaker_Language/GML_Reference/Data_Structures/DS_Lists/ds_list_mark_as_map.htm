ds_list_mark_as_map

Эта функция "пометит" (или "отметит") заданную позицию в ранее созданном списке DS как содержащую карту DS. Эта функция необходима при кодировании строк JSON (которые вы можете создавать с помощью json_encode()), но также может быть полезна при вложении структур данных, поскольку помеченные таким образом элементы будут автоматически собираться (уничтожаться) при уничтожении родительского списка DS. список DS уничтожается. Это означает, что вам не нужно вручную просматривать содержимое списка и уничтожать помеченные структуры данных по отдельности перед уничтожением "родительского" списка. Однако, если вы удалите позицию списка по отдельности, содержащаяся в ней структура данных не будет собрана, поэтому перед удалением позиции родительского списка следует вызвать соответствующую функцию уничтожения карты DS. Также обратите внимание, что если вы вызываете функцию ds_list_clear() на списке, все элементы, помеченные как карты, также будут уничтожены при очистке списка.

Также стоит отметить, что если отмеченная позиция меняется в списке, "метка" перемещается вместе с ней, поэтому если вы отметили позицию 3 в списке (например), а затем вставили еще 2 элемента под ней, чтобы она переместилась на позицию 5, она все равно будет отмечена как карта.

Синтаксис:

ds_list_mark_as_map(id, pos);

Аргумент Описание
id Идентификатор списка, который нужно отметить.
поз Позиция в списке, которую нужно отметить.

Возвращается:

Real (целое число - ID карты DS в случае успеха или -1 в случае неудачи)

Пример:

var sub_map = ds_map_create();
ds_map_add(sub_map, "player", player_array);
ds_map_add(sub_map, "enemy", enemy_array);
ds_list_add(j_list, sub_map);
ds_list_mark_as_map(j_list, 0);

Приведенный выше код создает карту DS, а затем заполняет ее двумя ключами, каждый из которых содержит массив значений. Затем эта карта добавляется в заданный список DS, а позиция "помечается" как таковая, чтобы впоследствии ее можно было правильно закодировать.
