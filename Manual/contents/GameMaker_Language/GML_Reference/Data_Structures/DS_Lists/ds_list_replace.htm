ds_list_replace

Эта функция заменит значение в заданной позиции на другое.

Синтаксис:

ds_list_replace(id, pos, val);

Аргумент Описание
id Идентификатор списка, который нужно изменить.
поз Позиция для замены значения, где 0 соответствует самому началу списка, а конечная позиция равна ds_list_size(id)-1.
val Новое значение, которым заменяется данное значение.

Возвращается:

Н/Д

Пример:

ds_list_replace(n_list, 3, name);

Приведенный выше код заменит значение, хранящееся на позиции 3 в списке, на значение, хранящееся в переменной "name".
