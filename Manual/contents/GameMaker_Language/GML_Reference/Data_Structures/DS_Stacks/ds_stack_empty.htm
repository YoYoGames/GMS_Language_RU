ds_stack_empty

С помощью этой функции вы можете проверить заданный стек DS на предмет того, пуст он (возвращается true) или нет (возвращается false).

Синтаксис:

ds_stack_empty(id);

Аргумент Описание
id Идентификатор структуры данных для проверки.

Возвращается:

Булево

Пример:

if (ai_count == 15) && (!ds_stack_empty(AI_stack))
{
ds_stack_clear(AI_stack);
alarm[0] = room_speed;
ai_count = 0;
}

Приведенный выше код проверяет переменную, достигла ли она определенного значения, и если достигла, то очищает стек DS, указанный в переменной "AI_stack", устанавливает сигнал тревоги и сбрасывает переменную в 0.
