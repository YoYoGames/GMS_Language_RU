ds_priority_delete_max

Эта функция вернет значение, имеющее наивысший приоритет в очереди, а затем удалит это значение (и приоритет) из структуры данных. Если более одного значения имеют одинаковый приоритет, то любое из них может быть возвращено в любом порядке, но все другие значения с тем же приоритетом все еще будут находиться в очереди.

Синтаксис:

ds_priority_delete_max(id);

Аргумент Описание
id Идентификатор используемой очереди приоритетов.

Возвращается:

Тип данных, хранящихся в приоритете

Пример:

if ai_move
{
script_execute(ds_priority_delete_max(ai_priority));
}

Приведенный выше код проверяет переменную экземпляра и, если она возвращает true, выполняет скрипт, индексированный в очереди приоритетов с наивысшим значением приоритета, а затем удаляет этот скрипт из очереди.
