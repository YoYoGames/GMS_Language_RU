ds_priority_find_min

С помощью этой функции вы можете найти значение, хранящееся в очереди приоритетов с наименьшим приоритетом, а если несколько значений имеют одинаковый приоритет, то любое из них может быть возвращено в любом порядке. Однако, в отличие от ds_priority_delete_min(), эта функция не удаляет значение из очереди.

Синтаксис:

ds_priority_find_min(id);

Аргумент Описание
id Идентификатор используемой очереди приоритетов.

Возвращается:

Вещественный или строка

Пример:

if ai_move
{
script_execute(ds_priority_find_min(ai_priority));
}

Приведенный выше код проверяет переменную экземпляра и, если она возвращает true, выполняет функцию сценария, проиндексированную в очереди приоритетов с наименьшим значением приоритета.
