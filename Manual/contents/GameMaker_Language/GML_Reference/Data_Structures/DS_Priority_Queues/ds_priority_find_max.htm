ds_priority_find_max

С помощью этой функции вы можете найти значение, хранящееся в очереди приоритетов с наивысшим приоритетом, а если несколько значений имеют одинаковый приоритет, то любое из них может быть возвращено в любом порядке. Однако, в отличие от ds_priority_delete_max(), эта функция не удаляет значение из очереди.

Синтаксис:

ds_priority_find_max(id);

Аргумент Описание
id Идентификатор используемой очереди приоритетов.

Возвращается:

Вещественный или строка

Пример:

if ai_move
{
script_execute(ds_priority_find_max(ai_priority));
}

Приведенный выше код проверяет переменную экземпляра и, если она возвращает true, выполняет функцию сценария, проиндексированную в очереди приоритетов с наивысшим значением приоритета.
