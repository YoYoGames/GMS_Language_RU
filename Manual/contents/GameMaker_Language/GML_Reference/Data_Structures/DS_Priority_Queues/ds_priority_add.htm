ds_priority_add

С помощью этой функции вы можете добавить значение (вещественное число или строку) в очередь приоритетов, одновременно присвоив ему значение приоритета.

Синтаксис:

ds_priority_add(id, val, priority);

Аргумент Описание
id Идентификатор очереди приоритетов для добавления.
val Значение для добавления в очередь приоритетов.
приоритет Приоритет добавляемой ценности.

Возвращается:

Н/Д

Пример:

ds_priority_add(ai_priority, AI_Search, 5);

Приведенный выше код добавляет функцию сценария в очередь приоритетов, указанную в переменной "ai_priority", и присваивает ей приоритет 5.
