ds_grid_set_disk

С помощью этой функции вы можете установить круговую область сетки на определенное значение. Вам необходимо указать начальную ячейку сетки (как координаты по оси x и y), а также радиус диска для установки и значение, на которое вы хотите установить диск, как показано на рисунке ниже. как показано на рисунке ниже:

Синтаксис:

ds_grid_set_disk(index, xm, ym, r, val);

Аргумент Описание
индекс Индекс сетки.
xm Позиция x диска на сетке.
ym Позиция y диска на сетке.
r Радиус диска на сетке.
val Значение, которое нужно установить для ячеек на диске.

Возвращается:

Н/Д

Пример:

ds_grid_set_disk(grid, ds_grid_width(grid) div 2, ds_grid_height(grid) div 2, 5, -4)

Приведенный выше код установит круглую область радиусом 5 ячеек в сетке DS, проиндексированной в переменной "grid", на значение -4.
