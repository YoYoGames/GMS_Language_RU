ds_grid_add_grid_region

Эта функция может быть использована для добавления значений всех ячеек, находящихся в исходной области сетки, к значениям в целевой сетке, как показано ниже:

ПРИМЕЧАНИЕ: Вы также можете использовать эту функцию в той же сетке для добавления значений из одной области сетки к значениям, хранящимся в другой (см. пример кода ниже).

Синтаксис:

ds_grid_add_grid_region(index, source, x1, y1, x2, y2, xpos, ypos);

Аргумент Описание
индекс Индекс сетки назначения.
источник Индекс исходной сетки.
x1 Левая позиция области ячеек для копирования из исходной сетки.
y1 Верхняя позиция области ячеек для копирования из исходной сетки.
x2 Правая позиция области ячеек для копирования из исходной сетки.
y2 Нижняя позиция области ячеек для копирования из исходной сетки.
xpos Позиция x на сетке назначения для добавления исходного региона.
ypos Позиция y на сетке назначения для добавления исходного региона.

Возвращается:

Н/Д

Пример:

ds_grid_add_grid_region(grid, grid, 0, 0, 1, 5, 2, 0)

Приведенный выше код копирует область ячеек с (0,0) по (1,5) сетки DS, индексированных в переменной "grid", и добавляет их к ячейкам с позиции (2,0) той же сетки DS.
