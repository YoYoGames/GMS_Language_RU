ds_grid_read

Эта функция может быть использована для преобразования строки, которая была создана ранее функцией ds_grid_write() обратно в сетку DS. Сетка DS должна быть создана ранее (см. пример ниже). Обратите внимание что если структура данных была создана в предыдущих версиях программы GameMaker необходимо добавить необязательный аргумент "legacy", установив его в true, так как формат строки изменился в этой версии.

Синтаксис:

ds_grid_read(index, string [, legacy]);

Аргумент Описание
индекс Индекс сетки для чтения.
строка Строка для считывания в сетку DS.
наследие(необязательно) Может быть как истинным, так и ложным или полностью отсутствовать.

Возвращается:

Н/Д

Пример:

grid = ds_grid_create(room_width div 32, room_height div 32);
ini_open("Save.ini");
ds_grid_read(grid, ini_read_string("Save", "0", ""));
ini_close()

Приведенный выше код создает сетку DS на основе размера комнаты (каждый квадрат 32x32 пикселей представляет собой одну ячейку сетки), а затем считывает ранее сохраненный набор данных сетки из ini-файла в новую сетку DS.
