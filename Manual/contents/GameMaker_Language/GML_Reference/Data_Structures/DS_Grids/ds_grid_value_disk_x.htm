ds_grid_value_disk_x

С помощью этой функции вы можете получить координату x (в пределах заданной дискообразной области сетки) искомого значения. Вы задаете индекс сетки DS (возвращаемый функцией ds_grid_create()) вместе с позициями x/y для центральной ячейки диска. Затем задается радиус (как целое число) вокруг центральной ячейки для поиска, после чего задается искомое значение. Если искомое значение не существует, то функция вернет -1.

Синтаксис:

ds_grid_value_disk_x(index, xm, ym, r, val);

Аргумент Описание
индекс Индекс сетки.
xm Позиция x диска на сетке.
ym Позиция y диска на сетке.
r Радиус диска на сетке.
val Значение, которое нужно найти.

Возвращается:

Реально

Пример:

if ds_grid_value_disk_exists(grid, 5, 5, 5, 5, val)
{
xpos = ds_grid_value_disk_x(grid, 5, 5, 5, 5, val);
ypos = ds_grid_value_disk_y(grid, 5, 5, 5, 5, val);
}

Приведенный выше код проверяет ds_grid на наличие определенного значения в области диска. Если оно найдено, то сохраняет x и y позицию значения в двух переменных для последующего использования.
