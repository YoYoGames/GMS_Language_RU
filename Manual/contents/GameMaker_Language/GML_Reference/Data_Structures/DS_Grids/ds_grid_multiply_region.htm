ds_grid_multiply_region

С помощью этой функции можно указать область сетки, в которой нужно умножить значение каждой ячейки на заданную величину.

ПРИМЕЧАНИЕ: Эта функция работает только с вещественными числами, а не со строками.

Синтаксис:

ds_grid_multiply_region(index, x1, y1, x2, y2, val);

Аргумент Описание
индекс Индекс сетки.
x1 Положение x левой части области в сетке.
y1 Положение y верхней части области в сетке.
x2 Положение x правой части области в сетке.
y2 Положение y нижней части области в сетке.
val Значение для умножения на ячейки региона.

Возвращается:

Н/Д

Пример:

ds_grid_multiply_region( mygrid, 5, 5, 10, 10, 2 )

Приведенный выше код возьмет все значения, найденные в заданной прямоугольной области сетки, и умножит каждое из них на 2.
