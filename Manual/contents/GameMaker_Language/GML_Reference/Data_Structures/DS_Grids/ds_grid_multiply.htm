ds_grid_multiply

Эта функция умножит значение заданной ячейки сетки на указанную величину.

ПРИМЕЧАНИЕ: Эта функция работает только с вещественными числами, а не со строками.

Синтаксис:

ds_grid_multiply(index, x, y, val);

Аргумент Описание
индекс Индекс сетки.
x Положение x ячейки в сетке.
y Положение y ячейки в сетке.
val Значение, которое нужно умножить на ячейку.

Возвращается:

Н/Д

Пример:

ds_grid_multiply(mygrid, 5, 5, 2)

Приведенный выше код умножит значение, хранящееся в данной ячейке сетки DS, на 2.
