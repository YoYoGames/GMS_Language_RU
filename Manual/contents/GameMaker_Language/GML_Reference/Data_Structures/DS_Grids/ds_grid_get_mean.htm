ds_grid_get_mean

Эта функция может быть использована для нахождения среднего значения для всех ячеек, находящихся в пределах определенной области сетки (значения всех ячеек складываются вместе, а затем делятся на общее количество ячеек, составляющих область), как показано на рисунке ниже:

Синтаксис:

ds_grid_get_mean(index, x1, y1, x2, y2);

Аргумент Описание
индекс Индекс сетки.
x1 Левый столбец ячеек региона.
y1 Верхний ряд ячеек региона.
x2 Правый столбец ячеек региона.
y2 Нижний ряд ячеек региона.

Возвращается:

Вещественный или строка

Пример:

val = ds_grid_get_mean(grid, 0, 0, 0, 5, 5)

Приведенный выше код установит переменную "val" в среднее значение, содержащееся в заданной области сетки DS, индексированной в переменной "grid".
