ds_map_find_value

С помощью этой функции можно получить значение по заданному ключу. Входными значениями функции являются (ранее созданная) карта DS для использования и ключ для проверки.

ПРИМЕЧАНИЕ: Если такого ключа не существует, то функция вернет неопределенное значение. Вы всегда должны проверять это с помощью is_undefined() функция

Синтаксис:

ds_map_find_value(id, key)

Аргумент Описание
id Идентификатор карты, которую нужно использовать.
ключ Ключ к разгадке.

Возвращается:

Вещественный, строковый или неопределенный

Пример:

количество = ds_map_find_value(inventory, "food");

Или, используя аксессор карты "?":

количество = инвентарь[? "еда"];

Приведенный выше код получит значение ключа "food" и сохранит его в переменной "amount".
