ds_map_find_next

Эта функция возвращает следующий ключ, хранящийся в карте DS после указанного в функции. Это может быть полезно, если вам нужно итеративно просматривать карту DS в поисках чего-либо, но по возможности этого следует избегать, так как это может быть медленно. Если такого ключ не существует, то функция вернет неопределенное значение. Вы всегда должны проверять это с помощью is_undefined() функция.

Синтаксис:

ds_map_find_next(id, key);

Аргумент Описание
id Идентификатор карты, которую нужно использовать.
ключ Ключ к поиску следующего.

Возвращается:

Вещественный, строковый или неопределенный

Пример:

var size = ds_map_size(inventory);
var key = ds_map_find_first(inventory);
for (var i = 0; i < size; i++;)
{
if (key != "gold")
{
key = ds_map_find_next(inventory, key);
}
else break;
}

Приведенный выше код создает некоторые временные переменные, затем получает размер карты DS и находит первый ключ, сохраненный компьютером в карте. Затем с помощью цикла for выполняется итерация по карте DS в поисках значения ключа "gold". Если он находит его, то прерывает цикл.
