ds_map_read

Эта функция принимает строку, которая ранее была создана функцией ds_map_write() и затем считывает ее в ранее созданную карту DS . Если карта, в которую считывается строка содержит какие-либо пары ключ/значение, они будут сначала очищены перед повторным созданием сохраненной карты. Обратите внимание, что если структура данных была создана в предыдущих версиях GameMaker Studio 2, необходимо добавить необязательный аргумент "legacy", установив его в true, поскольку в этой версии формат строк изменился.

Синтаксис:

ds_map_read(id, str [, legacy]);

Аргумент Описание
id Идентификатор структуры данных, в которую считывается строка
str Строка для чтения
наследие(необязательно) Может быть как истинным, так и ложным или полностью отсутствовать.

Возвращается:

Н/Д

Пример:

inventory = ds_map_create();
ini_open("map.ini");
var t_string = ini_read_string("Saved", "0", "");
if (t_string != "")
{
ds_map_read(inventory, t_string);
}
ini_close();

Приведенный выше код создает новую карту DS и сохраняет ее индекс id в переменной "inventory". Затем он открывает файл ini и считывает строку из этого файла во временную переменную "t_string". Наконец, он проверяет, чтобы убедиться, что (а не значение по умолчанию ""), и если это так, то считывает строку во вновь созданную DS-карту, после чего снова закрывает ini.
