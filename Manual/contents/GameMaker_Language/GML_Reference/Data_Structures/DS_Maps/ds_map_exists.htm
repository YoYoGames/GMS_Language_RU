ds_map_exists

Эта функция вернет true, если указанный ключ существует в (ранее созданной) карте DS , и false, если не существует.

Синтаксис:

ds_map_exists(id, key);

Аргумент Описание
id идентификатор структуры данных для проверки
ключ ключ для проверки

Возвращается:

Булево

Пример:

if !ds_map_exists(inventory, "potions")
{
ds_map_add(inventory, "potions", 1);
}

Приведенный выше код проверит карту DS, проиндексированную в переменной "inventory", на наличие ключа "potions" и, если его нет, добавит его на карту.
