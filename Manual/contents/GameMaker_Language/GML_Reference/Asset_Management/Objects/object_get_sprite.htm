объект_get_sprite

Эта функция сообщит вам, есть ли у проверяемого объекта спрайт или нет, и если есть, то вернет индекс этого спрайта, или -1, если нет. Обратите внимание, что это не функция экземпляра! Вы можете иметь объект без спрайта, а экземпляр того же объекта может иметь его, и наоборот, или даже иметь разные спрайты. Вы можете установить спрайт отдельного экземпляра с помощью переменной sprite_index переменной instance.

Синтаксис:

object_get_sprite(obj);

Аргумент Описание
obj Индекс объекта для проверки

Возвращается:

Real (идентификатор актива спрайта)

Пример:

var _spr = object_get_sprite(object_index);
if (sprite_index != _spr)
{
sprite_index = _spr;
}

В приведенном выше примере sprite_index экземпляра будет сверен со спрайтом object_index экземпляра. Если они не совпадают, то экземпляру будет присвоен тот же спрайт, что и у индекса объекта экземпляру.
