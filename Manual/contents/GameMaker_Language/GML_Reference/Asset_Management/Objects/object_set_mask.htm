object_set_mask

С помощью этой функции вы можете установить mask_index определенного объекта (более подробную информацию о масках см. в разделе Редактор объектов). Это означает, что все экземпляры этого объекта, созданные после изменения маски, будут созданы с этой новой маской, в то время как экземпляры, уже находящиеся в комнате, могут быть не затронуты. Вы можете установить значение -1, чтобы удалить маску спрайта и по умолчанию использовать маску, определенную для спрайта объекта (или без маски, если спрайт не был выбран). не выбран).

Обратите внимание, что это не функция экземпляра! Вы можете установить индекс маски отдельных экземпляров с помощью переменной mask_index и таким образом иметь десять экземпляров с разными и даже если вы измените индекс маски объекта с помощью этой функции, все экземпляры, которые находятся в комнате в данный момент, останутся прежними, и только экземпляры, созданные после вызова функции, будут начинаться с новой маской. с новой маской.

Синтаксис:

object_set_mask(index, spr);

Аргумент Описание
индекс Индекс объекта, который нужно изменить.
spr Новый спрайт, который будет использоваться в качестве маски объекта.

Возвращается:

Н/Д

Пример:

if (global.level == 10)
{
object_set_mask(obj_Platform, spr_mask_10);
}

Приведенный выше код проверяет значение глобальной переменной и затем изменяет индекс маски объекта "obj_Platform", если он равен десяти.
