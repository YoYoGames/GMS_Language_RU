объект_get_parent

Эта функция получит object_index любого родителя, который был назначен указанному объекту, или вернет -100, чтобы показать, что у объекта нет родителя, назначенного ему, или -1, если проверяемый объект не существует. Для получения дополнительной информации о родителях см. в разделе Редактор объектов.

Синтаксис:

object_get_parent(obj);

Аргумент Описание
obj Индекс объекта для проверки

Возвращается:

Реально

Пример:

par = object_get_parent(object_index);

Приведенный выше код получит родительский индекс объекта для экземпляра, выполняющего код, и сохранит возвращаемое значение в переменной "par".
