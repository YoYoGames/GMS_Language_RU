object_get_visible

Эта функция сообщит вам, был ли проверяемый объект отмечен как "видимый" (запускает событие рисования) или нет (не запускает событие рисования). Обратите внимание, что это не функция экземпляра! Таким образом, вы можете иметь видимый объект и невидимый экземпляр одного и того же объекта, и наоборот. Вы можете установить видимость отдельного экземпляра с помощью переменной visible переменной instance.

Синтаксис:

object_get_visible(obj);

Аргумент Описание
obj Индекс объекта для проверки.

Возвращается:

Булево

Пример:

if (!visible) && (object_get_visible(object_index))
{
visible = true;
}

Приведенный выше код проверит запущенный экземпляр на предмет того, виден он или нет, а также проверит индекс объекта экземпляра на предмет того, отмечен ли он как видимый или нет. Если экземпляр еще не виден, индекс объекта помечается как on, он установит значение "visible" в true для этого экземпляра.
