object_set_solid

С помощью этой функции можно отметить конкретный объект как цельный или нет (более подробную информацию о флаге "цельный" см. в разделе " Редактор объектов "). Это означает, что все экземпляры этого объекта, которые будут созданы после изменения параметра solid, будут созданы с этим новым состоянием, в то время как экземпляры, которые уже находятся в комнате, могут быть не затронуты.

Обратите внимание, что это не функция экземпляра! Вы можете установить флаг solid для отдельных экземпляров с помощью переменной твердый и таким образом десять экземпляров будут помечены как solid, даже если в IDE для данного объекта установлен флаг false. Даже если вы измените флаг solid объекта на true с помощью этой функции, все экземпляры, находящиеся в комнате, останутся такими, какими они были, и только экземпляры, созданные после вызова функции будут начинаться с флагом solid, установленным как true.

Синтаксис:

object_set_solid( index, solid );

Аргумент Описание
индекс Индекс объекта, который нужно изменить.
твердый Новая твердость объекта (true=твердый, false=не твердый).

Возвращается:

Н/Д

Пример:

if !object_get_solid(obj_Block)
{
object_set_solid(obj_Block, true);
}

Приведенный выше код проверяет флаг solid объекта "obj_Block", и если он равен false, то он изменяется на true.
