пользователь события

С помощью этой функции вы указываете экземпляру выполнить действия или код, который был помещен в одно из 16 событий, определенных пользователем. Эти события могут быть вызваны только таким образом или с помощью функции event_perform функцию.

Синтаксис:

event_user(numb);

Аргумент Описание
numb Номер события для вызова, от 0 до 15.

Возвращается:

Н/Д

Пример:

event_user(4);

Это вызовет определяемое пользователем событие 4.
