event_action (устаревшее)

Эта переменная, доступная только для чтения, возвращала индекс выполняемого в данный момент действия, начиная с 0 в предыдущих версиях GameMaker. Однако в GameMaker Studio 2 эта переменная уже устарела . Она была оставлена только для поддержки наследия и всегда будет возвращать 0, поскольку все действия объединяются вместе для повышения скорости выполнения.

Синтаксис:

событие_действие;

Возвращается:

Реально

Пример:

num = event_action;

Приведенный выше код сохраняет номер текущего действия в переменной "num".
