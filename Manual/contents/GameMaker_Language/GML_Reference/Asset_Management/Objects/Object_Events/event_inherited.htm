событие_наследованное

Эта функция вызывает текущее событие родительского объекта экземпляра. Обычно, когда у экземпляра есть родительский объект, он автоматически наследует все те же события, что и родитель, но если (например) у родительского объекта есть событие create и вы добавляете его в дочерний объект, все экземпляры дочернего объекта будут выполнять новое событие create, которое вы добавили, а не то, которое было в родительском объекте. Если вам нужно использовать и событие родительского объекта, и событие дочернего объекта одного и того же типа, следует использовать эту функцию, поскольку она запустит событие родительского объекта, прежде чем продолжить работу с остальным кодом или действиями, которые содержит дочернее событие.

Синтаксис:

event_inherited();

Возвращается:

Н/Д

Пример:

event_inherited();
switch (sprite_index)
{
case spr_Enemy_1: dmg += 2; break;
case spr_Enemy_4: dmg -= 1; break;
case spr_Enemy_10: dmg +=10; break;
}

Приведенный выше код вызывает наследуемое событие родителя (в котором мы инициализируем переменную "dmg"), а затем переходит к изменению переменной "dmg". Если бы не было указано родителя для экземпляра, выполняющего этот код, мы получим ошибку "неизвестная переменная", поскольку dmg не была определена.
