event_perform_object

Эта функция работает так же, как и event_perform() за исключением того, что на этот раз вы можете указать события от другого объекта. Здесь есть много опций, которые позволяют полностью смоделировать все но обратите внимание, что это буквально просто выполняет весь код в событии, и игра не будет ничего изменять, чтобы заставить его срабатывать вручную, например, если вы выберете событие нажатия клавиши, событие сработает но соответствующая клавиша не будет распознана как нажатая. Или если вы выполните событие тревоги, счетчик тревоги не будет установлен на -1, а продолжит отсчет. Полный список констант, доступных для этой функции на странице функции event_perform().

ПРИМЕЧАНИЕ: Действия в событии, вызванном с помощью этой функции, применяются к текущему экземпляру, а не к экземплярам данного объекта.

Синтаксис:

event_perform_object(obj, type, numb);

Аргумент Описание
obj Объект, для которого должно быть запущено событие.
тип Тип события, которое необходимо выполнить. См. ниже.
numb Конкретное событие, если оно необходимо (в противном случае просто используйте 0).

Возвращается:

Н/Д

Пример:

event_perform_object(obj_Player, ev_keypress, ord("W"));

Это выполнит событие, связанное с Keyboard Check Pressed > W key от объекта "obj_Player" в текущем экземпляре.
