room_speed (устаревшее)

В этой переменной хранится скорость работы всех комнат (и игры) в игровых кадрах в секунду. Обратите внимание, что это НЕ FPS (кадры в секунду), а скорее количество игровых шагов, которые GameMaker Studio 2 будет пытаться поддерживать каждую секунду.

ВАЖНО! Эта переменная поддерживается только для Legacy Support и не должна использоваться, так как она больше не устанавливает скорость для одной комнаты, а для ВСЕХ комнат в игре. Для изменения скорости игры вместо этого следует использовать функцию game_set_speed().

Синтаксис:

комнатная скорость;

Возвращается:

Реально

Пример:

alarm[0] = room_speed * 10;

В приведенном выше коде переменная room_speed используется для установки сигнала тревоги на 10 секунд. Использование переменной таким образом гарантирует, что сигнал тревоги будет подаваться в течение 10 секунд, независимо от того, какая скорость установлена для комнаты.
