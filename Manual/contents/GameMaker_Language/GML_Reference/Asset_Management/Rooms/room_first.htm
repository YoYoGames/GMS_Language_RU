первая комната

Эта переменная, доступная только для чтения, возвращает индекс самой первой комнаты в игре (он определяется порядком появления комнат в Менеджере комнат, а не порядком их создания). они были созданы).

Синтаксис:

комната_первая;

Возвращается:

Индекс

Пример:

если живет < 1
{
room_goto(room_first);
}

Приведенный выше код проверит переменную "lives" и, если она меньше 1, перейдет в первую комнату в игре.
