layer_sprite_get_speed

Эта функция может использоваться для получения текущего значения множителя скорости элемента спрайта. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или когда вы используете функцию layer_sprite_get_id()), и функция вернет реальное значение, которое представляет собой множитель скорости, используемый для анимации спрайта. Значение по умолчанию равно 1.

Синтаксис:

layer_sprite_get_speed(sprite_element_id);

Аргумент Описание
sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию

Возвращается:

Реально

Пример:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_speed(spr_id) > 0
{
layer_sprite_speed(spr_id, 0);
}

Приведенный выше код получит идентификатор слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот ID затем используется для проверки скорости анимации для элемента, и если он больше 0, то он устанавливается на 0.
