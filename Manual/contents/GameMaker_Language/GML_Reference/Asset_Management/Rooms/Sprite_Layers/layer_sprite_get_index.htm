layer_sprite_get_index

Эта функция может использоваться для получения текущего значения индекса изображения элемента спрайта. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или когда вы использовать функцию layer_sprite_get_id()), и функция вернет реальное значение, которое представляет собой индекс изображения, отображаемого для спрайта. Функция вернет -1, если либо элемент спрайта не существует, либо у элемента нет индекса изображения. либо элемент не существует, либо ему не назначен правильный спрайт.

Синтаксис:

layer_sprite_get_index(sprite_element_id);

Аргумент Описание
sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию

Возвращается:

Real (индекс текущего изображения спрайта или -1)

Пример:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_index(spr_id) < 4
{
layer_sprite_index(spr_id, 4);
}

Приведенный выше код получит идентификатор слоя для слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот идентификатор затем используется для проверки того, что индекс изображения для элемента меньше 4, и если это так, то он устанавливается на 4.
