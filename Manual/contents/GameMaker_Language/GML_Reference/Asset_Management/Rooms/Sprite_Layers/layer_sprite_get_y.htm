layer_sprite_get_y

Эта функция может быть использована для получения позиции y элемента спрайта в комнате. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или когда вы использовать функцию layer_sprite_get_id()), а функция вернет значение позиции y.

Синтаксис:

layer_sprite_get_y(sprite_element_id);

Аргумент Описание
sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию

Возвращается:

Реально

Пример:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_y(spr_id) < 0
{
layer_sprite_y(spr_id, room_height);
}

Приведенный выше код получит идентификатор слоя для слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот идентификатор затем используется для проверки позиции элемента y, и если он меньше 0, то элемент слоя перемещается в другое положение по оси y.
