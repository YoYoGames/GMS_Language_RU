слой_спрайта_существует

С помощью этой функции можно проверить, существует ли элемент спрайта на данном слое. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции слой_создать()) или имя слоя (в виде строки - это влияет на производительность) и идентификатор элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()) и функция вернет либо true, если элемент существует, либо false, если его нет.

ПРИМЕЧАНИЕ: Эта функция работает в пределах текущей целевой комнаты - по умолчанию это комната, в которой вызывается функция, - которую можно установить с помощью функции layer_set_target_room().

Синтаксис:

layer_sprite_exists(layer_id, sprite_element_id)

Аргумент Описание
идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки).
sprite_element_id Уникальное значение ID элемента спрайта для проверки

Возвращается:

Булево

Пример:

var lay_id = layer_get_id("Asset_Trees");
if layer_sprite_exists(lay_id, global.TreesSprites)
{
layer_sprite_destroy(lay_id, global.TreesSprites);
}

Приведенный выше код проверяет слой "Asset_trees" на существование заданного элемента спрайта, и если он существует, то уничтожает его (но не слой).
