layer_sprite_get_yscale

Эта функция используется для получения текущего множителя масштаба по оси y для элемента спрайта. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), и функция вернет реальное значение для масштаба по оси y, где 1 - нет масштабирования.

Синтаксис:

layer_sprite_get_yscale(sprite_element_id);

Аргумент Описание
sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию

Возвращается:

Реально

Пример:

var lay_id = layer_get_id("sprite_sky");
var spr_id = layer_sprite_get_id(lay_id, "Clouds");
if layer_sprite_get_yscale(spr_id) != 1
{
layer_sprite_yscale(spr_id, 1);
}

Приведенный выше код получит идентификатор слоя для слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот идентификатор затем используется для проверки значения yscale, и если оно не равно 1, то оно устанавливается равным 1.
