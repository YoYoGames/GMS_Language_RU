layer_sprite_xscale

С помощью этой функции вы можете изменить, должен ли данный элемент спрайта на слое масштабироваться по оси x или нет. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью команды layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете значение масштаба. Масштаб 1 означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться вниз (0,5, например, уменьшит ширину используемого спрайта вдвое), большие значения будут масштабироваться вниз (0,5, например, уменьшит ширину используемого спрайта вдвое). спрайта), большие значения будут масштабироваться вверх, а отрицательные значения будут переворачивать спрайт и масштабировать его, если только используемое значение не равно -1 (в этом случае используемый спрайт просто переворачивается вправо-влево относительно исходной позиции без масштабирования).

Синтаксис:

layer_sprite_xscale(sprite_element_id, xscale);

Аргумент Описание
sprite_element_id Уникальное значение ID элемента спрайта для изменения
xscale Значение xscale (по умолчанию равно 1)

Возвращается:

Н/Д

Пример:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1
{
layer_sprite_xscale(asset_sprite, 1);
layer_sprite_yscale(asset_sprite, 1);
}

Приведенный выше код проверит элемент спрайта, назначенный слою, на котором находится экземпляр, выполняющий код, и если он масштабируется в любом направлении, то установит масштаб по оси x и масштаб по оси y равным 1.
