угол наклона_слоя

С помощью этой функции можно изменить угол для заданного элемента спрайта на слое. Вы указываете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или когда вы использовать функцию layer_sprite_get_id()), а затем задаете значение угла, от 0 до 359 против часовой стрелки, где 0 - справа, 90 - сверху, 180 - слева и 270 - снизу. Если вы зададите значение больше 360, оно будет зациклено, чтобы привести его в диапазон от 0 до 359.

Синтаксис:

layer_sprite_angle(sprite_element_id, angle);

Аргумент Описание
sprite_element_id Уникальное значение ID элемента спрайта для изменения
угол Угол наклона спрайта (по умолчанию 0)

Возвращается:

Н/Д

Пример:

var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees");
if layer_sprite_get_angle(asset_sprite) != 0
{
layer_sprite_angle(asset_sprite, 0);
}

Приведенный выше код проверит элемент спрайта, назначенный слою, на котором находится экземпляр, выполняющий код, и если его угол не равен 0, то установит его в 0.
