layer_background_get_htiled

С помощью этой функции можно узнать, является ли элемент фона горизонтальной плиткой или нет. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет либо true, если элемент в данный момент является плиточным, либо false, если это не так.

Синтаксис:

layer_background_get_htiled(background_element_id);

Аргумент Описание
background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию

Возвращается:

Булево

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if !layer_background_get_htiled(back_id)
{
layer_background_htiled(back_id, true);
}

Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки того, является ли элемент черепичным по горизонтальной оси, и если он если нет, то он устанавливается для этого.
