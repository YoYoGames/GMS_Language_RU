layer_background_sprite

С помощью этой функции вы можете установить индекс спрайта фонового элемента. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете индекс спрайта, который будет использоваться. Изображение фонового элемента будет заменено новым спрайтом. Если вы укажете значение -1, то элемент не будет назначен спрайт (но он будет существовать и может быть назначен позже).

Синтаксис:

layer_background_sprite(background_element_id, sprite_index)

Аргумент Описание
background_element_id Уникальное значение ID элемента фона для изменения
sprite_index Индекс спрайта, который будет использоваться для фонового элемента

Возвращается:

Н/Д

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_sprite(back_id) != spr_Clouds
{
layer_background_sprite(back_id, spr_Clouds);
}

Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот ID затем используется для проверки спрайта, назначенного этому элементу, и установки его на спрайт "spr_Clouds". если он еще не установлен.
