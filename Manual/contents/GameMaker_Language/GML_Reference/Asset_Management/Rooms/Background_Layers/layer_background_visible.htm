layer_background_visible

С помощью этой функции можно переключить видимость фонового элемента. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или когда вы используете функцию layer_background_get_id()), а затем устанавливаете аргумент visible в true или false. При значении true элемент будет отображаться, а при значении false, элемент не будет отображаться. Обратите внимание, что это зависит от видимости слоя, и даже если фоновый элемент отмечен как видимый, он не будет нарисован, если слой, на котором он находится, отмечен как невидимый.

Синтаксис:

layer_background_visible(background_element_id, visible)

Аргумент Описание
background_element_id Уникальное значение ID элемента фона для изменения
видимый Переключатель видимости, который может быть истинным или ложным

Возвращается:

Н/Д

Пример:

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_visible(back)
{
layer_background_visible(back, false);
}
else
{
layer_background_visible(back, true);
}
}

Приведенный выше код проверяет нажатие кнопки мыши и, если оно обнаружено, переключает видимость фонового элемента, назначенного слою, на котором находится экземпляр, выполняющий код.
