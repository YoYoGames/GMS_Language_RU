индекс_фона_слоя

Эта функция может использоваться для установки индекса изображения фонового спрайта, который имеет несколько подизображений. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете индекс изображения, который будет использоваться. Если вы зададите значение вне диапазона суб-изображений, то индекс изображения будет зацикливаться.

Синтаксис:

layer_background_index(background_element_id, image_index);

Аргумент Описание
background_element_id Уникальное значение ID элемента фона для изменения
индекс Индекс изображения, используемый для фона

Возвращается:

Н/Д

Пример:

var lay_id = layer_get_id("Background_trees");
var back_id = layer_background_get_id(lay_id);
layer_background_index(back_id, 1);

Приведенный выше код получит идентификатор слоя для слоя с именем "Background_trees", а затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для изменения индекса изображения элемента.
