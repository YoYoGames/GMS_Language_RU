layer_background_vtiled

С помощью этой функции можно изменить, должен ли данный фоновый элемент на слое быть вертикально черепичным или нет. Вы задаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете значение tiling. Если значение равно true, то элемент будет отображаться вертикально по комнате, а если значение равно false, элемент не будет отображаться в виде плитки. Эта функция предназначена только для 2D-проектов и не будет работать корректно при использовании проекции 3D-камеры.

Синтаксис:

layer_background_vtiled(background_element_id, tiled);

Аргумент Описание
background_element_id Уникальное значение ID элемента фона для изменения
видимый Переключатель вертикальной черепицы, который может быть истинным или ложным

Возвращается:

Н/Д

Пример:

var back = layer_background_get_id(layer);
if !layer_background_get_htiled(back) || !layer_background_get_vtiled(back)
{
layer_background_htiled(back, true);
layer_background_vtiled(back, true);
}

Приведенный выше код проверит элемент фона, назначенный слою, на котором находится экземпляр, выполняющий код, и если он не имеет плитки в любом направлении, то установит его на плитку по вертикали и горизонтали.
