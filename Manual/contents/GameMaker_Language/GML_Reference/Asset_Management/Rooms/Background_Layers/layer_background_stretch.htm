layer_background_stretch

С помощью этой функции вы можете переключить спрайт фонового элемента на растягивание в соответствии с размерами комнаты или на сохранение разрешения 1:1. Вы передаете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью команды layer_background_create() или при использовании функции layer_background_get_id()), а затем устанавливаете аргумент stretch в true или false. При значении true спрайт элемента будет растянут, чтобы соответствовать комнаты (либо увеличивается, либо уменьшается в зависимости от размеров спрайта и комнаты), а при значении false спрайт элемента будет отображаться в своем собственном разрешении.

Синтаксис:

layer_background_stretch(background_element_id, stretch)

Аргумент Описание
background_element_id Уникальное значение ID элемента фона для изменения
стрейч Переключатель растяжения, который может быть истинным или ложным

Возвращается:

Н/Д

Пример:

if mouse_check_button_pressed(mb_left)
{
var back = layer_background_get_id(layer);
if layer_background_get_stretch(back)
{
layer_background_stretch(back, false);
}
else
{
layer_background_stretch(back, true);
}
}

Приведенный выше код проверяет нажатие кнопки мыши и, если оно обнаружено, переключает растяжение спрайта фонового элемента, назначенного слою, на котором находится экземпляр, выполняющий код.
