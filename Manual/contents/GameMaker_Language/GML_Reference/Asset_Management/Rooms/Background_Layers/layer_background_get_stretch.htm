layer_background_get_stretch

Эта функция может быть использована для получения растянутого состояния спрайта фонового элемента. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет либо true, если спрайт элемента в данный момент растянут, чтобы соответствовать комнате, либо false, если это не так. нет.

Синтаксис:

layer_background_get_stretch(background_element_id);

Аргумент Описание
background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию

Возвращается:

Булево

Пример:

var lay_id = layer_get_id("Background_sky");
var back_id = layer_background_get_id(lay_id);
if layer_background_get_stretch(back_id)
{
layer_background_stretch(back_id, false);
}

Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот ID затем используется для проверки, будет ли спрайт элемента растянут, чтобы соответствовать комнате и если он будет растянут, то это свойство будет установлено в false.
