room_set_viewport

С помощью этой функции вы можете установить свойства порта вида для любой комнаты в вашей игре, кроме текущей. Вы передаете индекс комнаты, в которой нужно установить порт вида, индекс порта вида (от 0 до 7), а также указываете, должен ли порт вида быть видимым при в начале комнаты или нет (установите значение true, чтобы сделать порт видимым, и false в противном случае). После этого задаются координаты x и y (соответствующие верхнему левому положению порта просмотра), а затем ширина и высота в пикселях.

Синтаксис:

room_set_viewport(rm, vind, vis, xport, yport, wport, hport);

Аргумент Описание
rm Индекс комнаты для установки
vind Индекс порта представления для установки
vis Видимость порта просмотра(true - видимый, false - невидимый)
xport Положение x для смотрового отверстия в комнате
yport Положение y порта просмотра в помещении
wport Ширина (в пикселях) порта просмотра
hport Высота (в пикселях) порта просмотра

Возвращается:

Н/Д

Пример:

global.myroom = room_add();
room_set_width(global.myroom, 640);
room_set_height(global.myroom, 480);
room_set_viewport(global.myroom, 0, true, 0, 0, 640, 480);

Это установит свойства порта просмотра [0] в комнате, проиндексированной в переменной "global.myroom".
