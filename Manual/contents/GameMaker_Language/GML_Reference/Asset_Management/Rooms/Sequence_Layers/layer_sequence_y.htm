слой_последовательности_y

С помощью этой функции можно задать положение вдоль оси Y (вертикальной) комнаты для заданного элемента последовательности. Вы предоставляете идентификатор элемента последовательности, возвращаемый функцией layer_sequence_create() или одной из функции элемента слоя вместе с заданной позицией по оси Y, и последовательность будет перемещена в новое положение.

Синтаксис:

layer_sequence_y(sequence_element_id, pos_y)

Аргумент Описание
sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться
позиция_y Положение по оси Y для перемещения элемента последовательности

Возвращается:

Н/Д

Пример:

if layer_sequence_exists(my_seq)
{
layer_sequence_x(my_seq, x);
layer_sequence_y(my_seq, y);
}

Приведенный выше код проверяет, существует ли элемент последовательности, на который ссылается переменная "my_seq", и если да, то он устанавливает позицию x/y на позицию экземпляра, выполняющего код.
