уничтожение_последовательности_слоев

С помощью этой функции вы можете уничтожить (удалить из комнаты) элемент последовательности. Вы передаете идентификатор элемента последовательности, возвращенный layer_sequence_create() или одной из функций элемента слоя, и последовательность будет уничтожена.

Синтаксис:

layer_sequence_destroy(sequence_element_id)

Аргумент Описание
sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться

Возвращается:

Н/Д

Пример:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sequence
{
layer_sequence_destroy(a[i])
}
}

Приведенный выше код получает идентификаторы всех элементов, назначенных слою экземпляра, выполняющего этот код. Затем код проверяет, являются ли какие-либо из возвращенных элементов активами последовательности, и если да, то они уничтожаются.
