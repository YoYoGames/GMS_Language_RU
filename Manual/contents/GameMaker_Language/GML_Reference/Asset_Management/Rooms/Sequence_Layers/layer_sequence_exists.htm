layer_sequence_exists

С помощью этой функции вы можете проверить, существует ли элемент последовательности на заданном слое. Вы передаете идентификатор слоя, который может быть строкой имени слоя - как определено в редакторе помещений - или уникальным идентификатором слоя - как возвращается функцией layer_get_id(), а также ID элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элементов слоя - и возвращает true, если данный элемент существует, или false в противном случае.

Синтаксис:

layer_sequence_exists(layer_id, sequence_element_id)

Аргумент Описание
идентификатор_слоя Уникальный идентификатор или имя слоя для проверки
sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться

Возвращается:

Булево

Пример:

if !layer_sequence_exists("Background", my_seq)
{
my_seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground);
layer_sequence_pause(my_seq);
}

Приведенный выше код проверяет, существует ли заданный элемент последовательности, и если нет, то создает новую последовательность на заданном слое, а затем приостанавливает ее.
