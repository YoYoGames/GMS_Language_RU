room_get_viewport

С помощью этой функции вы можете получить информацию о видовом порте в комнате, отличной от текущей. Вы указываете ID комнаты и индекс порта просмотра, который нужно получить (от 0 до 7), и функция возвращает массив из 5 индексов, где:

  • [0] = видимый (булево: true / false)
  • [1] = положение x (реальное)
  • [2] = положение y (реальное)
  • [3] = ширина (реальная)
  • [4] = высота (реальная)

Синтаксис:

room_get_viewport(rm, vind);

Аргумент Описание
rm Индекс комнаты для получения данных о вьюпорте
vind Индекс порта представления для получения

Возвращается:

Массив (5 элементов: видимый, x, y, ширина, высота)

Пример:

v_vals = room_get_viewport(rm_Game, 0);
if v_vals[0] == false
{
room_set_view(rm_Game, true, v_vals[1], v_vals[2], v_vals[3], v_vals[4]);
}

Приведенный выше код извлекает данные порта просмотра для заданной комнаты, затем проверяет, отмечен ли порт как видимый. Если это не так, данные порта вида устанавливаются, чтобы сделать его видимым.
