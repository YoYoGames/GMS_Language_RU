tilemap_set_width

Эта функция может быть использована для изменения размера элемента карты плиток. Вы передаете идентификатор элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и новую ширину карты тайлов в ячейках тайлов.

Синтаксис:

tilemap_set_width(tilemap_element_id, width)

Аргумент Описание
tilemap_element_id Уникальное значение идентификатора элемента карты плитки для установки ширины
ширина Значение ширины (в "ячейках")

Возвращается:

Н/Д

Пример:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
if tilemap_get_width(map_id) != room_width div 16
{
tilemap_set_width(map_id, room_width div 16);
}

Приведенный выше код проверяет ширину конкретной карты плиток, и если она не соответствует размеру, то ширина устанавливается.
