tilemap_y

Эта функция управляет положением по оси y комнаты элемента карты плитки актива на слое. Вы передаете ID элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а затем задаете значение y, которое необходимо использовать (на основе координат помещения).

Синтаксис:

tilemap_y(tilemap_element_id, y);

Аргумент Описание
tilemap_element_id Уникальное значение ID элемента карты плитки, которое необходимо изменить
y Положение по оси y для карты плитки

Возвращается:

Н/Д

Пример:

var lay_id = layer_get_id("Asset_sky");
var map_id = layer_tilemap_get_id(lay_id);
tilemap_y(map_id, irandom(room_height));

Приведенный выше код получает значение ID актива карты плиток, назначенного слою "Asset_sky", а затем устанавливает его позицию y на случайное значение между 0 и высотой комнаты.

Назад: Плитка Слои карты
