tilemap_get_mask

Эта функция может быть использована для получения значения битовой маски для заданной карты плиток. Вы передаете идентификатор элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или когда вы используете функцию layer_tilemap_get_id()), и функция вернет текущее значение маски или -1, если произошла ошибка, или 0, если маска не указана. Для получения дополнительной информации о битовых масках карты плиток, см. функцию tilemap_set_mask().

Синтаксис:

tilemap_get_mask(tilemap_element_id);

Аргумент Описание
tilemap_element_id Уникальное значение ID элемента карты плитки для получения маски

Возвращается:

Реальный (0 для отсутствия маски, -1 для ошибки)

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mask = tilemap_get_mask(map_id);
var new_mask = tile_mirror | tile_flip | tile_rotate | 255;
if mask != new_mask
{
tilemap_set_mask(map_id, new_mask);
}

Приведенный выше код получает ID карты плитки из данного слоя, а затем проверяет связанное с ней значение маски. Если оно не совпадает со значением, определенным в переменной "new_mask", то оно устанавливается на это значение.
