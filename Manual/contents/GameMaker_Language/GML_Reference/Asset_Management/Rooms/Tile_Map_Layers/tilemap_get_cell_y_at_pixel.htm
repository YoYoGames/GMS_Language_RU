tilemap_get_cell_y_at_pixel

С помощью этой функции вы можете получить положение оси y отдельной ячейки карты плитки, задав относительное положение оси y в помещении. Вы указываете ID элемента карты плитки (который вы получаете при создании элемента карты плитки, используя layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а также положение x и y в помещении, и функция вернет положение y ячейки в карте плитки для этой точки. Обратите внимание, что если значение находится за пределами области карты плиток, и ни одна ячейка не доступна, функция вернет -1.

Синтаксис:

tilemap_get_cell_y_at_pixel(tilemap_element_id, x, y);

Аргумент Описание
tilemap_element_id Уникальное значение ID элемента карты плитки для получения позиции y ячейки
x Позиция x в комнате, которую следует использовать для получения ячейки
y Позиция y в комнате, которую следует использовать для получения ячейки

Возвращается:

Целое число (положение ячейки оси y или -1, если произошла ошибка)

Пример:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
var _x = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var _y = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
tiledata = tilemap_get(map_id, _x, _y);

Приведенный выше код использует полученный идентификатор карты плитки, чтобы получить положение ячейки из набора координат комнаты, а затем сохранить данные для любой плитки, найденной там, в переменной экземпляра.
