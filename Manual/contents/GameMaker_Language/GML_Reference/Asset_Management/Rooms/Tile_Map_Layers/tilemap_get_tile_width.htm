tilemap_get_tile_width

С помощью этой функции вы можете получить ширину (в пикселях) каждой ячейки элемента карты плиток. Вы передаете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и функция вернет ширину ячейки набора плиток.

Синтаксис:

tilemap_get_tile_width(tilemap_element_id);

Аргумент Описание
tilemap_element_id Уникальное значение ID элемента карты плитки для получения ширины ячейки плитки

Возвращается:

Реально

Пример:

var lay_id = layer_get_id("Tiles_Walls");
var map_id = layer_tilemap_get_id(lay_id);
global.snap_x = tilemap_get_tile_width(map_id);
global.snap_y = tilemap_get_tile_height(map_id);

Приведенный выше код использует полученный ID карты плитки для получения ширины и высоты ячейки плитки и использует их для установки двух глобальных переменных.
