tile_set_flip

Эта функция может быть использована для установки заданного набора данных плитки, чтобы перевернуть плитку или нет. Вы передаете данные тайла, которые можно получить с помощью функции tilemap_get()и затем устанавливаете аргумент flip в true, если хотите, чтобы плитка была перевернута, или false, если хотите, чтобы плитка была в стандартном, неперевернутом состоянии. Функция вернет модифицированный набор данных плитки, который затем можно применить с помощью функции tilemap_set() функцию.

Синтаксис:

tile_set_flip(tiledata, flip)

Аргумент Описание
tiledata данные плитки для установки
флип Установите значение true, чтобы перевернуть, и false, чтобы оставить все как есть

Возвращается:

Реальные (плиточные)

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y);
var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y);
var data = tilemap_get(map_id, mx, my);
var bool = !tile_get_flip(data);
data = tile_set_flip(data, bool);
tilemap_set(map_id, data, mx, my);

Приведенный выше код получает ID карты плитки из заданного слоя, а затем получает позицию x и y ячейки для плитки под мышью. Эта позиция затем используется для получения данных плитки, которые переворачиваются и затем снова используются для установки плитки.
