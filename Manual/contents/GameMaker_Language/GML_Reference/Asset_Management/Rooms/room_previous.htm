номер_предыдущий

С помощью этой функции вы можете получить индекс комнаты до ввода комнаты в функцию. Например, вы можете использовать переменную комната для получения индекса текущей комнаты, а затем использовать эту функцию для поиска комнату, которая находится перед ней, как указано в диспетчере комнат. Если перед введенной вами комнатой нет комнаты, то возвращается -1. Обратите внимание, что эта функция не распознает и не принимает во внимание комнаты, которые были добавлены динамически с помощью room_add() или room_duplicate().

Синтаксис:

room_previous(numb);

Аргумент Описание
numb Индекс комнаты, из которой нужно получить предыдущую.

Возвращается:

Индекс

Пример:

if room_previous(room) != -1
{
room_goto_previous());
}

Приведенный выше код проверит, существует ли предыдущая комната, и если да, то перейдет в нее.
