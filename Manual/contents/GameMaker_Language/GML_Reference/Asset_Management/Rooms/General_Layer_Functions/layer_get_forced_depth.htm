layer_is_draw_depth_forced

Эта функция может быть использована для получения глубины Z, установленной для рендеринга слоев в помещении. См. layer_force_draw_depth() для получения дополнительной информации.

Синтаксис:

layer_get_forced_depth()

Возвращается:

Реально

Пример:

if layer_get_forced_depth() != 0
{
layer_force_draw_depth(true, 0);
}

Приведенный выше код проверяет, имеет ли глубина Z слоя принудительное значение 0 или нет, и если нет, то устанавливает глубину Z слоя равной 0.
