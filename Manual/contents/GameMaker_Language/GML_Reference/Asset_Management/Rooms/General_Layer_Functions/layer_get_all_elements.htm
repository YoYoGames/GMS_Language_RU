layer_get_all_elements

Вы можете использовать эту функцию для получения идентификаторов элементов заданного слоя. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это будет иметь на производительность), и функция вернет массив идентификаторов, где каждая запись в массиве является уникальным идентификатором элемента данного слоя. Например, если слой является слоем активов, массив будет заполнен значениями ID для каждого актива спрайта, назначенного слою. Обратите внимание, что использование кода для работы со слоями означает, что вы можете назначать различные типы элементов одному и тому же слою - таким образом, у вас могут быть спрайтовые активы наряду с экземплярами, например, в этом случае вы можете использовать функцию layer_get_element_type() для получения типа элемента, к которому относится идентификатор.

Синтаксис:

layer_get_all_elements(layer_id)

Аргумент Описание
идентификатор слоя Уникальное значение ID слоя, из которого нужно получить элементы (или имя слоя в виде строки).

Возвращается:

Массив (1D, заполненный идентификаторами элементов)

Пример:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

Приведенный выше код получает идентификаторы всех элементов экземпляра, назначенных слою экземпляра, на котором выполняется код. Затем код проверяет, являются ли какие-либо из возвращенных элементов спрайтовыми активами, и если да, то они уничтожаются.
