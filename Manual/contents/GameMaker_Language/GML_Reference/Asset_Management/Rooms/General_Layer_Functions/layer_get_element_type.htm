layer_get_element_type

Вы можете использовать эту функцию для получения типа элемента для данного элемента. Вы передаете уникальное значение ID элемента (например, возвращенное функцией, создавшей элемент, или из редактора комнат), и функция возвращает одну из следующих констант констант (или -1, если элемент не существует или значение ID ошибочно):

Постоянная Описание
layerelementtype_background Элемент является фоном.
layerelementtype_instance Элемент является экземпляром.
layerelementtype_sprite Элемент является активом спрайта.
layerelementtype_tilemap Элемент представляет собой карту плитки.
layerelementtype_particlesystem Элемент представляет собой систему частиц.
layerelementtype_tile Элемент является унаследованной фоновой плиткой (это справедливо только для проектов, которые были импортированы из предыдущих версий программы GameMaker).
layerelementtype_sequence Элемент является активом последовательности.

Обратите внимание, что эта функция наиболее полезна, когда у вас есть несколько различных типов элементов, назначенных одному слою, и что вы можете получить список всех элементов на любом данном слое с помощью функции layer_get_all_elements().

Синтаксис:

layer_get_element_type(element_id)

Аргумент Описание
слой Уникальное значение ID элемента для получения типа

Возвращается:

Константа (см. таблицу, выше) или -1, если элемент не существует или недопустим

Пример:

var a = layer_get_all_elements(layer);
for (var i = 0; i < array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_sprite_destroy(a[i])
}
}

Приведенный выше код получает идентификаторы всех элементов экземпляра, назначенных слою экземпляра, на котором выполняется код. Затем код проверяет, являются ли какие-либо из возвращенных элементов спрайтовыми активами, и если да, то они уничтожаются.
