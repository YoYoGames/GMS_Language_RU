layer_get_element_layer

Вы можете использовать эту функцию для получения ID слоя, на котором находится данный элемент. Вы передаете уникальное значение ID элемента (например, возвращенное функцией, создавшей элемент, или из редактора помещений), а функция возвращает уникальный ID слоя, на котором находится данный элемент. Если заданный ID элемента не является действительным, то функция вернет -1.

Синтаксис:

layer_get_element_layer(element_id)

Аргумент Описание
идентификатор элемента Уникальное значение ID элемента для получения слоя

Возвращается:

ID слоя или -1, если элемент недействителен (т.е. не существует).

Пример:

element = layer_get_element_layer(asset_1);

Приведенный выше код получает ID слоя для элемента с ID, хранящимся в переменной "asset_1", и сохраняет его в переменной.
