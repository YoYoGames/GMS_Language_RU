layer_element_move

Вы можете использовать эту функцию для перемещения элемента с одного слоя на другой. Вы передаете идентификатор элемента, возвращаемый функцией, используемой для создания элемента, или редактором помещений, или функцией layer_get_all_elements(), а затем указываете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность).

Синтаксис:

layer_element_move(element_id, layer_id)

Аргумент Описание
идентификатор элемента Уникальное значение ID элемента для перемещения
идентификатор слоя Уникальное значение ID слоя, в который нужно переместить элемент (или имя слоя в виде строки).

Возвращается:

Н/Д

Пример:

var a = layer_get_all_elements(layer);
asset_layer = layer_create(-100);
for (var i = 0; i Alt; array_length(a); i++;)
{
if layer_get_element_type(a[i]) == layerelementtype_sprite
{
layer_element_move(a[i], asset_layer)
}
}

Приведенный выше код получает элементы на слое, к которому приписан экземпляр, выполняющий код, затем проверяет, являются ли они спрайтовыми активами, и если да, то они перемещаются на слой с ID, хранящимся в переменной "asset_layer".
