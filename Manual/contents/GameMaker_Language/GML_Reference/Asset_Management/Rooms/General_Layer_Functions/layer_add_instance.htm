layer_add_instance

Эта функция может быть использована для перемещения данного экземпляра со слоя, на котором он находится в данный момент, на другой слой. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (как строка - это влияет на производительность) и ID экземпляра, который нужно переместить между слоями. Например, допустим, ваш игрок находится на слое, который находится на меньшей глубине, чем другой слой, и вы хотите, чтобы он появился за слоями между ними. двумя слоями. Вы можете использовать эту функцию, чтобы переключить игрока с переднего плана на фоновый слой и заставить его появиться за другими рисуемыми слоями.

ПРИМЕЧАНИЕ: Эта функция не может быть использована для добавления нового экземпляра в слой. Вы должны были создать экземпляр ранее и сохранить его ID в переменной.

ПРИМЕЧАНИЕ: Эта функция действует только в пределах текущей комнаты и не может быть использована, если область была изменена с помощью функции layer_set_target_room().

Синтаксис:

layer_add_instance(layer_id, instance_id)

Аргумент Описание
идентификатор слоя Уникальное значение ID слоя экземпляра, на который нужно нацелиться (или имя слоя в виде строки).
идентификатор экземпляра Уникальное значение идентификатора экземпляра, который необходимо переместить на целевой слой

Возвращается:

Н/Д

Пример:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Приведенный выше код сначала получит индекс ближайшего экземпляра к заданной позиции x/y и сохранит его в локальной переменной. Затем он получает уникальный идентификатор слоя экземпляра для слоя с именем "Instances Front" и перемещает найденный экземпляр на этот слой.
