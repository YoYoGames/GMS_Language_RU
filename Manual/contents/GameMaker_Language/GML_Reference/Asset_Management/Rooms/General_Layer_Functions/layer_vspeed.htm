layer_vspeed

Вы можете использовать эту функцию для установки вертикальной скорости (в пикселях на игровой кадр) слоя в пределах текущей комнаты. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и значение скорости, которое нужно установить, где положительное значение - вниз, а отрицательное - вверх.

Синтаксис:

layer_vspeed(layer_id, vspd)

Аргумент Описание
идентификатор слоя Уникальное значение ID слоя для установки вертикальной скорости
vspd Вертикальная скорость (в пикселях на игровой кадр) для установки

Возвращается:

Н/Д

Пример:

var lay_id = layer_get_id("Sprites");
if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0
{
layer_hspeed(lay_id, 0);
layer_vspeed(lay_id, 0);
}

Приведенный выше код проверяет заданные горизонтальную и вертикальную скорости слоя, и если они обе не равны 0, то устанавливает их в 0.
