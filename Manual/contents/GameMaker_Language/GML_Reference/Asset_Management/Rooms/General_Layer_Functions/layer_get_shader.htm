layer_get_shader

Эта функция используется для проверки того, назначен ли данному слою шейдер. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это повлияет на производительность). повлияет на производительность), и функция вернет либо индекс шейдера, которому назначен шейдер, либо -1, если шейдер не назначен.

Синтаксис:

layer_get_shader(layer_id)

Аргумент Описание
идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки).

Возвращается:

Индексшейдера (или -1, если шейдер не назначен)

Пример:

if layer_get_shader(layer) == -1
{
layer_shader(layer, shd_Sepia);
}

Приведенный выше код проверяет, назначен ли шейдер слою, на котором находится экземпляр, выполняющий код, и если нет, то назначает шейдер.
