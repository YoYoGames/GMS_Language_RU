layer_get_id

Эта функция может быть использована для получения уникального значения ID для данного слоя. В IDE все слои имеют имя и тип, и чтобы иметь возможность редактировать или изменять их через код, необходимо указать значение ID слоя. Эта функция используется для получения этот ID, используя имя (строку) слоя (как записано в IDE). Если вы создадите новый слой с помощью кода, используя функцию layer_create() то эта функция вернет уникальное значение ID. (динамически созданные слои не получают имен). Обратите внимание, что если вы зададите имя слоя, который не существует в текущей комнате, то вы получите ошибку и проект завершится аварийно.

Синтаксис:

layer_get_id(имя_слоя)

Аргумент Описание
имя_слоя Название слоя (строка)

Возвращается:

Идентификатор слоя Значение (реальное), или -1, если указанный слой не существует

Пример:

var near = instance_nearest(x, y, obj_Tree);
var layer_id = layer_get_id("Instances Front");
layer_add_instance(layer_id, near);

Приведенный выше код сначала получит индекс ближайшего экземпляра к заданной позиции x/y и сохранит его в локальной переменной. Затем он получает уникальный идентификатор слоя экземпляра для слоя с именем "Instances Front" и перемещает найденный экземпляр на этот слой.
