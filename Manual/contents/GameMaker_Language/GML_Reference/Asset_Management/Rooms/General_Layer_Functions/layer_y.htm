слой_y

Эту функцию можно использовать для установки положения слоя по оси y в пределах текущего масштабируемого помещения. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это повлияет на производительность), и функция переместит слой на заданное количество пикселей вдоль вертикальной оси комнаты.

Синтаксис:

layer_y(layer_id, y)

Аргумент Описание
идентификатор слоя Уникальное значение ID слоя для установки положения по оси y
y Позиция y в помещении, на которую нужно установить слой

Возвращается:

Н/Д

Пример:

var lay_id = layer_get_id("Sprites");
if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0
{
layer_x(lay_id, 0);
layer_y(lay_id, 0);
}

Приведенный выше код проверяет заданную позицию слоя, и если она не установлена в (0, 0), то она устанавливается в эту позицию.
