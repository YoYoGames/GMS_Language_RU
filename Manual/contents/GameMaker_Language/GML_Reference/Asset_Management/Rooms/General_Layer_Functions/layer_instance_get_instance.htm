layer_instance_get_instance

Эта функция может быть использована для получения уникального идентификатора экземпляра данного элемента экземпляра. Вы сообщаете ID элемента экземпляра (см. пример кода ниже, как его получить), и функция возвращает вещественное значение, которое представляет собой уникальный идентификатор экземпляра для данного элемента. Если элемент не является экземпляром, функция вернет -1.

Синтаксис:

layer_instance_get_instance(element_id)

Аргумент Описание
идентификатор элемента Уникальное значение ID элемента экземпляра для получения идентификатора

Возвращается:

Real (Уникальное значение идентификатора для экземпляра)

Пример:

elements = layer_get_all_elements("Instances");
for (var i = 0; i < array_length(elements); i++)
{
if (layer_get_element_type(elements[i]) == layerelementtype_instance)
{
var layerelement = elements[i];
var inst = layer_instance_get_instance(layerelement);
inst.x = inst.x + 10;
}
}

Приведенный выше код проверит все элементы экземпляра на слое, затем получит их уникальное значение ID и использует его для перемещения на 10px вправо.
