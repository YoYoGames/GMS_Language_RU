комната_постоянная

Эта переменная может быть использована для получения и установки флага постоянства для текущей комнаты. Если установлено значение true, комната считается постоянной, в этом случае каждый раз, когда вы покидаете комнату и возвращаетесь обратно, состояние экземпляров в этой комнате будет сохраняться. Если же значение false, то каждый раз, когда вы возвращаетесь в комнату, она будет возвращаться в исходное состояние. Следует отметить, что постоянная комната использует значительно больше памяти, чем обычная комната, поэтому не рекомендуется иметь их в игре слишком много.

Синтаксис:

room_persistent;

Возвращается:

Булево

Пример:

если lives < 1
{
room_persistent = false;
room_goto(rm_start);
}

Приведенный выше код проверяет переменную "lives", и если она меньше 1, то устанавливает постоянство комнат на false, а затем меняет комнаты.
