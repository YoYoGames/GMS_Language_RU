skeleton_animation_set_ext

Когда вы устанавливаете набор анимации для вашего скелетного анимационного спрайта, ему назначается дорожка 0. Однако вы можете назначить другие наборы на другие дорожки (1 и выше) и таким образом заставить ваш спрайт воспроизводить две (или более) анимаций одновременно. Обратите внимание, что спрайт должен быть разработан в вашей программе анимации с учетом этого, так, чтобы "базовая" анимация на дорожке 0 имела ключевые кадры только для определенных костей, а следующие дорожки имели ключевые кадры для других. Например, если у вас есть анимация "ходьбы", и ее ключевые кадры воздействуют только на ноги спрайта, вы можете создать набор анимаций головы и рук, которые можно установить на дорожках 1 и 2 соответственно. Вы можете иметь базовое движение и адаптировать его в реальном времени во время игры.

Синтаксис:

skeleton_animation_set_ext(animname, track);

Аргумент Описание
animname Имя (строка) используемого набора анимации.
трек Дорожка для используемого набора анимации.

Возвращается:

Н/Д

Пример:

if mouse_check_button(mb_left)
{
skeleton_animation_set_ext("bodyfight", 1);
}
else
{
skeleton_animation_set_ext("bodywalk", 1);
}

Приведенный выше код изменит набор анимации, используемый дорожкой 1, на набор "bodyfight" или "bodywalk" при нажатии левой кнопки мыши.
