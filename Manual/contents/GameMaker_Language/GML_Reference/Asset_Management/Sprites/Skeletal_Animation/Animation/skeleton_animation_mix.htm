skeleton_animation_mix

Вы можете легко переключать наборы анимаций с помощью функции skeleton_animation_set() но это может привести к пропуску или заиканию при смене одной анимации на другую. Чтобы предотвратить это, вы можете установить значение mix Обычно это нужно сделать в событии Create экземпляра со скелетной анимацией, так как его нужно установить только один раз, а GameMaker Studio 2 будет интерполировать все дальнейшие изменения спрайта, используя анимацию. интерполирует все дальнейшие изменения спрайта, используя наборы анимации в этом экземпляре. Обратите внимание, что значение длительности варьируется от 0 до 1, при значении 0,5 интерполяция от одного набора к другому будет "половинной".

Синтаксис:

skeleton_animation_mix(animfrom, animto, duration);

Аргумент Описание
animfrom Имя (строка) первого набора анимации для интерполяции.
анимто Имя (строка) второго набора анимации для интерполяции.
продолжительность Продолжительность интерполяции (от 0 до 1)

Возвращается:

Н/Д

Пример:

skeleton_animation_set("walk");
skeleton_animation_mix("walk", "jump", 0.2);
skeleton_animation_mix("jump", "walk", 0.4);

Приведенный выше код будет входить в событие Create Event экземпляра со скелетным анимационным спрайтом и устанавливает длительность анимационного микса для интерполяции между двумя анимационными наборами "ходить" и "прыгать".
