skeleton_animation_get_frames

Эта функция может использоваться для получения количества кадров, которые имеет любая заданная скелетная анимация. Вы передаете имя скелетной анимации (в виде строки, как определено в программе, используемой для создания анимации, или как возвращено с помощью функции skeleton_animation_get, а функция возвращает количество кадров, которое она имеет, в виде целочисленного значения. Функция возвращает 0, если указанная анимация не существует.

Синтаксис:

skeleton_animation_get_frames(anim_name);

Аргумент Описание
имя_анимации Имя анимации для получения кадров.

Возвращается:

Целое число

Пример:

var num = skeleton_animation_get_frames(skeleton_animation_get());
image_index = num -1;
image_speed = 0;

Приведенный выше код получит количество кадров в анимации, затем установит спрайт на последний кадр и остановит анимацию.
