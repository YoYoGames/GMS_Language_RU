skeleton_animation_set

После того как вы назначили скелетный анимационный спрайт своему экземпляру, вы можете определить, какой набор анимаций он должен использовать в любой момент времени с помощью этой функции. При создании спрайта (в Spine) вы определите несколько наборов анимации для различных состояний и присвоили каждой из них соответствующее имя. Эти имена вы будете использовать, чтобы указать GameMaker Studio 2, какой набор анимации использовать в любой момент игры. Например, если ваш спрайт предназначен для платформенной игры, то у вас может быть набор "прыжок", набор "бег" и набор "бездействие" - все они содержатся в одном спрайте, и вы можете переключаться между ними в игре с помощью этой функции.

Обратите внимание, что при установке новой анимации таким образом значение параметра индекс_изображения на 0, запуская новый набор анимации с начала цикла анимации. Если это не является желаемое поведение, то вы можете установить значение "микс" с помощью функции skeleton_animation_mix() и при установке новой анимации она будет плавно интерполироваться с предыдущим набором. Вам также следует убедиться, что в начале каждой анимации есть ключевой кадр, чтобы данные не задерживались в предыдущих анимациях, если вы не планируете использовать отдельные анимационные дорожки.

Синтаксис:

skeleton_animation_set(animname);

Аргумент Описание
animname Имя (строка) используемого набора анимации.

Возвращается:

Н/Д

Пример:

if keyboard_check_pressed(vk_space)
{
if skeleton_animation_get() != "jump"
{
skeleton_animation_set("jump");
}
}

Приведенный выше код изменит используемый набор анимации на набор "прыжок" при нажатии клавиши пробела, но только если текущий используемый набор еще не "прыжок".
