скелетный_костный_список

С помощью этой функции вы можете заполнить (предварительно созданный) список DS всеми именами костей, используемых как часть скелетной анимации спрайта. Эти имена будут строками и могут быть использованы в других функциях костей скелетной анимации для этих типов спрайтов.

Синтаксис:

skeleton_bone_list(sprite, list);

Аргумент Описание
спрайт Индекс спрайта скелетной анимации Spine для получения списка.
список Идентификатор списка DS для заполнения именами костей.

Возвращается:

Н/Д

Пример:

bone_list = ds_list_create();
skeleton_bone_list(sprite_index, bone_list);

Приведенный выше код создает список DS, затем заполняет его именами костей для последующего использования.

Спина: кости
