skeleton_attachment_set

Скелетный анимационный спрайт может иметь другие изображения, добавленные в качестве вложений, причем эти изображения добавляются в именованный слот (имя задается при создании слота вложения в вашей программе анимации), и они будут рисоваться вместе с анимацией текущего спрайта. С помощью этой функции можно установить вложение в заданный слот, при этом необходимо указать имена (в виде строк) слота и вложения. Эти имена определяются используемой программой анимации или (в случае вложения) когда вы вызываете skeleton_attachment_create().

Обратите внимание, что вы также можете передать sprite_index в качестве вложения, и этот спрайт будет использован, или вы можете использовать -1, чтобы удалить вложение из слота. Когда вы передаете индекс спрайта в качестве аргумента, создается слот вложения, используя имя спрайта в качестве строки для названия слота (например, используя spr_sword, будет создан слот вложения "spr_sword"), и слот будет использовать первый индекс изображения (0) этого спрайта, а также смещение начала координат x и y, с масштабом (1,1) и вращением 0.

Синтаксис:

skeleton_attachment_set(slot, attachment);

Аргумент Описание
слот Имя слота (строка) для получения вложения.
вложение Имя (в виде строки или sprite_index) прикрепленного изображения.

Возвращается:

Строка

Пример:

if skeleton_attachment_get("slot_leftHand") == ""
{
skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife"));
}

Приведенный выше код проверит текущее назначенное имя вложения для слота с именем "slot_leftHand", и если будет возвращена пустая строка, то будет прикреплен новый спрайт.
