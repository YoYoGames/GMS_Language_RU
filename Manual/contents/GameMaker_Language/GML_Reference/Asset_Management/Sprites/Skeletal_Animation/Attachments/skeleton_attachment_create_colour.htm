skeleton_attachment_create_colour

При создании скелетного анимационного спрайта можно назначить слоты для вложений и вложения, которые будут в них находиться. Это просто изображения (спрайты), которые отделены от анимации, но при прикреплении будут двигаться вместе с костью, к которой они прикреплены. к которой они прикреплены. Обычно вы назначаете эти вложения в программе анимации (Spine), но с помощью этой функции вы можете создавать свои собственные вложения во время выполнения, используя актив спрайта из вашей игры. функция требует, чтобы вы дали вложению имя (как строка), а затем установить sprite_index и индекс_изображения для использования, а также начало координат x и y (которое может отличаться от тех, которые заданы в свойствах спрайта), а также задать любые трансформации, которые будут применены к изображению при вложении, цвет, который будет смешиваться с изображением, и его альфа (прозрачность). значение. Если вложение было успешно создано, функция вернет 1, а если нет (вы указали неверный индекс спрайта или базовый спрайт не является спрайтом Spine), то вернется -1.

Стоит отметить, что экземпляр спрайта может иметь настройки цвета смешивания и альфа (image_blend и изображение_альфа), как и слот вложения (см. функцию skeleton_slot_colour_set()), и поэтому окончательный цвет и альфа, которые будет иметь назначенный прикрепляемый спрайт для слота, будет представлять собой композицию всех этих значений.

Синтаксис:

skeleton_attachment_create_colour(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot, color, alpha);

Аргумент Описание
имя Имя (в виде строки) создаваемого вложения.
спрайт Индекс sprite_index для получения изображения вложения.
инд Индекс_изображения для получения изображения вложения.
xorigin Начало координат x для используемого изображения.
yorigin Начало координат y для используемого изображения.
xscale Горизонтальное масштабирование изображения в виде множителя: 1 = нормальное масштабирование, 0,5 - половина и т.д...
yscale Вертикальное масштабирование изображения в виде множителя: 1 = нормальное масштабирование, 0,5 - половина и т.д....
гниение Поворот изображения: 0 = нормально, 90 = повернуть на 90° против часовой стрелки и т.д.
Цвет Используемое значение цвета (константа, целое число или шестнадцатеричное значение).
Альфа Используемое значение альфа (от 0 до 1).

Возвращается:

1 - если успешно, -1 - если нет

Пример:

skeleton_attachment_create_colour("меч", spr_Weapons, 0, 0, 80, 1, 1, 90, c_red, 1);
skeleton_attachment_create_colour("нож", spr_Weapons, 1, 0, 45, 1, 1, 90, c_yellow, 1);
skeleton_attachment_create_colour("арбалет", spr_Weapons, 0, 10, 30, 1, 1, 0, c_white, 0.5);
skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow"));

Приведенный выше код проверит текущее назначенное имя вложения для слота с именем "slot_leftHand", и если будет возвращена пустая строка, то будет прикреплен новый спрайт.
