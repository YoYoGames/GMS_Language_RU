скелетный_слот_список

С помощью этой функции вы можете заполнить (предварительно созданный) список DS именами всех слотов, созданных как часть скелетной анимации спрайта. Имена будут строками и могут быть использованы в других функциях слотов скелетной анимации для этих типов спрайтов.

Синтаксис:

skeleton_slot_list(sprite, list);

Аргумент Описание
спрайт Индекс спрайта скелетной анимации Spine для получения списка.
список Идентификатор списка DS для заполнения именами костей.

Возвращается:

Н/Д

Пример:

bone_list = ds_list_create();
skeleton_slot_list(sprite_index, bone_list);

Приведенный выше код создает список DS, затем заполняет его именами слотов для последующего использования.
