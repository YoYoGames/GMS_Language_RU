скелет_найти_слот

С помощью этой функции можно найти, какие слоты находятся в заданной позиции пространства комнаты в спрайте Spine, связанном с текущим экземпляром. Вы создаете список DS и предоставляете его ID вместе с с позицией x/y для проверки, и список будет заполнен строкой имен для каждого из доступных слотов крепления спрайта (включая любые модификации крепления). Обратите внимание, что список всегда сортируется в порядке убывания, начиная с самого верхнего слота.

Синтаксис:

skeleton_find_slot(x, y, list);

Аргумент Описание
x Х-позиция в комнате для проверки.
y Положение y в комнате для проверки.
список Идентификатор списка DS для заполнения картами DS.

Возвращается:

Н/Д

Пример:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);

Приведенный выше код создает список DS, а затем заполняет его именами слотов, найденных при положении мыши в комнате.
