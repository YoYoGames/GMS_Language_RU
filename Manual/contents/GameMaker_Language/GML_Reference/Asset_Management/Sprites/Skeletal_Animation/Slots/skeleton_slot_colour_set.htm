skeleton_slot_colour_set

С помощью этой функции вы можете задать цвет и альфа слота привязки, так что все спрайты, привязанные к нему, будут рисоваться с этими значениями смешивания. Имейте в виду, что у экземпляра спрайта может быть задан цвет наложения и альфа (смешивание изображений и альфа-спектр изображения), так же как и у вложения (см. функцию skeleton_attachment_create_colour()), и поэтому окончательный цвет и альфа, которые будет иметь назначенный спрайт вложения для данного слота, будут представлять собой комбинацию всех этих значений.

Синтаксис:

skeleton_slot_colour_set(slot, color, alpha);

Аргумент Описание
слот Имя слота для установки, строка
цвет Цвет для установки, либо целое число, либо константа, либо шестнадцатеричное значение.
альфа Альфа для установки от 0 до 1.

Возвращается:

Н/Д

Пример:

slot_list = ds_list_create();
skeleton_find_slot(mouse_x, mouse_y, slot_list);
if !ds_list_empty(slot_list)
{
for (var i = 0; i < ds_list_size(slot_list); ++i;)
{
if skeleton_slot_colour_get(slot_list[| i]) != c_white
{
skeleton_slot_colour_set(slot_list[| i], c_white, 1);
}
}
}

Приведенный выше код создает список DS, затем заполняет его именами слотов, найденных по положению мыши в комнате. Затем он проходит по списку слотов и сбрасывает цвет для найденных слотов на белый с альфой 1, если цвет не является уже белый.
