skeleton_collision_draw_set

С помощью этой функции можно включить(true) или выключить(false) отрисовку данных о столкновениях для текущего скелетного анимационного спрайта, используемого экземпляром. Если эта функция включена, ограничивающая рамка и точная маска столкновений будут будут нарисованы в виде контуров вокруг спрайта.

Синтаксис:

skeleton_collision_draw_set(flag);

Аргумент Описание
флаг Установите значение true, чтобы включить рисование, и false, чтобы выключить его.

Возвращается:

Н/Д

Пример:

if debug_mode == true
{
skeleton_collision_draw_set(true);
}

Приведенный выше код проверяет, запущена ли игра в режиме отладки, и если да, то переключает данные скелетных столкновений для экземпляра, который будет показан.
