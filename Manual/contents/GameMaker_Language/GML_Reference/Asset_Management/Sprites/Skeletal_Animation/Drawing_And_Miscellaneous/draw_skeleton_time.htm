рисовать_скелет_время

Эта функция отрисует заданную анимацию с использованием заданного скина в определенное время анимации. Значение времени должно находиться между 0 (начало) и конечной продолжительностью анимации, которую можно найти с помощью функции skeleton_animation_get_duration(). Вы можете установить значение времени больше, чем общая продолжительность анимации, и анимация вернется к началу, но при этом вы рискуете потерять точность с плавающей запятой, так как накопленное время становится больше.

Синтаксис:

draw_skeleton_time(sprite, animname, skinname, time, x, y, xscale, yscale, rot, color)

Аргумент Описание
спрайт Индекс спрайта для рисования.
animname Имя анимации, из которой нужно получить кадр (строка).
skinname Имя кожи, используемой для рисования (строка).
время Временной интервал, в котором будет рисоваться анимация (от 0 до конечной продолжительности, в секундах).
x Координата x места рисования спрайта.
y Координата y места рисования спрайта.
xscale Горизонтальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д....
yscale Вертикальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д....
гниение Поворот спрайта. 0=нормальный, 90=повернутый на 90 градусов против часовой стрелки и т.д.
цвет Цвет, с которым будет смешиваться спрайт.

Возвращается:

Н/Д

Пример:

var time += delta_time / 1000000;
var d = skeleton_animation_get_duration("walk");
if time > d time -= d;
draw_skeleton_time(sprite_index, "walk", "skin1", time, x, y, image_xscale, image_yscale, image_angle, c_white);

Приведенный выше код нарисует заданный скелетный анимационный спрайт, используя дельта-время для установки рисуемого кадра.
