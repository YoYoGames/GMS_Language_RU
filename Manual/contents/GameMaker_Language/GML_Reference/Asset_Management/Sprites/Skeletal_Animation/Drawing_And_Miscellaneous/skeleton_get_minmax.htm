skeleton_get_minmax

Эта функция возвращает массив с общим значением граничной области для всех отдельных граничных областей, назначенных скелетному анимационному спрайту. Возвращаемый массив будет состоять из 4 элементов:

  • [0] - минимальное положение x для всех ограничительных рамок
  • [1] - минимальное положение по оси y для всех ограничительных рамок
  • [2] - максимальное положение x для всех ограничительных рамок
  • [3] - максимальное положение по оси y для всех ограничивающих рамок

Синтаксис:

skeleton_get_minmax();

Возвращается:

Массив (4 элемента: xMin, yMin, xMax, yMax)

Пример:

var frame = skeleton_animation_get_frame(0);
if frame > 0 && frame < 2
{
var box = skeleton_get_minmax();
var ww = (box[2] - box[0]) / 2;
var hh = (box[3] - box[1]) / 2;
part_particles_create(global.p_sys, box[0] + ww, box[1] + hh, global.Stars, 10);
}

Приведенный выше код проверит текущий кадр анимации, назначенный дорожке 0, а затем вырвет несколько частиц из точки в середине общей граничной области спрайта.
