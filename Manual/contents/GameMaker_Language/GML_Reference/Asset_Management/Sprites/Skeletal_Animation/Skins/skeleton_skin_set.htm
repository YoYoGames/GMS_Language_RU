skeleton_skin_set

Скелетный анимационный спрайт может быть нарисован с различными текстурами (называемыми "скинами"), что позволяет использовать одну анимацию для различных целей. Например, у вас может быть общая анимация ходьбы, и она может быть использована вашими NPC-персонажей, врагов и т.д. в игре, просто изменив скин, которым она нарисована. Эти скины должны быть предварительно созданы и названы в вашей программе анимации, и это имя передается в функцию (как строка), чтобы установить спрайт использовать этот скин при рисовании.

Синтаксис:

skeleton_skin_set(skinname);

Аргумент Описание
skinname Имя (строка) кожи, которую нужно использовать.

Возвращается:

Н/Д

Пример:

if skeleton_skin_get() == "skin_Enemy1"
{
skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3");
}

Приведенный выше код проверяет текущий назначенный скин для анимации, и если это "skin_Enemy1", он выбирает и устанавливает новый скин из одного из трех вариантов.
