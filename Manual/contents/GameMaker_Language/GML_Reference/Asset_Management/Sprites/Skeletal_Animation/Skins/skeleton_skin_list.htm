список_кожи_скелета

С помощью этой функции вы можете заполнить (предварительно созданный) список DS всеми именами скинов, включенных как часть скелетного анимационного спрайта. Эти имена будут строками и могут быть затем использовать в других функциях скинов скелетной анимации для этих типов спрайтов.

Синтаксис:

skeleton_skin_list(sprite, list);

Аргумент Описание
спрайт Индекс спрайта скелетной анимации Spine для получения списка.
список Идентификатор списка DS для заполнения именами анимаций.

Возвращается:

Н/Д

Пример:

var list = ds_list_create();
skeleton_skin_list(sprite_index, list);
var num = ds_list_size(list);
skeleton_skin_set(list[| irandom(num - 1));
ds_list_destroy(list);

Приведенный выше код создает список DS, затем заполняет его именами скинов. Затем выбирается случайный скин и применяется к экземпляру перед уничтожением списка.
