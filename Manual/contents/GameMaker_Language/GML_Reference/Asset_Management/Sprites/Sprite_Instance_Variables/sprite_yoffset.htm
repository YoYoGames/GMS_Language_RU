sprite_yoffset

Эта переменная, доступная только для чтения, возвращает локальное смещение y (y-компонент начала координат, определенный в редакторе спрайтов) спрайта, который был назначен данному экземпляру. Это смещение y возвращается в пикселях и зависит от масштаб_изображения, так что (например) если квадратный спрайт 32x32 определен как имеющий начало координат y на 16, то при масштабировании этого спрайта на 2 эта переменная вернет 32. Если вам нужно немасштабированное смещение по y, используйте sprite_get_yoffset().

Синтаксис:

sprite_yoffset;

Возвращается:

Real (значение с плавающей точкой одинарной точности)

Пример:

если y - sprite_yoffset < 0
{
y = sprite_yoffset;
}

Приведенный выше код проверяет, находится ли верхний край спрайта экземпляра за пределами комнаты, и если да, то перемещает экземпляр так, чтобы весь спрайт был виден.
