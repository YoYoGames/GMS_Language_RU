sprite_xoffset

Эта переменная, доступная только для чтения, возвращает локальное смещение x (x-компонент начала координат, определенный в редакторе спрайтов) спрайта, который был назначен данному экземпляру. Смещение xoffset возвращается в пикселях и зависит от значения параметра image_xscale, так что (например) если квадратный спрайт 32x32 определен как имеющий начало x в 16, то при масштабировании этого спрайта на 2 эта переменная вернет 32. Если вам нужно немасштабированное смещение x, используйте sprite_get_xoffset().

Синтаксис:

sprite_xoffset;

Возвращается:

Real (значение с плавающей точкой одинарной точности)

Пример:

если x - sprite_xoffset < 0
{
x = sprite_xoffset;
}

Приведенный выше код проверяет, находится ли левый край спрайта экземпляра за пределами комнаты, и если да, то перемещает экземпляр так, чтобы весь спрайт был виден.
