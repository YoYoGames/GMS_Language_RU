изображение_альфа

Эта переменная используется для получения или установки значения альфа для спрайта. Альфа всегда рассчитывается как значение между 0 и 1, где 0 - полностью прозрачный, а 1 - полностью непрозрачный. Обратите внимание, что для того, чтобы изменения в этой переменной были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker Studio 2 по умолчанию нарисует спрайт), либо быть нарисованным с помощью одной из расширенных функций рисования, таких как draw_self() или draw_sprite_ext().

Синтаксис:

image_alpha;

Возвращается:

Вещественный (от 0 до 1 - значение с плавающей точкой одинарной точности)

Пример:

image_alpha = clamp(image_alpha - 0.01, 0, 1);

Приведенный выше код будет медленно уменьшать значение image_alpha, пока оно не достигнет 0.
