смешивание изображений

Эта переменная управляет "тонированием" экземпляра спрайта, значение по умолчанию -1 (но может быть и c_white). Любое другое значение (включая внутренние цветовые константы, такие как c_red или c_aqua) будет смешивать указанный цвет с исходным спрайтом. Обратите внимание, что для того, чтобы изменения в этой переменной были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker Studio 2 по умолчанию нарисует спрайт), либо быть нарисованным с помощью одной из расширенных функций рисования, таких как например draw_self() или draw_sprite_ext().

Обратите внимание, что на платформе HTML5 следует ограничить смешивание (если только не используется WebGL), так как каждый смешиваемый спрайт должен кэшироваться отдельно, поэтому наличие большого количества смешиваемых спрайтов может негативно сказаться на производительности (вы также можете установить размер кэша с помощью функции функция sprite_set_cache_size()).

Синтаксис:

image_blend;

Возвращается:

Real (значение с плавающей точкой одинарной точности)

Пример:

image_blend = make_colour_hsv(255, 255, random(255));

Приведенный выше код установит image_blend спрайта, определенного для данного экземпляра, в случайный цвет.
