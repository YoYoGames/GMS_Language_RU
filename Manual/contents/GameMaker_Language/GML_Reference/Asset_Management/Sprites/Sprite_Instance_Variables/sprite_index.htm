sprite_index

Эта переменная возвращает индекс текущего спрайта для экземпляра, или -1, если экземпляр не имеет связанного с ним спрайта. Вы можете изменить ее, чтобы дать экземпляру другой спрайт, задав ему имя спрайта из дерева ресурсов или используя переменную, в которой проиндексирован внешний загруженный спрайт. Изменение спрайта не изменяет индекс текущего видимого кадра, поэтому если вы измените спрайт на кадре номер 3, новый спрайт будет отрисован с видимым кадром этого кадра (при условии, что он имеет такое же количество кадров).

Синтаксис:

sprite_index;

Возвращается:

Real (целое число - значение индекса ресурса)

Пример:

with (obj_Check)
{
if !collision_line(x, y, other.x, other.y, obj_Wall, false, true)
{
sprite_index = spr_spotted;
}
else
{
sprite_index = spr_clear;
}
}

Приведенный выше код будет перебирать все экземпляры "obj_Check", проверяя наличие линии столкновения между ними и экземпляром, выполняющим код. Спрайт этих экземпляров будет изменен в зависимости от возвращаемого значения (true или false) для линии столкновения.
