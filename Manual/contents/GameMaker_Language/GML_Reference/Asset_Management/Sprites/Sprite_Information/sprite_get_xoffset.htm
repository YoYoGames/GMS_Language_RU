sprite_get_xoffset

Когда вы определяете спрайт в редакторе спрайтов, вам необходимо задать его начало координат. Это точка, в которой спрайт будет "прикреплен" или "нарисован" при использовании с экземпляром. Эта функция возвращает относительное смещение для оси x начала координат, исходя из того, что верхний левый угол является позицией 0,0, при этом +x означает право, а -x - лево от этого. Следующее изображение иллюстрирует это:

Синтаксис:

sprite_get_xoffset(index);

Аргумент Описание
индекс Индекс спрайта, для которого нужно найти смещение x.

Возвращает

Реально

Пример:

if x - sprite_get_xoffset(sprite_index) < 0
{
x = sprite_get_xoffset(sprite_index);
}

Приведенный выше код обеспечит сохранение экземпляра в комнате на основе спрайта (так что спрайт всегда будет виден).
