sprite_get_texture

Эта функция возвращает специальный указатель на страницу текстуры спрайта. Это значение может быть использовано в других функциях рисования, в частности, в функциях 2D-примитивов, а также в Шейдер функции. Вы можете получить больше информации о возвращаемой странице текстуры, используя различные функции texture_, найденные здесь.

Синтаксис:

sprite_get_texture(spr, subimg);

Аргумент Описание
spr Индекс используемого спрайта.
subimg Вложенное изображение спрайта для использования.

Возвращает

Указатель (идентификатор страницы текстуры)

Пример:

var tex;
tex = sprite_get_texture(spr_Wall, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(480, 0, 1, 0);
draw_vertex_texture(480, 640, 1, 1);
draw_vertex_texture(0, 640, 0, 1, 1);
draw_primitive_end();

Приведенный выше код нарисует полосу треугольника с 4 вершинами, текстурированную текстурой, хранящейся в переменной "tex".
