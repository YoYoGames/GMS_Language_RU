sprite_get_tpe

Эта функция возвращает специальное значение, относящееся к позиции данного спрайта и суб-изображения на странице текстуры, используемой вашей игрой. Таким образом, вы можете передавать это значение расширениям для использования вне скомпилированной игры, эффективно обходя функции рисования GameMaker Studio 2 и позволяя использовать спрайт в содержимом DOM, которое затем может быть нарисовано в любом месте окна, содержащего игровую канву. Эта функция представляет особый интерес для тех, кто хочет создавать кнопки и другие интерактивные средства за пределами элемента холста GameMaker Studio 2 на своей странице с помощью функции clickable_add().

ПРИМЕЧАНИЕ: Эта функция предназначена только для HTML5.

Синтаксис:

sprite_get_tpe(sprite, index);

Аргумент Описание
спрайт Индекс спрайта для поиска записи страницы текстуры.
индекс Индекс изображения спрайта.

Возвращает

Реально

Пример:

home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");

Приведенный выше код создает DOM-кнопку в позиции (32, 32) страницы, на которой запущен игровой холст. Кнопка использует спрайт, ссылающийся на страницу текстуры как "spr_MS_Home", и при нажатии на кнопку откроется новое окно для указанному URL и с определенными свойствами для этого окна.
