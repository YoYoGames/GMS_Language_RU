sprite_get_name

Эта функция возвращает имя в виде строки указанного спрайта. Это имя - то, которое было указано для спрайта в дереве ресурсов главного окна GameMaker Studio 2. Обратите внимание, что это только строка и не может быть использована для прямого обращения к спрайту - для этого вам понадобится индекс спрайта. Однако вы можете использовать эту строку для получения индекса спрайта, используя возвращаемую строку вместе с функцией asset_get_index().

Синтаксис:

sprite_get_name(index);

Аргумент Описание
индекс Индекс спрайта, имя которого нужно получить.

Возвращает

Строка

Пример:

str = sprite_get_name(sprite_index);

Приведенный выше код получит имя индекса спрайта для экземпляра, выполняющего код, и сохранит возвращаемое значение в переменной "str".
