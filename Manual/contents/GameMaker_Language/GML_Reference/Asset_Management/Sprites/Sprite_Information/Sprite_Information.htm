Информация о спрайте

Следующие функции можно использовать для получения информации об активе спрайта. Обратите внимание, что эти функции получают информацию из базового спрайта (т. е. актива спрайта, созданного в браузере активов). а не из спрайта, используемого в экземпляре. Например, вы можете использовать спрайт переменная экземпляра sprite_width для получения ширины спрайта, который используется в экземпляре, и эта ширина будет учитывать любое масштабирование, которое могло быть применено. Однако если вам нужно узнать ширину спрайта в обычном виде, без применения преобразований, вам нужно использовать функцию sprite_get_width(), приведенную ниже.

Для получения дополнительной информации о переменных экземпляра спрайта см. раздел " Переменные экземпляра".

  • sprite_get_name
  • sprite_get_number
  • sprite_get_speed
  • sprite_get_speed_type
  • sprite_get_width
  • sprite_get_height
  • sprite_get_xoffset
  • sprite_get_yoffset
  • sprite_get_bbox_bottom
  • sprite_get_bbox_left
  • sprite_get_bbox_right
  • sprite_get_bbox_top
  • sprite_get_bbox_mode
  • sprite_get_nineslice
  • sprite_get_tpe
  • sprite_get_texture
  • sprite_get_uvs
  • sprite_get_info
