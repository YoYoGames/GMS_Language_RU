сохранение спрайта

Эта функция может быть использована для сохранения любого под-изображения спрайта на диск, задав ему указанное имя файла. Спрайт должен быть добавлен во время выполнения (вы не можете сохранять спрайты, добавленные через IDE), и файл должен быть сохранен с расширением *.png.

ПРИМЕЧАНИЕ: В зависимости от выбранной целевой платформы вы ограничены в том, откуда вы можете сохранять и загружать файлы. Дополнительную информацию см. в разделе Работа с файлами.

Синтаксис:

sprite_save(ind, subimg, fname);

Аргумент Описание
инд Индекс спрайта для сохранения.
subimg Вложенное изображение спрайта для сохранения.
fname Имя файла для сохраненного спрайта.

Возвращает

Н/Д

Пример:

var surf, spr_custom;
surf = surface_create(32, 32);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
draw_sprite(spr_Body, 0, 0, 0);
draw_sprite(spr_Clothes, 0, 0, 0);
draw_sprite(spr_Hair, 0, 0, 0);
spr_custom = sprite_create_from_surface(surf, 0, 0, 32, 32, true, true, 16, 16);
surface_reset_target();
surface_free(surf);
sprite_save(spr_custom, 0, "Player_Custom_Sprite.png");
sprite_delete(spr_Custom);

Приведенный выше код создает поверхность и сохраняет ее индекс в локальной переменной "surf". Затем он нацеливается на эту поверхность, очищает ее и рисует несколько спрайтов друг над другом. Он создает новый спрайт из составного изображения, нарисованного на поверхности и присваивает его индекс локальной переменной "spr_Custom", после чего освобождает память, используемую поверхностью. Наконец, полученный спрайт сохраняется в файл и затем также удаляется из памяти.
