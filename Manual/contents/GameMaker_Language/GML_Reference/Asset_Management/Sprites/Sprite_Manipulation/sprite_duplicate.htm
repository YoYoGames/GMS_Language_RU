дублирование спрайта

Эта функция возвращает индекс вновь созданного спрайта, который является дубликатом (копией) того, который был введен в качестве аргумента "index". Если функция не выполнится, возвращается -1. Эта функция должна быть использована для копирования любых спрайты из исходных активов, прежде чем с ними можно будет производить какие-либо преобразования. Например, если вы хотите изменить ограничивающую рамку спрайта или установить его альфу из другого спрайта, вы должны сначала продублировать его, затем выполнить операцию на дублированном спрайте и использовать его. Дублированный спрайт при создании будет размещен на своей собственной уникальной текстурной странице, поэтому дублирование нескольких спрайтов приведет к созданию нескольких текстурных страниц и повлияет на производительность, поэтому используйте эту функцию только в случае необходимости. при необходимости.

ПРИМЕЧАНИЕ: Когда вы дублируете спрайт в GameMaker Studio 2, вы должны не забыть удалить его снова (с помощью функции sprite_delete()) когда он больше не нужен, иначе существует риск утечки памяти, что приведет к замедлению и, в конечном итоге, к краху вашей игры.

Синтаксис:

sprite_duplicate(index);

Аргумент Описание
индекс Индекс спрайта, который нужно продублировать.

Возвращает

Real (уникальное значение ID спрайта)

Пример:

new_spr = sprite_duplicate(sprite_index)

Приведенный выше код дублирует спрайт, используемый в настоящее время в качестве индекса спрайта экземпляра, и сохраняет индекс для этого нового спрайта в переменной "new_spr".
