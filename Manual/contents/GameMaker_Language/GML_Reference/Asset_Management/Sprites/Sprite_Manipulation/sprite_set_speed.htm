sprite_set_speed

Эта функция используется для установки базовой скорости и типа анимации для ресурса спрайта. Обычно это задается в редакторе спрайтов, но могут возникнуть моменты, когда вам потребуется установить это вручную для данного ресурса спрайта. Необходимо указать индекс спрайта, скорость анимации и тип скорости анимации. Тип скорости анимации может быть любой из следующих констант:

Постоянная Описание
spritespeed_framespersecond Спрайт был определен с анимацией в кадрах в секунду.
spritespeed_framespergameframe Спрайт был определен с анимацией в кадрах на один игровой кадр.


Выбор типа скорости, используемой для расчета анимации, очень важен, поскольку между двумя вариантами существует огромная разница. Следующие два примера иллюстрируют эту разницу:

  • Если у вас есть спрайт, который рисует 1 кадр в секунду, а скорость изображения установлена на 0,5, он будет рисоваться со скоростью 0,5 кадра в секунду. Если частота кадров в игре составляет 60 кадров в секунду, спрайт будет рисовать 1 кадр на каждые 120 игровых кадров.
  • Если у вас есть спрайт, который рисует 1 кадр на игровой кадр, и вы установили скорость изображения на 0,5, он будет рисовать 0,5 кадра на игровой кадр. Если частота кадров в игре составляет 60 кадров в секунду, спрайт будет рисовать 30 кадров на каждые 60 игровых кадров.

Узнать, какое значение скорости использовалось для анимации спрайта, можно с помощью функции sprite_get_speed(), а тип скорости анимации можно получить с помощью функции sprite_get_speed_type().

ВАЖНО! Эта функция влияет на ресурс спрайта, поэтому любые изменения, сделанные с помощью этой функции, повлияют на все экземпляры, использующие спрайт, а также на все последующие экземпляры. также на все будущие экземпляры.

Синтаксис:

sprite_set_speed(index, speed, type)

Аргумент Описание
индекс Индекс спрайта для получения типа анимации

Возвращается:

Постоянная (см. выше)

Пример:

sprite_set_speed(sprite_index, 10, spritespeed_framespersecond);

Приведенный выше код устанавливает базовую скорость анимации для спрайта, назначенного данному экземпляру, на 10 кадров в секунду.
