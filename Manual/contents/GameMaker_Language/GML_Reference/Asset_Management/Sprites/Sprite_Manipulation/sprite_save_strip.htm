sprite_save_strip

Эта функция создает изображение полосы из всех подизображений спрайта и сохраняет его на диске с указанным именем файла. Спрайт должен быть добавлен во время выполнения программы (нельзя сохранять спрайты, добавленные через IDE), а файл файл должен быть сохранен с расширением *.png.

ПРИМЕЧАНИЕ: В зависимости от выбранной целевой платформы вы ограничены в том, откуда вы можете сохранять и загружать файлы. Дополнительную информацию см. в разделе Работа с файлами.

Синтаксис:

sprite_save_strip(ind, filename);

Аргумент Описание
инд Индекс спрайта для изменения карты прозрачности.
имя файла Имя файла для сохраненной полосы спрайта.

Возвращает

Н/Д

Пример:

var spr, i;
spr = sprite_create_from_surface(0, 0, 32, 32, true, true, 16, 16);
for (i = 1; i < 8; i +=1)
{
sprite_add_from_surface(spr, i, 0, 32, 32, true, true, 16, 16);
}
sprite_save_strip(spr, "Custom_Player_Sprite.png");
sprite_delete(spr);

Приведенный выше код создает спрайт из поверхности, индексированной в переменной "surf", присваивая ее индекс локальной переменной "spr_Custom", а затем использует цикл for для перемещения по поверхности и захвата различных участков, которые добавляются в спрайт в качестве подизображений. Этот новый спрайт затем сохраняется как полоса png перед удалением из памяти.
