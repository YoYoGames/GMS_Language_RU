sprite_assign

Эта функция берет два ранее созданных (или включенных) индекса спрайта и копирует изображение из одного в другой. Таким образом, вы можете копировать (или "клонировать") один спрайт в другой индекс. Обратите внимание, что копирование в игровой ресурс невозможно. Вы должны были создать спрайт, в который будет производиться копирование, ранее с помощью функции sprite_add() или sprite_duplicate() функции.

Синтаксис:

sprite_assign(index, sprite);

Аргумент Описание
индекс Индекс спрайта, в который будет скопирован (т.е. перезаписан).
спрайт Спрайт, с которого нужно скопировать.

Возвращает

Н/Д

Пример:

if sprite_exists(global.Player_Sprite)
{
var t_spr = sprite_add("player.png", 16, true, true, 0, 0);
sprite_assign(global.Player_Sprite, t_spr);
sprite_delete(t_spr);
}
else
{
global.Player_Sprite = sprite_add("player.png", 16, true, true, 0, 0);
}

Приведенный выше код проверяет, содержит ли глобальная переменная "Player_Sprite" спрайт, и если содержит, то использует sprite_assign для замены его на тот, который был загружен из внешнего файла. Если переменная не содержит спрайта, он загружается и его индекс хранится в этой переменной.
