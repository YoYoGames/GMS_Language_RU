tileset_get_name

Эта функция возвращает имя в виде строки указанного набора плиток. Это имя - то, которое было указано для набора плиток в Asset Browser главного окна GameMaker Studio 2. Обратите внимание, что это только строка и не может быть использована для прямой ссылки на набор плиток - для этого вам понадобится индекс набора плиток. Однако вы можете использовать эту строку для получения индекса набора плиток, используя возвращаемую строку вместе с функцией asset_get_index().

Синтаксис:

tileset_get_name(index);

Аргумент Описание
индекс Индекс набора плиток для получения названия.

Возвращает

Строка

Пример:

var _l = layer_get_id("tilemap_trees");
var _m = layer_tilemap_get_id(_l);
var _t = tilemap_tileset(_m);
tileset_name = tileset_get_name(_t);

Приведенный выше код получит имя индекса набора плиток для заданного слоя, сохраняя возвращаемую строку в переменной tileset_name.
