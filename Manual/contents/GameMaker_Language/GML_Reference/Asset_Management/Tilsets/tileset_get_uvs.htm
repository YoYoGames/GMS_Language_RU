tileset_get_uvs

Эта функция возвращает массив с UV-координатами и другими данными для текстуры заданной плитки, установленной на странице текстуры. Функция возвращает массив со следующими 8 элементами:

  • [0] = левый
  • [1] = верх
  • [2] = право
  • [3] = низ
  • [4] = количество пикселей, которые компилятор активов обрезал с левой стороны набора плиток
  • [5] = количество пикселей, которые компилятор активов обрезал с верхней стороны набора плиток
  • [6] = нормализованный процент пиксельных данных от исходной ширины набора плиток, которые были сохранены на странице текстуры
  • [7] = нормализованный процент пиксельных данных от высоты исходного набора плиток, который был сохранен на странице текстуры

Этот массив затем можно использовать в других функциях рисования, особенно при работе в 3D или при использовании функций 2D-примитивов, а также при работе с функциями Шейдер функции.

Синтаксис:

tileset_get_uvs(tileset);

Аргумент Описание
набор плиток Индекс набора плиток, который необходимо использовать.

Возвращает

Массив (1D, 8 элементов)

Пример:

var tex = tileset_get_uvs(tl_Grass);
tex_left = tex[0];
tex_top = tex[1];
tex_right = tex[2];
tex_bottom = tex[3];

Приведенный выше код сохранит УФ-координаты для заданного набора плиток в локальном массиве, а затем присвоит значения переменным экземпляра.
