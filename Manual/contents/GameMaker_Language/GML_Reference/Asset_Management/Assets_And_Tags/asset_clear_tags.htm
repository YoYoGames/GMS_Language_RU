asset_clear_tags

С помощью этой функции вы можете очистить все теги, присутствующие на любом активе, из браузера активов. Вы передаете либо имя актива (в виде строки), либо его индекс, а если вы передаете значение индекса актива, то вам необходимо передать необязательный аргумент тип актива (константа), так как активы разных типов могут иметь одинаковый индекс, хотя они не могут иметь одинаковое имя. Доступные типы активов перечислены в таблице ниже:

Постоянная Описание
актив_объекта Данное имя относится к объекту.
актив_спрайт Данное имя относится к спрайту.
актив_звук Данное имя обозначает звук.
комната_активов Данное имя относится к помещению.
актив_плитки Данное название относится к набору плиток.
путь_актива Данное имя относится к пути.
актив_скрипт Данное имя относится к сценарию.
asset_font Данное название относится к шрифту.
временная шкала активов Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдер.
asset_animationcurve Данное название относится к кривой анимации.
последовательность активов Данное имя относится к последовательности.

Если функция успешно очистит теги, она вернет true, в противном случае она вернет false.

Синтаксис:

asset_clear_tags(name,[asset_type]);

Аргумент Описание
имя_или_индекс Название актива (строка) или значение его индекса (целое число).
[asset_type] ВАРИАНТ! Тип актива для очистки тегов, используется только при указании индекса для первого аргумента.

Возвращается:

Булево

Пример:

asset_clear_tags(obj_Enemy_Parent, asset_object);

Приведенный выше код очистит все теги из заданного объекта-актива.
