tag_get_assets

С помощью этой функции вы можете получить имена всех активов, которым был присвоен заданный тег или теги. Вы передаете либо одну строку тега, либо массив, где каждый элемент массива - это строка тега. Функция вернет массив, каждый элемент которого является именем (в виде строки) актива с заданным тегом. Если вам нужен уникальный индекс для актива, то вы можете использовать функцию asset_get_index() вместе с вместе с возвращаемым именем. Если активов с заданным тегом (тегами) нет, или если произошла ошибка (например, заданные теги не существуют), то будет возвращен пустой массив.

Синтаксис:

tag_get_assets(tags);

Аргумент Описание
теги Одна строка тега актива или массив с различными тегами актива.

Возвращается:

Массив

Пример:

backgrounds = ds_list_create();
var _assets = tag_get_assets("background");
for (var i = 0; i< array_length(_assets); ++i;)
{
if asset_get_type(_assets[i]) == asset_sprite
{
ds_list_add(backgrounds, asset_get_index(_assets[i]));
}
}

Приведенный выше код создает список, затем извлекает имена всех активов с тегом "background". Он циклически просматривает массив возвращенных имен, проверяя, являются ли какие-либо из них спрайтовыми активами, и если да, то уникальное значение индекса для актива добавляется в список для дальнейшего использования.
