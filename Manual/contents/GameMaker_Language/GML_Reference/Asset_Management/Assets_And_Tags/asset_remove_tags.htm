asset_remove_tags

С помощью этой функции вы можете удалить одну или несколько строк тегов для любого актива из браузера активов. Вы передаете либо имя актива (в виде строки), либо его индекс, а также либо одну строку тегов, либо массив, каждый элемент которого является одной строкой тегов. Если вы указываете значение индекса актива, то вам необходимо указать дополнительный аргумент типа актива (константа), так как активы разных типов могут иметь одинаковый индекс, хотя они не могут иметь одинаковое имя. Доступные типы активов перечислены в таблице ниже:

Постоянная Описание
актив_объекта Данное имя относится к объекту.
актив_спрайт Данное имя относится к спрайту.
актив_звук Данное имя обозначает звук.
комната_активов Данное имя относится к помещению.
актив_плитки Данное название относится к набору плиток.
путь_актива Данное имя относится к пути.
актив_скрипт Данное имя относится к сценарию.
asset_font Данное название относится к шрифту.
временная шкала активов Данное имя относится к временной линии.
asset_shader Данное имя относится к шейдер.
asset_animationcurve Данное название относится к кривой анимации.
последовательность активов Данное имя относится к последовательности.

Если функция успешно удалит метку (метки), она вернет true, в противном случае она вернет false.

Синтаксис:

asset_remove_tags(name_or_index, tags, [asset_type]);

Аргумент Описание
имя_или_индекс Название актива (строка) или значение его индекса (целое число).
теги Одна строка тега актива или массив с различными тегами актива.
[asset_type] ВАРИАНТ! Тип актива для удаления тегов, используется только при указании индекса для первого аргумента.

Возвращается:

Булево

Пример:

var _a = array_create(3);
_a[0] = "enemy";
_a[1] = "all_levels";
_a[2] = "boss";
asset_remove_tags(obj_Enemy_Boss_Parent, _a, asset_object);

Приведенный выше код создаст массив тегов, а затем удалит их из заданного объекта.
