asset_get_index

Вы можете использовать эту функцию для получения уникального идентификационного индекса для игрового актива из его названия. Если актив не найден, функция вернет значение -1, в противном случае она вернет уникальный идентификатор для проверяемого актива. Этот идентификатор можно можно использовать в других функциях, как и любое другое значение индекса, например, индекс sprite_index или индекс_пути, например. Обратите внимание, что хотя эта функция может использоваться для ссылки на активы из строк (см. пример ниже), вы всегда должны убедиться, что актив существует, прежде чем использовать его, иначе вы можете получить ошибки, которые приведут к краху вашей игры.

Синтаксис:

asset_get_index(name);

Аргумент Описание
имя Имя игрового актива для получения индекса (строка).

Возвращается:

Индекс актива (целое число)

Пример:

var obj = asset_get_index("obj_Enemy_" + string(global.Level));
if obj > -1
{
instance_create_layer(random(room_width), random(room_height), obj, "Enemy_Layer");
}

Приведенный выше код получит индекс объекта из строки, и если этот индекс существует, создаст экземпляр объекта в игре.
