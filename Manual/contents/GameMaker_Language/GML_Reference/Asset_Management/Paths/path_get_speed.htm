путь_получить_скорость

Эта функция возвращает коэффициент скорости для любой заданной позиции на пути. Позиция должна быть значением между 0 и 1 (вы можете использовать положение_путинапример, если вам нужна текущая позиция экземпляра), причем 0 - это начальная позиция, а 1 - конечная. Возвращаемое значение будет коэффициентом, на который фактически скорость_пути устанавливается. Например, на пути с двумя точками, 0 и 1, где коэффициент скорости точки 0 равен 100, а коэффициент скорости точки 1 равен 0, функция path_get_speed(path, 0.5) вернет 50.

Синтаксис:

path_get_speed(ind, pos);

Аргумент Описание
индекс Индекс пути для проверки.
поз Как далеко по пути нужно пройти для проверки. От 0 (начало) до 1 (конец), может быть десятичным.

Возвращается:

Реально

Пример:

скорость = path_get_speed(pth_Patrol, 0.5) / 20;

Это установит скорость вызывающего экземпляра на коэффициент скорости точки, находящейся ровно на половине заданного пути, деленный на 20. Таким образом, если найденный коэффициент скорости равен 100 по умолчанию, то скорость вызывающего экземпляра будет равна 5.
