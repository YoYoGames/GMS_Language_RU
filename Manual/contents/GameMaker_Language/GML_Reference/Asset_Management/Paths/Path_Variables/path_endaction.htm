путь_действия

Эта переменная может быть использована для получения или изменения реакции экземпляра, когда он достигает конца текущего пути. Обычно вы устанавливаете эту переменную при начале пути с помощью функции path_start() но вы можете захотеть изменить это поведение в зависимости от любого количества событий в вашей игре. Доступные значения выражаются с помощью следующих констант:

Постоянная Описание
путь_действия_остановки остановить путь
путь_действия_перезапуска продолжить с начальной позиции, переходя к началу, если путь не закрыт
путь_действия_продолжить начать путь заново с текущей позиции
путь_действия_обратный изменить скорость движения (проложить путь в обратном направлении)

Синтаксис:

path_endaction;

Возвращается:

Постоянная (см. описание)

Пример:

if (path_endaction == path_action_stop)
{
path_endaction = path_action_reverse;
}

Приведенный выше код проверит действие конца пути, и если оно установлено на stop, то действие конца будет изменено на reverse.
