путь_начало

Путь создается из серии определяющих точек, которые соединяются вместе и затем используются для планирования движения экземпляра. Они могут быть созданы с помощью кода или в редакторе путей, и они назначаются экземпляру для использования в игре. С помощью этой функции можно указать экземпляру, по какому пути следовать, с какой скоростью (измеряется в пикселях за шаг), как вести себя, когда он достигнет конца пути, и следовать ли абсолютному или относительному положению пути. позиция. Последняя часть означает, что он либо начинает и следует по пути точно так, как вы его спроектировали и разместили в редакторе путей (абсолютный), либо он начинает и следует по пути с позиции в которой был создан экземпляр (относительная).

Как видно из приведенного выше изображения, два экземпляра следуют по одному и тому же пути, начатому с помощью path_start(), но один из экземпляров следует по абсолютному пути, а другой - по относительному. Следует отметить, что при установке relative в false (абсолютный) экземпляр будет "прыгать" к началу пути независимо от того, в какой части комнаты он находится. комнате, где бы он ни находился.

Константы, используемые для определения поведения экземпляра, когда он достигает конца пути, следующие:

Постоянная Описание
путь_действия_остановки Завершить путь
путь_действия_перезапуска Продолжить путь с самого начала, снова переходя к начальной позиции, если путь не закрыт
путь_действия_продолжить Продолжить с текущей позиции
путь_действия_обратный Снова пройти назад по тропинке (достигается путем изменения скорости движения по тропинке)

Синтаксис:

path_start(path, speed, endaction, absolute);

Аргумент Описание
путь Индекс пути для запуска.
скорость Скорость, с которой нужно следовать по пути в пикселях за шаг, отрицательная означает движение назад.
endaction Что делать, когда достигнут конец пути (см. константы, перечисленные ниже).
абсолютный Должен ли вызывающий экземпляр следовать по абсолютному пути, как он находится в редакторе (true) или по относительному пути к его текущей позиции (false).

Возвращается:

Н/Д

Пример:

path_start(path, 4, path_action_reverse, 0);

Это заставит вызывающий экземпляр следовать по пути, индексированному в переменной "path", со скоростью 4 пикселя за шаг. Он будет следовать по пути относительно собственной позиции, а когда достигнет конца, изменит направление и вернется назад по пути путь, по которому он пришел.

Назад: Пути
Далее: path_end
