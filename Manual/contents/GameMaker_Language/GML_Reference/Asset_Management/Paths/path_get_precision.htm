path_get_precision

Эта функция возвращает, с какой точностью данный путь был "сглажен", и это будет целочисленное значение от 1 до 8. Хотя вы можете получить (и установить) это значение для прямолинейного пути, оно не будет иметь никакого влияния на то, как экземпляр использует путь, так как оно имеет значение только в том случае, если тип пути установлен на "гладкий".

Синтаксис:

path_get_precision(index);

Аргумент Описание
индекс Индекс пути для проверки.

Возвращается:

Реально

Пример:

if (path_get_kind(pth_Patrol))
{
if (path_get_precision(pth_Patrol)) != 8
{
path_set_precision(pth_Patrol, 8);
}
}

Приведенный выше код проверяет путь, индексированный в "pth_Patrol", на предмет его вида, и если он гладкий, то проверяет, насколько точным является сглаживание. Если она меньше 8, то устанавливается значение 8.
