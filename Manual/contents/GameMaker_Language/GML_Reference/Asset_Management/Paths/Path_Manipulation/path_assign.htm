путь_назначения

С помощью этой функции можно скопировать данные пути с одного пути на другой. Путь, в который копируется, сначала очищается (если в нем есть точки пути) и полностью перезаписывается путем из которого копируется. При этом ни один из путей не удаляется, и в результате получаются два пути с двумя разными индексами, но совершенно одинаковой формой и точками. В общем случае вы захотите использовать эту функцию на пути, созданном с помощью path_add(), так как если вы используете ее на активе пути, она будет постоянно влиять на путь для всех экземпляров в игре с момента использования функции и до конца игры.

Синтаксис:

path_assign(index, path);

Аргумент Описание
индекс Индекс пути, который нужно перезаписать.
путь Индекс пути, который будет перезаписывать 'index'.

Возвращается:

Н/Д

Пример:

mypath = path_add();
path_assign(mypath, choose(path_1, path_2, path_3));

Приведенный выше код создаст новый пустой путь, индексированный в переменной "mypath", а затем скопирует данные пути из одного из трех доступных ресурсов путей.
