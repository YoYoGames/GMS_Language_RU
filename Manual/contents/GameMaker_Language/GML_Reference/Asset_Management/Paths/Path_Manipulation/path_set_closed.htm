путь_закрыт

Эта функция может быть использована для обозначения заданного пути как открытого (false) или закрытого (true). У замкнутого пути начальная точка соединена с конечной точкой, образуя петлю, а у открытого пути начало и конец не связаны между собой. Эта функция изменяет фактический актив пути, Если это не то, что вам нужно, используйте функцию path_duplicate() для создания копии пути, а затем вызовите эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_set_closed(index, closed);

Аргумент Описание
индекс Индекс пути, который нужно изменить.
закрытый Закрыт ли путь (true) или нет (false).

Возвращается:

Н/Д

Пример:

path_set_closed(pth_Patrol, true);

Это установит заданный путь как замкнутый.
