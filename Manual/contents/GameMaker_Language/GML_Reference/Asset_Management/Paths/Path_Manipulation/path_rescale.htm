path_rescale

Эта функция может быть использована для изменения масштаба заданного пути по вертикальной и горизонтальной оси, перемещая каждую из точек пути в новое положение, соответствующее этому масштабу, вокруг центра пути. Эта функция изменяет фактический актив пути, и таким образом постоянно влияет на то, как этот путь используется всеми экземплярами в игре с момента использования функции до конца игры. Если это не то, что вам нужно, то следует использовать функцию типа path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_rescale(index, xscale, yscale);

Аргумент Описание
индекс Индекс пути для масштабирования.
xscale На что умножить текущий горизонтальный масштаб. По умолчанию масштаб равен 1.
yscale На что умножить текущий вертикальный масштаб. По умолчанию масштаб равен 1.

Возвращается:

Н/Д

Пример:

path_rescale(pth_AI, 3, 3);

Это изменит масштаб заданного пути до трехкратного исходного размера.
