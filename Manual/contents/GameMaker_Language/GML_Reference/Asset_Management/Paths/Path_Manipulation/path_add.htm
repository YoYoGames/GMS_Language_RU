путь_добавления

С помощью этой функции вы можете создать путь в GameMaker Studio 2 без использования редактора путей. Эта функция возвращает индекс пути, который должен быть сохранен в переменной и в дальнейшем использоваться в качестве ссылки для этого пути. Обратите внимание что созданный путь пуст, т.е. в нем нет определенных точек, поэтому вам придется использовать другие доступные функции для добавления точек к пути или использовать сетки MP для генерирования пути. После того, как вы закончили использовать путь или хотите создать новый и сохранить его индекс в той же переменной, сначала удалите старый путь с помощью функции путь_удалить чтобы предотвратить утечку памяти, которая что в конечном итоге может привести к краху вашей игры.

Синтаксис:

path_add();

Возвращается:

Реально

Пример:

global.newpath = path_add();

Это создаст новый путь и присвоит его индекс global.newpath.
