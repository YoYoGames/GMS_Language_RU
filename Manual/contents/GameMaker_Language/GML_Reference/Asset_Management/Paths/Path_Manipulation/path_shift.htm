сдвиг_пути

С помощью этой функции можно сдвинуть путь по горизонтальной и вертикальной оси. Обратите внимание, что это не будет иметь видимого эффекта в игре, если экземпляр, следующий за смещаемым путем, следует за ним относительно (а не абсолютно), так как в этом случае его позиция пути в этом случае не учитывается. Эта функция изменяет фактический актив пути и, таким образом, постоянно влияет на то, как этот путь используется всеми экземплярами в игре с момента использования функции до конца игры. Если это это не то, что вам нужно, то вам следует использовать функцию типа path_duplicate(), чтобы сначала создать копию пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_shift(index, xshift, yshift);

Аргумент Описание
индекс Индекс пути, который нужно изменить.
xshift Количество горизонтальных пикселей для сдвига контура. Отрицательный = влево, положительный = вправо.
yshift Количество вертикальных пикселей для сдвига контура. Отрицательный = вверх, положительный = вниз.

Возвращается:

Н/Д

Пример:

my_path = path_duplicate(pth_AI);
path_shift(my_path, 50, 50);
path_start(my_path, 4, path_action_reverse, true);

Приведенный выше код дублирует актив пути и сохраняет значение его ID в переменной. Затем эта переменная используется для смещения позиции дублированного пути, после чего экземпляр начинает путь.
