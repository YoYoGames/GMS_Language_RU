путь_приложения

С помощью этого кода вы можете добавить один путь к другому, эффективно соединив их вместе, и для того, чтобы это сработало, оба пути должны существовать ранее. Как правило, эту функцию лучше использовать для путей, созданных с помощью path_add(), так как если вы используете ее на активе пути, она будет постоянно влиять на путь для экземпляров в игре с момента использования функции и до конца игры.

ПРИМЕЧАНИЕ: Это не удалит добавляемый путь из игры. Он все еще там, только теперь в нем нет точек, и если вы больше не хотите его использовать, вам следует удалить его с помощью функции path_delete().

Синтаксис:

path_append(index, path);

Аргумент Описание
индекс Новый индекс пути.
путь Индекс пути, который будет добавлен к 'index'.

Возвращается:

Н/Д

Пример:

path_append(mypath, mppath);
path_delete(mppath);

Приведенный выше код добавляет данные пути, индексированные в переменной "mppath", к пути, индексированному в переменной "mypath", а затем удаляет "mppath" из памяти.
