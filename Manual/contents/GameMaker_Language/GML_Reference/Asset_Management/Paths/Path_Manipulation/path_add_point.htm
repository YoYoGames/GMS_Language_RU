точка_добавления_пути

С помощью этой функции можно добавить точку к заданному пути и установить коэффициент ее скорости. Точка добавляется в конец пути, независимо от положения точки, а коэффициент равен проценту от скорость_пути того скорость, с которой движется следующий экземпляр, когда он достигает этой точки пути. Если вы хотите поместить точку пути в какую-либо другую позицию, не являющуюся концом, используйте path_insert_point().

Если функция используется для пути из браузера активов, обратите внимание, что она изменит фактический актив и, таким образом, навсегда повлияет на то, как этот путь используется всеми экземплярами в игре с момента использования функции до конца игры. Если это не то, что вам нужно, то следует использовать функцию типа path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать на активе). не забудьте вызвать path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_add_point(index, x, y, speed);

Аргумент Описание
индекс Индекс пути, к которому нужно добавить точку.
x Координата x новой точки.
y Координата y новой точки.
скорость Коэффициент скорости точки (значение по умолчанию - 100).

Возвращается:

Н/Д

Пример:

my_path = path_add();
for (var i = 0; i <= 360; i += 36;)
{
path_add_point(my_path, x + lengthdir_x(50, i), y + lengthdir_y(50, i), 100);
}

Приведенный выше код создаст круговой путь вокруг позиции x/y экземпляра, выполняющего код.
