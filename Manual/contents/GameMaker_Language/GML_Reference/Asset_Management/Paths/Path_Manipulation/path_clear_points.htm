путь_очистить_точки

С помощью этой функции вы можете удалить все определения точек из пути, фактически создав "пустой" путь. Это не удаляет путь, для этого следует использовать функцию path_delete(), однако следует отметить, что эта функция изменяет фактический актив пути, а значит, будет постоянно влиять на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. Если это это не то, что вам нужно, то вам следует использовать функцию типа path_duplicate() для создания копии пути, затем вызвать эту функцию на дублированном активе (не забудьте вызвать функцию path_delete() на активе, когда он больше не нужен).

Синтаксис:

path_clear_points(index);

Аргумент Описание
индекс Индекс пути, который нужно очистить.

Возвращается:

Н/Д

Пример:

if path_get_number(mypath) > 0
{
path_clear_points(mypath);
}

Приведенный выше код проверяет, есть ли точки на пути, проиндексированном в переменной "mypath", и если есть, то очищает путь.
