путь_удаления

Вы можете использовать этот код для удаления пути из памяти. Если этот путь был создан динамически с помощью функции path_add()то переменная, содержащая индекс пути, больше не будет действительна для доступа к пути, так как она больше не существует а если путь был создан с помощью редактора путей, то этот путь больше не может быть доступен во всей игре, так как вы удаляете его навсегда.

Синтаксис:

path_delete(index);

Аргумент Описание
индекс Индекс пути для удаления.

Возвращается:

Н/Д

Пример:

var t_path = path_add();
if mp_grid_path(grid, t_path, x, y, obj_Player.x, obj_Player.y, 1)
{
path_assign(mypath, t_path);
}
path_delete(t_path);

Приведенный выше код создаст путь и сохранит его индекс в локальной переменной. Этот путь затем используется для хранения mp_grid_path() сгенерированный путь, который, если ему удается найти путь к цели, присваивается пути, индексированному в "mypath". Наконец, "t_path" удаляется.
