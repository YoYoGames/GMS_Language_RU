точка_вставки_пути

С помощью этой функции вы можете вставить новую точку в путь (путь может быть создан в редакторе путей или в коде с помощью функции path_add()). Точка будет добавлена в путь перед точкой "n" указанной в функции.

Синтаксис:

path_insert_point(index, n, x, y, speed);

Аргумент Описание
индекс Индекс пути, в который нужно вставить точку.
n Определяющая точка для вставки новой точки ДО.
x Координата x (относительно пути) новой точки.
y Координата y (относительно пути) новой точки.
скорость Коэффициент скорости точки.

Возвращается:

Н/Д

Пример:

path_insert_point(mypath, 0, 50, 50, 100);

В результате будет вставлена точка в самом начале пути, индексированного в переменной "mypath", в точке (50,50), с коэффициентом скорости 100%.
