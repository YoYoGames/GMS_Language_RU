audio_is_paused

Эта функция проверяет заданный звук, чтобы узнать, не приостановлен ли он в данный момент. Звук может быть либо одним экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() или звуковой актив, в этом случае будут проверены все экземпляры данного звука, и если какой-либо из них находится на паузе, функция вернет true, иначе она вернет false.

Синтаксис:

audio_is_paused(index);

Аргумент Описание
индекс Индекс звука для проверки.

Возвращается:

Булево

Пример:

if audio_is_paused(snd_Waterfall)
{
audio_resume_sound(snd_Waterfall);
}

Приведенный выше код проверяет, не приостановлен ли в данный момент звук, индексированный в переменной "snd_Waterfall", и если он возвращает true, то воспроизведение звука будет возобновлено.
