audio_stop_sound

Эта функция остановит заданный звук, если он воспроизводится в данный момент. Звук может быть либо одним экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() ), либо звуковой актив, в этом случае все экземпляры данного звука будут остановлены.

Синтаксис:

audio_stop_sound(index);

Аргумент Описание
индекс Индекс звука для остановки.

Возвращается:

Н/Д

Пример:

if !global.SFX
{
audio_stop_sound(snd_Waterfall);
}
else
{
audio_play_sound_at(snd_Waterfall, x, y, 0, 100, 300, 1, true, 1);
}

Приведенный выше код проверяет глобальную переменную "SFX" и если возвращает false, то останавливает звук, индексированный в переменной "snd_Waterfall", который играет в данный момент, а если возвращает true, он зацикливает звук.
