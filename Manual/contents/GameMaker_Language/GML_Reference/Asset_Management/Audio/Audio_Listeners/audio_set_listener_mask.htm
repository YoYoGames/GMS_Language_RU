audio_set_listener_mask

При использовании нескольких слушателей в системе вы можете установить битовую маску для звука, и он будет слышен только для отмеченного слушателя. Однако вы также можете установить глобальную маску с помощью этой функции, и все звуки, воспроизводимые в обычном режиме, будут слышны только тем слушателям, которые отмечены этой маской, без необходимости устанавливать маску для каждого звука. слушателей, отмеченных этой маской, без необходимости устанавливать маску для каждого звука отдельно.

Синтаксис:

audio_set_listener_mask(mask);

Аргумент Описание
маска Данные битовой маски, которые необходимо установить для слушателей

Возвращается:

Н/Д

Пример:

var num = audio_get_listener_count();
var mask = 0; for(var i = 0; i < num; ++i;)
{
var info = audio_get_listener_info(i);
var m = audio_listener_get_data(info[? "mask"]);
mask = mask | m;
ds_map_destroy(info);
}
audio_set_listener_mask(mask);

Приведенный выше код проверяет количество доступных слушателей, затем перебирает их и получает биты их масок, которые затем объединяются для создания однобитовой маски, применяемой к глобальному слушателю.
