audio_listener_position

С помощью этой функции можно изменить положение слушателя в трехмерном аудиопространстве. На рисунке ниже показано положение слушателя в аудиопространстве по умолчанию:

ПРИМЕЧАНИЕ: если у вас несколько слушателей, следует использовать функцию audio_listener_set_position().

Как вы можете видеть, положение по умолчанию - (0, 0, 0), но обычно вы используете эту функцию для перемещения слушателя вместе с объектом игрока в вашей игре и таким образом изменяете способ, которым звук, созданный эмиттерами, слышен игроку, например, на приведенном ниже изображении игры "сверху вниз" экземпляр игрока устанавливает слушатель, который заставит звук от различных эмиттеров "меняться" при перемещении игрока по уровню. на изображении игры сверху вниз, экземпляр игрока устанавливает слушателя, который заставит звук от различных эмиттеров "меняться" по мере перемещения игрока по уровню:

Синтаксис:

audio_listener_position(x, y, z);

Аргумент Описание
x Позиция x слушателя (по умолчанию 0).
y Позиция y слушателя (по умолчанию 0).
z Положение z слушателя (по умолчанию 0).

Возвращается:

Н/Д

Пример:

если скорость > 0
{
audio_listener_position(x, y, 0);
}

Приведенный выше код проверяет, превышает ли скорость экземпляра плеера 0, и если да, то обновляет аудиослушатель до текущей позиции x/y.
