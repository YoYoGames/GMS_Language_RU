ориентация аудио_слушателя

С помощью этой функции можно изменить ориентацию слушателя в трехмерном аудиопространстве. Направление взгляда и направление вверх основаны на векторах которые определяются из заданных относительных положений x, y и z, и по умолчанию имеют значения (0, 0, 1000) для направления взгляда и (0, 1, 0) для направления вверх, как показано на рисунке ниже:

ПРИМЕЧАНИЕ: если у вас несколько слушателей, вы должны использовать функцию audio_listener_set_orientation().

Изменение ориентации слушателя с помощью этой функции изменит то, как звуки, создаваемые звуковыми эмиттерами в игровой комнате, будут восприниматься игроком вашей игры. В приведенном ниже примере звуки, создаваемые излучателем, когда слушатель находится в положении по умолчанию будут казаться исходящими снизу и справа от слушателя, но при новом положении и ориентации слушателя они будут восприниматься как исходящие сверху и справа.

Синтаксис:

audio_listener_orientation(lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Аргумент Описание
lookat_x Вектор вида x (по умолчанию 0).
lookat_y Вектор вида y (по умолчанию 0).
lookat_z Вектор вида z (по умолчанию 1000).
up_x Вектор x вверх (по умолчанию 0).
up_y Вектор y вверх (по умолчанию 1).
up_z Вектор z вверх (по умолчанию 0).

Возвращается:

Н/Д

Пример:

xt = x + dcos(direction);
yt = y - dsin(direction);
zt = z - dsin(zdirection);
audio_listener_position(x, y, z)
audio_listener_orientation(xt, yt, zt, 0, 0, 1)

В приведенном выше коде используются три переменные для установки положения и ориентации 3D аудиослушателя.
