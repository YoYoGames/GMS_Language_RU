audio_listener_set_orientation

С помощью этой функции вы можете изменить ориентацию данного слушателя в трехмерном аудиопространстве. По умолчанию индекс слушателя равен 0, но вы можете использовать функцию audio_get_listener_info() для получить различные индексы, доступные для целевой платформы.

Вектор взгляда и вектор вверх основываются на значениях, которые определяются из заданных относительных положений x, y и z, и по умолчанию имеют значения (0, 0, -1) для вектора взгляда и (0, 1, 0) для вектора вверх, как показано на рисунке ниже:

Изменение заданной ориентации слушателя с помощью этой функции изменит то, как звуки, создаваемые звуковыми излучателями, расположенными вокруг игровой комнаты, будут восприниматься игроком вашей игры. В приведенном ниже примере звуки, создаваемые излучателем, когда слушатель находится в положении по умолчанию, будут казаться исходящими снизу и справа от слушателя, но при новом положении и ориентации слушателя они будут восприниматься как исходящие сверху и справа.

Синтаксис:

audio_listener_set_orientation(index, lookat_x, lookat_y, lookat_z, up_x, up_y, up_z);

Аргумент Описание
индекс Слушатель для установки ориентации.
lookat_x Вектор вида x (по умолчанию 0).
lookat_y Вектор вида y (по умолчанию 0).
lookat_z Вектор вида z (по умолчанию -1).
up_x Вектор x вверх (по умолчанию 0).
up_y Вектор y вверх (по умолчанию 1).
up_z Вектор z вверх (по умолчанию 0).

Возвращается:

Н/Д

Пример:

var _m = camera_get_view_mat(view_camera[0]);
audio_listener_set_position(global.Player_Listener, _m[0], _m[1], _m[2]);
audio_listener_set_orientation(info[? "index"], _m[3], _m[4], _m[5], _m[6], _m[7], _m[8]);

Приведенный выше код извлекает матрицу вида для вида камеры [0], а затем использует ее для установки положения и ориентации аудиослушателя с идентификатором, хранящимся в глобальной переменной "Player_Listener".
