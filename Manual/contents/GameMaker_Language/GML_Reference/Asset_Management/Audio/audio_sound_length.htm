аудио_звук_длительность

Эта функция возвращает длительность заданного звука в секундах. Звук может быть либо ссылкой из индекса для отдельного воспроизводимого звука, который был сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() или фактический звуковой актив из браузера активов.

Синтаксис:

audio_sound_length(index);

Аргумент Описание
индекс Индекс звука для проверки.

Возвращается:

Реально

Пример:

var len;
len = audio_sound_length(snd_Beam);
audio_play_sound(snd_Beam, 1, false);
alarm[0] = room_speed * len;

Приведенный выше код получает длительность (в секундах) звука, проиндексированного в переменной "snd_Beam", затем воспроизводит звук и устанавливает сигнал тревоги, когда звук закончит воспроизведение, используя длительность звука для расчета времени, необходимого для тревоги.
