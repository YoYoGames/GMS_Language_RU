audio_create_stream

С помощью этой функции вы можете создать новый звуковой индекс, который затем может быть использован в обычных аудиофункциях для потоковой передачи аудио непосредственно из внешнего источника файла OGG. Функция требует имя файла (которое может быть, например, включенным файлом) и вернет новый звуковой индекс для использования. Обратите внимание, что после того, как звук больше не нужен, необходимо вызвать функцию audio_destroy_stream() с индексом звука, чтобы удалить его из иначе может произойти утечка памяти, что приведет к замедлению и, в конечном итоге, к краху вашей игры.

ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.

Синтаксис:

audio_create_stream(filename);

Аргумент Описание
имя файла Файл (только OGG) для потокового воспроизведения аудио.

Возвращается:

Звуковой указатель

Пример:

snd = audio_create_stream("Music/Track1.ogg");
audio_play_sound(snd, 0, true);

Приведенный выше код создает новый индекс звука в переменной "snd" из заданного файла, затем воспроизводит этот звук.
