audio_sound_set_listener_mask

Эта функция может быть использована для установки битовой маски для звука, чтобы он воспроизводился только от указанных слушателей. Вы можете создать битовую маску с помощью функций audio_get_listener_info() и затем с помощью побитового или ("|") создать пользовательскую маску с теми слушателями, от которых вы хотите, чтобы звук воспроизводился, а затем применить эту маску к звуку после его воспроизведения. Эта маска будет отменяет маску по умолчанию или ту, которую вы задали с помощью функции audio_set_listener_mask().

ПРИМЕЧАНИЕ: Эта функция может быть использована только для звука после того, как он был установлен для воспроизведения, и требует идентификатор звука, возвращенный функцией, используемой для воспроизведения звука.

Синтаксис:

audio_sound_set_listener_mask(soundID, mask);

Аргумент Описание
soundID Уникальный идентификатор звука, для которого нужно установить маску
маска Данные битовой маски, которые необходимо установить для звука

Возвращается:

Н/Д

Пример:

var snd = audio_play_sound(snd_PlayerDead, 10, false);
audio_sound_set_listener_mask(snd, global.PlayerMask);

Приведенный выше код воспроизводит звук, затем устанавливает слушателя (слушателей) для воспроизведения, используя данные маски, хранящиеся в глобальной переменной.
