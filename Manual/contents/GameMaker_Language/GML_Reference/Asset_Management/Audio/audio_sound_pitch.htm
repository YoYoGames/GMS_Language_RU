audio_sound_pitch

Эта функция может быть использована для изменения высоты тона заданного звука. Звук может быть либо ссылкой на индекс отдельного воспроизводимого звука, который был сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() или фактический звуковой актив из браузера активов. Если это индекс воспроизводимого звука, то будет изменен только этот экземпляр, однако при использовании звукового актива из браузера активов, все последующие воспроизводимые экземпляры этого звука будут изменены.

Аргумент pitch является множителем высоты тона, то есть вводимое значение умножает текущую высоту тона на эту величину, поэтому значение по умолчанию 1 не изменяет высоту тона, в то время как значение меньше 1 понижает высоту тона, а больше 1 - повышает. увеличит высоту тона. Для этой функции лучше использовать небольшие приращения, так как значение меньше 0 или больше 5 может быть не слышно. Стоит отметить, что общее допустимое изменение высоты тона ограничено (1/256) - 256 октавами, поэтому любое значение больше или меньше не будет зарегистрировано.

ПРИМЕЧАНИЕ: Приведенное выше значение зажатого диапазона - это то, к чему GameMaker Studio 2 пытается привязать диапазон, но это значение не гарантируется на всех целевых платформах. Например, iOS, зажимается до (1/256) - 8, поэтому вам может понадобиться экспериментировать на каждой целевой платформе и иметь разные версии звукового ресурса, каждая из которых предварительно сдвинута, если вам нужны более высокие или более низкие значения октавы.

Синтаксис:

audio_sound_pitch(index, pitch);

Аргумент Описание
индекс Индекс звука, который нужно изменить.
шаг Множитель шага (по умолчанию 1).

Возвращается:

Н/Д

Пример:

var s_engine = audio_play_sound(snd_CarEngine, 10, false);
switch (gear)
{
case 1: audio_sound_pitch(s_engine, 0.8); break;
case 2: audio_sound_pitch(s_engine, 0.9); break;
case 3: audio_sound_pitch(s_engine, 0.95); break;
case 4: audio_sound_pitch(s_engine, 1); break;
case 5: audio_sound_pitch(s_engine, 1.2); break;
}

Приведенный выше код изменит высоту тона звука, воспроизводимого из звука, индексированного в переменной "s_engine", в зависимости от значения переменной "gear".
