audio_is_playing

Эта функция проверяет заданный звук, чтобы узнать, воспроизводится ли он в данный момент. Звук может быть либо одним экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() или звуковой актив, в этом случае будут проверены все экземпляры данного звука, и если какой-либо из них воспроизводится, функция вернет true, в противном случае она вернет false. Обратите внимание, что эта функция вернет true, если проверяемый звук был ранее приостановлен с помощью функции audio_pause_sound() функция.

Синтаксис:

audio_is_playing(index);

Аргумент Описание
индекс Индекс звука для проверки.

Возвращается:

Булево

Пример:

if !audio_is_playing(snd_Waterfall)
{
audio_play_sound_at(snd_Waterfall, x, y, 0, 300, true, 1);
}

Приведенный выше код проверяет, играет ли в данный момент звук, индексированный в переменной "snd_Waterfall", и если он возвращает false, то звук будет зациклен на позиции комнаты с расстоянием падения 300 и низким приоритетом. 300 и низким приоритетом.
