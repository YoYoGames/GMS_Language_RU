audio_queue_sound

Эта функция добавит данные из буфера в аудио очередь, которую вы ранее создали с помощью функции audio_create_play_queue(). Вы указываете индекс очереди, в которую нужно добавить данные, и ID буфера для получения данных, а также позицию (смещение) в буфере и количество байт для добавления. После добавления аудио из буфера в очередь, буфер не может быть удален, пока вы не освободите очередь с помощью функции audio_free_play_queue(), а свойства буфера должны соответствовать свойствам очереди, в которую вы добавляете звук.

Синтаксис:

audio_queue_sound(queueIndex, bufferId, bufferOffset, bufferLength);

Аргумент Описание
queueIndex Индекс очереди для добавления.
bufferId Буфер для добавления в очередь.
bufferOffset Смещение в исходном буфере, с которого следует начинать.
bufferLength Длина буфера (количество байтов в буфере).

Возвращается:

Н/Д

Пример:

len = async_load[? "data_len"];
audio_buff = buffer_create(len, buffer_fast, 1);
buffer_copy(async_load[? "buffer_id"], 0, len, buff, 0);
audio_queue_sound(audio_queue, audio_buff, 0, len);
audio_play_sound(audio_queue, 10, 0);

Приведенный выше код будет вызван в событии асинхронной аудиозаписи и присвоит некоторые записанные данные буферу, который затем добавляется в очередь аудио. Затем происходит воспроизведение.
