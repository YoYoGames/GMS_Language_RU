audio_emitter_get_gain

Эта функция возвращает текущее значение усиления (громкости), установленное для данного аудиоизлучателя, обычно в диапазоне от 0 до 1, где 0 - тишина, а 1 - полная громкость. Обратите внимание, что на некоторых платформах вы можете иметь коэффициент усиления больше 1, хотя значение 1 считается "полная громкость", а большее значение может привести к обрыву звука.

Синтаксис:

audio_emitter_get_gain(emitter);

Аргумент Описание
эмиттер Индекс эмиттера, который необходимо использовать.

Возвращается:

Реально

Пример:

if audio_emitter_get_gain(emitter_player) < 1
{
audio_emitter_gain(emitter_player, 1);
}

Приведенный выше код проверяет текущее усиление данного эмиттера, и если оно меньше 1, то устанавливается в 1.
