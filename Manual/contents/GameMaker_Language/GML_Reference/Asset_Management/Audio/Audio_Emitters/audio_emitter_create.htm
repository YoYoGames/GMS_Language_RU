audio_emitter_create

Эта функция создает новый аудиоэмиттер и возвращает его индекс. Этот индекс должен храниться в переменной для всех последующих функций, связанных с этим эмиттером, а затем, когда он больше не нужен, он должен быть удален из памяти с помощью функции функция audio_emitter_free() чтобы предотвратить утечку памяти, которая может привести к краху вашей игры.

Синтаксис:

audio_emitter_create();

Возвращается:

Целое число (идентификатор эмиттера)

Пример:

s_emit = audio_emitter_create();

Приведенный выше код создает новый аудиоэмиттер и присваивает его индекс переменной "s_emit".
