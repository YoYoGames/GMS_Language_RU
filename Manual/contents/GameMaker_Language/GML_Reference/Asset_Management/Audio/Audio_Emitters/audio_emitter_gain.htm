audio_emitter_gain

Эта функция устанавливает максимальное усиление (громкость) для звука. Воспринимаемая громкость звука может меняться в зависимости от величины спада и положения относительно слушателя, но при установке усиления с помощью этой функции полная громкость никогда не превысит заданного значения усиления. На изображении ниже показано, как усиление влияет на громкость излучателя, когда значение fall-off больше 0:

Эта функция изменит громкость звука во время его воспроизведения, а также всех последующих звуков, воспроизводимых через данный эмиттер. Обратите внимание, что на некоторых платформах вы можете иметь коэффициент усиления больше 1, хотя значение 1 считается "полной громкостью", а большее значение может привести к клиппированию или искажению звука.

ПРИМЕЧАНИЕ: на конечную громкость также будет влиять глобальное усиление звука, которое было установлено функцией audio_master_gain().

Синтаксис:

audio_emitter_gain(emitter, gain);

Аргумент Описание
эмиттер Индекс излучателя, который нужно изменить.
усиление Максимальное усиление (по умолчанию 1).

Возвращается:

Н/Д

Пример:

if up
{
gain += 0.05;
if gain >= 1 up = false;
}
else
{
gain += 0.05;
if gain <= 0 up = true;
}
audio_emitter_gain(s_emit, gain);

Приведенный выше код устанавливает переменную "gain" в различные значения, а затем использует эту же переменную для установки усиления эмиттера, индексированного в переменной "s_emit".
