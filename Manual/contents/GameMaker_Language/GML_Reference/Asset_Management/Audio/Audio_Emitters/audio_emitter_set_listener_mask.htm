audio_emitter_set_listener_mask

Эта функция может быть использована для установки битовой маски для эмиттера, чтобы все звуки, воспроизводимые через эмиттер, воспроизводились только от указанных слушателей. Вы можете создать битовую маску с помощью параметров audio_get_listener_info() и затем с помощью побитового или ("|") создать пользовательскую маску с теми слушателями, от которых требуется воспроизведение звука, а затем применить эту маску к эмиттеру. Эта маска отменяет маску по умолчанию или той, которую вы задали с помощью функции audio_set_listener_mask().

Синтаксис:

audio_emitter_set_listener_mask(emitterID, mask);

Аргумент Описание
emitterID Уникальный идентификатор излучателя для установки маски
маска Данные битовой маски, которые необходимо установить для звука

Возвращается:

Н/Д

Пример:

s_emit = audio_emitter_create();
audio_emitter_set_listener_mask(snd, global.PlayerMask);

Приведенный выше код создает эмиттер звука, а затем устанавливает слушателя (слушателей) для воспроизведения, используя данные маски, хранящиеся в глобальной переменной.
