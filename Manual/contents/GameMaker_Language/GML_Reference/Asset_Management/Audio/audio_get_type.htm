audio_get_type

При добавлении звука в GameMaker Studio 2 он может быть либо потоковым, либо в памяти. Если вам нужно узнать, является ли данный индекс звука потоковым или нет, вы можете использовать эту функцию, которая вернет 1 для потокового звука, 0 для звука в памяти и -1, если произошла ошибка или индекс не указывает на правильный звуковой ресурс.

Синтаксис:

audio_get_type(index);

Аргумент Описание
индекс Индекс звука для проверки.

Возвращается:

Реально

Пример:

type = audio_get_type(snd_Music_1);

Приведенный выше код проверяет тип аудио, проиндексированный в переменной "snd_Music_1", и сохраняет возвращаемое значение в переменной "type".
