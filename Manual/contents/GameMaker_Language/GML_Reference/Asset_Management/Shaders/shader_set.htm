шейдер_set

С помощью этой функции вы можете установить цель рисования на заданный шейдер, и все дальнейшее рисование будет выполняться с его помощью. Завершить использование шейдера можно с помощью функции shader_reset().

ПРИМЕЧАНИЕ: Вы можете установить шейдер только в событии Draw Event экземпляра.

Синтаксис:

shader_set(shader);

Аргумент Описание
шейдер Индекс шейдера, который нужно использовать.

Возвращается:

Н/Д

Пример:

shader_set(shader_Glass);
draw_self();
shader_reset();

Приведенный выше код установит вершину шейдера, используемую для отрисовки, а затем отрисует текущий спрайт, используемый для экземпляра с помощью этого шейдера.
