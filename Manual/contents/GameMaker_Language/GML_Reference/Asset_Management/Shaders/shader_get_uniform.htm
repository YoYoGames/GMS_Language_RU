shader_get_uniform

Поскольку значение шейдерной константы нельзя изменить в самом шейдере, необходимо задать его перед вызовом шейдера с помощью одной из доступных функций uniform set. Однако для этого необходимо сначала вызвать эту функцию чтобы получить "хэндл" константы шейдера, которую вы хотите изменить.

ПРИМЕЧАНИЕ: Хотя шейдер состоит из двух отдельных программ (вершинной и фрагментной), эта функция не делает различий между ними и возвращает хэндл константы шейдера любой из них.

Синтаксис:

shader_get_uniform(shader, uniform);

Аргумент Описание
шейдер Индекс шейдера, который нужно использовать.
униформа Константа шейдера, к которой нужно получить управление (строка).

Возвращается:

Real (Единая ручка)

Пример:

shader_params= shader_get_uniform(shd_glass, "u_vRefractColour");

Приведенный выше код получит хэндл константы шейдера "u_vRefractColour".
