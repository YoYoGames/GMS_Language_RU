shader_enable_corner_id

С помощью этой функции вы можете установить глобальное состояние для всех используемых шейдеров, при включении которого шейдер "крадет" 2 бита из входных значений цвета. Первый - из младшего бита значения красного цвета, а второй - из младшего младшего бита значения синего цвета. Эти значения затем могут быть восстановлены в шейдере, чтобы определить, с какой вершиной вы имеете дело (т.е. с каким углом).

Синтаксис:

shader_enable_corner_id(enable);

Аргумент Описание
включить Включите(true) или отключите(false) эту функцию.

Возвращается:

Н/Д

Пример:

shader_enable_corner_id(true);

Приведенный выше код позволит использовать биты цвета для идентификатора угла для всех шейдеров.
