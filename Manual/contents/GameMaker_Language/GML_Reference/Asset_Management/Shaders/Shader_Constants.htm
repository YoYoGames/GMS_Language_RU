Константышейдера

Помимо шейдерных функций и констант, определенных в справочных страницах OpenGL ES Shading Language (GLSL ES), существует также ряд шейдерных констант, доступных только в GameMaker Studio 2.

Следующие константы матрицы дисплея могут быть использованы в качестве индексов массива при использовании константы массива шейдера gm_Matrices:

Постоянная Описание
MATRIX_VIEW Эта константа индекса массива хранит индекс текущей матрицы представления. Возвращаемый индекс будет использоваться как значение массива при вызове константы gm_Matrices в коде шейдера.
МАТРИЧНАЯ_ПРОЕКЦИЯ Эта константа индекса массива хранит индекс текущей матрицы проекции. Возвращаемый индекс будет использоваться как значение массива при вызове константы gm_Matrices в коде шейдера.
MATRIX_WORLD Эта константа индекса массива хранит индекс текущей матрицы мира. Это можно использовать для таких вещей, как освещение, если у вас есть информация о свете в мировом пространстве. Возвращаемый индекс будет использоваться как значение массива при вызове константы gm_Matrices в коде шейдера.
МАТРИЧНЫЙ_МИР_ПРОСМОТРА Эта константа индекса массива содержит индекс результата умножения матриц мира и вида. Это часто используется для таких вещей, как туман. Возвращаемый индекс будет использоваться как значение массива при вызове константы gm_Matrices в коде шейдера.
МАТРИЧНАЯ_ПРОЕКЦИЯ_МИРОВОГО_ВИДА Эта индексная константа массива содержит индекс результата перемножения матриц мира, вида и проекции. Это обычная матрица преобразования, используемая для позиций вершин. Возвращаемый индекс будет использоваться как значение массива при вызове константы gm_Matrices в коде шейдера.
MATRIX_MAX Это не индекс массива, а скорее возвращает размер массива матрицы в вершинном шейдере.
MAX_VS_LIGHTS Это не индекс массива, а скорее возвращает количество огней в вершинном шейдере.

Следующая константа также доступна для освещения:

Постоянная Описание
MAX_VS_LIGHTS Максимальное количество точечных и направленных огней, доступных в шейдере

Следующие предопределенные матричные формы и константы могут быть использованы в вашем шейдере для доступа к специфическим значениям GameMaker Studio 2:

Постоянная Описание
gm_Matrices[matrix] Эта константа массива возвращает матрицу преобразования и является одной из доступных предопределенных форм, которые GameMaker Studio 2 создает для использования в редакторе кода шейдеров. Индекс массива выбирается из одной из перечисленных выше констант.
gm_BaseTexture Это константа 2D-сэмплера, которая возвращает текстуру текущего объекта, как установлено в GameMaker Studio 2. Таким образом, это будет текущий спрайт, поверхность или текстура, которая обычно используется при рисовании без вызова шейдера.
gm_LightingEnabled С его помощью можно получить или установить освещение GameMaker Studio 2 при использовании 3D.
gm_FogStart Это можно использовать для получения расстояния, на котором полигоны начинают смешиваться с цветом тумана.
gm_RcpFogRange Это можно использовать для получения расстояния, на котором туман максимален и уже ничего не видно.
gm_PS_FogEnabled Возвращает true или false, если на GPU включен пиксельный туман или нет.
gm_FogColour С его помощью можно получить цвет тумана, используемый в GameMaker Studio 2.
gm_VS_FogEnabled Возвращает true или false, если на GPU включен вершинный туман или нет.
gm_AlphaTestEnabled Это можно использовать для получения альфа-тестирования в шейдере. См. gpu_set_alphatestenable() более подробную информацию об альфа-тестировании.
gm_AlphaRefValue Это можно использовать для получения текущего опорного значения альфа-тестирования. См. gpu_set_alphatestref() дополнительную информацию об эталоне альфа-тестирования.
