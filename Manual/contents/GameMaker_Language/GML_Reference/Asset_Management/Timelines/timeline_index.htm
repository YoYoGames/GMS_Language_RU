временной_индекс

Эта переменная содержит индекс временной линии, связанной в данный момент с экземпляром. Вы можете установить его на определенную временную линию, чтобы использовать ее, или установить его на -1, чтобы прекратить использование временной линии для данного экземпляра (если для экземпляра не определена временная линия, возвращается значение -1). Обратите внимание, что это не запускает временную линию - для этого используйте переменную временная_линия.

Синтаксис:

временной_индекс;

Возвращается:

Real (целое число - значение индекса ресурса)

Пример:

if timeline_exists(global.tl)
{
timeline_index = global.tl;
}

Приведенный выше код присвоит экземпляру, выполняющему код, временную линию, индексированную в переменной "global.tl", если такая временная линия существует.
