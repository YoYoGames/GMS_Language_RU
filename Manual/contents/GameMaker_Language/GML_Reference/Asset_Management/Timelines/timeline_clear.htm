временная_чистота

С помощью этой функции вы можете очистить определенную временную линию от "моментов", удалив все коды и действия для этой временной линии и оставив ее пустой.

Синтаксис:

timeline_clear(ind);

Аргумент Описание
инд Индекс временной линии, которую нужно очистить.

Возвращается:

Н/Д

Пример:

if timeline_position > 200
{
timeline_clear(global.tl);
timeline_index = -1;
}

Приведенный выше код очистит указанную временную линию, индексируемую переменной "global.tl", от всех моментов, когда был пройден определенный момент.
