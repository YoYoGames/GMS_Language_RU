размер временной шкалы

С помощью этой функции вы можете получить общее количество активных моментов для временной шкалы (активный момент - это момент, в который добавленкод или DnD™). Это может быть удобно при создании динамических временных шкал с помощью функций timeline_moment_add_script() и timeline_moment_clear() можно проверить, есть ли в данной временной шкале нужное количество активных моментов или их нет вообще.

Синтаксис:

timeline_size(ind);

Аргумент Описание
инд Индекс временной шкалы, из которой берется информация.

Возвращается:

Н/Д

Пример:

if timeline_size(global.tl) == 0
{
timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), spawn_enemy);
}

Приведенный выше код проверяет заданный размер временной шкалы, и если он возвращает 0 (т.е. временная шкала не имеет активных моментов), то добавляет функцию скрипта, которая будет использоваться в случайный момент временной шкалы, проиндексированный в глобальной переменной "tl".
