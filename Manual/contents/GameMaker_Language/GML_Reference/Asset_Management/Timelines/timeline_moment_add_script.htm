таймлайн_момент_добавить_скрипт

С помощью этой функции вы можете динамически добавлять скрипт-функцию к временным линиям в любой заданный "момент" в пределах этой временной линии, где "момент" эквивалентен одному игровому тику (или шаг). Таким образом, вы можете создать новую временную линию с помощью функции timeline_add() и добавлять различные модели поведения в любой момент, или просто модифицировать ранее созданный ресурс таймлайна с помощью новых моделей поведения. Обратите внимание что функция не может требовать дополнительных аргументов при использовании, и если вы используете ее для изменения актива Timeline, присутствующего в Asset Browser, то все экземпляры, использующие эту временную линию, будут затронуты, и изменение продлится до конца игры (вызов game_restart() также не сбросит изменения).

Синтаксис:

timeline_moment_add_script(ind, step, script);

Аргумент Описание
инд Указатель линии времени для добавления момента.
шаг Момент (шаг) для добавления.
скрипт Индекс функции сценария для добавления в момент.

Возвращается:

Н/Д

Пример:

global.tl = timeline_add();
var i = room_speed * 60;
repeat(3)
{
timeline_moment_add_script(global.tl, i, choose(Attack_1, Attack_2, Attack_3);
i += room_speed * 60;
}

Приведенный выше код создаст новую временную линию и сохранит ее индекс в переменной "global.tl". Затем он добавит на линию времени три сценария, выбранные случайным образом с интервалом в одну минуту.
