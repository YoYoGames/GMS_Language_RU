экземпляр_последовательности

Это встроенная переменная, которая является частью переменных экземпляра, созданных для каждого экземпляра объекта в вашей игре. Если экземпляр управляется последовательностью, эта переменная будет содержать структуру экземпляра последовательности для последовательности, управляющей экземпляром, в противном случае она будет неопределена. Это переменная только для чтения и не может быть изменена.

Обратите внимание, что эта переменная станет неопределенной после завершения управляющей Последовательности, даже если ее элемент Последовательности все еще существует, и снова получит структуру, если этот элемент Последовательности будет воспроизведен.

Синтаксис:

экземпляр_последовательности

Возвращается:

Булево

Пример:

if (in_sequence)
{
sequence_instance.speedScale = 2;
}

Приведенный выше код проверяет in_sequence переменную, и если она равна true (то есть экземпляр управляется последовательностью), то он изменит шкалу скорости этой последовательности на 2.
