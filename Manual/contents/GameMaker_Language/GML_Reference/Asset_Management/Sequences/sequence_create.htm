последовательность_создать

С помощью этой функции вы можете создать новый объект последовательности, в который затем можно добавить дорожки. Функция возвращает struct, к которому можно обратиться для настройки созданной последовательности. Содержимое этой структуры подробно описано на этой странице. Структура объекта последовательности может быть использована для создания экземпляров последовательности на слое помещения с помощью функции layer_sequence_create(). Обратите внимание, что при создании объектов последовательности таким способом их следует удалить, вызвав функцию последовательность_уничтожить() когда они больше не нужны, иначе произойдет утечка памяти, которая что может замедлить работу и в конечном итоге привести к краху вашей игры.

Синтаксис:

sequence_create();

Возвращается:

Структура

Пример:

myseq = sequence_create();
myseq.length = 120;
myseq.loopmode = seqplay_pingpong;

Приведенный выше код создает новый объект sequence struct и устанавливает его длину воспроизведения и режим цикла.
