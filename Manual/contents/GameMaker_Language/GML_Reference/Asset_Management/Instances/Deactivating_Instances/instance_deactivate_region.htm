экземпляр_деактивировать_регион

С помощью этой функции можно определить область внутри помещения для деактивации экземпляров, которые были ранее активированы. Эта область может быть помечена как "внутри" или "снаружи", как показано на следующем изображении:

На изображении выше видно, что экземпляр "яблоко" всегда неактивен, потому что, даже если сам спрайт не перекрывает регион, ограничивающая рамка перекрывает его. Таким образом, экземпляры считаются находящимися в указанном регионе, когда их ограничивающая рам ка перекрывает его, а состояние маски столкновений (точное или нет) не принимается во внимание. Обратите внимание, что деактивация не является мгновенной, и экземпляр, деактивированный таким образом, не будет считаться неактивным до окончания события, в котором была вызвана функция. вызвана.

ПРИМЕЧАНИЕ: Если вы деактивируете экземпляр при запуске комнаты (т.е. из кода создания комнаты, или из события создания экземпляра в комнате), все экземпляры, размещенные в комнате из редактора комнат, будут выполнять свое событие создания до деактивации.

ПРЕДУПРЕЖДЕНИЕ: Деактивация экземпляров, у которых включена физика, НЕ остановит взаимодействие их фикстур в рамках физической симуляции. Для этого вы должны установить их phy_active переменную в true или false при активации/деактивации экземпляров.

Синтаксис:

instance_deactivate_region(left, top, width, height, inside, notme);

Аргумент Описание
слева Координата x левой части прямоугольной области для деактивации.
топ Координата y вершины прямоугольной области для деактивации.
ширина Ширина области для деактивации.
высота Высота области, которую необходимо деактивировать.
внутри Деактивировать ли экземпляры внутри региона (true) или снаружи (false).
notme Исключать ли вызывающий экземпляр из деактивации (true) или нет (false).

Возвращается:

Н/Д

Пример:

instance_activate_all();
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_deactivate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false, false);

Приведенный выше код активирует все экземпляры, а затем деактивирует область внутри комнаты.
