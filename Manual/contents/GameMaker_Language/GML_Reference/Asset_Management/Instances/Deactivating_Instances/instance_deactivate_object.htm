экземпляр_деактивировать_объект

С помощью этой функции вы можете деактивировать один экземпляр или все экземпляры определенного объекта из всех тех, которые были активированы ранее. Обратите внимание, что если вы деактивировали экземпляр или объект, который был помечен как Persistent, то вам нужно будет снова активировать его с помощью функции instance_activate_object() перед сменой комнаты, иначе он не будет перенесен и будет отброшен. Обратите внимание, что деактивация не является мгновенной, и экземпляр, деактивированный таким образом, не будет считаться неактивным до окончания события, в котором была вызвана функция.

ПРИМЕЧАНИЕ: Если вы деактивируете экземпляр при запуске комнаты (т.е. из кода создания комнаты или из события создания экземпляра в комнате), то все экземпляры, размещенные в комнате из редактора комнат, будут выполнять свое событие создания до деактивации.

ПРЕДУПРЕЖДЕНИЕ: Деактивация экземпляров, у которых включена физика, НЕ остановит взаимодействие их фикстур в рамках физической симуляции. Для этого вы должны установить их phy_active переменную в true или false при активации/деактивации экземпляров.

Синтаксис:

instance_deactivate_object(obj);

Аргумент Описание
obj Объект или экземпляр для деактивации (также можно использовать ключевое слово all ).

Возвращается:

Н/Д

Пример:

instance_deactivate_object(obj_Enemy);
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Приведенный выше код деактивирует все экземпляры объекта "obj_Enemy", а затем активирует регион в комнате.
