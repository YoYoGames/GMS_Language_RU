экземпляр_деактивировать_слой

С помощью этой функции вы можете деактивировать все экземпляры, назначенные определенному слою. Необходимо указать ID слоя, который может быть либо именем слоя, записанным в редакторе кода (в виде строки), либо фактическим значением ID слоя, возвращаемым функцией layer_create(), и обратите внимание, что с помощью этой функции можно деактивировать только слои экземпляра. Обратите внимание, что если вы деактивировали слой, в котором есть экземпляры объектов. помечены как постоянные, то вам необходимо снова активировать слой с помощью функции instance_activate_layer() перед сменой комнаты, иначе все постоянные экземпляры на слое не будут не будут перенесены и будут отброшены. Обратите внимание, что деактивация не является мгновенной, и экземпляр, деактивированный таким образом, не будет считаться неактивным до окончания события, в котором была вызвана функция.

ПРИМЕЧАНИЕ: Если вы деактивируете экземпляр при запуске комнаты (т.е. из кода создания комнаты или из события создания экземпляра в комнате), то все экземпляры, размещенные в комнате из редактора комнат, будут выполнять свое событие создания до деактивации.

ПРЕДУПРЕЖДЕНИЕ: Деактивация экземпляров с включенной физикой НЕ остановит взаимодействие их светильников в рамках физической симуляции. Для этого вы должны установить их phy_active переменную в true или false при активации/деактивации экземпляров.

Синтаксис:

instance_deactivate_layer(obj);

Аргумент Описание
слой Строка (или значение) идентификатора слоя, который будет использоваться

Возвращается:

Н/Д

Пример:

instance_deactivate_layer("Enemy Layer");
var _vx = camera_get_view_x(view_camera[0]);
var _vy = camera_get_view_y(view_camera[0]);
var _vw = camera_get_view_width(view_camera[0]);
var _vh = camera_get_view_height(view_camera[0]);
instance_activate_region(_vx - 64, _vy - 64, _vw + 128, _vh + 128, false);

Приведенный выше код деактивирует все экземпляры, назначенные слою "Enemy_Layer", а затем активирует регион в комнате.
