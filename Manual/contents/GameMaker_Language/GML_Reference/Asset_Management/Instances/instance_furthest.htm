экземпляр_самый дальний

Эта функция проверяет все экземпляры заданного объекта, чтобы определить, какой из них находится дальше всего от заданной точки x/y. Все проверки будут проводиться от заданной позиции x/y до начала координат (позиция x/y) экземпляров указанного объекта. Если ни одного экземпляра объекта, функция вернет ключевое слово noone, но если экземпляры существуют, то она вернет значение id найденного экземпляра. Обратите внимание, что если экземпляр, выполняющий код, имеет тот же индекс объекта, что и проверяемый объект, то он будет включен в проверку (это включает проверку родительских объектов, если вызывающий экземпляр также является дочерним по отношению к родительскому).

Синтаксис:

instance_furthest(x, y, obj);

Аргумент Описание
x Позиция x для проверки экземпляров, находящихся далеко от.
y Позиция y для проверки наличия экземпляров, расположенных далеко от.
obj Объект для проверки экземпляров.

Возвращается:

Real (либо значение ID экземпляра, либо ключевое слово noone)

Пример:

var inst;
inst = instance_furthest(x, y, object_index);
if inst != id
{
draw_line(x, y, inst.x, inst.y);
}

Приведенный выше код найдет самый удаленный экземпляр, который имеет тот же индекс объекта, что и экземпляр, выполняющий код, и сохранит его id в переменной. Затем эта переменная проверяется на то, совпадает ли она с id вызывающего экземпляра, и, если это не так. нет, между двумя экземплярами проводится линия.
