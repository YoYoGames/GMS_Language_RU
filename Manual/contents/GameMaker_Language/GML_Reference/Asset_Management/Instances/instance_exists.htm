экземпляр_существует

Эта функция может быть использована двумя способами в зависимости от того, что вы хотите проверить. Вы можете указать ей object_index для проверки, в этом случае функция вернет true, если в текущей комнате существуют активные экземпляры указанного объекта, или вы можете передать ей id экземпляра, в этом случае функция вернет true, если этот конкретный экземпляр существует и активен в текущей комнате.

Обратите внимание, что эта функция не учитывает те экземпляры, которые были деактивированы с помощью функций деактивации экземпляров.

Синтаксис:

instance_exists(obj);

Аргумент Описание
obj Объект или экземпляр, существование которого необходимо проверить.

Возвращается:

Булево

Пример:

if !instance_exists(obj_Enemy)
{
score += 200;
room_goto(rm_hiscores);
}

Приведенный выше код проверяет, существуют ли экземпляры объекта "obj_Enemy", и если нет, то добавляет к переменной "score" и меняет комнату.
