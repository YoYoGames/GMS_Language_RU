instance_id_get

С помощью этой функции вы можете получить уникальное значение ID любого экземпляра из активного в данный момент списка экземпляров. Вы указываете индекс в списке экземпляров, по которому нужно получить ID, и функция возвращает значение для хранения в переменной.

Синтаксис:

instance_id_get(index);

Аргумент Описание
индекс Индекс в списке экземпляров от 0 до (количество экземпляров - 1).

Возвращается:

Значение идентификатора экземпляра (реальное)

Пример:

for (var i = 0; i < instance_count; ++i;)
{
var temp_id = instance_id_get(i);
with (temp_id)
{
speed += 0.1;
}
}

Приведенный выше код пройдется по всем экземплярам в комнате и добавит 0,1 к их скорости.
