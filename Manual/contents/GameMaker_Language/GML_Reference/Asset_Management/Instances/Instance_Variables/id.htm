id

Эта переменная, доступная только для чтения, содержит уникальный идентификационный номер экземпляра. Каждому экземпляру, который вы создаете - будь то с помощью кода или путем добавления в комнату в редакторе комнат - присваивается номер, который используется для внутренней идентификации этого экземпляра. и переменная id - это то, что вы можете использовать для ссылки на него. Номер id также возвращается (и может быть сохранен в переменной) при создании экземпляра с помощью функции instance_create_layer() или instance_create_depth() , а также в других функциях экземпляра.

ПРИМЕЧАНИЕ: Значение этой переменной не совпадает с идентификатором, который присваивается экземпляру в редакторе помещений, а также отличается от значения переменной идентификатор экземпляра массив который содержит все идентификаторы всех активных в данный момент экземпляров.

Синтаксис:

id;

Возвращается:

Real (целое число - значение идентификатора экземпляра)

Пример:

for(var i = 0; i < instance_count; i++;)
{
if instance_id[i] != id
{
instance_id[i].scr += 5;
}
}

Приведенный выше код добавляет 5 к переменной "scr" для каждого активного экземпляра в комнате, кроме того, который выполняет код. Это происходит путем перебора ВСЕХ активных экземпляров (используя массив "instance_id" для возврата ID каждого активного экземпляра) и сравнения их со встроенной переменной "id ID) и сравнивая их со встроенной переменной "id", которая является ID исходного экземпляра, выполняющего код.
