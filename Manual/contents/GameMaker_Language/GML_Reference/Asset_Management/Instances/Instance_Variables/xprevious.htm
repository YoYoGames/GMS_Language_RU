xprevious

Эта встроенная переменная возвращает предыдущую позицию x для данного экземпляра. Эта переменная будет установлена непосредственно перед началом события begin step, но она также может быть установлена через код в любое время, что означает, что вы можете дать ей собственное пользовательское значение если в этом возникнет необходимость.

Синтаксис:

xprevious;

Возвращается:

Real (значение с плавающей точкой одинарной точности)

Пример:

if x != xprevious || y != yprevious
{
moved = true;
}

Приведенный выше код проверяет, есть ли разница между предыдущими значениями xprevious и yprevious и текущими значениями x и y. Если есть, то переменная устанавливается в true.
