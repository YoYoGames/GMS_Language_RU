сигнализация

Этот одномерный массив используется для получения текущего значения для любых сигналов тревоги, которые могут быть у экземпляра, или может использоваться для установки этих сигналов. В каждый экземпляр объекта встроено двенадцать сигналов тревоги, и каждый из них имеет свое собственное событие, которое произойдет, когда эта переменная достигнет 0.

Сигналы тревоги отсчитываются в начале каждого шага, после события Begin Step, но до основного события Step, и до того, как для шага что-либо будет отображено. Следует отметить, что сигнал тревоги не завершается, когда он достигает 0 (хотя событие так как на следующем шаге он опустится до -1, поэтому если вам нужно остановить сигнал тревоги по какой-либо причине, вы должны установить значение его массива на -1 , а не на 0. Время тревоги рассчитывается в игровых шагах, при этом значение 30 равно 30 шагам, 60 - 60 шагов и т.д.

ПРИМЕЧАНИЕ: Сигнал тревоги, в котором нет никаких действий или кода, не будет запущен. Однако даже при наличии только комментария и отсутствии кода сигнал тревоги будет отсчитывать время.

Вы можете установить массив сигналов тревоги напрямую (и получить текущее значение сигнала тревоги также напрямую, используя массив), но при некоторых обстоятельствах это не самый подходящий метод, поэтому вы также можете использовать функцию alarm_set() для установить сигнал тревоги, и функцию alarm_get() для получения значения сигнала тревоги.

Синтаксис:

тревога[val];

Возвращается:

Целое число (-1, если сигнализация не работает)

Пример:

if canshoot
{
if keyboard_check_pressed(vk_space)
{
canshoot = false;
alarm[0] = room_speed;
instance_create_layer(x, y, "Bullets", obj_Bullet);
}
}

Приведенный выше код проверяет переменную, затем, если она истинна, проверяет нажатие клавиши, и если она истинна, то в конце устанавливает переменную в false, устанавливает тревогу (где она снова будет установлена в true) и создает экземпляр.
