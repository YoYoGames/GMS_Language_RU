видимый

Экземпляр можно пометить как видимый или невидимый, установив эту переменную в true (видимый) или false (невидимый). Это позволяет GameMaker Studio 2 пропустить событие рисования для данного экземпляра, поэтому при использовании этой переменной следует соблюдать осторожность. так как это означает, что никакой код, помещенный в событие рисования, не будет выполняться, пока экземпляр не виден. Также обратите внимание, что если слой, которому назначен данный экземпляр, помечен как невидимый, установка этой переменной в значение true не будет иметь никакого эффекта, пока сам слой не будет помечен как видимый.

Синтаксис:

видимым;

Возвращается:

Булево

Пример:

if other.visible = true
{
x = xprevious;
y = yprevious;
}

Приведенный выше код проверит флаг видимости экземпляра "другой" в событии столкновения и, если он установлен в true, переместит этот экземпляр в его предыдущую позицию.
