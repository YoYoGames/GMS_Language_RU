Переменные экземпляра

Когда вы создаете новый объект, он будет поставляться с определенными переменными, уже инициализированными значениями по умолчанию. Эти переменные мы называем переменными экземпляра, поскольку они будут создаваться для каждого экземпляра объекта, который вы помещаете в игровые комнаты, и после создания экземпляра значения этих переменных уникальны для этого и только этого экземпляра. Некоторые из этих переменных вы будете использовать часто, например x и y, в то время как другие нужны меньше, в зависимости от того, что вы хотите сделать. но в целом они очень полезны, и по возможности вы должны использовать их, а не создавать свои собственные переменные.

Большинство переменных экземпляра можно как устанавливать, так и считывать, что позволяет вам изменять свойства и поведение экземпляра, просто изменяя значение определенной переменной - например, вы можете запретить экземпляру рисовать, просто установив переменную visible встроенной переменной false. Ниже приведены различные переменные, которые инициализируются для всех экземпляров всех объектов в вашей игре.

ВАЖНО! Если экземпляр находится в последовательности, то некоторые из этих переменных - например, image_xscale / image_yscale / image_angle / x / y - будут перезаписаны, когда последовательность обновляет каждый шаг после начала воспроизведения.

Эти переменные относятся к общим свойствам экземпляра:

  • id
  • видимый
  • твердый
  • постоянный
  • глубина
  • слой
  • сигнализация

Эти переменные связаны с положением и перемещением экземпляра:

  • направление
  • трение
  • гравитация
  • направление силы тяжести
  • hspeed
  • vspeed
  • скорость
  • xstart
  • ystart
  • x
  • y
  • xprevious
  • yprevious

Свойства объекта

Следующая переменная содержит уникальный индексный идентификатор объекта, на основе которого был создан экземпляр:

  • объектный_индекс

Все эти переменные связаны со спрайтом, назначенным экземпляру, и могут быть использованы для изменения того, что и как рисуется:

  • sprite_index
  • ширина спрайта
  • высота спрайта
  • sprite_xoffset
  • sprite_yoffset
  • изображение_альфа
  • угол изображения
  • смешивание изображений
  • индекс_изображения
  • номер_изображения
  • скорость изображения
  • масштаб_изображения
  • масштаб_изображения

Эти переменные относятся к маске столкновений:

  • индекс_маски
  • bbox_bottom
  • bbox_left
  • bbox_right
  • bbox_top

Эти переменные связаны с путями и тем, как экземпляр взаимодействует с одним из них, если он назначен:

  • индекс_пути
  • позиция_пути
  • путь_позиция_предыдущий
  • скорость движения
  • масштаб_пути
  • ориентация_пути
  • путь_действия

Эти переменные предназначены для настройки экземпляра на использование временной шкалы:

  • временной_индекс
  • хронология_пробега
  • временная_скорость
  • временная_позиция
  • временная_петля

Последовательности

Следующие переменные хранят информацию о Последовательности, которая может управлять экземпляром в любой момент времени:

  • в_последовательности
  • экземпляр_последовательности

Существует большое количество встроенных переменных для использования с физическими функциями GameMaker Studio 2, и чтобы было понятнее, их можно найти в разделе руководства, который охватывает все, что связано с моделированием физики:

  • Физика
