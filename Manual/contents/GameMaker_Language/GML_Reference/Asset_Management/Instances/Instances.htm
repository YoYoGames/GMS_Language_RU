Экземпляры

Экземпляры создаются из объектов, и после создания их можно изменять и манипулировать ими с помощью кода GML и действий DnD™. Основные функции, связанные с экземплярами, которые доступны и наиболее часто используемыми являются:

  • экземпляр_создания_слоя
  • глубина_создания_экземпляра
  • уничтожение экземпляра
  • экземпляр_существует
  • экземпляр_изменения
  • экземпляр_копии
  • экземпляр_найти
  • экземпляр_самый дальний
  • экземпляр_ближайший
  • номер экземпляра
  • место_объекта
  • список_мест_экземпляров
  • позиция экземпляра
  • список_позиций_экземпляров

Обратите внимание, что есть две дополнительные встроенные переменные (это переменные глобальной области видимости ) и одна функция, связанная с использованием экземпляров, которая может быть полезна для получения определенных деталей:

  • instance_id_get
  • идентификатор экземпляра
  • счётчик экземпляров

Существует ряд встроенных переменных, доступных в каждом экземпляре объекта, которые можно использовать для управления различными аспектами отображения экземпляра, использования других активов, реакции на столкновения и т.д... все они перечислены здесь здесь:

  • Переменные экземпляра

У экземпляров есть переменная массива, которую можно использовать для получения или установки значений тревожного события, но есть также две функции, которые можно использовать для того же самого:

  • alarm_get
  • установка_тревоги

Наконец, у вас также есть возможность деактивировать экземпляры, что означает, что они не будут обрабатывать никаких своих событий, хотя они все еще будут существовать в игре. Функции, связанные с этим, доступны на следующей страница:

  • Деактивация экземпляров
