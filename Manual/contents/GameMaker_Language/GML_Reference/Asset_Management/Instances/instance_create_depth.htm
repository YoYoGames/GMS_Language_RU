глубина_создания_экземпляра

С помощью этой функции вы можете создать новый экземпляр указанного объекта в любой заданной точке комнаты и на указанной глубине. Глубина может быть любым значением, где чем меньше глубина, тем "ближе" к камере будут нарисованы объекты, а чем больше глубина, тем дальше. чем больше глубина, тем дальше, поэтому экземпляр с глубиной -200 будет нарисован поверх экземпляра с глубиной +300 (например).

Обратите внимание, что эта функция фактически создаст слой комнаты для экземпляра, поскольку все экземпляры должны находиться на слое в комнате, но поскольку это управляемый слой (т.е. не тот, который вы создали с помощью кода или в комнате, а тот, который GameMaker Studio 2 создала его автоматически), вы не можете получить к нему доступ, и слой переменная экземпляра вернет значение -1.

ВАЖНО! Существует минимальная и максимальная глубина слоя от -16000 до 16000. Все, что размещено на глубине вне этого диапазона, не будет нарисовано, хотя все события будут проходить в обычном режиме.

Эта функция возвращает id нового экземпляра, который затем может быть сохранен в переменной или использован для доступа к этому экземпляру. Обратите внимание, что эта функция также вызовет событие Create Event создаваемого экземпляра, прежде чем продолжить код или действия для события, вызвавшего функцию.

Синтаксис:

instance_create_depth(x, y, depth, obj);

Аргумент Описание
x Позиция x, в которой будет создан объект
y Позиция y, в которой будет создан объект
глубина Глубина, на которую следует назначить созданный экземпляр
obj Индекс объекта для создания экземпляра объекта

Возвращается:

Real (значение идентификатора экземпляра)

Пример:

var inst = instance_create_depth(x, y, -10000, obj_Bullet);
with (inst)
{
speed = other.shoot_speed;
direction = other.image_angle;
}

Приведенный выше код создает новый экземпляр объекта "obj_Bullet" и сохраняет идентификатор экземпляра в переменной. Эта переменная затем используется для присвоения скорости и направления новому экземпляру.
