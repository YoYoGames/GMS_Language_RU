font_exists

Эта функция возвращает, существует ли шрифт с указанным индексом или нет. Вы можете проверить индексы шрифтов, определенные в браузере активов, или шрифты, которые были добавлены с помощью функций типа font_add().

Синтаксис:

font_exists(ind);

Аргумент Описание
инд Индекс шрифта для проверки.

Возвращается:

Булево

Пример:

if font_exists(fnt_Main)
{
draw_set_font(fnt_Main);
}

При этом активный шрифт для рисования будет установлен на fnt_Main, если он существует.

Назад: Шрифты
