размер_текстуры_страницы

Эта встроенная переменная может быть использована для получения или установки размера страницы текстуры при использовании функции font_add(). При добавлении шрифта с помощью этой функции GameMaker Studio 2 будет создаст кэш текстурной страницы с необходимыми глифами до размера, определенного этой переменной. Процесс происходит следующим образом:

  • Каждый используемый символ кэшируется из шрифта в текстурную страницу.
  • Когда страница текстур переполнена (потому что вы используете больших персонажей или много разных персонажей), GameMaker Studio 2 удалит персонажей из кэша и заменит их новыми, которые вам нужны
  • GameMaker Studio 2 старается не делать текстуру слишком большой, поскольку не все устройства могут справиться с большими текстурами, и старается не делать текстуру слишком маленькой, поскольку удаление и повторное добавление символов в текстуру занимает процессорное время и это приводит к медленной работе игры.

Если вы считаете, что вам требуется больший или меньший размер страницы текстуры, чем 1024px, используемый по умолчанию в GameMaker Studio 2, вы можете установить значение с помощью этой встроенной переменной. Если вы не используете функцию font_add(), то установка этого параметра не повлияет на производительность вашей игры, так как шрифты, добавленные в IDE, кэшируются на обычных текстурных страницах (что можно контролировать в редакторе шрифтов и опциях игры для целевой платформы). Опции игры для целевой платформы).

Синтаксис:

font_texture_page_size;

Возвращается:

Real (максимальная ширина/высота в пикселях)

Пример:

font_texture_page_size = 512;
newfont = font_add("Arial", 24, true, true, 32, 128);

Приведенный выше код устанавливает для страницы текстуры кэша шрифтов максимальный размер 512px, а затем добавляет новый шрифт в игру.
