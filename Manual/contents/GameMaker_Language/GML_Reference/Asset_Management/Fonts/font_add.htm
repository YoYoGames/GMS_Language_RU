font_add

Эта функция может быть использована для добавления шрифта в вашу игру из тех шрифтов, которые установлены в системе, на которой она запущена. Вы можете определить размер шрифта (в пунктах), а также то, должен ли шрифт быть жирным или курсивным, и вы можете также определить диапазон символов, которые необходимо включить. Функция возвращает значение индекса, которое должно быть сохранено в переменной, так как оно будет необходимо во всех дальнейших кодах, обращающихся к этому шрифту, или возвращает -1, если функция не работает по какой-либо причине. причине.

Для целевого модуля HTML5 эта функция может использоваться только для добавления веб-шрифтов и будет работать, только если WebGL не включен - если WebGL включен, вы должны использовать общий ресурс шрифта (т.е. использовать draw_set_font() вместе со шрифтом из Asset Browser) или растровые шрифты (см. font_add_sprite()).

ВАЖНО! При работе с HTML5 курсив и полужирный текст поддерживаются только в том случае, если актив шрифта или сам веб-шрифт содержит эти символы. Они не будут синтезированы из базового шрифта.

Любой другой шрифт может существовать или не существовать на компьютере или устройстве, на котором запущена ваша игра, поэтому использование этой функции должно быть ограничено этими стандартными шрифтами:

Для всех других (не веб-) целей можно использовать эту функцию для добавления шрифта из файла. Файл должен быть включен в пакет игры с помощью функции Included Files программы GameMaker Studio 2, и это должен быть файл шрифта формата *.ttf или *.otf, что удобно для добавления нестандартных шрифтов, например, азиатских или арабских.

ВНИМАНИЕ!!! Если вы включаете такой файл шрифта в игру, он должен быть лицензирован для распространения вместе с игрой.

При загрузке шрифта из файла таким способом размер шрифта указывается в пикселях, а первое и последнее значения игнорируются, то есть все глифы шрифта будут добавлены. Шрифты, добавленные таким образом, назначаются своей собственной текстуре странице, поэтому при использовании этой функции следует соблюдать осторожность, так как она увеличивает количество смен текстур при рисовании. Также стоит отметить, что шрифты могут выглядеть немного больше при отрисовке, так как глифы могут иметь части, которые рисуются за пределами граничного поля, определенного для шрифта. Также следует помнить, что существует ограничение на 200 глифов, которые могут быть отрисованы в одном кадре для определенного шрифта.

По умолчанию в добавленных шрифтах включено сглаживание. Однако вы можете проверить состояние AA в любое время с помощью функции font_add_get_enable_aa() и вы можете включить/выключить AA с помощью функции font_add_enable_aa().

ПРИМЕЧАНИЕ: Когда вы загружаете шрифт в GameMaker Studio 2: Studio, вы должны не забыть удалить его снова (с помощью функции font_delete()), когда он больше не нужен, иначе в противном случае существует риск утечки памяти, что приведет к замедлению и, в конечном итоге, к краху вашей игры.

Синтаксис:

font_add(name, size, bold, italic, first, last);

Аргумент Описание
имя Имя шрифта, который будет добавлен (например, 'Arial'), или путь к файлу, если шрифт является включенным файлом *. ttf или *.otf.
размер Размер шрифта - пункты для веб-шрифтов, пиксели для файловых шрифтов.
полужирный Является ли шрифт полужирным (true) или нет (false).
курсив Является ли шрифт курсивным (true) или нет (false).
первый Первый символ для включения (если вы не уверены, выбирайте 32).
последний Последний символ, который нужно включить (если вы не уверены, выбирайте 128).

Возвращается:

Реально

Пример:

newfont = font_add("Arial", 24, true, true, 32, 128);

Это создаст новый шрифт размером 24pt, использующий "Arial", который является полужирным и курсивным, а индекс для этого нового шрифта хранится в переменной newfont. Диапазон шрифтов включает заглавные и строчные буквы, цифры и все общепринятые знаки препинания. пунктуацию.
