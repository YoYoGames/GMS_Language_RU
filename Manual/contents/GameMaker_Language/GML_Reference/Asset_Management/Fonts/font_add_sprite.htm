font_add_sprite

С помощью этой функции вы можете использовать "спрайтовую полосу" для создания нового актива шрифта, где каждое подизображение будет отдельным символом или буквой. Индекс - это номер карты UTF8 для первого используемого символа, а диапазоны для использования такие же, как и для обычного шрифта. как для обычного шрифта, начиная с "!", но помните, что вы должны включать только то, что вам нужно, и что вы должны использовать функцию ord() чтобы получить правильное значение UTF8 для начальной буквы. Созданный спрайтовый шрифт будет иметь подизображения, установленные на отдельные глифы шрифта, где "первый" глиф будет иметь image_index равный 0. Вы можете определить "пространство" как любой символ, например например, одна строка размером с пробел, и все пробелы в тексте будут отображаться с такой шириной (но выбранное изображение никогда не будет отображено), или, если вы не предоставили спрайт для пробела, то ширина самого широкого символа в шрифте спрайта будет использоваться вместо этого.

Вы также можете установить пропорциональное(true) или непропорциональное(false) расстояние между буквами шрифта. При пропорциональном шрифте расстояние между буквами определяется их фактической шириной (например, буква "i" занимает меньше места, чем буква "w", в то время как непропорциональный шрифт будет размещаться на основе ширины суб-изображения, как моноширинный шрифт. Наконец, вы можете определить пространство, которое нужно оставить между каждой буквой при написании, и это может быть любое целое значение, при этом 0 означает отсутствие пространства (буквы будут "соприкасаться", если шрифт пропорциональный). буквы будут "соприкасаться", если они пропорциональны). Функция возвращает значение индекса, которое должно быть сохранено в переменной, так как оно будет необходимо во всех дальнейших кодах, которые будут ссылаться на этот шрифт, или возвращает -1, если функция не работает по какой-либо причине. причине.

Также стоит отметить, что в карте символов не может быть пробелов, поэтому если вы, например, используете '.' и '0', но не '/', вам нужно либо включить '/' в любом случае, либо оставить пустое подизображение между двумя используемыми символами. двумя используемыми символами.

Сам спрайт может быть активом спрайта из браузера ассетов или загруженным из файла с помощью sprite_add для получения индекса загруженного спрайта (который затем используется в функции). Важно отметить, что для выравнивания шрифта функции (например draw_set_halign()) работают так, как ожидается, при использовании спрайтового шрифта, его начало должно быть установлено в левом верхнем углу. Если вы используете другие значения, то вам необходимо учитывать смещение начала координат при рисовании текста с использованием шрифта.

ПРИМЕЧАНИЕ: Когда вы загружаете шрифт в GameMaker Studio 2, вы должны не забыть удалить его снова (с помощью функции font_delete()), когда он больше не нужен, иначе существует риск утечки памяти, что приведет к замедлению и в конечном итоге к краху вашей игры.

Синтаксис:

font_add_sprite(spr, first, prop, sep);

Аргумент Описание
spr Спрайт, на основе которого будет добавлен шрифт.
первый Индекс первого доступного символа.
реквизит Установить пропорциональный шрифт или нет.
сеп Пробел, который нужно оставить между каждой буквой.

Возвращается:

Реально

Пример:

global.Font = font_add_sprite(spr_HUD_Font, ord("!"), true, 2);

Приведенный выше код создаст новый актив шрифта из актива спрайта и сохранит индекс нового шрифта в переменной "global.Font" для последующего использования.
