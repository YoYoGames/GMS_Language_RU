font_replace_sprite_ext

Эта функция заменит спрайтовый шрифт, который вы создали ранее с помощью функции font_add_sprite_ext(). Она использует "спрайтовую полосу", где каждое подизображение является отдельным символом или буквой, а изображения отображаются на буквы на основе аргумента "string_map" функции. Этот аргумент представляет собой строку, с помощью которой вы можете сообщить программе GameMaker Studio 2, в каком порядке расположены суб-изображения спрайтового шрифта, и она автоматически сопоставит их с буквами при написании текста. Так, например, если у вас есть string_map "AaBbCcDdEeFfGgHh", то ваш спрайтовый шрифт должен иметь суб-изображения, упорядоченные таким образом.

Вы также можете установить пропорциональное(true) или непропорциональное(false) расстояние между буквами шрифта. При пропорциональном шрифте расстояние между буквами определяется их фактической шириной (например, буква "i" занимает меньше места, чем буква "w", в то время как непропорциональный шрифт будет размещаться на основе ширины суб-изображения, как моноширинный шрифт. Наконец, вы можете определить пространство, которое нужно оставить между каждой буквой при написании, и это может быть любое целое значение, при этом 0 означает отсутствие пространства (буквы будут "соприкасаться", если шрифт пропорциональный). буквы будут "соприкасаться", если они пропорциональны). Функция возвращает значение индекса, которое следует сохранить в переменной, так как оно понадобится во всех дальнейших кодах, которые будут ссылаться на этот шрифт.

Если функция завершится неудачно, то она завершится молча, не заменив заданный шрифт спрайта.

ПРИМЕЧАНИЕ: Эта функция требует, чтобы вы ранее добавили шрифт спрайта с помощью функции font_add_sprite(), поскольку она заменит его и не может быть использована для шрифтов, которые включены как часть ресурсов игры.

Синтаксис:

font_replace_sprite_ext(font, spr, string_map, prop, sep);

Аргумент Описание
шрифт Шрифт для замены.
spr Спрайт, на основе которого будет добавлен шрифт.
карта строк Строка, из которой берется порядок суб-изображений спрайта.
реквизит Установить пропорциональный шрифт или нет.
сеп Пробел, который нужно оставить между каждой буквой.

Возвращается:

Н/Д

Пример:

font_replace_sprite_ext(global.Font, spr_CalcFont, "0123456789+-*/=", true, 2);

Приведенный выше код заменит актив спрайтового шрифта, который индексируется в переменной "global.Font", на спрайтовые изображения, индексируемые в переменной "spr_CalcFont".
