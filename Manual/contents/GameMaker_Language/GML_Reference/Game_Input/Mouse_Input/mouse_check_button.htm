кнопка_проверки_мыши

Эта функция возвращает true, если проверяемая кнопка мыши удерживается нажатой, и false, если не удерживается. Вы задаете кнопку мыши для проверки из одной из следующих констант:

Постоянная Описание
mb_left Левая кнопка мыши
mb_middle Средняя кнопка мыши (это может быть актуально не для всех целевых платформ)
mb_right Правая кнопка мыши
mb_side1* Боковая кнопка мыши 1
mb_side2* Боковая кнопка мыши 2
mb_any Любая из кнопок мыши
mb_none Отсутствие кнопки мыши


* ПРИМЕЧАНИЕ: Кнопки mb_side1 и mb_side2 доступны только в Windows, macOS, Ubuntu и HTML5.

Синтаксис:

mouse_check_button(numb);

Аргумент Описание
numb Какую константу кнопки мыши проверять.

Возвращается:

Булево

Пример:

if mouse_check_button(mb_left)
{
instance_create_layer(mouse_x, mouse_y, "Effects", obj_Star);
}

Приведенный выше код будет проверять наличие левой кнопки мыши и при каждом ее удержании будет создавать экземпляр объекта, индексированного в "obj_Star".
