Ввод с помощью мыши

Мышиный ввод принимается на всех платформах (на мобильных устройствах он принимается как однократное касание экрана - если вам нужно использовать мультитач, вы должны использовать специфические функции устройства) и имеет несколько констант которые используются для указания нажимаемых кнопок. Эти константы показаны в следующей таблице:

Постоянная Описание
mb_left Левая кнопка мыши
mb_middle Средняя кнопка мыши (это может быть актуально не для всех целевых платформ)
mb_right Правая кнопка мыши
mb_side1* Боковая кнопка мыши 1
mb_side2* Боковая кнопка мыши 2
mb_any Любая из кнопок мыши
mb_none Отсутствие кнопки мыши


* ПРИМЕЧАНИЕ: Кнопки mb_side1 и mb_side2 доступны только в Windows, macOS, Ubuntu и HTML5.

Для управления стандартными кнопками мыши существуют следующие функции:

  • кнопка_мыши
  • кнопка_проверки_мыши
  • mouse_check_button_pressed
  • mouse_check_button_released
  • очистка_мыши
  • кнопка_мыши
  • колесо_мыши_вверх
  • колесо_мыши_вниз
  • мышь_x
  • мышь_y

Обратите внимание, что существует также набор оконных функций, связанных с использованием мыши на объектах рабочего стола:

  • window_mouse_get_x
  • window_mouse_get_y
  • window_mouse_set
  • window_view_mouse_get_x
  • window_view_mouse_get_y
  • window_views_mouse_get_x
  • window_views_mouse_get_y
