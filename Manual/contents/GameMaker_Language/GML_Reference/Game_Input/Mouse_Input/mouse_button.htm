кнопка_мыши

Эта переменная, доступная только для чтения, возвращает кнопку мыши, которая в данный момент нажата (в данный момент, как в этом шаге), и может возвращать любую из специальных констант мыши, кроме mb_any.

Синтаксис:

кнопка_мыши;

Возвращается:

Постоянная

Пример:

if mouse_button = mb_left
{
x -= 1;
}

Это перемещает текущий экземпляр влево, если левая кнопка мыши нажата.
