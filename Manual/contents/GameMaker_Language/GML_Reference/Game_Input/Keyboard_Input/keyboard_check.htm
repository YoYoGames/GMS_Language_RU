keyboard_check

С помощью этой функции можно проверить, удерживается ли клавиша нажатой или нет. В отличие от keyboard_check_pressed() или keyboard_check_released() функции, которые срабатывают только один раз при нажатии или отпускании клавиши, эта функция срабатывает на каждом шаге, в течение которого клавиша удерживается нажатой. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице " Ввод с клавиатуры ".

Синтаксис:

keyboard_check(key);

Аргумент Описание
ключ Ключ для проверки состояния вниз.

Возвращается:

Boolean

Пример:

if keyboard_check(vk_left)
{
x -= 5;
}

Приведенный выше код будет проверять, нажата ли клавиша со стрелкой, и перемещать экземпляр на 5 пикселей влево каждый шаг, когда возвращается true.
