keyboard_set_numlock

С помощью этой функции можно включить или выключить блокировку номера клавиатуры (для включения установлено значение true, для выключения - false ).

ПРИМЕЧАНИЕ: Эта функциональность доступна только в целевых сборках Windows и не будет работать на любом другом устройстве.

Синтаксис:

keyboard_set_numlock(value);

Аргумент Описание
значение Установите значение true для "включено" и false для "выключено".

Возвращается:

Н/Д

Пример:

if keyboard_get_numlock()
{
keyboard_set_numlock(false);
}
else
{
keyboard_set_numlock(true);
}

Приведенный выше пример кода получит состояние клавиши Numberlock и, если она включена (true), то установит ее в положение off (false) и наоборот.
