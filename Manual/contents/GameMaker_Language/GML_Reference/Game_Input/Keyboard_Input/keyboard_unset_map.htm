keyboard_unset_map

С помощью этой функции вы можете очистить все повторно отображенные клавиши, чтобы они вернулись в состояние по умолчанию, т.е. все клавиши отобразились на себя.

Синтаксис:

keyboard_get_map(key);

Возвращается:

Н/Д

Пример:

if keyboard_check_pressed(vk_escape)
{
keyboard_unset_map();
}

Приведенный выше пример кода сбросит все сопоставленные клавиши к настройкам по умолчанию, если пользователь нажмет клавишу "escape".
