keyboard_get_map

Иногда вам может понадобиться получить ascii код для сопоставленного ключа (например, чтобы узнать, сопоставлен ли он уже), что и возвращает эта функция. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице ввода с клавиатуры.

Синтаксис:

keyboard_get_map(key);

Аргумент Описание
ключ Это ключ, из которого вы хотите получить сопоставленное значение

Возвращается:

Реально

Пример:

if keyboard_get_map(ord("A")) = ord("A")
{
keyboard_set_map(ord("A"), keyboard_lastkey);
}

Приведенный выше пример кода сначала проверит, была ли "A" сопоставлена с другой клавишей, и если нет, то сопоставит ее с последней клавишей, которую нажал пользователь.
