keyboard_check_pressed

С помощью этой функции можно проверить, была ли нажата клавиша или нет. В отличие от keyboard_check() эта функция выполняется только один раз при каждом нажатии клавиши, поэтому, чтобы она сработала снова, клавиша должна быть сначала отпущена, а затем снова нажата. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_* констант, перечисленных на главной странице " Ввод с клавиатуры ".

Синтаксис:

keyboard_check_pressed(key);

Аргумент Описание
ключ Клавиша для проверки состояния нажатия.

Возвращается:

Булево

Пример:

if keyboard_check_pressed(vk_anykey)
{
room_goto_next();
}

Приведенный выше код перейдет в следующую комнату, если игрок нажмет любую из клавиш клавиатуры (работает как запрос "Press Any Key to Continue").
