клавиатура_клавиша_нажатия

С помощью этой функции можно имитировать нажатие любой клавиши на клавиатуре. При ее использовании клавиша будет отмечена как нажатая до тех пор, пока не будет вызвана соответствующая функция отпускания (см. keyboard_key_release()). Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице " Ввод с клавиатуры ".

Обратите внимание, что после вызова этой функции, если клавиша физически нажата на клавиатуре, это событие нажатия не будет зарегистрировано снова, пока клавиша не будет физически отпущена (срабатывает событие отпускания и останавливает эту функцию), или была вызвана соответствующая функция освобождения клавиши, и клавиша была нажата снова.

Синтаксис:

keyboard_key_press(key);

Аргумент Описание
ключ Клавиша для имитации нажатия.

Возвращается:

Н/Д

Пример:

keyboard_key_press(vk_space);

Это будет имитировать нажатие пробела.

Назад: Ввод с клавиатуры
Далее: keyboard_key_release
