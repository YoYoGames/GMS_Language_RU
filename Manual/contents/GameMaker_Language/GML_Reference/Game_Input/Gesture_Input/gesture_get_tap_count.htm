gesture_get_tap_count

Эта функция используется для проверки того, включен или отключен подсчет касаний. Функция возвращает true, если она включена, и false в противном случае, и вы можете включить или отключить подсчет касаний с помощью функции gesture_tap_count(). Если функция включена, каждое событие касания будет иметь дополнительный элемент DS карты "tapcount", который будет регистрировать количество касаний, а если функция включена, это означает, что все события касания будут срабатывать, т.е. два касания будут срабатывать как при одинарном касании, так и при двойном. и двойное касание, при этом количество касаний при одиночном касании будет равно 1, а при двойном - 2. Значение счета переключений будет сброшено на 0 по истечении времени, установленного для обнаружения двойного переключения (см. функцию gesture_double_tap_time()). Если счетчик касаний отключен, то первое касание не будет зарегистрировано, пока не пройдет время двойного касания и не будет обнаружено второе касание. Обратите внимание, что по умолчанию эта функция включена.

Синтаксис:

gesture_get_tap_count();

Возвращается:

Bool

Пример:

if !gesture_get_tap_count()
{
getsure_tap_count(true);
}

Приведенный выше код проверяет, включен ли подсчет касаний, и если нет, то он включается.
