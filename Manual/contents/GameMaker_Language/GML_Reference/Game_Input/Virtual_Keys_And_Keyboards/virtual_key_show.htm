виртуальный_ключ_показывать

После того как вы создали виртуальную клавишу для своих устройств, вам может понадобиться отладка и проверка того, что она правильно расположена и работает должным образом. Для этого нужно вызвать эту функцию, используя индекс виртуального ключа, который вы хотите увидеть (предварительно созданный и сохраненный с помощью virtual_key_add()), которая нарисует цветной прямоугольник для отображения клавиши на экране. Как только вы будете удовлетворены, рекомендуется создать и поместить свою собственную графику в своей области.

ПРИМЕЧАНИЕ: На рисуемый прямоугольник будут влиять параметры смешивания, цвета и альфа, установленные соответствующими функциями рисования.

Синтаксис

virtual_key_show(index);

Аргумент Описание
индекс Индекс виртуального ключа, который нужно показать.

Возвращается:

Н/Д

Пример:

if global.Debug
{
virtual_key_show(global.Left);
}
else
{
virtual_key_hide(global.Left);
}

Приведенный выше код проверяет глобальную переменную "global.Debug" и если она истинна, то виртуальная клавиша, индексированная в переменной "global.Left", будет нарисована на экране, а если ложна, то клавиша будет скрыта.
