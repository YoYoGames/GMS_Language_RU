keyboard_virtual_show

Эта функция может быть использована для отображения виртуальной клавиатуры на устройстве, на котором запущена игра. При вызове этой функции необходимо предоставить одну из следующих констант для каждого из первых трех аргументов:

  • Тип клавиатуры: Эта константа используется для установки того, какой набор клавиш будет доступен на виртуальной клавиатуре. Доступны следующие типы клавиатуры:
Тип клавиатуры Описание
kbv_type_default Тип клавиатуры по умолчанию для текущей системы.
kbv_type_ascii Клавиатура только ASCII.
kbv_type_url Обычная клавиатура, оптимизированная для ввода URL-адресов. Обычно имеет клавишу ".com" или другие доменные клавиши, а также клавиши "/" и ".".
kbv_type_email Обычная клавиатура, оптимизированная для ввода электронной почты. Обычно содержит символы "@" и ".".
kbv_type_numbers Клавиатура, использующая только цифры, обычно отображается как цифровая панель.
kbv_type_phone Клавиатура телефонной панели. Обычно только для цифр с клавишами "*" и "#".
kbv_type_phone_name Клавиатура, оптимизированная для ввода как номера телефона, так и имени. Обычно похожа на клавиатуру ASCII, но с ограниченным выбором специальных символов.

  • Тип возврата: Эта константа используется для установки того, что будет отображаться на клавише возврата/действия виртуальной клавиатуры. Доступны следующие типы возврата:
Тип возврата Описание
kbv_returnkey_default Название клавиши возврата по умолчанию для текущей системы.
kbv_returnkey_go Устанавливает заголовок клавиши возврата на "Go".
kbv_returnkey_google Устанавливает заголовок клавиши возврата на "Google", или на общий значок поиска в некоторых случаях.
kbv_returnkey_join Устанавливает заголовок клавиши возврата на "Go".
kbv_returnkey_next Устанавливает заголовок клавиши возврата на "Далее".
kbv_returnkey_route Устанавливает заголовок клавиши возврата на "Маршрут".
kbv_returnkey_search Устанавливает заголовок клавиши возврата на "Поиск", или на общий значок поиска в некоторых случаях.
kbv_returnkey_send Устанавливает для заголовка клавиши возврата значение "Отправить".
kbv_returnkey_yahoo Устанавливает заголовок ключа возврата на "Yahoo", или на общий значок поиска в некоторых случаях.
kbv_returnkey_done Устанавливает для заголовка клавиши возврата значение "Готово".
kbv_returnkey_continue Устанавливает заголовок клавиши возврата на "Продолжить".
kbv_returnkey_emergency Устанавливает название клавиши возврата на "Экстренный вызов".

  • Тип автокапитализации: Эта константа используется для определения типа автокапитализации слов, набираемых с помощью виртуальной клавиатуры. Доступны следующие типы автокапитализации:
Тип автокапитализации Описание
kbv_autocapitalize_none Автокапитализация отключена.
kbv_autocapitalize_words Слова будут автозаглавными.
kbv_autocapitalize_sentences Предложения будут автозаглавными.
kbv_autocapitalize_characters Все символы будут написаны заглавными буквами.

Последним аргументом является включение/выключение предикативного текста, и он должен быть установлен в true, чтобы разрешить его, и false в противном случае, но обратите внимание, что то, что он разрешен, не означает, что он будет использоваться, так как это зависит от предпочтений пользователя устройства. В режиме предикативного текста виртуальная клавиатура не будет генерировать обычные события нажатия клавиш GameMaker Studio 2. Вместо этого она будет обновлять только последний нажатый символ и переменные строки клавиатуры. Это связано с тем, что невозможности определить, произошло ли изменение во внутреннем текстовом поле, используемом для обнаружения нажатия клавиш, от фактической клавиши виртуальной клавиатуры или от текстового предложения. В этих случаях вы захотите прочитать ввод строки_клавиатуры, а не чем считывать любой необработанный ввод клавиш.

Важно также отметить, что пользователь получит разные клавиатуры с разными возможностями в зависимости от ОС платформы, со следующими оговорками для использования по назначению:

  • UWP: Реализация UWP не поддерживает return_key_type, autocapitalization_type или predictive_text_enabled. Передача этих параметров в функцию не вызовет никаких ошибок, они просто не будут иметь никакого влияния на клавиатуру.

  • Android: На Android тип клавиатуры kbv_type_phone_name не поддерживается и вместо него отображается клавиатура ASCII, а тип клавиши возврата может быть только kbv_returnkey_go, kbv_returnkey_search, kbv_returnkey_next, kbv_returnkey_send или kbv_returnkey_done. Если на данной платформе используются другие типы клавиш возврата, система по умолчанию будет использовать заменяющие их клавиши возврата, перечисленные ниже:
Недоступная клавиша возврата Сменный ключ
kbv_returnkey_google
kbv_returnkey_yahoo 		kbv_returnkey_search
kbv_returnkey_join
kbv_returnkey_route
kbv_returnkey_emergency 		kbv_returnkey_go
kbv_returnkey_continue kbv_returnkey_next

  • AndroidTV/ FireTV: Пользовательские типы клавиш возврата не поддерживаются на клавиатурах ASCII - всегда будет отображаться клавиша возврата по умолчанию.

  • tvOS: Предикативный текст и автокапитализация не поддерживаются на tvOS, а высота клавиатуры, возвращаемая системными событиями и функцией keyboard_virtual_height(), будет по умолчанию равна высоте экрана, поскольку клавиатура занимает весь экран, а функции определения размеров прямоугольника клавиатуры отключены на tvOS. экрана и из-за того, что функции прямоугольных размеров клавиатуры отключены в tvOS. Также обратите внимание, что события физической (bluetooth) клавиатуры не будут транслироваться, пока не будет открыта виртуальная клавиатура.

Вызов этой функции сгенерирует системное асинхронное событие, в котором карта async_load DS будет заполнена следующими парами ключ/значение:

  • " event_type" - это будет "статус виртуальнойклавиатуры", когда срабатывают функции виртуальной клавиатуры.

  • "screen_height" - высота виртуальной клавиатуры (в пикселях). Это значение будет равно 0, если клавиатура невидима.

  • "keyboard_status" - текущее состояние клавиатуры, возвращается в виде одной из следующих строк:
    • "скрытие"
    • "скрытый"
    • "показывающий"
    • "видимый"

Синтаксис

keyboard_virtual_show(keyboard_type, return_key_type, autocapitalization_type, predictive_text_enabled);

Аргумент Описание
тип клавиатуры Определяет набор клавиш, доступный на виртуальной клавиатуре.
тип_ключа_возврата Определяет, что будет отображаться на клавише возврата/действия виртуальной клавиатуры.
тип_автокапитализации Определяет, как/если слова, набранные с помощью виртуальной клавиатуры, будут автокапитализированы.
предсказательный_текст_включен Установите значение true/false для включения/выключения предикативного ввода текста.

Возвращается:

Н/Д

Пример:

если input == false
{
input = true;
keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false);
}

Приведенный выше код вызовет виртуальную клавиатуру ОС, если заданная переменная не установлена в true.
