высота клавиатуры_виртуальная_высота

Эта функция возвращает текущую высоту в пикселях виртуальной клавиатуры, основываясь на размере дисплея. Если клавиатура не видна, будет возвращен 0.

Синтаксис

keyboard_virtual_height();

Возвращается:

Int

Пример:

if keyboard_virtual_status() == true
{
key_h = keyboard_virtual_height();
}

Приведенный выше код проверит состояние виртуальной клавиатуры ОС и, если она видна, установит переменную на высоту клавиатуры.
