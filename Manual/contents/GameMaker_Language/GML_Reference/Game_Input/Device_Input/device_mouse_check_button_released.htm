device_mouse_check_button_released

Эта функция возвращает true или false в зависимости от того, было ли освобождено указанное вами касание устройства или нет. Аргумент device указывает на номер касания устройства, который может быть от 0 до n, и обратите внимание, что максимальное число прикосновений зависит от используемого устройства и ОС (большинство устройств поддерживают не менее 4 одновременных прикосновений). Эта функция срабатывает только один раз при действии отпускания, и константы, перечисленные на этой странице, могут быть использованы для проверки кнопок мыши. Обратите внимание, что mb_right будет обнаружена только в том случае, если обнаружено двойное касание и удерживается на втором касании перед отпусканием (это поведение можно отключить с помощью функции device_mouse_dbclick_enable()).

Синтаксис:

device_mouse_check_button_released(device, button);

Аргумент Описание
устройство Устройство (от 0 до n), которое проверяется.
кнопка Кнопка устройства, которое проверяется.

Возвращается:

Булево

Пример:

if device_mouse_check_button_released(0, mb_left)
{
press=false;
}

Приведенный выше код проверяет, было ли нажато устройство, и устанавливает переменную в false, если было.

Назад: Ввод устройства
Далее: device_mouse_dbclick_enable
