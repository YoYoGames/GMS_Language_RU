device_mouse_check_button_pressed

Эта функция возвращает true или false в зависимости от того, было ли указанное вами устройство "затронуто" (нажато) или нет. Аргумент device указывает на номер касания, который может быть от 0 до n, а максимальное число максимальное количество касаний, которое может быть обнаружено, будет сильно зависеть от используемого устройства и ОС, под управлением которой оно работает (большинство устройств поддерживают не менее 4 одновременных касаний). Эта функция срабатывает только один раз при фактическом нажатии, а константы перечисленные на этой странице, могут быть использованы для проверки наличия кнопок мыши. Обратите внимание, что mb_right будет обнаружена только в случае двойного касания и удержания при втором касании (это поведение можно отключить с помощью функции device_mouse_dbclick_enable()).

Синтаксис:

device_mouse_check_button_pressed(device, button);

Аргумент Описание
устройство Устройство (от 0 до n), которое проверяется.
кнопка Кнопка устройства, которое проверяется.

Возвращается:

Булево

Пример:

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

Приведенный выше код проверяет, было ли нажато устройство, и устанавливает переменную в true, если да.

Назад: Ввод устройства
