gamepad_set_option

Эта функция может быть использована для установки любой из доступных опций геймпада. Вы указываете "слот" геймпада для установки опции, а также строку опции, которую нужно установить, и значение, которое нужно использовать. Доступная строка опций зависит от платформы, на которой выполняется проект. на которой выполняется проект, как указано ниже:

Вариант Платформа Описание Значение
"allow_rotation" tvOS Установите, будет ли пульт Siri Remote заблокирован на вращение (т.е. может использоваться только вертикально) или нет. При разблокировке (значение true) вращение пульта изменит ввод, например, поворот на 90º влево заставит "правую" сторона сенсорной панели вернется "вверх", и т.д... таким образом, весь ввод будет относиться к ориентации устройства. При блокировке (значение false) ввод не будет изменяться относительно положения контроллера и всегда будет обрабатываться так, как если бы контроллер находился в вертикальном положении. Булево
"dpad_absolute" tvOS Устанавливает, принимает ли сенсорная панель Siri Remote абсолютный или относительный ввод. В абсолютном режиме (значение true) центром сенсорной панели считается позиция (0,0) - т. е. место, где аналоговый стик находится в состоянии покоя или где находится центр и движение вокруг этого положения будет генерировать ввод. Если установлено относительное значение (значение false ), то позиция (0,0) будет рассматриваться как место, где было сделано первоначальное касание пэда, и все вводимые данные будут генерироваться из этой позиции, а не из позиции будут генерироваться из этой позиции, а не из центра пэда. Булево

Синтаксис:

gamepad_set_option(device, option_name, value);

Аргумент Описание
устройство Какой "слот" устройства геймпада установить.
имя_опции Имя опции, которую нужно установить (строка, см. таблицу выше)
значение Значение для установки опции (может быть булевым, вещественным, целочисленным или строковым - см. таблицу выше).

Возвращается:

Н/Д

Пример:

for(var i = 0; i < 12; ++i;)
{
if gamepad_is_connected(i) && gamepad_get_description(i) == " tvOS SiriRemote"
{
если gamepad_get_option(i, "allow_rotation") == false
{
gamepad_set_option(i, "allow_rotation", true);
}
}
}

Приведенный выше код перебирает все слоты геймпада и проверяет наличие "Siri Remote" на платформе tvOS . Если таковой обнаружен, он устанавливает пульт ДУ для разрешения вращения ввода.
