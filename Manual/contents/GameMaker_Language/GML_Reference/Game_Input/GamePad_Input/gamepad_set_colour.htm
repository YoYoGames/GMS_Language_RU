gamepad_set_colour

Эта функция может использоваться для установки цвета светодиодов в контроллере PlayStation. Вы указываете слот устройства для установки, а затем задаете цвет, который может быть любой из цветовых констант или значением цвета, созданным с помощью специальных цветовых функций, или HEX-значением (например, $FFFFFFF).

Синтаксис:

gamepad_set_colour(device, color);

Аргумент Описание
устройство Какой "слот" устройства геймпада установить.
цвет Цвет для использования.

Возвращается:

Н/Д

Пример:

если здоровье < 10
{
gamepad_set_colour(0, c_red);
}

Приведенный выше код установит красный цвет светодиодов контроллера PlayStation, если переменная здоровья упадет ниже 10.
