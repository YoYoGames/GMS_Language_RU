gamepad_get_device_count

Эта функция сообщит вам одно из двух. Либо количество подключенных геймпадов, либо количество доступных "слотов" для подключения геймпадов. Фактическое возвращаемое значение будет зависеть от платформы и внутренней конфигурации этой платформы, поэтому данную функцию следует использовать вместе с функцией gamepad_is_connected() чтобы быть уверенным в точном количестве подключенных пэдов в любой момент времени.

Синтаксис:

gamepad_get_device_count();

Возвращается:

Реально

Пример:

var gp_num = gamepad_get_device_count();
for (var i = 0; i < gp_num; i++;)
{
if gamepad_is_connected(i)
{
global.gp[i] = true;
}
else
{
global.gp[i] = false;
}
}

Приведенный выше код перебирает доступные геймпады (или слоты для геймпадов) и затем проверяет каждый из них на наличие подключенного геймпада. возвращаемое значение затем используется для установки глобального массива в true или false для использования в будущих проверках.
