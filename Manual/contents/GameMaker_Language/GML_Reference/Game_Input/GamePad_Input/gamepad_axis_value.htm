значение_оси_геймпада

Вы можете использовать эту функцию для получения значений различных осей для данного слота геймпада. Вы передаете индекс слота геймпада для проверки, а также либо константу индекса оси (как указано здесь), либо целочисленное значение между 0 и gamepad_axis_count()-1. Возвращаемое значение будет от -1 до 1 для каждой из доступных горизонтальных и вертикальных осей.

Обратите внимание, что возвращаемое значение нормализуется с учетом значения, установленного для мертвой зоны оси (с помощью функции gamepad_set_axis_deadzone()), и поэтому всегда будет находиться в диапазоне от -1 до 0 и от 0 до 1. Например, установка мертвой зоны на 0,2 означает, что нажатие стика вправо начнет возвращать значение от 0 до 1 только тогда, когда необработанное значение оси превысит 0,2 - так, например, когда необработанное значение равно 0,2, возвращаемое значение будет 0, когда необработанное значение равно 0,5, возвращаемое значение будет 0. возвращается значение 0, если необработанное значение равно 0,5, возвращается значение 0,375, или если необработанное значение равно 0,9, возвращается значение 0,875.

Синтаксис:

gamepad_axis_value(device, axisIndex);

Аргумент Описание
устройство Какой "слот" устройства геймпада нужно проверить.
axisIndex Индекс оси для проверки (см. список констант).

Возвращается:

настоящий

Пример:

var haxis = gamepad_axis_value(0, gp_axislh);
var vaxis = gamepad_axis_value(0, gp_axislv);
direction = point_direction(0, 0, haxis, vaxis);
speed = point_distance(0 ,0, haxis, vaxis) * 5;

Приведенный выше код устанавливает скорость и направление движения экземпляра на основе возвращаемых значений левого аналогового регулятора геймпада, подключенного к устройству "слот" 0.
