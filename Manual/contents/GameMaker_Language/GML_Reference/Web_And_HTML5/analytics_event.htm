аналитика_событие

Эта функция отправит указанный текст поставщику аналитики, которого вы настроили через опции HTML5 Game Options. Эта функция может быть использована для создания пользовательской аналитики для отслеживания чего-то вне рамок используемого провайдера.

Синтаксис:

analytics_event(string);

Аргумент Описание
строка Строка для отправки провайдеру

Возвращается:

Н/Д

Пример:

if hs_new
{
analytics_event("Новый hiscore от " + string (score));
}

Приведенный выше код проверит переменную на истинность, и если это так, то будет запущено специальное аналитическое событие с указанной строкой.

Назад: Web и HTML5
