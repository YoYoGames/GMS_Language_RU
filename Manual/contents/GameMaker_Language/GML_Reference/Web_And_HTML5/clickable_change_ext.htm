clickable_change_ext

С помощью этой функции вы можете изменить спрайт и позицию кликабельной иконки, созданной ранее с помощью функции clickable_add(). Имейте в виду, что позиция основана на позиции окна, а не холста (0,0) и что на спрайт нужно ссылаться непосредственно из страницы текстуры (см: sprite_get_tpe()). Эта функция также позволяет изменить масштаб используемого спрайта (в виде множителя, так что по умолчанию используется 1, 0,5 - половина, а 2 - удвоение) и альфа-значение от 0 (полностью прозрачный) до 1 (полностью непрозрачный) для конечного значка на экране.

Синтаксис:

clickable_change_ext(index, tpe, x, y, alpha, scale)

Аргумент Описание
индекс Индекс кликабельного значка для изменения.
tpe Запись страницы текстуры для используемого спрайта
x Новая позиция x в окне.
y Новая позиция y в окне.
шкала Масштаб значка (по умолчанию 1).
альфа Альфа изображения значка (по умолчанию 1).

Возвращается:

Н/Д

Пример:

switch (room)
{
case rm_Menu: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32, 2, 1); break;
case rm_Game: clickable_change_ext(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32, 1, 0.5); break;
}

Приведенный выше код изменит индекс изображения и положение кликабельной иконки, индексируемой в переменной "global.Help", в зависимости от текущей комнаты, изменяя при этом масштаб и альфа иконки.
