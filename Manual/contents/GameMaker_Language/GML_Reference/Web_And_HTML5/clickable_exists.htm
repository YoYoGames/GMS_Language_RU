clickable_exists

Эта функция возвращает, была ли создана кликабельная DOM-иконка с указанным индексом или нет. Обратите внимание, что значение, используемое для проверки, должно быть инициализировано ранее, иначе вы получите ошибку, приводящую к закрытию GameMaker Studio 2.

Синтаксис:

clickable_exists(index);

Аргумент Описание
индекс Индекс кликабельного объекта для проверки.

Возвращается:

Булево

Пример:

if !clickable_exists(home_but)
{
home_but = clickable_add(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0");
}

Приведенный выше код проверяет переменную "home_but" на предмет того, существует ли она уже, и если нет, то создает кликабельную DOM-иконку в позиции (32, 32) страницы, на которой запущен игровой холст. Иконка использует спрайт, на который ссылается текстуры страницы как "spr_MS_Home", и при нажатии на иконку открывается новое окно для указанного URL и с определенными свойствами для этого окна.
