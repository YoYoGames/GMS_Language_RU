кликабельное_изменение

С помощью этой функции вы можете изменить спрайт и положение кликабельной иконки, созданной ранее с помощью функции clickable_add(). Обратите внимание, что позиция основана на позиции окна, а не холста (0,0). и что спрайт должен быть указан непосредственно на странице текстуры (см: sprite_get_tpe()).

Синтаксис:

clickable_change(index, tpe, x, y)

Аргумент Описание
индекс Индекс кликабельного значка для изменения.
tpe Запись страницы текстуры для используемого спрайта.
x Новая позиция x в окне.
y Новая позиция y в окне.

Возвращается:

Н/Д

Пример:

switch (room)
{
case rm_Menu: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 1), 32, 32); break;
case rm_Game: clickable_change(global.Help_Icon, sprite_get_tpe(spr_MS_Help, 0), 200, 32); break;
}

Приведенный выше код будет изменять индекс изображения и положение кликабельной иконки, индексируемой в переменной "global.Help", в зависимости от текущей комнаты.
