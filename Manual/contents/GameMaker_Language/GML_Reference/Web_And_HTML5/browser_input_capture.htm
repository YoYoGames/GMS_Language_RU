browser_input_capture

С помощью этой функции вы можете установить, должно ли окно браузера захватывать весь ввод (установите значение false) или игра должна захватывать ввод (установите значение true). Обратите внимание, что эта функция предназначена только для использования с модулем HTML5.

Синтаксис:

browser_input_capture(enable)

Возвращается:

Н/Д

Пример:

browser_input_capture(true);

Приведенный выше код устанавливает игру на захват всего ввода браузера на целевой платформе HTML5.

Назад: Web и HTML5
Далее: webgl_enabled
