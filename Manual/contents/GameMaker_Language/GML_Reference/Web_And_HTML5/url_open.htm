url_open

Это откроет указанный URL в браузере выбранного целевого устройства или, если вы используете модуль HTML5, в текущем открытом браузере.

Синтаксис:

url_open(url);

Аргумент Описание
url URL (адрес веб-сайта) для ссылки.

Возвращается:

Н/Д

Пример:

url_open("http://yoyogames.com");

Это откроет домашнюю страницу YoYo Games в текущем окне.

Назад: Web и HTML5
