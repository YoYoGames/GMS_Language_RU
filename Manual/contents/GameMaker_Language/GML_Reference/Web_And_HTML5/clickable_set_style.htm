clickable_set_style

Эта функция позволяет установить свойства стиля CSS для данной кнопки с помощью пар ключ/значение в предоставленном файле Карта DS. Вам необходимо предварительно создать элемент кнопки (используя clickable_add()) и DS Map и предоставить сохраненные индексы для каждого из них в качестве аргументов.

Синтаксис:

clickable_set_style(index, map)

Аргумент Описание
индекс Индекс кликабельного значка для стилизации.
карта Индекс карты DS Map для установки стиля.

Возвращается:

Н/Д

Пример:

var map = ds_map_create();
ds_map_add(map, "opacity", "0.5");
clickable_set_style(button, map);
ds_map_destroy();

Приведенный выше код изменит стиль нажимаемой кнопки с индексом, хранящимся в переменной "button".
