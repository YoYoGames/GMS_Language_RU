буфер_sha1

В криптографии SHA-1 - это криптографическая функция хэширования, разработанная Агентством национальной безопасности США и используемая в нескольких широко используемых приложениях и протоколах, таких как популярный Git где она используется для проверки изменений файлов. Эта функция принимает входные данные из буфера и возвращает 160-битный дайджест сообщения в формате ASCII. Таким образом, вы можете сгенерировать безопасный ключ, который можно хранить и использовать для проверки целостности информации, отправляемой на внешний сервер (или получаемой от него) (например). При применении этой функции к буферам вы должны указать идентификатор буфера, затем значение смещения (в байтах) для начала, а затем размер (опять же в байтах) области для хеширования.

ПРИМЕЧАНИЕ: SHA-1 не является полностью безопасным и может быть взломан. См. эту страницу для получения дополнительной информации.

Синтаксис:

buffer_sha1(buffer, offset, size);

Аргумент Описание
буфер Индекс используемого буфера.
смещение Значение смещения данных.
размер Размер буфера.

Возвращается:

Строка

Пример:

check_string = buffer_sha1(buff, 0, buffer_get_size(buff));

Приведенный выше код создаст хэш sha1 для полных данных, хранящихся в буфере, индексируемом переменной "buff", и сохранит возвращенный хэш в переменной "check_string".
