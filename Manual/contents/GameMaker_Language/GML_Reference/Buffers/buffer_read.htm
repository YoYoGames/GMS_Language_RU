буферное_чтение

Эта функция может быть использована для чтения данных из ранее созданного буфера. Возвращаемое значение будет зависеть от типа считываемых данных, который сам по себе определяется следующими константами:

Постоянная Описание
буфер_u8 Беззнаковое 8-разрядное целое число. Это положительное значение от 0 до 255.
буфер_с8 Знаковое 8-разрядное целое число. Это может быть положительное или отрицательное значение от -128 до 127 (0 считается положительным).
буфер_u16 Беззнаковое 16-разрядное целое число. Это положительное значение от 0 до 65 535.
буфер_с16 Подписанное 16-разрядное целое число. Это может быть положительное или отрицательное значение от -32,768 до 32,767 (0 считается положительным).
буфер_u32 Беззнаковое 32-разрядное целое число. Это положительное значение от 0 до 4 294 967 295.
буфер_с32 Подписанное 32-разрядное целое число. Это может быть положительное или отрицательное значение от -2,147,483,648 до 2,147,483,647 (0 считается положительным).
буфер_u64 Беззнаковое 64-разрядное целое число.
буфер_f16 16-разрядное плавающее значение. Это может быть положительное или отрицательное значение в диапазоне +/- 65504. (В настоящее время не поддерживается!)
буфер_f32 32-разрядное плавающее значение. Это может быть положительное или отрицательное значение в диапазоне +/-16777216.
буфер_f64 64-разрядное плавающее число.
buffer_bool Булево значение. Может быть только 1 или 0(true или false).
буферная_строка Струна любого размера.
буферный_текст Строка любого размера без завершающего нулевого символа.

Если функция выполнится успешно, то она вернет значение заданного типа, однако если она не выполнится, то это приведет к ошибке runner.

ПРИМЕЧАНИЕ: Использование неправильного типа данных для считываемых данных приведет к возврату ошибочных значений.

Синтаксис:

buffer_read(buffer, type)

Аргумент Описание
буфер Индекс буфера для чтения.
тип Тип данных, которые должны быть считаны из буфера (см. список констант выше).

Возвращается:

Вещественный/Строка

Пример:

var cmd = buffer_read(buff, buffer_s16);

Приведенный выше код считывает из буфера с id, хранящегося в переменной "buff", знаковое 16-битное значение в локальную переменную "cmd".
