буферная_декомпрессия

С помощью этой функции вы можете распаковать ранее сжатый буфер, используя сжатие zlib. Вы передаете идентификатор буфера для распаковки, и функция возвращает новый идентификатор буфера, который содержит несжатые данные. Если декомпрессия не удалась (например, вы предоставляете буфер, который не был сжат), то функция вернет значение меньше 0.

Синтаксис:

buffer_decompress(buffer);

Аргумент Описание
буфер Индекс буфера для распаковки.

Возвращается:

Идентификатор буфера

Пример:

var cmpBuff = buffer_load("Player_Save.sav");
var srcBuff = buffer_decompress(cmpBuff);
global.DataString = buffer_read(srcBuff, buffer_string);

Приведенный выше код сначала загрузит сохраненный буфер, затем распакует его и, наконец, прочитает строковые данные из распакованного буфера в глобальную переменную.
