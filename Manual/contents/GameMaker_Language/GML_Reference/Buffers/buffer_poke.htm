буферная спица

С помощью функции buffer_write() вы можете записывать данные в заданный буфер в текущей позиции "seek", при этом каждая порция данных будет продвигаться вперед по этой позиции на записываемые или считываемые байты. Однако может возникнуть необходимость изменить заданный фрагмент данных, не меняя текущую позицию поиска, и тогда вы можете использовать эту функцию. Вы просто передаете функции индекс буфера, а затем позицию смещения от буфера (в байтах) в пределах этого буфера для записи, а также тип данных и записываемое значение.

Синтаксис:

buffer_poke(buffer, offset, type, value);

Аргумент Описание
буфер Индекс используемого буфера.
смещение Позиция смещения (в байтах) в буфере для записи заданных данных.
тип Тип данных, которые должны быть записаны в буфер (см. список констант здесь).
значение Данные для добавления в буфер, в соответствии с указанным типом.

Возвращается:

Н/Д

Пример:

buffer_poke(buff, 3, buffer_u8, color_get_blue(image_blend));

Приведенный выше код добавит значение синей составляющей цвета, используемого для наложения изображения, в буфер, индексированный в переменной "buff", на третью позицию в буфере и как беззнаковое 8-битное значение.
