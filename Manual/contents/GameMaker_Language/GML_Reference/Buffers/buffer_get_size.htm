размер буфера_get_size

С помощью этой функции вы можете получить размер заданного буфера в байтах.

Синтаксис:

buffer_get_size(index);

Аргумент Описание
индекс Индекс буфера для получения размера.

Возвращается:

Реально

Пример:

var _size = buffer_get_size(player_data);
var _temp = buffer_create(_size, buffer_fixed, 0);

Приведенный выше код создаст новый буфер и сохранит его индекс в локальной переменной "_temp", причем размер этого нового буфера будет равен размеру буфера, проиндексированного в переменной "player_data".
