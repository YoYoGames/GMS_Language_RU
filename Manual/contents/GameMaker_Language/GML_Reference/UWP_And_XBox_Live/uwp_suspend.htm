uwp_suspend

Эта функция указывает системе, что приложение завершило сохранение данных после обнаружения запроса на приостановку. Вызов этой функции является обязательным требованием для представления Xbox One, и ее всегда следует использовать вместе с функцией uwp_is_suspending(), как показано в примере ниже.

Синтаксис:

uwp_suspend();

Возвращается:

Н/Д

Пример:

if uwp_is_suspending()
{
scr_Save_Game_Data();
uwp_suspend();
}

Приведенный выше код проверяет, переходит ли приложение в режим приостановки, и если да, то вызывает сценарий для сохранения игровых данных перед приостановкой игры.
