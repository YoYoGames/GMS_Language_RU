uwp_license_trial_version

Эта функция может быть использована для проверки того, находится ли игра под пробной лицензией или нет. Если да, то функция вернет true, в противном случае - false.

Синтаксис:

uwp_license_trial_version();

Возвращается:

Булево

Пример:

if uwp_license_trial_version()
{
global.LevelCap = 10;
}

Приведенный выше код проверяет, не является ли приложение пробным, и если да, то устанавливает глобальную переменную.

Назад: UWP и XBox Live
