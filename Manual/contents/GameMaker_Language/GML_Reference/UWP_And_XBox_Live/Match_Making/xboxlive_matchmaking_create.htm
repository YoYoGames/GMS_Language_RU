xboxlive_matchmaking_create

Эта функция создаст многопользовательскую сессию, используя шаблон сессии, бункер подбора и шаблон Secure Device Association, который вы должны были предварительно создать в приборной панели XDP. Идентификатор пользователя, который вы можете получить для конкретного игрового площадки, вызвав xboxlive_user_for_pad() функции и видимость будет одной из следующих констант:

Постоянная Описание
xboxlive_match_visibility_open указывает, что к сеансу могут присоединиться другие.
xboxlive_match_visibility_private указывает, что сессия не видна пользователям, которые не являются членами сессии.
xboxlive_match_visibility_usetemplate указывает, что должно использоваться значение, используемое в шаблоне сессии (обычно используется это значение, так как вы не можете отменить значение шаблона, если оно было установлено в XDP).

Функция возвращает уникальное значение Request ID, которое затем может быть использовано для определения правильного Social Asynchronous Event для этой функции. Это событие будет запущено после создания сессии и будет содержать DS Map в переменной async_load со следующими парами ключ/значение:

  • "requestid" - содержит идентификатор запроса, возвращенный вызывающей функцией
  • "status" - содержит строку "session_created" для информирования о том, какой тип события был вызван.
  • "sessionid" - содержит идентификатор сессии, или -1 в случае неудачи
  • "error" - содержит 0 в случае успеха, или -1 в случае неудачи
  • "correlationid" - хэндл сессии для корреляции с функцией xboxlive_matchmaking_join_session()

Синтаксис:

xboxlive_matchmaking_create(user_id, visibility, template, hopper, sdatemplate, [matchattributes]);

Аргумент Описание
идентификатор пользователя Указатель идентификатора пользователя для использования
видимость Одна из перечисленных выше констант
шаблон Имя шаблона сеанса
бункер Название бункера для спичек
sdatemplate Имя шаблона ассоциации защищенных устройств
matchattributes [ВАРИАНТ]

Возвращается:

Идентификатор запроса (целое число)

Пример:

var userID = xboxlive_user_for_pad(global.PadIndex);
result = xboxlive_matchmaking_create(userID, xboxlive_match_visibility_usetemplate, "MatchTicketSession", "MatchTicketHopper", "PeerServerTraffic");

Приведенные выше действия позволят получить идентификатор пользователя на данном геймпаде, а затем инициализировать для него сеанс подбора.
