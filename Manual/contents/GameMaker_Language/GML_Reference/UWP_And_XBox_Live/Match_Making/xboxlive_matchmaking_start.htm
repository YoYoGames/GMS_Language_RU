xboxlive_matchmaking_start

Эта функция должна быть вызвана до того, как вы сможете начать любую многопользовательскую сессию для данного пользователя. Она принимает указатель ID пользователя, который вы можете получить для конкретной игровой панели, вызвав функцию xboxlive_user_for_pad() и инициализирует API мультиплеера.

Синтаксис:

xboxlive_matchmaking_start(user_id);

Аргумент Описание
идентификатор пользователя Указатель идентификатора пользователя для использования

Возвращается:

Н/Д

Пример:

global.UserID = xboxlive_user_for_pad(global.PadIndex);
xboxlive_matchmaking_start(global.UserID);

Приведенный выше код получает идентификатор пользователя для заданного индекса пэда, а затем инициализирует многопользовательские возможности.
