xboxlive_matchmaking_send_invites

Эта функция вызывает системный диалог Xbox Live, позволяющий пользователю отправлять приглашения в свою игру. Вы указываете локального пользователя, который должен управлять диалогом (и чей список друзей будет использоваться диалогом), уникальное значение ID сессии для сеанса, в который они хотят пригласить людей, и текст по умолчанию для отображения в диалоге. Идентификатор сеанса можно получить из async_load DS карты, созданной функцией xboxlive_matchmaking_create().

Функция вернет -1, если аргументы недействительны или пользователь не зарегистрирован в Xbox Live, или значение больше 0 в противном случае.

Синтаксис:

xboxlive_matchmaking_send_invites(from_user, session_to_invite_into, text_to_add_to_invite);

Аргумент Описание
от_пользователя Локальный идентификатор пользователя для управления диалогом.
сессия_к_включению Уникальный идентификатор сессии для сессии, к которой нужно присоединиться.
текст_к_добавлению_к_приглашению Короткий текст для добавления к приглашению.

Возвращается:

Целое число

Пример:

if gamepad_button_check_pressed(0, gp_face1)
{
var _uid = xboxlive_user_for_pad(0);
xboxlive_matchmaking_send_invites(_uid, global.session_ID, "Присоединяйтесь к моей игре!");
}

Приведенный выше код проверяет нажатие кнопки геймпада, и если оно обнаружено, то открывается диалог Xbox Live для приглашения в сессию.
