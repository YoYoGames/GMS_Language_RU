xboxlive_matchmaking_session_leave

Эта функция заставляет пользователя, который создал или нашел указанную сессию, покинуть ее. Вы можете получить указатель идентификатора сессии из async_load карты DS, которая возвращается в асинхронном социальном событии, когда вы создали или нашли сессию (см. xboxlive_matchmaking_create() для более подробной информации).

Синтаксис:

xboxlive_matchmaking_session_leave(session_id);

Аргумент Описание
идентификатор сессии Указатель идентификатора сессии, которую нужно покинуть.

Возвращается:

Н/Д

Пример:

if gamepad_button_check_pressed(0, gp_start)
{
xboxlive_matchmaking_session_leave(global.SessionID);
}

Приведенный выше код проверяет нажатие кнопки геймпада, и если таковое обнаружено, то пользователь выходит из данной сессии матчмейкинга.
