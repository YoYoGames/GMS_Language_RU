xboxlive_read_player_leaderboard

Функция возвращает информацию о таблице лидеров для заданного имени пользователя. Таблица лидеров должна быть создана ранее на приборной панели XDP для вашей игры, а вызов функции вызовет обратный вызов Social Asynchronous Event, который содержит async_load карту, заполненную соответствующими парами ключ/значение. Ключ "id" этой карты DS используется для идентификации правильного обратного вызова (для любого асинхронного события может быть более одной функции запуска), и IT будет сопряжен с константой achievement_leaderboard_info, а также рядом других пар ключ/значение. Точное содержимое карты показано ниже:

  • " id" - Для этой функции это должно быть achievement_leaderboard_info
  • "leaderboardid" - Уникальный ID для доски лидеров, определенный на приборной панели провайдера.
  • "numentries" - Количество записей в таблице лидеров, которые вы получили.
  • "PlayerN" - Имя игрока, где "N" - целочисленное значение, соответствующее его позиции в списке участников таблицы лидеров.
  • "PlayeridN" - Уникальный идентификатор пользователя игрока, "N".
  • "RankN" - Ранг игрока "N" в таблице лидеров.
  • "ScoreN" - Счет игрока "N".


Для установки свойств фильтра функция требует указать одну из следующих констант:

Постоянная Описание
фильтр_достижений_всех_игроков Получите все баллы.
фильтр_достижений_только_для_друзей Получить только оценки друзей в порядке возрастания.
достижение_фильтр_фаворитов_только Получение только избранных оценок в порядке возрастания
фильтр_достижений_друзья_алт Получить оценки только друзей в порядке убывания.
достижение_фильтр_фаворитов_alt Получение только избранных оценок в порядке убывания.

Синтаксис:

xboxlive_read_player_leaderboard(leaderboard_name, user_name, num_items, friend_filter);

Аргумент Описание
имя_лидера Название таблицы лидеров для чтения, установленное на XDP.
имя пользователя Имя пользователя для чтения.
количество_элементов Количество элементов для извлечения.
фильтр друзей Одна из констант фильтра (см. описание выше).

Возвращается:

Реально

Пример:

var _user = xboxlive_user_for_pad(0);
var _uid = xboxlive_user_id_for_user(_user);
xboxlive_read_player_leaderboard("MyLeaderboard", _uid, 10, achievement_filter_all_players);

Приведенный выше код получает идентификатор пользователя и затем использует его для запроса всех данных игрока для первых 10 мест заданной таблицы лидеров.
