xboxlive_stats_setup

Эта функция должна быть вызвана до того, как вы сможете использовать любые другие функции Xbox Live stat, и просто инициализирует необходимые библиотеки в системе. Аргумент "user_id" - это необработанный идентификатор пользователя, возвращаемый функцией xboxlive_get_user(), а "service_config" и "title_id" - это уникальный ID вашей игры в Xbox Live Dev Center.

Синтаксис:

xboxlive_stats_setup(user_id, service_config_id, title_id);

Аргумент Описание
идентификатор пользователя Указатель ID пользователя для проверки.
service_config_id Это строка config_id для игры.
идентификатор названия Уникальный идентификатор вашей игры в Xbox Dev Center.

Возвращается:

Н/Д

Пример:

var user = xboxlive_get_user(0); xboxlive_stats_setup(user, "4d61a1aa-61ac-4541-badd-31f91244fea6", $1244FEA6);

Приведенный выше код инициализирует систему статистики для заданного идентификатора пользователя.
