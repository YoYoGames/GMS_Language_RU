xboxlive_stats_delete_stat

Эта функция может быть использована для удаления статистики из менеджера статистики для заданного ID пользователя. Вы указываете ID пользователя, который возвращается (например) функцией xboxlive_get_user(), затем строка stat для удаления. Это очистит значение stat и удалит его из менеджера stat, что означает, что оно больше не будет возвращено функциями xboxlive_stats_get_stat_names() или xboxlive_stats_get_stat().

Функция вернет -1, если произошла ошибка или идентификатор пользователя недействителен, или любое другое значение, если функция была вызвана успешно.

Синтаксис:

xboxlive_stats_delete_stat(user_id, stat);

Аргумент Описание
идентификатор пользователя Указатель идентификатора пользователя для удаления статистики
статистика Статистика для удаления (строка)

Возвращается:

Реально

Пример:

for(var i = 0; i < xboxlive_get_user_count(); i++;)
{
user_id[i] = xboxlive_get_user(i);
xboxlive_stats_delete_stat(user_id[i], "HighScore");
}

Приведенный выше код перебирает всех подключенных пользователей и удаляет указанную статистику из менеджера статистики для каждого из них.
