xboxlive_stats_get_stat

Эта функция может использоваться для получения одного значения статистики из менеджера статистики для данного пользователя. Вы передаете идентификатор пользователя, возвращаемый (например) функцией xboxlive_get_user(), и затем строку статистики, заданную при ее создании с помощью одной из функций xboxlive_stats_set_stat_*. Возвращаемое значение может быть либо строкой, либо вещественным числом (в зависимости от проверяемого стата), либо GML-константой undefined, если стат не существует или произошла ошибка.

Синтаксис:

xboxlive_stats_get_stat(user_id, stat);

Аргумент Описание
идентификатор пользователя Указатель идентификатора пользователя для получения имен статистов
статистика Статистика, которую нужно установить (строка)

Возвращается:

Вещественный, строковый или неопределенный

Пример:

если (game_over == true)
{
if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100)
{
var _val = (global.LevelsFinished / global.LevelsTotal) * 100;
xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val);
}
}

Приведенный выше код проверяет переменную, и если она истинна , то проверяется значение статистики "PercentDone". Если это значение меньше 100, генерируется значение, и статистика устанавливается на это значение.
