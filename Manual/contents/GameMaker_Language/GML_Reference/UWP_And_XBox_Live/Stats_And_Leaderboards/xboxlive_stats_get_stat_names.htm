xboxlive_stats_get_stat_names

Эта функция может быть использована для получения всех определенных статистических данных из менеджера статистики для данного пользователя. Вы вводите идентификатор пользователя, возвращаемый (например) функцией xboxlive_get_user(), и функция вернет массив строк, содержащих статистику для данного пользователя. Если произошла ошибка или у пользователя нет статистики, массив будет возвращен, но количество элементов будет равно 0.

Синтаксис:

xboxlive_stats_get_stat_names(user_id);

Аргумент Описание
идентификатор пользователя Указатель идентификатора пользователя для получения имен статистов

Возвращается:

Массив

Пример:

var _stat_str = xboxlive_stats_get_stat_names(user_id);
for (var i = 0; i < array_length(_stat_str); ++i;)
{
xboxlive_stats_delete_stat(user_id, _stat_str[i]);
}

Приведенный выше код извлекает все определенные статистики для указателя ID пользователя, хранящегося в данной переменной, а затем проходит по массиву и удаляет указанную статистику из менеджера статистик.
