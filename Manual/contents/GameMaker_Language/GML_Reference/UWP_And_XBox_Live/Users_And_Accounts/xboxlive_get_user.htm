xboxlive_get_user

С помощью этой функции вы можете получить указатель идентификатора пользователя для индексированного пользователя. Если пользователь не существует, функция вернет константу pointer_null. Количество пользователей, вошедших в систему в данный момент, можно узнать с помощью функции xboxlive_get_user_count().

ВАЖНО: Эта функция должна использоваться только с xboxlive_get_user_count() - не используйте ее с геймпадами. Используйте xboxlive_user_for_pad() вместо нее.

Синтаксис:

xboxlive_get_user(index);

Аргумент Описание
индекс Индекс (целое число) для получения идентификатора пользователя.

Возвращается:

Указатель

Пример:

for(var i = 0; i < xboxlive_get_user_count(); ++i;)
{
user_id[i] = xboxlive_get_user(i);
}

Приведенный выше цикл перебирает всех зарегистрированных пользователей и сохраняет их уникальный идентификационный указатель в массиве.
