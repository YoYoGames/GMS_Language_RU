xboxlive_agegroup_for_user

Эта функция возвращает одну из четырех констант (показанных ниже), указывающих на возрастной диапазон, присвоенный указанному указателю идентификатора пользователя.

Синтаксис:

xboxlive_agegroup_for_user(user_id);

Аргумент Описание
идентификатор пользователя Идентификатор пользователя (указатель) для проверки

Возвращается:

Постоянная

Постоянная Описание
xboxlive_agegroup_unknown Возрастная группа пользователя неизвестна
xboxlive_agegroup_child Возрастная группа пользователей - от 8 до 12 лет (включительно)
xboxlive_agegroup_teen Возрастная группа пользователей - от 13 до 17 лет (включительно, и от 13 до 19 лет в Южной Корее)
xboxlive_agegroup_adult Возрастная группа пользователей - 18 лет и старше (или 20 лет и старше в Южной Корее

Пример:

var _age = xboxlive_agegroup_for_user(xboxlive_get_activating_user());
if _age != xboxlive_agegroup_adult
{
global.ContentControl = true;
}
else global.ContentControl = false;

Приведенный выше код проверяет возрастную группу активирующего пользователя и устанавливает глобальную переменную в true или false в зависимости от возвращаемой константы.
