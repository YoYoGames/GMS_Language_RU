xboxlive_sprite_add_from_gamerpicture

С помощью этой функции вы можете получить изображение геймера для заданного указателя ID пользователя. Функция работает асинхронно, и будет вызывать асинхронное событие Image Loaded чтобы сообщить вам, что функция завершена (как в случае с HTML5/URL-вызовами к sprite_add()). Асинхронное событие Image Loaded будет иметь карту DS, хранящуюся во встроенной переменной async_load. Карта содержит следующую информацию:

  • " filename": Полный путь к запрашиваемому файлу.
  • " id": Идентификатор загруженного ресурса. Он будет совпадать с переменной, которой вы присвоили ресурс.
  • "status": Возвращает значение меньше 0 в случае ошибки.

ПРИМЕЧАНИЕ: Когда вы динамически загружаете спрайт в вашу игру во время выполнения, вы должны помнить, чтобы удалить его снова с помощью функции sprite_delete() когда он больше не нужен, В противном случае существует риск утечки памяти, что приведет к замедлению и в конечном итоге к краху вашей игры.

Синтаксис:

xboxlive_sprite_add_from_gamerpicture(user_id, imagesize, xorig, yorig);

Аргумент Описание
идентификатор пользователя Идентификатор пользователя (указатель) для получения изображения геймера
imagesize Размер в пикселях возвращаемого спрайта
xorig Укажите x-позицию начала координат в спрайте
yorig Укажите положение начала координат по оси y в спрайте
