xboxlive_pad_for_user

Пользователи Xbox могут назначать различные пэды своему идентификатору пользователя, каждый из которых заполняет "слот". Эти слоты нумеруются от 0 и могут содержать любой индекс пэда, поэтому слот 0 может быть назначен пэду 3, а слот 1 - пэду 2. Эта функция может быть использована для получения пэда, назначенного на любой заданный слот, и особенно полезна, если пользователь оставляет геймпад без ввода в течение определенного времени, так как он отключается сам по себе, поэтому вы можете проверить любой ID пользователя с помощью этой функции убедиться, что у него есть активный геймпад, и если он не активен, то отреагировать соответствующим образом, например, поставить игру на паузу. Если геймпад активен, то будет возвращено значение его индекса, если нет, то функция вернет -1.

Обратите внимание, что если пэд в каком-либо слоте становится неактивным, последующие слоты "сдвигаются" на освободившееся место, поэтому если у вас есть три активных пэда, назначенных на слоты 0, 1 и 2, а пэд в слоте 0 неактивен, слоты 1 и к теперь станут слотами 0 и 1, поэтому слот 0 всегда должен содержать активный пэд, если только все пэды не выключены (в этом случае он вернет -1).

Синтаксис:

xboxlive_pad_for_user(user_id, slot);

Аргумент Описание
идентификатор пользователя Идентификатор пользователя (указатель) для проверки
идентификатор пользователя Индекс (целое число) слота геймпада для проверки

Возвращается:

Указатель

Пример:

if (xboxlive_pad_for_user(user_id, 0) == -1)
{
for(var i = 0; i < gamepad_get_device_count(); i++;)
{
var u_id = xboxlive_user_for_pad(i);
if (u_id == -1)
{
async_ID = xboxlive_pad_for_user(i, u_id);
}
}
}

Приведенный выше код проверяет слот 0 для заданного указателя User ID на предмет наличия назначенных геймпадов, и если нет, то назначается один.
