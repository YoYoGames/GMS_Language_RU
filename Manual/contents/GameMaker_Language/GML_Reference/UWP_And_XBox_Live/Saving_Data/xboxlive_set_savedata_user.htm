xboxlive_set_savedata_user

Эта функция определяет, что будущие операции с файлами, которые работают в области сохранения игры (т.е. все записи в файлы и чтения из файлов, которые не находятся в области бандла), будут связаны с указанным указателем ID пользователя. Эта функция может вызываться так часто для перенаправления данных сохранения соответствующему пользователю, или вы можете использовать константу pointer_null для сохранения в общую область хранения машины.

Синтаксис:

xboxlive_set_savedata_user(user_id);

Аргумент Описание
идентификатор пользователя Идентификатор пользователя (указатель), который нужно установить для сохранения, или указатель_null

Возвращается:

Н/Д

Пример:

if (xboxlive_get_savedata_user() != user_id[0])
{
xboxlive_set_savedata_user(user_id[0]);
}

Приведенный выше код проверяет, назначен ли пользователь в качестве цели сохранения, и если нет, то он назначается.
