uwp_is_suspending

Эта функция возвращает true для одного кадра (и false в противном случае) аналогично функции os_is_paused(). После этого у игры есть одна секунда, чтобы сохранить состояние игры, прежде чем она прежде чем она будет потенциально завершена системой. Как только игра закончит сохранение данных, она должна вызвать функцию uwp_suspend() чтобы приостановить игру. Возможно, игра также должна войти в состояние паузы, чтобы в случае ее возобновления системой система (если она не была полностью завершена), игрок не будет возвращен обратно в игровую ситуацию.

Синтаксис:

uwp_is_suspending();

Возвращается:

Булево

Пример:

if uwp_is_suspending()
{
scr_Save_Game_Data();
uwp_suspend();
}

Приведенный выше код проверяет, переходит ли приложение в режим приостановки, и если да, то вызывает сценарий для сохранения игровых данных перед приостановкой игры.

Назад: UWP и XBox Live
