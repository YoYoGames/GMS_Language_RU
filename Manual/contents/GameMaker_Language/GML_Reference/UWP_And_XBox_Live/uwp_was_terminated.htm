uwp_was_terminated

С помощью этой функции можно узнать, была ли игра ранее выключена системой (т.е. это не первый запуск игры). Если это было так, то функция вернет true, в противном случае она вернет false. Примером может быть проверка, не была ли игра ранее завершена системой, а затем (если возвращается true) загрузка предыдущего этапа, на котором находился игрок. Если игра предназначена для XBox One, то требуемое поведение игр в случае обнаружения этого факта будет отличаться в разных версиях документации по сертификации, поэтому стоит проверить последнюю версию, чтобы узнать, что должна делать игра .

Синтаксис:

uwp_was_terminated();

Возвращается:

Булево

Пример:

if xboxlivewas_terminated()
{
buffer_load("Player_Save.sav");
global.GameLoaded = true;
}

Приведенный выше код проверяет, не было ли приложение ранее завершено системой, и если да, то загружает буфер и устанавливает глобальную переменную.
