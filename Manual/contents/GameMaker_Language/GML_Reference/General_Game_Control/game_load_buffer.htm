игровой_загрузочный_буфер

С помощью этой функции вы можете загрузить состояние игры, которое было сохранено ранее. Игра загружается из ранее созданного буфера "grow" (см. раздел "Буферы") , и в этом буфере должно быть сохранено состояние игры. с помощью game_save_buffer() функция.

Синтаксис:

game_load_buffer(buffer);

Аргумент Описание
буфер Идентификатор буфера для загрузки.

Возвращается:

Н/Д

Пример:

if keyboard_check_pressed(ord("L"))
{
if global.Checkpoint game_load_buffer(save_buff);
}

Приведенный выше код загрузит ранее сохраненное состояние игры из буфера, проиндексированного в переменной "save_buff", только если глобальная переменная равна true, когда игрок нажмет клавишу "L".
