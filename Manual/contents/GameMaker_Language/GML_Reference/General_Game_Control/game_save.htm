игра_сохранение

Это устаревшая функция, которая может быть использована для сохранения текущего состояния игры, и больше не рекомендуется к использованию. Вместо этого используйте функции File для создания пользовательской системы сохранения, в которой вы сохраняете и загружаете только и загружать только определенные игровые данные.

Эта функция сохранит состояние игры как есть, однако не сохранит динамические ресурсы, используемые в это время, такие как структуры данных, поверхности, активы, добавленные во время выполнения и т.д. Если такой файл сохранения будет загружен, любые обновления игры, которые были сделанные после сохранения, не будут видны, так как будет восстановлена версия игры, которая использовалась для создания сохранения.

Обратите внимание, что файлы сохранения, созданные с помощью этой функции, могут не поддерживаться обновленными Время выполнения поэтому замена этой функции на новую систему сохранения необходима для обеспечения совместимости с будущими обновлениями GameMaker и, по причинам, перечисленным выше, обновлениями вашей собственной игры.

Синтаксис:

game_save(filename);

Аргумент Описание
имя файла Имя файла, в который будет сохранена игра.

Возвращается:

Н/Д

Пример:

if keyboard_check_pressed(ord("S"))
{
global.Saved = true;
game_save("Save.dat");
}

Приведенный выше код установит глобальную переменную в true, а затем сохранит игру в файл "Save.dat" при нажатии клавиши "S".
