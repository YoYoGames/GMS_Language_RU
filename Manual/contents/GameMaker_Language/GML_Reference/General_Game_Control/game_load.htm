игровая_загрузка

С помощью этой функции вы можете загрузить игру, которая была ранее сохранена с помощью функции game_save(). Обратите внимание, что при этом восстанавливается версия игры, которая использовалась для создания сохранения, поэтому любые обновления, сделанные после этого, не будут видны. видны. Для получения дополнительной информации прочтите страницу game_save().

Синтаксис:

game_load(filename);

Аргумент Описание
имя файла Имя файла для загрузки.

Возвращается:

Н/Д

Пример:

if keyboard_check_pressed(ord("L"))
{
if global.Save game_load("Save.dat");
}

Приведенный выше код загрузит ранее сохраненную игру, если глобальная переменная равна true, когда игрок нажимает клавишу "L".
