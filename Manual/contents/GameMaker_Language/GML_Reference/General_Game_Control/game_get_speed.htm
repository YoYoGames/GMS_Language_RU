game_get_speed

Эта функция может быть использована для получения скорости игры либо как количество игровых кадров в секунду, либо как количество микросекунд на игровой кадр. Обратите внимание, что это не фактическое значение скорости бега FPS (для этого используйте переменную fps_real переменная) а скорее количество игровых кадров (FPS), которые игра будет пытаться поддерживать каждую секунду, или длительность каждого игрового кадра в микросекундах, которую игра будет пытаться поддерживать (MPF). Когда вы используете эту функцию, вам необходимо задать одну из следующих констант констант, которые определят тип возвращаемого значения:

Постоянная Описание
скорость_игры_fps Получает скорость игры, используя количество кадров в секунду.
скорость_микросекунды Получает скорость игры, используя микросекунды на кадр.

Так, например, если скорость игры установлена на 30 в Опциях игры и вы используете тип FPS, то функция вернет 30, но если вы используете MPF, то функция вернет 33333.

Синтаксис:

game_get_speed(type);

Аргумент Описание
тип Тип метода, используемого для получения скорости игры (см. константы выше).

Возвращается:

Реально

Пример:

if game_get_speed(gamespeed_fps) != 60
{
game_set_speed(60, gamespeed_fps);
}

Приведенный выше код проверяет, запущена ли игра с игровой скоростью 60 FPS, и если нет, то устанавливается значение 60 FPS.
