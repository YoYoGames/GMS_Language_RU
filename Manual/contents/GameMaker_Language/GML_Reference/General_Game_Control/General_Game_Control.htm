Общее управление игрой

Все эти функции и переменные связаны с функциональностью, специфичной для игры, например, загрузка/сохранение или перезапуск и т.д..

Следующие встроенные переменные глобального диапазона полезны для отладки вашей игры - они могут быть использованы, например, как вывод в файл журнала в случае возникновения проблем - а также а также для отображения текущего ID сборки и т.д... для пользователя:

  • игровой идентификатор
  • game_save_id
  • имя_игрового_дисплея
  • имя_игрового_проекта

Следующие функции используются для общего управления игрой:

  • конец игры
  • перезапуск игры
  • игровая_загрузка
  • игровой_загрузочный_буфер
  • игра_сохранение
  • игровой_сохранённый_буфер
  • game_get_speed
  • game_set_speed

Чтобы помочь вам создавать и поддерживать высокие баллы для ваших игр, GameMaker Studio 2 создает глобальный массив высоких баллов из 10 мест, к которым вы можете обращаться, добавлять и изменять, чтобы создавать свои собственные списки высоких баллов без лишних хлопот. списки высоких результатов без лишних хлопот. Эта функциональность является локальной для игры и работает на всех платформах, поэтому вы можете легко хранить и поддерживать базовую таблицу высоких результатов, используя эти функции вместе с функциями обработки файлов.

Существуют следующие функции, работающие с внутренним списком высоких результатов.

  • highscore_add
  • имя_высокой_оценки
  • значение_высокой_оценки
  • highscore_clear

GameMaker Studio 2 также включает функцию для вывода результатов на экран в виде списка, однако она очень проста, и рекомендуется использовать ее для отладки, а не для чего-либо еще, поскольку создать собственную таблицу результатов очень просто. легко и дает вам гораздо больше контроля над тем, как она отображается:

  • draw_highscore

Наконец, у нас есть функция, позволяющая назначить спрайт курсору мыши:

  • курсор_спрайт
