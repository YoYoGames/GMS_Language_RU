строка_ниже

С помощью этой функции вы можете заставить строку содержать только символы нижнего регистра.

ПРИМЕЧАНИЕ: Эта функция распознает только 26 букв английского алфавита от A до Z.

Синтаксис:

string_lower(string);

Аргумент Описание
строка Строка для преобразования в нижний регистр.

Возвращается:

Строка

Пример:

str1 = "Hello World";
str2 = string_lower(str1);

Приведенный выше код установит str2 в значение "hello world".
