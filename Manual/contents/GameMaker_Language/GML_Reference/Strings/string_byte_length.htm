длина_байта_строки

Эта функция возвращает количество байтов в строке, но следует учитывать, что из-за того, что они хранятся в формате UTF8, это не будет равно длине строки.

Синтаксис:

string_byte_length(string);

Аргумент Описание
строка Строка для измерения количества байтов.

Возвращается:

Реально

Пример:

bytesize = string_byte_length("Hello World");

Это установит bytesize на количество байт в "Hello World".
