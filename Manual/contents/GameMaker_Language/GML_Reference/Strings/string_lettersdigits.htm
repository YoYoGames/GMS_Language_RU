строка_буквенных цифр

Эта функция возвращает копию заданной строки с удаленными буквами и цифрами, что означает, что ее можно использовать для удаления ненужных символов (таких как "#" или "?"), например, из имени пользователя или пароля.

ПРИМЕЧАНИЕ: Эта функция возвращает только 26 букв английского алфавита от A до Z (в верхнем или нижнем регистре) и цифры 0 - 9.

Синтаксис:

string_lettersdigits(string);

Аргумент Описание
строка Строка для преобразования в буквы и цифры.

Возвращается:

Строка

Пример:

if (string_length(password) > string_length(string_lettersdigits(password)))
{
draw_text(32,32, "Неверный пароль! Только буквы и цифры, пожалуйста!");
}

Приведенный выше код проверяет длину строки на длину той же строки, но с удаленными символами. Если они не совпадают, на экран выводится сообщение.
