string_pos_ext

Эта функция возвращает позицию символа подстроки в строке, выполняя поиск вперед по строке, начиная с позиции, заданной в качестве начальной. Функция возвращает 0, если искомая строка не найдена, или позицию первого символа искомой строки, если она найдена. Имейте в виду, что для старой поддержки строки индексируются с 1, поэтому 1 - это первая позиция в строке, а не 0, как вы могли бы ожидать.

Синтаксис:

string_pos_ext(substr, str, start_pos);

Аргумент Описание
субстрат Подстрока, которую нужно искать в строке.
str Струна.
start_pos Начальная позиция для поиска.

Возвращается:

Целое число

Пример:

if (string_pos_ext(",", text, 20) != 0)
{
string_insert(name, text, string_pos_ext(",", text, 20));
}

Приведенный выше код ищет в строке, хранящейся в переменной "text", запятую после 20-го символа, и если находит ее, то вставляет в эту позицию подстроку "name".
