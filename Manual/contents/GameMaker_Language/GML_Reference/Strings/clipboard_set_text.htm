clipboard_set_text

Эта функция устанавливает в буфер обмена заданную строку. Вы можете установить пустую строку "", чтобы эффективно очистить буфер обмена от текста.

ПРИМЕЧАНИЕ: Эта функция действительна только для Windows цели.

Синтаксис:

clipboard_set_text(string);

Аргумент Описание
строка Текст (в виде строки), который будет храниться в буфере обмена.

Возвращается:

Н/Д

Пример:

if clipboard_has_text()
{
str = clipboard_get_text();
clipboard_set_text("");
}

Приведенный выше код проверяет буфер обмена на наличие текста и, если он есть, считывает его как строку в переменную "str". Наконец, буфер обмена очищается путем установки в него пустой строки.
