string_char_at

Вы можете использовать эту функцию для возврата определенного символа в определенной позиции в строке, при этом индекс для первого символа начинается с 1. Если символ не найден или строка короче, чем значение индекса, возвращается пустая строка "". возвращается.

Синтаксис:

string_char_at(str, index);

Аргумент Описание
str Строка для проверки.
индекс Позиция, с которой нужно получить персонажа.

Возвращается:

Строка

Пример:

str1 = "Hello World";
str2 = string_char_at(str1, 7);

Это установит str2 на седьмой символ ("H" считается первым) в строке str1, в данном случае"W".
