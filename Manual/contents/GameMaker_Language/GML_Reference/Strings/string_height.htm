высота_строки

Эта функция возвращает высоту (в пикселях) введенной строки, учитывая разделение строк и любые разрывы строк, которые может иметь текст. Это очень удобно для расчета расстояния между элементами текста на основе самой высокой из букв. составляющих строку, как она была бы нарисована с помощью draw_text() используя текущий определенный шрифт.

Синтаксис:

string_height(string);

Аргумент Описание
строка Строка для измерения высоты.

Возвращается:

Реально

Пример:

var hh = string_height("ABCDEFGHIJKLMNOPQRSTUVWXYZ");
draw_text(32, 32, string(score));
draw_text(32, 32 + hh, string(lives);

Приведенный выше код получит высоту заданной строки, а затем нарисует две строки текста, используя возвращенную высоту строки в качестве разделителя.
