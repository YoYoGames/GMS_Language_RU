string_set_byte_at

Эта функция устанавливает байт непосредственно в строке (на основе формата UTF8) и возвращает копию строки с изменениями.

ПРИМЕЧАНИЕ: Эта функция работает невероятно медленно, поэтому тщательно подумайте, нужна ли она и где вы ее используете.

Синтаксис:

string_set_byte_at(str, pos, byte);

Аргумент Описание
str Строка для изменения байта.
поз Позиция в строке (начиная с 1) для изменения байта.
байт Новое значение байта.

Возвращается:

Строка

Пример:

str = string_set_byte_at("hello", 2, 97);

Приведенный выше код изменит значение байта второй буквы в строке и таким образом установит переменную str в значение "hallo".
