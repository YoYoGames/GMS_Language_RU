строка_повтора

Эта функция просто возвращает одну и ту же строку, повторенную заданное количество раз над самой собой.

Синтаксис:

string_repeat(str, count);

Аргумент Описание
str Строка для повторения.
считать Количество повторений строки.

Возвращается:

Строка

Пример:

str1 = "Hello World";
str2 = string_repeat(str1, 3);

Приведенный выше код установит str2 в значение "Hello WorldHello WorldHello World".
