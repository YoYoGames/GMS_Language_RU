string_last_pos

Эта функция возвращает позицию символа экземпляра подстроки в строке, осуществляя поиск от конца строки к началу (таким образом, обратная функция string_pos(). Функция вернет 0 если искомая строка не найдена, или позицию первого символа искомой строки, если она найдена. Имейте в виду, что для старой поддержки строки индексируются с 1, поэтому 1 - это первая позиция в строке, а не 0, как вы могли бы ожидать.

Синтаксис:

string_last_pos(substr, str);

Аргумент Описание
субстрат Подстрока, которую нужно искать в строке.
str Струна.

Возвращается:

Целое число

Пример:

if (string_last_pos(",", text) != 0)
{
string_insert(name, text, string_last_pos(",", text));
}

Приведенный выше код ищет в строке, хранящейся в переменной "text", запятую, начиная поиск с конца строки. Если она найдена, то в эту позицию вставляется подстрока "name".
