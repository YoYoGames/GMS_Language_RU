счётчик строк

Эта функция возвращает количество повторений заданной подстроки в определенной строке. Таким образом можно проверить, сколько раз повторяется одна буква или фраза в участке хранимого текста.

Синтаксис:

string_count(substr, str);

Аргумент Описание
субстрат Подстрока, на наличие которой нужно проверить строку.
str Строка для проверки.

Возвращается:

Реально

Пример:

str1 = "Hello World";
ocount = string_count("o", str1 );

Это установит переменную ocount на количество букв "o" в str1, в данном случае 2.
