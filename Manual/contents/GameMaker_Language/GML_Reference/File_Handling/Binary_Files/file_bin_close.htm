file_bin_close

Закрывает ранее открытый двоичный файл. Вы передаете значение идентификатора файла, возвращаемое функцией file_bin_open().

ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5.

Синтаксис:

file_bin_close(binfile);

Аргумент Описание
binfile Идентификатор файла, который нужно закрыть.

Возвращается:

Н/Д

Пример:

file = file_bin_open("myfile.bin", 2);
file_bin_rewrite(file);
file_bin_close(file);

Это позволит открыть файл из того же каталога, что и игра, и присвоить его индекс переменной "file". Затем файл перезаписывается (очищается от данных) и снова закрывается.
