file_bin_open

Эта функция откроет двоичный файл с указанным именем. Режим указывает, что можно сделать с файлом:

  • 0 = чтение
  • 1 = запись
  • 2 = и чтение, и письмо

Если файл не существует, он создается, и функция возвращает id файла, который должен быть использован в других функциях. Вы можете одновременно открывать несколько файлов (не более 32), но не забудьте закрыть их, когда закончите работу с ними.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5.

Синтаксис:

file_bin_open(fname, mode);

Аргумент Описание
fname Имя файла для чтения.
режим индикатор того, что можно сделать с файлом.

Возвращается:

Реально

Пример:

file = file_bin_open("myfile.bin", 2);

Это откроет файл из той же директории, что и игра, и присвоит его индекс переменной "file".
