file_bin_seek

Эта функция перемещает текущую позицию чтения в файле на указанную позицию. Вы передаете значение идентификатора файла, возвращаемое функцией file_bin_open()и для добавления файла переместите позицию на величину размер файла перед записью.

ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5.

Синтаксис:

file_bin_seek(binfile, pos);

Аргумент Описание
binfile Идентификатор файла для чтения.
поз Позиция в файле, на которую следует перейти.

Возвращается:

Н/Д

Пример:

file = file_bin_open("myfile.bin", 2);
size = file_bin_size(file);
file_bin_seek(file, size);

Это позволит открыть файл из той же директории, что и игра, и присвоить его индекс переменной "file", затем получить размер файла и установить следующую позицию записи на этот размер.
