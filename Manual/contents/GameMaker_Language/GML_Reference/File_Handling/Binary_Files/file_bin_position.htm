позиция файла_bin_position

Эта функция возвращает текущую позицию в байтах, где 0 - первая позиция, файла с заданным идентификатором файла. Вы передаете значение идентификатора файла, возвращаемое функцией file_bin_open().

ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5.

Синтаксис:

file_bin_position(binfile);

Аргумент Описание
binfile Идентификатор файла, в котором нужно получить позицию.

Возвращается:

Реально

Пример:

pos = file_bin_position(file);

Это сохранит текущую позицию в переменной "pos".
