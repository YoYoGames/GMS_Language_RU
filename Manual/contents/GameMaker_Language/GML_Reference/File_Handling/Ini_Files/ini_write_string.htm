ini_write_string

Вы можете использовать эту функцию для записи строки (текста) в файл данных ini. Ini файлы разбиваются на секции, а затем каждая секция разбивается на пары ключ - значение. Таким образом, типичный ini файл будет выглядеть примерно так:

Стоит отметить, что использование кавычек и экранирующих символов при записи строк ini может привести к проблемам при последующем при последующем чтении данных, так как они могут быть преждевременно усечены. Например, если написать следующее:

ini_write_string("test2", "section", "hello \"Fritz\");

Синтаксис:

ini_write_string(section, key, value);

Аргумент Описание
раздел Раздел .ini для записи.
ключ Ключ в соответствующем разделе .ini для записи.
значение Строка для записи в соответствующее место назначения.

Возвращается:

Н/Д

Пример:

ini_open("savedata.ini");
ini_write_string("Save", "Player", global.Name);
ini_close();

Приведенный выше код открывает ini-файл для чтения и записи, затем записывает строку, хранящуюся в глобальной переменной "Name", в раздел "Save" с ключом "Player", после чего снова закрывает файл.
