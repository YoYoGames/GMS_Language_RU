ini_read_string

Вы можете использовать эту функцию для чтения строки (текста) из файла данных ini. Ini файлы разбиваются на секции, а затем каждая секция разбивается на пары ключ - значение. Таким образом, типичный ini файл будет выглядеть примерно так:

Синтаксис:

ini_read_string(section, key, default);

Аргумент Описание
раздел Раздел .ini для чтения.
ключ Ключ в соответствующем разделе .ini для чтения.
по умолчанию Строка, которая возвращается, если строка не найдена в определенном месте (или файл .ini не существует). Должна быть строкой.

Возвращается:

Строка

Пример:

ini_open("savedata.ini");
global.name = ini_read_string("player", "name", "Player1");
ini_close();

Это откроет файл "savedata.ini" и установит global.name в строку "save1" > "Name" в нем, затем снова закроет .ini. Если в строке "save1" > "Name" нет строки, или файл "savedata.ini" отсутствует. файл, global.name будет установлен на "Player1".
