ini_close

Эта функция должна быть вызвана в тот момент, когда вы закончили чтение или запись в любой открытый ini-файл. Если вы не используете эту функцию после использования любой из функций записи ini, то на диск ничего не будет записано, так как информация о файле хранится в памяти до тех пор, пока не будет вызвана эта функция. в памяти до вызова этой функции, что приводит к записи. Если вы попытаетесь открыть ini, не закрыв предварительно другой (или тот же самый), вы также получите ошибку.

Эта функция также возвращает строку с закодированным в ней файлом ini. Эту строку можно сохранить на сервере и/или использовать снова вместе с функцией ini_open_from_string() для повторного создания ini.

Синтаксис:

ini_close();

Возвращается:

Строка

Пример:

ini_open("savedata.ini");
score = ini_read_real("save1", "score", 0);
ini_close();

Это откроет файл 'savedata.ini' и установит значение score в разделе "save1" с ключом "score", затем снова закроет .ini. Если в разделе "save1" нет значения "score" или отсутствует файл "savedata.ini", счет будет установлен в 0 и будет создан новый ini_файл.
