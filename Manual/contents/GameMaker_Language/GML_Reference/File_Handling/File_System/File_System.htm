Файловая система

В GameMaker Studio 2 существует ряд функций, которые позволяют манипулировать файлами всех типов различными способами:

  • файл_существует
  • файл_удалить
  • переименование файла
  • копирование_файла
  • файл_найти_первым
  • файл_найти_следующий
  • найти_файл_закрыть
  • атрибуты файла

  • имя_файла
  • имя_файла
  • имя_дира
  • имя_диска
  • имя_текста
  • filename_change_ext

Наконец, есть несколько специальных функций для получения имен файлов, которые можно использовать для загрузки и сохранения файлов с фильтрами типа файла. Эти функции открывают окно проводника и тем самым заставляют игру потерять фокус. Это означает, что любой ввод и поэтому мы рекомендуем (при использовании этих функций в событии клавиатуры) использовать событие " вверх", а не "вниз" или "нажато".

ПРИМЕЧАНИЕ: Эти функции Windows и macOS только.

ВНИМАНИЕ! Эти функции могут работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для дополнительную информацию.

  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
