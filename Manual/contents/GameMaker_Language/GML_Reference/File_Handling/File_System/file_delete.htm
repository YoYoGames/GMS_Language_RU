файл_удалить

Эта функция удалит указанный файл из системы. Следует отметить, что эта функция удалит только те файлы, которые GameMaker Studio 2 может создавать и разбирать: ini файлы, текстовые и бинарные файлы, или файлы, созданные для хранения ресурсов игры, таких как спрайты и поверхности. для хранения ресурсов, созданных в игре, таких как спрайты или поверхности. Однако она не будет удалять никакие другие файлы. Функция также вернет true, если файл был успешно удален, или false при любых других обстоятельствах.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Для получения дополнительной информации обратитесь к разделу " Ограничения файловой системы ".

Синтаксис:

file_delete(fname);

Аргумент Описание
fname Имя файла для удаления.

Возвращается:

Булево

Пример:

if file_exists("level.txt")
{
file_delete("level.txt");
}

Это позволит проверить наличие файла и, если он существует, удалить его.
