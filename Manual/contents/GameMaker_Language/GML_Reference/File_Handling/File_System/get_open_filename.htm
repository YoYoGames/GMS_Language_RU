get_open_filename

Эта функция открывает диалог и запрашивает у игрока имя файла, который нужно открыть с помощью заданного фильтра. Фильтр имеет вид " name1|mask1|name2|mask2|... ", где маска может содержать различные варианты с точкой с запятой между ними и Вы также можете использовать "*" для обозначения любой строки. Например: " bitmaps|*.bmp;*.wmf", попросит пользователя выбрать имя растрового файла из системы, но только тех, которые имеют два указанных расширения (bmp и png). Эта функция не открывает сам файл! Она только возвращает строку с полным именем и путем к файлу. Если пользователь нажимает кнопку "Отмена", возвращается пустая строка "".

При использовании этой функции важно отметить, что она предоставляет вам определенные права доступа к этому файлу на все время игры. Таким образом, получив от пользователя путь к файлу, вы сможете обращаться к нему снова и снова без лишних вопросов. Однако, поскольку эта функция предназначена для чтения файла, предоставленные разрешения предназначены только для чтения, а сохранение не будет разрешено ОС. Если вам потребуется чтение и запись в файл, вы можете получить разрешение с помощью функции get_save_filename().

ПРИМЕЧАНИЕ: Эта функция действует только для целей Windows и macOS.

Синтаксис:

get_open_filename(filter, fname);

Аргумент Описание
фильтр Фильтр типа файла и/или его имени.
fname Предлагаемое имя файла для использования.

Возвращается:

Строка

Пример:

var file;
file = get_open_filename("text file|*.txt", "");
if file != ""
{
file_text_open_read(file);
}

Приведенный выше код попросит пользователя выбрать текстовый файл для открытия, затем проверит имя и путь к файлу, и если он существует, то откроет файл для чтения.
