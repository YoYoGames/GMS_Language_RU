копирование_файла

Эта функция скопирует указанный файл, дав ему новое имя, которое вы выберете. Если вы копируете файл Android из области Included Files (которая будет находиться внутри вашего APK), он будет сначала скопирован в память, поэтому имейте в виду. что копирование очень больших файлов из APK может привести к тому, что на устройствах закончится память. В HTML5-цели мы используем HTML5 Local Storage API для сохранения на стороне клиента, и эта функция не поддерживает операции с двоичными файлами, поэтому предполагается, что вы копирование текстового файла.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Для получения дополнительной информации обратитесь к разделу " Ограничения файловой системы ".

Синтаксис:

file_copy(fname, newname);

Аргумент Описание
fname Имя файла для копирования.
новое имя Имя нового, скопированного файла.

Возвращается:

Н/Д

Пример:

if file_exists("level1.txt")
{
file_copy("level1.txt", "level2.txt");
}

Это позволит проверить наличие файла и, если он существует, скопировать его.
