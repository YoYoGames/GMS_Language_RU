найти_файл_закрыть

Должна быть вызвана после обработки файлов, открытых с помощью file_find_first() и file_find_next() функций для освобождения памяти. Функции поиска файлов открывают дескрипторы в файловый и они занимают минимальное количество памяти, которая (со временем) накапливается. Поэтому вы всегда должны вызывать эту функцию после нахождения нужных вам файлов, чтобы "закрыть" эти дескрипторы.

Синтаксис:

file_find_close();

Возвращает

Н/Д

Пример:

file_find_close();

Это приведет к закрытию и освобождению памяти, связанной с обработчиками функции поиска файлов.
