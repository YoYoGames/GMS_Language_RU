имя_файла

Эта функция возвращает часть пути указанного файла, включая последний обратный слэш.

Синтаксис:

filename_path(fname);

Аргумент Описание
fname Имя файла и путь к нему.

Возвращается:

Строка

Пример:

path = filename_path(working_directory + "Test.ini");

Приведенный выше код получает путь (в виде строки) к указанному файлу.
