файл_найти_первым

Эта функция возвращает имя первого файла, который удовлетворяет маске и атрибутам. Если такого файла не существует, то возвращается пустая строка. Маска может содержать путь и содержать символы подстановки, например, 'C:\temp\*.doc'. задают дополнительные файлы, которые вы хотите увидеть, поэтому обычные файлы всегда возвращаются, если они удовлетворяют маске. Вы можете сложить следующие константы, чтобы увидеть нужный вам тип файлов (если вы не хотите добавлять никаких атрибутов, используйте 0):

Постоянная Описание
fa_readonly Файлы только для чтения
fa_hidden Скрытые файлы
fa_sysfile Системные файлы
fa_volumeid Файлы с идентификатором тома
fa_directory Справочники
fa_archive Архивные файлы

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

ПРИМЕЧАНИЕ: Эта функция вообще не будет работать на HTML5-цели, а флаги фильтра будут работать только на Windows (используйте 0 для всех остальных платформ).

Синтаксис:

file_find_first(mask, attr);

Аргумент Описание
маска Маска, которую следует использовать для поиска.
attr Конкретный атрибут файла, который нужно искать.

Возвращается:

Строка

Пример:

if directory_exists("\User Content")
{
file = file_find_first("/User Content/*.doc", fa_readonly);
}

Он проверит, существует ли указанный каталог, затем, если существует, перейдет в него и вернет первый найденный файл doc "только для чтения".
