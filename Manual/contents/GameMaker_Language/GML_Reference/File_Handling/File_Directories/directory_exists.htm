каталог_существует

Эта функция вернет true, если указанный каталог существует, или false, если его нет. Указанное имя должно включать полный путь, а не относительный, и по умолчанию вы не можете получить доступ к каталогам, находящимся вне пакета игры поскольку все игры находятся в "песочнице" (см. раздел о файловой системе для получения дополнительной информации).

Синтаксис:

directory_exists(dname)

Аргумент Описание
dname Имя каталога для поиска.

Возвращается:

Булево

Пример:

if directory_exists(working_directory + "Saves/")
{
file = file_find_first(working_directory + "Saves/*.doc", fa_readonly);
}

Он проверит, существует ли указанный каталог, затем, если существует, перейдет в него и вернет первый найденный файл doc "только для чтения".
