Файловые директории

При работе с файлами могут возникнуть моменты, когда вам потребуется указать каталог, в котором хранятся файлы, и в GameMaker Studio 2 есть некоторые специальные функции и переменные, относящиеся к различным возможным каталогам, которые могут вам понадобиться:

  • каталог_существует
  • каталог_создать
  • уничтожение каталога
  • каталог temp_directory
  • рабочий_каталог
  • директория программы

ПРИМЕЧАНИЕ: Вы не можете создавать или уничтожать каталоги на HTML5.
