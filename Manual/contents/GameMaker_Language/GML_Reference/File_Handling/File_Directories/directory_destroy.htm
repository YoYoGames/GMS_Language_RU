уничтожение каталога

Эта функция удалит каталог с заданным именем в области сохранения.

ПРИМЕЧАНИЕ: Эта функция не будет работать на HTML5, так как вы не можете создавать или удалять каталоги в локальном хранилище браузера.

ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации.

Синтаксис:

directory_destroy(dname)

Аргумент Описание
dname Имя каталога для удаления.

Возвращается:

Н/Д

Пример:

if directory_exists("DLC")
{
directory_destroy("DLC");
}

При этом проверяется, существует ли указанный каталог в локальной папке данных, и если да, то он удаляется.

Назад: Файловые каталоги
Далее: temp_directory
