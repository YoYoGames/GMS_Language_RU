load_csv

Эта функция загрузит файл формата CSV и преобразует его в сетку DS, возвращая уникальное значение ID для созданной сетки.

Синтаксис:

load_csv(filename)

Аргумент Описание
имя файла Имя файла для открытия (в виде строки)

Возвращается:

Индекс сетки DS (реальный)

Пример:

file_grid = load_csv("spreadsheet.csv");
var ww = ds_grid_width(file_grid);
var hh = ds_grid_height(file_grid);
var xx = 32;
var yy = 32;
for (var i = 0; i < ww; i++;)
{
for (var j = 0; j < hh; j++;)
{
draw_text(xx, yy, string(file_grid[# i, j]));
yy += 32;
}
yy = 32;
xx += 32;
}

Приведенный выше код открывает заданный файл CSV и сохраняет возвращаемую сетку DS в переменной "file_grid". Затем эта сетка будет разобрана в паре фор-лопов, а ее содержимое выведено на экран.
