gif_open

С помощью этой функции вы можете создать пустое изображение формата GIF, готовое к добавлению данных. При вызове функции необходимо указать ширину и высоту GIF (в пикселях, верхнего предела размера нет, за исключением доступной памяти), и функция вернет уникальное значение ID, используемое для идентификации подарка в последующих функциях, или вернет -1, если gif не удалось инициализировать (например, если ширина/высота слишком велики для доступной памяти). Вы также можете указать необязательный аргумент для задания "прозрачного цвета" для GIF. Это значение RGB (без альфа-компонента), и оно очистит GIF до этого цвета, прежде чем в него что-либо будет добавлено. Если вы не укажете аргумент "чистый цвет", будет использоваться цвет по умолчанию будет использоваться черный цвет по умолчанию.

Обратите внимание, что при использовании этой функции вы должны вызвать gif_save() для завершения создания GIF перед открытием другого файла для записи (поэтому для каждого gif_open() должно быть сопутствующее gif_save()).

Синтаксис:

gif_open(width, height, [clear_colour]);

Аргумент Описание
ширина Ширина gif-изображения для создания
высота Высота gif-изображения для создания
[clear_colour] [Необязательно] Цвет (RGB), к которому нужно очистить gif

Возвращается:

Целое число

Пример:

if save_gif == true
{
if count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
else
{
gif_save(gif_image, "GameCapture.gif");
count = 0;
save_gif = false;
}
count++;
}

Приведенный выше код создаст файл изображения GIF с 30 кадрами, взятыми с поверхности приложения, а затем сохранит его.
