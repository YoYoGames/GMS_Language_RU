display_mouse_set

С помощью этой функции вы можете изменить или установить положение мыши на игровом дисплее, что может быть полезно, например, в играх FPS. Функция будет работать только тогда, когда игра находится в фокусе, а использование ALT + Tab разблокирует мышь.

Синтаксис:

display_mouse_set(x, y);

Аргумент Описание
x Координата x на дисплее.
y Координата y на дисплее.

Возвращается:

Н/Д

Пример:

display_mouse_set(display_get_width() / 2, display_get_height() / 2);

Приведенный выше код центрирует мышь на игровом дисплее.
