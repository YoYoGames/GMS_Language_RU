window_set_fullscreen

С помощью этой функции вы можете установить полноэкранный режим окна игры(true) или нет(false). Обратите внимание, что для macOS цели, вы должны снять флажок "Start In Fullscreen" и установить флажок "Allow игроку изменять размер игрового окна" в Опциях игры , иначе эта функция не сработает. Также обратите внимание, что эта функция не будет работать на HTML5, если не добавить в качестве "кликабельного" обратный вызов (подробнее см. здесь ).

Синтаксис:

window_set_fullscreen(full);

Аргумент Описание
полный Устанавливать ли экран в полноэкранный режим (true) или нет (false).

Возвращается:

Н/Д

Пример:

if mouse_check_button_pressed(mb_left)
{
if window_get_fullscreen()
{
window_set_fullscreen(false);
}
else
{
window_set_fullscreen(true);
}
}

Приведенный выше код проверяет нажатие кнопки мыши и затем переводит окно в полноэкранный режим, если оно еще не переведено, или переводит его в оконный режим, если оно уже переведено.
