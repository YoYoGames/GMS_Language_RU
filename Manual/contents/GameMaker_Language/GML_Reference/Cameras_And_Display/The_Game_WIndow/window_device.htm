оконное устройство

Эта функция возвращает текущий указатель устройства d3d, который затем можно (например) передать DLL или Dylib в Windows и macOS.

ПРИМЕЧАНИЕ: Эта функция была упразднена в GameMaker Studio 2 в пользу os_get_info().

Синтаксис:

window_device();

Возвращается:

Указатель

Пример:

gfx_pointer = window_device()

Приведенный выше код будет хранить указатель устройства d3d в переменной.
