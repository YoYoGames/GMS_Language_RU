размер_окна

С помощью этой функции можно изменить размер игрового окна. Если вы используете эту функцию для изменения размера игрового окна, содержимое окна будет масштабироваться в соотношении 1:1 к фоновому холсту, что сохранит четкость и ясность изображения. и четким (хотя масштаб будет изменен, если окно отличается от размера комнаты или вида). Например, если у вас есть окно размером 200 x 150 пикселей, и игрок растягивает окно, перетаскивая границы, конечный размер окна может быть 400 x 300 пикселей, но холст рисунка будет по-прежнему 200 x 150. холст все еще 200 x 150, что дает размытое изображение, поскольку соотношение пикселей теперь 1:2. Однако если вы используете эту функцию для установки размера окна, соотношение 1:1 сохраняется, даже если изображение масштабируется в соответствии с новым размером окна.

Синтаксис:

window_set_size( w, h );

Аргумент Описание
w Новая ширина окна.
h Новая высота окна.

Возвращается:

Н/Д

Пример:

if mouse_check_button_pressed(mb_left)
{
global.scale += 1;
window_set_size(200 * global.scale, 150 * global.scale);
}

Приведенный выше код будет изменять размер окна в зависимости от значения переменной "global.scale" при нажатии кнопки мыши.
