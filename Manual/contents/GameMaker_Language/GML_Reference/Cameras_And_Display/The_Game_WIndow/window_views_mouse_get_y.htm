window_views_mouse_get_y

Эта функция возвращает координату y мыши относительно всех активных представлений и возвращает одно и то же значение мышь_y.

ПРИМЕЧАНИЕ: Об обычных функциях мыши см. раздел " Ввод мыши".

Синтаксис:

window_views_mouse_get_y();

Возвращается:

Реально

Пример:

myy = window_views_mouse_get_y();

Это установит myy в координату y мыши в окне относительно всех активных представлений.
