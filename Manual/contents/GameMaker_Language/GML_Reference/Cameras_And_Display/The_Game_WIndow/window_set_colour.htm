цвет_окна

Эта функция позволяет установить цвет фона игрового окна. Этот цвет представляет собой цвет, который будет использоваться для тех областей игрового окна, которые не заняты никакими видами. Следующее изображение иллюстрирует это:

Приведенное выше изображение имеет два вида с двумя портами просмотра, каждый из которых нарисован в разных позициях. Это растягивает окно игры, чтобы вместить оба порта, и использует цвет окна для закрашивания фона там, где вид не отображается.

Синтаксис:

window_set_colour(color);

Аргумент Описание
цвет Цвет для установки области.

Возвращается:

Н/Д

Пример:

if window_get_colour() != c_black
{
window_set_colour(c_black);
}

Приведенный выше код проверит цвет окна, чтобы увидеть, установлен ли он как черный или нет, и если нет, то установит его черным.
