display_get_orientation

Эта функция возвращает одну из двух констант GameMaker Studio 2, позволяющих определить, в каком режиме находится устройство, на котором запущена игра, - альбомном или портретном (см. таблицу ниже). Обратите внимание, что эта функция может неправильно определить ориентацию устройства при использовании в целевом модуле HTML5. Однако это легко имитируется с помощью следующего сценария:

return (browser_width < browser_height);

Он вернет true для портрета и false для пейзажа.

Синтаксис:

display_get_orientation()

Возвращается:

Постоянная

Постоянная Описание
отображение_ландшафта Устройство удерживается горизонтально, т.е. самый длинный край расположен слева направо, а кнопка меню находится справа.
отображение_ландшафта_перевернутого Как и выше, только теперь кнопка меню находится слева.
отображаемый_портрет Устройство удерживается вертикально, т.е. самый длинный край находится сверху вниз, а кнопка меню - внизу.
display_portrait_flipped Как и выше, только теперь кнопка меню находится вверху.

Пример:

if display_get_orientation() == display_landscape
{
global.Config = 0;
}
else
{
global.Config = 1;
}

Приведенный выше код проверяет ориентацию устройства и устанавливает глобальную переменную в зависимости от возвращенного значения.
