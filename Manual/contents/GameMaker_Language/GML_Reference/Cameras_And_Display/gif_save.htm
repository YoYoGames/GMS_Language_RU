gif_save

С помощью этой функции вы можете сохранить анимацию в формате GIF. Вы передаете индекс GIF (возвращаемый функцией gif_open()), а также имя файла для сохранения. Обратите внимание, что GameMaker Studio 2 не добавляет автоматически не добавляет расширение файла .gif, поэтому вам следует включить его в строку имени файла, если вы хотите, чтобы сохраненный файл был идентифицирован как GIF. Созданный GIF будет палетирован с помощью универсальной палитры 884 (дополнительную информацию см. здесь ).

Обратите внимание, что если функция успешна и GIF сохранен правильно, то она вернет 0, в противном случае она вернет -1.

Синтаксис:

gif_save(gif_index, fname);

Аргумент Описание
gif_index Идентификатор gif для сохранения
fname Имя файла, которое будет использоваться для gif

Возвращается:

Целое число

Пример:

if save_gif == true
{
if count == 0
{
gif_image = gif_open(room_width, room_height);
}
else if count < 30
{
gif_add_surface(gif_image, application_surface, 6/100);
}
else
{
gif_save(gif_image, "GameCapture.gif");
count = 0;
save_gif = false;
}
count++;
}

Приведенный выше код создаст файл изображения GIF с 30 кадрами, взятыми с поверхности приложения, а затем сохранит его.
