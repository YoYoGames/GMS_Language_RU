экран_сохранить_часть

С помощью этой функции вы можете сохранить скриншот части игры в том виде, в котором она сейчас отображается на игровом дисплее. Координаты выделения основаны на абсолютном размере экрана (или окна, если оно не полноэкранное), поэтому (0,0) - это всегда верхний левый верхний угол экрана (или окна). Ниже приведено изображение, показывающее, как работает эта функция:

Файл будет сохранен в рабочем каталоге игры. Важно отметить, что функция предназначена для вызова в событии Draw GUI End Event, что должно дать согласованные результаты на всех платформах. Использование ее в любом другом событии может не сработать, как ожидается, и дать разные результаты на разных платформах и даже устройствах.

ПРИМЕЧАНИЕ: Эта функция не работает на цели HTML5.

Синтаксис:

screen_save_part(fname,x,y,w,h);

Аргумент Описание
fname Имя файла для сохранения снимка экрана
x Координата x левого верхнего угла области экрана для захвата
y Координата y левого верхнего угла области экрана для захвата
w Ширина области экрана для захвата
h Высота области экрана для захвата

Возвращается:

Н/Д

Пример:

if keyboard_check_pressed(ord("S"))
{
screen_save_part("Screens\Screen_" + string(num++) + ".png", 0, 0, 100, 100)
}

Приведенный выше код делает скриншот части игрового экрана размером 100 пикселей в квадрате, когда пользователь нажимает клавишу "S". Этот снимок экрана сохраняется в определенной папке и нумеруется с помощью переменной, которая затем увеличивается.
