view_set_visible

Эта функция может использоваться для установки видимости заданного порта представления. Функция принимает индекс порта представления (от 0 до 7) и булево значение true, если он виден, и false, если нет.

Синтаксис:

view_set_visible(view_port, visible)

Аргумент Описание
вид_порта Порт просмотра, на который будет нацелено представление (0 - 7)
видимый Переключатель видимости(true - видимый, false - невидимый)

Возвращается:

Н/Д

Пример:

if !view_get_visible(0)
{
view_set_visible(0, true);
}

Приведенный выше код проверяет, является ли порт представления[0] видимым, и если нет, то он устанавливается в видимый.
