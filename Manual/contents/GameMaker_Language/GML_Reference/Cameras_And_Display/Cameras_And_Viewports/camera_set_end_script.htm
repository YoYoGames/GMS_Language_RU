camera_set_end_script

Эта функция может быть использована для задания функции сценария, которая будет вызываться в конце каждого игрового кадра, когда камера назначена на видимый и активный порт представления, после того, как все для этого представления камеры было отрисовано. Вы задаете уникальное значение ID камеры (возвращаемое различными camera_create() функции) и ID функции, которая должна быть вызвана. Порядок, в котором функции прикрепляются к камерам и собственно отрисовка вида камеры происходит следующим образом:

  • Камеры для всех видимых и активных портов просмотра имеют свою функцию обновления , вызываемую как
  • Затем для каждого вида:
    • Вызывается функция begin для камеры для этого вида
    • Для этого вида выполняются события рисования
    • Вызывается функция end для камеры
  • Перейдите к следующему активному видимому виду и повторите...

Синтаксис:

camera_set_end_script(camera_id, script)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры.
скрипт Скрипт для запуска каждого игрового кадра

Возвращается:

Н/Д

Пример:

camera_set_end_script(view_camera[0], endCamera);

Приведенный выше код устанавливает конечный сценарий для камеры, назначенной на порт просмотра [0].
