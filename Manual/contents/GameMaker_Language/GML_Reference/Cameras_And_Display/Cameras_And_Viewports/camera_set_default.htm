camera_set_default

Эта функция может быть использована для установки камеры по умолчанию для использования пользовательской камеры, которую вы ранее создали с помощью одной из функций camera_create() функций. Когда вы создаете комнату без активных портов просмотра или камеры обзора, GameMaker Studio 2 все равно использует камеру для отображения действий в игре. Эта камера называется камерой по умолчанию и может быть установлена и управляться (и даже уничтожена) так же, как и любая другая камера.

Синтаксис:

camera_set_default(camera_id)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры

Возвращается:

Н/Д

Пример:

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);
camera_set_default(new_cam);

Приведенный выше код создает новую камеру, а затем назначает ее в качестве используемой по умолчанию камеры.
