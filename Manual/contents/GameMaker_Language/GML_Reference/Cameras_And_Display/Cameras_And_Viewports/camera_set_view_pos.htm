camera_set_view_pos

Вы можете использовать эту функцию для обновления положения вида камеры в помещении. Вы задаете уникальное значение ID камеры (как возвращается различными camera_create() функциями), а затем задаете позиции x и y. для установки камеры.

Синтаксис:

camera_set_view_pos(camera_id, x, y)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры.
x Положение x для размещения вида (в комнате)
y Позиция y для размещения вида (в комнате)

Возвращается:

Н/Д

Пример:

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

Приведенный выше код установит положение камеры просмотра для камеры, назначенной на порт просмотра[0].
