уничтожение камеры

С помощью этой функции вы можете уничтожить любую камеру. При вызове функции вы передаете уникальное значение ID камеры, которое вы получаете из функций camera_create_* или из view_camera массива при использовании Room Editor для настройки порта просмотра и вида. Вы никогда не должны уничтожать камеру, которая в данный момент назначена на видимый вид, если только вы не назначаете новую камеру на этот вид в том же шаге, и вы всегда должны уничтожать любые камеры, которые вы создали с помощью кода, когда их нет. которые вы создали с помощью кода, если они больше не нужны в игре, чтобы предотвратить утечку памяти, а также вы можете уничтожить камеры по умолчанию, если они назначены в редакторе комнат, но вы должны назначить новую камеру на порт вида (или отключить ее), иначе вы получите странные результаты. иначе вы получите странные результаты.

Синтаксис:

camera_destroy(camera_id)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры.

Возвращается:

Real (уникальное значение идентификатора камеры)

Пример:

camera_destroy(view_camera[0]);
view_camera[0] = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1);

Приведенный выше код уничтожает камеру, назначенную в данный момент порту просмотра [0], затем создает новую камеру и назначает ее ID этому порту.
