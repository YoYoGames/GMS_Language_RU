camera_set_view_angle

Вы можете использовать эту функцию для обновления угла обзора камеры в помещении. Вы указываете уникальное значение ID камеры (как возвращается различными camera_create() функциями), а затем указываете угол, который вы на который вы хотите установить угол обзора камеры. Значение по умолчанию равно 0°, положительные значения поворачивают камеру против часовой стрелки, т.е. установка значения 90 повернет камеру на 90° влево.

Синтаксис:

camera_set_view_angle(camera_id, angle)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры
угол Новый угол, на который будет установлен вид камеры

Возвращается:

Н/Д

Пример:

camera_set_view_angle(view_camera[0], point_direction(x, y, mouse_x, mouse_y));

Приведенный выше код установит угол обзора камеры для камеры, назначенной на порт просмотра[0].
