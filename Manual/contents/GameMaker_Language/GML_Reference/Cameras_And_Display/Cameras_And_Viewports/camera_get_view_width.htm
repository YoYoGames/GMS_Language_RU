camera_get_view_width

Эта функция может быть использована для получения ширины (в пикселях) заданной камеры. Обратите внимание, что эта функция действительна только для камер, созданных с помощью функции camera_create_view() или для тех, которые были добавлены в редактора.

Синтаксис:

camera_get_view_width(camera_id)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры

Возвращается:

Реально

Пример:

var vw = camera_get_view_width(view_camera[0]) / 2;
var vh = camera_get_view_height(view_camera[0]) / 2;
camera_set_view_pos(view_camera[0], x - vw, y - vh);

Приведенный выше код извлекает ширину и высоту камеры, назначенной на view port[0], а затем устанавливает ее положение относительно центра.
