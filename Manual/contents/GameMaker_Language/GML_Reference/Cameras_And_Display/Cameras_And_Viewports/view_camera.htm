просмотр_камеры

Этот массив содержит уникальный идентификатор камеры, назначенный данному порту просмотра, и может быть установлен на новую камеру или прочитан для получения текущей камеры, возвращая -1, если камера не назначена. В комнате может быть активно до 8 портов просмотра (значения массива от 0 до 7), и вы можете назначить камеру на любой из них, просто установив эту переменную на значение ID камеры (как возвращается функциями camera_create() или camera_create_view()). Если вы назначили камеру на порт просмотра через редактор комнаты, эта переменная будет содержать идентификатор камеры для этого порта.

Синтаксис:

view_camera[0 ... 7];

Возвращается:

Индекс камеры или -1, если камера не назначена

Пример:

if view_camera[0] == -1
{
view_camera[0] = camera_create_view(0, 0, room_width, room_height);
}

Приведенный выше код проверяет, есть ли в порте просмотра [0] камера, прикрепленная к нему, и если нет, то она создается и устанавливается на него.
