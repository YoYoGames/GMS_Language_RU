view_get_yport

Эта функция может быть использована для получения позиции y заданного порта просмотра.

Синтаксис:

view_get_yport(view_port)

Аргумент Описание
вид_порта Порт просмотра, на который будет нацелено представление (0 - 7)

Возвращается:

Реально

Пример:

if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2)
{
view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2));
}

Приведенный выше код проверит положение y порта просмотра[0], и если оно не там, где нужно, оно будет установлено в это положение.
