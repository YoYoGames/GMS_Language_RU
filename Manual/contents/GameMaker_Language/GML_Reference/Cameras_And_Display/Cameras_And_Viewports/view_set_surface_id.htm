view_set_surface_id

С помощью этой переменной вы можете задать содержимое порта просмотра для рисования на поверхности. При работе с поверхностями часто требуется захватить всю видимую область экрана на поверхность, и поэтому вы назначите ее порту просмотра с помощью эту функцию. Это означает, что все, что отображается в выбранном представлении порта, теперь будет рисоваться на назначенной поверхности, а содержимое этого порта представления больше не будет отображаться, что означает, что вам нужно будет либо:

  • Включите новый вид и рисуйте поверхность только в этом виде (используя вид_текущий для проверки того, какой вид рисуется)
  • Нарисуйте поверхность в событии Draw GUI экземпляра, поскольку слой GUI не зависит от представлений.

При использовании этой функции вы указываете индекс порта вида (от 0 до 7) и индекс поверхности (либо индекс поверхность_приложения либо уникальное значение индекса, возвращаемое функцией поверхность_создать()) или, если поверхность была назначена ранее и вы хотите ее удалить, значение -1. Для получения дополнительной информации об установке порта просмотра на поверхность см. также переменную вид_поверхности_ид.

Синтаксис:

view_set_surface_id(view_port, surf)

Аргумент Описание
вид_порта Порт просмотра, на который будет нацелено представление (0 - 7)

Возвращается:

Реально

Пример:

if view_get_surface_id(0) == -1
{
view_set_surface_id(0, global.vSurf);
}

Приведенный выше код проверит, была ли поверхность назначена порту вида[0], и если нет, то она будет назначена.
