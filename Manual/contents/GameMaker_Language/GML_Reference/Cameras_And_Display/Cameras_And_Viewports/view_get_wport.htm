view_get_wport

Эта функция может быть использована для получения ширины заданного порта просмотра.

Синтаксис:

view_get_wport(view_port)

Аргумент Описание
вид_порта Порт просмотра, на который будет нацелено представление (0 - 7)

Возвращается:

Реально

Пример:

if view_get_wport(0) != (display_get_width () / 2)
{
view_set_wport(0, display_get_width() / 2);
}

Приведенный выше код проверяет ширину порта просмотра[0], и если она не равна половине ширины дисплея, то она устанавливается в это значение.
