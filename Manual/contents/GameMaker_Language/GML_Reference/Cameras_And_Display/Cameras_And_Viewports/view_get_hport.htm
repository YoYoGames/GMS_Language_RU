view_get_hport

Эта функция может быть использована для получения высоты заданного порта просмотра.

Синтаксис:

view_get_hport(view_port)

Аргумент Описание
вид_порта Порт просмотра, на который будет нацелено представление (0 - 7)

Возвращается:

Реально

Пример:

if view_get_hport(0) != (display_get_height () / 2)
{
view_set_hport(0, display_get_height() / 2);
}

Приведенный выше код проверяет высоту порта просмотра[0], и если она не равна половине высоты дисплея, то она устанавливается на это значение.
