camera_get_view_speed_x

Эта функция может использоваться для получения скорости движения заданной камеры вдоль оси x (горизонтальное движение). Возвращаемое значение будет в пикселях на один игровой кадр.

Синтаксис:

camera_get_view_speed_x(camera_id)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры

Возвращается:

Реально

Пример:

var xs = camera_get_view_speed_x(view_camera[0]);
var ys = camera_get_view_speed_y(view_camera[0]);
if xs != 5 || ys != 5
{
camera_set_view_speed(view_camera[0], 5, 5);
}

Приведенный выше код извлекает xspeed и yspeed камеры, назначенной порту view[0], а затем проверяет, совпадают ли они с заданным значением. Если нет, то камера просмотра устанавливается на эту скорость.

Назад: Камеры и порты для просмотра
