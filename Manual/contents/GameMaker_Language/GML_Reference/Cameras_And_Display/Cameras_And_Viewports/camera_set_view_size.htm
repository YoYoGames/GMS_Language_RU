camera_set_view_size

Вы можете использовать эту функцию для обновления размера камеры обзора в помещении. Вы задаете уникальное значение ID камеры (как возвращается различными camera_create() функциями), а затем указываете ширину и высоту (в пикселях) для установки камеры.

Синтаксис:

camera_set_view_size(camera_id, width, height)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры.
ширина Ширина обзора камеры в пикселях
высота Высота вида камеры в пикселях

Возвращается:

Н/Д

Пример:

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

Приведенный выше код установит размер камеры просмотра для камеры, назначенной на порт просмотра[0].
