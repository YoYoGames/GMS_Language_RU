camera_get_proj_mat

Эта функция может быть использована для получения матрицы проекции. Функция возвращает значение идентификатора матрицы, которое затем может быть использовано в других матричных функциях или для установки проекции матрицы другой камеры (с помощью camera_set_proj_mat()).

Синтаксис:

camera_get_proj_mat(camera_id)

Аргумент Описание
идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры

Возвращается:

Идентификатор матрицы (реальный)

Пример:

mat = camera_get_proj_mat(camera_view[0]);

Приведенный выше код получает матрицу проекции для камеры, назначенной на порт просмотра[0].
