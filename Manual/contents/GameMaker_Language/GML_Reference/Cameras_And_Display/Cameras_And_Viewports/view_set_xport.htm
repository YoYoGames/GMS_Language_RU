view_set_xport

Эта функция используется для установки x-положения заданного порта просмотра. Вы указываете индекс порта представления (от 0 до 7) и новое положение, в которое его нужно поместить.

Синтаксис:

view_set_xport(view_port, x)

Аргумент Описание
вид_порта Порт просмотра, на который будет нацелено представление (0 - 7)
x Новое положение x

Возвращается:

Н/Д

Пример:

if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2)
{
view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2));
}

Приведенный выше код проверит положение x порта представления[0], и если он находится не там, где должен быть, он будет установлен в это положение.
