cloud_string_save

Эта функция зафиксирует строку в выбранном облачном сервисе для хранения. Функция возвращает уникальное значение id, которое затем должно быть использовано в соответствующем асинхронном событии для идентификации карты DS, возвращаемой в качестве "обратного вызова". от облачного сервиса. Строка должна содержать всю информацию, которую вам нужно сохранить для вашей игры, поскольку вы можете сохранить только один единственный "блок данных" в облаке, и повторный запуск этой функции перезапишет любые ранее сохраненные значения (как и при использовании cloud_file_save() функция). Описание должно быть короткой строкой информации, описывающей сохранение, например: "Level2, Stage2".

Для получения дополнительной информации о возвращаемых асинхронных данных, пожалуйста, обратитесь к функции cloud_synchronise().

Синтаксис:

cloud_string_save(string, description);

Аргумент Описание
строка Строка данных для загрузки.
описание Краткое описание хранимых данных.

Возвращается:

Реально

Пример:

var t_str = "";
for (var i = 0; i < 10; i++;)
{
t_str += string(global.Highscore[i]) + "|"
}
save_check = cloud_string_save(t_str, "Current Highscores");
var file = file_text_open_write("Highscores.txt");
file_text_write_string(file, t_str);
file_text_close(file);

Приведенный выше код создает строку из значений, хранящихся в глобальном массиве "Highscores", а затем записывает эту строку в облачный сервис для хранения, а также записывает ее в файл для локального хранения.
