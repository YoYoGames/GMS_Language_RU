http_set_request_crossorigin

С помощью этой функции вы можете установить тип кросс-оригинала, который будет использоваться при загрузке изображений из файла. Функция предназначена исключительно для платформы HTML5 и полезна только при загрузке спрайтов из файла с помощью функции sprite_add() функция. Обратите внимание, что при установке значения "use-credentials" вам больше не нужно указывать абсолютный путь к загружаемому спрайту, вместо этого можно указать относительный путь (как показано в примере ниже). По умолчанию тип происхождения кросса установлен на "anonymous".

Синтаксис:

http_set_request_crossorigin(origin_type);

Аргумент Описание
тип происхождения Используемый тип происхождения креста (строка)

Возвращается:

Н/Д

Пример:

if string_lower(http_get_request_crossorigin()) != "use-credentials"
{
http_set_request_crossorigin("use-credentials");
}
sprite_add("sprites/player_outfit_1.png", 0, false, false, 0, 0);

Приведенный выше код сначала проверит установленный в данный момент тип cross origin и если он не "use-credentials", то он будет установлен на "use-credentials", а затем спрайт будет добавлен из файла.
