достижение_загрузки_прогресса

Эта функция отправит запрос на сервер для получения информации обо всех доступных достижениях. Она вызовет обратный вызов Social Asynchronous Event, который содержит async_load карту, заполненную соответствующими парами ключ/значение. Ключ id этой карты DS используется для идентификации правильного обратного вызова (может быть может быть более одной триггерной функции для любого данного асинхронного события), и будет сопряжен с константой achievement_achievement_info, а также рядом других пар ключ/значение для каждого игрока. Точное содержимое карты показано ниже:

ПРИМЕЧАНИЕ: Эта функция предназначена только для iOS только.

  • " id" - Для этой функции это должно быть achievement_achievement_info.
  • " numentries" - Количество достижений в списке.
  • " AchN"- Название достижения, где "N" - целочисленное значение, соответствующее его месту в списке достижений.
  • " AchCompN" - Насколько полно выполнено достижение "N" в процентах от 0 до 100 (строка).

achievement_load_progress()

if os_type == os_ios

Это отправит запрос на получение информации о достижениях и сгенерирует асинхронный обратный вызов со специальной картой async_load DS, содержащей следующие данные:

var ident = ds_map_find_value(async_load, "id");
if ident == achievement_achievement_info
{
var numentries = ds_map_find_value(async_load, "numentries");
for(var i = 0; i < numentries; i++;)
{
ach_id[i] = ds_map_find_value(async_load, "Ach" + string(i));
comp[i] = ds_map_find_value(async_load, "AchComp" + string(i));
}
}

Приведенный выше код проверяет возвращенную DS Map в асинхронном событии Social и, если ее "id" совпадает с проверяемой константой, перебирает записи в карте для каждого из достижений и сохраняет их название и процент выполнения.
