Достижения и таблицы лидеров

Игры, созданные для устройств под управлением iOS или Android, также могут подключаться к одному из двух API для таблиц лидеров и достижений. Для iOS у вас есть iTunes Connect, а для Android вы можете использовать Google Play. они используют функции, найденные на этой странице. Эти функции позволяют вашей игре отправлять информацию о набранных очках и достижениях в таблицы лидеров, которые вы установили в аккаунте разработчиков для создаваемой вами игры, и это отличный способ привлечь игроков к социальной стороне игры, поскольку они могут увидеть, чего достигли другие люди, игравшие в вашу игру, и попытаться превзойти их.

При использовании этих функций следует учитывать, что они не отслеживают достижения и баллы автоматически за вас. Они отображают только те данные, которые вы отправляете им из своей игры, поэтому вы должны написать свой собственный код достижений и для отслеживания состояния отдельных людей в игре, а затем использовать эти функции для добавления данных в выбранную сеть (однако вы можете запросить информацию о прогрессе достижений с помощью функции achievement_load_progress() функцию).

Некоторые из этих функций будут вызывать обратный вызов Social Asynchronous Event, который будет содержать DS Map данных, относящихся к к функции, которая его вызвала. Те функции, которые запускают событие, также детализируют пары ключ/значение карты, возвращаемые обратным вызовом, однако обратный вызов социального события также может быть вызван тем, что ваша игра получает внешних уведомлений о вызовах и т.д..

ПРИМЕЧАНИЕ: Чтобы эти функции работали на iOS, у вас должна быть действующая учетная запись Game Center и идентификатор приложения iOS для вашей игры.

ПРИМЕЧАНИЕ: Чтобы эти функции работали в Android, вы должны отправить свою игру в Google Play и установить необходимые разделы, а также активировать API в опцияхAndroid Game Options для вашей игры. вашей игры.
