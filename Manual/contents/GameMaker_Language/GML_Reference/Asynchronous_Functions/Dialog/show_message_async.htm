show_message_async

Эта функция открывает окно и показывает пользователю сообщение, которое вы задаете в функции. Это асинхронная функция, и поэтому GameMaker Studio 2 не блокирует устройство, на котором она выполняется, в ожидании ответа, а просто продолжает выполнять события в обычном режиме. Как только пользователь нажимает кнопку "Ок", запускается асинхронное событие Диалог запускается асинхронное событие, которое в течение только этого события, будет иметь ds_map, хранящийся в переменной async_load.

Эта карта будет содержать два ключа, "id" и "status", причем "id" будет значением, которое было возвращено функцией при вызове, а "status" будет либо true, если кнопка "Ок" была нажата, либо false, если сообщение было отменено (если это возможно, так как все целевые платформы позволяют отменять диалоги). или false, если сообщение было отменено (где это возможно, так как не все целевые платформы позволяют отменять диалоги).

ПРИМЕЧАНИЕ: Эта функция предназначена только для отладки и тестированияи не должна использоваться в выпущенных играх. Для этих целей вы должны создать пользовательский интерфейс для получения ввода от игроков, чтобы иметь полный контроль над тем, как выглядят и ведут себя диалоги.

Синтаксис:

show_message_async(string);

Аргумент Описание
Строка Сообщение, которое нужно показать пользователю.

Расширенный пример:

Событие нажатия левой кнопки мыши (например) объекта, который показывает сообщение, будет иметь следующий код:

msg = show_message_async("Сейчас вы попадете в магазин");

Вышеописанное покажет сообщение с заданной строкой. Идентификатор сообщения хранится в переменной "msg" и будет использован в асинхронном событии Dialogs, как показано ниже:

var i_d, stat;
i_d = ds_map_find_value(async_load, "id");
if i_d == msg
{
if ds_map_find_value(async_load, "status")
{
url_open("https://play.google.com/store");
}
}

Приведенный выше код сверяет ключ "id" возвращаемой DS Map со значением, хранящимся в переменной "msg". Если они совпадают, то проверяется, была ли нажата кнопка "Ок" (а не закрыто/отменено окно), и если возвращается true, то открывается url. и если возвращается true, то открывается url.
