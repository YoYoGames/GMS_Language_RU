рисовать_свет_определять_направление

Эта функция предназначена для определения направленного света, где направление и интенсивность задаются значениями, введенными для единичного вектора x, y и z (это означает, что они должны быть между -1 и 1, где 1 или -1 - полная интенсивность, а 0 - отсутствие интенсивности). можно задать цвет, который также влияет на воспринимаемую интенсивность света, так как некоторые цвета кажутся "темнее" других. Вы также должны присвоить свету индексный номер, который будет использоваться в других функциях для ссылки на него. на него.

ПРИМЕЧАНИЕ: Имеется только 8 аппаратных светильников, поэтому одновременно могут быть включены только 8 определенных светильников (хотя можно определить и больше).

Синтаксис:

draw_light_define_direction(ind, x, y, z, col);

Аргумент Описание
инд Индексный номер света (произвольный)
x x-компонент светового вектора
y Компонент y светового вектора
z Компонента z светового вектора
col Цвет, который будет использоваться для освещения (либо константа, либо вещественное или шестнадцатеричное значение).

Возвращается:

Н/Д

Пример:

draw_set_lighting(true);
draw_light_define_direction(1, 0, 1, 0, c_white);
draw_light_enable(1, true);

Приведенный выше код включит освещение для всей сцены, затем определит белый направленный свет в пространстве комнаты и, наконец, включит этот свет.
