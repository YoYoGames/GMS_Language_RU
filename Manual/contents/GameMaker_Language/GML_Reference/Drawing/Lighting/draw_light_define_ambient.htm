draw_light_define_ambient

Эта функция используется для управления окружающим светом сцены, то есть светом, который есть в сцене даже без определения точечных или направленных источников света. По сути, это общий цвет и яркость (или темнота) сцены. По умолчанию используется цвет c_black.

Синтаксис:

draw_light_define_ambient(col);

Аргумент Описание
col Цвет, который будет использоваться (либо константа, либо вещественное или шестнадцатеричное значение).

Возвращается:

Н/Д

Пример:

draw_light_define_ambient(c_white);

Приведенный выше код определяет окружающее освещение как белое.
