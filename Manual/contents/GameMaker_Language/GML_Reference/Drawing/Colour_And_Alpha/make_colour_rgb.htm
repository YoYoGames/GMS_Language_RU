make_colour_rgb

GameMaker Studio 2 предоставляет эту функцию (а также другие) для того, чтобы пользователь мог создавать свои собственные цвета. Эта функция состоит из трех компонентов: красного, зеленого и синего компонентов цвета, который вы хотите создать. который вы хотите создать. Синтаксис:

make_colour_rgb(red, green, blue);

Аргумент Описание
красный Красный компонент цвета
зеленый Зеленый компонент цвета
синий Синий компонент цвета

Возвращается:

Int

Пример:

col = make_colour_rgb(100, 145, 255);

Приведенный выше код использует функцию для создания цвета и сохранения его значения в переменной "col" для последующего использования.
