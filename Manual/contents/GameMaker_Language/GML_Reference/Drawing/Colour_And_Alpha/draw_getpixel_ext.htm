draw_getpixel_ext

С помощью этой функции вы можете получить полное abgr 32-битное значение любого пикселя, который рисуется на текущей цели рендеринга. Это означает, что результаты будут зависеть от события, в котором вызывается функция, а также от используемой целевой поверхности. используется.

ПРИМЕЧАНИЕ: Эта функция приведет к значительному снижению производительности, поэтому ее следует использовать только в случае крайней необходимости.

Синтаксис:

draw_getpixel_ext(x, y);

Аргумент Описание
x Координата x пикселя для проверки
y Координата y пикселя для проверки

Возвращается:

Реально

Пример:

col = draw_getpixel_ext(mouse_x, mouse_y);
alpha = (col >> 24) & 255;
blue = (col >> 16) & 255;
green = (col >> 8) & 255;
red = col & 255;

Приведенный выше код получит 32-битное значение цвета в позиции мыши, а затем разделит его на составляющие значения, сохранив их в переменных.
