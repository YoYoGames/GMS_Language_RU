Цвет и альфа

GameMaker Studio 2 имеет обширный набор функций для манипулирования цветом и альфа-каналом рисуемого на экране изображения (о смешивании изображений с использованием специальных режимов наложения и других специфических для PGU функций см. см. раздел о GPU Control. Используя эти функции, вы можете разделять цвета на составные части, создавать новые цвета и манипулировать альфой отрисовки, чтобы изменить, как различные различные объекты в вашей игре рисуются вместе. Помимо этих функций, существует также ряд констант, которые можно использовать в качестве базовых цветов в этих (и других) функциях:

Обратите внимание, что вы также можете создавать цвета из их шестнадцатеричного значения, используя предварительно символ "$", и эти значения также могут быть использованы в цветовых функциях для получения оттенков компонентов или для смешивания и т.д.. Шестнадцатеричное значение для этих цветов разбивается на три части в формате Синий / Зеленый / Красный, поэтому, например, для определения пурпурного цвета можно использовать:

col = $983c95

Где 98 - синий компонент, 3c - зеленый компонент и 95 - красный компонент.

Следующие функции можно использовать для получения компонентных оттенков, составных оттенков, насыщенности и светимости выбранного цвета, а также альфа и других свойств:

  • color_get_blue
  • color_get_green
  • color_get_red
  • цвет_получить_оттенок
  • color_get_saturation
  • color_get_value
  • draw_getpixel
  • draw_getpixel_ext
  • draw_get_colour
  • draw_get_alpha

Эти функции можно использовать для создания цветов из необработанных входных значений цветовых компонентов:

  • make_colour_hsv
  • make_colour_rgb
  • слияние_цветов

Следующие функции можно использовать для настройки различных параметров рисования на экране, включая альфа, цвет и смешивание:

  • draw_clear
  • draw_clear_alpha
  • draw_set_alpha
  • draw_set_colour
