draw_get_colour

Эта функция возвращает текущий цвет рисования, который используется для рисования форм, текста, примитивов и нетекстурированных 3D моделей. Его можно установить с помощью функции draw_set_colour() функция.

Синтаксис:

draw_get_colour()

Возвращается:

Реально

Пример:

if draw_get_colour() != c_black
{
draw_set_colour(c_black);
}

Приведенный выше код сначала проверяет текущий цвет рисования, а затем, если он не установлен в c_black, устанавливает его в c_black.
