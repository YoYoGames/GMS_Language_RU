color_get_red

Эта функция возвращает количество красного цвета, использованного для создания заданного цвета. Значение находится в диапазоне от 0 до 255, где 0 - нет красного, а 255 - весь красный. Следующее изображение иллюстрирует это:

Синтаксис:

color_get_red(col);

Аргумент Описание
col Цвет для проверки

Возвращается:

Int

Пример:

r_comp = color_get_red(c_teal);

Приведенный выше код получит красную составляющую цветовой константы "c_teal" и сохранит ее значение в переменной "r_comp".
