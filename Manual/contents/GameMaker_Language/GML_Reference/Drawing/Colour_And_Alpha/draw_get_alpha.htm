draw_get_alpha

Эта функция возвращает текущее значение альфы рисунка, которое находится в диапазоне от 0 (полностью прозрачный) до 1 (полностью непрозрачный). Альфа рисунка влияет на прозрачность всех функций рисования и может быть установлена с помощью функции draw_set_alpha() функция.

Синтаксис:

draw_get_alpha()

Возвращается:

Реально

Пример:

if draw_get_alpha() != 1
{
draw_set_alpha(1);
}

Приведенный выше код сначала проверяет текущую альфу draw, а затем, если она не установлена в 1, устанавливает ее в 1.
