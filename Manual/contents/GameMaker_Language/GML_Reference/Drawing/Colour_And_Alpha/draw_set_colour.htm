draw_set_colour

С помощью этой функции вы можете установить базовый цвет рисования для игры. Это значение будет влиять на все дальнейшее рисование, где это необходимо, включая шрифты, формы, примитивы и 3D. Если какие-либо из этих активов рисуются с измененным собственным значением цвета, это значение будет проигнорировано.

Синтаксис:

draw_set_colour(col);

Аргумент Описание
col Цвет, который будет установлен для рисования.

Возвращается:

Н/Д

Пример:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Приведенный выше код нарисует текст в указанной позиции с эффектом тени, созданным измененными альфа и цветом рисунка.
