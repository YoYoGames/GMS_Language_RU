draw_set_alpha

С помощью этой функции вы можете установить базовый альфа-фактор рисования для игры. Это значение может быть установлено от 0 до 1, при этом 0 будет полностью прозрачным, а 1 - полностью непрозрачным (значение по умолчанию), и будет влиять на все дальнейшее рисование, включая фоны, спрайты, шрифты, примитивы и 3D.

Обратите внимание, что если вы рисуете любой из активов и передаете аргумент альфа как часть функции (например, при использовании функции draw_sprite_ext()), то значение альфа, заданное здесь, будет отменяется, поэтому при рисовании спрайта с помощью draw_sprite_ext() с аргументом alpha, равным 1, спрайт будет нарисован полностью непрозрачным, независимо от значения alpha, установленного с помощью draw_set_alpha() (это будет работать так же, как при использовании нарисовать_себя() и установке изображение_альфа экземпляра). Также обратите внимание, что когда экземпляр рисует по умолчанию (т.е. у него есть спрайт, но нет события Draw Event), то переменная image_alpha будет привязана к альфе, установленной с помощью draw_set_alpha(), так что image_alpha равная 1 и draw alpha 0.5 нарисует спрайт с альфой 0.5, а image_alpha равная 0.25 и draw alpha 0,5 нарисует спрайт с альфой 0,25.

Синтаксис:

draw_set_alpha(alpha);

Аргумент Описание
альфа Альфа для установки (между 0 и 1)

Возвращается:

Н/Д

Пример:

draw_set_alpha(0.5);
draw_set_colour(c_black);
draw_text(x+5, y+5, "LEVEL 1");
draw_set_alpha(1);
draw_set_colour(c_white);
draw_text(x, y, "LEVEL 1");

Приведенный выше код нарисует текст в указанной позиции с эффектом тени, созданным измененными альфа и цветом рисунка.
