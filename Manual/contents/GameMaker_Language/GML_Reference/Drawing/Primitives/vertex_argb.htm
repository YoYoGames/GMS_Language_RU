вершина_argb

Эта функция устанавливает значения ARGB для вершины, определяемой в данный момент для пользовательского примитива. Вы предоставляете буфер для записи данных, а также значения красного, зеленого, синего и альфа. которые вы хотите использовать в виде одного 32-битного беззнакового целого числа - альфа-образец в старших 8 битах, затем красный образец, зеленый образец и, наконец, синий образец в младших 8 битах. Вы можете использовать шестнадцатеричную нотацию ($AARRGGBB) как вещественное число или использовать любой функции make_colour_*() для определения значения цвета.

Синтаксис:

vertex_argb(buffer, argb);

Аргумент Описание
буфер Буфер для записи информации.
a Альфа-значение для цвета (0 - 255).
r Красная составляющая цвета (0 - 255).
g Зеленая составляющая цвета (0 - 255).
b Синяя составляющая цвета (0 - 255).

Возвращается:

Н/Д

Пример:

vertex_argb(buff, $FFFFFFFF);

Приведенный выше код установит значения ARGB определяемой вершины в белый цвет.
