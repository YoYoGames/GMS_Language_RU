конец_вершины

С помощью этой функции вы завершаете создание пользовательского примитива. После вызова этой команды примитив может быть использован в vertex_submit() функции для использования в шейдере или вы можете заморозить буфер (делая буфер вершин используется только для чтения и намного быстрее) с помощью функции vertex_freeze().

Синтаксис:

vertex_end(buffer);

Аргумент Описание
буфер Буфер, в который нужно закончить запись.

Возвращается:

Н/Д

Пример:

vertex_end(buff);

Приведенный выше код завершает определение пользовательского примитива.
