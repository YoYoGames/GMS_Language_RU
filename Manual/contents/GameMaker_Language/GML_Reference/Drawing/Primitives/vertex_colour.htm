цвет_вершины

Если определенный вами формат вершины принимает значение цвета, вы можете использовать эту функцию для добавления этих данных к вершине, определяемой для текущего примитива. Функция требует буфера для хранения данных и принимает либо цветовую константу, или шестнадцатеричное значение (используя стандартный формат GameMaker Studio 2 BGR, например: $FF0000 для синего цвета), а также значение альфа от 0 (прозрачный) до 1 (полностью непрозрачный).

Синтаксис:

vertex_colour(buffer, color, alpha);

Аргумент Описание
буфер Буфер для записи информации.
цвет Цвет для этой вершины (может быть константой или шестнадцатеричным значением).
альфа Значение альфа для вершины (от 0 до 1).

Возвращается:

Н/Д

Пример:

vertex_colour(b, c_white, 1);

Приведенный выше код установит для определяемой вершины белый цвет с альфа-значением 1.
