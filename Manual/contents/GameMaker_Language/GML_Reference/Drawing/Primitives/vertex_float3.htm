вершина_плоская3

Эта функция добавит три значения с плавающей точкой к данным вершины. Вершина должна быть правильно отформатирована, чтобы принять ее с помощью функции vertex_format_add_custom() функцию.

Синтаксис:

vertex_float3(buffer, float, float, float);

Аргумент Описание
буфер Буфер для записи информации.
float Первое входное значение.
float Второе входное значение.
float Третье входное значение.

Возвращается:

Н/Д

Пример:

vertex_float3(buff, 0.05, 0.01, room_width / x);

Приведенный выше код добавит три значения с плавающей точкой к определяемым данным вершин.
