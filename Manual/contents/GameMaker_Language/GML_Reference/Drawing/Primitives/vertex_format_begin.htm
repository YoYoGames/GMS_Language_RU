vertex_format_begin

Прежде чем вы сможете определить новый формат вершин, вы должны сообщить GameMaker Studio 2, что делаете это с помощью данной функции. Вы должны сначала вызвать эту функцию, затем определить значения формата с помощью соответствующих функций и, наконец, вызвать функцию vertex_format_end() чтобы завершить определение и вернуть "handle" нового формата.

Синтаксис:

vertex_format_begin();

Возвращается:

Н/Д

Пример:

vertex_format_begin();
vertex_format_add_colour();
vertex_format_add_position();
my_format = vertex_format_end();

Приведенный выше код создаст новый формат вершин, содержащий только значения цвета и позиции, а затем сохранит идентификатор формата в переменной "my_format".
