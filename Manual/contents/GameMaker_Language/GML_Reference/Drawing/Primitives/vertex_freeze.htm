вершинная_заморозка

Эта функция может быть использована для "замораживания" вершинного буфера. Этот буфер становится доступным только для чтения, то есть, если вам понадобится изменить его, вам придется удалить весь буфер и создать его заново. Замороженный буфер может быть передан в шейдер быстрее, чем обычный динамический буфер, и рекомендуется для тех эффектов, которые требуют неизменного пользовательского примитива на протяжении всего уровня или игры.

Синтаксис:

vertex_freeze(buffer);

Аргумент Описание
буфер Буфер для замораживания.

Возвращается:

Н/Д

Пример:

vertex_freeze(buff);

Приведенный выше код заморозит заданный буфер вершин.
