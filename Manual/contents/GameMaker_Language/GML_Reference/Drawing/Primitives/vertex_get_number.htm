вершина_получить_число

С помощью этой функции можно узнать количество отдельных вершин, определенных в любом данном вершинном буфере.

Синтаксис:

vertex_get_number(buffer);

Аргумент Описание
буфер Буфер вершин для проверки.

Возвращается:

Реально

Пример:

v_num = vertex_get_number(v_buffer);

Приведенный выше код сохранит в переменной количество вершин, хранящихся в данном вершинном буфере.
