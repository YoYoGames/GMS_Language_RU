draw_primitive_begin_texture

Эта функция должна быть вызвана до того, как вы определите вершины текстурированного примитива. Вы должны указать тип используемого примитива (см. draw_primitive_begin() для дополнительной информации) и идентификатор текстуры, которая может быть спрайтом или активом фонового изображения. Идентификатор актива можно получить из функций sprite_get_texture()например (используйте -1 для отсутствия текстуры).

ПРИМЕЧАНИЕ: Для того чтобы текстура повторялась, ее размер должен быть равен степени двойки, т.е. 32x32, 128x128 и т.д..

Синтаксис:

draw_primitive_begin_texture(kind, tex)

Аргумент Описание
добрый Вид примитива, который вы собираетесь нарисовать.
tex Текстура для использования с примитивом.

Возвращается:

Н/Д

Пример:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 0, 0, 0);
draw_vertex_texture(640, 0, 1, 0);
draw_vertex_texture(0, 480, 0, 1);
draw_vertex_texture(640, 480, 1, 1);
draw_primitive_end();

Приведенный выше код нарисует полосу треугольника с 4 вершинами (образуя прямоугольник), текстурированную текстурой, хранящейся в переменной "tex", и вся текстура будет использована для покрытия готового примитива.
