draw_primitive_begin

Эта функция должна быть вызвана до того, как вы сможете определить какие-либо примитивы. Существует 6 типов примитивов, которые можно определить с помощью следующих констант:

  • pr_pointlist
  • pr_linelist
  • pr_linestrip
  • pr_trianglelist
  • pr_trianglestrip
  • pr_trianglefan

На следующем изображении показано, как они должны выглядеть, а также порядок, в котором следует определять вершины:

Обратите внимание, что на некоторых платформах(Windows, UWP, XBox) тип pr_trianglefan не поддерживается изначально. и поэтому GameMaker Studio 2 выполняет преобразование при компиляции игры, чтобы заставить их работать. Это означает, что на этих платформах тип pr_trianglefan будет использоваться гораздо медленнее, чем другие. Также обратите внимание, что для использования этой функцию на HTML5, необходимо включить WebGL в опциях игры.

Синтаксис:

draw_primitive_begin(kind)

Аргумент Описание
добрый Вид примитива, который вы собираетесь нарисовать.

Возвращается:

Н/Д

Пример:

var _steps = 20;
var _xx = 50;
var _yy = 50;
var _radius = 30;
draw_primitive_begin(pr_trianglefan);
draw_vertex(_xx, _yy);
for(var i = 0; i <= _steps; ++i;)
{
draw_vertex(_xx + lengthdir_x(_radius, 270 * i / _steps), _yy + lengthdir_y(_radius, 270 * i / _steps));
}
draw_primitive_end();

Приведенный выше код нарисует три четверти окружности, созданной из примитивов.
