вершинная_нормаль

Эта функция добавит данные о нормали поверхности к вершине, определяемой в данный момент для пользовательского примитива. Вы предоставляете буфер для записи данных, а также x, y и z компоненты нормали.

Синтаксис:

vertex_normal(buffer, nx, ny, nz);

Аргумент Описание
буфер Буфер для записи информации.
nx Компонента x нормали поверхности.
ny Компонента y нормали поверхности.
nz Компонента z нормали поверхности.

Возвращается:

Н/Д

Пример:

vertex_normal(buff, 0, 1, 1);

Приведенный выше код установит нормаль поверхности определяемой вершины.
