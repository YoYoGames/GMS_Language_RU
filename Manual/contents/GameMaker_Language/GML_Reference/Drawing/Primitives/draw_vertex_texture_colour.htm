draw_vertex_texture_colour

Эта функция определяет положение текстурированной вершины для примитива. Окончательный вид примитива будет зависеть от типа примитива, выбранного для рисования (см. draw_primitive_begin() для получения дополнительной информации), порядка добавления к нему вершин, положения начальной и конечной точек, которые вы задаете для образца текстуры, а также от установленных вами значений цвета и альфа. Для сохранения внешнего вида текстуры при изменении только альфа-функции, значение -1 (или c_white) может быть использовано для аргумента color. Чтобы завершить и нарисовать примитив, необходимо вызвать команду draw_primitive_end().

ПРИМЕЧАНИЕ: Для того чтобы текстура повторялась, ее размер должен быть равен степени двойки, т.е. 32x32, 128x128 и т.д..

Синтаксис:

draw_vertex_texture_colour(x, y, xtex, ytex, col, alpha)

Аргумент Описание
x Координата x вершины.
y Координата y вершины.
xtex Координата x в текстуре.
ytex Координата y в текстуре.
col Цвет для смешивания с текстурой в данной вершине (-1 или c_white для отсутствия смешивания).
альфа Альфа, с которой будет рисоваться эта вершина (0-1).

Возвращается:

Н/Д

Пример:

draw_set_colour(c_white);
var tex = sprite_get_texture(spr_Background, 0);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture_colour(0, 0, 0, 0, c_fuchsia, 1);
draw_vertex_texture_colour(640, 0, 1, 0, c_yellow, 1);
draw_vertex_texture_colour(0, 480, 0, 1, c_aqua, 1);
draw_vertex_texture_colour(640, 480, 1, 1, c_lime, 1);
draw_primitive_end();

Приведенный выше код нарисует полосу треугольника с 4 вершинами (образуя прямоугольник), текстурированную текстурой, хранящейся в переменной "tex", и вся текстура будет использована для покрытия готового примитива, и она будет смешана с четырьмя различными цветами.
