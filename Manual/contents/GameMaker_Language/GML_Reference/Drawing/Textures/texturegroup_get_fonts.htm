texturegroup_get_fonts

С помощью этой функции вы можете получить индекс шрифта каждого из шрифтов, назначенных текстурным страницам в заданной группе текстур. Вы передаете строку ID текстурной группы (как определено в редакторе текстурных групп), и функция возвращает одномерный массив где каждая запись содержит индекс шрифта для ресурса шрифта. Если функция не работает - т.е. задана неверная группа, или группа не имеет назначенной ей текстуры - то массив будет пуст (длина 0).

Синтаксис:

texturegroup_get_fonts(tex_id);

Аргумент Описание
tex Имя группы текстур для проверки (строка).

Возвращается:

одномерный массив

Пример:

var _tex_array = texturegroup_get_fonts("MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
show_debug_message("Font " + string(i) + " Index:" + string(tex_array[i]));
}

Приведенный выше код получит индексы шрифтов для группы текстур "MainMenu", а затем отобразит эти идентификаторы в окне консольного вывода.
