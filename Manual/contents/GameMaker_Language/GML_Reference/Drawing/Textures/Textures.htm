Текстуры

Следующие функции связаны с "текстурами", т.е. изображениями, которые хранятся в памяти VRAM для использования в качестве спрайтов или других элементов в вашей игре. Первый набор функций используются для получения информации о конкретных изображениях, их положении или размере на странице текстур, а также для установки определенных характеристик текстур:

  • texture_get_width
  • texture_get_height
  • texture_get_texel_width
  • texture_get_texel_height
  • texture_get_uvs
  • texture_set_stage
  • текстурный_глобальный_масштаб

GameMaker Studio 2 также имеет ряд функций, связанных с управлением VRAM, связанным как с отдельными текстурами, так и с целыми группами текстур:

  • draw_texture_flush
  • текстурная_выборка
  • текстурная_промывка
  • sprite_flush
  • sprite_flush_multi
  • sprite_prefetch
  • sprite_prefetch_multi
  • draw_flush

Наконец, у нас есть следующие функции, которые в основном используются для отладки проекта и обеспечения эффективного использования текстурной памяти:

  • texture_debug_messages
  • texture_is_ready
  • texturegroup_get_textures
  • texturegroup_get_sprites
  • texturegroup_get_fonts
  • texturegroup_get_tilesets
