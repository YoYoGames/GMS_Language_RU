texturegroup_get_textures

Эта функция может быть использована для получения идентификаторов текстурных страниц отдельных страниц, составляющих группу текстур. Вы передаете строку ID текстурной группы (как определено в редакторе Texture Group Editor), и функция возвращает одномерный массив, где каждый элемент запись в массиве - это ID одной текстурной страницы. Если функция не работает - т.е. указана неверная группа, или группа не имеет назначенной ей текстуры - массив будет пуст (длина 0).

Синтаксис:

texturegroup_get_textures(tex_id);

Аргумент Описание
tex_id Имя группы текстур для проверки (строка).

Возвращается:

одномерный массив

Пример:

var _tex_array = texturegroup_get_textures("MainMenu");
for (var i = 0; i < array_length(_tex_array); ++i;)
{
if texture_is_ready(_tex_array[i])
{
texture_prefetch(_tex_array[i]);
}
}

Приведенный выше код получит идентификаторы текстурных страниц для группы текстур "MainMenu", затем проверит, распакованы ли они, и если да, то поместит их в VRAM.
