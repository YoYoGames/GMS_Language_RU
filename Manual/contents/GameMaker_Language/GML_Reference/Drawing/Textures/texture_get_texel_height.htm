texture_get_texel_height

Тексель, или элемент текстуры, - это фундаментальная единица текстурного пространства, используемая в компьютерной графике. Текстуры представляются массивами текселей, так же как изображения представляются массивами пикселей, и эта функция возвращает высоту одного Тексель из текстурной страницы используемого актива изображения.

Синтаксис:

texture_get_texel_height(tex);

Аргумент Описание
tex Указатель актива страницы текстуры для использования

Возвращается:

Реально

Пример:

var tex = sprite_get_texture(sprite_index, 0);
tex_w = texture_get_texel_width(tex);
tex_h = texture_get_texel_height(tex);

Приведенный выше код получит ширину и высоту текселя текстуры, взятой из актива спрайта.
