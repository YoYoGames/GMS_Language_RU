draw_surface_stretched_ext

Эта функция делает то же самое, что и draw_surface_stretched() с добавлением возможности установить смешивание цветов и значение альфа для поверхности при ее рисовании (аналогично функции draw_surface_ext()).

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

draw_surface_stretched_ext(id, x, y, w, h, col, alpha);

Аргумент Описание
id Уникальное значение ID поверхности для рисования.
x Позиция x в месте рисования поверхности.
y Положение по оси y места рисования поверхности.
w Ширина, на которой следует рисовать поверхность.
h Высота, на которой следует рисовать поверхность.
цвет Цвет, которым будет окрашена поверхность.
альфа Альфа, с помощью которой нужно смешивать поверхность.

Возвращается:

Н/Д

Пример:

draw_surface_stretched_ext(surf, x, y, 200, 200, c_white, 0.5);

Это нарисует заданную поверхность с левым углом в позиции экземпляра x/y, и она будет растянута на область 200x200 пикселей без наложения, но с частичной прозрачностью.
