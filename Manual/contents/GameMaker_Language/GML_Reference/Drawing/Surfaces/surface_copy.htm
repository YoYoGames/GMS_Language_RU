поверхностное_копирование

Эта функция просто берет изображение с одной поверхности и копирует его на другую поверхность в указанной локальной позиции внутри этой поверхности (где позиция (0,0) - левый верхний угол поверхности назначения). Если на поверхности назначения уже уже есть информация, она будет перезаписана копией, и функция никак не изменяет исходную поверхность.

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_copy(destination, x, y, source);

Аргумент Описание
направление Идентификатор поверхности, на которую нужно скопировать другую поверхность.
x Позиция x для копирования.
y Позиция y для копирования.
источник Идентификатор поверхности, которую нужно скопировать.

Возвращается:

Н/Д

Пример:

if view_current == 0
{
surface_copy(surf, 0, 0, 0, temp_surf);
}
else
{
draw_surface(surf, 0, 0);
}

Приведенный выше код проверяет текущий вид, и если это вид[0], то копирует поверхность, указанную в переменной "temp_surf", на поверхность, указанную в переменной "surf". Если текущий вид не является видом[0] то на экран выводится поверхность "surf".
