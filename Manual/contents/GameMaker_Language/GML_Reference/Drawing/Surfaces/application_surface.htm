поверхность_приложения

Эта встроенная переменная глобального масштаба может использоваться для доступа к поверхности приложения с помощью любой из функций поверхности. Эта поверхность постоянно доступна, и именно на ней выполняется основная часть рисования в GameMaker Studio 2.

Синтаксис:

поверхность_приложения;

Возвращается:

Указатель

Пример:

surface_resize(application_surface, display_get_gui_width(), display_get_gui_height())

Приведенный выше код изменит размер поверхности приложения так, чтобы соотношение с слоем GUI было 1:1.
