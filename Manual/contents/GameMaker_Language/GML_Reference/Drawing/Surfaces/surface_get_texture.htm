поверхность_get_текстура

Эта функция возвращает специальный указатель на текстуру поверхности. Это значение затем может быть использовано в других функциях рисования, в частности, в общих функциях 3D и некоторых функциях 2D-примитивов.

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

surface_get_texture(surface_id);

Аргумент Описание
идентификатор поверхности Идентификатор поверхности для получения ширины.

Возвращается:

Указатель

Пример:

var tex;
tex = surface_get_texture(surf);
draw_primitive_begin_texture(pr_trianglestrip, tex);
draw_vertex_texture(0, 480, 0, 0);
draw_vertex_texture(640, 480, 1, 0);
draw_vertex_texture(640, 480, 1, 1);
draw_vertex_texture(0, 480, 0, 1);
draw_primitive_end();

Приведенный выше код нарисует полосу треугольника с 4 вершинами, текстурированную текстурой, хранящейся в переменной "tex", которая сама берется из ранее созданной поверхности.
