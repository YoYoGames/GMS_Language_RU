поверхность_сохранения

Эта функция сохранит поверхность на диск, используя заданное имя файла. Поверхность должна быть сохранена как файл формата *.png.

Синтаксис:

surface_save(surface_id, fname);

Аргумент Описание
идентификатор поверхности Идентификатор поверхности, которую нужно установить в качестве цели рисования.
fname Имя сохраненного файла изображения.

Возвращается:

Н/Д

Пример:

if keyboard_check_pressed(ord("S")
{
surface_save(surf, "test.png");
}

Приведенный выше код проверит, нажимает ли пользователь клавишу "S" на клавиатуре, и если нажимает, то сохранит поверхность, проиндексированную в переменной "surf", на диск.
