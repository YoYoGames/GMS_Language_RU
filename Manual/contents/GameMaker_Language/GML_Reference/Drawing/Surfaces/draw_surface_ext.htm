нарисовать_поверхность_экст

Эта функция рисует заданную поверхность, как в функции нарисовать_поверхность() но с дополнительными опциями для изменения масштаба, смешивания, вращения и альфа-функции рисуемой поверхности. Изменение этих значений никак не изменяет ресурс (только то, как он рисуется).

ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней.

Синтаксис:

draw_surface_ext(id, x, y, xscale, yscale, rot, col, alpha);

Аргумент Описание
id Уникальное значение ID поверхности для рисования.
x Позиция x в месте рисования поверхности.
y Положение по оси y места рисования поверхности.
xscale Горизонтальная шкала.
yscale Вертикальная шкала.
гниение Поворот или угол для рисования поверхности.
col Цвет, которым будет смешиваться поверхность.
альфа Альфа-прозрачность для рисования поверхности.

Возвращается:

Н/Д

Пример:

draw_surface_ext(surf, 0, 0, 0, 2, 2, 0, c_red, 0.5);

Приведенный выше код нарисует поверхность, индексированную в переменной "surf" в позиции (0,0) в комнате, с удвоенным исходным масштабом, с красной и полупрозрачной смесью.
