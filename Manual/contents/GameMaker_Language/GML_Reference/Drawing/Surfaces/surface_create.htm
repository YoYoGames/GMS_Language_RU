поверхность_создать

Эта функция используется для создания поверхности и возвращает индекс поверхности, который должен быть сохранен в переменной для последующих вызовов функции. Когда поверхность создается впервые, она может содержать "шум", так как по сути это просто область памяти, отведенная для этой цели (и эта память все еще может содержать информацию), поэтому вы можете захотеть очистить поверхность перед использованием с функцией типа draw_clear_alpha().

Настоятельно рекомендуется создавать все поверхности с размером, равным степени 2, то есть, например, 16, 128, 512 или 1024 пикселей. Это не совсем обязательно на некоторых платформах (например, Windows и MacOS), но это, безусловно, повысит совместимость на этих платформах. совместимость на этих платформах, в то время как для HTML5 и устройств это необходимо, и очень важно, чтобы вы помнили об этом, иначе у вас могут возникнуть проблемы.

Синтаксис:

surface_create(w, h);

Аргумент Описание
w Ширина создаваемой поверхности.
h Высота создаваемой поверхности.

Возвращается:

Реально

Пример:

if !surface_exists(surf)
{
surf = surface_create(1024, 1024);
surface_set_target(surf);
draw_clear_alpha(c_black, 0);
surface_reset_target();
view_surface_id[0] = surf;
}

Приведенный выше код проверяет, существует ли поверхность, и если нет, то создает поверхность шириной 1024 пикселя и высотой 1024 пикселя, присваивая индекс переменной "surf". Затем цель рисования устанавливается на новую поверхность, которая очищается и становится прозрачной, после чего цель рисунка сбрасывается на экран. Наконец, поверхности присваивается вид.
