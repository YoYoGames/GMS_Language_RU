draw_enable_drawevent

С помощью этой функции вы можете включить(true) или выключить(false) событие рисования для всех экземпляров в игре, таким образом предоставляя вам контроль над тем, как и когда происходит рисование. "пропуск кадров". Обратите внимание, что это не просто предотвращает отрисовку экземпляров на экране, а полностью подавляет событие draw, что означает, что следует быть осторожным, поскольку любая игровая логика, присутствующая в этом событии, не будет запущена. также. Важно понимать, что если вы вызовете эту функцию в самом начале игры, до того, как будет отрисован начальный кадр (т.е. событие Create Event первого объекта в первой комнате игры), то окно игры не будет отрисовано. Это может быть полезно для онлайн-проектов, где требуется сервер, которому не нужно ничего рендерить.

Синтаксис:

draw_enable_drawevent(enable);

Аргумент Описание
Включить Установите значение true или false.

Возвращается:

Н/Д

Пример:

frame_skip ++;
if (frame_skip mod 5) == 0
{
draw_enable_drawevent(true);
}
else
{
draw_enable_drawevent(false);
}

Приведенный выше код проверяет переменную и, если возвращает true, включает событие draw, в противном случае событие draw отключается.
