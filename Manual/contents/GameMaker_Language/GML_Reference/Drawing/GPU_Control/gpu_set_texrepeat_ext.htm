gpu_set_texrepeat_ext

Эта функция может быть использована для установки того, повторяет ли один "слот" сэмплера заданную текстуру при использовании Шейдеры в GameMaker Studio 2. Если установить значение true, то текстура будет повторяться текстуру, если uv-координаты находятся вне диапазона 0-1, в то время как значение false означает отсутствие повторения. Скорее всего, эти функции используются для повторения текстуры в 3D, но для того, чтобы они работали и не перетягивали изображения из остальной страницы текстуры, используемый спрайт должен быть помечен как находящийся на "отдельной странице текстуры" в редакторе спрайтов.

ПРИМЕЧАНИЕ: Эта настройка будет отменена значением, установленным при вызове функции gpu_set_texrepeat().

Синтаксис:

gpu_set_texrepeat_ext(sampler_id, enable);

Аргумент Описание
sampler_id Идентификатор сэмплера из шейдера.
включить Включить или отключить фильтрацию текстур(true / false)

Возвращается:

Н/Д

Пример:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
else
{
gpu_set_texfilter_ext(s_tex, true);
}

Приведенный выше код проверяет, включена или выключена фильтрация текстур для определенного ID сэмплера (хранится в локальной переменной), и переключает ее соответствующим образом.
