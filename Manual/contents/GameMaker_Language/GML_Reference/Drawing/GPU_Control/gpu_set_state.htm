gpu_set_state

Эта функция устанавливает текущее состояние GPU, используя переданную карту DS Map. Переданная карта может быть создана с помощью функции gpu_get_state().

Синтаксис:

gpu_set_state(ds_map);

Аргумент Описание
ds_map Состояние GPU для установки в качестве DS Map.

Возвращается:

Н/Д

Пример:

gpu_set_state(gpu_map);

Приведенный выше код устанавливает состояние GPU, используя карту, заданную в переменной "gpu_map".
