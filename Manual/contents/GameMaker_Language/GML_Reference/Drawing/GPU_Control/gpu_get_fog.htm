gpu_get_fog

Эта функция может быть использована для получения настроек тумана. Функция возвращает 4-элементный массив 1D со следующей информацией:

  • [0] = включенный тумблер (булево значение, либо истина, либо ложь), по умолчанию false
  • [1] = цвет (реальный), по умолчанию c_black
  • [2] = начальное расстояние (реальное), по умолчанию 0
  • [3] = конечное расстояние (реальное), по умолчанию 1

Обратите внимание, что вы можете изменить эти значения и передать полный массив в gpu_set_fog() функции в качестве одного аргумента.

Синтаксис:

gpu_get_fog();

Возвращается:

4-элементный одномерный массив (элементы массива см. выше)

Пример:

var fog_a = gpu_get_fog();
fog_a[1] = c_red;
gpu_set_fog(fog_a);

Приведенный выше код получает текущие настройки тумана, затем устанавливает элемент color массива в c_red, после чего снова устанавливает туман, используя измененный массив.
