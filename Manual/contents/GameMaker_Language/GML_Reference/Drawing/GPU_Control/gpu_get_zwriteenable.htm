gpu_get_zwriteenable

Эта функция может использоваться для получения информации о том, включена ли функция z-writing (функция возвращает true) или нет (функция возвращает false). По умолчанию значение z-writing включено, поэтому функция возвращает true.

Синтаксис:

gpu_get_zwriteenable();

Возвращается:

Булево

Пример:

if gpu_get_zwriteenable() == false
{
gpu_set_zwriteenable(true);
}

Приведенный выше код проверяет, включена или нет z-запись, и если она отключена, то снова включается.
