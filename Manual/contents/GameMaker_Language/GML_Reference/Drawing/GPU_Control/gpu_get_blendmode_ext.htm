gpu_get_blendmode_ext

Эта функция может использоваться для получения текущего расширенного режима наложения, используемого для рисования. Функция возвращает двухэлементный массив 1D со следующими элементами:

  • [0] = Режим наложения источника (по умолчанию bm_src_alpha)
  • [1] = Режим наложения назначения (по умолчанию bm_inv_src_alpha)

Значениями массива будут одни из следующих констант ("s" обозначает значение, взятое из источника, а "d" - значение из пункта назначения):

Постоянная Коэффициент смешивания (красный, зеленый, синий, альфа)
bm_zero (0, 0, 0, 0)
bm_one (1, 1, 1, 1)
bm_src_colour (Rs, Gs, Bs, As)
bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As)
bm_src_alpha (As, As, As, As, As)
bm_inv_src_alpha (1-Ас, 1-Ас, 1-Ас, 1-Ас, 1-Ас)
bm_dest_alpha (Ad, Ad, Ad, Ad, Ad)
bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad)
bm_dest_colour (Rd, Gd, Bd, Ad)
bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad)
bm_src_alpha_sat (f, f, f, 1), где f = min(As, 1-Ad)

Обратите внимание, что вы можете изменить эти значения и передать полный массив в gpu_set_blendmode_ext() функции в качестве одного аргумента.

Синтаксис:

gpu_get_blendmode_ext();

Возвращается:

Двухэлементный одномерный массив (константы см. выше)

Пример:

var bm = gpu_get_blendmode_ext();
bm[0] = bm_src_alpha;
gpu_set_blendmode_ext(bm);

Приведенный выше код получает текущий расширенный режим наложения, изменяет источник, а затем снова устанавливает расширенный режим наложения.
