gpu_get_zfunc

Эта функция может быть использована для получения режима сравнения z. Возвращаемое значение будет одной из следующих констант (значение по умолчанию - cmpfunc_lessequal):

  • cmpfunc_never
  • cmpfunc_less
  • cmpfunc_equal
  • cmpfunc_lessequal
  • cmpfunc_greater
  • cmpfunc_notequal
  • cmpfunc_greaterequal
  • cmpfunc_always

Синтаксис:

gpu_get_zfunc();

Возвращается:

Постоянная (см. таблицу выше)

Пример:

if gpu_get_zfunc() != cmpfunc_greater
{
gpu_set_zfunc(cmpfunc_greater);
}

Приведенный выше код проверяет, установлен ли метод z-тестирования на cmpfunc_greater, и если нет, то он устанавливается на эту константу.
