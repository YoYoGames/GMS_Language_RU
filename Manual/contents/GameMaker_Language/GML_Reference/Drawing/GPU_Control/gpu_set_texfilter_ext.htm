gpu_set_texfilter_ext

Эта функция может использоваться для установки линейной интерполяции для одного "слота" сэмплера при использовании Шейдеры в GameMaker Studio 2. Если эта функция включена(true), то текстура сэмплера текстура будет сглажена, а если эта функция отключена(false), то изображения будут рисоваться на основе ближайшего пикселя. По умолчанию используется значение, установленное в глобальных игровых параметрах вашей игры, или значение, установленное с помощью функции gpu_set_texfilter().

ПРИМЕЧАНИЕ: Эта настройка будет отменена значением, установленным при вызове функции gpu_set_texfilter().

ПРИМЕЧАНИЕ: На цели HTML5 эта функция будет работать только при включенном WebGL.

Синтаксис:

gpu_set_texfilter_ext(sampler_id, enable);

Аргумент Описание
sampler_id Идентификатор сэмплера из шейдера.
включить Включить или отключить фильтрацию текстур(true / false)

Возвращается:

Н/Д

Пример:

var s_tex = shader_get_sampler_index(shader_glass, "s_NoiseSampler");
if gpu_get_texfilter_ext(s_tex)
{
gpu_set_texfilter_ext(s_tex, false);
}
else
{
gpu_set_texfilter_ext(s_tex, true);
}

Приведенный выше код проверяет, включена или выключена фильтрация текстур для определенного ID сэмплера (хранится в локальной переменной), и переключает ее соответствующим образом.
