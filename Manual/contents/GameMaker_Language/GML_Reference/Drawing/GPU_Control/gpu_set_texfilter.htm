gpu_set_texfilter

Эта функция может использоваться для настройки фильтрации текстур (линейной интерполяции) всех изображений, рисуемых на игровом экране. Когда эта функция включена(true), все текстуры будут сглажены при отрисовке (это касается и спрайтов, так как они тоже считаются текстурами), Это означает, что при масштабировании или перемещении, если соотношение пикселей не 1:1, будет наблюдаться "размазывание" по различным пикселям, что может сделать изображение размытым в зависимости от используемого художественного стиля. Если этот параметр отключен(false), то изображения будут отрисовываться по ближайшему пикселю при масштабировании или перемещении, что может привести к появлению "блочных" изображений. Значение по умолчанию - false, его также можно изменить в глобальных параметрах игры для отдельных целевых платформ.

ПРИМЕЧАНИЕ:Эта настройка отменяет любую интерполяцию текстурной стадии, установленную для шейдеров с помощью функции gpu_set_texfilter_ext().

ПРИМЕЧАНИЕ: На цели HTML5 эта функция будет работать только при включенном WebGL.

Синтаксис:

gpu_set_texfilter(enable);

Аргумент Описание
включить Включить или отключить фильтрацию текстур(true / false)

Возвращается:

Н/Д

Пример:

if gpu_get_texfilter()
{
gpu_set_texfilter(false);
}
else
{
gpu_set_texfilter(true);
}

Приведенный выше код проверяет, включена или выключена фильтрация текстур, и переключает ее соответствующим образом.
