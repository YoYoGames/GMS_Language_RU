Mipmapping

В этом разделе содержатся все функции, связанные с использованием функций mipmapping. Перед использованием этих функций вы должны включить mipmapping для необходимых текстурных страниц в Texture Group Editor и/или включить mipmapping для тех спрайтов, которые были настроены на использование только одной текстурной страницы в General Game Options. На изображении ниже показана разница, которую может внести mipmapping при рендеринге вашего проекта:

После того как вы включили мипмаппинг для проекта или страницы текстуры, вы можете использовать следующие функции для получения и установки значений, которые изменят внешний вид проекта при запуске.

Перечисленные ниже функции можно использовать для получения различных значений, связанных с мипмаппингом в GameMaker Studio 2:

  • gpu_get_tex_mip_filter
  • gpu_get_tex_mip_filter_ext
  • gpu_get_tex_mip_bias
  • gpu_get_tex_mip_bias_ext
  • gpu_get_tex_min_mip
  • gpu_get_tex_min_mip_ext
  • gpu_get_tex_max_mip
  • gpu_get_tex_max_mip_ext
  • gpu_get_tex_max_aniso
  • gpu_get_tex_max_aniso_ext
  • gpu_get_tex_mip_enable
  • gpu_get_tex_mip_enable_ext

Перечисленные ниже функции можно использовать для установки различных значений, связанных с mipmapping в GameMaker Studio 2:

  • gpu_set_tex_mip_filter
  • gpu_set_tex_mip_filter_ext
  • gpu_set_tex_mip_bias
  • gpu_set_tex_mip_bias_ext
  • gpu_set_tex_min_mip
  • gpu_set_tex_min_mip_ext
  • gpu_set_tex_max_mip
  • gpu_set_tex_max_mip_ext
  • gpu_set_tex_max_aniso
  • gpu_set_tex_max_aniso_ext
  • gpu_set_tex_mip_enable
  • gpu_set_tex_mip_enable_ext
