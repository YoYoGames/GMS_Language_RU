gpu_set_tex_mip_enable

С помощью этой функции можно изменить, будет ли мипмаппинг выключен, включен для всего или включен только для групп текстур, выбранных в менеджере групп текстур. Для работы функции требуется одна из перечисленных ниже констант.

Постоянная Описание
mip_off Mipmapping отключен.
mip_on Mipmapping для всех текстур включен.
mip_markedonly Mipmapping включен для текстур, у которых он включен в опциях группы текстур (по умолчанию).

Синтаксис:

gpu_set_tex_mip_enable(setting);

Аргумент Описание
настройка Настройка mipmap (константа, по умолчанию: mip_markedonly).

Возвращается:

Н/Д

Пример:

if gpu_get_tex_mip_enable != mip_on
{
gpu_set_tex_mip_enable(mip_on);
}

Приведенный выше код проверит, включен ли mipmapping, и если нет, то включит его.
