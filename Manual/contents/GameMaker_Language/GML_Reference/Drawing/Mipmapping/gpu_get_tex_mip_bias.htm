gpu_get_tex_mip_bias

С помощью этой функции можно получить значение смещения мипмапы, где значение 0 означает отсутствие смещения, 1 - первая мипмапа, 2 - вторая мипмапа и т.д.. Это значение контролирует скорость, с которой меняется карта мипов, и обычно делает рендеринг текстур тем более размытыми, чем выше значение и чем больше "расстояние", на котором происходит просмотр. Обратите внимание, что этот параметр может возвращать и отрицательные значения, в этом случае отрисованные текстуры будут более четкими на большем расстоянии, чем меньше значение.

Синтаксис:

gpu_get_tex_mip_bias();

Возвращается:

Реальный (по умолчанию: 0)

Пример:

if gpu_get_tex_mip_bias() != 0
{
gpu_set_tex_mip_bias(0);
}

Приведенный выше код проверяет текущее смещение mipmap и если оно не равно 0, то устанавливается на 0.
