эффект_создания_нижнего

С помощью этой функции вы можете создать простой эффект под всеми экземплярами вашей комнаты (на самом деле он создается на глубине 100000). Если эффект не является ef_rain или ef_snow, вы можете определить положение x/y для создания эффекта. а размер может быть равен 0, 1 или 2, где 0 - маленький, 1 - средний и 2 - большой.

Стоит отметить, что эти эффекты можно включать и выключать, а также приостанавливать их отрисовку с помощью функций part_system_automatic_draw() и part_system_automatic_update() с соответствующим значением индекса системы частиц (где 0 - для эффектов ниже, а 1 - для эффектов выше).

Доступными константами для различных видов частиц являются:

Постоянная Пример Описание
эф_облако Эффект, создающий случайные облачные частицы разного размера
эф_эллипс Эффект, создающий расширяющиеся эллипсы
ef_explosion Эффект, создающий расширяющиеся затухающие взрывы
эф_фейерверк Эффект, создающий множество мелких частиц для создания взрыва фейерверка
эф_блик Эффект, создающий блестящую точку, которая вспыхивает и затухает
дождь Эффект, создающий частицы дождя, спускающиеся с верхней части экрана
эф_ринг Эффект, создающий расширяющиеся и исчезающие круги
эф_дым Эффект, создающий маленькие струйки дыма
ef_smokeup Эффект, создающий дымовой шлейф, поднимающийся вверх по экрану
эф_снег Эффект, создающий множество снежных частиц, падающих вниз по экрану
эф_искра Эффект, создающий небольшую искру
эф_стар Эффект, создающий звездные частицы

Синтаксис:

effect_create_below(kind, x, y, size, color);

Аргумент Описание
добрый Вид эффекта (используйте одну из констант, перечисленных выше).
x Позиционирование x эффекта, если это имеет значение.
y Позиционирование эффекта по оси y, если это важно.
размер Размер эффекта.
цвет Цвет эффекта.

Возвращается:

Н/Д

Пример:

if speed > 0
{
effect_create_below(ef_smoke, x, y, 0, c_gray);
}

Приведенный выше код будет создавать небольшую струйку серого дыма на каждом шаге, когда скорость экземпляра больше 0 в координатах x,y экземпляра.
