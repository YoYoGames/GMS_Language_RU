часть_системы_обновление

С помощью этой функции вы можете заставить систему частиц рисовать все текущие частицы на экране. Если part_system_automatic_draw() выключена, то эта функция будет показывать частицы при используется в событии draw экземпляра, или может быть использована, когда цель рисования установлена на поверхность (используя surface_set_target()), чтобы нарисовать частицы в системе на этой поверхности.

Синтаксис:

part_system_drawit(ind);

Аргумент Описание
инд Индекс системы частиц для отрисовки частиц.

Возвращается:

Н/Д

Пример:

if surface_exists(surf)
{
surface_set_target(surf);
part_system_drawit(global.psys);
surface_reset_target();
}

Приведенный выше код проверяет, существует ли поверхность, индексированная в переменной "surf", и если существует, то устанавливает цель рисования на поверхность, рисует систему частиц с идентификатором, хранящимся в глобальной переменной, а затем сбрасывает цель рисования. чтобы на экране снова появился обычный рисунок.
