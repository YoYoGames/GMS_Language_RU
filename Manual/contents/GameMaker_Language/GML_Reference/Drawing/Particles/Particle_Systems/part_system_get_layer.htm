part_system_get_layer

Эта функция может быть использована для получения уникального значения ID слоя для заданной системы частиц. Вы передаете значение ID системы частиц (возвращаемое функцией part_system_create_layer()) и функция вернет значение ID слоя.

Синтаксис:

part_system_get_layer(ind);

Аргумент Описание
инд Значение идентификатора системы частиц для получения идентификатора слоя

Возвращается:

Real (уникальное значение ID слоя или 0, если слой является внутренне управляемым)

Пример:

if (part_system_get_layer(global.p_sys) != "effects_layer")
{
part_system_layer(global.p_sys, "effects_layer";
}

Приведенный выше код проверит систему частиц, чтобы узнать, на каком слое она находится, и если она не находится на заданном слое, то она будет изменена.

Назад: Системы частиц
