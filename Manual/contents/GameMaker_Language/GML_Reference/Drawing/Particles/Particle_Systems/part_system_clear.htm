часть_системы_очистить

С помощью этой функции вы можете очистить индексированную систему до состояния по умолчанию, удалив все эмиттеры и сбросив глубину и положение (если они были изменены). Будьте осторожны, используя эту функцию, так как если у вас есть какие-либо установки экземпляра, разрывы или любое другое действие, использующее эмиттер, связанный с этой системой, вы получите ошибку, если только вы не используете эту part_emitter_exists() для проверки. Также нет необходимости вызывать part_emitter_destroy() функцию так как это происходит автоматически.

ПРИМЕЧАНИЕ: эта функция очистит видимые частицы в комнате, но она не очищает свойства частиц и не удаляет их из памяти. Для этого вам следует использовать функции part_type_clear() и part_type_destroy().

Синтаксис:

part_system_clear(ind);

Аргумент Описание
инд Индекс системы частиц для очистки.

Возвращается:

Н/Д

Пример:

part_system_clear(global.Sname);

Приведенный выше код очистит систему частиц, индексированную в глобальной переменной "Sname", до состояния по умолчанию.
