часть_частиц_создать

Эта функция идеально подходит для тех эффектов, которые не требуют функциональности, предлагаемой эмиттерами частиц (например, для создания дыма от ракеты или простого эффекта взрыва), поскольку она позволяет быстро и легко создавать частицы в любом месте игровой комнаты. Обратите внимание, что для использования этой функции необходимо предварительно создать систему частиц и тип частиц.

Синтаксис:

part_particles_create(ind, x, y, parttype, number);

Аргумент Описание
инд Индекс системы частиц.
x Координата x места создания частиц.
y Координата y места создания частиц.
тип детали Индекс (тип) создаваемых частиц.
номер Количество частиц для создания.

Возвращается:

Н/Д

Пример:

if mouse_check_button(mb_left)
{
part_particles_create(sname, mouse_x, mouse_y, p_CursorEffect, 5);
}

Приведенный выше код проверяет, нажата ли кнопка мыши, и если он возвращает true, то генерирует 5 частиц в позиции мыши.
