тип_части_альфа3

Эта функция может быть использована для установки градиента с тремя значениями альфа (прозрачности) для каждой созданной частицы данного типа. Первая альфа - это значение, с которым все частицы будут начинаться, вторая альфа - это значение, которое будет смешиваться до половины времени жизни частицы, а третья альфа - это значение, с которым частица будет заканчиваться. и третья альфа - это та, с которой частица закончит свое существование. В течение жизни частицы будет происходить плавное изменение градиента от одной альфы к другой.

Синтаксис:

part_type_alpha3(ind, alpha1, alpha2, alpha3);

Аргумент Описание
инд Индекс типа частицы, который необходимо изменить.
альфа1 Начальная альфа частицы.
альфа2 Альфа полуокружности частицы.
альфа3 Конечная альфа частицы.

Возвращается:

Н/Д

Пример:

part_type_alpha3( part_Health, 0.5, 1, 0);

Приведенный выше код заставит все созданные частицы типа, указанного в переменной "part_Health", иметь альфа-смесь от 0.5 до 1 до 0 в течение их жизни.
