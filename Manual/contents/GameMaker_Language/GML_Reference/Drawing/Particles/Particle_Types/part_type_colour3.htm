тип_детали_цвет3

Эта функция может быть использована для установки трехцветного градиента для каждой созданной частицы заданного типа. Первый цвет - это цвет, с которым все частицы будут начинаться, второй цвет - это цвет, который будет смешиваться до половины времени жизни частицы. и третий цвет - это цвет, с которым частица закончится. В течение жизни частицы будет происходить плавное градиентное изменение цветов от одного к другому.

Синтаксис:

part_type_colour3( ind, color1, color2, color3 );

Аргумент Описание
инд Индекс типа частицы, который необходимо изменить.
цвет1 Цвет, с которого будет начинаться частица.
цвет2 Цвет, которым частица будет обладать в середине своего жизненного цикла.
цвет3 Цвет, на котором будет заканчиваться частица.

Возвращается:

Н/Д

Пример:

part_type_colour3( part_Health, c_red, c_white, c_maroon);

Приведенный выше код заставит все частицы, созданные по типу, указанному в переменной "part_Health", иметь смесь цветов от красного до белого и бордового в течение их жизни.
