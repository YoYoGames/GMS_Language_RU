part_type_colour_mix

С помощью этой функции вы можете задать для заданного типа частиц случайное смешение двух цветов.

Синтаксис:

part_type_colour_mix(ind, color1, color2);

Аргумент Описание
инд Индекс типа частицы, который необходимо изменить.
цвет1 Первый цвет, из которого будет взята смесь.
цвет2 Второй цвет, из которого будет взята смесь.

Возвращается:

Н/Д

Пример:

part_type_colour_mix(global.P_Damage, c_red, c_yellow);

Приведенный выше код установит цвет для каждой частицы, испускаемой частицей типа, указанного в глобальной переменной "P_Damage", как случайную смесь между красным и желтым цветами.
