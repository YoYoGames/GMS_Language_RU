тип_детали_чистый

С помощью этой функции вы можете "сбросить" частицу, вернув все значения для каждой из функций, относящихся к частице (жизнь, цвет, альфа, ориентация и т.д.), к их значениям по умолчанию. Обратите внимание, что эта функция не удаляет частицы которые в данный момент видны в комнате, с экрана, для этого вы должны использовать функцию part_particles_clear().

Синтаксис:

part_type_clear(ind);

Аргумент Описание
инд Индекс типа частицы, которую нужно очистить.

Возвращается:

Н/Д

Пример:

part_type_clear(global.explode_part);

Приведенный выше код очистит тип частицы, указанный в глобальной переменной "explode_part", до значений по умолчанию.
