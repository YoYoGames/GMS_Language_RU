тип_части_спрайта

Эта функция может использоваться для установки типа частиц для использования пользовательского спрайта из игровых активов. Вы можете выбрать спрайт, а затем указать GameMaker Studio 2 использовать этот спрайт и анимировать его или нет, и если вы выберете анимированный, то вы также можете выбрать чтобы суб-изображения изменялись с таким интервалом, что анимация будет начинаться и заканчиваться в начале и конце времени жизни каждой созданной частицы (так что если у вас время жизни установлено на 15 шагов минимум и 30 шагов максимум, независимо от того, какое независимо от того, какое значение имеет время жизни, анимация будет подстроена под него, причем для более короткого времени жизни она будет работать быстрее, а для более длинного - медленнее). Вы также можете выбрать случайное подизображение, чтобы, если спрайт не анимирован, для создания спрайта было выбрано другое изображение. или если частица анимирована, она начнет анимацию в случайной точке.

Синтаксис:

part_type_sprite(ind, sprite, animate, stretch, random);

Аргумент Описание
инд Индекс типа частицы, который необходимо изменить.
спрайт Индекс спрайта для установки типа частиц.
анимировать Следовать ли анимации спрайта нормально (true) или нет (false).
стрейч Растягивать ли анимацию спрайта в соответствии с временем жизни частиц (true) или нет (false).
случайный Выбирать ли случайное подизображение (true) или нет (false).

Возвращается:

Н/Д

Пример:

global.p1 = part_type_create();
part_type_sprite(global.p1 , spr_Coins, 1, 0, 0);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Приведенный выше код устанавливает тип частицы, указанный в глобальной переменной "p1", на форму спрайта, указанного в "spr_Coins", а затем устанавливает различные другие свойства частицы.
