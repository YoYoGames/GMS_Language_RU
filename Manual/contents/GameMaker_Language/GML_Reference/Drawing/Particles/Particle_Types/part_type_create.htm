тип_части_создать

С помощью этой функции вы можете создать новый тип частицы, а возвращаемое значение должно быть сохранено в переменной для использования во всех последующих функциях частиц.

Синтаксис:

part_type_create();

Возвращается:

Целое число (идентификатор частицы)

Пример:

mypart = part_type_create();

Это создаст новый тип частицы, сохранив ее индекс в переменной "mypart".
