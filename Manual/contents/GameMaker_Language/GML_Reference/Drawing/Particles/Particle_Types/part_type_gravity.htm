тип_части_гравитации

Эта функция устанавливает силу тяжести, которая будет действовать на каждую создаваемую частицу данного типа. Значение силы гравитации добавляется к скорости частицы на каждом шаге и обычно является небольшим значением, например 0,5, а направление - это направление гравитационного "притяжения" и соответствует стандартным направлениям GameMaker Studio 2: 0° - вправо, 90° - вверх, 180° - влево и 270° - вниз.

Синтаксис:

part_type_gravity(ind, grav_amount, grav_direction);

Аргумент Описание
инд Индекс типа частицы, который необходимо изменить.
grav_amount Сила притяжения.
гравитационное направление Направление силы тяжести.

Возвращается:

Н/Д

Пример:

part_type_shape(global.p1, pt_shape_pixel);
part_type_size(global.p1, 1, 3, 0, 0);
part_type_scale(global.p1, 1, 1);
part_type_colour1(global.p1, c_white);
part_type_alpha2(global.p1, 1, 0);
part_type_speed(global.p1, 2, 4, 0, 0);
part_type_direction(global.p1, 0, 180, 0, 0);
part_type_gravity(global.p1, 0.20, 270);
part_type_orientation(global.p1, 0, 0, 0, 1);
part_type_blend(global.p1, 1);
part_type_life(global.p1, 15, 60);

Приведенный выше код установит различные значения частиц, включая гравитацию, которая будет добавлять 0,2 к скорости каждый шаг с направлением 270º, таким образом, частица будет тянуться "вниз" к нижней части экрана.
