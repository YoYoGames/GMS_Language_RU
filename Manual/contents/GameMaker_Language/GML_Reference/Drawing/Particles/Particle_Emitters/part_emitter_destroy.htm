часть_излучателя_уничтожить

Эта функция удалит указанный излучатель из данной системы и очистит его из памяти (это также остановит производство любых частиц данным излучателем, но НЕ удалит их из комнаты). Эта функция всегда должна вызывать, когда данный эмиттер больше не нужен системе, чтобы предотвратить утечку памяти и ошибки.

Синтаксис:

part_emitter_destroy( ps, ind );

Аргумент Описание
ps Система частиц для уничтожения эмиттера.
инд Индекс излучателя, который нужно уничтожить.

Возвращается:

Н/Д

Пример:

if part_emitter_exists(global.Sname, p_emit)
{
part_emitter_destroy(global.Sname, p_emit1);
}

Приведенный выше код проверит, существует ли эмиттер частиц, указанный в переменной "p_emit", в системе частиц give, и если да, то он будет уничтожен.

Назад: Излучатели частиц
Далее: part_emitter_destroy_all
