часть_излучателя_создать

Эта функция должна использоваться для создания нового эмиттера и назначения его заданной системе частиц. Функция возвращает значение индекса для нового эмиттера, которое должно храниться в переменной и использоваться во всех последующих функциях, ссылающихся на эмиттер, а сам эмиттер должен быть уничтожен, если он больше не используется, чтобы предотвратить утечку памяти (это можно сделать с помощью специальных функций уничтожения эмиттера или путем уничтожения всей системы частиц, к которой принадлежит эмиттер).

Синтаксис:

part_emitter_create(ps);

Аргумент Описание
ps Система частиц, в которой будет создан эмиттер.

Возвращается:

Реально

Пример:

p_emit1 = part_emitter_create(Sname);

Это создаст новый эмиттер частиц и сохранит его индекс в переменной "p_emit".
