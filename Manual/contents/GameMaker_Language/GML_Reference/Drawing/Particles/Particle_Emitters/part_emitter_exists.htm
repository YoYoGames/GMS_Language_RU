часть_излучателя_существует

С помощью этой функции вы можете проверить, существует ли данный эмиттер частиц с индексом в данной системе или нет. Обратите внимание, что если проверяемая переменная является неинициализированной переменной (которой эмиттер частиц в противном случае присвоил бы свой индекс) это приведет к ошибке.

Синтаксис:

part_emitter_exists(ps, ind);

Аргумент Описание
ps Система частиц для проверки наличия эмиттера.
инд Индекс эмиттера для поиска.

Возвращается:

Булево

Пример:

if part_emitter_exists(sname, p_emit1)
{
part_emitter_burst(sname, p_emit1, part_1, 30);
}

Приведенный выше код проверит наличие эмиттера, указанного в переменной "permit", и если он существует, то выпустит из него 30 частиц.
