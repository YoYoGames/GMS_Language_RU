нарисовать_круглую_прямую

С помощью этой функции можно нарисовать либо контур закругленного прямоугольника, либо заполненный закругленный прямоугольник, где позиция (x1,y1) - левый верхний угол, а позиция (x2,y2) - правый нижний угол. Вы можете определить, насколько точным будет рисование углов с помощью функции draw_set_circle_precision(), но в данной функции для них используется фиксированный радиус (если вам нужно изменить радиус угла, используйте функцию draw_roundrect_ext()). Обратите внимание, что для рисования прямоугольника с нужными размерами могут потребоваться другие значения (+/-1 по x, y, ширине или высоте) из-за различий в разных поддерживаемых платформах.

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_roundrect(x1, y1, x2, y2, контур);

Аргумент Описание
x1 Координата x левого верхнего угла закругленного прямоугольника (как у обычного прямоугольника).
y1 Координата y левого верхнего угла закругленного прямоугольника (как у обычного прямоугольника).
x2 Координата x нижнего правого угла закругленного прямоугольника (как если бы это был обычный прямоугольник).
y2 Координата y нижнего правого угла скругленного прямоугольника (как если бы это был обычный прямоугольник).
конспект Будет ли закругленный прямоугольник рисоваться заполненным (false) или в виде контура шириной в один пиксель (true).

Возвращается:

Н/Д

Пример:

draw_set_colour(c_lime);
draw_roundrect(50, 100, 250, 200, 0);

Это нарисует заполненный закругленный прямоугольник с теоретическим (до изогнутого) верхним левым углом на (50,100) и теоретическим нижним правым углом на (250,200).
