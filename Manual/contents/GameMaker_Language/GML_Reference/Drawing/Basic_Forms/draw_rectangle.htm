нарисовать_прямоугольник

С помощью этой функции можно нарисовать либо контур прямоугольника, либо заполненный прямоугольник, где позиция (x1,y1) - левый верхний угол, а позиция (x2,y2) - правый нижний угол. Обратите внимание, что для рисуемого прямоугольника могут потребоваться различные значения (+/-1 по x, y, ширине или высоте), чтобы нарисованный прямоугольник имел нужные размеры из-за различий в различных поддерживаемых платформах.

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_rectangle(x1, y1, x2, y2, контур);

Аргумент Описание
x1 Координата x левого верхнего угла прямоугольника.
y1 Координата y левого верхнего угла прямоугольника.
x2 Координата x правого нижнего угла прямоугольника.
y2 Координата y правого нижнего угла прямоугольника.
конспект Будет ли прямоугольник рисоваться заполненным (false) или в виде контура шириной в один пиксель (true).

Возвращается:

Н/Д

Пример:

draw_set_colour(c_yellow);
draw_rectangle(100, 100, 300, 200, true);

Это позволит нарисовать контур прямоугольника, левый верхний угол которого находится в точке (100,100), а правый нижний - в точке (300,200).
