draw_healthbar

С помощью этой функции можно нарисовать цветную полосу для отображения постоянной величины. Хотя в функции используется слово "healthbar", вы можете использовать ее для отображения чего угодно, если отображаемая величина представляет собой процентное значение от 0 до 100 (процент от любого значения может быть рассчитан по формуле (КУРСОВОЕ_ЗНАЧЕНИЕ / МАКСИМУМ_ЗНАЧЕНИЯ) * 100), поэтому, например, вы можете использовать эту функцию для отображения силы, здоровья, маны, времени или чего-либо еще, что может быть полезно для отображения в виде столбика.

Вы можете установить различные параметры, которые изменяют визуальный аспект полосы, и они показаны на рисунке ниже (обратите внимание, что здесь используются цвета c_green для максимального значения и c_red для минимального значения, но вы можете использовать цвета по своему усмотрению). вы можете использовать любые цвета по своему усмотрению):

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_healthbar(x1, y1, x2, y2, amount, backcol, mincol, maxcol, direction, showback, showborder);

Аргумент Описание
x1 Координата x левой части панели здоровья
y1 Координата y верхней части панели здоровья
x2 Координата x правой части панели здоровья
y2 Координата y нижней части панели здоровья
сумма Переменная, определяющая общее состояние здоровья (от 0 до 100)
backcol Цвет фона для бара
mincol Цвет полоски при отсутствии здоровья (0)
maxcol Цвет полоски при полном здоровье (100)
направление Где "закреплена" полоса (0 = слева, 1 = справа, 2 = сверху, 3 = снизу)
showback Рисуется ли фон бара (true) или нет (false). Если false, аргумент backcol не принимается во внимание.
показать границу Имеют ли элементы бара черную рамку шириной 1px (true) или нет (false).

Возвращается:

Н/Д

Пример:

var pc;
pc = (hp / max_hp) * 100;
draw_healthbar(100, 100, 500, 200, pc, c_black, c_red, c_lime, 0, true, true)

Приведенный выше код использует процентное значение, полученное из переменных "hp" и "hp_max", для рисования стандартной красно-зеленой панели здоровья.
