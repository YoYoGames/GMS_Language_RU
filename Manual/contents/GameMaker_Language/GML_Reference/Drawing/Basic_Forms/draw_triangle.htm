нарисовать_треугольник

С помощью этой функции можно нарисовать либо контур треугольника, либо заполненный треугольник.

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_triangle(x1, y1, x2, y2, x3, y3, контур);

Аргумент Описание
x1 Координата x первого угла треугольника.
y1 Координата y первого угла треугольника.
x2 Координата x второго угла треугольника.
y2 Координата y второго угла треугольника.
x3 Координата x третьего угла треугольника.
y3 Координата y третьего угла треугольника.
конспект Будет ли треугольник рисоваться заполненным (false) или в виде контура шириной в один пиксель (true).

Возвращается:

Н/Д

Пример:

draw_set_colour(c_aqua);
draw_triangle(50, 50, 200, 50, 50, 50, 200, 0);

В результате будет нарисован заполненный аквамаринового цвета равнобедренный прямоугольный треугольник, первый угол которого расположен в точке (50,50), второй - в точке (200,50), а третий - в точке (50,200).
