draw_roundrect_colour_ext

С помощью этой функции можно нарисовать либо контур закругленного прямоугольника, либо заполненный закругленный прямоугольник, где позиция (x1,y1) - левый верхний угол, а позиция (x2,y2) - правый нижний угол. Если прямоугольник заполнен, то аргументы color будут использоваться для создания цветового градиента от центра к краям, где цвет 1 - это цвет центра, а цвет 2 - цвет краев. Вы также должны указать значения радиуса для осей x и y (в пикселях), и углы будут закруглены на эти величины. Вы можете определить, насколько точно будут прорисованы углы, с помощью функции draw_set_circle_precision(). Обратите внимание, что для рисуемого прямоугольника могут потребоваться другие значения (+/-1 по осям x, y, ширине или высоте), чтобы нарисованный прямоугольник имел нужные размеры из-за различий в разных поддерживаемых платформах.

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_roundrect_colour_ext(x1, y1, x2, y2, xrad, yrad, col1, col2, outline);

Аргумент Описание
x1 Координата x левой части закругленного прямоугольника.
y1 Координата y вершины скругленного прямоугольника.
x2 Координата x правой части закругленного прямоугольника.
y2 Координата y нижней части скругленного прямоугольника.
xrad Радиус кривой вдоль оси x от углов прямоугольника.
yrad Радиус кривой вдоль оси y от углов прямоугольника.
кол1 Центральный цвет.
кол2 Цвет внешнего края.
конспект Является ли прямоугольник контуром (true) или заполненным (false).

Возвращается:

Н/Д

Пример:

var dist = point_distance(x, y, mouse_x, mouse_y) / 10;
var col = make_colour_hsv(clamp(dist, 0, 255), 255, 255);
draw_roundrect_colour_ext(x - 50, y - 50, x + 50, y + 50, dist, dist, col, c_white, 0);

В этом случае рисуется заполненный квадрат с закругленными углами, радиус которого определяется расстоянием от экземпляра, выполняющего рисование, до мыши. Это значение также используется для расчета цвета смешивания центра.
