нарисовать_кнопку

Эта функция нарисует очень простую прямоугольную "кнопку", используя выбранный в данный момент цвет рисунка и альфа, где аргумент up определяет, как будет выглядеть эффект скошенного края, как показано на изображении ниже:

ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных следующие входные данные: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого.

Синтаксис:

draw_button(x1, y1, x2, y2, up);

Аргумент Описание
x1 Координата x левой части кнопки
y1 Координата y верхней части кнопки
x2 Координата x правой части кнопки
y2 Координата y нижней части кнопки
вверх Находится ли кнопка вверху(true) или внизу(false)

Возвращается:

Н/Д

Пример:

draw_button(100, 100, 200, 150, !mouse_check_button(mb_left));

Это нарисует кнопку, которая будет выглядеть нажатой, если удерживать левую кнопку мыши.
