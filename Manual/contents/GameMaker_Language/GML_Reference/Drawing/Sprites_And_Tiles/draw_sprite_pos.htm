draw_sprite_pos

С помощью этой функции можно нарисовать спрайт, искаженный в области, определенной четырьмя угловыми координатами. Первыми двумя аргументами являются рисуемый спрайт и подизображение спрайта (такое же, как в функции draw_sprite()) но следующие - это те, которые определяют положение каждого из четырех углов ограничивающей области данного спрайта. Их следует задавать по часовой стрелке, поэтому первой координатой будет верхняя левая, затем верхняя правая, затем нижняя правая и, наконец, нижняя левая. правая и, наконец, нижняя левая. Вы также можете задать значение альфа для спрайта, чтобы нарисовать его прозрачным. На рисунке ниже показано, как работает эта функция:

ВНИМАНИЕ! Изображение выше приведено только для примера, и если вы используете эту функцию на спрайте, вы получите другие результаты и можете столкнуться с "обрезанием" текстуры из-за того, что спрайт строится из квадранта примитивов.

ПРИМЕЧАНИЕ: Эта функция полезна только для растровых спрайтов и не работает со спрайтами SWF или JSON (Spine).

ПРИМЕЧАНИЕ: При использовании этой функции параметр Автоматически обрезать для текстурных страниц должен быть отключен.

Синтаксис:

draw_sprite_pos(sprite, subimg, x1, y1, x2, y2, x3, y3, x4, y4, alpha);

Аргумент Описание
спрайт Индекс спрайта для рисования.
subimg Подimg (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте).
x1 Первая координата x.
y1 Первая координата y.
x2 Вторая координата x.
y2 Вторая координата y.
x3 Третья координата x.
y3 Третья координата y.
x4 Четвертая координата x.
y4 Четвертая координата y.
альфа Альфа спрайта (от 0 до 1, где 0 - прозрачный, а 1 - непрозрачный).

Возвращается:

Н/Д

Пример:

draw_sprite_pos(sprite_index, image_index, x - 100, y - 50, x - 50, y +150, x + 100, y + 200, x + 100, y, 1);

Приведенный выше код нарисует спрайт, связанный с экземпляром, выполняющим код, с искажением вокруг x / y позиции экземпляра и с полностью непрозрачной альфой.
