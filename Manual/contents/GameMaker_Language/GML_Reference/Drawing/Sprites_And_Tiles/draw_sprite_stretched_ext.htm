draw_sprite_stretched_ext

Эта функция делает то же самое, что и draw_sprite_stretched() с добавлением возможности установить смешивание цветов и значение альфа для спрайта при его рисовании (аналогично функции draw_sprite_ext()).

ПРИМЕЧАНИЕ: Смешивание цветов рекомендуется только для HTML5-цели, когда включен WebGL, хотя вы можете установить цвет смешивания, если он не включен, и он будет смешивать спрайт как обычно. Однако при таком смешивании создается дубликат спрайта, который затем сохраняется в кэше и используется при необходимости. Это далеко не оптимально, и если вы используете многократное изменение цвета, это замедлит производительность вашей игры, если вы не активируете WebGL. если вы не хотите использовать WebGL, то вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это, если потребуется, используя функцию sprite_set_cache_sizee().

Синтаксис:

draw_sprite_stretched_ext(sprite, subimg, x, y, w, h, color, alpha);

Аргумент Описание
спрайт Индекс спрайта для рисования.
subimg Подimg (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте).
x Координата x места рисования спрайта.
y Координата y места рисования спрайта.
w Ширина области, которую будет занимать растянутый спрайт.
h Высота области, которую будет занимать растянутый спрайт.
цвет Цвет, с которым будет смешиваться спрайт. c_white - отображать его в обычном режиме.
альфа Альфа спрайта (от 0 до 1, где 0 - прозрачный, а 1 - непрозрачный).

Возвращается:

Н/Д

Пример:

draw_sprite_stretched_ext(sprite_index, image_index, x, y, sprite_width, sprite_height / 2, c_white, 0.5);

В результате будет нарисован назначенный экземпляру спрайт и его подизображение с левым углом в позиции x /y экземпляра. Его ширина устанавливается равной ширине спрайта, а высота - высоте спрайта, деленной на два. Он также будет нормально смешиваться, но будет иметь частично прозрачное альфа-значение.
