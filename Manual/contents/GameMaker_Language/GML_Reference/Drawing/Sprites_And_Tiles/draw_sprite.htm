рисовать_спрайт

Эта функция рисует заданный спрайт и суб-изображение в определенном месте игровой комнаты. Для спрайта вы можете использовать переменную экземпляра индекс спрайта для получить текущий спрайт, который назначен экземпляру, выполняющему код, или вы можете использовать любой другой актив спрайта. То же самое относится и к суб-изображению, поскольку оно также может быть задано переменной экземпляра индекс_изображения которая установит суб-изображение, выбранное для текущего спрайта экземпляра (обратите внимание, что вы можете нарисовать другой спрайт и все равно использовать значение суб-изображения для текущего экземпляра), или вы можете использовать любое другое значение для этого, чтобы нарисовать определенное суб-изображение выбранного спрайта. Если значение больше, чем количество субизображений, то GameMaker Studio 2 будет автоматически перебирать номера, чтобы выбрать соответствующее изображение (например, если рисуемый спрайт имеет 5 субизображений, пронумерованных от 0 до 4 и мы установили значение индекса равным 7, то функция нарисует суб-изображение 3, пронумерованное 0). Наконец, x и y position - это положение в комнате, в котором будет нарисован спрайт, и оно центрируется по смещению x и y спрайта.

ПРИМЕЧАНИЕ: Эта функция может работать не так, как ожидается, при использовании спрайтов со скелетной анимацией, и вы можете обнаружить, что функция рисует только первый кадр позы по умолчанию. Вместо нее следует использовать функции draw_skeleton_*.

Синтаксис:

draw_sprite(sprite, subimg, x, y);

Аргумент Описание
спрайт Индекс спрайта для рисования.
subimg Под-изображение (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте).
x Координата x места рисования спрайта.
y Координата y места рисования спрайта.

Возвращается:

Н/Д

Пример:

draw_sprite(sprite_index, image_index, x, y);
draw_sprite(spr_Halo, 0, x, y-32);

Это нарисует назначенный экземпляру спрайт (sprite_index) с текущим суб-изображением в позиции x и y экземпляра в комнате. Затем будет нарисован первый кадр спрайта с индексом "spr_Halo" в том же положении x и y. но на 32 пикселя "выше".
