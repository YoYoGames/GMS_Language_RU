draw_enable_swf_aa

С помощью этой функции можно включить или отключить сглаживание (AA) для векторных спрайтов формата SWF. AA просто сглаживает края векторных изображений, чтобы придать им более приятный вид. Количество используемого AA зависит от значения, установленного с помощью функции draw_set_swf_aa_level(). По умолчанию эта функция отключена.

Синтаксис:

draw_enable_swf_aa(enable);

Аргумент Описание
включить Включить(true) или выключить(false) AA для всех спрайтов SWF.

Возвращается:

Н/Д

Пример:

if draw_get_swf_aa_level() == 0
{
draw_enable_swf_aa(true);
draw_set_swf_aa_level(0.5);
}

Приведенный выше код проверит значение AA для спрайтов формата SWF, и если оно равно 0, то включит AA и установит значение 0,5.
