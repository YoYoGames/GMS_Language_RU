нарисовать_плитку

С помощью этой функции вы можете нарисовать любую заданную плитку из набора плиток - конкурировать с поворотами, переворотами и зеркалами - задавая рамку (если она анимирована) и положение в комнате. Значение ID набора плиток является индексом набора плиток, установленным в IDE, а данные плитки data - это "сгусток" данных, который содержит всю информацию об индексе плитки и примененных преобразованиях. Вы можете использовать функцию tilemap_get() для получения данных плитки, которыми затем можно манипулировать с помощью функций плитки, или создать их самостоятельно, используя следующие константы плитки вместе со значением индекса создаваемой плитки:

Постоянная Описание
вращение_плитки Используется для установки/получения бита поворота блока данных плитки.
зеркальная плитка Используется для установки/получения зеркального бита блока данных плитки.
плитка_флип Используется для установки/получения флип-бита блока данных плитки.
маска_индекса_плитки Специальная константа, которая используется для "и"-соединения с блобом данных плитки для извлечения индекса плитки.


Для получения конечных плиточных данных необходимо побитовое "или" этих констант со значением индекса плитки:

var tiledata = 8 | tile_rotate | tile_flip;

Приведенный выше код просто создает локальную переменную для хранения tiledata для плитки с индексом 8 в наборе плиток, и она поворачивается и переворачивается. Затем она может быть использована для отрисовки плитки с этим индексом из указанного вами набора плиток. и плитка будет нарисована на глубине слоя экземпляра, выполняющего рисование, в позиции x/y, которую вы укажете в комнате. Обратите внимание, что вы также можете задать рамку для рисования. Если индексированная плитка является частью анимации то здесь можно задать номер кадра (от 0 до количества кадров - 1), и индексированная плитка будет просто индикатором того, какой набор анимированных плиток вы используете, а рисоваться будет именно данный кадр. Если плитка не является частью анимированного набора, то кадр должен быть равен 0.

Синтаксис:

draw_tile(tile set, tiledata, frame, x, y)

Аргумент Описание
набор плиток Идентификатор набора плиток для использования
tiledata Данные плитки для использования
рама Номер кадра для используемой анимированной плитки (по умолчанию 0 для неанимированных плиток)
x Позиция x в помещении для рисования
y Позиция y в помещении для рисования

Возвращается:

Реально

Пример:

var lay_id = layer_get_id("Tiles_sky");
var map_id = layer_tilemap_get_id(lay_id);
var data = tilemap_get(map_id, 0, 0);
tile_set_flip(data, true);
draw_tile(set_Wall, data, 0, x, y);

Приведенный выше код получит данные о плитке из заданного tilemap, затем установит флаг "flip" и использует эти данные для отрисовки плитки, перевернутой в позиции экземпляра.
