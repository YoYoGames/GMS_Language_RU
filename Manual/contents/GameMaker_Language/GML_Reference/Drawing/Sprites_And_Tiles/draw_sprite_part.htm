нарисовать_часть

С помощью этой функции вы можете нарисовать часть любого спрайта в заданной позиции в комнате. Как и в случае с нарисовать_спрайт() вы можете указать спрайт и суб-изображение для рисования, затем вы должны указать относительные координаты внутри спрайта области, которую нужно выбрать для рисования. Это означает, что левая позиция 0 и верхняя позиция 0 будут левым верхним углом спрайта, и все дальнейшие координаты должны быть взяты из этой позиции. На рисунке ниже показан пример как это работает:

Следует отметить, что если страница текстуры допускает автоматическое обрезание, то эта функция может не работать так, как ожидается, поскольку при создании текстурной страницы будет удалено лишнее "пустое" пространство вокруг спрайта. Чтобы решить эту проблему, необходимо в настройках страницы текстуры (в редакторе группы текстур) отключить опцию Automatic Crop.

ПРИМЕЧАНИЕ: При рисовании с помощью этой функции, смещение спрайта по x и y игнорируется, и часть спрайта будет нарисована с левым верхним углом в заданной позиции x / y в комнате.

ПРИМЕЧАНИЕ: Эта функция полезна только для растровых спрайтов и не работает со спрайтами SWF или JSON (Spine).

Синтаксис:

draw_sprite_part(sprite, subimg, left, top, width, height, x, y);

Аргумент Описание
спрайт Индекс спрайта для рисования.
subimg Подimg (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте).
слева Позиция x на спрайте левого верхнего угла области для рисования.
топ Позиция y на спрайте левого верхнего угла области для рисования.
ширина Ширина области для рисования.
высота Высота области для рисования.
x Координата x места рисования спрайта.
y Координата y места рисования спрайта.

Возвращается:

Н/Д

Пример:

draw_sprite_part(sprite_index, image_index, 4, 0, sprite_width-16, sprite_height-16, x, y );

Это отрисует назначенный экземпляр спрайта (sprite_index) и его текущий кадр анимации (image_index), однако при этом будет сбрито 4px поля по ширине с обеих сторон и 8 pixel поля по высоте от нижней части оригинального 24x24 пикселей.
