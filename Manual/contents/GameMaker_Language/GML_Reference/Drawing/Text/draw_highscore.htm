draw_highscore

Эта простая функция нарисует текущий список внутренних рейтингов, используя установленные в данный момент шрифт, цвет и альфа-значения, в указанном прямоугольнике. Вы указываете координаты левого верхнего угла и правого нижнего угла прямоугольной области для рисования текста, а GameMaker Studio 2 позаботится об этом. прямоугольной области для рисования текста, а GameMaker Studio 2 позаботится обо всем остальном - расстояние и положение будут выполнены автоматически.

Синтаксис:

draw_highscore( x1, y1, x2, y2 );

Аргумент Описание
x1 Координата x левой части прямоугольника Highscore.
y1 Координата y вершины прямоугольника Highscore.
x2 Координата x правой части прямоугольника Highscore.
y2 Координата y нижней части прямоугольника Highscore.

Возвращается:

Н/Д

Пример:

draw_highscore(100, 100, room_width - 100, room_height - 100);

Это нарисует таблицу высоких результатов в прямоугольнике в центре комнаты с границей в 100px.
