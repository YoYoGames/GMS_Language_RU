draw_text_ext_transformed_colour

Эта функция представляет собой комбинацию базовой нарисовать_текст() функции с draw_text_ext(), нарисовать_текст_трансформированный() и нарисовать_текст_цвет() функциями. Как таковая, это наиболее общая из всех функций для рисования текста и дает большую гибкость при рисовании. Обратите внимание, что аргумент "ширина" основан на масштабе, равном 1, поэтому если масштаб другой, это значение должно быть изменено пропорционально. Например, если базовая ширина для переноса строки равна 300, а масштаб установлен на 2, то текст будет выглядеть неправильно, выходя за пределы заданной ширины. ширину. Вместо этого вам следовало бы установить ширину 150, чтобы компенсировать масштабирование.

ПРИМЕЧАНИЕ: Градиентное смешивание недоступно для HTML5, если не включен WebGL, хотя вы все равно можете задать цвета смешивания, и шрифт будет смешиваться с первым заданным цветом. Однако при таком смешивании создается дубликат шрифта, который затем сохраняется в кэше и используется по мере необходимости, что далеко не оптимально, а если вы используете многократное изменение цвета, это замедлит производительность вашей игры. Вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это в случае необходимости используя функцию font_set_cache_size().

Синтаксис:

draw_text_ext_transformed_colour(x, y, string, sep, w, xscale, yscale, angle, c1, c2, c3, c4, alpha);

Аргумент Описание
x Координата x нарисованной строки.
y Координата y нарисованной строки.
строка Нить, которую нужно натянуть.
сеп Расстояние в пикселях между строками текста.
w Максимальная ширина строки в пикселях перед переводом строки.
xscale Горизонтальная шкала.
yscale Вертикальная шкала.
угол Угол наклона текста.
c1 Цвет для верхней левой части нарисованного текста.
c2 Цвет для верхней правой части нарисованного текста.
c3 Цвет для правой нижней части нарисованного текста.
c4 Цвет для левой нижней части нарисованного текста.
альфа Альфа для текста.

Возвращается:

Н/Д

Пример:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
draw_text_ext_transformed_colour(200, 200, "My name is" + global.name +".", 20, 200, 0.5, 0.5, 180, c_blue, c_blue, c_navy, c_navy, 1);

В результате будет нарисован заданный текст с нисходящим градиентом от синего к темно-синему, в позиции (200,200) в комнате, повернутый вверх ногами и с размером шрифта вдвое меньше обычного. Между каждой линией должно быть 20 пикселей, если нарисовано более одна линия, а максимальная ширина каждой линии составляет 200 пикселей.
