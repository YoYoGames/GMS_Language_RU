draw_text_ext_transformed

Эта функция представляет собой комбинацию базовой нарисовать_текст() функции с draw_text_ext() и нарисовать_текст_трансформированный() позволяет масштабировать и поворачивать текст, сохраняя при этом определенный межстрочный интервал и максимальную ширину строки. Обратите внимание, что аргумент "ширина" основан на масштабе, равном 1, поэтому, если масштаб другой, это значение должно быть изменено пропорционально. Например, если базовая ширина для переноса строки равна 300, а вы установили масштаб 2, то текст будет выглядеть неправильно, выходя за пределы заданной ширины. Вместо этого вам следовало бы установить ширину 150, чтобы компенсировать масштабирование.

Синтаксис:

draw_text_ext_transformed(x, y, string, sep, w, xscale, yscale, angle);

Аргумент Описание
x Координата x нарисованной строки.
y Координата y нарисованной строки.
строка Нить, которую нужно натянуть.
сеп Расстояние в пикселях между строками текста.
w Максимальная ширина строки в пикселях перед переводом строки.
xscale Горизонтальная шкала.
yscale Вертикальная шкала.
угол Угол наклона текста.

Возвращается:

Н/Д

Пример:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_ext_transformed(room_width / 2, room_height / 2, keyboard_string, 10, 300, 2, 2, image_angle);

Приведенный выше код нарисует заданный текст в центре комнаты, с максимальной длиной строки 300 пикселей, расстоянием между каждой строкой 10 пикселей, вращающийся по кругу и масштабированный до удвоенного исходного размера.
