рисовать_текст_текст_цвет

Эта функция представляет собой комбинацию базовой нарисовать_текст() функции с нарисовать_текст_текст() и нарисовать_текст_цвет() позволяющая определить цвета градиента для текста, а также межстрочный интервал и максимальную ширину строки.

ПРИМЕЧАНИЕ: Градиентное смешивание недоступно для HTML5, если не включен WebGL, хотя вы все равно можете задать цвета смешивания, и шрифт будет смешиваться с первым заданным цветом. Однако при таком смешивании создается дубликат шрифта, который затем сохраняется в кэше и используется по мере необходимости, что далеко не оптимально, а если вы используете многократное изменение цвета, это замедлит производительность вашей игры. Вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это в случае необходимости используя функцию font_set_cache_size().

Синтаксис:

draw_text_ext_colour(x, y, string, sep, w, c1, c2, c3, c4, alpha);

Аргумент Описание
x Координата x нарисованной строки.
y Координата y нарисованной строки.
строка Нить, которую нужно натянуть.
сеп Расстояние в пикселях между строками текста.
w Максимальная ширина строки в пикселях перед переводом строки.
c1 Цвет для верхней левой части нарисованного текста.
c2 Цвет для верхней правой части нарисованного текста.
c3 Цвет для правой нижней части нарисованного текста.
c4 Цвет для левой нижней части нарисованного текста.
альфа Альфа для текста.

Возвращается:

Н/Д

Пример:

draw_text_ext_colour(200, 200, keyboard_string, 5, 300, c_blue, c_blue, c_navy, c_navy, 1);

Приведенный выше код будет рисовать текст, который пользователь набирает на клавиатуре, разбивая его на новые строки каждый раз, когда длина строки превышает 300 пикселей. Код также будет поддерживать разделение в 5 пикселей между строками, если это произойдет. Каждая строка текста будет окрашена с помощью синего градиента: светло-синий вверху и темно-синий внизу.
