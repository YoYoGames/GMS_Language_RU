нарисовать_текст_трансформированный

Эта функция рисует текст аналогично функции нарисовать_текст() только теперь вы можете выбрать масштаб текста по горизонтальной или вертикальной оси (эффективно растягивая или сжимая его), а также нарисовать текст под углом. GameMaker Studio 2 рисовать его под углом (где 0 - нормальный, а каждый градус больше 0 поворачивает текст против часовой стрелки).

Синтаксис:

draw_text_transformed(x, y, string, xscale, yscale, angle);

Аргумент Описание
x Координата x нарисованной строки.
y Координата y нарисованной строки.
строка Нить, которую нужно натянуть.
xscale Горизонтальный масштаб (по умолчанию 1).
yscale Вертикальный масштаб (по умолчанию 1).
угол Угол наклона текста.

Возвращается:

Н/Д

Пример:

draw_set_halign(fa_center);
draw_set_valign(fa_middle);
image_angle += 1;
draw_text_transformed(room_width / 2, room_height / 2, "GAME OVER!", 2, 2, image_angle);

Приведенный выше код нарисует заданный текст в центре комнаты, вращаясь по кругу и масштабируя его до удвоенного исходного размера.
