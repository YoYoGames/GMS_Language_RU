draw_get_halign

Эта функция используется для получения настройки выравнивания текста по горизонтальной оси и возвращает одну из перечисленных ниже констант.

Синтаксис:

draw_get_halign();

Возвращается:

Постоянная

Постоянная Выравнивание
fa_left
fa_center
fa_right

Пример:

if draw_get_halign() != fa_left
{
draw_set_halign(fa_left);
}
draw_text(100, 32, "Score: " + string(score));

Приведенный выше код проверит, является ли текущее выравнивание текста левосторонним, и если нет, то оно будет установлено, после чего будет нарисован текст.
