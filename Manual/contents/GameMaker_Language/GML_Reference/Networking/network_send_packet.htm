network_send_packet

С помощью этой функции вы можете отправить "пакет" данных по сети. Функция принимает идентификатор сокета для подключения, затем вы должны указать идентификатор буфера, который содержит данные для отправки (для получения дополнительной информации о буферах см. раздел " Справочник - Буферы") и, наконец, размер (в байтах) пакета данных. Пакеты, отправленные с помощью этой функции, имеют такой формат, чтобы игра GameMaker Studio 2, принимающая данные, могла правильно "разделить" их. функция возвращает количество байт отправленных данных или число меньше 0, если отправка не удалась. Стоит отметить, что окончательный размер отправляемых данных, возвращаемый этой функцией, также включает в себя информация заголовка GameMaker, которая составляет дополнительные 12 байт.

Синтаксис:

network_send_packet(socket, buffer, size);

Аргумент Описание
розетка Идентификатор используемого сокета.
буфер Идентификатор буфера, из которого будут получены данные.
размер Размер (в байтах) данных.

Возвращается:

Реально

Пример:

buff = buffer_load("player_save.dat");
network_send_packet(sock, buff, buffer_get_size(buff));

Приведенная выше информация загружает ранее сохраненные данные буфера в память и возвращает идентификатор буфера для хранения в переменной "buff". Затем этот полный буфер отправляется в виде пакета по сети, используя сокет, идентифицированный переменной "sock".
