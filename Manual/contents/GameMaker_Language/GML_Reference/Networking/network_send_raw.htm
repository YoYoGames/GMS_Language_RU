network_send_raw

С помощью этой функции вы можете отправить "сырой" пакет данных по сети. Функция принимает идентификатор сокета для подключения, затем вы должны указать идентификатор буфера, который содержит данные для отправки (для получения дополнительной информации о буферах см. раздел Справочник - Буферы) и, наконец, размер (в байтах) пакета данных. Отправляемые данные никак не форматируются GameMaker Studio 2, и принимающие устройства получат их в виде потока, что означает, что вам придется обрабатывать их самостоятельно. что означает, что вам придется обрабатывать его самостоятельно. Функция возвращает количество байт отправленных данных или число меньше 0, если отправка не удалась.

Синтаксис:

network_send_raw(socket, buffer, size);

Аргумент Описание
розетка Идентификатор используемого сокета.
буфер Идентификатор буфера, из которого будут получены данные.
размер Размер (в байтах) данных.

Возвращается:

Реально

Пример:

buff = buffer_load("player_save.dat");
network_send_raw(sock, buff, buffer_get_size(buff));

Приведенная выше информация загружает ранее сохраненные данные буфера в память и возвращает идентификатор буфера для хранения в переменной "buff". Затем этот полный буфер отправляется в виде пакета необработанных данных по сети, используя сокет, идентифицируемый переменной переменной "sock".
