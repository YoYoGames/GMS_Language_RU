network_connect_raw_async

С помощью этой функции вы можете отправить запрос на подключение к серверу. Функция принимает идентификатор сокета для подключения (см. network_create_socket()) и требует, чтобы вы указали IP-адрес для подключения (строка), а также порт для подключения, и в случае неудачи соединения будет возвращено значение меньше 0. Разница между этой функцией и network_connect_async()в том, что эта может подключаться к любому серверу и ничего не делает с исходными данными, что означает, что вы должны сами реализовать протоколы на стороне сервера. Обратите внимание, что эта функция является асинхронной, генерируя событие Asynchronous Networking типа network_type_non_blocking_connect.

Синтаксис:

network_connect_raw_async(socket, url, port);

Аргумент Описание
розетка Идентификатор используемого сокета.
url URL или IP для подключения (строка).
порт Порт, к которому необходимо подключиться.

Возвращается:

Реально

Пример:

client = network_create_socket(network_socket_tcp);
network_connect_raw_async(client, "www.macsweeneygames.com", 6510);

Приведенный выше код создаст новый TCP сокет и попытается создать через него "сырое" асинхронное соединение с заданным URL на порту 6510.
