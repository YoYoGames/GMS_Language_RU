сетевое_решение

Эта функция возвращает IP-адрес заданного URL.

Синтаксис:

network_resolve(url);

Аргумент Описание
url URL для получения IP-адреса (строка).

Возвращается:

Строка

Пример:

game_ip = network_resolve("www.macsweeneygames.com");

Приведенный выше код вернет IP-адрес заданного URL и сохранит его в переменной "game_ip".
