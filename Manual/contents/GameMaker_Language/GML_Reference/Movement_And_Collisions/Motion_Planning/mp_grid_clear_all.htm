mp_grid_clear_all

С помощью этой функции вы можете очистить сетку MP от всех "запрещенных" ячеек.

Синтаксис:

mp_grid_clear_all(id);

Аргумент Описание
id Индекс mp_grid, который будет использоваться

Возвращается:

Булево

Пример:

if !instance_exists(obj_Player)
{
mp_grid_clear_all(grid);
}

Приведенный выше код очистит mp_grid, индексированную в переменной "grid", пометив все ячейки как свободные, если в комнате больше нет экземпляра объекта "obj_Player".
