mp_grid_destroy

Вы можете использовать эту функцию, чтобы уничтожить указанную mp_grid (как определено функцией mp_grid_create()) и освободить выделенную ей память. Очень важно, чтобы вы вызывали эту функцию каждый раз, когда или вы можете получить утечку памяти, что означает, что ваша игра будет замедляться с течением времени и в конечном итоге завершится крахом.

ПРИМЕЧАНИЕ: Использование функций mp_grid_* на сетке после ее уничтожения приведет к ошибке.

Синтаксис:

mp_grid_destroy(id);

Аргумент Описание
id Индекс mp_grid, которая должна быть уничтожена

Возвращается:

Н/Д

Пример:

if timer = 0
{
mp_grid_destroy(grid);
room_goto(rm_Menu);
}

Приведенный выше код уничтожит mp_grid, проиндексированную в переменной "grid", если переменная "timer" равна 0, а затем перейдет в другую комнату.
