mp_linear_step_object

С помощью этой функции вы указываете экземпляру сделать "шаг" к определенной точке, заданной значениями xgoal и ygoal. Размер шага (то есть на сколько пикселей экземпляр должен перемещаться каждый шаг) указывается значением stepize, и если экземпляр уже находится в заданной позиции, он остановится и не будет двигаться дальше, в отличие от других, более простых функций, таких как move_towards_point(). Размер шага - это также расстояние "вперед". на котором объект будет проверять каждый шаг на столкновение, и вы можете выбрать, остановится ли экземпляр при столкновении с объектом или экземпляром по вашему выбору.

Синтаксис:

mp_linear_step_object(xgoal, ygoal, stepize, obj);

Аргумент Описание
xgoal Положение цели x.
ygoal Положение цели по оси y.
размер шага Скорость перемещения экземпляра в пикселях за шаг.
obj Объект, который должен преградить путь. Может быть индексом объекта, идентификатором экземпляра или специальным ключевым словом, все

Возвращается:

Булево

Пример:

if mp_linear_step_object(mouse_x, mouse_y, 5, obj_Wall)
{
instance_create_layer(x, y, "Effects", obj_Explosion);
instance_destroy();
}

Приведенный выше код заставит объект двигаться в направлении мыши со скоростью 5 пикселей за шаг, проверяя столкновения только с объектом "obj_Wall". Когда объект достигнет позиции мыши, он создаст объект "obj_Explosion" и уничтожит себя.
