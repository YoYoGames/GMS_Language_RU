mp_linear_path_object

Эта функция рассчитывает прямой путь от текущей позиции экземпляра до точки, заданной значениями "xgoal" и "ygoal". Она использует указанный размер шага, такой же, как в функции mp_linear_step(). Указанный путь должен уже существовать и будет перезаписан новым путем, а функция вернет, был ли найден полный путь (true) или нет (false). Полный путь найден только в том случае, если не было столкновения с указанным объектом или экземпляром, и если возвращается false, то путь все равно будет сгенерирован, но он будет проходить только до того места, где путь был заблокирован.

ПРИМЕЧАНИЕ: Эта функция не перемещает экземпляр. Она только задает путь, а для перемещения необходимо использовать функции Path.

Синтаксис:

mp_linear_path_object(path, xgoal, ygoal, stepize, obj);

Аргумент Описание
путь Индекс пути, который будет использоваться
xgoal Положение цели x.
ygoal Положение цели по оси y.
размер шага Скорость перемещения экземпляра в пикселях за шаг.
obj Объект, который должен преградить путь. Может быть индексом объекта, идентификатором экземпляра или специальным ключевым словом,all

Возвращается:

Булево

Пример:

if mp_linear_path_object(path, mouse_x, mouse_y, 4, obj_Wall)
{
path_start(path, 4, 0, 0);
}

Приведенный выше код проверяет наличие столкновения с "obj_Wall" на пути между объектом, запускающим код, и позицией x/y мыши. Если столкновений нет и сгенерирован полный путь, то запускается объект по сгенерированному пути.
