mp_grid_get_cell

С помощью этой функции вы можете проверить любую заданную ячейку mp_grid на предмет того, была ли она отмечена как занятая или нет, указав индекс (ранее созданной) mp_grid и координаты x и y проверяемой ячейки. Если она была занята или проверяемая позиция находится вне границ сетки, то функция вернет -1, в противном случае она вернет 0.

Синтаксис:

mp_grid_get_cell(id, x , y);

Аргумент Описание
id Индекс mp_grid, который будет использоваться
x1 Координата x сетки для проверки.
y1 Координата y сетки для проверки.

Возвращается:

Реально

Пример:

if mp_grid_get_cell(grid, mouse_x div 16, mouse_y div 16) == -1
{
image_blend = c_red;
}
else
{
image_blend = c_lime;
}

Приведенный выше код проверит ячейку mp_grid, соответствующую положению мыши, и если она занята, то установит переменную image_blend в красный цвет, а если не занята, то в зеленый.
