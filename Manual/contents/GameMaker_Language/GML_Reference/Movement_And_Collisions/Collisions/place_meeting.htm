место_заседания

С помощью этой функции можно проверить позицию на столкновение с другим экземпляром или со всеми экземплярами объекта, используя маску столкновения того экземпляра, который запускает код проверки. При использовании этой функции вы фактически просите GameMaker Studio 2 переместить экземпляр в новую позицию, проверить наличие столкновения, вернуться назад и сообщить вам, было ли обнаружено столкновение или нет. Это будет работать для точных столкновений, но только если и у экземпляра, и у проверяемого объекта выбраны маски точных столкновений. в противном случае применяются только столкновения с ограничивающей рамкой. Эта функция также принимает специальные ключевые слова all и other. Обратите внимание, что если вам нужно получить уникальный идентификатор экземпляра объекта, с которым происходит столкновение, вы должны следует использовать instance_place().

Обратите внимание, что перед проверкой заданные координаты x/y будут приведены к ближайшему целому числу. выполняется.

Синтаксис:

place_meeting(x, y, obj);

Аргумент Описание
x Позиция x для проверки.
y Позиция y для проверки.
obj Экземпляр или объект для проверки.

Возвращается:

Булево

Пример:

if keyboard_check(vk_left)
{
if !place_meeting(x - 5, y, obj_wall) x -=5;
}

Приведенный выше код проверяет, нет ли столкновения слева от экземпляра, и перемещает экземпляр, если его нет.
