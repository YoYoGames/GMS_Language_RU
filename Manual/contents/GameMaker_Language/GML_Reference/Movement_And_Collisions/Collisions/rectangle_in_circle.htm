прямоугольник_в_круге

Эта функция проверяет заданную вами прямоугольную область на предмет отсутствия столкновений, полного попадания в границы назначения или просто касания с заданной круговой областью. Если они вообще не соприкасаются, функция вернет значение 0, если источник полностью находится в пределах места назначения, то функция вернет 1, а если они просто пересекаются, то вернет 2. Изображение ниже иллюстрирует это:

Синтаксис:

rectangle_in_circle(sx1, sy1, sx2, sy2, x, y, rad);

Аргумент Описание
sx1 Координата x левой стороны исходного прямоугольника.
си1 Координата y верхней стороны исходного прямоугольника.
sx2 Координата x правой стороны исходного прямоугольника.
си2 Координата y нижней стороны исходного прямоугольника.
x Координата x центра окружности
y Координата y центра окружности.
rad Радиус вокруг центральной точки, в котором следует проверить наличие столкновения.

Возвращается:

Реально

Пример:

inst = instance_nearest(x, y, obj_Bullet);
if instance_exists(inst)
{
if rectangle_in_circle(inst.x - 5, inst.y - 5, inst.x + 5, inst.y + 5, x, y - 25, 20) > 0
{
hit = true;
}
}

Приведенный выше код использует функцию rectangle_in_circle для проверки столкновения в пределах круговой области и прямоугольника вокруг найденного экземпляра. Если столкновение есть (либо край перекрывается, либо охватывается), то переменная будет установлена в true.
