Вы можете использовать эту функцию, чтобы проверить, столкнется ли вызывающий экземпляр с любым другим экземпляром объекта или со всеми экземплярами в вашей игре. Для правильного разрешения столкновения экземпляр, выполняющий код, должен иметь действительную маску столкновения маска (либо для самого спрайта, либо через mask_index), и он будет регистрировать столкновения только с теми экземплярами, которые также имеют действительную маску.

Функция проверяет, нет ли столкновений, если вызывающий экземпляр будет помещен в определенную позицию, и вы можете предоставить дополнительный аргумент для уточнения проверки, чтобы проверить, свободна ли позиция от столкновений с экземплярами данного типа. Обратите внимание, что если не указан дополнительный идентификатор объекта, то проверка будет выполнена для всех экземпляров в пределах комнаты. Проверка столкновений будет либо точной, либо основанной на граничной области экземпляра, в зависимости от того, какой тип маски столкновений был выбран. но для корректной работы точных столкновений все экземпляры в проверке должны иметь точные маски столкновений.

Обратите внимание, что перед выполнением проверки заданные координаты x/y будут приведены к ближайшему целому числу.

Синтаксис:

place_empty(x, y, [object_id]);

Аргумент Описание
x Позиция x для проверки.
y Позиция y для проверки.
[object_id] Необязательно! Объект для проверки.

Возвращается:

Булево

Пример:

if place_empty(mouse_x, mouse_y, obj_Enemy)
{
x = mouse_x;
y = mouse_y;
}

Приведенный выше код будет проверять столкновение с любым другим экземпляром объекта "obj_Enemy", как если бы вызывающий экземпляр был помещен в ту же позицию, что и мышь. Если столкновение не обнаружено, то для экземпляра устанавливаются координаты x/y устанавливаются равными координатам мыши.
