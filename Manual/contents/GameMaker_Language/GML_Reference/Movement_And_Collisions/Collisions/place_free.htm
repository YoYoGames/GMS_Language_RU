место_свободно

С помощью этой функции можно проверить, столкнется ли вызывающий экземпляр с любым экземпляром, отмеченным как твердый в вашей игре. Следует отметить, что для того, чтобы это сработало, экземпляр, выполняющий код, должен иметь действительную маску столкновения маска (либо для самого спрайта, либо через mask_index), и он будет регистрировать столкновения только с теми экземплярами, помеченными как твердые, у которых также есть действительная маска.

Сама функция работает следующим образом: берет экземпляр и проверяет его на столкновения только с твердыми отмеченными экземплярами при размещении в позиции, заданной аргументами x/y. Проверка столкновений может быть как точной, так и основанной на ограничивающей рамки экземпляра, в зависимости от того, какая маска столкновений была выбрана.

Обратите внимание, что перед выполнением проверки заданные координаты x/y будут приведены к ближайшему целому числу.

Синтаксис:

place_free(x, y);

Аргумент Описание
x Позиция x для проверки.
y Позиция y для проверки.

Возвращается:

Булево

Пример:

if place_free(mouse_x, mouse_y)
{
x = mouse_x;
y = mouse_y;
}

Приведенный выше код проверит столкновение с любым экземпляром с твердым флагом, если вызывающий экземпляр будет помещен в то же положение, что и мышь. Если столкновение не обнаружено, то координаты x/y экземпляра устанавливаются равными координатам мыши.
