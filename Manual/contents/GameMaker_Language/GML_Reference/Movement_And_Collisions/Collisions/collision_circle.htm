столкновение_круга

Collision_circle создает вокруг точек x1,y1 круговую зону заданного пользователем радиуса, а затем проверяет эту зону на столкновение с любым экземпляром объекта, указанным в аргументе "obj". Эта проверка может быть точной или нет, но для включения точных столкновений объект или экземпляр, который вы проверяете, должен также иметь включенные точные столкновения для своего спрайта. В противном случае проверка по умолчанию основывается на граничных коробках. На следующем изображении показано, как это работает:

Здесь экземпляр в центре использует круг столкновений для проверки наличия шариков. объектов. Теперь синие объекты не имеют точной границы, и, как вы можете видеть, даже если спрайт не касается круга, столкновение все равно может произойти (даже если вы установите параметр exact в функции в true), так как граница область этого спрайта выходит за пределы круговой области, определяемой collision_circle. С другой стороны, зеленые шарики будут считаться столкнувшимися только в том случае, если реальный спрайт пересекает заданную окружность. Помните, что для точного учета столкновений и спрайт объекта, и функция столкновения должны быть отмечены как точные. Следует также отметить, что возвращаемым значением функции может быть id любого из экземпляров, считающихся столкнувшимися.

Обратите внимание, что вместо индекса объекта вы можете указать экземпляр id для проверки конкретного экземпляра, или ключевые слова экземпляра all, или other (в зависимости от события и текущей области видимости кода).

Синтаксис:

collision_circle(x1, y1, rad, obj, prec, notme);

Аргумент Описание
x1 Координата x центра окружности для проверки.
y1 Координата y центра круга для проверки.
rad Радиус (расстояние в пикселях от центра до края).
obj Объект для проверки столкновений экземпляров.
prec Основана ли проверка на коллизиях с точностью до пикселя (true = медленно) или на его граничной области в целом (false = быстро).
notme Должен ли вызывающий экземпляр, если он релевантен, быть исключен(true) или нет(false).

Возвращается:

Идентификатор экземпляра или нет

Пример:

if collision_circle(x, y, 20, obj_Cursor, false, true)
{
image_index = 1;
}
else image_index = 0;

Приведенный выше код проверит круглую область с радиусом 20 пикселей на столкновение с "obj_Cursor", и если оно есть, то установит image_index объекта, выполняющего код, равным 1, а если нет, то установит image_index объекта в 0.
