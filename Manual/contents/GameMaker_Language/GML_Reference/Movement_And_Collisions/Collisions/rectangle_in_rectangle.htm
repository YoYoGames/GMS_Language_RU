прямоугольник_в_прямоугольнике

Эта функция проверит две заданные вами прямоугольные области на предмет того, не сталкивается ли исходный прямоугольник, находится ли он полностью в границах прямоугольника назначения, или они просто соприкасаются. Если они вообще не соприкасаются, функция вернет 0, если исходный прямоугольник полностью находится внутри конечного, то вернет 1, а если они просто перекрываются, то вернет 2. Изображение ниже иллюстрирует это:

Синтаксис:

rectangle_in_rectangle(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2);

Аргумент Описание
sx1 Координата x левой стороны исходного прямоугольника.
си1 Координата y верхней стороны исходного прямоугольника.
sx2 Координата x правой стороны исходного прямоугольника.
си2 Координата y нижней стороны исходного прямоугольника.
dx1 Координата x левой стороны целевого прямоугольника для проверки.
dy1 Координата y верхней стороны целевого прямоугольника для проверки.
dx2 Координата x правой стороны целевого прямоугольника для проверки.
dy2 Координата y нижней стороны целевого прямоугольника для проверки.

Возвращается:

Реально

Пример:

switch(rectangle_in_rectangle(x - 50, y - 50, x + 50, y + 50, mouse_x - 50, mouse_y - 50, mouse_x + 50, mouse_y + 50))
{
case 0: image_blend = c_yellow; break;
case 1: image_blend = c_red; break;
case 2: image_blend = c_orange; break;
}

Приведенный выше код проверяет две прямоугольные области на столкновение и устанавливает экземпляр image_blend на основе возвращаемого значения.
