прямоугольник_в_треугольнике

Эта функция проверит заданную вами прямоугольную область на предмет отсутствия столкновений, полного попадания в границы назначения или просто касания с заданной треугольной областью. Если они вообще не соприкасаются, функция вернет 0, если источник полностью находится в пределах места назначения, то вернет 1, а если они просто пересекаются, то вернет 2. Изображение ниже иллюстрирует это:

Синтаксис:

rectangle_in_triangle(sx1, sy1, sx2, sy2, dx1, dy1, dx2, dy2, dx3, dy3);

Аргумент Описание
sx1 Координата x левой стороны исходного прямоугольника.
си1 Координата y верхней стороны исходного прямоугольника.
sx2 Координата x правой стороны исходного прямоугольника.
си2 Координата y нижней стороны исходного прямоугольника.
x1 Координата x первой стороны треугольника для проверки.
y1 Координата y первой стороны треугольника для проверки.
x2 Координата x второй стороны треугольника для проверки.
y2 Координата y второй стороны треугольника для проверки.
x3 Координата x третьей стороны треугольника для проверки.
y3 Координата y третьей стороны треугольника для проверки.

Возвращается:

Реально

Пример:

var inst = instance_nearest(x, y, obj_Player);
if instance_exists(inst)
{
var x1 = x + lengthdir_x(100, image_angle - 45);
var y1 = y + lengthdir_y(100, image_angle - 45);
var x2 = x + lengthdir_x(100, image_angle + 45);
var y2 = y + lengthdir_y(100, image_angle + 45);
if rectangle_in_triangle(inst.bbox_left, inst.bbox_top, inst.bbox_right, inst.bbox_bottom, x, y, x1, y1, x2, y2) == 1
{
can_see = true;
}
}

Приведенный выше код использует функцию rectangle_in_triangle в качестве "конуса зрения" для проверки наличия экземпляра "obj_player". Если полное граничное поле находится в пределах заданной треугольной области, то переменная будет установлена в true.
