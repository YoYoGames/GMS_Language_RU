перемещение_контакта_твердого

Эта функция будет перемещать экземпляр, выполняющий код, на заданное количество пикселей в указанном направлении, пока не встретит экземпляр с флажком solid отмеченным как true. Вы можете использовать -1 или 0 для максимального расстояния, которое по умолчанию равно 1000px, т.е.: GameMaker Studio 2 будет перемещать экземпляр постоянно на 1000 пикселей, пока он не выйдет из столкновения.

Синтаксис:

move_contact_solid( dir, maxdist );

Аргумент Описание
dir Направление, в котором нужно двигаться.
maxdist Максимальное расстояние, которое может пройти объект (-1 или 0, значение по умолчанию - 1000 пикселей).

Возвращается:

Н/Д

Пример:

if !place_meeting(x, y + 1, obj_Floor)
{
move_contact_solid(270, -1);
}

Приведенный выше код будет проверять под экземпляром на столкновение с "obj_Floor", и если такового нет, то будет перемещать его вниз, пока не произойдет столкновение или не будет пройдено 1000 пикселей.
