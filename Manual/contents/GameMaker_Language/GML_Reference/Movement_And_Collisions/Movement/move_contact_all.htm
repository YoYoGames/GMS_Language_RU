move_contact_all

Эта функция будет перемещать экземпляр, выполняющий код, на заданное количество пикселей в указанном направлении, пока он не встретит любой другой экземпляр с действующей маской. Вы можете использовать -1 или 0 для maxdist, которое по умолчанию равно 1000px, т.е.: GameMaker Studio 2 будет перемещать экземпляр постоянно на 1000 пикселей, пока он не выйдет из столкновения.

Синтаксис:

move_contact_all(dir, maxdist);

Аргумент Описание
dir Направление, в котором нужно двигаться.
maxdist Максимальное расстояние, которое может пройти объект (-1 или 0, значение по умолчанию - 1000 пикселей).

Возвращается:

Н/Д

Пример:

if !place_meeting(x, y + 1, all)
{
move_contact_all(270, -1);
}

Приведенный выше код будет проверять экземпляр на наличие столкновения, и если его нет, то он будет перемещать его вниз, пока не появится или пока не будет пройдено 1000 пикселей.
