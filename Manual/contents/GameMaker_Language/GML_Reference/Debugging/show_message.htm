показать_сообщение

Эта функция создает всплывающее окно сообщения, в котором отображается заданная строка и кнопка с надписью "Ok" для его закрытия.

ПРИМЕЧАНИЕ: Эта функция предназначена для отладки только для целей Desktop и UWP, а для всех остальных целей она устарела.

Синтаксис:

show_message(str);

Аргумент Описание
str Строка для отображения во всплывающем сообщении.

Возвращается:

Н/Д

Пример:

var tot = 0;
for (var i = 0; i < 10; i += 1)
{
tot += inv[i];
}
show_message("Total = " + string(tot));

Приведенный выше код будет циклически просматривать значения, хранящиеся в массиве "inv", и добавлять их к переменной "tot", а затем выводить сообщение с итоговой суммой.
