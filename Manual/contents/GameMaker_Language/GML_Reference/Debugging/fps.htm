fps

В GameMaker Studio 2 есть два основных способа, которые можно использовать для определения скорости, с которой работает ваша игра. скорость игры (как указано в опциях игры) и fps (кадры в секунду). Эти значения часто путают, но в основном одно из них - это количество игровых шагов, которые GameMaker Studio 2 должна выполнить за секунду, а другое (fps) - количество шагов процессора, которые GameMaker Studio 2 действительно выполняет за секунду, вплоть до максимального значения скорости самой комнаты. Чтобы получить истинное значение fps, т.е. фактическое количество шагов процессора за один игровой шаг, используйте переменную fps_real переменную.

Эта переменная, доступная только для чтения, возвращает текущий fps в виде целочисленного значения. Обратите внимание, что функция будет обновляться только один раз на каждом шаге вашей игры, поэтому может показаться, что значение "прыгает" от одного к другому, но это вполне нормально.

Синтаксис:

fps

Возвращается:

Реально

Пример:

if debug_mode
{
draw_text(32, 32, "FPS = " + string(fps));
}

Приведенный выше код проверит, находится ли игра в режиме отладки, и если да, то отобразит текущий fps на экране.

Назад: Отладка
Далее: fps_real
