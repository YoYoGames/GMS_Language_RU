режим отладки

Эта переменная, доступная только для чтения, возвращает true, если игра ведется в режиме отладки, и false, если игра ведется в обычном режиме.

Синтаксис:

режим отладки

Возвращается:

Булево

Пример:

if debug_mode
{
ini_open("Cheats.ini");
}
else
{
ini_open("Game.ini");
}

Приведенный выше код открывает другой ini-файл в зависимости от значения переменной debug_mode, доступной только для чтения.

Назад: Отладка
Далее: debug_event
