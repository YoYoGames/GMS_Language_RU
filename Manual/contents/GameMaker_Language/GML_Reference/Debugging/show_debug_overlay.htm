show_debug_overlay

Эта функция может использоваться для включения и выключения стандартной отладочной накладки при тестировании игры и по умолчанию отключена. Отладочная накладка показывает графическую полосу использования CPU/GPU в самом игровом окне вместе с текущим реальным значением fps, количество смен текстур и количество вершинных партий (обратите внимание, что смены текстур и вершинных партий никогда не будут равны нулю и обычно показывают значения 2 или 3, поскольку даже при пустой комнате и отсутствии объектов GameMaker Studio 2 все равно приходится рисовать и выполнять пакетную обработку). вещи).

Этот бар разделен на секции, каждая из которых составляет 1/60 секунды. Как видно из приведенного выше изображения, полоса состоит из различных цвета:

  • Зеленый - обработка ввода/вывода (т.е.: клавиатура, мышь, геймпад, сеть и т.д...)
  • Красный - Скорость обновления события шага
  • Желтый - Время, необходимое для проведения розыгрыша
  • Оранжевый -Время обновления отладки, которое обычно видно только при использовании модуля отладки
  • Белый - оставшееся время GPU, то есть время, потраченное на ожидание, пока GPU закончит рендеринг кадра, прежде чем начнется следующий.
  • Cyan - Время отрисовки текста
  • Grey - Время, необходимое для очистки экрана на каждом этапе рисования
  • Blue - Время, необходимое для работы сборщика мусора
  • Темно-красный цвет - промывка GPU, т.е. сколько времени требуется GPU для очистки изображений из памяти.

Используя эту функцию, вы можете добавить наложение в режиме отладки или нет, и таким образом вы сможете увидеть, насколько эффективно работает ваша игра, и получить визуальную подсказку о том, как она использует доступные ресурсы, без лишних затрат, связанных с запуском отладчика. рядом.

ПРИМЕЧАНИЕ: Эта функция не будет работать на цели HTML5 из-за того, что она работает иначе, чем все остальные цели.

Синтаксис:

show_debug_overlay(enable);

Аргумент Описание
включить включение(true) или выключение(false) наложения отладки.

Возвращается:

Н/Д

Пример:

if global.debug
{
show_debug_overlay(true);
}
else
{
show_debug_overlay(false);
}

Приведенный выше код будет включать или выключать отладку в зависимости от значения глобальной переменной.
