переменная_глобальная_гет

С помощью этой функции вы можете получить значение из заданной именованной глобальной переменной. Вы указываете имя глобальной переменной, значение которой необходимо получить, в виде строки (см. пример кода ниже), и функция возвращает значение хранящееся в глобальной переменной, или undefined, если переменная не существует.

ВАЖНО! Если глобальная переменная, которую вы получаете, не существует, то функция вернет ключевое слово undefined, и вы можете получить ошибки, которые остановят работу игры, поэтому, если вы сомневаетесь, используйте функцию используйте функцию переменная_глобальная_существует сначала.

Синтаксис:

variable_global_get(name);

Аргумент Описание
имя Имя глобальной переменной, которую нужно получить (в виде строки)

Возвращается:

Значение (любой тип данных) или undefined (если именованная переменная не существует)

Пример:

if variable_global_exists("enemy_num")
{
show_debug_message("enemy_num = " + string(variable_global_get("enemy_num"));)
}

Приведенный выше код проверит, существует ли глобальная переменная, и если да, то выведет ее на консоль.
