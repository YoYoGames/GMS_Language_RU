переменная_объекта_get

С помощью этой функции вы можете получить значение из заданной именованной переменной. Вы предоставляете уникальное значение ID экземпляра (которое можно найти в Свойствах экземпляра в редакторе комнаты). редактора или возвращается при вызове функции instance_create_layer()), а также имя переменной, значение которой нужно получить в виде строки (см. пример кода ниже). Функция вернет значение переменной или undefined, если переменная не существует.

ВАЖНО! Если переменная, которую вы получаете, не существует, то функция вернет ключевое слово undefined, и вы можете получить ошибки, которые остановят работу игры, поэтому, если вы сомневаетесь, используйте функцию переменная_экземпляр_существует first.

Синтаксис:

variable_instance_get(instance_id, name);

Аргумент Описание
идентификатор экземпляра Уникальное значение ID экземпляра для использования
имя Имя переменной, которую нужно получить (в виде строки)

Возвращается:

Значение (любой тип данных) или undefined (если именованная переменная не существует)

Пример:

if variable_instance_exists(id, "shields")
{
var ss = variable_instance_get(id, "shields");
}
else
}
var ss = -1;
}

Приведенный выше код проверяет, существует ли переменная, и если существует, то значение, которое она содержит, извлекается и сохраняется в локальной переменной. Если она не существует, то локальная переменная устанавливается в -1.
