is_real

Эта функция возвращает, является ли данная переменная действительным числом (единичным, двойным или целым) или нет. В некоторых случаях вы хотите проверить, является ли переменная вещественным числом, и тогда вы используете эту функцию. Она не возвращает реальное число, а скорее true или false, поэтому значение, например, "fish" вернет false, а значение 200 - true.

ПРИМЕЧАНИЕ: Эта функция вернет false для любых значений перечислений, поскольку они хранятся как int64. Правильной функцией для проверки этого типа данных является is_int64().

Синтаксис:

is_real(n);

Аргумент Описание
n Аргумент для проверки.

Возвращается:

Булево

Пример:

if is_real(val)
{
score += val;
}

Приведенный выше код проверяет переменную "val" на то, является ли она действительным числом, и если да, то добавляет ее к счету.
