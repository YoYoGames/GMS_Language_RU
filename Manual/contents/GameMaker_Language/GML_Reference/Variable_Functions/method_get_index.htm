метод_get_index

С помощью этой функции вы можете получить индекс сценария, в котором был определен метод. Если метод не был определен в скрипте, то функция вернет значение -1, в противном случае она вернет значение индекса для скрипта.

Синтаксис:

method_get_index(method);

Аргумент Описание
метод Переменная метода для проверки

Возвращается:

Целое число

Пример:

var _index = method_get_index(light_setup);
if _index != -1
{
show_debug_message(script_get_name(_index));
}

Приведенный выше код проверит индекс скрипта на наличие метода "light_setup", и если он не равен -1, то выведет имя скрипта в консоль.
