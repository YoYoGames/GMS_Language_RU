is_method

Эта функция может использоваться для проверки и определения того, является ли переменная переменной метода (в этом случае она вернет true) или нет (в этом случае она вернет false).

Синтаксис:

is_method(n);

Аргумент Описание
n Переменная для проверки.

Возвращается:

Булево

Пример:

if is_method(get_vec)
{
show_debug_message("Переменная метода!");
}

Приведенный выше код проверяет переменную на предмет того, является ли она методом, и если функция возвращает true, то в консоль выводится отладочное сообщение.
