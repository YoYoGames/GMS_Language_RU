is_bool

Эта функция возвращает, является ли данная переменная булевой(true или false) или нет. В некоторых случаях необходимо проверить, является ли переменная в GameMaker Studio 2 булевым значением. и тогда вы используете эту функцию.

Синтаксис:

is_bool(n);

Аргумент Описание
n Аргумент для проверки.

Возвращается:

Булево

Пример:

if is_bool(val)
{
global.Sound = val;
}
else
{
global.Sound = true;
}

Приведенный выше код проверяет переменную "val" на то, является ли она действительным числом, и если да, то использует ее для установки глобальной переменной.
