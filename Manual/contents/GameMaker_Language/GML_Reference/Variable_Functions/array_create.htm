массив_создать

С помощью этой функции вы можете создать одномерный массив заданного размера. Вы сообщаете функции длину массива, который нужно создать, и она возвращает "хэндл" для массива, который вы можете присвоить переменной. Массивы, созданные таким образом, будут иметь каждая запись будет инициализирована 0, если вы не укажете (необязательное) инициализирующее значение. Если вы укажете дополнительное значение для инициализации массива, то все индексы в новом массиве будут установлены в это значение вместо 0. в этом случае будет иметь большую производительность.

Синтаксис:

array_create(size, [value]);

Аргумент Описание
размер Размер создаваемого массива.
значение Значение, используемое для инициализации всех индексов массива (необязательно).

Возвращается:

Ссылка на массив

Пример:

instance_array = array_create(100, noone);

Приведенный выше код создаст новый массив из 100 записей, каждая из которых будет установлена на ключевое слово noone, а затем присвоит его переменной "instance_array".
