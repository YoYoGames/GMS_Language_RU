переменная_объекта_набора

С помощью этой функции вы можете установить значение заданной переменной в экземпляре. Вы предоставляете уникальное значение ID экземпляра (которое можно найти в Свойствах экземпляра в редакторе помещения или возвращается при вызове функции instance_create_layer()), а также имя переменной, значение которой нужно установить, в виде строки (см. пример код ниже), и, наконец, значение, которое нужно установить (может быть любым допустимым типом данных). Если переменная еще не существует в экземпляре, она будет создана, а затем ей будет присвоено значение.

Синтаксис:

variable_instance_set(instance_id, name, val);

Аргумент Описание
идентификатор экземпляра Уникальное значение ID экземпляра для использования
имя Имя переменной, которую нужно установить (в виде строки)
val Значение, в которое нужно установить переменную

Возвращается:

Н/Д

Пример:

if !variable_instance_exists(id, "shields")
{
variable_instance_set(id, "shields", 0);
}

Приведенный выше код проверит, существует ли переменная экземпляра в вызывающем экземпляре, и если нет, то она будет создана и установлена в 0.
