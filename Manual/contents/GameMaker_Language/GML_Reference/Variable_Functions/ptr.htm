ptr

Эта функция пытается преобразовать заданное значение в тип данных указателя, где значение должно быть либо real, либо string, либо int64, либо int32, или ptr. Любое другое значение приведет к аварийному завершению игры с сообщением об ошибке.

Синтаксис:

ptr(n);

Аргумент Описание
n Значение для преобразования.

Возвращается:

Указатель

Пример:

if !is_ptr(val)
{
val = ptr(application_surface);
}

Приведенный выше код проверяет переменную "val" на наличие в ней указателя, и если его нет, то ему присваивается указатель.
