array_resize

С помощью этой функции вы можете изменить размер существующего массива на новый. Вы предоставляете массив, размер которого нужно изменить, а также новое количество индексов для массива, и функция изменит размер этого массива. Обратите внимание, что эта функция предназначена для изменения размера массива до меньшей длины, так как вы можете изменить размер, просто установив новый индекс в массиве. При этом, если вы используете ее для изменения размера массива, то все новые индексы будут установлены в значение по умолчанию 0.

Эта функция также может быть использована для многомерных массивов, если при указании индекса массива вы укажете, какое измерение вы хотите изменить, следуя этой схеме:

// Изменение размера первой размерности массива
array_resize(my_array, 10);

// Изменение размера второй размерности массива
array_resize(my_array[0], 10);

// Изменение размера третьей размерности массива
array_resize(my_array[0][0], 10);

// и т.д....

Синтаксис:

array_resize(array_index, new_size);

Аргумент Описание
индекс_массива Индекс массива для изменения размера.
новый_размер Новый размер массива (целое число, начиная с 0).

Возвращается:

Н/Д

Пример:

if array_length(a) > 10
{
array_resize(a, 10);
}

Приведенный выше код проверяет длину массива, и если он имеет более 10 индексов, то его размер изменяется.
