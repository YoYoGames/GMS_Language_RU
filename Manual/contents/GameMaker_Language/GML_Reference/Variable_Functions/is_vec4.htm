is_vec4

В некоторых случаях вы хотите проверить и узнать, какой тип данных имеет переменная в GameMaker Studio 2, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того. от того, является ли значение вектором из 4 значений или нет.

Синтаксис:

is_vec4(n);

Аргумент Описание
n Аргумент для проверки.

Возвращается:

Булево

Пример:

if !is_vec4(val)
{
show_debug_message("Not a 4 value vector!");
}

Приведенный выше код проверяет переменную "val" на наличие в ней vec4, и если это не так, то выводит сообщение в консоль отладки.
