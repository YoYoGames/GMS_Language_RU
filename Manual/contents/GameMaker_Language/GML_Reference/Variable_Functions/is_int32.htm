is_int32

Эта функция возвращает, является ли данная переменная 323-битным целым числом или нет. В некоторых случаях вы хотите проверить и узнать, какой тип данных имеет переменная в GameMaker Studio 2, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того, является ли данное значение целым числом 32 или нет.

Синтаксис:

is_int32(n);

Аргумент Описание
n Аргумент для проверки.

Возвращается:

Булево

Пример:

if !is_int32(val)
{
show_debug_message("Not a 32 bit integer!");
}

Приведенный выше код проверяет переменную "val" на то, содержит ли она int32, и если нет, то выводит сообщение в консоль отладки.
