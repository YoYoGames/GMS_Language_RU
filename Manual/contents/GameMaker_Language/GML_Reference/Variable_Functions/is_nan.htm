is_nan

Эта функция возвращает, является ли данная переменная NaN (не числом) или нет, возвращая true, если является, и false, если не является.

Синтаксис:

is_nan(n);

Аргумент Описание
n Аргумент для проверки.

Возвращается:

Булево

Пример:

if is_nan(global.value)
{
show_debug_message("Value is not a number");
}

Приведенный выше код проверяет глобальную переменную "value" на то, является ли она числом или нет, и показывает отладочное сообщение, если это не так.
