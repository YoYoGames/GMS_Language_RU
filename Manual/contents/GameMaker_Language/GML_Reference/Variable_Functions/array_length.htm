длина_массива

С помощью этой функции вы можете получить длину (количество записей) размерности массива. Вы вводите значение индекса массива, и функция возвращает целочисленное значение, представляющее количество записей, которые содержит массив. Эта функция также может быть использована для многомерных массивов, если при указании индекса массива указать, длину какого измерения вы хотите получить, следуя следующей схеме:

// Возвращает первую размерность массива
val = array_length(my_array);

// Возвращает вторую размерность массива
val = array_length(my_array[0]);

// Возвращает третью размерность массива
val = array_length(my_array[0][0]);

// и т.д....

Синтаксис:

array_length(array_index);

Аргумент Описание
индекс_массива Индекс массива для проверки.

Возвращается:

Целое число

Пример:

for (var i = 0; i < array_length(a); ++i;)
{
a[i] = -1;
}

Приведенный выше код будет циклически просматривать массив и устанавливать для каждого элемента значение -1.
