instanceof

Эта функция может быть использована для получения имени функции, которая была использована для создания структуры, когда структура была создана с помощью функции-конструктора и функции новый оператор. Вы передаете переменную со ссылкой на struct для проверки, и функция возвращает либо строку с именем функции, либо undefined. Обратите внимание, что если вы передадите функции литерал struct (т.е. struct, созданный без использования функции-конструктора), то она просто вернет строку "struct". Эта функция также может быть использована для проверки того, является ли ссылка на структуру слабая ссылка или в этом случае функция вернет строку "weakref" вместо имени функции, создавшей struct. Более подробную информацию можно найти в функции weak_ref_create()).

Синтаксис:

instanceof(struct_id);

Аргумент Описание
struct Ссылка на структуру, которую следует использовать.

Возвращается:

Строка или неопределенное значение

Расширенный пример:

В этом примере мы должны сначала определить функцию, которая будет использоваться в качестве конструктора для struct:

function init_struct(_a, _b, _c) конструктор
{
a = _a;
b = _b;
c = _c;
}

Затем эту функцию можно использовать вместе с оператором new для создания структуры и заполнения ее переменными, установленными на значения аргументов, используемых в функции:

mystruct = new init_struct(10, 100, "Hello World");

Затем мы можем получить имя функции, которая была использована, следующим образом:

var _name = instanceof(mystruct);
if is_string(_name)
{
show_debug_message(_name);
}
