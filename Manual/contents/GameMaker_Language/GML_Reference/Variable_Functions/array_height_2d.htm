array_height_2d(Исправлено)

ВНИМАНИЕ! Эта функция устарела, так как массивы больше не ограничиваются только 1 или 2 измерениями, и как таковая эта функция предоставляется только для поддержки старых проектов. Все новые проекты должны использовать текущий способ создания и доступа к многомерным массивов (более подробную информацию см. здесь ).

С помощью этой функции вы можете получить высоту (количество записей) первого измерения двумерного массива. Вы задаете массив для проверки, а на выходе функции вы узнаете, сколько начальных записей он содержит. Вы можете получить количество записей для второго измерения массива с помощью функции array_length_2d.

Синтаксис:

array_height_2d(array_index);

Аргумент Описание
индекс_массива Индекс массива для проверки.

Возвращается:

Реально

Пример:

for (var i = 0; i < array_height_2d(a); ++i;)
{
for (var j = 0; j < array_length_2d(a, i); ++j;)
{
a[i, j] = -1;
}
}

Приведенный выше код будет циклически просматривать двумерный массив и устанавливать для каждой записи значение -1.
