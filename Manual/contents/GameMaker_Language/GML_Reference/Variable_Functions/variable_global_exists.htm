переменная_глобальная_существует

С помощью этой функции вы можете проверить, существует ли переменная в глобальной области видимости или нет. Вы передаете имя глобальной переменной, которую нужно проверить, в виде строки (см. пример кода ниже), и функция вернет true, если если глобальная переменная с заданным именем существует, или false в противном случае.

Синтаксис:

variable_global_exists(name);

Аргумент Описание
имя Имя глобальной переменной для проверки (в виде строки)

Возвращается:

Булево

Пример:

if !variable_global_exists("enemy_num"))
{
global.enemy_num = instance_number(obj_Enemey_Parent);
}

Приведенный выше код проверит, существует ли глобальная переменная "enemy_num", и если нет, то она будет создана и ей будет задано значение.
