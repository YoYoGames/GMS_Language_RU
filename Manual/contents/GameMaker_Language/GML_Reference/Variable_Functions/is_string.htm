is_string

Эта функция возвращает, является ли данная переменная строковым числом или нет. В некоторых случаях вы хотите проверить, является ли переменная в GameMaker Studio 2 строкой, а не вещественным числом, и тогда вы можете использовать эту функцию. функцию. Она возвращает не строку, а true или false, поэтому значение, например, "fish" для n вернет true, но значение 200 для n вернет false.

Синтаксис:

is_string(n);

Аргумент Описание
n Аргумент для проверки.

Возвращается:

Булево

Пример:

if is_string(val)
{
name = "Player: " + val;
}

Приведенный выше код проверяет переменную "val" на наличие в ней строки, и если она содержит строку, то добавляет ее в другую строку, которая затем присваивается переменной "name".
