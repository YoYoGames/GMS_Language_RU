переменная_экземпляр_существует

С помощью этой функции вы можете проверить, существует ли переменная области видимости экземпляра или нет. Вы предоставляете уникальное значение ID экземпляра (которое можно найти в свойствах экземпляра в редакторе комнаты или оно возвращается при вызове функции), а также имя переменной для проверки в виде строки (см. пример кода ниже). instance_create_layer()), а также имя переменной для проверки в виде строки (см. пример кода ниже). Функция вернет true, если переменная с заданным именем существует для данного экземпляра, и false в противном случае.

Синтаксис:

variable_instance_exists(instance_id, name);

Аргумент Описание
идентификатор экземпляра Уникальное значение ID экземпляра для проверки
имя Имя переменной для проверки (в виде строки)

Булево

Пример:

if !variable_instance_exists(id, "shields"))
{
shields = 0;
}

Приведенный выше код проверит, существует ли переменная "shields" в экземпляре, выполняющем код, и если нет, то она будет создана и инициализирована в 0.
