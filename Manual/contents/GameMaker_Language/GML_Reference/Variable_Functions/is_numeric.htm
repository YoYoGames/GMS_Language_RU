is_numeric

Эта функция возвращает, является ли данная переменная числовым значением (real, int32, int64 или boolean) или нет. В некоторых случаях вы хотите проверить, имеет ли переменная в GameMaker Studio 2 числовое значение, и тогда тогда вы можете использовать эту функцию. Функция вернет true, если введенное значение является числовым, и false в противном случае.

Синтаксис:

is_numeric(n);

Аргумент Описание
n Вход для проверки.

Возвращается:

Булево

Пример:

if is_numeric(val)
{
current_val += val;
}

Приведенный выше код проверяет переменную "val" на то, является ли она числовым значением, и если да, то добавляет его к заданной переменной.
