is_vec3

В некоторых случаях вы хотите проверить и узнать, какой тип данных имеет переменная в GameMaker Studio 2, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того. в зависимости от того, является ли значение вектором из 3 значений или нет.

Синтаксис:

is_vec3(n);

Аргумент Описание
n Аргумент для проверки.

Возвращается:

Булево

Пример:

if !is_vec3(val)
{
show_debug_message("Not a 3 value vector!");
}

Приведенный выше код проверяет переменную "val" на наличие в ней vec3, и если это не так, то выводит сообщение в консоль отладки.
