physics_particle_group_delete

С помощью этой функции вы можете удалить группу частиц из физического моделирования в текущей комнате. Функция принимает уникальный идентификатор группы, которую нужно удалить, возвращаемый функцией physics_particle_group_end().

Синтаксис:

physics_particle_group_delete(ind)

Аргумент Описание
инд Индекс (ID) группы частиц для удаления.

Возвращается:

Н/Д

Пример:

physics_particle_group_delete(gp1);

Приведенный выше код удалит из симуляции все частицы, составляющие группу с идентификатором, хранящимся в переменной "gp1".
