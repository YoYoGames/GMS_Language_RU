physics_particle_delete_region_poly

С помощью этой функции вы можете удалить все частицы, которые попадают в границы определенной многоугольной области, из физической симуляции в текущей комнате. Функция принимает (предварительно созданный) ds_list() содержащий x/y позицию каждой точки многоугольника, причем четные позиции в списке являются координатами x, а нечетные - координатами y, т.е. для треугольника список будет состоять из шести позиций, где 0, 2 и 4 - координаты x, а 1, 2 и 4 - координаты y. 4 - координаты x, а 1, 3 и 5 - координаты y. Затем полигон будет использоваться для удаления всех частиц, которые попадают в определенную область.

Обратите внимание, что заданный многоугольник должен иметь не менее трех точек и не более 8.

Синтаксис:

physics_particle_delete_region_poly(pointList)

Аргумент Описание
pointList Список ds_list точек для создания многоугольника.

Возвращается:

Н/Д

Пример:

var list = ds_list_create(p_list);
for (var i = 0; i < 5; i ++;)
{
ds_list_add(p_list, mx[i]);
ds_list_add(p_list, my[i]);
}
physics_particle_delete_region_poly(p_list);
ds_list_destroy(p_list);

Приведенный выше код удалит все частицы, находящиеся в многоугольной области, определенной точками, добавленными в список DS.
