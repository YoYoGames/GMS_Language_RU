физика_удаление_приспособления

Эта функция удаляет (или "отвязывает") приспособление от экземпляра или экземпляров. Для этого требуется уникальный "id" привязанного приспособления (возвращаемый функцией physics_fixture_bind() и Она удалит все определенные в данный момент физические свойства для экземпляра, что позволит вам переопределить новое приспособление и привязать его к экземпляру. Таким образом, вы можете изменить физические свойства экземпляра без необходимости уничтожать и создавать его заново. его.

Синтаксис:

physics_remove_fixture(id, fixture)

Аргумент Описание
id Идентификатор экземпляра, из которого нужно удалить приспособление
приспособление Идентификатор приспособления, которое должно быть удалено из экземпляра

Возвращается:

Н/Д

Пример:

physics_remove_fixture(id, my_fix);

Приведенный выше код удалит из экземпляра приспособление с "id", хранящимся в переменной "my_fix".
