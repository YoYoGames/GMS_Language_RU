physics_fixture_set_awake

Когда вы привязываете приспособление к экземпляру с помощью physics_fixture_bind() это возвращает "id" для связанного приспособления. Вы можете использовать этот идентификатор для получения значения трения связанного приспособления(не "базового" приспособления) в любое время с помощью этой функции.

Синтаксис:

physics_get_friction(fixture)

Аргумент Описание
приспособление идентификатор связанного приспособления

Возвращается:

Реально

Пример:

var fric = physics_get_friction(fix_id);
physics_set_friction(fix_id, fric + 0.1);

Приведенный выше код получает текущее значение трения для связанных физических свойств экземпляра, а затем устанавливает их на другое значение.
