physics_fixture_set_angular_damping

Если вспомнить любой вращающийся объект в "реальном мире", если только он не оснащен двигателем или не находится в космосе, то со временем он замедляется под воздействием внешних сил (например, трения с окружающим воздухом). Мы можем использовать функцию physics_fixture_set_angular_damping() в GameMaker Studio 2 для имитации этого эффекта и уменьшения скорости вращения объектов в физическом мире, поскольку без нее любой вращающийся объект будет продолжать вращаться бесконечно. Параметры демпфирования должны находиться в диапазоне от 0 до бесконечности, при этом 0 означает отсутствие демпфирования, а бесконечность - полное демпфирование. Обычно используется значение демпфирования между 0 и 1, но при необходимости можно использовать любое неотрицательное значение.

Синтаксис:

physics_fixture_set_angular_damping(fixture, damping)

Аргумент Описание
приспособление индекс приспособления
демпфирование угловое демпфирование приспособления, обычно от 0 до 1

Возвращается:

Н/Д

Пример:

physics_fixture_set_angular_damping(fix_Ball, 0.1);

Приведенный выше код установит угловое демпфирование приспособления, индексируемого в "fix_ball", равным 0,1.
