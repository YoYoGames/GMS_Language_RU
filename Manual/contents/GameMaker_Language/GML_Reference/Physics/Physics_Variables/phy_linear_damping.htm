phy_linear_damping

Эта переменная может быть использована для установки линейного демпфирования экземпляра или для получения текущего линейного демпфирования. Демпфирование - это величина "сопротивления" движению вперед, которое оказывает экземпляр с включенной физикой, причем меньшее значение позволяет экземпляру двигаться и ускоряться быстрее, а более высокое значение требует более сильного толчка.

Синтаксис:

phy_linear_damping;

Возвращается:

Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику)

Пример:

if place_meeting(phy_position_x, phy_position_y, obj_Water)
{
phy_linear_damping = 10;
}
else
{
phy_linear_damping = 3;
}

Приведенный выше код проверит столкновение между вызывающим экземпляром и экземплярами "obj_Water" и соответствующим образом изменит линейное затухание.
