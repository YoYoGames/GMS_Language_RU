phy_kinematic

Эта переменная, доступная только для чтения, возвращает true, если экземпляр классифицируется как кинематический объект, или false, если это не так. Кинематический объект - это объект, который имеет бесконечную массу (плотность равна 0), но может двигаться. Поэтому, чтобы сделать экземпляр кинематическим, вы сначала создадите статический экземпляр, а затем установите одну или несколько переменных экземпляра, связанных с движением (например: phy_speed_x, phy_speed_y, или phy_angular_velocity)

Синтаксис:

phy_kinematic;

Возвращается:

Булево (или неопределено, если экземпляр не включен в физику)

Пример:

if !phy_kinematic
{
phy_speed_x = 5;
}

Приведенный выше код проверяет, является ли экземпляр кинематическим, и если нет, то устанавливает горизонтальную скорость равной 5.
