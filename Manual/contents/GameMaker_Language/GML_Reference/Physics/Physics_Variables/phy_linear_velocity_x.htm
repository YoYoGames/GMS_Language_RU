phy_linear_velocity_x

Эта переменная может использоваться для получения или изменения x-компонента вектора линейной скорости экземпляра и определяется в пикселях в секунду (для пикселей на шаг см. phy_speed_x). Изменение этого параметра для статического экземпляра (т.е. экземпляра с плотностью 0) превратит его в кинематический экземпляр

Синтаксис:

phy_linear_velocity_x;

Возвращается:

Real (значение с плавающей точкой одинарной точности, или неопределено, если в экземпляре не включена физика)

Пример:

если (phy_linear_velocity_x != 0)
{
phy_linear_velocity_x = 0;
}

Приведенный выше код проверит x-компонент вектора линейной скорости и, если он не равен 0, то установит его в 0.
