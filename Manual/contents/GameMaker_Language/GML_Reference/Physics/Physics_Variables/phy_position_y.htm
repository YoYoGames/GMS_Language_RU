phy_position_y

Эта переменная может быть использована для получения (или установки) позиции y экземпляра в физическом мире игровой комнаты. Обратите внимание, что физический мир может выдавать ошибки при перемещении экземпляров путем непосредственной установки этой переменной, так как это прервет непрерывное моделирование. Эта переменная является физическим эквивалентом переменной экземпляра y.

Синтаксис:

phy_position_y;

Возвращается:

Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику)

Пример:

xx = phy_position_x;
yy = phy_position_y;

Приведенный выше код хранит позицию экземпляра x и y в двух переменных.
