физический_шарнир_удалить

После того, как два экземпляра с физическими представлениями были связаны суставом, его можно удалить в любое время. Обычно это происходит автоматически, когда один из двух экземпляров уничтожается, или когда комната заканчивается, но бывают случаи, когда вы может возникнуть необходимость сделать это вручную. В этих случаях можно воспользоваться данной функцией.

Следует также отметить, что при уничтожении экземпляра, являющегося частью зубчатого соединения, зубчатое соединение должно быть удалено с помощью этой функции до того, как будут удалены все экземпляры, участвующие в формировании зубчатого соединения (но оставшиеся соединения будут удалены автоматически), например, в событии destroy. (но остальные соединения будут удалены автоматически), например, в событии destroy экземпляра.

Синтаксис:

physics_joint_delete(joint)

Аргумент Описание
совместный Индекс сустава, который вы хотите удалить

Возвращается:

Н/Д

Пример:

var reactionForceX, reactionForceY, reactionForce;
if shipJoint
{
reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
reactionForce = sqrt((reactionForceX + reactionForceX) + (reactionForceY + reactionForceY));
if reactionForce > 2
{
physics_joint_delete(shipJoint);
shipJoint = -1;
}
}

Приведенный выше код проверяет, содержит ли переменная "shipJoint" индекс сустава, и если да, то рассчитывает силу, приложенную к этому суставу, используя две константы. Наконец, если общая сила больше 2, сустав удаляется.
