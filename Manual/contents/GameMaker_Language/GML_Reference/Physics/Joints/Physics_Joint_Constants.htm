Физика Совместные константы

В GameMaker Studio 2 GML включено большое количество констант, специфичных для соединений между креплениями в физическом мире. Их можно использовать в сочетании с многими функциями суставов, чтобы задавать или получать информацию от них в реальном времени во время работы физического моделирования. Однако следует помнить, что при их вызове выполняются сложные вычисления, поэтому их следует использовать с осторожностью и только при необходимости, и обратите внимание, что многие из них уникальны для определенного типа суставов. Также обратите внимание, что хотя вы можете получить все эти значения с помощью соответствующей функции, вы можете установить только те, которые помечены как не доступные только для чтения.

В общем случае эти константы используются вместе со следующими функциями:

  • физический_шарнир_получить_значение
  • физическое_соединение_установить_значение

Следующие константы могут быть применены к любому из имеющихся типов соединений:

Постоянная Описание Только чтение
phy_joint_anchor_1_x Координата x первой опорной точки сустава в помещении Да
phy_joint_anchor_1_y Координата y первой опорной точки сустава в помещении Да
phy_joint_anchor_2_x Координата x второй опорной точки сустава в помещении Да
phy_joint_anchor_2_y Координата y второй опорной точки сустава в помещении Да
phy_joint_reaction_force_x Это сила реакции, приложенная ко второму экземпляру в суставе в положении привязки x Да
phy_joint_reaction_force_y Это сила реакции, приложенная ко второму экземпляру в суставе в положении привязки y Да
phy_joint_reaction_torque Это крутящий момент, прикладываемый ко второму экземпляру в суставе в положении якоря Да


Эти константы предназначены для тех шарниров, к которым присоединен двигатель (револьверный, призматический, колесный):

Постоянная Описание Только чтение
phy_joint_max_motor_force Значение, указанное при создании сустава для максимального усилия двигателя Нет
phy_joint_max_motor_torque Значение, указанное при создании соединения для максимального крутящего момента двигателя Нет
phy_joint_motor_force Текущая сила двигателя Да
phy_joint_motor_speed Текущая скорость вращения двигателя Нет
phy_joint_motor_torque Текущий крутящий момент двигателя Да


Для револьверного шарнира вы можете использовать следующую константу (а также константы двигателя, если он был добавлен):

Постоянная Описание Только чтение
фи_суставной_угол Угол, который образует линия между двумя опорными точками сустава. Этот угол рассчитывается с использованием координат физического мира
( не координат комнаты в GameMaker Studio 2 ) в радианах. координаты) в радианах.		 Да
phy_joint_angle_limits Включите или отключите ограничение угла для сустава. Установите значение true для включения или false для отключения. Нет
phy_joint_upper_angle_limit Верхний предел угла для сустава в градусах. Нет
phy_joint_lower_angle_limit Нижний предел угла для сустава в градусах. Нет


Для призматического соединения можно использовать следующую константу:

Постоянная Описание Только чтение
phy_joint_translation Получает расстояние между координатами x/y якоря и локальными координатами x/y. Да
phy_joint_speed Текущая скорость движения сустава. Да


Для расстояния, сварного шва и колесных соединений можно использовать следующие константы (как и для шкивных соединений):

Постоянная Описание Только чтение
phy_joint_damping_ratio Коэффициент демпфирования является безразмерным и определяет "пружинистость" соединения. Значение этой константы обычно находится в диапазоне от 0 до 1, но может быть и больше, а при значении 1 демпфирование является критическим, то есть все колебания должны исчезнуть. Нет
phy_joint_frequency Это вернет (или установит) частоту колебаний для сустава, в герцах, и обычно частота должна быть меньше половины частоты временного шага, установленного функцией physics_world_update_speed(). Нет
phy_joint_length_1 Это вернет длину сустава от первой локальной координаты x/y до первой якорной координаты x/y (только для дистанционных суставов, может быть считано только из) Да
phy_joint_length_2 Это вернет длину сустава от второй локальной координаты x/y до второй якорной координаты x/y (только для суставов с расстоянием, может быть записано только в) Нет


Для фрикционного соединения можно использовать следующие константы:

Постоянная Описание Только чтение
phy_joint_max_torque Максимальное значение крутящего момента для соединения. Нет
phy_joint_max_force Максимальное значение силы для сустава. Нет


Для веревочного соединения можно использовать следующую константу:

Постоянная Описание Только чтение
phy_joint_max_length Максимальное удлинение для соединения между двумя точками крепления. Нет
