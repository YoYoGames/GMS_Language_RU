Используя ряд предопределенных констант, вы можете попросить GameMaker Studio 2 сообщить вам ряд сведений о состоянии любого сустава. Это очень полезно, поскольку дает возможность удалять суставы или изменять поведение экземпляра в зависимости от того, что вам нужно в данный момент. в зависимости от того, что вам нужно в данный момент. Существует ряд констант, которые можно использовать в этой функции, и их можно найти здесь: Physics Joint Constants, но имейте в виду, что при их вызове выполняются сложные вычисления. поэтому их следует использовать с осторожностью и только при необходимости, и учтите, что многие из них уникальны для определенного типа суставов.

Если свойство не существует (например, если вы проверяете шарнир шкива на призматический перевод), то возвращаемым значением будет 0.

Синтаксис:

physics_joint_get_value(joint, value)

Аргумент Описание
совместный Индекс сустава, который вы хотите проверить
значение Константа для совместного имущества, которое вы хотите проверить

Возвращается:

Реально

Пример:

if shipJoint
{
var reactionForceX = physics_joint_get_value(shipJoint, phy_joint_reaction_force_x);
var reactionForceY = physics_joint_get_value(shipJoint, phy_joint_reaction_force_y);
var reactionForce = point_distance(0, 0, reactionForceX, reactionForceY);
if reactionForce > 2
{
physics_joint_delete(shipJoint);
shipJoint = -1;
}
}

Приведенный выше код проверяет, содержит ли переменная "shipJoint" индекс сустава, и если да, то рассчитывает силу, приложенную к этому суставу, используя две константы. Наконец, если общая сила больше 2, сустав удаляется.
