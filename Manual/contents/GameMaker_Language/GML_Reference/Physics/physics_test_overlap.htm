физика_тест_перекрытие

Эта функция может быть использована для проверки и определения, перекрывает ли физическое тело (т.е. приспособление экземпляра) или будет ли оно перекрывать друг друга при вращении и размещении в заданной позиции в комнате. аргумент "angle" - это угол поворота, который имеет (или будет иметь) вызывающий экземпляр в проверяемой позиции. угол поворота, который имеет (или будет иметь) вызывающий экземпляр в проверяемой позиции, а аргумент "obj" может быть либо одним идентификатором экземпляра, либо индексом объекта, либо ключевые слова все или другие.

Синтаксис:

physics_test_overlap(xpos, ypos, angle, obj);

Аргумент Описание
xpos Х положение в комнате для проверки
ypos Положение y в комнате для проверки
угол Угол для проверки (вызывающего экземпляра)
obj Объект для проверки

Возвращается:

Булево

Пример:

if physics_test_overlap(x+20, y-35, 0, obj_Bomb)
{
alarm[0] = room_speed;
ignited = true;
}

Приведенный выше код проверяет позицию на наличие перекрытия физического приспособления, и если оно есть, то устанавливает переменную и сигнал тревоги.
