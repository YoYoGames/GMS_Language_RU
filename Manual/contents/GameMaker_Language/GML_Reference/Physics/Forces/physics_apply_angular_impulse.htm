физика_применить_угловой_импульс

Эта функция придаст угловой импульс экземпляру с включенной физикой. Этот импульс установит угловое вращение на заданную величину, игнорируя текущий крутящий момент, по сути, устанавливая количество "вращения", которое имеет приспособление. Если вы хотите применить угловую силу к экземпляру с помощью крутящего момента, то вам следует использовать функцию physics_apply_torque().

Синтаксис:

physics_apply_angular_impulse(impulse)

Аргумент Описание
импульс Импульс (в Ньютон-метрах), который необходимо приложить

Возвращается:

Н/Д

Пример:

if keyboard_check(vk_up)
{
physics_apply_angular_impulse(10);
}

Приведенный выше код будет давать приспособлению с включенной физикой угловой импульс при нажатии клавиши.
