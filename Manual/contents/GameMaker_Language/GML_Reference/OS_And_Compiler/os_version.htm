os_version

Эта переменная сообщает вам номер версии ОС, на которой запущена ваша игра. Например, если вы запускаете ее на Windows 10, os_version будет равна 655360.

ПРИМЕЧАНИЕ: Если вам нужна дополнительная информация об ОС Windows, вы можете воспользоваться функцией environment_get_variable().

Синтаксис:

os_version

Возвращается:

Реально

Пример:

if (os_type == os_android) && (os_version > 10)
{
global.GFX = 1;
}

Приведенный выше код проверяет тип операционной системы и номер версии, если они оба верны, то глобальная переменная устанавливается в 1.
