scheduler_resolution_set

Эта функция используется для изменения разрешения планировщика потоков Windows. Значение разрешения, указанное в аргументе, должно быть в миллисекундах.

По умолчанию, GameMaker Studio 2 используется разрешение 1 миллисекунда, что является наименьшим допустимым разрешением и обеспечивает наилучшую производительность. Изменение разрешения на более высокое значение может отрицательно сказаться на производительности, но это также может снизить энергопотребление.

Вы можете указать -1 в качестве аргумента, чтобы вернуть разрешение к значению по умолчанию (установленному Windows). Обратите внимание, что изменение разрешения планировщика повлияет на все процессы, запущенные одновременно с вашей игрой.

ПРИМЕЧАНИЕ: Эта функция предназначена Windows только.

Синтаксис:

scheduler_resolution_set(milliseconds);

Аргумент Описание
миллисекунды Новое значение разрешения (в миллисекундах) или -1 по умолчанию

Возвращается:

Н/Д

Пример:

scheduler_resolution_set(2);

В этом примере разрешение планировщика потоков Windows установлено на 2 миллисекунды.
