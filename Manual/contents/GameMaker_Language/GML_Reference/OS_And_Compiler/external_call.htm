внешний_вызов

Если вы создали вызов внешней функции для dll или dylib, используя external_define()вы можете использовать эту функцию для последующего вызова этой функции. Вы передаете имя ранее определенной функции, а также а также все аргументы, которые она требует (каждый аргумент должен иметь правильный тип, либо вещественный, либо строковый), и функция возвращает результат внешнего вызова.

Синтаксис:

external_call(id, args[0...15]);

Аргумент Описание
id Имя функции, которую вы хотите вызвать
args[0...10] Различные аргументы, которые вы хотите передать внешней функции

Возвращается:

Значение (тип возвращаемого значения будет зависеть от определенной функции)

Пример:

my_function = external_define("MyDLL.dll", "MyMin", dll_cdecl, ty_real, 2, ty_real, ty_real);
var _a = external_call(my_function, x, y);

Приведенный выше пример кода вызывает ранее определенную внешнюю функцию и сохраняет возвращаемое значение в локальной переменной.
