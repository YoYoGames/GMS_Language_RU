Предпочтения редактора кривых анимации Параметры кривых анимации используются для определения определенных свойств при работе с редактором кривых анимации. Доступны следующие опции: Автоматическая реакция на удаление кривой: Этот параметр можно использовать для управления всплывающим сообщением, которое показывается при удалении кривой из редактора анимационных кривых. По умолчанию здесь отображается предупреждающее сообщение, но вы можете установить значение "Хорошо", и диалог не будет показан.
