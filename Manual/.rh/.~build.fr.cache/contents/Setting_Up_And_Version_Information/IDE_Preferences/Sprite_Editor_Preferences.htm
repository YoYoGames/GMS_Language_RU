Предпочтения редактора спрайтов Параметры редактора спрайтов используются для определения внешнего вида и функций редактора спрайтов. Существуют следующие параметры: Начало спрайта по умолчанию: Этот параметр позволяет установить положение начала координат по умолчанию для всех новых ресурсов спрайта, которые вы создаете. Значение по умолчанию - верхний левый угол. Вид маски столкновений спрайта по умолчанию: Эта опция позволяет установить тип маски столкновений по умолчанию, которую следует использовать для новых спрайтов. Можно выбрать следующие варианты: прямоугольник, повернутый прямоугольник, эллипс, ромб или точная / точная на кадр. По умолчанию используется прямоугольник. Режим анимации спрайтов по умолчанию: Эта опция позволяет установить режим анимации по умолчанию для спрайтов в редакторе спрайтов. Вы можете выбрать либо "Кадры в секунду", либо"Кадры на игровой кадр", при этом по умолчанию используется значение "Кадры в секунду". Скорость анимации спрайта по умолчанию: Эта опция позволяет установить значение скорости анимации по умолчанию при создании нового актива спрайта. Обратите внимание, что это значение связано с режимом анимации, и, например, 3 кадра в секунду это не то же самое, что 30 кадров на игровой кадр, поэтому убедитесь, что вы правильно установили это значение в соответствии с предпочитаемым режимом. Значение по умолчанию - 30. Ширина спрайта по умолчанию: Эта опция позволяет установить ширину по умолчанию для всех новых создаваемых спрайтов. Значение по умолчанию - 64. Высота спрайта по умолчанию: Эта опция позволяет установить высоту по умолчанию для всех новых спрайтов. Значение по умолчанию - 64. Применять изменение размера и масштаба к началу спрайта: Если этот флажок установлен, то при изменении размера или масштаба спрайта его начало будет изменено, а новое начало будет перемещено/масштабировано в соответствии с новыми значениями спрайта. Обратите внимание, что это применимо только к пользовательским началам спрайтов, которые не заблокированы в редакторе спрайтов. Если начало заблокировано или спрайт использует одно из положений по умолчанию - например, "Центр" или "Сверху слева" и т.д., - то эта опция будет будет выделена серым цветом и недоступна для изменения. По умолчанию включена. Следующая подкатегория также существует для редактора спрайтов: Диалоги подтверждения Доступные здесь параметры управляют сообщениями, появляющимися при необходимости подтверждения какого-либо действия. Вы можете изменить эти настройки, чтобы показывать сообщение всегда или отклонять его при определенном действии (Да или Нет): Автоматическая реакция на импорт файлов: Когда вы импортируете новое изображение, а одно уже существует, по умолчанию вам будет предложено продолжить импорт и перезаписать существующее изображение или отменить операцию. Этот параметр позволяет подавить это сообщение и всегда либо отменять импорт, выбрав "Нет", либо перезаписывать текущий спрайт, выбрав "Да". Значение по умолчанию - "Показывать сообщение". Автоматическая реакция на сбой загрузки изображения: Когда вы импортируете новый образ и импорт не удается, по умолчанию вам будет показано предупреждающее сообщение, информирующее вас о том, что произошел сбой. Эта опция позволяет отключить это сообщение, выбрав "Ок". Значение по умолчанию - "Показать сообщение".
