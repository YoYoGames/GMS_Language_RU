Предпочтения редактора объектов Параметры редактора объектов используются для определения определенных свойств окна редактора объектов. Основной раздел содержит следующие опции: Щелчок для открытия метода события: Этот параметр определяет, нужно ли выполнить двойной или одинарный щелчок на событии объекта, чтобы открыть его. По умолчанию установлено значение двойного щелчка. Наследуемые события альфа: Эта опция позволяет изменить альфа, с которой будут рисоваться все унаследованные события в списке событий объекта. По умолчанию установлено значение 0.5 (значения должны быть между 0 и 1). Формат метки события: Здесь вы можете включить или выключить различные возможные тексты, которые могут быть применены к метке события в списке событий редактора объектов. По умолчанию включено только описание события. Добавить содержимое события по умолчанию: По умолчанию эта опция включена, и все, что она делает, это добавляет простой комментарий в редактор кода для всех новых событий, чтобы напомнить вам, что вы можете добавить туда свой код. Содержание события по умолчанию: Здесь вы можете редактировать комментарии, которые по умолчанию добавляются в окно кода для новых событий. Следующая подкатегория также существует для редактора объектов: Диалоги подтверждения В этом разделе рассматриваются диалоговые окна, которые могут быть показаны при использовании редактора объектов, при этом доступна следующая опция: Автоматическая реакция на преобразование DnD™ в Gamemaker Language: Вы можете использовать правую кнопку мыши в рабочей области Drag and Drop™, чтобы выбрать опцию для преобразования всех Drag and Drop™для этого события в код GML. По умолчанию здесь выводится предупреждающее сообщение с вопросом, хотите ли вы продолжить, но вы можете установить значение "Хорошо", и диалог не будет показан до преобразования. Автоматическая реакция на превышение лимита вершин физики: Эта опция может использоваться для управления всплывающим сообщением, которое показывается, когда вы превышаете допустимое количество вершин в физическом редакторе формы. По умолчанию здесь отображается предупреждающее сообщение, но вы можете установить значение "Ок", и диалог не будет показан. Автоматический выбор при удалении события: Этот параметр можно использовать для управления всплывающим сообщением, которое показывается при удалении события в списке событий редактора объектов. Действием по умолчанию здесь является показ предупреждающего сообщения, но вы можете установить значение "Хорошо", и диалог не будет показан.
