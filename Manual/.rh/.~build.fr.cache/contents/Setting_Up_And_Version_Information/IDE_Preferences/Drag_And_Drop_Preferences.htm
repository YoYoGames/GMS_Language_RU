ПредпочтенияDrag And Drop™ Параметры Drag and Drop™ используются для определения того, как выглядит и как ощущается редактор сценариев Drag and Drop™. Ниже перечислены различные доступные параметры, а также описание того, что каждый из них делает. Расположение панели инструментовDnD™ по умолчанию: По умолчанию панель инструментов DnD™(где хранятся все различные библиотеки действий) располагается справа от редактора, но вы можете использовать эту опцию, чтобы она располагалась по умолчанию слева. Режим видимости обзора узлаDnD™: Обзор узла - это короткий описательный текст, который появляется рядом с определенными действиями в рабочей области действий, например, вот это действие Destroy Instance: Вы можете включить их всегда, включить их только если действие свернуто (настройка по умолчанию), или вообще отключить их. Автоматическая реакция на преобразование DnD™в GameMaker Language: Когда вы конвертируете событие Drag and Drop или сценарий в код GML, по умолчанию вы получите сообщение, предупреждающее о том, что вы собираетесь сделать, и спрашивающее, хотите ли вы продолжать. Если установить этот параметр на "OK", то изменение просто завершится без показа сообщения. Автоматическая реакция на удаление узла DnD™: Когда вы удаляете узел события Drag and Drop, по умолчанию вы получите сообщение с предупреждением о том, что вы собираетесь сделать, и вопросом, хотите ли вы продолжить. Установка этого параметра на "ОК" просто разрешит удаление без показа сообщения. Максимальный размер для миниатюр действий панели инструментов DnD™: В нижней части панели инструментов действий находится ползунок для изменения размера иконок перетаскивания. Этот параметр будет использоваться для определения максимально допустимого размера (по умолчанию 64px). Минимальный размер для миниатюр действий панели инструментов DnD™: В нижней части панели инструментов действий находится ползунок для изменения размера иконок перетаскивания. Этот параметр будет использоваться для определения минимально допустимого размера (по умолчанию 16px). Размер по умолчанию для миниатюр действий на панели инструментов DnD™: Этот параметр будет использоваться для определения размера по умолчанию для всех иконок действий в панели инструментов (по умолчанию 32px). Максимальное количество недавно использованных действий для отображения: На панели инструментов действий в верхней части находится библиотека, в которой хранятся действия, которые вы использовали в последнее время. С помощью этой опции можно изменить максимальное количество последних действий, которые будет показано (по умолчанию 5). Толщина цепи, соединяющей действия DnD™: Определяет толщину графической цепочки, связывающей различные действия в блоке кода DnD (по умолчанию 3px). Время опускания мыши для начала перетаскивания узлов (мс): Когда вы щелкаете на узле и перетаскиваете его, вы перепозиционируете узел в коде действия. Однако если вы на мгновение задержите мышь, а затем перетащите, вы переместите положение узла в рабочей области, но не в порядке кода. Этот параметр устанавливает количество миллисекунд, которое необходимо выждать при нажатой кнопке мыши, прежде чем узел будет перемещен (не перепозиционирован). По умолчанию - 350. Скрыть панель обзора узлов DnD™: Здесь вы можете выбрать, скрыть ли панель обзора редактора кода действий. По умолчанию она отключена, и ее включение уберет из редактора текстовый список цепочек действий. Также есть подменю для комментариев DnD™: Параметры комментариев управляют различными цветовыми и форматирующими параметрами для любых комментариев, которые были были добавлены к сценариям действий Drag and Drop™. К этим параметрам относятся: Цвет: Цвет по умолчанию для текста, записываемого в окно (по умолчанию ARGB #FF53E270). Имя шрифта: Имя шрифта, который необходимо использовать (доступные шрифты зависят от установленных в вашей системе). Стиль: Используемый стиль шрифта, например, жирный или курсив (обратите внимание, что не во всех окнах можно использовать одинаковые стили, а также это зависит от используемого шрифта). Размер: Размер шрифта для данного окна вывода. Сглаживание: Здесь вы можете установить режим сглаживания для рендеринга шрифта выходных окон. Вы можете позволить IDE решать это, выбрав Use Default, или принудительно включить или выключить , выбрав одну из этих опций.
