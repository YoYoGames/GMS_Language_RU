Предпочтения отладки Настройки здесь связаны с отладчикомGameMaker Studio 2, и у вас есть следующие опции: Автоматическая реакция на закрытие отладчика: Эта опция позволяет установить автоматическую реакцию при закрытии отладчика. Если установить значение "Показать сообщение", вам будет показан диалог, когда вы захотите закрыть отладчик, в то время как при установке значения "Да" или "Нет" этот диалог не будет показан, а будет выполнено указанное задание. Если установить значение "Да" или "Нет", то диалог не будет показан, а задание будет выполнено, как указано. Включить отладку в реальном времени: Отметив этот флажок, вы сможете отлаживать свои игровые проекты в реальном времени при их запуске из IDE. Эта опция установлена по умолчанию. Порт отладчика: Здесь задается номер порта, который будет использовать модуль отладки. Значение по умолчанию - 6509. Показывать значки событий в представлении ресурсов отладчика: Установите, следует ли показывать значки в представлении ресурсов отладчика. По умолчанию этот параметр установлен.
