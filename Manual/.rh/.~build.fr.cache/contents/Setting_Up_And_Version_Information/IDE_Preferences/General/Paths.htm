Предпочтения пути Параметры Path управляют различными путями к файлам, которые требуются элементам в IDE GameMaker Studio 2, а также для любых внешних редакторов, которые вам необходимы. Для IDE существуют следующие опции: Временный каталог: Место для сохранения всех временных папок. По умолчанию в Windows это \Users\<Имя пользователя>\AppData\Local\Temp\GameMakerStudio2 А на macOS это: /var/folders//GameMakerStudio2 Каталог кэша IDE: Место для сохранения кэша IDE. По умолчанию в Windows это \Users\<Имя пользователя>\AppData\Roaming\GameMakerStudio2\Cache А на macOS это: /Users/<Имя пользователя>/.config/GameMakerStudio2/Cache Каталог кэша активов: Место для сохранения кэша активов для каждого проекта. По умолчанию в Windows это \Users\<Имя пользователя>\AppData\Roaming\GameMakerStudio2\Cache А на macOS это: /Users//.config/GameMakerStudio2/Cache Расположение "Мои проекты": Место, где GameMaker Studio 2 будет первоначально создавать новые проекты. По умолчанию в Windows это \Users\<Имя пользователя>\Documents\GameMakerStudio2 А в macOS это /Users/<Имя пользователя>/GameMakerStudio2 Automatically delete temp directory on close: Установка этого флажка заставит GameMaker Studio 2 автоматически удалять папку Temp, которую она создает в проекте для компиляции и т.д.. Этот параметр включен по умолчанию, а снятие флажка выключит ее (это означает, что вам придется вручную удалять все временные файлы). Автоматически удалять кэш активов при закрытии: Если этот флажок установлен, то папка кэша компилятора активов будет удаляться при выходе из GameMaker Studio 2. По умолчанию эта функция выключена, и ее включение будет означать, что каждый раз при загрузке и запуске любого проекта кэш нужно будет создавать заново (что может занять время в зависимости от размера игры). Автоматически удалять кэш IDE при закрытии: Если этот флажок установлен, то папка кэша компилятора IDE будет удаляться при выходе из GameMaker Studio 2. По умолчанию этот параметр выключен, и его включение будет означать, что при каждом запуске GameMaker Studio 2 кэш нужно будет создавать заново. Удалить временную папку: Нажатие этой кнопки приведет к удалению временной папки проекта. Удалить кэш активов: Нажатие этой кнопки приведет к удалению кэша активов компилятора для проекта. Удалить кэш IDE: Нажатие этой кнопки приведет к удалению кэша IDE Для задания путей к внешним редакторам существуют следующие варианты: Путь к внешнему редактору/просмотрщику для файлов SWF: Если вы работаете со спрайтами в формате SWF, вы можете установить этот параметр на путь к предпочитаемому редактору/просмотрщику, и когда вы нажмете кнопку Редактировать изображение в редакторе спрайтов, он будет открываться указанная программа. По умолчанию путь не задан. Путь к внешнему редактору/просмотрщику для файлов Spine: Если вы работаете со спрайтами в формате Spine, вы можете установить этот параметр на путь к предпочитаемой программе просмотра/редактору, и когда вы нажмете кнопку Редактировать изображение в редакторе спрайтов, тогда будет открываться указанная программа.
