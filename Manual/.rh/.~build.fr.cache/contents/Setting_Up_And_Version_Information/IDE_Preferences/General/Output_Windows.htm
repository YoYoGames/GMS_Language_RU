Предпочтения окна вывода Окно вывода управляют различными цветовыми и форматирующими опциями для различных окон вывода в IDE. Каждое из различных окон имеет одинаковый набор опций, которые перечислены ниже: Цвет: Цвет по умолчанию для текста, записываемого в окно. Имя шрифта: Имя шрифта, который необходимо использовать (доступные шрифты зависят от установленных в вашей системе). Стиль: Используемый стиль шрифта, например, жирный или курсив (обратите внимание, что не во всех окнах можно использовать одинаковые стили, а также это зависит от используемого шрифта). Размер: Размер шрифта для данного окна вывода. Сглаживание: Здесь вы можете установить режим сглаживания для рендеринга шрифтов выходных окон. Вы можете позволить IDE решить это, выбрав Use Default, или принудительно включить или Off , выбрав одну из этих опций.
