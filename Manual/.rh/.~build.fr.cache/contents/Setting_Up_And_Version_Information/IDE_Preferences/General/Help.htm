Предпочтения справки Файл справки разработан для работы как веб-страница, и хотя файлы хранятся локально, для их просмотра требуется использование внешнего браузера. Здесь можно изменить следующие настройки, связанные с файлом справки: Язык руководства: Здесь задается язык, на котором будет работать IDE. По умолчанию это "English". Порт справочного руководства: Здесь задается номер порта, который будет использовать веб-сервер Manual Micro. Значение по умолчанию - 51291.
