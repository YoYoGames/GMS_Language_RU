Предпочтения наблюдателя за файлами File Watcher - это название части IDE, которая следит за файлами, относящимися к проекту, на предмет изменений, вызванных внешними программами (например, редактирование изображения во внешнем редакторе) или внутренними изменениями, такими как редактирование сценария. актив. В целом, эти настройки не нужно редактировать, но на некоторых системах - например, на тех, которые используют SSD в качестве накопителя для сохранения файлов проекта - наблюдатель файлов может сообщать вам об изменениях, когда вы делаете такие вещи, как простое сохранение проекта или редактирование любого актива проекта. или при редактировании любого актива проекта, и в этом случае вам может понадобиться изменить эти настройки. Доступны следующие настройки: Включить File Watcher: Здесь можно включить или отключить систему File Watcher. Если эта функция отключена, IDE больше не будет замечать файлы, измененные внешними источниками, поэтому следует быть осторожным при отключении этой функции. этой функции. По умолчанию система File Watcher включена. Установите допуск File Watcher в MS после сохранения В IDE: Это настраивает время между сохранением файла и проверкой изменений наблюдателем за файлами. Вы можете изменить это значение, если вы сохраняете проект, а File Watcher ошибочно появляется сообщение о том, что что-то изменилось, в этом случае вы увеличиваете задержку до тех пор, пока сообщение не перестанет появляться.
