Последние предпочтения Windows Настройки здесь содержат параметры отображения элементов в списке " Недавние окна": Переместить сфокусированные окна в начало списка: Если этот параметр включен и вы щелкните на любом элементе в списке Недавние окна, выбранный элемент будет перемещен в начало списка. По умолчанию эта функция отключена. Отображать только открытые в данный момент окна: Включение этого параметра заставит список Recent Windows удалять те элементы, которые больше не открыты в рабочей области (т.е. закрытие любого окна также удалит его из списка). По умолчанию эта функция отключена. Максимальное количество окон для отображения: Устанавливает максимальное количество ранее открытых окон, которые будут отображаться в списке Недавние окна.
