Предпочтения ввода Параметрыввода управляют определенными аспектами ввода с помощью мыши и клавиатуры в IDE. Доступны следующие опции: Медленное время двойного щелчка (мс): Некоторые области IDE требуют более медленного двойного щелчка для корректной работы, например, при переименовании ресурса (быстрый двойной щелчок откроет редактор ресурсов, а медленный двойной щелчок позволит вам переименовать его). Этот параметр позволяет установить скорость обнаружения, значение по умолчанию составляет 500 мс. Время блокировки прокрутки (мс): Этот параметр определяет время, в течение которого GameMaker Studio 2 должен ждать между прокручиваемыми окнами, которые перекрываются. По сути, когда вы прокручиваете окно "A", курсор может навестись на другой прокручиваемой области в окне "B", поэтому это предпочтение определяет, как долго нужно ждать после остановки прокрутки "A", прежде чем можно будет прокрутить "B", чтобы случайно не начать прокручивать разные панели. Значение по умолчанию равно 500 мс. Обернуть курсор вокруг экрана при панорамировании: Если эта функция включена, курсор будет огибать границы экрана при панорамировании рабочего пространства. По умолчанию включено. Скорость прокрутки: Этот параметр управляет скоростью прокрутки в различных рабочих пространствах. Значение по умолчанию равно 1. Расстояние перетаскивания вкладки до размаксимизации рабочего стола (px): Когда вы перетаскиваете вкладку рабочего пространства из ее положения по умолчанию в IDE, если в это время активно только одно рабочее пространство, то окно GameMaker Studio 2 разворачивается и перемещаться с помощью мыши (если активно более одной рабочей области, то будет создан новый экземпляр IDE для размещения рабочей области). Этот параметр определяет, насколько далеко нужно перетащить вкладку, чтобы рабочая область распознала ее, а значение по умолчанию составляет 50px. Минимальный уровень масштабирования для взаимодействия с содержимым окна (%): Этот параметр управляет точкой отсечения масштаба, при которой мышь больше не сможет взаимодействовать с окнами рабочей области (т.е. добавлять узлы, перетаскивать окна и т.д.). По умолчанию значение равно 40. Для масштабирования с помощью колесика мыши требуется клавиша Ctrl: При установке флажка во всех редакторах (и в основной рабочей области) будет включено масштабирование с помощью / + колесико мыши , в то время как снятие флажка включит масштабирование только с помощью колесика мыши Значок MMB. Ctrl + MMB для открытия унаследованного события вместо ручного: Когда эта опция отмечена, удерживая / + среднюю кнопку мыши на объекте, вы откроете наследуемое событие объекта (если оно есть). а не руководство. Добавляет смещение при выделении текста для предотвращения перескакивания строк: Когда эта опция отмечена, высота задуманной строки при выделении одной строки в любом текстовом окне будет смещена, что делает ее менее подверженной вертикальной неточности. Эта функция включена по умолчанию. Включить режим ноутбука на всех устройствах: Установка этого флажка отобразит кнопку Laptop Mode на IDE, что позволит вам включать или выключать его по мере необходимости, независимо от того, является ли работающее устройство GameMaker Studio 2 является ноутбуком или нет.
