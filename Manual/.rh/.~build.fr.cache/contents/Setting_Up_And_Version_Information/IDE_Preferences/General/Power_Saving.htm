Параметры энергосбережения Параметры энергосбережения здесь влияют на то, сколько энергии CPU и GPU будет использоваться для генерации IDE GameMaker Studio 2 (по сути, это снижает базовую частоту кадров, на которой работает программа, и таким образом снижает энергопотребление). Варианты здесь следующие: Частота кадров переменного тока (%): Базовый модификатор частоты кадров IDE при подключении к входу питания. По умолчанию - 100%. Частота кадров при питании от батареи (%): Базовый модификатор частоты кадров IDE при использовании питания от батареи. По умолчанию - 35%. Фоновая частота кадров (%): Базовый модификатор частоты кадров IDE, когда GameMaker Studio 2 не находится в фокусе (независимо от источника питания). По умолчанию - 15%.
