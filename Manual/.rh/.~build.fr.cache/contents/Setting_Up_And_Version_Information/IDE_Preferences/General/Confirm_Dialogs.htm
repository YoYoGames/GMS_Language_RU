Подтверждение предпочтений диалогового окна Параметры диалога подтверждения касаются некоторых общих всплывающих окон подтверждения, которые появляются при выполнении определенных действий в IDE GameMaker Studio 2. Опции следующие: Автоматическая реакция на закрытие рабочей области с помощью сочетания клавиш: Упомянутое выше сочетание клавиш можно использовать для закрытия любой открытой вкладки, но иногда вы можете случайно закрыть пустую вкладку, которая вам необходима. Чтобы предотвратить это, вы можете можете установить опцию "Показать сообщение" (опция по умолчанию), чтобы спросить, действительно ли вы хотите закрыть эту пустую вкладку. Другие настройки: "Нет", чтобы ничего не делать, если используется ярлык, или "Хорошо", чтобы закрыть вкладку без сообщения. Автоматическая реакция на перезапуск запущенной сборки: Когда вы перезапускаете сборку, а другая предыдущая сборка все еще запущена, вас спросят, хотите ли вы сначала остановить текущую сборку. Если вы не отметите это сообщение, оно будет подавлено сообщение и заставит GameMaker Studio 2 считать, что вы нажали "Ок" в качестве ответа. По умолчанию для этого параметра установлено значение "Показать сообщение". Автоматический ответ на предупреждение при попытке отладки YYC: Цель YYC компилируется в нативный код и поэтому не может быть отлажена с помощью встроенного отладчика. Когда вы попытаетесь использовать отладчик на цели YYC, вы получите предупреждение информирующее вас об этом, а также о том, что проект будет скомпилирован с использованием VM. По умолчанию для этого параметра установлено значение "Показать сообщение", но если установить его на "OK", это предупреждение будет подавлено. Автоматическая реакция на сброс макета проекта: Когда вы выбираете сброс макета проекта (из меню "Макеты"), по умолчанию вам будет показано предупреждающее сообщение о том, что это приведет к закрытию всех открытых окон ресурсов. Однако вы можете подавить это сообщение и автоматически сбросить макет, выбрав "Ок", или ничего не делать, выбрав "Нет". Значение по умолчанию - "Показать Сообщение".
