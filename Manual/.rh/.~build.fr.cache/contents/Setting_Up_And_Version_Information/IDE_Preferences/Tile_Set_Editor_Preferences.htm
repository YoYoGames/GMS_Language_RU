Предпочтения редактора набора плиток В настройках редактора наборов плиток есть следующая опция для автоплитки: Цвет фона автоплитки: Этот параметр устанавливает цвет фона в редакторе наборов плиток в разделе создания автоплиток. По умолчанию используется цвет (rgba) #808080FF. В настройках редактора наборов плиток также есть подраздел " Диалоги подтверждения ": Опция, доступная здесь, управляет сообщением, появляющимся при изменении размера ячеек набора плиток. размер ячейки набора плиток, что повлияет на отрисовку слоев карты плиток помещения (в результате чего слои будут обрезаны). По умолчанию вам будет предложено продолжить или отменить операцию, а эта опция позволяет подавить это сообщение, выбрав "Хорошо". что позволит продолжить изменение ячеек набора плиток независимо от влияния на слои карты помещения.
