Предпочтения браузера активов Предпочтения браузера активов используются для определения определенных свойств браузера активов и его элементов. его элементов. Основными параметрами являются: Показывать быстрый доступ: При включении этой опции в верхней части браузера активов будет отображаться раздел "Быстрый доступ". Эта опция включена по умолчанию. Создавать папки по умолчанию в новом проекте: При включении этой опции все новые проекты будут начинаться с уже созданными папками активов по умолчанию, а при выключении будет создана только папка комнаты с комнатой по умолчанию, требуемой для всех проектов. Эта опция включена по умолчанию. Максимальное количество активов, открываемых одновременно без предупреждения: Вы можете выбрать несколько активов для одновременного открытия, используя / + для их выбора, а затем щелкнув правой кнопкой мыши на и выбрав пункт "Открыть все". Если вы выбрали больше, чем указано в этой опции, то вам будет показано предупреждение, так как одновременное открытие слишком большого количества активов может вызвать проблемы с производительностью. с производительностью на низкопроизводительных машинах. Значение по умолчанию равно 10. Максимальное количество недавно отредактированных элементов: Этот параметр регулирует количество активов, перечисленных в разделе "Последние" браузера активов. Значение по умолчанию равно 10. Размер цветовой палитры: Здесь можно установить количество цветов, доступных в палитре браузера активов для групп активов. Значение по умолчанию - 16, минимальное значение - 16, максимальное - 100. Применять цвета активов в Быстром доступе: Если флажок установлен, этот параметр будет использовать цвета, установленные для любого данного типа актива, для окраски актива в разделе быстрого доступа браузера. При снятии флажка цвет актива не будет использоваться. Эта опция включена по умолчанию. Цвета группы переопределяют цвета дочерних активов: Если флажок установлен, то для окрашивания всех активов внутри группы будет использоваться групповой цвет верхнего уровня, игнорируя цвет, установленный для отдельного актива. По умолчанию этот параметр отключен. Создайте диалоговый стиль актива: Когда вы нажимаете кнопку "Добавить актив" в браузере активов, появляется окно, показывающее различные активы, которые могут быть добавлены в ваш проект. Это окно может быть представлено в виде списка или в виде сетки, и этот параметр определяет, что будет отображаться по умолчанию. Значение по умолчанию - сетка. Порядок сортировки по умолчанию: С помощью этого параметра можно установить порядок сортировки активов в браузере по умолчанию: по алфавиту или от А до Я или от Я до А. По умолчанию - от А до Я. Поумолчанию сортировать сначала папки: Если этот параметр включен, браузер активов будет сортировать сначала папки, затем все "свободные" активы, а если этот параметр отключен, то все активы и папки будут сортироваться вместе. Это включено по умолчанию. Двойной щелчок по спрайту открывает редактор изображений: По умолчанию двойной щелчок на любом активе открывает этот актив для редактирования, но часто при работе со спрайтами, после первоначальной настройки, когда вы создаете новый актив, вы открываете его для редактирования изображения, а не свойств спрайта. При выборе этой опции откроется редактор изображений для актива спрайта, а не сам актив спрайта. Вы все еще можете открыть редактор спрайтов, щелкнув правой кнопкой мыши и выбрав Edit. По умолчанию эта опция выключена. Двойной щелчок на объекте открывает все события объекта: По умолчанию двойной щелчок на любом активе открывает этот актив для редактирования, но часто при работе с объектами, после первоначальной настройки при создании нового актива, вы открываете его для редактирования только событий, а не свойств объекта. При выборе этой опции различные события объекта будут открыты в новом рабочем пространстве, при этом каждому событию будет отведена своя вкладка в новой рабочей области. Вы все еще можете открыть редактор объектов, щелкнув правой кнопкой мыши и выбрав Edit. По умолчанию эта опция выключена. Разгруппировка порядка комнат удаляет пустую группу: Если эта функция включена, при разгруппировке всех помещений, принадлежащих к папке группы, папка группы также будет удалена. Если это действие отключено, то при разгруппировке комнат останется пустая папка группы. в обозревателе активов. Эта функция включена по умолчанию. Масштаб по умолчанию (%): Этот параметр устанавливает процент масштабирования по умолчанию для браузера активов при запуске нового проекта. Значение по умолчанию равно 100%. Минимальный масштаб (%): Этот параметр устанавливает минимальное значение масштаба, допустимое для браузера активов. По умолчанию - 50%. Максимальный масштаб (%): Этот параметр устанавливает максимальное значение масштаба, допустимое для браузера активов. По умолчанию - 250%. Показывать линии деревьев: Здесь вы можете включить/выключить отображение линий деревьев, которые показываются сбоку от активов в браузере активов. По умолчанию флажок установлен, чтобы показывать линии деревьев. Разворачивать дерево при изменении фильтра: Этот параметр заставит браузер активов развернуть все свернутые узлы дерева при изменении параметров фильтра. Отключение опции предотвратит это. По умолчанию включена. Диалоги подтверждения Здесь вы можете выбрать поведение диалоговых окон подтверждения при работе с активом из браузера активов. Доступны следующие варианты: Автоматическая реакция на удаление актива: При удалении актива из браузера активов обычно появляется предупреждение с вопросом, уверены ли вы, что хотите продолжить. При установке этого параметра на "ОК" сообщение будет подавлено. как если бы вы нажали кнопку "OK", чтобы продолжить. Значение по умолчанию - "Показать сообщение". Автоматическая реакция на удаление группы Room Order: При удалении папки группы, содержащей комнаты, из браузера активов обычно появляется предупреждение, спрашивающее, уверены ли вы и хотите ли вы приступить к удалению (в котором перечислены все комнаты в группе). которые будут удалены). Если установить для этого параметра значение "OK", сообщение будет заблокировано, как если бы вы нажали кнопку OK, чтобы продолжить и удалить все помещения в группе, а также саму папку группы. Значение по умолчанию - "Показать Сообщение". Автоматическая реакция на открытие нескольких активов: Как упоминалось выше в общих настройках браузера активов, при одновременном открытии нескольких активов из браузера активов вы можете получить предупреждение, если их количество превысит указанное. При установке этого параметра в значение "ОК" предупреждение будет подавлено, как если бы вы нажали кнопку "ОК" для продолжения работы. Значение по умолчанию - "Показать сообщение". Автоматическая реакция на преобразование DnD™в GameMaker Language: Когда вы выбираете опцию преобразования узла DnD™в GML, вам будет предложено подтвердить изменение, поскольку его нельзя отменить. При установке этого параметра на "OK" предупреждение будет подавлено, как если бы вы нажали кнопку "OK" для продолжения работы. Значение по умолчанию - "Показать сообщение". Автоматическая реакция на установку родителя комнаты с помощью перетаскивания узла: В разделе "Наследование" окна Room Manager можно перетащить комнату на другую комнату, чтобы сделать ее дочерней по отношению к комнате, на которую она перетаскивается. При выполнении вы получите предупреждение с вопросом, уверены ли вы и хотите ли вы продолжить. Если установить для этого параметра значение "OK", сообщение будет заблокировано, как если бы вы нажали кнопку OK, чтобы продолжить. Значение по умолчанию - "Показать сообщение". Автоматическая реакция на ошибку дублирования имени группы: Когда вы создаете групповую папку для организации активов и называете ее так же, как другую папку в браузере, появится сообщение с вопросом, хотите ли вы продолжить и иметь две папки с одинаковыми именами. Если установить для этого параметра значение "OK", предупреждение будет подавлено, как если бы вы нажали кнопку "OK", чтобы продолжить. Значение по умолчанию - "Показать сообщение". Автоматическая реакция на установку родительской комнаты с помощью перетаскивания узла: Когда вы перетаскиваете комнату на другую комнату в Менеджере комнат, а затем отпускаете ее, появляется сообщение с вопросом, хотите ли вы сделать эту комнату "дочерней" по отношению к той. перетаскиваемой комнаты. Если установить для этого параметра значение "OK", предупреждение будет подавлено, как если бы вы нажали кнопку "OK", чтобы продолжить. Значение по умолчанию - "Показать сообщение".
