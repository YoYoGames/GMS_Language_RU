Предпочтения редактора шрифтов Параметры редактора шрифтов используются для определения поведения редактора шриф тов. Ниже перечислены различные доступные опции, а также описание того, что каждая из них делает. Обновить шрифты, импортированные из проектов GameMaker Studio 1: Если этот флажок установлен, GameMaker Studio 2 попытается обновить шрифты и регенерировать их текстуру. Это произойдет только в том случае, если: Это предпочтение включено Шрифт доступен на машине GameMaker Studio 2 может найти подходящее лицо в семействе шрифтов (это наиболее вероятная причина несоответствия при импорте). Обратите внимание, что это также произойдет, если шрифт GameMaker Studio 1 не имеет кэшированной текстуры шрифта. Если устаревший шрифт не может быть обновлен, то GameMaker Studio 2 по умолчанию будет использовать кэшированную текстуру шрифта, и редактор шрифтов больше не позволит вам редактировать шрифт, показывая возможность резервного копирования и обновления на другой: Нажав на эту кнопку, вы сможете редактировать шрифт, либо добавив отсутствующий шрифт, либо установив другой, при этом резервная копия шрифта хранится в папке font asset, как .yy.gms1 и .png.gms1. Чтобы восстановить эти файлы, вручную удалите файлы .yy и замените их этими, удалив расширение .gms1 из имен файлов. Важно! Все активы хранят свои имена в файле .yy, поэтому вы должны отредактировать их и поместить правильное имя ресурса также в поле (если оно отличается). Автоматический выбор для диалога неудачной регенерации содержимого: Когда вы собираетесь регенерировать текстуру шрифтового актива, вам показывается предупреждение. Установка этого параметра на "ОК" означает, что шрифт будет регенерирован без предупреждения. показано. Автоматический выбор для диалога о неудачной регенерации шрифта: При неудачной регенерации текстуры шрифтового актива отображается предупреждение. Установка этого параметра в "ОК" позволяет пропустить показ сообщения. Автоматический выбор для обновления импортированных устаревших шрифтов: При импорте устаревшего проекта вы можете получить предупреждение об устаревших шрифтах и предложение обновить их. Установка этого параметра на "ОК" приведет к автоматическому обновлению без показа этого сообщения.
