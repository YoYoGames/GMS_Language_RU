Предпочтения редактора сплайнов Предпочтения редактора сплайнов используются в основном для определения того, как выглядят гладкие траектории в редакторе контуров, хотя они влияют и на некоторые другие аспекты IDE (например, редактор форм физики). Editor). Существуют следующие параметры: Точность точки вставки контура (px): Устанавливает значение точности (в пикселях) для вставки точек на пути, т.е. мышь должна находиться в этом диапазоне, чтобы вставить точку, а не добавить новую. По умолчанию - 20px. Ширина точки привязки: задает ширину точек привязки в пикселях. По умолчанию 8px; Высота точки привязки: Установите высоту точек привязки в пикселях. По умолчанию 8px; Цвет контура: Цвет, который будет использоваться для отображения контура при его редактировании. По умолчанию это ( ARGB) #FFFFFFFF. Цвет выделенного сегмента пути: Цвет, который будет использоваться для выделения выбранных сегментов пути. По умолчанию (ARGB) #FFFFFFFF. Цвет недопустимого пути: Цвет, который будет использоваться для выделения недопустимого пути. По умолчанию (ARGB) #FF0000FF. Цвет опорных точек: Цвет, который будет использоваться для рисования опорных точек контура. По умолчанию (ARGB) #FFC0C0C0. Цвет недействительных опорных точек: Цвет, который будет использоваться для рисования опорных точек контура. По умолчанию (ARGB) #FF800000. Цвет выделенной опорной точки: Цвет, который будет использоваться для рисования опорных точек пути, которые были выделены (выбраны). По умолчанию (ARGB) #FF00A4FF. Цвет выделенных недействительных опорных точек: Цвет, который будет использоваться для рисования опорных точек, которые выделены, но недействительны. По умолчанию (ARGB) #FFA400FF. Цвет опорной точки при наведении мыши: Цвет, который будет использоваться для рисования активных опорных точек. По умолчанию (ARGB) #FFFFFFFF. Цвет недействительной опорной точки при наведении мыши: Цвет, который будет использоваться для рисования активных, но недействительных опорных точек. По умолчанию (ARGB) #FF0000FF. Цвет наложения маски: задает цвет наложения для масок, если они включены (см. выше). По умолчанию (ARGB) #FF404040.
