Предпочтения редактора изображений Параметры редактора изображений используются для определения того, как выглядит и ощущается окно редактора изображений. Ниже перечислены общие параметры: Развернуть наложение элементов управления спрайтом по умолчанию: Если этот флажок установлен, элементы управления спрайтом и информация о кадре будут отображаться в редакторе изображений каждый раз, когда вы открываете спрайт для редактирования (его можно закрыть с помощью кнопки свернуть , но при последующем редактировании спрайта он снова откроется). Если флажок не установлен, элементы управления спрайтом и информация о кадре будут скрыты при открытии спрайта. спрайта. По умолчанию флажок не установлен. Режим зацикливания по умолчанию: Кнопка цикла в редакторе изображений может быть настроена на работу в двух различных режимах: либо воспроизведение один раз, при котором анимация остановится, когда дойдет до последнего кадра, либо цикл с начала, при котором анимация вернется или "пинг-понг", когда анимация будет идти вверх по изображениям до конца, а затем снова вниз по ним до начала. По умолчанию здесь используется значение "loop". Рисовать тонкий контур выделения: Если флажок установлен, то контур выделения будет иметь абсолютную ширину 1px, независимо от масштаба изображения. Если флажок снят, то контур выделения будет иметь относительную ширину 1px. и масштабироваться вместе с изображением. По умолчанию этот параметр отмечен. Основной цвет контура выделения: Контур раздела представляет собой пунктирную линию, чередующуюся между двумя цветами. Этот параметр позволяет установить первый из этих цветов (включая альфа) и имеет значение по умолчанию ARGB #FFFFFD0. Вторичный цвет контура выделения: Контур раздела представляет собой пунктирную линию, которая чередуется между двумя цветами. Этот параметр позволяет задать второй из этих цветов (включая альфа), значение по умолчанию - ARGB #0000D0. Цвет фона редактора: При редактировании кадра спрайта изображение может увеличиваться или уменьшаться, или не помещаться в окне рисунка целиком, в этом случае будет отображаться цвет фона, который можно задать здесь. Значение по умолчанию - ARGB #282828FF. Параметры редактора изображений также имеют следующие подкатегории: Справочная информация Здесь вы можете задать параметры фона, на котором будут редактироваться изображения спрайтов. По умолчанию это сетка из серых квадратов, но вы можете использовать следующие настройки, чтобы изменить ее: Стиль холста: Здесь вы устанавливаете "стиль" для фона, который может быть либо " Блочный узор" или " Одноцветный". Одноцветный просто устанавливает фон на выбранный вами цвет (с помощью параметра Цвет 1, как объясняется ниже), в то время как блочный узор является параметром по умолчанию (который также может быть изменен с помощью остальных опций, показанных ниже). По умолчанию - " Блочный узор". Размер блока холста: Когда вы выбрали Block Pattern для фона, здесь можно задать размер отдельного блока. По умолчанию - 16px, минимальное значение - 2px, максимальное - 256px. Основной цвет холста: Если для фона выбран Single Colour для фона, то используется именно этот цвет. Вы можете щелкнуть по образцу цвета, чтобы открыть редактор цветов и изменить его. При использовании опции Block Pattern, этот параметр задает первый цвет для чередующихся блоков. По умолчанию используется цвет ( ARGB) #FFC3C3C3. Холст вторичного цвета (стиль блочного рисунка): Если для фона выбран Block Pattern для фона, то цвет, используемый для чередования с цветом 1 (см. выше), будет именно этот. Вы можете щелкнуть по образцу цвета, чтобы открыть редактор цветов и изменить его. Цвет по умолчанию - (ARGB) #FF808080. Диалоги подтверждения Здесь можно выбрать поведение диалогов подтверждения при выполнении различных действий над текущим спрайтом. К ним относятся: Автоматическая реакция на удаление кадров: Когда вы удаляете кадр, по умолчанию появляется сообщение с вопросом, уверены ли вы, что хотите это сделать. Установив этот параметр на "OK", вы можете удалить это сообщение и разрешить удаление кадра без дальнейших сообщений. По умолчанию установлено значение "Показать сообщение". Автоматическая реакция на замену изображений при импорте стрипов: Когда вы выбираете "Импортировать полосу" в выпадающем меню "Изображение", вы получите сообщение с предупреждением о том, что вы перезапишете все изображения, уже имеющиеся в ресурсе спрайта, с возможностью продолжить или нет. Здесь можно указать, показывать ли это сообщение, или же IDE просто выбирает опцию "Да" или "Нет". Значение по умолчанию - "Показать сообщение". Автоматический ответ на фиксацию изменений при смене инструмента: Когда вы меняете инструменты рисования после редактирования кадра, по умолчанию появляется сообщение с вопросом, хотите ли вы зафиксировать внесенные изменения. Установив этот параметр на "ОК", вы можно убрать это сообщение и разрешить фиксацию редактирования кадра без последующих сообщений при смене инструмента. Если вы выберете "Нет", то редактирование рамки будет молча отменено при смене инструментов. По умолчанию установлено значение "Показать сообщение". Автоматическая реакция на предупреждение о заблокированном слое: Если вы попытаетесь отредактировать заблокированный слой, то по умолчанию появится сообщение, информирующее вас о том, что вы не можете этого сделать. Если вы установите для этого параметра значение "OK", то сообщение не будет показано и редактирование будет тихо провалено. Автоматический ответ на предупреждение о групповом выделении: Если вы выделите несколько слоев в редакторе слоев, а затем попытаетесь рисовать или редактировать, то по умолчанию будет показано сообщение о том, что это невозможно. Вы можете выбрать "OK" здесь, чтобы убрать это сообщение и молча не редактировать изображения вместе, пока не будет выбран только один слой. По умолчанию установлено значение "Показать сообщение". Настройки сетки Сетка Параметры настройки используются для задания способа представления сетки изображений, при этом доступны следующие варианты: Расстояние между горизонтальными линиями сетки (px): Это значение задает расстояние между горизонтальными линиями сетки изображений. Значение по умолчанию равно 1px. Расстояние между вертикальными линиями сетки (px): Это значение задает расстояние между вертикальными линиями сетки изображений. Значение по умолчанию равно 1px. Отображать сетку: Если этот флажок установлен, то редактор изображений будет открываться с включенной сеткой изображений. По умолчанию выключено. Привязка к сетке: Если этот флажок установлен, то редактор изображений будет автоматически привязывать определенные инструменты к сетке изображения. По умолчанию выключено. Цвет линии сетки: Здесь можно задать цвет (включая альфа-значение) для сетки изображения. По умолчанию это ( ARGB) #FF000000. Инструменты Здесь вы можете установить различные параметры для работы и отображения некоторых инструментов рисования. Если вы используете Arc Tool , то здесь можно задать различные цвета, используемые для рисования свойств узла: Цвет опорных точек: задает цвет для различных опорных точек сплайна/дуги. По умолчанию ( ARGB) #FFFF0000. Цвет контрольной точки: Установите цвет для каждой из "ручек" контрольной точки на опорной точке. По умолчанию (ARGB) #FFFF0000. Цвет касательной линии: задает цвет линий, связывающих контрольные точки с привязками. По умолчанию (ARGB) #FF00FFFF. Цвет выделенной точки: задает цвет, который будет использоваться для всех выделенных опорных точек на сплайне/дуге. По умолчанию (ARGB) #FF00FF00. Размер точки: Установите размер различных точек привязки и контрольных точек, которые будут использоваться. По умолчанию - 10px.
