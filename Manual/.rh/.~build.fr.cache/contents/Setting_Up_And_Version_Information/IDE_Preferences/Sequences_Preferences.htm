Последовательности Предпочтения Предпочтения последовательностей используются для определения определенных свойств редактора последовательностей и его элементов. его элементов. Основными опциями являются: Показывать рамку холста во вновь созданных последовательностях: Если опция включена, интеллектуальная направляющая рамки холста будет отображаться на холсте редактора последовательностей для всех новых последовательностей. Эта опция может быть включена/выключена по необходимости в свойствах холста в самом редакторе. Значение по умолчанию - включено. Ширина кадра холста для вновь создаваемых последовательностей: Здесь вы можете установить ширину рамки по умолчанию для всех новых последовательностей. Значение по умолчанию - 1366px. Высота кадра холста длявновь создаваемых последовательностей: Здесь вы можете установить высоту рамки по умолчанию для всех новых последовательностей. Значение по умолчанию - 768px. Пауза при щелчке на холсте: Если этот параметр отмечен, щелчок в любом месте холста редактора последовательностей приостанавливает воспроизведение последовательности. Эта опция включена по умолчанию. Задержка выбора перекрытия в миллисекундах: Этот параметр определяет, как долго нужно ждать, прежде чем появится панель инструментов элементов для перекрывающихся элементов на холсте редактора последовательностей. Значение по умолчанию - 1000 мс. Поле вставки точки пути перевода: На холсте редактора последовательностей, когда вы добавили актив и затем создали дорожку параметров положения, будут нарисованы пунктирные линии, чтобы показать путь, по которому актив будет двигаться при перемещении. Если подвести мышь близко к этой линии, то можно добавить к ней новую точку дорожки параметров, а этот параметр задает расстояние от линии перевода, на котором должна находиться мышь, чтобы маркер точки был показан. Значение по умолчанию 6px. Автоматически создавать дорожки параметров по умолчанию: Если этот параметр включен, он автоматически добавляет дорожки параметров для позиции, масштаба, вращения и начала координат ко всем новым дорожкам, созданным на панели дорожек последовательности. По умолчанию отключена. Автоматически растягивать вновь созданные активы: Эта опция растягивает все вновь созданные ключи активов до длины последовательности. По умолчанию отключена. Высота дорожки кривой анимации в доп. листе: С помощью этого параметра можно установить, сколько линий дорожек параметров, использующих кривые, будет занимать лист dopw в режиме просмотра ключевых кадров. По умолчанию - 2, но вы можете выбрать 1 или 3. Диалоги подтверждения Confirm Dialogs используются для изменения сообщений, которые отображаются в редакторе последовательности при выполнении определенных действий. Вы можете изменить следующие сообщения: Автоматический выбор для встраивания отредактированной кривой: Если вы попытаетесь вставить отредактированную анимационную кривую, то по умолчанию вам будет показано сообщение для подтверждения этого действия. Однако если установить этот параметр в значение "Да", то сообщение не будет отображаться сообщение и встраивание будет происходить каждый раз. Если установить значение "Нет", то встраивание не будет происходить каждый раз. Автоматический выбор для преобразования кривой в ключевые кадры: Если вы попытаетесь преобразовать кривую анимации в ключевые кадры, по умолчанию будет показано сообщение с вопросом, хотите ли вы продолжить. Однако при установке этого параметра на "Ok" не будет показано сообщение и просто не позволит вам преобразовать кривую. Автоматический выбор показа предупреждения при редактировании внешней кривой: Если вы попытаетесь отредактировать кривую, примененную к дорожке параметров, и окажется, что это внешний актив Animation Curve, вам по умолчанию будет показано предупреждение. Однако установка этот параметр на "Ok", сообщение не будет показано, и вы сможете редактировать кривую. Умные гиды Здесь вы можете выбрать поведение интеллектуальных направляющих при работе в редакторе последовательностей. Доступны следующие опции: Включить интеллектуальные направляющие: Здесь можно включить или выключить интеллектуальные направляющие на холсте редактора последовательностей. По умолчанию эта функция включена. Включить линейки: Здесь можно включить или выключить интеллектуальные направляющие на холсте редактора последовательностей. По умолчанию эта функция включена. Привязать линейки к краям: Эта опция позволяет линейкам "привязываться" к краям активов на холсте секвенций (и наоборот). По умолчанию включена. Расстояние привязки: Это расстояние (в пикселях), на котором элемент должен находиться от линейки (или линейка от элемента), чтобы он "привязался" к позиции. Значение по умолчанию - 6px.
