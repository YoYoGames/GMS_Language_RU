Предпочтения редактора помещений Параметры редактора помещений используются для определения внешнего вида и поведения редактора помещений. Общие настройки следующие: Расстояние начала затухания цепи (px): Устанавливает расстояние от родительского окна, на котором звено цепи будет исчезать. По умолчанию 10px. Расстояние остановки затухания цепи (px): Устанавливает расстояние от родительского окна до конца затухания звена цепи. По умолчанию - 100px. Минимальная альфа цепочки: Устанавливает конечное значение альфа после точки падения затухания для альфа-цепочки. По умолчанию 0,3. Расстояние между глубинами слоев по умолчанию: Когда вы создаете несколько слоев в редакторе помещений, им автоматически присваивается глубина, основанная на их положении в списке слоев. Эта глубина увеличивается на величину, указанную здесь. По умолчанию значение равно 100, поэтому каждый слой будет иметь глубину предыдущего слоя + 100. Ширина комнаты по умолчанию: задает начальную ширину, которая будет выбрана по умолчанию для новой комнаты. По умолчанию это 1024px. Высота помещения по умолчанию: Устанавливает начальную высоту, которая будет выбрана по умолчанию для новой комнаты. По умолчанию - 768px. Добавьте слои комнаты по умолчанию: Когда вы создаете любой новый ресурс комнаты, GameMaker Studio 2 сгенерирует для вас несколько "базовых" слоев - фоновый слой и слой экземпляра. Вы можете отключить это поведение здесь, чтобы новые комнаты не будут иметь предварительно созданных слоев. По умолчанию эта функция включена. Показывать виды помещения: Если в комнате активны виды, это отображается контуром, где границы вида будут находиться в комнате, и видимость этой области можно переключить с помощью специальной кнопки в самом редакторе. Однако вы можете выбрать, чтобы границы вида были невидимы при открытии комнаты, сняв флажок с этой опции (кнопку все равно можно использовать, чтобы показать их снова). Значение по умолчанию - on, что делает границы вида видимыми при открытии комнаты. Общие параметры цвета: Они связаны с различными цветами, которые используются во всем интерфейсе редактора помещений, и если вы дважды щелкните на любом из образцов, вы сможете изменить цвет для этой настройки. Настройка мастер-громкости: Настройка мастер-громкости для воспроизведения предварительного просмотра в редакторе помещений, значение от 0 до 1. По умолчанию - 1. Для редактирования доступны дополнительные опции из перечисленных ниже подкатегорий: Диалоги подтверждения Параметры " Диалоги подтверждения" используются для изменения сообщений, которые отображаются в редакторе помещений при выполнении определенных действий. Вы можете изменить следующие сообщения: Автоматическая реакция на обновление глубин слоев: при изменении глубины слоя появляется сообщение, предупреждающее о том, что при этом также автоматически обновляются все остальные глубины слоев. С помощью этой опции вы можете выбрать, чтобы сообщение отображаться, обновлять слои, как если бы вы нажали "Ок", или не обновлять ничего, как если бы вы нажали "Нет". Автоматическая реакция на предупреждение о блокировке слоя: Если вы попытаетесь удалить слой в комнате, а он окажется заблокированным, по умолчанию вам будет показано сообщение. Однако при установке этого параметра на "Ок" сообщение не будет показано и просто не позволит вам удалить слой. Автоматическая реакция на предупреждение о блокировке элемента: Если вы попытаетесь удалить элемент, который оказался заблокированным, то по умолчанию вам будет показано сообщение. Однако если установить этот параметр в значение "Ok", сообщение не будет показано и просто не позволит вам удалить элемент. Автоматическая реакция на предупреждение о блокировке слоя владельца элемента: Если вы попытаетесь удалить элемент на слое, а слой окажется заблокированным (даже если элемент не заблокирован), то по умолчанию будет показано сообщение о том, что это невозможно сделать. невозможно. Однако если установить этот параметр в значение "Ok", то сообщение не будет показано, и вы просто не сможете удалить элемент. Создание нового слоя актива при перетаскивании последовательностей/ спрайтов на неподходящие слои: Если вы попытаетесь добавить спрайт (или последовательность) на слой, который не является слоем активов, то по умолчанию будет показано сообщение, информирующее о том, что это невозможно и спросит, хотите ли вы создать новый слой активов для перетаскиваемого элемента. Однако при установке этого параметра на "Ok" сообщение не будет показано и автоматически будет создан новый слой для спрайтов (и последовательностей). Автоматическая реакция на удаление элемента родительской комнаты: Здесь вы можете выбрать, что произойдет, если вы удалите элемент, который является частью родительской комнаты (и поэтому присутствует во всех дочерних комнатах). По умолчанию отображается предупреждающее сообщение, и вы сами решаете, что делать. но вы можете установить значение "Удалить все", чтобы подавить сообщение и продолжить удаление элемента из родительской комнаты и всех дочерних комнат, или выбрать "Сохранить все", чтобы подавить сообщение и прервать удаление. Автоматическая реакция на удаление слоя родительской комнаты: Здесь вы можете выбрать, что произойдет, если вы удалите слой, который является частью родительской комнаты (и поэтому присутствует во всех дочерних комнатах). По умолчанию отображается предупреждающее сообщение и вы можете но вы можете установить значение "Удалить все", чтобы подавить сообщение и продолжить удаление слоя из родительской комнаты и всех ее дочерних комнат, или выбрать "Сохранить все", чтобы подавить сообщение и прервать удаление. Автоматическая реакция на ошибку перетаскивания недопустимого актива: Если вы попытаетесь добавить актив в слой, который не предназначен для данного типа актива (например, добавить путь в слой экземпляра), то по умолчанию будет показано сообщение, чтобы информирующее вас о том, что это невозможно сделать. Однако при установке этого параметра на "Ok" сообщение не будет показано, и вы просто не сможете добавить актив. Обратите внимание, что этот параметр не применяется к экземплярам, спрайтам или последовательностям. Создание нового слоя экземпляра при перетаскивании объектов на неподходящие слои: Если вы попытаетесь добавить экземпляр объекта в слой, который не является слоем экземпляра, то по умолчанию будет показано сообщение, информирующее вас о том, что это невозможно и спросит, хотите ли вы создать новый слой для перетаскиваемого объекта. Однако при установке этого параметра на "Ok" сообщение не будет показано и автоматически будет создан новый слой для экземпляра. Автоматическая реакция на ошибку перетаскивания при недопустимом выборе слоя: Здесь вы можете выбрать, что произойдет при перетаскивании ресурса в редактор помещения, если в редакторе слоев выбрано несколько слоев (или не выбран ни один слой). вообще). По умолчанию отображается предупреждающее сообщение, но вы можете установить этот параметр на "OK", чтобы подавить сообщение и продолжить работу, не добавляя никаких ресурсов. Автоматическая реакция на ошибку перетаскивания заблокированного или невидимого слоя: Здесь вы можете выбрать, что произойдет, если вы перетащите ресурс на слой, который был заблокирован или отмечен как невидимый. По умолчанию отображается предупреждающее сообщение, но вы можете можно установить значение "OK", чтобы подавить сообщение и продолжить работу, не добавляя никаких ресурсов. Настройки сетки Параметры настройки сетки используются для изменения работы наложения сетки, при этом доступны следующие опции: Горизонтальное расстояние между ячейками сетки (px): Этот параметр задает горизонтальное расстояние между ячейками сетки комнаты в пикселях. Значение по умолчанию - 32px. Вертикальное расстояние между ячейками сетки (px): Устанавливает вертикальное расстояние между ячейками сетки помещения в пикселях. Значение по умолчанию - 32px. Отображать сетку: Эта опция включает или выключает сетку помещения при первом открытии помещения. По умолчанию она включена, но даже если она выключена, вы можете включить ее снова с помощью соответствующей кнопки в редакторе комнаты. Привязка к сетке: С помощью этого параметра вы можете установить, будут ли активы помещения по умолчанию привязываться к сетке помещения или нет при первоначальном открытии помещения. По умолчанию этот параметр включен, но даже если он выключен, вы можете включить его снова, используя соответствующую кнопку в редакторе комнаты. Цвет линии сетки: Этот параметр позволяет задать цвет (включая альфа-компонент) линий сетки помещения. Значение по умолчанию ( ARGB) #30FFFFFF. Умные гиды В этом разделе подробно описаны Параметры интеллектуальных направляющих для редактора помещений. Доступны следующие опции: Включить интеллектуальные направляющие: Этот параметр используется для включения/выключения использования интеллектуальных направляющих в редакторе помещений. По умолчанию включено. Включить линейки: Этот параметр включает/выключает боковые линейки в редакторе помещений. По умолчанию включена. Привязать линейки к краям: При включении этого параметра направляющие, перетащенные в редакторе комнат из линеек, будут привязаны к любым объектам, размещенным в комнате. По умолчанию включена. Расстояние привязки: Этот параметр указывает редактору помещений, на каком расстоянии от объекта должна находиться направляющая, прежде чем она будет "привязана" к стороне объекта.
