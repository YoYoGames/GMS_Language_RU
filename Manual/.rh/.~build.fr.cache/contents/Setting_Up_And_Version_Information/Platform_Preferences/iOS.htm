ПредпочтенияiOS В настройках iOS есть следующие параметры: Подавить сборку и запуск: По умолчанию при создании проекта iOS и последующей сборке конечного исполняемого пакета GameMaker Studio 2 будет собирать пакет, а затем пытаться запустить его на всех подключенных устройствах. Установка этого параметра подавить сборку приложения и его запуск на устройстве и просто создать проект XCode на главном компьютере Mac Путь установкиmacOS: Путь установки файлов проекта на компьютере Mac для сборки. Идентификатор команды по умолчанию: Здесь вы можете добавить свой идентификатор команды по умолчанию, присвоенный вам компанией Apple. Этот идентификатор команды будет использоваться, когда файлы вашей игры будут отправлены в Xcode для создания приложения и позволит Xcode генерировать необходимые сертификаты подписи. Обратите внимание, что эта настройка будет применяться по умолчанию ко всем играм, созданным для iOS, но ее можно изменить для каждого проекта в общих параметрах игры для iOS.
