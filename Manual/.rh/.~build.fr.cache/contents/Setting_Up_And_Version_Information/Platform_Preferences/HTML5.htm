Предпочтения HTML5 Параметры HTML5 имеют следующие опции: Тип упаковки по умолчанию: При создании конечного исполняемого пакета HTML5 вы можете выбрать между Package as loose files - который просто создаст все необходимые файлы в папке - и Package as Zip - который создаст сжатый файл ZIP , содержащий игру и все дополнительные файлы. По умолчанию вам будет предоставлен выбор, какой из этих двух вариантов вы хотите выбрать при компиляции, но вы можете установить это предпочтение, чтобы всегда использовать один или другой вариант. Обратите внимание, что когда эта опция установлена на Show Dialogue, в реальном диалоговом окне, которое появляется при компиляции, выбор "Remember Packaging Option" установит это предпочтение на опцию который вы выберете. Порт веб-сервера по умолчанию: позволяет изменить порт по умолчанию для всех устройств, созданных через диспетчер устройств. Значение по умолчанию - 51264. Адрес веб-сервера по умолчанию: Здесь вы можете изменить адрес веб-сервера по умолчанию для устройств, которые вы добавляете в диспетчер устройств. Значение по умолчанию - 127.0.0.1. Разрешенные IP-адреса веб-сервера по умолчанию: Здесь вы можете добавить любые разрешенные IP-адреса для веб-сервера, и они будут установлены по умолчанию для всех устройств, добавленных через Target Manager.
