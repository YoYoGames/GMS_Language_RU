Вход IDE и навигация IDE GameMaker Studio 2 поддерживает ввод данных с помощью мыши и клавиатуры, и многие операции могут быть выполнены с помощью одного или другого или обоих. В общем случае вы можете нажать левую кнопку мыши для выбора чего-либо, использовать / + для выбора нескольких элементов, удерживать для перетаскивания элементов в различные доки или на рабочем пространстве, а также для открытия контекстно-специфических меню. Обратите внимание, что если вы используете GameMaker Studio 2 на системе macOS и пользуетесь однокнопочной мышью, пожалуйста, используйте + , чтобы получить щелчок правой кнопкой мыши . Существует также большое количество сочетаний клавиш, которые можно использовать для навигации по различным элементам рабочей области и редакторам ресурсов, и мы быстро рассмотрим некоторые из наиболее важных (полный список можно найти здесь ): / + "Z" : Это отменяет предыдущее и работает в большинстве редакторов. У вас также есть несколько уровней отмены, поэтому вы можете нажать эту кнопку несколько раз, чтобы "откатить" изменения. / + "Y" : Это позволит повторить предыдущее действие, которое было отменено, и работает в большинстве редакторов. Как и в случае с отменой, у вас есть несколько уровней повтора. / + "S" : Это заставит GameMaker Studio 2 сохранить ваш проект. / + "T" : Откроется окно Goto, которое позволяет быстро найти и перейти к любому активу в вашей игре, а также выполнить поиск в Preferences и Game Options. Другие способы навигации по рабочему пространству включают использование сочетания клавиш / + + для перемещения между любыми открытыми окнами в указанном направлении, а также нажав и удерживая среднюю кнопку мыши , а затем перетаскивая мышь, можно перемещаться по рабочему пространству. При перемещении по различным рабочим пространствам и окнам вы можете использовать правую кнопку мыши на любом текстовом поле, чтобы открыть контекстное меню, которое будет содержать следующие пункты поля по умолчанию: В некоторых ситуациях эти опции могут быть развернуты в зависимости от редактора или окна, находящегося в фокусе. Помимо этих способов ввода, имеется ограниченная поддержка перьевых устройств, а также специальный режим для тех, кто работает над проектами с помощью ноутбука. Оба этих режима описаны ниже: Режим ноутбука Если вы используете GameMaker Studio 2 на ноутбуке, то в верхней части IDE у вас будет дополнительная опция Laptop Mode: По умолчанию он включен, но при желании его можно отключить, переключив эту кнопку. Режим ноутбука сочетается с некоторыми инструментами в IDE для улучшения работы с тачпадом, упрощая 3 основных вида взаимодействия с мышью: панорамирование, масштабирование и прокрутка. Для этого используются две клавиши-модификатора: Левая / и Левая . Когда включен режим ноутбука, Левая / и Левая "зарезервированы" для взаимодействия с мышью, например, для панорамирования редактора комнаты, где Левая / - масштабирование, Левая - прокрутка и панорамирование. Прокрутка в режиме ноутбука ведет себя так же, как и панорамирование, поэтому она должна быть интуитивно понятной и достаточно точной, чтобы вам не нужно было использовать полосу прокрутки. Обратите внимание, что, удерживая эти кнопки, вы должны просто перемещать мышь - не щелкать и не перетаскивать, а просто перемещать - и что если есть общая комбинация клавиш, которая обычно требует использование / или , вы должны использовать Правые клавиши клавиатуры вместо этого. Поддержка сенсорного экрана Напоследок отметим, что IDE для GameMaker Studio 2 также имеет минимальную поддержку сенсорных экранов. На всех операционных системах вы можете использовать сенсорный экран для нажатия и перетаскивания элементов в основной рабочей области, и мы поддерживаем 2 одновременных указателей, при этом второе нажатие будет выполнять щелчок правой кнопкой мыши. Обратите внимание, что в Windows 8 и выше IDE GameMaker Studio 2 также поддерживает перьевые устройства.
