Меню Верхнюю часть IDE GameMaker Studio занимает панель меню. Здесь вы найдете ряд выпадающих меню, которые можно использовать для навигации IDE, выполнять задания или открывать дополнительные окна с инструментами, которые помогут вам в создании игр. Обратите внимание, что здесь перечислены меню, доступные по умолчанию, но некоторые редакторы активов добавляют дополнительные меню в зависимости от контекста. Эти дополнительные объясняются на страницах, относящихся к редактору, который их создал.
