Меню макетов Это меню связано с сохранением и загрузкой различных макетов IDE. Вы можете настроить IDE по своему усмотрению, при этом некоторые элементы (например, браузер активов, или окна вывода) расположены иначе, чем по умолчанию. Это позволяет вам настроить GameMaker Studio 2 в соответствии с вашим рабочим процессом. С помощью опций этого меню вы можете сохранить макеты текущего проекта в виде файла *.xml, который затем можно загрузить в любой проект, чтобы установить одинаковые макеты. Опции этого меню следующие: Загрузить макет - откроется проводник файлов, в котором можно выбрать ранее сохраненный файл макета *.xml для загрузки. Сохранить макет - Откроется проводник файлов, где вы можете назвать и сохранить текущий макет IDE в виде файла *.xml. Reset Layout - сброс текущей раскладки IDE к настройкам по умолчанию.
