Меню Windows Это меню предназначено для открытия общих пристыкованных окон IDE. Поскольку доки могут быть переупорядочены, а их содержимое может быть удалено как отдельные окна или даже полностью закрыто, это окно позволяет вам снова открыть их, если вы обнаружите, что они вам для чего-то нужны. Окна, которые можно открыть отсюда, - это окно браузера активов и все различные окна вывода (Output, Search, Source Control, Точки останова и Ошибки компиляции/синтаксиса). Далее вы можете открыть окно Инспектор. Это окно используется для проверки свойств выбранного в данный момент элемента IDE, и эти свойства можно редактировать по мере необходимости. После опции Инспектор находится список Недавние окна, а также возможность открыть окно Переменные среды, в котором можно увидеть различные пути, на которые указывают различные переменные среды. на которые указывают различные переменные окружения, используемые в Предпочтениях. Эти переменные нельзя редактировать (ни их названия, ни пути/настройки), но вы можете выбрать и скопировать/вставить любую из них для использования в другом месте. в другом месте. Наконец-то у вас появилась возможность открыть окна учебника. Это позволит вновь открыть все окна, связанные с учебным проектом. но только если этот проект был создан в рамках учебного курса (в противном случае эта опция будет недоступна). Информацию о создании собственных обучающих проектов см. в разделе Рынок (поскольку все учебники создаются, размещаются и загружаются оттуда).
