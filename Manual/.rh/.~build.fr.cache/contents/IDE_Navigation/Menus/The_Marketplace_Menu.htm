Меню рынка Помимо создания собственных активов и кода для своих проектов, у вас также есть доступ к GameMaker Studio 2 Marketplace, где вы можете загружать как бесплатные, так и платные активы для добавления в свои проекты. Подробнее о Торговой площадке вы можете узнать в разделе в разделе GameMaker Marketplace, а опции из этого меню приведены ниже: Перейти на торговую площадку - откройте окно торговой площадки, где вы можете просматривать активы. Моя библиотека - просмотр текущей библиотеки содержимого Marketplace. Моя история покупок - Эта опция откроет браузер и переведет вас на страницу учетной записи пользователя, вошедшего в систему, чтобы вы могли просмотреть историю покупок для этой учетной записи. Обновить существующий пакет - Эта опция откроет мастер обновления пакета, чтобы вы могли обновить существующий пакет Marketplace, который вы ранее создали. Создать пакет - С помощью этой опции вы можете открыть мастер создания пакета, чтобы создать новый пакет активов или учебник для Marketplace.
