Меню "Файл В меню "Файл" вы можете изменить, сохранить или импортировать проекты, а также установить предпочтения IDE GameMaker Studio 2. Доступны следующие опции: Новый проект - Эта кнопка предназначена для создания нового проекта с нуля в GameMaker Studio 2. Откроется диалог "Сохранить", и вам будет предложено выбрать место для сохранения нового проекта. После того, как вы дадите проекту имя и место его сохранения, вы перейдете в новую, пустую рабочую область, чтобы начать работу над проектом. Файлы проектов сохраняются с расширением *.yyp. Открыть проект - откроется проводник файлов, в котором можно выбрать для загрузки ранее созданный файл проекта *.yyz. Обратите внимание, что при импорте, если проект был создан с использованием версии ниже 2.3.0, то вы получите следующее предупреждение: Если вы решите продолжить, то откроется проводник файлов, и вы можете дать проекту обновленное имя и место для его сохранения, так что оригинальный проект не будет перезаписан. После преобразования проекта в новый формат 2.3.0+ он откроется в IDE, и вы получите сообщение о том, что преобразование прошло успешно. что преобразование прошло успешно. Вы должны проверить отчет о совместимости, который был создан в процессе преобразования, и посмотреть, есть ли что-то, что нужно исправить, прежде чем пытаться запустить или скомпилировать проект. Обратите внимание, что если вы получаете повторяющиеся ошибки при попытке открыть проект, созданный в версии, предшествующей 2.3.0, то вам следует связаться с YoYo Games (используя опцию "Сообщить об ошибке" в меню "Помощь") , предоставив оригинальный файл *.yyp, который вы пытаетесь конвертировать. Импорт проекта - Здесь вы можете импортировать проекты, которые ранее были экспортированы в архив *.yyz (см. раздел "Экспорт проекта", ниже), или - если у вас есть проект, созданный с помощью GameMaker: Studio 1.4 от YoYo Games - вы можете выбрать эту опцию, чтобы импортировать файл *.gmx или *.gmz из этой линейки продуктов. Как и при открытии проекта, если файл *.yyz был создан в версии GameMaker Studio 2 до обновления 2.3.0, то его нужно будет конвертировать, так как формат файла проекта изменился с этим обновлением. ПРИМЕЧАНИЕ: Чтобы импортировать предыдущий проект GameMaker в GameMaker Studio 2, он должен быть создан с помощью версии 1.4, и даже в этом случае он может не работать непосредственно при импорте. Очень многое отличается между GameMaker Studio 2 и GameMaker: Studio 1.4, поэтому ваши старые проекты могут потребовать доработки, чтобы они заработали. Пожалуйста, обратитесь к статье Перенос игры GM:S 1.4 в GameMaker Studio 2 на YoYo Games. База знаний и раздел руководства, посвященный устаревшим функциям и сценариям совместимости. ВАЖНО! При импорте проекта *.yyz, содержащего ресурсы шрифтов, может потребоваться "регенерация" шрифтов путем их удаления/добавления или изменения шрифта в редакторе шрифтов, а затем установки его обратно. его обратно. Это происходит потому, что существующая информация о шрифтах сохраняется на сайте *.yyz при экспорте, а при последующем импорте или на другой машине в проекте могут быть не установлены необходимые шрифты (или могут быть установлены другие шрифты с тем же именем). может быть установлен другой шрифт с тем же именем). Это следует делать только в том случае, если у вас возникли проблемы с отображением шрифта после импорта. Последние проекты - Вы можете использовать эту опцию для просмотра недавно использованных проектов. При выборе этой опции вновь откроется Стартовая страница (если вы ее закрыли), и вы сможете выбрать недавно использованный проект из находящегося там списка. Сохранить проект - сохранит вашу игру в том же месте, которое вы указали при создании проекта (или там, откуда вы его открыли). Сохранить проект как - То же самое, что и выше, только вы указываете местоположение и новое имя проекта. Экспорт проекта - Экспортирует проект в один файл *.yyz. Очень полезно для отправки копий людям или для сохранения резервных копий. Предпочтения - Выбрав этот пункт, вы попадете в окно Предпочтения, где вы можете настроить внешний вид GameMaker Studio 2. Новая IDE - При выборе этого пункта будет открыт новый экземпляр IDE для использования (он будет открыт на стартовой странице). Мой аккаунт - Эта опция откроет ваш веб-браузер по умолчанию и приведет вас на страницу вашего аккаунта YoYo. Выход - при выборе этого пункта вы выйдете из GameMaker Studio 2. Вам нужно будет снова войти в систему с той же лицензией (или любой другой), чтобы иметь возможность использовать продукт в дальнейшем. ВАЖНО! Если вы использовали пробную лицензию GameMaker Studio 2, а затем приобрели одну из других лицензий (Desktop, Mobile, UWP, Web или Console), вам может потребоваться выйти из IDE, затем перезапустить GameMaker Studio 2 и снова войти в систему, чтобы убедиться, что новая лицензия применена правильно (хотя это должно произойти автоматически).
