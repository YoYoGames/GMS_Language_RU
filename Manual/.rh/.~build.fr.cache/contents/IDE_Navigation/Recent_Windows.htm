Последние окна Список " Недавние окна" можно открыть из Windows меню, в нем отображается короткий список (по по умолчанию он содержит максимум 10 элементов) окон, которые вы открывали ранее. Вначале список будет пустым, но по мере открытия различных окон в IDE (например, спрайта, скрипта, объекта и т.д.) список будет пополняться. Если вам нужно вернуться к любому из этих ранее открытых окон, просто нажмите левую кнопку мыши на элементе в списке, и рабочая область перейдет к этому окну. Если вы нажмете правую кнопку мыши на элементе, то откроется следующее меню: Здесь вы можете выбрать удаление элемента (или элементов, поскольку вы можете использовать + или / + для выбора нескольких) из списка или перемещение их в начало списка. В меню RMB также есть дополнительная опция, позволяющая полностью очистить список. Стоит отметить, что Recent Windows может быть прикреплен к боковым сторонам IDE, и вы также можете настроить некоторые Параметры, чтобы изменить его поведение.
