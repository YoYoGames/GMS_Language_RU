Нарисовать линию Это действие нарисует линию в заданном положении в комнате. Вы указываете начальную x/y позицию, а затем конечную x/y позицию, и линия будет проведена между ними с использованием текущего цвета рисования. Позиция может быть либо абсолютной позицией внутри комнаты, либо позицией относительно экземпляра, вызывающего действие. ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах. Синтаксис действия: Аргументы: Аргумент Описание x Начальное положение x y Начальная позиция y x2 Финишное положение x y2 Финишное положение y Пример: Приведенный выше код блока действий рисует розовую линию через верхнюю часть комнаты.
