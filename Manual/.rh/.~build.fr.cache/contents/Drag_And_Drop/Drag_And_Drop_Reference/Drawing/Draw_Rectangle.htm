Нарисовать прямоугольник Это действие нарисует прямоугольник в заданной позиции внутри комнаты. Вы указываете верхнюю левую позицию и нижнюю правую позицию, и прямоугольник будет нарисован между ними, причем позиция может быть либо абсолютной позицией внутри комнаты, либо позицией относительно экземпляра, вызывающего действие. Вы также можете установить, может ли прямоугольник быть заполнен или очерчен, установив флажок заполнения в нижней части. ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах. Синтаксис действия: Аргументы: Аргумент Описание Слева Левая позиция, с которой начинается рисование Топ Верхняя позиция, с которой следует начинать рисовать Справа Правильная позиция для завершения рисования Дно Нижняя позиция для завершения рисования Пример: Приведенный выше код блока действий рисует два прямоугольника, один зеленый и заполненный, а другой желтый и только контур, в позиции относительно экземпляра, вызывающего действия.
