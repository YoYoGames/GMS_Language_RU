Get Draw Colour С помощью этого действия вы можете получить цвет, используемый для любых фигур (прямоугольник/эллипс/линия) или для любого рисуемого текста. Действие вернет цвет в указанную вами целевую переменную, которая может быть создана ранее или может быть быть новой временной (если вы установите флажок "Temp"). Синтаксис действия: Пример: Приведенный выше блок действий получает текущий цвет рисования и, если он не установлен в $ffffffff ( белый), то он устанавливается в этот цвет.
