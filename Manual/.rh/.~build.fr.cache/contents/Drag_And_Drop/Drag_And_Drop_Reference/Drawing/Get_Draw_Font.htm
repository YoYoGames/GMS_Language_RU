Get Draw Font С помощью этого действия можно получить значение ID шрифта, назначенного в данный момент для рисования. Возвращаемое значение будет либо -1, если шрифт не назначен, либо значение >= 0, которое представляет собой константу шрифта из браузера активов. Возвращаемое значение будет сохранено в указанной вами целевой переменной, которая может быть создана ранее или может быть новой временной переменной (если вы установите флажок "Temp"). Синтаксис действия: Пример: Приведенный выше код блока действий проверяет текущий шрифт, используемый для рисования, и если он не является заданный актив, то он устанавливается на этот актив.
