Получить выравнивание текста Это действие получит выравнивание шрифта для всех текущих действий рисования текста. Вы можете выбрать, горизонтальное или вертикальное выравнивание проверять, и действие вернет одно из следующих постоянных значений, которые представляют различные тип горизонтального или вертикального выравнивания: Текст выровнен по горизонтали влево Текст выровнен по центру по горизонтали Текст выровнен по горизонтали вправо Текст выровнен по вертикали к верху Текст вертикально выровнен по центру Текст выровнен по вертикали к низу Возвращаемая константа будет сохранена в указанной вами целевой переменной, которая может быть создана ранее или может быть новой временной переменной (если вы установите флажок "Temp"). Синтаксис действия: Пример: Приведенный выше код блока действий проверяет горизонтальное выравнивание текста и если оно не установлено в fa_left , то оно устанавливается в это значение.
