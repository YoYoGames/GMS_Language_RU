Установить выравнивание текста Это действие установит выравнивание шрифта для всех последующих действий рисования текста. Вы можете установить горизонтальное выравнивание слева, по центру или справа, а вертикальное выравнивание - сверху, посередине или снизу, и текст будет выравниваться относительно x/y как показано на рисунках ниже: Текст выровнен по горизонтали влево Текст выровнен по центру по горизонтали Текст выровнен по горизонтали вправо Текст выровнен по вертикали к верху Текст вертикально выровнен по центру Текст выровнен по вертикали к низу Обратите внимание, что это действие устанавливает выравнивание для всех последующих вызовов рисования текста, поэтому его можно вызвать, например, один раз в начале игры - в любом событии, а не только в событии рисования - и все действия с рисованным текстом будут использовать установленное выравнивание. Однако если вам нужно, чтобы несколько действий с рисующим текстом были выровнены по-разному, то вам придется вызывать это действие перед каждым действием с рисующим текстом, который вы хотите выровнять по-разному. Синтаксис действия: Аргументы: Аргумент Описание H.Выравнивание Значение выравнивания по горизонтали (слева, по центру, справа) V.Выравнивание Значение вертикального выравнивания (верхнее, среднее, нижнее) Пример: Приведенный выше код блока действий устанавливает шрифт, цвет рисунка и выравнивание для некоторого текста. который будет нарисован.
