Draw Healthbar С помощью этого действия вы можете нарисовать цветную полосу для графического представления значения. Отображаемая величина является процентным значением и должна находиться в диапазоне 0 - 100, однако вы можете использовать и большие значения с помощью формулы (CURRENT_Value / MAXIMUM_value) * 100, например, вы можете использовать это для отображения силы, здоровья, маны, времени или чего-либо еще, что может быть полезно для отображения в виде столбика. Помимо значения для рисования, вы можете задать направление для рисования панели здоровья (слева направо или справа налево и сверху вниз или снизу вверх), а также позицию, либо в пределах комнаты, либо относительно экземпляра, вызывающего действие. Помимо позиции, можно также задать цвет, который будет смешиваться с каждым из четырех углов панели здоровья, создавая градиенты от красного к зеленому (например, но вы можете использовать любые цвета).
