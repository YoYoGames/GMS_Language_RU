Установить шрифт С помощью этого действия можно задать шрифт, который будет использоваться для всех последующих действий по рисованию текста. При нажатии на диалог откроется следующее окно, в котором можно выбрать шрифт, который вы хотите использовать: Обратите внимание, что это установит шрифт, который будет использоваться для всех последующих вызовов рисования текста, и поэтому его можно вызвать один раз в начале игры - в любом событии, не только в событии Draw - и все действия по рисованию текста будут использовать установленный шрифт. Однако если в вашей игре требуется несколько шрифтов, то вам нужно будет вызвать это действие перед рисованием любой текст с нужным шрифтом.
