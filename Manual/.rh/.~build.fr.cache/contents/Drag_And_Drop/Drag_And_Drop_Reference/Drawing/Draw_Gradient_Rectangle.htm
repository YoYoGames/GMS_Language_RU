Нарисовать градиентный прямоугольник Это действие нарисует прямоугольник в заданной позиции в комнате, используя набор цветов смешивания для создания градиента. Вы указываете верхнюю левую позицию и нижнюю правую позицию, и прямоугольник будет нарисован между ними, а позиция может быть либо абсолютной позицией в комнате, либо позицией относительно экземпляра, вызывающего действие. Затем вы задаете цвета, которые будут смешиваться с каждым из углов, а также можете установить, может ли прямоугольник быть заполнен или очерчен, установив флажок в нижней части. ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах. Синтаксис действия: Аргументы: Аргумент Описание Слева Левая позиция, с которой начинается рисование Топ Верхняя позиция, с которой следует начинать рисовать Справа Правильная позиция для завершения рисования Дно Нижняя позиция для завершения рисования Слева вверху Цвет для смешивания из левого верхнего угла Верхняя правая часть Цвет для смешивания в правом верхнем углу Внизу слева Цвет для смешивания в левом нижнем углу Внизу справа Цвет для смешивания в правом нижнем углу Пример: Приведенный выше код блока действий рисует градиентный прямоугольник относительно относительно позиции экземпляра с некоторым текстом над ним.
