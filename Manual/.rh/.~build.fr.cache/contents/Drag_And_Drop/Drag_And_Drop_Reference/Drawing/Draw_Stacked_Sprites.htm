Рисование сложенных спрайтов Это действие нарисует спрайт заданного количества спрайтов один за другим в заданном положении в комнате. Вы указываете спрайт для рисования и порядок расположения, который может быть либо ряд (горизонтально, слева направо), либо столбец (вертикально, сверху вниз), а также количество спрайтов для рисования и позицию. Позиция может быть абсолютной позицией в комнате или относительной позицией экземпляра, выполняющего рисование, а расстояние между изображениями будет основано на следующих параметрах ширине или высоте спрайта. Обратите внимание, что при этом рисуется статическое изображение - начальное одиночное изображение (кадр 0) данного спрайта - и все последующие кадры будут проигнорированы, как и любые преобразования, добавленные путем изменения переменных экземпляра (например, image_xscale или image_blend). ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах. Синтаксис действия: Аргументы: Аргумент Описание Спрайт Спрайт для рисования Порядок укладки Порядок рисования (в строке или столбце) Номер Количество спрайтов для рисования x Позиция x для рисования y Позиция y для рисования Пример: Приведенный выше код блока действий получает количество экземпляров объекта "obj_Player" и затем использует его для рисования на экране нескольких маркерных спрайтов.
