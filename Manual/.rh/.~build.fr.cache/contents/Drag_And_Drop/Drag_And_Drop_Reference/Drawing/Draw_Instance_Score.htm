Оценка экземпляра рисунка С помощью этого действия можно нарисовать значение счета экземпляра. При использовании любого действия score в GameMaker Studio 2 у вызывающего его экземпляра будет добавлена новая переменная области видимости экземпляра под названием score. Эту переменную можно проверять и устанавливать, как и любую другую переменную. В случае с этим действием переменная будет выведена на экран в заданной позиции вместе с любым текстом подписи, который вы хотите предоставить. Текст подписи будет нарисован перед значением оценки, и если текст должен быть обернут символом "", и оба текста будут отформатированы с использованием текущего шрифта, цвета и выравнивания. ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах. Синтаксис действия: Аргументы: Аргумент Описание Надпись Строка, которую нужно нарисовать вместе со значением оценки (предшествует и может быть оставлена пустой) x Позиция x, в которой будет нарисован счет y Позиция y для рисования счета Пример: Приведенный выше код блока действий рисует счет, используя заданное форматирование рисования.
