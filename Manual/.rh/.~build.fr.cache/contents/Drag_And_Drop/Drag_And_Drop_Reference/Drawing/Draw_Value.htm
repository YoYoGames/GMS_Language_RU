Draw Value С помощью этого действия вы можете вывести на экран строковую надпись и/или переменную или значение. Аргумент caption - это строка, которую вы хотите добавить перед значением, и при необходимости ее можно оставить пустой, но если вы хотите включить текст, его следует заключить в кавычки. а значение может быть переменной, фиксированным значением или другой строкой - в этом случае его тоже следует заключить в кавычки, но его нельзя оставлять пустым. Затем вы указываете позицию, в которой должна быть нарисована строка, либо как абсолютную позицию или как позицию относительно экземпляра, рисующего текст. Обратите внимание, что вид текста при рисовании зависит от используемого шрифта, цвета и выравнивания. Все эти параметры можно изменить перед рисованием, как показано в примере ниже. ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах. Синтаксис действия: Аргументы: Аргумент Описание Надпись Надпись для рисования (строка), можно оставить пустой Значение Значение для рисования (вещественное, переменное или строка) x Позиция x для рисования y Позиция y для рисования Пример: Приведенный выше код блока действий устанавливает шрифт, цвет шрифта и затем выравнивает текст для рисования от перед тем, как нарисовать значение "level_num" вместе с надписью "Level: ".
