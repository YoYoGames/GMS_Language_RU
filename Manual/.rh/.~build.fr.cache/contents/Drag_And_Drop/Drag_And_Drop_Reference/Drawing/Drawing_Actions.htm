Библиотека действий при рисовании Библиотека действий рисования - это место, где вы можете найти действия, необходимые для рисования спрайтов, текста или фигур, а также установить определенные свойства рисования. Большинство из этих действий предназначены только для использования в различных событиях рисования объекта и могут не работать, если используются за пределами события Draw Event. Исключением являются действия Set, которые можно добавить в любое событие, и после этого они будут влиять на все рисование для всех экземпляров. Важно отметить, что если вы добавите какие-либо действия в основное событие Draw Event объекта, то он не будет рисовать спрайт, который был назначен данному экземпляру, пока вы явно не укажете GameMaker Studio 2 нарисовать его, используя действие например Draw Self. В принципе, GameMaker Studio 2 будет по умолчанию рисовать любой спрайт, назначенный экземпляру, только если в событии Draw Event нет ничего другого.
