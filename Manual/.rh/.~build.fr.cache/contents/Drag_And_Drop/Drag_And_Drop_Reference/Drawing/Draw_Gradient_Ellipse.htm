Нарисовать градиентный эллипс Это действие нарисует эллипс в заданном месте комнаты, используя набор цветов смешивания для создания градиента. Вы задаете левое верхнее и правое нижнее положение области, в которую должен "вписаться" эллипс, и эллипс будет нарисован между ними. Положение может быть либо абсолютным положением в комнате, либо положением относительно экземпляра, вызывающего действие, и вы можете задать цвета для смешивания от края эллипса и от его центра. Эллипс может быть нарисован заполненным или очерченным, установив флажок заполнения в нижней части. ПРИМЕЧАНИЕ: Это действие предназначено только для использования в различных событиях рисования и не будет рисовать ничего, если использовать его в других местах. Синтаксис действия: Аргументы: Аргумент Описание Слева Левая позиция, с которой начинается рисование Топ Верхняя позиция, с которой следует начинать рисовать Справа Правильная позиция для завершения рисования Дно Нижняя позиция для завершения рисования Центр Цвет смешивается от центра Край Цвет для смешивания с краем Пример: Приведенный выше код блока действий рисует градиентный эллипс в позиции экземпляра "obj_Player".
