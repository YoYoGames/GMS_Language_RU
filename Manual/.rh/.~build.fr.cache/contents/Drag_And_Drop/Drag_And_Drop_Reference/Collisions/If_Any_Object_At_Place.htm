If Any Object At Place Это действие используется для проверки наличия экземпляра любого объекта, расположенного в позиции, основанной на маске столкновений вызывающего экземпляра. Вы задаете позицию, которая может быть абсолютной позицией или позицией относительно экземпляра, и действие вернет true, если столкновение есть, или false, если его нет. Вы также можете установить флаг "Not", чтобы проверить, нет ли столкновения в данной позиции, и тогда действие вернет true, если столкновений не обнаружено, и false в противном случае. Столкновения рассчитываются на основе маски столкновений вызывающего экземпляра, перекрывающей маску столкновений любого экземпляра в данной позиции (как если бы он был "помещен" в данную позицию). ВАЖНО! Столкновения будут регистрироваться только для тех экземпляров, у которых есть действительная маска столкновения, т.е. у них есть спрайт, назначенный на sprite_index, или спрайт, назначенный на mask_index. Если один из экземпляров в столкновения не имеет маски столкновения, то столкновение не будет обнаружено, независимо от того, что экземпляр рисует в это время.
