Загрузить игру Это действие загружает игру, которая ранее была сохранена с помощью действия Save Game. Вы указываете имя файла, который нужно загрузить (в виде строки, включая расширение), и игра будет загружена и продолжит работу с сохраненной точки. Обратите внимание, что это не универсальная система сохранения/загрузки, и попытка загрузить игру, сохраненную в предыдущем запуске проекта, может привести к ошибкам (особенно если вы использовали такие вещи, как структуры данных или частицы), и поэтому вы должны пытаться загрузить только те игры, которые были сохранены в процессе прохождения одной игры (для таких вещей, как контрольные точки). Также обратите внимание, что файл не будет загружен до окончания текущего события или сценария, поэтому любые действия после вызова действия загрузки все равно будут выполнены. Более подробный подход к загрузке сохраненных данных игры см. в разделе Действия с файлами. Синтаксис действия: Аргументы: Аргумент Описание Временная шкала Ресурс временной шкалы для назначения экземпляру Пример: Приведенный выше код блока действий проверит глобальную переменную и, если она меньше или равна 0, то будет загружена ранее сохраненная игра.
