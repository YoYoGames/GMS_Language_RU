Сохранить игру Это действие выполняет базовое сохранение игры в заданный файл. Вы указываете имя файла (в виде строки и включая расширение), и действие выполнит ограниченное сохранение из текущего состояния игры. Важно понимать, что действие предназначено для создания "моментального снимка" текущего состояния игры и используется скорее как система контрольных точек, чем как универсальная система сохранения, поскольку оно не сохраняет динамические ресурсы (например, структуры данных или частицы) . Поэтому использовать это действие следует очень осторожно, чтобы не получить ошибок или утечек памяти. Для более комплексного подхода к сохранению игровых данных см. раздел Действия с файлами. Синтаксис действия: Аргументы: Аргумент Описание Имя файла Имя - в виде строки и с расширением - для сохранения файла (если файл уже существует, он будет перезаписан). Пример: Приведенный выше код блока действий проверяет наличие столкновения с другим экземпляром и, если таковое обнаружено, проверяет переменную экземпляра. Если переменная возвращает значение true, то игра сохраняется, а переменная устанавливается в значение false.
