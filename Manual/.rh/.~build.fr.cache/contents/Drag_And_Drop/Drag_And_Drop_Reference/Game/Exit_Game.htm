Выходная игра С помощью этого действия вы можете завершить игру, вызвав событие Other - Game End Event. Обратите внимание, что это произойдет не в момент вызова действия, а в конце текущего события или сценария, поэтому любые действия, которые вы выполняете после вызова этого действия, будут выполняться до тех пор, пока не будет достигнут конец события/скрипта. Обратите внимание, что это действие имеет следующие ограничения: Он сообщит об ошибке на iOS цели, так как это противоречит условиям их рыночной площадки. Он не будет иметь никакого эффекта при вызове на HTML5 и Windows UWP цели. Он будет работать на Windows (включая игры, созданные для Steam), Android, Ubuntu (Linux ) и Mac.
