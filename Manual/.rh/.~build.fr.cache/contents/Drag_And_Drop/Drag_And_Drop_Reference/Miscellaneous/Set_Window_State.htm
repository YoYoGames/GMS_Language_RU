Установить состояние окна С помощью этого действия вы можете установить состояние окна игры в полноэкранный или оконный режим. Здесь есть три варианта: Toggle - Если выбрана эта опция, то при каждом вызове действия окно будет переключаться между полноэкранным и оконным режимами. Fullscreen - переводит окно в полноэкранное состояние из оконного. Windowed - возвращает окно в состояние windowed, если оно находится в полноэкранном режиме. Обратите внимание, что вид игры в полноэкранном состоянии будет зависеть от настроек Game Options - Graphics для ваших целевых платформ, так как если игра не имеет какого-либо механизма адаптации к различным соотношениям сторон и разрешениям, вы можете получить растяжение (если опция игры установлена на Full Scale), или вы можете получить черные полосы (если опция игры установлена на Keep Aspect Ratio). В общем случае, если у вас нет ничего, чтобы с различными разрешениями, вам следует использовать Keep Aspect Ratio для целевых платформ, на которые вы хотите компилировать.
