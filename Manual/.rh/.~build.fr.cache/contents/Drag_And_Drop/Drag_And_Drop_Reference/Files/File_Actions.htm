Файлы Библиотека действий Действия с файлами работают с двумя различными типами файлов - файлами буфера и файлами Ini, а также имеют определенные общие действия с файлами для переименования или копирования существующих файлов. Буферные файлы создаются путем сохранения данных из ранее созданного буфера, которые затем могут быть загружены в буфер в любое время в будущем. Более подробную информацию о буферах вы можете найти здесь: Действия с буфером. Файлы Ini - это маленькие, легкие файлы, совместимые с большинством платформ. Они идеально подходят для хранения небольших фрагментов информации, например, предпочтений интерфейса, локальных высоких результатов, данных об уровне и т.д... и очень просты в использовании. Ini файлы не для использования этих действий, и вы можете читать из несуществующего Ini файла и просто получите возвращаемое значение по умолчанию (которое вы укажете), однако мы рекомендуем вам создать хотя бы "базовый" ini файл для открытия и изменения перед использованием действий. Этот базовый ini файл можно создать, просто вызвав действие Open Ini File, за которым следует Close Ini File, так как закрытие файл будет записан на диск, или вы можете включить его в Included Files браузера активов. Если в качестве базового Ini вы используете файл, включенный в Браузер активов, вам также следует прочитать раздел руководства о том, как работает файловая система. ПРИМЕЧАНИЕ: Для игр с локализацией очень важно, чтобы акцентированные буквы могли быть прочитаны из внешних файлов. Это означает, что вам следует сначала создать ini-файл в формате UTF8, а затем добавить его в GameMaker Studio 2 в качестве включенного файла чтобы он экспортировался при запуске игры и использовался вместо стандартного ini-файла в формате ANSI, который создается функциями ini по умолчанию, если он не был задан ранее. Таким образом, вы сможете правильно читать и писать в нем, сохраняя все ударения и нелатинские буквы. сохраняются все ударения и нелатинские буквы.
