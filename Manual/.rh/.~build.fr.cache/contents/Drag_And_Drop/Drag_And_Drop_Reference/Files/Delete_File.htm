Удалить файл С помощью этого действия вы можете удалить файл, который был создан ранее. Вы указываете имя файла (в виде строки), который нужно удалить. Обратите внимание, что вы не можете удалить файлы, включенные в пакет игры, только те, которые были созданы с помощью действий Close Ini или Save Buffer (подробнее см. раздел " Файловая система "). Синтаксис действия: Аргументы: Аргумент Описание Имя файла Имя (в виде строки) файла для удаления Пример: Приведенный выше код блока действий проверяет, существует ли файл "checkpoint.sav", и если существует, то проверяет, существует ли файл "checkpoint_OLD.sav". то проверяется, существует ли файл "checkpoint_OLD.sav". Если этот файл также существует, то он удаляется, а затем оригинальный файл "checkpoint.sav" переименовывается в "checkpoint_OLD.sav". Наконец, буфер сохраняется как "checkpoint.sav" (по сути, мы создаем резервную копию сохраненного файла буфера при каждом сохранении, так что всегда есть "текущее" сохранение и "старый" файл сохранения).
