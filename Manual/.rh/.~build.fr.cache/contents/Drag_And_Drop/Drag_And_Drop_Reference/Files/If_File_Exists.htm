If File Exists С помощью этого действия вы можете проверить, существует ли файл с заданным именем. Вы выбираете файл для проверки (как жало и включая расширение), и функция возвращает true, если он существует, и false, если нет. Обратите внимание, что функция вернет true, если файл существует в пакете игры как Included File или если файл был сохранен во время работы игры (см. раздел " Файловая система " для получения дополнительной информации). Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Синтаксис действия: Аргументы: Аргумент Описание Тип Тип данных для чтения, либо строка (заключенная в кавычки ""), либо вещественное число Пример: Приведенный выше код блока действий создаст переменную для хранения данных буфера, а затем проверит, не был ли был ли ранее создан файл сохранения буфера. Если файл существует, то он загружается в созданную нами переменную буфера, если же файла не существует, то он создается, записывается и затем сохраняется, готовый к загрузке в следующий раз.
