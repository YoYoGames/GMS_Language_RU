Буфер загрузки С помощью этого действия можно загрузить буфер, который ранее был сохранен с помощью действия Save Buffer. Действие создаст новый буфер, загрузит в него файл данных и вернет уникальное значение ID для идентификации буфера в переменной. Вы указываете имя файла для загрузки (в виде строки) и переменную для хранения возвращаемого идентификатора буфера. Обратите внимание, что если вы отметили переменную как временную локальную переменную, вам необходимо убедиться, что вы удалите буфер, созданный этой функцией, перед иначе вы потеряете ссылку на него и получите утечку памяти при удалении переменной. Синтаксис действия: Аргументы: Аргумент Описание Имя файла Имя (в виде строки) файла, который нужно открыть Цель Целевая переменная для хранения уникального значения ID для буфера, созданного при загрузке Пример: Приведенный выше код блока действий создаст переменную для хранения данных буфера, а затем проверит, не был ли файл сохранения буфера создан ранее. файл сохранения буфера был создан ранее. Если файл существует, то он загружается в созданную нами переменную буфера, если же файла не существует, то он создается, записывается и затем сохраняется, готовый к загрузке в следующий раз.
