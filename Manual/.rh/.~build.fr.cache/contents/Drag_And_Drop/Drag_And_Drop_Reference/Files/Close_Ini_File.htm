Закрыть Ini файл Это действие закроет текущий открытый файл Ini. Это действие следует вызывать в тот момент, когда вы закончили чтение или запись в любой открытый файл Ini (файлы Ini открываются с помощью действия Open Ini File). Если вы не используете это действие после использования действия Write To Ini File, то на диск ничего не будет записано, так как информация о файле хранится в памяти до вызова этого действия, после чего она извлекается из памяти и записывается в файл на диске. Также обратите внимание, что если вы попытаетесь открыть Ini файл, не закрыв предварительно другой (или тот же самый), вы получите ошибку. Синтаксис действия: Пример: Приведенный выше код блока действий откроет ini файл для использования, затем получит значение, связанное с ключом "name" в заголовке "player". Если возвращаемое имя совпадает со значением имени по умолчанию (т.е. файл, раздел или ключ не существует), то перед закрытием файла в него записывается глобальная переменная (и записываются новых данных на диск).
