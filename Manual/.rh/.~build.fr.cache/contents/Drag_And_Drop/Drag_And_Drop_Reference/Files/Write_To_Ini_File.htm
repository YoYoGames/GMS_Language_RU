Write To Ini File С помощью этого действия вы можете записать числовое значение или строку в файл Ini. Вы должны предварительно открыть Ini файл для чтения/записи с помощью действия Открыть Ini файл, а затем с помощью этого действия указать тип данных для записи (либо вещественное число, либо строка, заключенная в ""), а также заголовок секции, ключ в этой секции и тип данных. данных для записи (вещественное число или строка, заключенная в ""), а также заголовок секции, ключ в этой секции и значение для ключа. Для наглядного представления структуры Ini файла см. это изображение: Как вы можете видеть, файлы Ini разбиваются на секции - где каждая секция имеет свое собственное имя заголовка - и затем каждая секция разбивается на различные пары ключ/значение (подобно структуре данных Map). Обратите внимание, что вы можете создавать Ini файлы самостоятельно, используя любую программу для обработки текста (например, Блокнот в Windows) и затем включить их в качестве включенного файла для чтения и записи в вашей игре. ВАЖНО! Попытка прочитать данные из Ini файла, который не был открыт ранее с помощью действия Open Ini File, приведет к серьезной ошибке и краху вашей игры. ВАЖНО! При этом данные записываются только в память, а чтобы окончательно записать данные в файл на диске, необходимо вызвать Close Ini File, иначе данные не будут сохранены.
