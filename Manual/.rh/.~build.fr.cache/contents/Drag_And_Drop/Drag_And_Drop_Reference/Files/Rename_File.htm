Переименовать файл С помощью этого действия вы можете переименовать любой файл, сохраненный вашей игрой. Вы указываете имя файла, которое нужно изменить (в виде строки), а затем новое имя (также в виде строки и включая расширение файла), и файл будет переименован. Обратите внимание, что вы не можете переименовать ни один файл, который был включен в игру в качестве включенного файла (более подробную информацию см. в разделе " Файловая система "). Синтаксис действия: Аргументы: Аргумент Описание Имя файла Имя (в виде строки) файла для переименования Новое имя Новое имя (в виде строки) для файла Пример: Приведенный выше код блока действий проверяет, существует ли файл "checkpoint.sav", и если существует, то проверяет, существует ли файл "checkpoint_OLD.sav". то проверяется, существует ли файл "checkpoint_OLD.sav". Если этот файл также существует, то он удаляется, а затем оригинальный файл "checkpoint.sav" переименовывается в "checkpoint_OLD.sav". Наконец, буфер сохраняется как "checkpoint.sav" (по сути, мы создаем резервную копию сохраненного файла буфера при каждом сохранении, так что всегда есть "текущее" сохранение и "старый" файл сохранения).
