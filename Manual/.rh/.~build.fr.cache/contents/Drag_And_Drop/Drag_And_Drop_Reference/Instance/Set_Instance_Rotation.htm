Set Instance Rotation Этот блок действий устанавливает угол поворота ( image_angle ) экземпляра. Угол измеряется в градусах: вправо - 0°, вверх - 90°, влево - 180° и вниз - 270°. Установите эту переменную в 0, чтобы сбросить экземпляр, что означает, что он будет рисовать любой назначенный спрайт, как он был определен в редакторе спрайтов. Обратите внимание, что для того, чтобы изменения в этом действии были заметны, экземпляр должен либо не иметь события рисования (и тогда GameMaker Studio 2 по умолчанию нарисует спрайт), либо быть нарисованным с помощью действия Draw Self. Синтаксис действия: Аргументы: Аргумент Описание Угол Новый угол, который будет использоваться (от 0 до 360). Пример: Приведенный выше код блока действий устанавливает новый спрайт, а также ряд других свойств для отображения этого спрайта, включая его поворот на 180°.
