If Instance Exists Это действие можно использовать для проверки, существует ли экземпляр определенного объекта в игровой комнате, или существует ли сам определенный экземпляр. Вы указываете объект из проводника активов, и если какой-либо экземпляр существует в комнате, то действие вернет true, в противном случае оно вернет false. Обратите внимание, что здесь можно использовать переменную с уникальным ID конкретного экземпляра (например, возвращаемым действием Create Object Instance) или даже ID экземпляра возвращенный редактором комнаты при добавлении экземпляра. Если вы отметите аргумент "Not", то действие проверит, не существует ли какой-либо экземпляр, и если таковых не найдено, то вернет true, а если хотя бы один существует, то вернет false. Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Теперь эти действия будут выполняться, если "if" оценивается как true, в то время как любые действия, отложенные в другом месте, будут выполняться после блока "if". Синтаксис действия: Аргументы: Аргумент Описание Объект Имя объекта, экземпляр которого необходимо проверить (или идентификатор экземпляра). Пример: Приведенный выше код блока действий проверяет наличие столкновения и возвращает уникальное значение ID экземпляра. с которым произошло столкновение. Это значение сохраняется во временной переменной, а затем проверяется, соответствует ли оно экземпляру, который в данный момент находится в комнате (если столкновение не обнаружено, то в переменной будет ключевое слово "noone"), и если соответствует, то переменная устанавливается в экземпляр, с которым произошло столкновение.
