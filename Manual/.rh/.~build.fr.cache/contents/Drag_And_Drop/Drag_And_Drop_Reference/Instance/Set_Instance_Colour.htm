Установить цвет экземпляра Этот блок действий устанавливает цвет image_blend для "смешивания" со спрайтом экземпляра. Значение по умолчанию - шестнадцатеричный код $FFFFFFFF (который представляет белый цвет в формате ARGB ), и это отрисует спрайт без смешивания и с полной альфой. Любое другое значение (включая внутренние цветовые константы, такие как c_red или c_aqua) будет смешивать указанный цвет с оригинальным спрайтом, а также устанавливать компонент image_alpha для данного экземпляра, но только если установлен флажок Использовать альфу. При этом будет перезаписано любое значение альфа, установленное ранее с помощью действия Set Instance Alpha. Обратите внимание, что для того, чтобы изменения в этом действии были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker Studio 2 по умолчанию нарисует спрайт), либо быть нарисованным с помощью действия Draw Self. Важно важно отметить, что на платформе HTML5 следует ограничить смешивание (если только не используется WebGL), поскольку каждый смешиваемый спрайт должен кэшироваться отдельно, поэтому наличие большого количества смешиваемых спрайтов может негативно сказаться на производительности.
