Set Instance Scale Этот блок действий устанавливает значения image_xscale и image_yscale для экземпляра. Входные значения здесь являются модификаторами, которые будут применены к спрайту, назначенному данному экземпляру, где масштаб 1 (значение по умолчанию) означает отсутствие масштабирования (1:1), меньшие значения будут уменьшать масштаб (0,5, например, уменьшит ширину спрайта в два раза). масштабирование (1:1), меньшие значения будут уменьшать масштаб (0.5, например, уменьшит ширину спрайта вдвое), а большие значения будут увеличивать масштаб. Вы можете использовать отрицательные значения, чтобы перевернуть спрайт и масштабировать его, если только используемое значение не равно ровно -1 (в этом случае спрайт просто переворачивается или зеркально отражается относительно своего начала координат без масштабирования). Вы можете установить горизонтальный или вертикальный масштаб относительно текущих значений, в этом случае вы будете добавлять или вычитать новое значение из текущих значений масштаба. Обратите внимание, что изменение этих значений также изменит способ обнаружения столкновений экземпляром, если только вы не задали отдельную mask_index (маску столкновений) в редакторе объектов. Обратите внимание, что чтобы изменения в этой переменной были видны, экземпляр должен либо не иметь события рисования (и тогда GameMaker Studio 2 по умолчанию нарисует спрайт), либо быть нарисованным с помощью действия Draw Self. Синтаксис действия: Аргументы: Аргумент Описание Горизонтальный Используемый коэффициент вертикального масштабирования (по умолчанию 1). Вертикальный Используемый коэффициент горизонтального масштабирования (по умолчанию 1). Пример: Приведенный выше код блока действий устанавливает новый спрайт, а также ряд других свойств для отображения этого спрайта, включая установку его масштабирования на три четверти от базового размера по обеим осям.
