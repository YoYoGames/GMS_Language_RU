Получить сигнал тревоги Этот блок действий извлекает значение обратного отсчета сигнала тревоги и сохраняет его в заданной целевой переменной. Если вы отметите целевую переменную как "Temp", то это создаст новую временную (локальную) переменную для хранения возвращаемого значения, как показано в примере ниже в примере ниже. Синтаксис действия: Аргументы: Аргумент Описание Сигнализация Сигнал тревоги. Цель Целевая переменная, используемая для хранения значения сигнала тревоги. Пример: Приведенный выше код блока действий создает локальную (временную) переменную и сохраняет в ней текущее значение тревоги[0]. Затем это значение проверяется, и если оно меньше 0, то сигнал тревоги устанавливается на 60.
