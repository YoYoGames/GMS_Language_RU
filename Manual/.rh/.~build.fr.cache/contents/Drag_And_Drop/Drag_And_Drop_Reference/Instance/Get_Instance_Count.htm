Get Instance Count С помощью этого действия вы можете узнать количество экземпляров любого объекта, активного в данный момент в комнате. Вы можете выбрать любой объект из списка ресурсов или использовать специальное ключевое слово все , чтобы получить количество всех активных экземпляров. Вам также необходимо указать целевую переменную для возврата значения, чтобы вы могли использовать его в дальнейших действиях. Если вы обозначите целевую переменную как "Temp", то будет создана новая временная (локальную) переменную для хранения возвращаемого значения, как показано в примере ниже. Синтаксис действия: Аргументы: Аргумент Описание Объект Имя объекта для проверки. Цель Переменная, на которую будет нацелено возвращаемое значение. Пример: Приведенный выше код блока действий создает локальную (временную) переменную, а затем присваивает ей количество экземпляров объекта "obj_Squirrel". Затем это значение проверяется, и если количество экземпляров больше 100, экземпляр уничтожается.
