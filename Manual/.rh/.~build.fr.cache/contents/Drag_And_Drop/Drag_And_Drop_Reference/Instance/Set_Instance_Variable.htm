Установить переменную экземпляра С помощью этого действия вы можете установить одну из встроенных переменных. Все экземпляры всех объектов имеют ряд переменных, созданных для хранения важной информации, такой как текущая позиция x/y, скорость, направление, видимость и многое другое. Переменные экземпляра, которые вы можете использовать, следующие: Координата X - положение вдоль оси x в помещении. Y Coordinate - Положение вдоль оси y в помещении. Direction - Направление движения экземпляра (по умолчанию 0, где вправо - 0°, вверх - 90°, влево - 180° и вниз - 270°). Скорость - скорость (в пикселях на игровой кадр), с которой движется экземпляр. Горизонтальная скорость - горизонтальная скорость (в пикселях на игровой кадр), с которой движется экземпляр. Отрицательная скорость означает, что он движется влево, положительная скорость - вправо, по умолчанию - 0. Вертикальная скорость - вертикальная скорость (в пикселях на игровой кадр), с которой движется экземпляр. Отрицательная скорость означает, что он движется вверх, положительная скорость - вниз, а значение по умолчанию равно 0. Visible - Переменная переключения видимости (по умолчанию true, если в редакторе объектов не установлено иное, установите значение false, чтобы сделать невидимым). Обратите внимание, что невидимый экземпляр по-прежнему выполняет все свои события, кроме события Draw. Persistent - Переменная переключения постоянства (по умолчанию false, если в редакторе объектов не установлено иное, установите значение true, чтобы сделать его постоянным). Это позволит экземпляру "сохраняться" из комнаты в комнату, т.е. экземпляр не будет очищен в конце комнаты, а будет перенесен в следующую комнату. Обратите внимание, что когда экземпляр персистентен он не будет вызывать события Create и Destroy, но будет вызывать события Clean Up, Room End и Room Start. Layer - Уровень, к которому был назначен экземпляр. Глубина - Значение глубины, присвоенное слою. Sprite - Ресурс спрайта, назначенный экземпляру. Индекс изображения - индекс изображения назначенного спрайта. Это значение устанавливает кадр спрайта, назначенного данному экземпляру, и если значение больше, чем количество кадров в анимации, она будет "зацикливаться", так что спрайт с 5 кадрами, если он установлен на кадр 7, на самом деле покажет кадр 2. Угол изображения - угол, под которым должен быть установлен экземпляр (по умолчанию 0, где вправо - 0°, вверх - 90°, влево - 180° и вниз 270°). Альфа изображения - значение альфа для экземпляра (по умолчанию 1) от 0 до 1, где 0 - полностью прозрачный, а 1 - полностью непрозрачный. Image Blend - Цвет, который будет смешиваться со спрайтом экземпляра (по умолчанию белый). Скорость изображения - Скорость анимации для данного экземпляра. Horizontal Scale - горизонтальный масштаб, в котором должен быть нарисован спрайт экземпляра (это также влияет на ограничивающую рамку экземпляра и столкновения). бокс и коллизии). Масштаб 1 означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться вниз (0.5, например, уменьшит ширину спрайта вдвое), большие значения будут масштабироваться вверх, а отрицательные значения будут зеркально отражать спрайт и масштабировать его, за исключением следующих случаев значение равно -1 (в этом случае спрайт просто зеркально отображается по оси y без масштабирования). Vertical Scale - вертикальный масштаб, в котором должен быть нарисован спрайт экземпляра (это также влияет на ограничивающую рамку экземпляра и столкновения). бокс и коллизии). Масштаб 1 означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться вниз (например, 0.5 уменьшит высоту спрайта вдвое), большие значения будут масштабироваться вверх, а отрицательные значения перевернут спрайт и масштабируют его, за исключением следующих случаев значение равно -1 (в этом случае спрайт будет просто перевернут вдоль оси y без масштабирования). Mask - индекс маски для экземпляра. Значение по умолчанию -1 (это означает, что экземпляр будет использовать маску столкновений для но если установить значение для любого другого ресурса спрайта, то вместо спрайта, назначенного на индекс спрайта, будет использоваться маска столкновений для этого экземпляра. Friction - Трение, которое будет использоваться в каждом игровом кадре игры для замедления экземпляра. Это значение будет просто вычитаться из текущего значения скорости в каждом игровом кадре (или прибавляться, если значение скорости отрицательное). Гравитация - Гравитация будет прикладывать постоянную силу в направлении гравитации (см. ниже) экземпляра, влияя на его скорость (по умолчанию установлено значение 0). Обратите внимание, что гравитация является кумулятивной силой и будет ускорять объект, если вы решите не ограничивать конечную скорость. Направление гравитации - Этот параметр можно использовать для задания направления движения, когда гравитация экземпляра больше 0 (по умолчанию 270). Обратите внимание, что направления в GameMaker Studio 2 рассчитываются как 0° - вправо, 90° - вверх, 180° - влево и 270° - вниз. Score - установить переменную score. Lives - установить переменную lives. Здоровье - установить переменную здоровья. Выбрав встроенную переменную для использования, вы затем устанавливаете значение для этой переменной. Используемое значение будет зависеть от выбранной переменной (см. список выше для значений по умолчанию и диапазонов), и вы можете отметить значение, чтобы оно было относительно текущего значение переменной. Вы также можете добавить дополнительные переменные для установки в том же действии, нажав на значок плюса рядом с и выбрать другую переменную экземпляра и задать ей значение. Синтаксис действия: Аргументы: Аргумент Описание Переменная Встроенная переменная для установки Значение Значение, в которое нужно установить переменную Пример: Приведенный выше код блока действий получает значения для предыдущей позиции x/y, а затем сравнивает их с текущей позицией x/y. Если они не совпадают, то индекс спрайта для данного экземпляра изменяется, а индекс изображения устанавливается в 0.
