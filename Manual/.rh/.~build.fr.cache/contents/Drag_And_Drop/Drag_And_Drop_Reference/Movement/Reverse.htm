Реверс Это действие используется для отмены определенных значений, связанных с направлением и движением. Доступны следующие варианты: Изменение направления просто добавит 180° к текущему направлению (если необходимо, зациклится на 360°). например, если изменить направление на 270°, то получится 90°), изменение горизонтальной или вертикальной скорости просто умножит заданный вектор скорости на -1, а выбор гравитации сделает то же самое, что и выбор направления, только для только для направления гравитации. Синтаксис действия: Аргументы: Аргумент Описание направление Направление для реверса Пример: Приведенный выше код блока действий проверяет столкновение экземпляра с "obj_Wall" относительно текущей позиции влево и вправо. Если таковое обнаружено, то горизонтальная скорость изменяется на противоположную. Затем то же самое делается над и под экземпляром, при обнаружении столкновения изменяется вертикальная скорость.
