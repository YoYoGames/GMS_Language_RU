Установить переменную направления Это действие используется для задания направления для экземпляра, которое не фиксировано с шагом 45°. Вы задаете значение направления от 0° до 360° - где вправо - 0°, вверх - 90°, влево - 180° и вниз - 270° - а также можете установить флаг relative, чтобы добавить/вычесть значение, которое вы задаете, к текущему направлению. Синтаксис действия: Аргументы: Аргумент Описание Направление Направление от 0° до 360° (значения вне диапазона будут завернуты). Пример: Приведенный выше код блока действий проверяет, удерживается ли левая кнопка мыши. и если она удерживается, то устанавливает направление на случайное значение от 0 до 360 и скорость на 5.
