Установить направление Случайно Это действие используется для установки направления движения экземпляра на одно или несколько случайных направлений, выбранных из 8 кардинальных точек. Вы можете щелкнуть одну или несколько из 8 стрелок направления внутри действия, и действие выберет одно из них случайным образом для использования в качестве направления для экземпляра. Если вы не выберете ни одну из стрелок, действие не повлияет на текущее направление экземпляра. Синтаксис действия: Пример: Приведенный выше код блока действий выбирает либо true, либо false и сохраняет результат во временной (локальной) переменной. Затем эта переменная проверяется, и если это true, то выбирается один набор случайных направлений, а если false, то другой.
