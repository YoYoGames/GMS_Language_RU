Установленное направление Фиксированное Это действие используется для задания фиксированного направления движения. Доступные направления обозначены различными кнопками в блоке действий, и щелчок на любой из них задает это направление. Обратите внимание, что направления в GameMaker Studio 2 измеряются в градусах: вправо - 0°, вверх - 90°, влево - 180° и вниз - 270°. Синтаксис действия: Пример: Приведенный выше код блока действий проверяет наличие нажатия левой кнопки мыши на экземпляре, и если таковое обнаружено, он устанавливает направление вниз (270°) и скорость на 5.
