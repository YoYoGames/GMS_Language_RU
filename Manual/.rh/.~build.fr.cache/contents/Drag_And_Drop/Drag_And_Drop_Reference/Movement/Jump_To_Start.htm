Перейти к началу Это действие используется для перемещения экземпляра из его текущего положения в положение, в котором он был первоначально создан. Исходное положение, в котором был создан экземпляр, хранится в двух переменных экземпляра, называемых xstart и ystart. Эти переменные могут быть использованы в любом другом действии, а также могут быть установлены для изменения записанного начального положения. Синтаксис действия: Пример: Приведенный выше код блока действий проверяет, не находится ли верхняя часть ограничивающей рамки экземпляра ниже высоты помещения, и если это так, то перемещает экземпляр обратно в позицию, в которой он был создан.
