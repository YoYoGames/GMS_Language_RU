Clear Grid С помощью этого действия вы можете очистить структуру данных сетки до одного значения (каждая ячейка в сетке будет установлена на заданное значение). Вы передаете переменную, содержащую индекс сетки (как возвращается действием Создать сетку) и затем задаете значение, до которого нужно очистить сетку. Синтаксис действия: Аргументы: Аргумент Описание Сетка Индекс (хранится в переменной) сетки, которую нужно очистить Значение Значение, до которого нужно очистить сетку Пример: Приведенный выше код блока действий очищает данную структуру данных сетки, чтобы в каждой ячейке было ключевое слово noone. каждой ячейке, если глобальная переменная меньше или равна 0.
