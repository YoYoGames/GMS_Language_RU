Получить количество элементов списка Вы можете использовать это действие, чтобы узнать, сколько записей содержится в списке. Вы предоставляете переменную, которая содержит индекс списка (возвращенный действием Создать список), и целевую переменную для хранения возвращаемого числа элементов в списке (которая может быть помечена как временная локальная переменная, которая будет использоваться до конца сценария или события). Синтаксис действия: Аргументы: Аргумент Описание Список Индекс (хранится в переменной), по которому можно получить количество элементов Цель Целевая переменная для хранения возвращаемого количества элементов Пример: Приведенный выше код блока действий получает размер списка, на который ссылается глобальная и затем использует цикл for для его просмотра и уничтожения экземпляров с идентификаторами, хранящимися в списке.
