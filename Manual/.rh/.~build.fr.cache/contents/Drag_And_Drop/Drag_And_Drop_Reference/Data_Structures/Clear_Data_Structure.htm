Очистить структуру данных Это действие очистит все значения из заданной структуры данных. Сначала необходимо выбрать тип структуры данных для очистки, а затем указать индекс структуры данных (индекс - это значение, которое возвращается и хранится в переменной при создании структуры данных). структуру данных). Очищенная структура данных считается "пустой" (т.е. в ней нет записей), за исключением сеток . В случае с сетками это действие просто очистит все ячейки сетки до 0. ПРИМЕЧАНИЕ: Это не освобождает структуру данных из памяти, а только очищает ее от любой информации. Чтобы освободить память, отведенную для структуры данных, необходимо использовать действие Free Data Structure. Синтаксис действия: Аргументы: Аргумент Описание Структура Тип структуры данных, которую вы очищаете (список, карта, сетка или стек). Переменная Переменная, содержащая индекс структуры данных для очистки Пример: Приведенный выше код блока действий проверяет, не пуста ли структура данных, на которую ссылается в глобальной переменной не пуста, и если это так, то структура данных очищается.
