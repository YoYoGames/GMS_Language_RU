Если структура данных существует С помощью этого действия вы можете проверить переменную, чтобы узнать, индексирует ли она структуру данных или нет. Сначала нужно выбрать тип структуры данных для проверки (список, карта, сетка или стек), а затем указать переменную, которая содержит структуру данных индекс (возвращаемый действиями Create List, Create Map, Create Grid и Create Stack). Действие вернет true, если структура данных существует, и false, если не существует. Обратите внимание, что в действии можно поставить флаг "not", чтобы проверить, не хранит ли переменная индекс структуры данных. Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Эти действия теперь будут выполняться, если "if" оценивается в true, в то время как любые действия, отброшенные в другом месте, будут выполняться после блока "if". Синтаксис действия: Аргументы: Аргумент Описание Структура Тип структуры данных для проверки (список, карта, сетка или стек) Переменная Переменная, содержащая индекс структуры данных Пример: Приведенный выше код блока действий проверяет, существует ли заданная структура данных и если она существует, то она освобождается из памяти.
