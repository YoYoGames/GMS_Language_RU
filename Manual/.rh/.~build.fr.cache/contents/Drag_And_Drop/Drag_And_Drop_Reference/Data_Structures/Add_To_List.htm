Добавить в список Это действие можно использовать для добавления нового значения (любого типа данных) в список, и это значение будет добавлено в конец. Вы передаете переменную, в которой хранится индекс списка (возвращенный действием Создать список) и значение для хранения. Синтаксис действия: Аргументы: Аргумент Описание Список Индекс (хранится в переменной) списка для добавления в список Значение Значение для добавления в список Пример: Приведенный выше код блока действий создает глобальную переменную области видимости, а затем создает новую структуру данных списка, присваивая ее индексное значение глобальной переменной. Затем область видимости изменяется таким образом, чтобы все экземпляры объекта "obj_Enemy_Parent" добавили свое уникальное значение ID экземпляра в список.
