Удалить запись на карте С помощью этого действия вы можете удалить ключ (и соответствующее ему значение) из структуры данных карты. Вы предоставляете переменную, которая содержит индекс карты (возвращается действием Create Map), а затем указываете "ключ" (это это идентификатор значения в карте), и действие удалит этот ключ из карты. Обратите внимание, что это действие ничего не возвращает, и если вам нужно значение для данного ключа, то перед его удалением следует воспользоваться действием Get Map Value. Синтаксис действия: Аргументы: Аргумент Описание Карта Индекс (хранится в переменной) карты, из которой нужно удалить значение Ключ Ключ для использования (вещественный или строковый) Пример: Приведенный выше код блока действий извлекает значение определенного ключа из структуры данных map и затем проверяет, меньше или равно ли это значение 0.
