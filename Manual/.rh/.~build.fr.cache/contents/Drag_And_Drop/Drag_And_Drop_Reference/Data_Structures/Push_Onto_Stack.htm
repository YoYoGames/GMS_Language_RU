Push Onto Stack С помощью этого действия вы можете поместить значение в структуру данных стека. Вы предоставляете переменную, которая содержит индекс стека (возвращается действием Create Stack), а затем указываете значение, которое нужно протолкнуть на вершину. При использовании стековых значения всегда добавляются на "вершину" стека, и никогда в конец или в середину (если вам нужна такая функциональность, используйте структуру данных list). Синтаксис действия: Аргументы: Аргумент Описание Стек Индекс (хранится в переменной) стека для использования Значение Значение для переноса в стек Пример: Приведенный выше код блока действий создает переменную экземпляра и новую структуру данных стека. Индекс стека хранится в новой переменной, а затем выполняется цикл, который создает 10 экземпляров и помещает их уникальные значения ID в стек.
