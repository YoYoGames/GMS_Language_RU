Удалить из списка Это действие можно использовать для удаления элемента с индексом из заданного списка. Вы передаете переменную, которая хранит индекс списка (возвращенный действием Создать список) и позицию индекса в списке для удаления, где индексная позиция находится в диапазоне от 1 до (длина списка -1). Это действие ничего не возвращает, поэтому если вам нужно значение в позиции индекса, то перед удалением индекса следует использовать Get List Item At. Синтаксис действия: Аргументы: Аргумент Описание Список Индекс (хранится в переменной) списка, из которого нужно удалить список Индекс Индекс в списке для удаления Пример: Приведенный выше код блока действий проверяет наличие столкновения в позиции экземпляра и, если уникальное значение ID для экземпляра сохраняется во временной переменной, а затем проверяется, существует ли оно в структуре данных списка. Если оно существует, то элемент удаляется из списка, а экземпляр уничтожается.
