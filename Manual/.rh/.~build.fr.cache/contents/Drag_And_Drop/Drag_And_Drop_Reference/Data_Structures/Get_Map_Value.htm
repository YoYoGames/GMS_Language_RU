Получить значение карты С помощью этого действия вы можете получить значение, связанное с заданным ключом в структуре данных карты. Вы предоставляете переменную, которая содержит индекс карты (как возвращается действием Create Map), а затем задаете "ключ" (который это идентификатор в карте для значения), а также предоставляете целевую переменную для хранения возвращаемого значения (которая может быть помечена как временная локальная переменная). ПРИМЕЧАНИЕ: Если вы зададите ключ, который не существует в карте, действие вернет неопределенный результат. Проверить это можно с помощью действия If Undefined. Синтаксис действия: Аргументы: Аргумент Описание Карта Индекс (хранится в переменной) карты, из которой нужно получить значение Ключ Ключ, который нужно получить (вещественный или строковый) Цель Целевая переменная для хранения возвращаемого значения карты Пример: Приведенный выше код блока действий проверяет наличие столкновения в позиции экземпляра, и если таковое найдено, то выполняется поиск по карте, чтобы получить значение из ключа "damage", и это значение затем используется для изменения вызывающей переменной экземпляра "hp".
