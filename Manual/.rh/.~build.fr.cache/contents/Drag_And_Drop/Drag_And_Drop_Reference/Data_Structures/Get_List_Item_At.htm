Get List Item At С помощью этого действия вы можете восстановить индексную позицию элемента в заданном списке. Вы предоставляете переменную, которая хранит индекс списка (возвращенный действием Создать список) и позицию индекса в списке, чтобы а также целевую переменную для хранения возвращаемого значения элемента (которая может быть помечена как временная локальная переменная, которая будет использоваться до конца сценария или события). ПРИМЕЧАНИЕ: Если вы зададите позицию, которая находится за пределами заданного размера списка (т.е. позиция 11 в списке из 10 значений), то действие вернет неопределенное значение. Вы можете проверить это с помощью действия If Undefined. Синтаксис действия: Аргументы: Аргумент Описание Список Индекс (хранится в переменной) списка, из которого нужно удалить список Индекс Позиция индекса в списке, из которого нужно получить элемент Цель Целевая переменная для хранения возвращаемого значения Пример: Приведенный выше код блока действий получает размер списка, на который ссылается глобальная переменная, а затем использует цикл for. чтобы пройти по нему и уничтожить экземпляры с идентификаторами, хранящимися в списке.
