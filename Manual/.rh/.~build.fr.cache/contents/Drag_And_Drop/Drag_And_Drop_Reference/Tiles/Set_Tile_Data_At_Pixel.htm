Установить данные плитки на пиксель С помощью этого действия можно установить данные плитки для ячейки плитки, находящейся в определенном положении на слое карты плитки. Данные плитки - это просто значение, отражающее индекс плитки вместе со значениями flip, mirror, rotate, а также любые пользовательские маски плитки, которые вы использовали. В этом действии сначала указывается слой, на который нужно нацелиться, используя имя слоя (строка), заданное в редакторе помещения, а затем оси x и y в помещении. для которых нужно установить данные плитки. Обратите внимание, что данные плитки будут установлены для "ячейки" карты плитки, в которой находится заданная позиция. Например, если набор плиток имеет размер 16x16, то слой карты плиток будет содержать плитки в ячейках размером 16x16 пикселей, поэтому если вы используете это действие для указания позиции комнаты (8, 8), вы измените плитку в ячейке (0, 0) карты плиток. ПРИМЕЧАНИЕ: Выбранный слой должен быть определен как слой карты плиток в редакторе помещений, иначе могут возникнуть ошибки. Последним аргументом действия являются сами данные плитки. Обычно данные тайла для ячейки карты тайла получают с помощью действия Get Tile Data At Pixel, а затем манипулируют ими с помощью действия Set Tile Data Transform, прежде чем снова задать их с помощью этого действия. Более подробную информацию о данных тайла см. в разделе GML " Функции карты тайла". Синтаксис действия: Аргументы: Аргумент Описание Слой Слой с картой плиток, на который нужно нацелиться X Положение вдоль горизонтальной оси (X) для установки данных плитки Y Положение по вертикальной оси (Y) для установки данных плитки Данные Данные плитки, используемые для установки ячейки в позицию Пример: Приведенный выше код блока действий проверяет наличие щелчка мыши и при его получении он извлекает данные плитки для ячейки, совпадающей с позицией мыши. Затем эти данные изменяются и возвращаются в слой карты плиток, поворачивая плитку в этой ячейке.
