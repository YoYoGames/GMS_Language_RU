Получить преобразование данных о плитке С помощью этого действия вы можете узнать, были ли данные плитки преобразованы одним из трех способов: перевернуты, зеркально отображены или повернуты. Данные плитки - это просто значение, отражающее индекс плитки вместе с примененными преобразованиями. Обычно перед использованием этого действия необходимо сначала получить данные плитки с помощью Get Tile Data In Cell или Get Tile Data At Pixel. Действие вернет значение true, если выбранное преобразование было применено к данным плитки, или false в противном случае, а возвращаемое значение будет сохранено в целевой переменной, которая может быть создана ранее или может быть новой временной переменной (если вы установите флажок "Temp"). флажок). Более подробную информацию о тайловых данных можно найти в разделе GML " Функции тайловой карты". Синтаксис действия: Аргументы: Аргумент Описание Плитка Данные плитки для получения преобразования из плитки Преобразование Преобразование для проверки Пример: Приведенный выше код блока действий получает данные плитки в позиции мыши, а затем проверяет, не было ли к плитке применено преобразование вращения. Если это так, то преобразование сбрасывается, и данные плитки снова устанавливают плитку в это положение.
