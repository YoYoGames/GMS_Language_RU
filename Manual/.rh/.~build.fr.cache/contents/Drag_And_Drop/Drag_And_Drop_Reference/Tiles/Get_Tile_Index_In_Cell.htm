Получить индекс плитки в ячейке С помощью этого действия можно получить индекс плитки на заданном слое. Слой задается из имени слоя (строка, как используется для названия слоя в редакторе комнат), а затем указывается строка и столбец ячейки в слое для получения индекса плитки. индекс плитки. Наконец, вы задаете переменную target для хранения индекса плитки, где 0 - отсутствие плитки, а целое положительное число - индекс плитки. Обратите внимание, что переменная target может быть помечена как временная (локальная) переменная, которая может использоваться до окончания события. Если нет, то индекс плитки для этой ячейки устанавливается на 0, а временная (локальная) переменная увеличивается, прежде чем цикл выполнит еще одну проверку. выполняет еще один набор проверок.
