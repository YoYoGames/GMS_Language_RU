Получить данные о плитке в ячейке С помощью этого действия можно получить данные плитки для ячейки ячейки на слое карты плитки. Данные плитки - это просто значение, отражающее индекс плитки, а также значения флипа, зеркала и поворота, а также любые пользовательские маски плитки, которые вы использовали. В этом действии сначала указывается слой, на который нужно нацелиться, используя имя слоя (строку), определенное в редакторе помещений, а затем столбец и строку слоя карты плитки, для которых нужно получить данные плитки. для. Ячейка" - это область на сетке карты плиток, в которой находится плитка, данные которой вы хотите получить, поэтому, если ваши плитки имеют размер 16x16, например, а комната имеет размер 1024x768, карта плиток будет иметь 64 столбца и 48 строк. ПРИМЕЧАНИЕ: Выбранный слой должен быть определен как слой карты плиток в редакторе помещений, иначе могут возникнуть ошибки. Возвращенное значение будет сохранено в целевой переменной, которая может быть создана ранее или может быть новой временной переменной (если вы установите флажок "Temp"). Для получения дополнительной информации о данных тайлов см. раздел GML " Функции карты тайлов".
