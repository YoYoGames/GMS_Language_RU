Набор плиток С помощью этого действия можно задать слою использование наборов плиток, отличных от созданных в браузере активов. Вы задаете имя слоя (строка, как определено в редакторе помещений), а затем ресурс набора плиток для использования, и все плитки в помещении на этом слое будут нарисованы с новым набором плиток. Синтаксис действия: Аргументы: Аргумент Описание Слой Слой, на котором будет установлен новый набор плиток. Набор плиток Набор плиток, используемый для слоя. Пример: Приведенный выше код блока действий проверяет, используют ли плитки на слое "Floor_Tiles" набор плиток "tl_PalaceRuins", и если нет, то они будут установлены на его использование.
