Получить переменную Tilemap С помощью этого действия можно получить любую из заданного количества переменных для элемента карты плитки на слое. Когда вы создаете слой плитки в редакторе помещений, этот слой содержит элемент карты плитки, который затем заполняется плитками из набора плиток. Этот элемент карты плитки имеет определенные значения, связанные с ним, такие как позиция смещения, используемый набор плиток, ширина, высота и т.д.. Полный список значений, которые вы можете получить, следующий: Координата X: Положение вдоль оси x на слое, где был размещен элемент карты плитки. Координата Y: Положение вдоль оси y на слое, где был размещен элемент карты плитки. Столбцы: Ширина (определяется как количество столбцов ячеек плитки) элемента карты плитки. Ряды: Высота (определяется как количество рядов ячеек плитки) элемента карты плитки. Ширина плитки: ширина одной ячейки плитки. Высота плитки: Высота одной ячейки плитки. Ресурс набора плиток: Ресурс набора плиток, который был назначен элементу карты плиток для рисования. Текущий кадр: Текущий кадр, рисуемый для анимированных плиток. Mask: Данные маски для элемента карты плитки. Глобальная маска: данные глобальной маски для всех элементов карты плитки. При использовании этого действия вы указываете имя слоя (строка, как определено в редакторе помещений), из которого нужно получить данные элемента карты плитки, затем тип данных, которые вы хотите получить (как показано в списке выше). Возвращенное значение будет сохранено в целевой переменной, которая может быть создана ранее или может быть новой временной переменной (если вы установите флажок "Temp"). Обратите внимание, что вы можете получить дополнительные значения, нажав на значок плюса рядом с и выбрать другую переменную и указать другую переменную для хранения возвращаемого значения. Синтаксис действия: Аргументы: Аргумент Описание Слой Уровень, из которого будут получены данные Переменная Переменная для получения значения (как указано выше) Пример: Приведенный выше код блока действий проверяет, используют ли плитки на слое "Floor_Tiles" использует набор плиток "tl_PalaceRuins", и если нет, то он будет установлен для использования.
