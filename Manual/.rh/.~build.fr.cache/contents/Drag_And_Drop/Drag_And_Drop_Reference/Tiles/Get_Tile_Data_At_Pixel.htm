Получить данные о плитке на пикселе С помощью этого действия можно получить данные плитки для ячейки на слое карты плитки из позиции. Данные плитки - это просто значение, отражающее индекс плитки вместе со значениями флипа, зеркала и поворота, а также любыми пользовательскими масками плитки, которые которые вы использовали. В этом действии сначала указывается слой, на который нужно нацелиться, используя имя слоя (строку), определенное в редакторе помещения, а затем оси x и y в помещении, для которых нужно получить данные плитки. Возвращаемое значение данных тайла будет совпадать с любой ячейкой карты тайлов, находящейся в данной точке. Например, если ваш набор плиток имеет размер 16x16, то слой карты плиток будет содержать плитки в ячейках размером 16x16 пикселей, поэтому если вы используете это действие для нацеливания на позицию комнаты (8, 8), вы получите данные плитки для ячейки (0, 0) карты плитки. ПРИМЕЧАНИЕ: Выбранный слой должен быть определен как слой карты плиток в редакторе помещений, иначе могут возникнуть ошибки. Возвращенное значение будет сохранено в целевой переменной, которая может быть создана ранее или может быть новой временной переменной (если вы установите флажок "Temp"). Для получения дополнительной информации о данных тайлов см. раздел GML " Функции карты тайлов". Синтаксис действия: Аргументы: Аргумент Описание Слой Слой с картой плиток, на который нужно нацелиться X Положение вдоль горизонтальной оси (X) для установки данных плитки Y Положение по вертикальной оси (Y) для установки данных плитки Пример: Приведенный выше код блока действий проверяет наличие щелчка мыши и при его получении он извлекает данные плитки для ячейки, совпадающей с позицией мыши. Затем эти данные изменяются и возвращаются в слой tile-map, поворачивая плитку в этой ячейке.
