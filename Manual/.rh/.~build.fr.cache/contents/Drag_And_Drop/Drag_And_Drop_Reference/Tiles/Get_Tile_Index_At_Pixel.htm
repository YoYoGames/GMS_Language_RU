Получить индекс плитки в пикселе Это действие используется для получения индекса плитки на заданном слое. Слой задается из имени слоя (строка, используемая для названия слоя в редакторе комнат), а затем задается позиция x и y для поиска плитки, индекс которой нужно получить. Плитка будет та, в которую попадает заданная позиция в комнате, а индекс будет возвращен в переменную target, где 0 - нет плитки, а целое положительное число - индекс плитки. Позиции x и y могут быть помечены как относительные к позиции вызывающего экземпляра, а переменная target может быть помечена как временная (локальная) переменная, которая может использоваться до конца события. Синтаксис действия: Аргументы: Аргумент Описание Слой Слой, на котором будет установлена плитка x Позиция x для проверки y Позиция y для проверки Цель Целевая переменная для хранения индекса плитки Пример: Приведенный выше код блока действий проверяет наличие нажатия левой кнопки мыши и если таковая обнаружена, он получает индекс ячейки плитки, в которую попадает позиция мыши. Затем проверяется индекс плитки, и если он не равен 0, то он устанавливается в 0.
