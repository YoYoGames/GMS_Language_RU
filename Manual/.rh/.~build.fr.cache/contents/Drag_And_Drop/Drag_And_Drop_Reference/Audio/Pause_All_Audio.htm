Pause All Audio Это действие можно использовать для приостановки всех воспроизводимых в данный момент звуков, но это не приостановит воспроизведение звуков, воспроизводимых после вызова этого действия. Вы можете запустить их воспроизведение снова с помощью действия Resume All Audio.
