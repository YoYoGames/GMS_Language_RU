Возобновить все аудиозаписи Это действие можно использовать для возобновления всех ранее приостановленных звуков, которые были приостановлены с помощью Pause Audio или Pause All Audio. Это действие не будет иметь эффекта на те звуки, которые не были приостановлены ранее. Синтаксис действия: Пример: Приведенный выше код блока действий проверяет глобальную переменную, и если ее значение равно true, то весь звук приостанавливается, в противном случае он возобновляется.
