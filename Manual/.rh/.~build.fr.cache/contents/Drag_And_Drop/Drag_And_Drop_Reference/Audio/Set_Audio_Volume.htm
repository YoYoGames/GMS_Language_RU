Настройка громкости звука С помощью этого действия можно установить громкость заданного звука. Вы выбираете звук в проводнике активов, а затем устанавливаете значение громкости. Громкость может быть от 0 (тихо) до 1 (полная громкость), шкала линейная, так что значение 0,5 означает половина громкости. Обратите внимание, что это действие повлияет на все экземпляры звука, которые воспроизводятся в данный момент в комнате, и повлияет на все будущие экземпляры воспроизводимого звука. Синтаксис действия: Аргументы: Аргумент Описание Звук Звуковой ресурс для установки громкости Объем Новая громкость для звука (от 0 до 1, по умолчанию 1) Пример: Приведенный выше код блока действий проверяет глобальную переменную на истинность или ложность. Если она истинна, то громкость данного звука извлекается и сохраняется во временной локальной переменной. Затем проверяется, не равно ли это значение 0,5, и если нет, то громкость звука устанавливается равной 0,5. Если значение глобальной переменной равно false, то громкость звука устанавливается равной 1.
