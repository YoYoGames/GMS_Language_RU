Библиотека звуковых действий GameMaker Studio 2 имеет полноценный звуковой движок, основанный на форматах *.ogg, *.mp3 и *.wav. Звуки этих типов, добавленные в IDE, могут быть использованы в вашей игре с помощью основных звуковых действий, показанных ниже. Для более сложных вещей, чем базовые звуковые эффекты или воспроизведение одного Вы можете обратиться к расширенным аудиофункциям, посвященным потоковому аудио, позиционированию аудио - для придания 3D-звука - и группировке аудио. ВАЖНО! При использовании звука в HTML5-цели следует помнить, что не все браузеры поддерживают Web Audio, поэтому при запуске ваш проект может не воспроизводить звук. Вы можете получить общее представление о поддержке Web Audio по следующей ссылке ссылка: Могу ли я использовать Web Audio?
