Получить высоту звука Это действие извлекает высоту тона предоставленного вами звукового ресурса. Возвращаемое значение будет множителем для фактического записанного тона образца, где значение 1 означает отсутствие изменений. Более подробную информацию о высоте тона см. в действии Set Audio Pitch. Синтаксис действия: Аргументы: Аргумент Описание Звук Звуковой ресурс для получения высоты тона Цель Целевая переменная для хранения возвращаемого шага Пример: Приведенный выше код блока действий сначала получает высоту тона для заданного звука и сохраняет ее во временной локальную переменную. Затем проверяется нажатие клавиши, и если оно обнаружено, то значение высоты тона проверяется на то, меньше ли оно 2. Если да, то звук останавливается, к переменной высоты тона добавляется 0,1, и затем эта переменная используется для установки высоты тона звука снова перед воспроизведением. высоты тона звука перед его воспроизведением.
