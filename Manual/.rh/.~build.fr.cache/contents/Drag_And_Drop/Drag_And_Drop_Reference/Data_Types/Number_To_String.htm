Number To String Это действие преобразует вещественное число в строку. Если число не является целым и имеет десятичные знаки, они будут округлены до ближайших двух знаков, поэтому, например, число 123.45678 вернет строку "123.46". Синтаксис действия: Аргументы: Аргумент Описание Значение Вещественное число для преобразования Цель Целевая переменная для хранения возвращаемого значения Пример: Приведенный выше код блока действий принимает десятичное значение и преобразует его в целое число, сохраняя возвращаемое значение во временной локальной переменной.
