String To Number Это действие может преобразовать строку в число. Вы предоставляете строку для изменения, а затем указываете целевую переменную для возврата реального числа (эта переменная может быть помечена как временная локальная переменная). Обратите внимание, что если в строке есть символы кроме цифр и десятичной точки, то она не будет преобразована, хотя если строка начинается с цифры, то она будет преобразована, а остальная часть строки исключена. Важно отметить, что при передаче десятичных дробей в это действие действие, возвращаемая строка будет округлена только до двух знаков после запятой. Две строки считываются из буфера и сохраняются во временных локальных переменных, после чего одна из них преобразуется в вещественное число и сохраняется в другой временной локальной переменной.
