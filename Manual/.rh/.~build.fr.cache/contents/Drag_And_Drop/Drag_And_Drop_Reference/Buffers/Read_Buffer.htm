Буфер чтения Это действие можно использовать для чтения данных из ранее созданного буфера. Вы предоставляете уникальное значение ID буфера (возвращаемое действием Create Buffer) и указываете переменную для хранения возвращаемого значения. Если вы отметите переменную как временную локальную переменную, то она будет создана для вас и отброшена в конце события или сценария действия. Все данные возвращаются в виде строки, поэтому если вы храните какие-либо реальные числа, вам придется использовать одно из действий типа данных, чтобы преобразовать их из строки в значение. Следует также помнить, что буферы считываются и записываются последовательно, поэтому при чтении из буфера позиция "seek" будет перемещаться на следующую запись, а это значит, что для чтения различных фрагментов данных вы просто вызываете это действие необходимое количество раз и сохраняете каждый возвращенный фрагмент данных в переменную. Синтаксис действия: Аргументы: Аргумент Описание Буфер Индекс буфера (хранится в переменной) Цель Целевая переменная для хранения возвращенных данных Пример: Приведенный выше код блока действий устанавливает позицию чтения/записи буфера в начало буфера и затем считывает два фрагмента данных, сохраняя их (в виде строк) во временных локальных переменных.
