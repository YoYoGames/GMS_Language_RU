Буфер копирования Это действие можно использовать для копирования всех данных, хранящихся в одном буфере, в другой. При использовании двух буферов и копировании из одного в другой оба буфера должны быть предварительно созданы с помощью действия Create Buffer, а в качестве аргументов from/to указывается уникальный идентификатор буфера. ПРИМЕЧАНИЕ: Нельзя копировать в один и тот же буфер. Синтаксис действия: Аргументы: Аргумент Описание Буфер Индекс буфера (хранится в переменной) для копирования из Копировать в Индекс буфера (хранится в переменной) для копирования в Пример: Приведенный выше код блока действий создает новый буфер и сохраняет его ID во временной переменной. Затем он проверяет, существует ли буфер, индексированный в данной глобальной переменной, и если существует, то он копируется во временный буфер, а позиция чтения/записи сбрасывается на начало, а если не существует, то он создается.
