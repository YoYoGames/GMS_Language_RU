Получить положение буфера Это действие можно использовать для получения текущей позиции для чтения/записи в буфер. Когда вы создаете буфер, позиция чтения/записи устанавливается на 0 байт, и по мере добавления данных в буфер эта позиция будет последовательно перемещаться, что означает, что вы можете использовать это действие для получения количества байт в буфере, на которое он установлен в любой момент времени. Вы предоставляете уникальное значение ID буфера, возвращаемое действием Create Buffer, а затем указываете целевую переменную для возврата значение позиции. Целевая переменная может быть помечена как временная локальная переменная, в этом случае она будет создана для вас и удалена из памяти в конце текущего сценария действия или события. Синтаксис действия: Аргументы: Аргумент Описание Буфер Индекс буфера (хранится в переменной) Позиция Позиция чтения/записи (в байтах) в буфере Пример: Приведенный выше код блока действий получает позицию чтения/записи для заданного буфера и если она не равна 0 (т.е. находится в начале буфера), то она устанавливается в начало буфера.
