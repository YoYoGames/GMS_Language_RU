Буфер поиска Это действие можно использовать для перемещения (поиска) к позиции в буфере. Вы указываете уникальное значение ID буфера, возвращаемое действием Create Buffer, а затем значение смещения в байтах для перемещения. Значение смещения будет будет применяться от начала буфера, поэтому вызов этой функции переместит позицию чтения/записи в начало буфера, затем переместит на заданное количество байт и установит новую позицию чтения/записи. Однако, если вы отметите смещение как относительное, то позиция чтения/записи будет перемещена вперед (или назад, если смещение отрицательное) от текущей позиции.
