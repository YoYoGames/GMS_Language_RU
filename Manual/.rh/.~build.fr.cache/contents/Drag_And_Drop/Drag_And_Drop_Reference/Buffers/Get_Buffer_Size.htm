Получить размер буфера Это действие можно использовать для получения размера буфера в байтах. Вы предоставляете уникальное значение ID буфера, возвращаемое действием Create Buffer, а затем указываете целевую переменную для возврата значения размера. Цель может быть помечена как временная локальная переменная, в этом случае она будет создана для вас и удалена из памяти в конце текущего сценария действия или события. Синтаксис действия: Аргументы: Аргумент Описание Буфер Индекс буфера (хранится в переменной) Цель Целевая переменная для хранения возвращаемого значения Пример: Приведенный выше код блока действий получает размер буфера, а затем использует это значение для установки чтения/записи в конец текущих сохраненных данных.
