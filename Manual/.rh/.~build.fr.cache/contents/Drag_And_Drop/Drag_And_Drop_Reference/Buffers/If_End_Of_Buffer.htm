If End Of Buffer Это действие можно использовать для проверки того, достигла ли позиция чтения/записи конца буфера. Вы предоставляете уникальное значение ID буфера, возвращаемое действием Create Buffer, и действие возвращает true, если позиция чтения/записи достигла конца всех доступных данных в буфере, или false, если нет. Вы можете установить флаг "not", чтобы действие проверяло, не находится ли позиция чтения/записи в конце буфера. Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Синтаксис действия: Аргументы: Аргумент Описание Буфер Индекс буфера (хранится в переменной) Пример: Приведенный выше код блока действий создает список DS в экземпляре и затем циклически просматривает данные в заданном буфере, считывая их и добавляя в список.
