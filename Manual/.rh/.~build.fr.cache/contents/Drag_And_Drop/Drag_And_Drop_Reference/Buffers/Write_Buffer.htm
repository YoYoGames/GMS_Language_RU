Буфер записи Это действие можно использовать для записи данных в буфер. Вы указываете уникальный идентификатор буфера для записи (возвращаемый действием Create Buffer), а затем значение для записи, которое может быть либо вещественным, либо строковым, но обратите внимание, что все значения хранятся как строки, поэтому при чтении из буфера (с помощью Read Buffer) вам может понадобиться использовать одно из действий типа данных, чтобы преобразовать его из строки обратно в исходное значение. Помните также, что буферы считываются и записываются последовательно, поэтому при записи в буфер позиция "seek" сдвигается на одну позицию, а это значит, что для записи различных фрагментов данных вы просто вызываете это действие необходимое количество раз. Синтаксис действия: Аргументы: Аргумент Описание Буфер Индекс буфера (хранится в переменной) Значение Значение для добавления в буфер Пример: Приведенный выше код блока действий создает глобальную переменную, а затем буфер, присваивая уникальный идентификатор буфера переменной.
