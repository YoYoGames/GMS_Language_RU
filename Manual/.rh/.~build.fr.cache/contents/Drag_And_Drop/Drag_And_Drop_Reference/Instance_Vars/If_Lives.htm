If Lives Это действие используется для проверки значения переменной экземпляра lives с помощью определенного выражения. Вы задаете тип выражения для проверки и значение для проверки текущей переменной lives, а оператор "if" будет всегда будет возвращать либо true, либо false в зависимости от используемых выражений и значений. Доступными выражениями являются: Equals to - Переменная и значение в точности равны. Меньше - Переменная меньше значения Больше чем - Переменная больше, чем значение Less than or Equal to - Переменная меньше или равна значению Greater than or Equal to - Переменная больше или равна значению Если вы отметите аргумент Not, то вышеперечисленные выражения будут отрицаемыми, например, "equals to" станет " not equals to",таким образом, вы будете проверять, не равно ли значение lives заданному значению. ВАЖНОЕ ПРИМЕЧАНИЕ: Из-за проблем с точностью плавающей точки, проверка на точное равенство двух значений может вернуть ложное значение, так как одно из них может быть равно 1, а другое - 1.00000000000001. другое может быть 1.00000000000001. Этого можно избежать, используя действие Decimal to Integer перед проверкой или используя выражения "больше чем" или "меньше чем". Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Эти действия теперь будут выполняться, если "if" оценивается в true, в то время как любые действия, отброшенные в другом месте, будут выполняться после блока "if". Синтаксис действия: Аргументы: Аргумент Описание Не Устанавливается для проверки, если выражение не имеет значения true. Выражение Тип выражения, используемый для проверки. Значение Значение, по которому проверяются жизни. Пример: Приведенный выше код блока действий проверит значение lives, чтобы узнать, меньше или равно ли оно 0, и если это так, то комната будет перезапущена.
