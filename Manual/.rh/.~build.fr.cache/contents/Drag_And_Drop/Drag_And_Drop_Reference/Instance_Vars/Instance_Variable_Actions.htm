Переменная экземпляра Библиотека действий Действия с переменными экземпляра предназначены для начала работы над игрой и используются для создания и поддержания определенных переменных экземпляра для счета, жизней и здоровья (подробнее о переменных можно узнать здесь). Эти действия будут создавать эти переменные для вас в любое время, когда они используются, и предназначены скорее для удобства или для новичков, а если вам нужно более сложный контроль над игровыми значениями, вам следует создавать свои собственные переменные с помощью соответствующих действий (см. здесь).
