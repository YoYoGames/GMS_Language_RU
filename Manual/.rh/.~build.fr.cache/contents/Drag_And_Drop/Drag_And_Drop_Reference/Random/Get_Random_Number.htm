Получить случайное число Это действие генерирует случайное число в пределах набора минимальных и максимальных значений. Вы можете выбрать тип генерируемого числа (десятичное или целое число), а затем задать диапазон, в котором будет генерироваться число, указав минимальное и максимальное значение. Сгенерированное случайное значение будет возвращено в целевую переменную (которая может быть помечена как временная локальная переменная). ПРИМЕЧАНИЕ: При генерации случайного целого числа действие вернет целое значение между минимальным и максимальным значением включительно, поэтому диапазон 5 - 10 может вернуть 5 или 10, а также любое целое число между ними. Однако при генерации случайной десятичной дроби никогда не вернет максимальное значение, поэтому диапазон от 5 до 10 может вернуть 5, но не вернет 10 (хотя может вернуть 9.999999999999). Синтаксис действия: Аргументы: Аргумент Описание Тип Тип используемых чисел (целое или десятичное) Минимум Минимальное число для случайного диапазона Максимальный Максимальное число для случайного диапазона Цель Целевая переменная для хранения возвращаемого значения Пример: Приведенный выше код блока действий генерирует случайное значение и присваивает его временной локальной переменной. Затем это значение используется в цикле повтора, в котором выполняются дальнейшие случайные вызовы для создания позиций в комнате. Эти позиции затем используются для создания экземпляров чисел.
