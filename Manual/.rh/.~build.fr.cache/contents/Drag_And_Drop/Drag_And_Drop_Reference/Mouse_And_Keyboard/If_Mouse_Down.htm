If Mouse Down С помощью этого действия вы можете проверить, удерживается ли выбранная кнопка в данный момент. Действие будет возвращать true на каждом шаге, когда кнопка удерживается нажатой, и false в противном случае, хотя если вы нажмете модификатор not, то сможете проверить, не удерживается ли кнопка, т.е. действие будет возвращать true, когда кнопка не нажата, и false, если нажата. Если вам нужно проверить только одно нажатие кнопки, используйте действие If Mouse Pressed. Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Эти действия теперь будут выполняться, если "if" оценивается в true, в то время как любые действия, отброшенные в другом месте, будут выполняться после блока "if".
