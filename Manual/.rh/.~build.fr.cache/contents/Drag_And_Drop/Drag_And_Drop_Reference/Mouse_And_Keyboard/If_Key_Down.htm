Если клавиша нажата С помощью этого действия вы можете опросить клавиатуру, чтобы узнать, удерживается ли в данный момент какая-либо клавиша. Действие будет возвращать true на каждом шаге, когда клавиша удерживается нажатой, и false в противном случае, хотя если вы нажмете модификатор not, то сможете проверить, не нажата ли клавиша, т.е. действие будет возвращать true, когда ни одна клавиша не нажата, и false, если нажата. Если вам нужно проверить только одно нажатие клавиши, используйте действие If Key Pressed.
