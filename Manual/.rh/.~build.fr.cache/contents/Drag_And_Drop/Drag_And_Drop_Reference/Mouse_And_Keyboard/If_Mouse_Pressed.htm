При нажатии мыши Это действие проверяет первоначальное нажатие на кнопку мыши. Оно возвращает true на одном игровом шаге (кадре), когда кнопка нажата изначально, и false во всех остальных случаях. Если вы включите модификатор not, это действие будет обратным и будет проверять, не была ли нажата кнопка, возвращая false на том игровом шаге (кадре), на котором она была нажата, и true во всех остальных случаях. Если вам нужно проверить, удерживается ли кнопка нажатой, а а не только начальное событие нажатия, то используйте действие If Mouse Down. Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Эти действия теперь будут выполняться, если "if" оценивается в true, в то время как любые действия, отброшенные в другом месте, будут выполняться после блока "if". Синтаксис действия: Аргументы: Аргумент Описание Кнопка мыши Название кнопки мыши для проверки (левая, средняя, правая). Не Отрицание проверки (истина становится ложью и наоборот) Пример: Приведенный выше код блока действий опрашивает состояние кнопки мыши на каждом шаге и, если она удерживается нажатой, он проверяет начальное нажатие кнопки мыши вниз. Если проверка истинна при первоначальном нажатии кнопки вниз, устанавливается скорость экземпляра, затем, пока кнопка нажата, смешивание спрайта изменяется на красное. Если мышь не удерживается, проверка выполняется при отпускании мыши, чтобы сбросить цвет смешивания на белый и установить скорость на 0.
