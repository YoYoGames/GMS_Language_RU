Скрыть виртуальную клавиатуру Это действие можно использовать для скрытия виртуальной клавиатуры на устройстве, на котором запущена игра. Вызов этого действия генерирует системное асинхронное событие, в котором карта async_load DS map будет заполнена следующими парами ключ/значение: " event_type" - это будет "статус виртуальнойклавиатуры", когда срабатывает от действий виртуальной клавиатуры. "screen_height" - высота виртуальной клавиатуры (в пикселях). Это значение будет равно 0, если клавиатура невидима. "keyboard_status" - текущее состояние клавиатуры, возвращается в виде одной из следующих строк: "скрытие" "скрытый" "показывающий" "видимый" Синтаксис действия: Пример: Приведенный выше код блока действий проверяет наличие действия нажатия кнопки мыши, и если оно обнаружено обнаружено, то выполняется проверка, отображается ли виртуальная клавиатура ОС. Если нет, то она устанавливается на отображение - с использованием типа цифровой клавиатуры - а если уже отображается, то скрывается.
