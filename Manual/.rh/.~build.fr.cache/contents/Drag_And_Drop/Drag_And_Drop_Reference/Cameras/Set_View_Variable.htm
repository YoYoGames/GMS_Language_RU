Установить переменную вида Это действие позволяет установить определенные встроенные переменные, связанные с тем, как порт просмотра будет отображать вид камеры на комнату. Вы выбираете переменную порта просмотра для редактирования, затем задаете цель порта просмотра (от 0 до 7) и, наконец, новое значение для переменной. Порт просмотра - это область экрана, которая будет использоваться для рисования данного вида камеры (как установлено в редакторе помещения), и различные переменные, которые вы можете установить таким образом, описаны в таблице ниже: Переменная Описание Camera Эта функция предназначена для настройки области просмотра для отображения другого вида камеры. Вам необходимо ввести уникальное значение ID камеры, на которую нужно перейти (см. здесь более подробную информацию). Visibility Это переключатель видимости для порта просмотра, который принимает значение true или false. При установке значения true порт просмотра станет видимым (т.е. будет рисовать на экране), а значение false не будет ничего рисовать на экране. ничего на экран. View Port X Coordinate Это X-положение порта просмотра в игровом окне. Обратите внимание, что это значение действительно необходимо, только когда активно более одного порта просмотра - например, для создания эффекта разделенного экрана - и если оно используется, когда активен только один порт, вы получите неожиданные результаты. View Port Width Это ширина (в пикселях) порта просмотра. Ширина порта просмотра (или комбинированных портов просмотра, если активны несколько портов) определяет ширину игрового окна или фонового холста в начале игры, поэтому изменение этого значения после начала игры не окажет видимого влияния на размер игрового окна, если не вызвать функцию GML. не окажет видимого влияния на размер игрового окна, если только оно не будет вызвано вместе с функцией GML window_set_size().
