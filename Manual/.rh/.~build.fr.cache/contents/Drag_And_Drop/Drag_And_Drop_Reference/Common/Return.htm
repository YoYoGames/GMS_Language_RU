Возврат Это действие используется при создании функций блока действий и служит для возврата значения экземпляру, вызывающему сценарий. Возвращаемое значение может быть чем угодно, что может хранить переменная - например, строкой. например, строка, числовое значение, указатель или идентификатор ресурса, или же это может быть сама переменная. Обратите внимание, что когда мы вызываем действие return в функции, функция завершается, возвращая заданное значение, и никакие дальнейшие действия после return вызываться не будут, Хотя, если в событии после вызова функции есть действия, они все равно будут выполнены. Синтаксис действия: Аргументы: Аргумент Описание Возврат Возвращаемое значение. Пример: Приведенная выше функция блока действий устанавливает временную переменную на ключевое слово никто, а затем проверяет входную переменную, чтобы узнать, какое значение она имеет в данный момент. Если значение переменной больше 100, то создается экземпляр, и его уникальный идентификатор экземпляра сохраняется во временной переменной, которая затем возвращается из функции.
