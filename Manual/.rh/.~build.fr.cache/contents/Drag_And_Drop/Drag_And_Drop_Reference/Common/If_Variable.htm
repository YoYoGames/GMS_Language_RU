Переменная If Это действие используется для проверки состояния заданной переменной на соответствие заданному значению с помощью определенного выражения. Оператор "if" всегда возвращает либо true, либо false, а выражения для проверки могут быть любыми из следующих следующих: Equals to - Переменная и значение в точности равны. Меньше - Переменная меньше значения Больше чем - Переменная больше, чем значение Less than or Equal to - Переменная меньше или равна значению Greater than or Equal to - Переменная больше или равна значению Если вы отметите аргумент "Not", то вышеперечисленные выражения будут отрицаемыми, например, "equals to" станет " not equals to". Также обратите внимание, что ни аргументы переменной, ни аргументы значения не ограничиваются числовыми значениями и могут быть строкой, указателем, идентификатором ресурса или чем-либо еще, что функция может вернуть или использовать (см. раздел о типах данных для получения дополнительной информации), но вы не можете сравнить два например, если переменная содержит строку, а вы пытаетесь проверить, равна ли она числу, вы получите синтаксическую ошибку. ВАЖНОЕ ПРИМЕЧАНИЕ: Из-за проблем с точностью плавающей точки, проверка на точное равенство двух значений может вернуть ложное значение, так как одно из них может быть равно 1, а другое - 1.00000000000001. другое может быть 1.00000000000001. Этого можно избежать, используя действие Decimal to Integer перед проверкой или используя выражения "больше чем" или "меньше чем". Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Эти действия теперь будут выполняться, если "if" оценивается в true, в то время как любые действия, отброшенные в другом месте, будут выполняться после блока "if". Синтаксис действия: Аргументы: Аргумент Описание Переменная Имя переменной для проверки. Не Устанавливается для проверки, если выражение не имеет значения true. Выражение Тип выражения, используемый для проверки. Значение Значение, по которому проверяется переменная.
