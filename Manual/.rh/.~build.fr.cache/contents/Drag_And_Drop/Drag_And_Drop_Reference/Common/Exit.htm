Выход Это действие будет выполнять одно из двух действий в зависимости от контекста, в котором оно вызывается. Если оно вызывается внутри события, то оно завершит событие и перейдет к следующему, пропуская любой код блока действий, который может быть размещен после него. Однако если он используется внутри функции блока действий, то он выйдет только из функции и вернется к событию, вызвавшему функцию, выполняя все последующие действия. Обратите внимание, что вызов в середине функции не вернет никакого значения (для этого используется действие return ). Синтаксис действия: Пример: Приведенный выше код блока действий проверяет переменную, и если она больше или равна 100, событие завершается, в противном случае переменная увеличивается на единицу и спрайту экземпляра присваивается случайный масштаб.
