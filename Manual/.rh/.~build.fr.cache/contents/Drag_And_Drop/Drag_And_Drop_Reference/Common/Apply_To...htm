Apply To... Это действие может быть использовано для изменения области действия для всех последующих действий, связанных с этим действием. Как и в случае с действием if, если вы отбросите последующие действия справа от действия Apply to..., то они будут выполняться из области видимости объекта или экземпляра. Область действия выбирается из всплывающего меню, отображаемого при нажатии на маленькую стрелку в правом верхнем углу действия, и таким образом вы можете указать другому набору объектов - или даже одному экземпляру, если у вас есть идентификатор экземпляра - выполнить несколько действий, прежде чем продолжить работу с кодом текущего объекта. Обратите внимание, что для добавления действий в блок "Применяется к", их следует опустить сбоку от действия, как показано на изображении ниже: Теперь эти действия будут выполняться из объекта или экземпляра, на который нацелен блок "Applies To", в то время как любые действия, брошенные в другом месте, будут выполняться после блока "Applies To". Синтаксис действия: Пример: Приведенный выше код блока действий проверяет переменную на то, равна ли она 100, и если да, то изменяет область видимости на экземпляры " obj_Player" и говорит им выполнить действие Set Sprite ( все экземпляры этого объекта в комнате изменят спрайт в этот момент), а затем продолжает добавлять 1 к переменной в экземпляре, выполняющем полный блок кода.
