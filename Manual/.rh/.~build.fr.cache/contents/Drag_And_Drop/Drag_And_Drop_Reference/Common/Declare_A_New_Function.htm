Объявление новой функции Это действие можно использовать для создания новой определяемой пользователем функции, которая затем может быть вызвана в любое время в вашем коде. Определяемая пользователем функция - это функция, которую вы определяете для выполнения задачи, не охваченной доступными действиями DnD™ или функциями бегущей строки GML, Они могут быть глобальными, экземплярами или локальными по области применения (более подробную информацию об области применения см. здесь ), а также могут использоваться в качестве конструкторов для создания структур. Чтобы узнать больше об использовании этого действия для создания глобальных функций, обратитесь к разделу Создание функций действия. Вы также можете пометить функцию как статическую, это означает, что она будет определена только один раз при первом использовании. Чтобы использовать это действие, вы можете добавить его в событие экземпляра, а затем дать имя функции, которую вы хотите объявить. Затем вы указываете необходимые имена аргументов для каждого из параметров, которые будет принимать функция. Если функция будет использоваться в качестве конструктора (т.е. она будет создавать структуру), то вам нужно установить флаг Constructor, а затем для вызова функции использовать действие New. Если функция будет использоваться только локально в одном и том же событии и нигде больше, то можно также выбрать флаг Temp . После определения имени и параметров вы можете добавить действия, которые должны выполняться при вызове функции в вашем коде, переместив их в сторону как показано на рисунке ниже: Примеры использования в качестве обычной функции и функции-конструктора вы можете увидеть ниже. Синтаксис действия: Аргументы: Аргумент Описание Функция Имя новой функции. Аргумент0 ... АргументХ Имена различных аргументов для каждого из параметров, которые должны быть переданы в функцию.
