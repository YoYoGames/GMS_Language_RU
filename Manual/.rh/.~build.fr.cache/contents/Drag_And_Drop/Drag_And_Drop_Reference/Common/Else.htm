Else Действие "else" используется только в сочетании с If Variable (или любым другим действием "if") и представляет собой способ определения набора действий, которые должны быть выполнены, если оценка "if" возвращает ложное значение. Например, вы можете создать действие "если", которое проверяет, что переменная здоровья игроков меньше или равна 0, и если это значение равно true, вы уничтожаете экземпляр. Однако если вы хотите выполнить другой набор действий кода, если здоровье игрока не меньше 0, то вы можете добавить "else" и затем указать действия, которые будут выполняться, пока if будет иметь значение false. Обратите внимание, что для добавления действий в блок "else" их следует опустить сбоку от действия, как показано на изображении ниже: Любые действия, брошенные в другом месте, будут выполнены после блока "else" блока и его содержимого.
