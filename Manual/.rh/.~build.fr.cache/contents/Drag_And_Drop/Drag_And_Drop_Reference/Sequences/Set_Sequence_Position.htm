Установить позицию последовательности С помощью этого действия вы можете установить положение данного элемента последовательности в комнате. Вы указываете ID элемента последовательности (возвращенный действием Создать последовательность), а затем новое положение по X и Y. Если вы установить флажок "относительная", то значения, указанные для X и Y, будут добавлены к текущей позиции последовательности, в противном случае они будут абсолютной позицией. Синтаксис действия: Аргументы: Аргумент Описание Переменная Идентификатор элемента последовательности для установки позиции. X Новая позиция x в комнате для последовательности. Y Новое положение y в комнате для последовательности. Пример: Приведенный выше код блока действий извлекает текущую позицию элемента последовательности ID, хранящийся в переменной "my_seq". Затем эти значения используются для установки позиции последовательности, перемещая ее на 10 пикселей вправо.
