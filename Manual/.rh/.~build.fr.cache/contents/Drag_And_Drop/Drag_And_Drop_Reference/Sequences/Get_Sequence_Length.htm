Получить длину последовательности С помощью этого действия вы можете получить длину (в кадрах) заданного элемента последовательности в комнате. Вы предоставляете идентификатор элемента последовательности (возвращенный действием Создать последовательность), и это действие вернет его длину в целевой переменной. Это будет количество кадров, в течение которых выполняется последовательность. Обратите внимание, что целевая переменная может быть помечена как временная (локальная), которая будет создана для действия и может использоваться во всех последующих действиях для события, в котором происходит действие. Синтаксис действия: Аргументы: Аргумент Описание Переменная ID элемента последовательности для получения длины. Цель Целевая переменная для хранения длины. Пример: Приведенный выше блок действий извлекает длину идентификатора элемента последовательности, хранящегося в переменной "seq_id". Это значение затем используется для установки позиции головки того же элемента последовательности на 10 кадров до его конца.
