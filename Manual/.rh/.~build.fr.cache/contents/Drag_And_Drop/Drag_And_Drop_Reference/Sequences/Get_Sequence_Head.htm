Получить головку последовательности С помощью этого действия вы можете получить текущее положение головки воспроизведения заданного элемента последовательности в комнате. Вы указываете ID элемента последовательности (возвращенный действием Create Sequence), и это действие вернет текущий кадр, на котором находится головка воспроизведения последовательности, в переменную target. Информацию о головке воспроизведения см. в разделе Использование листа допинга. Обратите внимание, что целевая переменная может быть помечена как временная (локальная), которая будет создана для действия и может использоваться во всех последующих действиях для события, в котором происходит действие. Синтаксис действия: Аргументы: Аргумент Описание Переменная Идентификатор элемента последовательности для получения позиции головки. Цель Целевая переменная для фиксации положения головы. Пример: Приведенный выше блок действий извлекает текущую позицию головки идентификатора элемента последовательности, хранящегося в переменной "seq_id". Затем это значение используется для установки позиции головки того же элемента последовательности, сдвигая его на 4 кадра.
