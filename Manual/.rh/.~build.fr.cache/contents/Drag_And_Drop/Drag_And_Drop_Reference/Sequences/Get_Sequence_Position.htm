Получить позицию последовательности С помощью этого действия вы можете получить текущую позицию для данного элемента последовательности в комнате. Вы предоставляете идентификатор элемента последовательности (возвращенный действием Создать последовательность), и это действие возвращает его положение по X и Y в целевых переменных. Обратите внимание, что целевые переменные могут быть помечены как временные (локальные) переменные, которые будут созданы для данного действия и могут быть использованы во всех последующих действиях для события, в котором выполняется действие. Цель Y Целевая переменная для хранения позиции y. Пример: Приведенный выше код блока действий извлекает текущую позицию элемента последовательности ID, хранящийся в переменной "my_seq". Затем эти значения используются для установки позиции последовательности, перемещая ее на 10 пикселей вправо.
