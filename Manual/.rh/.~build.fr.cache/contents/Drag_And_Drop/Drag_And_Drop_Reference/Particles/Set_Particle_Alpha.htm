Set Particle Alpha Это действие устанавливает альфа-фактор на момент создания частицы, а также альфа-фактор на протяжении ее жизни. Сначала вы указываете значение ID типа частицы (оно должно быть таким же, как значение, возвращенное при создании частицы с помощью действия Create Particle Type ), а затем вы можете установить значения альфа, которые будут использоваться. Вы можете выбрать три значения альфа-спектра, и частица будет смешиваться между ними в течение времени жизни частицы. Например, если частица имеет значения альфа 0.5, 1 и 0 и время жизни 60 игровых кадров, то когда она создается (кадр 0) она будет иметь альфу 0,5, затем она будет исчезать до полной альфы 1 (кадр 30), а затем станет невидимой с альфой 0 (кадр 60). Синтаксис действия: Аргументы: Аргумент Описание Тип Уникальное значение ID для типа частиц, для которых нужно установить цвет Начало Начальная альфа для использования Средний В промежутке между альфа-диапазоном использовать Конец Окончательная альфа-версия для использования Пример: Приведенный выше код блока действий создаст новый тип частиц и присвоит его уникальное ID значение глобальной переменной.
