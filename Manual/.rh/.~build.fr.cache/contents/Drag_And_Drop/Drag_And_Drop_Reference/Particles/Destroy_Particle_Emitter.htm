Уничтожить излучатель частиц Это действие "уничтожит" данный эмиттер частиц, то есть освободит память, используемую эмиттером частиц. Вы указываете ID системы частиц, к которой принадлежит эмиттер (возвращается действием Create Particle System) и затем ID фактического эмиттера (возвращаемый действием Create Particle Emitter). Это действие следует вызывать всякий раз, когда вам больше не нужен эмиттер частиц в вашей игре, или когда вы хотите заново создать излучатель частиц (например, непосредственно перед вызовом Game Restart). Обратите внимание, что если система частиц, которой был назначен эмиттер, не была отмечена как Persistent, то вызывать это действие не нужно, так как эмиттер будет автоматически очищен на стороне комнаты вместе с системой частиц. Синтаксис действия: Аргументы: Аргумент Описание Система Значение идентификатора системы, содержащей излучатель Излучатель Значение идентификатора излучателя, который нужно уничтожить Пример: Приведенный выше код блока действий проверит переменную экземпляра и если она больше 100, то уничтожит эмиттер, назначенный данному экземпляру, а затем перейдет в следующую комнату.
