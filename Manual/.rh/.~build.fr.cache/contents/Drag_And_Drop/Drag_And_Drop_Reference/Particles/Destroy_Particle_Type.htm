Уничтожить тип частиц Это действие "уничтожит" данный тип частиц, созданный действием Create Particle Type (т.е. освободит память, используемую типом частиц). Это действие следует вызывать всякий раз, когда вам больше не нужен тип частиц в вашей игре, или когда вы хотите заново создать тип частиц (например, перед вызовом перезапуска игры).
