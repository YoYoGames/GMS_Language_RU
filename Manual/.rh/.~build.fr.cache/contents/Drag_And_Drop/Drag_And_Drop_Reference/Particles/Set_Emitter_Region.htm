Установить регион излучателя Это действие используется для установки области и формы эмиттера в системе частиц. Вы указываете ID системы частиц, к которой принадлежит эмиттер (возвращается действием Создать систему частиц), а затем ID фактического эмиттера (возвращается действием Create Particle Emitter), а затем задаете форму эмиттера в пределах заданных значений граничной области. Возможными формами эмиттера являются: Для определения фигур задаются координаты "ограничивающей рамки", в которую будет помещена данная (в случае прямоугольника, эллипса и ромба) или первую и вторую координаты для начальной и конечной точек (в случае линии). Все частицы, которые вырываются или вытекают из эмиттера, будут созданы в пределах в границах фигуры в пределах этой ограничительной области. Вы также можете задать распределение частиц с помощью опции: Линейный: Частицы будут созданы в произвольном месте в пределах заданной формы. Гаусс: Частицы будут создаваться преимущественно в центре фигуры и меньше по краям. Инверсный гаусс: Частицы будут создаваться преимущественно по краям фигуры, а в центре их будет меньше. Синтаксис действия: Аргументы: Аргумент Описание Система Уникальный идентификатор системы, к которой принадлежит излучатель Излучатель Уникальный идентификатор излучателя для установки формы Форма Форма частицы, которая будет использоваться Распространение Распределение частиц в пределах заданной формы Слева Положение левой стороны ограничительной рамки фигуры (в пикселях) Топ Положение вершины ограничивающей рамки фигуры (в пикселях) Справа Положение правой стороны ограничительной рамки фигуры (в пикселях) Дно Положение нижней части ограничительной рамки фигуры (в пикселях) Пример: Приведенный выше код блока действий создаст излучатель частиц, установит его регион, а затем настроит его на испускание потока частиц.
