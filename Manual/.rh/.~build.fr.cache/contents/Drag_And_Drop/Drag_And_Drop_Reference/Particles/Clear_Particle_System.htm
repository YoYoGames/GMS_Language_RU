Clear Particle System С помощью этого действия вы можете очистить данную систему частиц до состояния по умолчанию, удалив все эмиттеры (также нет необходимости вызывать действие Destroy Particle Emitter, поскольку это происходит автоматически) и убрать все частицы с экрана. Вы предоставляете уникальное значение ID системы для очистки, полученное при создании системы действием Create Particle System. Необходимо соблюдать осторожность при использовании этого действия, так как оно уничтожает все эмиттеры, связанные с системой частиц - это означает, что любой экземпляр в комнате, использующий эмиттер, может выдать ошибку, так как эмиттер больше не существует, поэтому убедитесь, что это действие вызывается только тогда, когда вы не собираетесь создавать новые частицы из этого эмиттера (более подробную информацию об эмиттерах частиц см. в разделе Создать эмиттер частиц). ПРИМЕЧАНИЕ: Это действие очищает только видимые частицы в комнате, но не очищает свойства частиц и не удаляет их из памяти. Для этого следует использовать действие Уничтожить тип частицы.
