Установить гравитацию частиц Это действие моделирует действие силы тяжести на все частицы заданного типа. Сначала вы указываете значение ID типа частицы (оно должно совпадать со значением, использованным при создании частицы с помощью действия Create Particle Type ), а затем можете задать силу гравитации и направление. Сила является кумулятивным значением и будет добавляться к вектору скорости каждый игровой кадр, поэтому сила гравитации 1 будет добавлять единицу к значению скорости каждый игровой кадр, в то время как значение направления это просто направление, в котором гравитация должна "тянуть" частицу. Обратите внимание, что и для силы , и для направления можно использовать отрицательные значения, а направление в GameMaker Studio 2 рассчитывается так: 0° вправо, 90° вверх, 180° влево и 270° вниз.
