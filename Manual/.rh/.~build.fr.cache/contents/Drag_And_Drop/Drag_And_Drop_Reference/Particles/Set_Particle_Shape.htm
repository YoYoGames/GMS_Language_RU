Установить форму частиц Это действие можно использовать для установки встроенной формы спрайта, которую частица должна использовать при рисовании. Вы предоставляете уникальное значение ID типа, для которого нужно установить форму (полученное при создании типа с помощью действия Create Particle Type) и форму, которую нужно использовать. Формы частиц - это просто белые спрайты, которые были созданы для GameMaker Studio 2 специально для использования в качестве частиц и состоят из следующих форм: ПРИМЕЧАНИЕ: Если вы используете это действие, вы не должны использовать действие Set Particle Sprite, так как вы можете иметь только спрайт или форму, но не оба. Синтаксис действия: Аргументы: Аргумент Описание Тип Значение идентификатора типа для установки формы Форма Форма, используемая для типа частиц Пример: Приведенный выше код блока действий создаст новый тип частиц и присвоит его уникальное ID значение глобальной переменной.
