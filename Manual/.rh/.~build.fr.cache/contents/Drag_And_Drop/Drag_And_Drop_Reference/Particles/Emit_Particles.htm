Излучают частицы Это действие используется для создания частиц из эмиттера. Вы указываете ID системы частиц, к которой принадлежит эмиттер (возвращается действием Create Particle System), затем фактический ID эмиттера (возвращается действием Create Particle Emitter) и, наконец, тип частицы для создания (возвращается действием Create Particle Type). действием Create Particle Emitter) и, наконец, тип создаваемой частицы (возвращаемый действием Create Particle Type). Затем вы можете выбрать "Burst" или "Поток" частиц из эмиттера, а также указать их количество, которое вы хотите создать. При выборе потоковой передачи частиц вам не нужно помещать это действие в событие игрового кадра, такое как событие Step или событие Draw, а просто вызвать его один раз в событии Create (например), и эмиттер будет передавать частицы каждый игровой кадр, пока вы не вызовете это действие снова, остановив его на 0. пока вы не вызовете это действие снова, чтобы остановить его (установив значение count в 0). Если вы решите разрывать частицы, то вы можете вызвать это действие в любом выбранном вами событии, и частицы будут созданы один раз.
