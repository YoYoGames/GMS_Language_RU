Обновление системы частиц Это действие заставит данную систему частиц обновить позицию, цвет, альфу и т.д.. Вы предоставляете уникальное значение ID системы для обновления (полученное при создании системы с помощью действия Create Particle System), и каждый раз, когда вы вызываете это действие, данная система будет обновляться на один игровой кадр, что означает, что вы можете, например, вызвать это действие в цикле Repeat и заставить систему частиц обновляться несколько раз за в одном игровом кадре (полезно, когда вы хотите, чтобы уровень имел какой-либо эффект, требующий присутствия частиц в самом начале, например, снег или дождь). Вы также можете использовать это действие, когда система частиц была приостановлена с помощью Pause Particle System, чтобы вручную обновить частицы в системе. Синтаксис действия: Аргументы: Аргумент Описание Система Уникальное значение ID для обновляемой системы Пример: Приведенный выше код блока действий создаст цикл, который затем обновит систему частиц 200 раз, прежде чем продолжить.
