Система частиц "Пауза Это действие приостанавливает работу заданной системы частиц. Вы указываете уникальное значение ID системы, которую нужно приостановить, полученное при создании системы действием Create Particle System. Приостановленная система частиц будет продолжать рисовать порожденные частицы, но они больше не будут обновлять свой цвет, положение или альфа-спектр, а также завершат свой жизненный цикл и исчезнут, пока вы не возобновите работу системы. Вы можете выбрать приостановку или возобновление работы системы с помощью опции "Состояние". Обратите внимание, что это не приостанавливает работу эмиттеров, создающих потоки частиц, и не останавливает эмиттер, выбрасывающий частицы - эмиттеры по-прежнему будут создавать частицы при вызове, но когда система приостановлена, частицы ничего не будут делать и будут отрисовываться точно пока вы не сбросите состояние паузы или не вызовете Update Particle System. Синтаксис действия: Аргументы: Аргумент Описание Система Уникальное значение ID для системы, которую необходимо приостановить Государство Приостановить или возобновить работу системы частиц Пример: Приведенный выше код блока действий проверит глобальную переменную, и если она вернет значение true, то данная система частиц будет приостановлена, в противном случае она будет возобновлена.
