Если кнопка геймпада отпущена Это действие проверяет, когда кнопка геймпада была отпущена для заданного индекса геймпада. Оно возвращает true при единственном игровом тике, когда кнопка была отпущена, и false во всех остальных случаях. Если вы включите модификатор not, это действие будет обратным и будет проверять, не была ли отпущена кнопка, возвращая false на игровом тике, на котором она была отпущена, и true во всех остальных случаях. Если проверяемая кнопка является аналоговой, то отпускание кнопки не будет обнаружено, пока значение кнопки не станет ниже порога, установленного с помощью действия Set Gamepad Button Threshold. Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Теперь эти действия будут выполняться, если значение "if" будет равно true, в то время как любые действия, отброшенные в другом месте, будут выполняться после блока "if".
