Подключите геймпад Вы можете использовать это действие, чтобы проверить, подключен ли к данному "слоту" геймпада геймпад. Вы указываете индекс геймпада (это номер "слота", который занимает геймпад), а возвращаемое значение будет сохранено в целевой переменной которую вы предоставите для дальнейшего использования (она может быть помечена как временная локальная переменная, которую вы создаете для этого события или сценария). Синтаксис действия: Аргументы: Аргумент Описание Геймпад Индекс геймпада для проверки. Цель Целевая переменная для хранения возвращаемого значения. Пример: Приведенный выше код блока действий запускает цикл for, чтобы проверить, подключен ли геймпад подключен к любому из 12 доступных "слотов". Если геймпад обнаружен, глобальная переменная устанавливается на номер текущего слота, и цикл прерывается.
