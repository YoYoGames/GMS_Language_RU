Комплект геймпада Axis Deadzone Это действие можно использовать для установки "мертвой зоны" оси стиков геймпада. Вы указываете индекс геймпада для установки, а затем задаете значение от 0 до 1, и если входная величина меньше заданного значения, ось стиков геймпада считается Обратите внимание, что это глобальная настройка, которая будет влиять на все оси подключенных стиков. Обратите внимание, что это не изменяет вывод действия Get Gamepad Axis, которое всегда будет возвращать значение от -1 и 0, и 0 и 1. Например, установка мертвой зоны на 0,2 означает, что нажатие стика вправо будет начнет возвращать значение от 0 до 1, только когда необработанное значение оси превысит 0,2 - таким образом, когда необработанное значение равно 0,2, возвращаемое значение будет равно 0, когда необработанное значение равно 0,5, возвращаемое значение будет равно 0,375, или если необработанное значение равно 0,9, возвращаемое значение будет равно 0,875. значение будет 0,875.
