Получить количество геймпадов Вы можете использовать это действие, чтобы получить количество геймпадов, подключенных к устройству, на котором запущена ваша игра. Возвращаемое число будет от 0 до 12. Синтаксис действия: Аргументы: Аргумент Описание Цель Целевая переменная для хранения возвращаемого значения. Пример: Приведенный выше код блока действий получает количество подключенных геймпадов и присваивает это значение временной локальной переменной. Затем проверяется, больше ли это значение 0, и соответствующим образом устанавливается глобальная переменная.
