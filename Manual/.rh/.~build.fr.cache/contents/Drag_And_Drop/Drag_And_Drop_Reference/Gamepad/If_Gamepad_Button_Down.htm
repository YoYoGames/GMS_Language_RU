При нажатии кнопки геймпада вниз С помощью этого действия вы можете опросить геймпад, чтобы узнать, удерживается ли в данный момент заданная кнопка. Действие будет возвращать true на каждом шаге, когда кнопка удерживается нажатой, и false в противном случае, хотя если вы нажмете на модификатор not вы можете проверить, не удерживается ли кнопка, т.е. действие будет возвращать true, если кнопка не удерживается, и false, если удерживается. Если вам нужно проверить нажатие только одной кнопки, используйте действие If Gamepad Button Pressed. Если проверяемая кнопка является аналоговой, то она не будет считаться удерживаемой, пока значение кнопки не превысит порог, установленный с помощью действия Set Gamepad Button Threshold. Обратите внимание, что для добавления действий в блок "if", их следует опустить сбоку от действия, как показано на изображении ниже: Эти действия теперь будут выполняться, если "if" оценивается в true, в то время как любые действия, отброшенные в другом месте, будут выполняться после блока "if". Не Отрицание проверки (истина становится ложью и наоборот) Пример: Приведенный выше код блока действий каждый шаг опрашивает состояние кнопки геймпада 0 и, если кнопка удерживается нажатой, он проверяет начальное нажатие кнопки вниз. Если проверка истинна при первоначальном нажатии, воспроизводится звук, затем, пока кнопка удерживается, скорость экземпляра устанавливается на 10. Если кнопка не удерживается вниз, проверка выполняется при отпускании кнопки, чтобы снова сбросить скорость до 0.
