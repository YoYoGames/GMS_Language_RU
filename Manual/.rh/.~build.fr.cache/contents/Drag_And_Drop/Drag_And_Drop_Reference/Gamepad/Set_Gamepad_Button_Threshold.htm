Установка порога нажатия кнопки геймпада Это действие можно использовать для установки "порога" аналоговых кнопок геймпада. Вы указываете индекс геймпада для установки, а затем задаете значение от 0 до 1, и если входное значение аналоговой кнопки меньше заданного значения, кнопка геймпада будет считаться равной 0. Обратите внимание, что это глобальная настройка, которая будет влиять на все аналоговые кнопки, подключенные к геймпаду. Это значение будет использоваться во всех проверках нажатия, отжатия и отпускания для данного геймпада, но будет игнорироваться действием Get Gamepad Trigger.
