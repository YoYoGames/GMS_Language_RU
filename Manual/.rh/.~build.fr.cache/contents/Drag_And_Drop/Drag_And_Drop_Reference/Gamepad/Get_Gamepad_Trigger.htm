Получить триггер геймпада Вы можете использовать это действие для получения значений различных триггеров с данного геймпада. Вы указываете индекс геймпада (это номер "слота", который занимает геймпад) и выбираете триггер для проверки (левый или правый). Возвращаемое значение будет от 0 до 1, где 0 - отсутствие давления на триггер, а 1 - полное давление. Это возвращаемое значение будет сохранено в переменной target, которую вы зададите для дальнейшего использования. Обратите внимание, что эта функция не учитывает настройку порога срабатывания. Синтаксис действия: Аргументы: Аргумент Описание Геймпад Индекс геймпада. Триггер Курок для проверки (левый/правый) Цель Целевая переменная для хранения возвращаемого значения. Пример: Приведенный выше код блока действий создает локальную (временную) переменную, а затем использует ее для хранения текущего значения для левого триггера.
