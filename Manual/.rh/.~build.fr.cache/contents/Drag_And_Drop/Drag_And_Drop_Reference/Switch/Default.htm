По умолчанию Это действие создаст новый случай по умолчанию в пределах Switch. Случай по умолчанию - это случай, который будет выполняться, если входное значение не соответствует никаким другим константам случая. Действия, которые добавляются к случаю по умолчанию, должны быть добавлены сбоку от действия, подобно тому, как вы добавляете действие в "если": Синтаксис действия: Пример: Приведенный выше код блока действий получает значение, хранящееся в глобальной переменной, а затем сравнивает его с различными возможными случаями. Если хоть один из случаев совпадает со значением, то будет выполнено действие, предусмотренное этим случаем, в противном случае будет оценен следующий случай. Если ни один из случаев не соответствует введенному значению, то вместо него будет выполнен случай по умолчанию.
