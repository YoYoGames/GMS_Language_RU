Дело Это действие создаст новый случай внутри Переключить. Вы указываете значение для случая, и оно сравнивается с входным значением для переключателя, и если они совпадают, то выполняются действия, добавленные в этот случай. Действия, которые добавляются в Случай, чтобы выполняться только в этом Случае, должны быть отброшены в сторону действия, подобно тому, как вы добавляете действие в "если": Обратите внимание, что значение Case должно быть константой, которая может быть любым типом данных (вещественным, строковым и т.д...), при условии, что оно оценивается в одно значение при компиляции - поэтому вы не можете использовать выражения, зависящие от переменных. Также обратите внимание, что Case нельзя использовать вне Switch, и попытка сделать это приведет к ошибке. Синтаксис действия: Аргументы: Аргумент Описание Постоянная Постоянное значение, которое должно быть оценено по отношению к входному значению переключателя Пример: Приведенный выше код блока действий получает значение, хранящееся в переменной, а затем сравнивает его с различными возможными случаями. Если хоть один из случаев совпадает со значением, то выполняется действие, соответствующее этому случаю, в противном случае оценивается следующий случай.
