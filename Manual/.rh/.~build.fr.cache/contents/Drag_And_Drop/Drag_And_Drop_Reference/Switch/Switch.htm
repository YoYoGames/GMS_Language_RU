Переключатель Это действие создаст новый оператор Switch, в который затем можно добавить другие случаи для оценки. Вы задаете значение, которое будет использоваться для сравнения со случаями в операторе Switch, а затем добавляете различные действия Case, подобно тому, как вы добавляете действие в операторе "if". как если бы вы добавили действие к "if", т.е. опустив его сбоку от действия, а не под ним: Обратите внимание, что вы не можете добавить любой другой тип действия в переключатель, не добавив предварительно хотя бы один случай. Синтаксис действия: Аргументы: Аргумент Описание Значение Значение, которое будет использоваться переключателем для оценки Пример: Приведенный выше код блока действий получает значение, хранящееся в переменной, а затем сравнивает его с различными возможными случаями. Если хоть один из случаев совпадает со значением, то выполняется действие, соответствующее этому случаю, в противном случае оценивается следующий случай.
