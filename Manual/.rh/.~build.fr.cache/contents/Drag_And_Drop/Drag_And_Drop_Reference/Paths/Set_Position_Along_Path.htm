Установить положение вдоль траектории Это действие устанавливает положение экземпляра вдоль пути. Значение нормализуется от 0 до 1, поэтому если вы установите его, например, на 0,5, экземпляр будет перемещен точно в середину пути. Обратите внимание, что для того, чтобы действие имело эффект, необходимо вызвать действие Start Following Path для назначения пути экземпляру. Синтаксис действия: Аргументы: Аргумент Описание Позиция Позиция (от 0 до 1) вдоль пути для размещения экземпляра Пример: Приведенный выше код блока действий запускает экземпляр по пути, а затем генерирует случайное десятичное значение между 0 и 1. Это значение хранится во временной локальной переменной и используется для установки положения экземпляра на пути.
