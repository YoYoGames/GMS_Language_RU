Установить состояние временной шкалы Это действие можно использовать для запуска, остановки, приостановки или возобновления временной шкалы. Временная шкала должна быть сначала установлена для экземпляра с помощью действия Set Instance Timeline, а затем вы можете использовать это действие в любое время, чтобы изменить ее состояние из следующих вариантов Вариант Описание Начало Запустите временную шкалу, назначенную данному экземпляру (если временная шкала была приостановлена, это снова установит начальную позицию в момент 0). Резюме Возобновить приостановленную временную шкалу (если временная шкала была остановлена и не приостановлена, это ничего не даст). Приостановлено Приостановить временную шкалу, назначенную данному экземпляру (во время приостановки моменты временной шкалы не будут продвигаться вместе с кадрами игры). Остановить Остановите временную шкалу, назначенную данному экземпляру.
