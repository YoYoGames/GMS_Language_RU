Установите скорость временной шкалы Это действие можно использовать для установки скорости, с которой временная шкала будет перемещаться по содержащимся в ней моментам. Обычно в каждом игровом кадре вы хотите, чтобы позиция на временной шкале увеличивалась на 1, однако вы можете изменить это значение, установив это действие на другое значение. Выбранное значение будет рассматриваться как множитель, поэтому, например, значение 2 заставит временную шкалу работать с удвоенной скоростью, а значение 0,5 заставит ее работать с вдвое меньшей скоростью. Для моментов это означает, что если значение больше единицы, то в одном игровом кадре может произойти несколько моментов (моменты никогда не пропускаются), а если скорость меньше единицы, то после того, как момент пробежал, будут кадры, в которых ничего не происходит до следующего момента. где ничего не происходит, пока не запустится следующий момент - если вы установите скорость временной шкалы на 0.1, например, вы получите момент на одном игровом кадре, затем 9 игровых кадров без моментов, а на десятом игровом кадре будет запущен следующий момент. Если вы установите временную шкалу для запуска с помощью действий Set Instance Timeline или Set Timeline State, но не установите скорость, то временная шкала по умолчанию будет иметь значение 1 момент на игровой кадр. Синтаксис действия: Аргументы: Аргумент Описание Скорость Скорость - в виде множителя - для временной шкалы (по умолчанию 1). Пример: Приведенный выше код блока действий назначит временную шкалу экземпляру, вызывающему действия, а также установит начальный момент на 10, а затем скорость временной шкалы на 2.
