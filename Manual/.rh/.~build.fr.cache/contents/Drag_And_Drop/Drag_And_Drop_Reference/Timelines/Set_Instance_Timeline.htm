Временная шкала экземпляра набора Это действие назначает экземпляру временную шкалу из браузера ассетов. Вы указываете временную шкалу для назначения, а затем можете выбрать зацикливание временной шкалы при достижении последнего момента (следующий кадр после запуска последнего момента будет зациклен на к первому моменту временной шкалы), а также запустить временную шкалу немедленно или нет. Если вы не запускаете временную шкалу сразу, вы можете использовать действие Set Timeline State, чтобы запустить ее позже в другом событии или сценарии. Синтаксис действия: Аргументы: Аргумент Описание Временная шкала Ресурс временной шкалы для назначения экземпляру Петля Флаг, чтобы временная шкала зацикливалась после последнего момента (по умолчанию установлен флаг) Начальная шкала времени Флаг, чтобы временная шкала начиналась немедленно или нет (по умолчанию флаг установлен). Пример: Приведенный выше код блока действий назначит временную шкалу экземпляру, вызывающему действия, а также установит начальный момент на 10, а затем скорость временной шкалы на 2.
