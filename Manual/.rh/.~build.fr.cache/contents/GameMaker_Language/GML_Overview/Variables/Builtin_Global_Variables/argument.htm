аргумент Эта переменная содержит массив, который используется вместе с переменной argument_count, доступной только для чтения, в функциях или методах скрипта. Каждая позиция массива содержит входное значение для функции и предназначена специально для использования в функциях с переменным аргументом. Обратите внимание, что существует также ряд независимых переменных глобальной области видимости, которые также могут быть использованы в пользовательских функциях для ссылки на различные входные аргументы: argument0, argument1, argument2, и т.д. вплоть до .argument15 ПРИМЕЧАНИЕ: Это больше не является обязательным требованием и предоставляется больше для поддержки наследия, чем для чего-либо еще. Все определяемые пользователем функции будут принимать переменные аргументы, и любой аргумент, переданный в функцию, который не является частью именованных входа, по умолчанию будет инициализирован как undefined, что можно проверить с помощью функции is_undefined(). Синтаксис: аргумент[n] аргумент1 аргумент2 ... аргумент15 Возвращается: Значение (может быть любого типа данных, передаваемых в функцию) Пример: function my_array() { var i, arg; for (i = 0; i < 5; i += 1;) { if argument_count > i { arg[i] = argument[i] } else { arg[i] = -1; } } } Приведенный выше код использует переменную argument_count вместе с массивом argument для инициализации переменных и является примером того, как разрешить пользовательской функции принимать переменное количество аргументов.
