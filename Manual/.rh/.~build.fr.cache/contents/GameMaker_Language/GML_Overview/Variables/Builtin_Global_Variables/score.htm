счет Эта переменная является глобальной и используется для хранения числового значения, которое обычно используется для оценки игрока. Эта переменная предназначена только для поддержки старых проектов от предыдущих версий GameMaker и должна не использоваться в новых проектах поскольку в будущем она может быть устаревшей. Синтаксис: оценка; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример switch (object_index) { case obj_Enemy_Fighter: score += 10; break; case obj_Enemy_Mage: score += 25; break; case obj_Enemy_Boss: score += 100; break; } instance_destroy(); Приведенный выше код проверяет индекс объекта экземпляра, выполняющего код, с помощью переключателя, а затем добавляет различные суммы в переменную score в зависимости от того, какой это объект.
