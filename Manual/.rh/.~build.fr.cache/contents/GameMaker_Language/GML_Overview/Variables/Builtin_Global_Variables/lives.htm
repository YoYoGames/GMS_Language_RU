жизни Эта переменная является глобальной и используется для хранения числового значения, которое обычно используется для жизней игроков. Эта переменная предназначена только для поддержки старых проектов предыдущих версий программы GameMaker и должна не использовать в новых проектах поскольку в будущем она может быть устаревшей. Синтаксис: жизни; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример if (lives < 5) { lives += 1; } Приведенный выше код проверяет переменную lives, и если она меньше 5, к ней прибавляется 1.
