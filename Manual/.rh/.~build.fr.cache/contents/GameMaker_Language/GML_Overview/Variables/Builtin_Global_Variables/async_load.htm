async_load Эта переменная является глобальной и используется для хранения карты DS при использовании в асинхронных событиях, и -1 во всех остальных случаях. Фактическое содержимое карты DS будет зависеть от типа обратного вызова асинхронного события, а также от функции, которая была использована для запуска события, поэтому обратитесь к на отдельные страницы этих событий для получения подробной информации о возможном содержимом карты DS. Синтаксис: async_load; Возвращается: Real (целочисленное значение) Расширенный пример: sprite = sprite_add("site.com/path/image.png", 16, true, true, 0, 0); Приведенный выше код будет вызван в событии для загрузки спрайта из внешнего URL. Затем вызывается асинхронное событие Image Loaded Asynchronous Event, в котором разбирается карта async_load: если ds_map_find_value(async_load, "id") == sprite { if (ds_map_find_value(async_load, "status") >= 0) { sprite_index = sprite; } } Приведенный выше код сначала проверит ID карты async_load, затем проверит статус обратного вызова. Если значение больше или равно 0 (сигнализирует об успехе), результат обратного вызова будет использован для установки индекса спрайта экземпляра экземпляра на только что загруженное изображение.
