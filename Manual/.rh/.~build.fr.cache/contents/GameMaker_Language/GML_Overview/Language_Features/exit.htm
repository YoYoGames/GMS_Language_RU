выход Оператор exit имеет следующий синтаксис: выход; exit просто завершает выполнение текущей функции, метода или события сценария. Обратите внимание, что здесь есть небольшая разница в использовании в зависимости от области применения: Если вы используете exit в функции или методе сценария, он просто выйдет из функции и вернется к коду, который вызвал функцию. Если вы используете exit в блоке кода внутри события объекта, он завершит все событие, даже если есть различные отдельные блоки кода после вызова exit. Если вы используете exit в родительском событии и вызываете это событие с помощью event_inherited() в дочернем объекте, только родительское событие завершится, а дочернее событие (которое вызвало event_inherited()) будет продолжаться. При использовании в событии exit обычно используется для того, чтобы избежать выполнения экземпляром дальнейшего кода, когда определенное условие было выполнено (или нет). Приведенный ниже код описывает пример его использования, в данном случае в событии столкновения, хотя его можно использовать в любом событии. if (!visible) { exit; } other.hp -= attack; other.coins -= 4; coins += 4; Приведенный выше код проверяет, не виден ли текущий экземпляр, в этом случае он выходит из блока кода, в противном случае он продолжает выполнение остальной части кода.
