Например, вызывается следующее: throw ("Hello World!"); приведет к появлению следующей ошибки не обработанного исключения: Вы можете, однако, взять на себя это сообщение об ошибке и использовать свой собственный код обработчика, вызвав функцию функцию exception_unhandled_handler(). Эта функциявремени выполнения позволяет вам предоставить пользовательский метод, который будет вызываться всякий раз, когда в вашей игре возникают необработанные исключения.
