array_get С помощью этой функции вы можете получить значение по индексу в массиве. Функция требует, чтобы вы указали переменную, содержащую массив, и индекс для получения значения из этого массива. Обратите внимание, что если указанный индекс массива выходит за пределы границ то игра завершится с ошибкой. Эта функция также может быть использована для многомерных массивов, если вы укажете, какое измерение вы хотите получить при указании индекса массива, по следующей схеме: // 1D массив array_get(my_array, 0); // 2D массив array_get(my_array[0], 0); // 3D массив array_get(my_array[0][0], 0); // и т.д.... Синтаксис: array_get(variable, index); Аргумент Описание переменная Переменная, в которой хранится массив. индекс Индекс массива, из которого нужно получить значение. Возвращается: Любой допустимый тип данных, который может содержать массив. Пример: for (var i = 0; i < 10; ++i;) { show_debug_message(array_get(my_array, i)); } Приведенный выше код выведет на консоль первые 10 элементов из заданных.
