array_pop С помощью этой функции вы можете получить значение из конца массива, не зная его длины. Вы передаете переменную, в которой хранится массив, и функция возвращает содержимое последней записи в массиве. Если массив пуст, функция вернет undefined. Синтаксис: array_pop(variable); Аргумент Описание переменная Переменная, в которой хранится массив. Возвращается: Значение (любой тип данных) или не определено Пример: var _lastscore = array_pop(score_array); draw_text(32, 32, "Last Score = " + string(_lastscore)); Приведенный выше код выберет последнее значение из заданного массива и выведет его вместе с текстом на экран.
