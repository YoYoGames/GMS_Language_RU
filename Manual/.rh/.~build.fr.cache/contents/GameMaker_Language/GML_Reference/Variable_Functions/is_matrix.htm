матрица Эта функция возвращает, относится ли данная переменная к матрице или нет. В некоторых случаях вы хотите проверить и узнать, какой тип данных имеет переменная в GameMaker Studio 2, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того, является ли значение матрицей или нет. Синтаксис: is_matrix(n); Аргумент Описание n Аргумент для проверки. Возвращается: Булево Пример: if !is_matrix(val) { show_debug_message("Not a valid matrix!"); } Приведенный выше код проверяет переменную "val" на наличие в ней матрицы, и если это не так, то выводит сообщение в консоль отладки.
