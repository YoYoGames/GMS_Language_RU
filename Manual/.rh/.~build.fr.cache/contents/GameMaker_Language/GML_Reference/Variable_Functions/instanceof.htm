Вы передаете переменную со ссылкой на struct для проверки, и функция возвращает либо строку с именем функции, либо undefined. Обратите внимание, что если вы передадите функции литерал struct (т.е. struct, созданный без использования функции-конструктора), то она просто вернет строку "struct".
