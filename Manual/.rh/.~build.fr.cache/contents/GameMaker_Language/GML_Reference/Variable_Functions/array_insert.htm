array_insert С помощью этой функции вы можете вставить значение (или значения) в массив в любую заданную позицию. Функция требует, чтобы вы предоставили переменную, содержащую массив, индекс (позицию) в массиве для вставки, а также хотя бы одно значение для вставки. хотя по желанию вы можете указать дополнительные аргументы, и все они будут вставлены в массив в последовательном порядке, начиная с заданного индекса. Обратите внимание, что если заданный индекс находится за пределами длины массива, значения будут добавлены по заданному индексу, а все пустые места в массиве между его последним значением и вставленными значениями будут установлены в значение по умолчанию 0. Синтаксис: array_insert(variable, index, value, [value], [value], [etc...]); Аргумент Описание переменная Переменная, в которой хранится массив. индекс Индекс (позиция) в массиве для вставки значения (значений) значение Значение для вставки [значение], [значение], [и т.д.]. (Необязательно) Дополнительные значения для вставки в массив Возвращается: Н/Д Пример: array_insert(score_array, score_rank, current_score); Приведенный выше код вставит текущий результат в массив score по индексу, определенному переменной score_rank.
