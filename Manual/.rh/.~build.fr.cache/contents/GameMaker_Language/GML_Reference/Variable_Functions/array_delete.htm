массив_удалить С помощью этой функции вы можете удалить значение (или значения) из массива в любой заданной позиции. Функция требует, чтобы вы предоставили переменную, содержащую массив, индекс (позицию) в массиве для удаления, а также количество значений для удаления. Синтаксис: array_delete(variable, index, number); Аргумент Описание переменная Переменная, в которой хранится массив. индекс Индекс (позиция) в массиве для удаления значения (значений). номер Количество значений для удаления. Возвращается: Н/Д Пример: array_delete(score_array, 0, 10); Приведенный выше код удалит первые 10 значений в заданном массиве, начиная с индекса 0 (таким образом, из массива будут удалены индексы с 0 по 9).
