int64 Эта функция пытается преобразовать заданное значение в 64-битное целое число, где значение должно быть либо вещественным, либо строкой, либо int64, либо int32, либо ptr. Любое приведет к аварийному завершению игры с сообщением об ошибке. Проверить, хранит ли переменная значение int64, можно с помощью функции is_int64(). Синтаксис: int64(val); Аргумент Описание val Значение для преобразования. Возвращается: 64-разрядное целое число Пример: steam_handle= int64(global.fileReadString); Приведенный выше код преобразует значение, хранящееся в глобальной переменной, в 64-битное целое число.
