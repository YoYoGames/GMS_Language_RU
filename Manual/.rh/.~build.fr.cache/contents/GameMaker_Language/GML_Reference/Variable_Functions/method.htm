метод С помощью этой функции вы можете привязать любую функцию, которая была определена ранее, к данному экземпляру или структуре, создавая переменную метода, которая может быть использована позже. использовать позже. Вы указываете ID экземпляра (это должен быть экземпляр, который активен и находится в комнате, и не может быть ID объекта) или ссылку на структуру, а также ID функции, которую вы хотите связать. Функция вернет метод, который может быть вызван из переменной, которой она присвоена (см. пример кода ниже). Стоит отметить, что вы можете привязывать как встроенные функции, так и пользовательские определенные пользователем, а также можно указать undefined в качестве аргумента экземпляра/структуры, что означает, что текущая self область видимости будет использоваться для привязки. Синтаксис: method(struct_ref_or_instance_id, function); Аргумент Описание struct_ref_or_instance_id Уникальная ссылка или идентификационное значение структуры или экземпляра для использования (или не определено для себя) функция Идентификатор функции, которую нужно использовать Возвращается: Метод Пример: var _inst = instance_position(mouse_x, mouse_y, obj_Enemy); if instance_exists(_inst) { enemy_func = method(_inst, enemy_ai); } Приведенный выше код проверит, существует ли экземпляр в заданной позиции, и если да, то функция привязывается к экземпляру и возвращается как переменная метода "enemy_func".
