bool Эта функция попытается преобразовать заданное значение в булевский тип данных, где значение будет возвращено как true, если оно больше 0,5, и false в противном случае. Синтаксис: bool(n); Аргумент Описание n Значение для преобразования. Возвращается: Булево Пример: if !is_bool(val) { val = bool(val); } Приведенный выше код проверяет переменную "val" на то, является ли она булевой, и если нет, то преобразует ее в булевую.
