variable_instance_get_names С помощью этой функции можно получить массив, заполненный именами переменных экземпляра или глобальных переменных игры. Когда вы передаете значение идентификатора экземпляра, каждый каждая запись в массиве будет строкой имени переменной, соответствующей переменной области видимости экземпляра, которая была создана в данном экземпляре. Однако если вы передать ключевое слово global, каждая запись в массиве будет строкой имени переменной, соответствующей переменной глобальной области видимости. Синтаксис: variable_instance_get_names(instance_id/global); Аргумент Описание instance_id/global Уникальное значение ID экземпляра для проверки или ключевое слово global Возвращается: Массив (каждая запись - строка) Пример: var str = ""; var array = variable_instance_get_names(id); show_debug_message("Переменные для " + object_get_name(object_index) + string(id)); for (var i = 0; i < array_length(array); i++;) { str = array[i] + ":" + string(variable_instance_get(id, array[i])); show_debug_message(str); } Приведенный выше код получит массив всех переменных области видимости экземпляра для экземпляра, выполняющего блок кода, а затем отобразит их вместе с их значениями в отладочном выводе.
