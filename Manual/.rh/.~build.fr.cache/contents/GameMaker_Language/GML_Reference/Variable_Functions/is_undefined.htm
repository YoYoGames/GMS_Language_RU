is_undefined Эта функция возвращает, определена ли данная переменная или нет. В некоторых случаях вы хотите проверить, какой тип данных имеет переменная в GameMaker Studio 2, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того. в зависимости от того, определено значение или нет. ПРИМЕЧАНИЕ: Эта функция не может быть использована для проверки существования переменной. Используйте variable_instance_exists() или переменная_глобальная_существует() вместо нее. Синтаксис: is_undefined(n); Аргумент Описание n Аргумент для проверки. Возвращается: Булево Пример: var val = ds_map_find_value(map, 13); if is_undefined(val) { show_debug_message("Map entry does not exist!"); } Приведенный выше код проверяет переменную "val" на то, является ли она неопределенной или нет, и показывает отладочное сообщение, если это так.
