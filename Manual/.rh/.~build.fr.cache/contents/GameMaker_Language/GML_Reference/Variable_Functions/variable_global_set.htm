переменная_глобальная_установка С помощью этой функции вы можете установить значение заданной глобальной переменной. Вы указываете имя глобальной переменной, значение которой нужно установить, в виде строки (см. пример кода ниже), а затем значение, которое нужно установить (это может быть любой допустимый тип данных). Если глобальная переменная еще не существует в игре, она будет создана, а затем ей будет присвоено значение. Синтаксис: variable_global_set(name, val); Аргумент Описание имя Имя глобальной переменной для установки (в виде строки) val Значение, на которое нужно установить глобальную переменную Возвращается: Н/Д Пример: if !variable_global_exists("enemy_num") { variable_global_set("enemy_num", 0); } Приведенный выше код проверит, существует ли глобальная переменная, и если нет, то она будет создана и установлена в 0.
