is_int64 Эта функция возвращает, является ли данная переменная 64-битным целым числом или нет. В некоторых случаях вы хотите проверить и узнать, какой тип данных имеет переменная в GameMaker Studio 2, и тогда вы используете эту функцию. Она возвращает true или false в зависимости от того, является ли значение int64 или нет. Синтаксис: is_int64(n); Аргумент Описание n Аргумент для проверки. Возвращается: Булево Пример: if !is_int64(val) { show_debug_message("Not a 64bit integer!"); } Приведенный выше код проверяет переменную "val" на то, содержит ли она int64, и если нет, то выводит сообщение в консоль отладки.
