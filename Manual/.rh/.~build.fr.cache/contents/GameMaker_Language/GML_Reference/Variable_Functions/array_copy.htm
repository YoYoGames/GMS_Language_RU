array_copy С помощью этой функции вы можете скопировать весь массив или его часть в другой массив в любой позиции. Вам необходимо предоставить исходный и конечный массивы (оба должны быть созданы ранее), а также позицию в исходном массиве для копирования из и позицию в массиве назначения для копирования. Наконец, необходимо указать длину массива (или длину нужной части) для копирования. Если копируемые данные превышают длину массива назначения, массив будет будет расширен, чтобы принять эти данные. Эта функция также может быть использована для многомерных массивов, если при указании индекса массива вы укажете, какое измерение вы хотите скопировать, следуя этой схеме: // Копирование из первого измерения массива // во второе измерение массива array_copy(item_array, 0, inventory_array[0], 0, len); // Копирование из третьего измерения массива // в первое измерение массива array_copy(item_array[0][0], 0, inventory_array, 0, len); // и т.д.... Синтаксис: array_copy(dest, dest_index, src, src_index, length); Аргумент Описание dest Идентификатор массива, в который нужно скопировать. dest_index Индекс в массиве для копирования. src Идентификатор массива, из которого будет производиться копирование. src_index Индекс массива, с которого следует начать копирование. длина Длина (количество индексов массива) для копирования. Возвращается: Н/Д Пример: if !array_equals(inventory_array, item_array) { var len = array_length(inventory_array); array_copy(item_array, 0, inventory_array, 0, len); } Приведенный выше код проверит два массива на наличие в них эквивалентных значений, и если это не так, то код скопирует все содержимое массива "inventory_array" в массив "item_array".
