Переменные функции Функции на этой странице предназначены для работы с различными переменными и типами переменных при использовании языка GameMaker в ваших играх. Более подробную информацию о переменных можно найти на странице Переменные и область действия переменных. Доступны следующие функции переменных: переменная_экземпляр_существует variable_instance_get_names переменная_имена_обстоятельств_счёт переменная_объекта_get переменная_объекта_набора переменная_глобальная_существует переменная_глобальная_гет переменная_глобальная_установка переменная_структура_существует переменная_структура_get variable_struct_set переменная_структура_удаления variable_struct_get_names переменная_структура_имена_счет instanceof Существует несколько функций, связанных с использованием массивов. Они предназначены для обеспечения гибкости кода и позволяют создавать более функциональные и динамичные массивы при создании игр. К таким функциям относятся: массив_создать array_copy массивы_равных array_get array_set массив_толчков array_pop array_insert массив_удалить сортировка массивов длина_массива array_resize array_length_1d(deprecated) array_length_2d(deprecated) array_height_2d(устаревшее) Все эти следующие функции связаны с использованием переменных метода: метод метод_get_self метод_get_index Наконец, у вас также есть следующие функции для определения типа переменной (более подробную информацию о типах данных вы можете найти здесь, а таблицы типов- здесь): is_string is_real is_numeric is_bool массив is_struct is_method is_ptr is_int32 is_int64 is_vec3 is_vec4 матрица is_undefined is_nan бесконечность typeof bool ptr int64 Назад: Ссылка наGML Следующее: Управление активами © Copyright YoYo Games Ltd. 2021 Все права защищены
