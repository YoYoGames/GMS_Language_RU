массив_толчков С помощью этой функции вы можете поместить значение (или значения) в конец массива без необходимости знать длину массива. Функция требует, чтобы вы предоставили переменную, содержащую массив, а также хотя бы одно значение для вставки, хотя по желанию вы можете указать дополнительные аргументы, и все они будут вытолкнуты в массив в последовательном порядке. Синтаксис: array_push(variable, value, [value], [value], [etc...]); Аргумент Описание переменная Переменная, в которой хранится массив. значение Значение для переноса в конец массива [значение], [значение], [и т.д.]. ВАРИАНТ! Дополнительные значения, которые необходимо поместить в массив Возвращается: Н/Д Пример: array_push(score_array, obj_Player1.scr, obj_Player2.scr, obj_Player3.scr, obj_Player4.scr); Приведенный выше код поместит четыре значения в конец заданного массива.
