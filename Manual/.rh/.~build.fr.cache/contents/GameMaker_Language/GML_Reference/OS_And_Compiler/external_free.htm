внешний_свободный Эта функция освобождает память, связанную с dll или dylib с заданным именем. Это следует делать всякий раз, когда данный файл больше не нужен в игре, обычно (например, в событии Game End). Синтаксис: external_free(id); Аргумент Описание id Имя dll или dylib, которую вы хотите освободить Возвращается: Н/Д Пример: external_free("MyDLL.dll"); Приведенный выше код примера освободит память, связанную с заданной dll.
