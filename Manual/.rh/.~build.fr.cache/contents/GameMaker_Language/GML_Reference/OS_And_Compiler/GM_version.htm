GM_version Эта константа хранит номер версии, определенный в Опциях игры для каждой целевой платформы. Значение хранится в виде строки. Синтаксис: GM_version; Возвращается: Строка Пример: draw_text(32, 32, date_time_string(GM_build_date)); draw_text(32, 64, "v" + GM_version); Приведенный выше код принимает константы GM date и version и выводит их на экран.
