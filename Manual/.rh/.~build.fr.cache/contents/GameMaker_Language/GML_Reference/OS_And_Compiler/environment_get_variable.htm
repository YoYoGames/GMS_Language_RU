environment_get_variable Эта функция возвращает значение (строка) переменной окружения с заданным именем (также строка). Вы можете получить доступные переменные окружения на macOS и Ubuntu(Linux), набрав " env" в приложении терминала, а для получения информации о переменных окружения Windows, если вы используете командную строку, введите " echo%PATH%", а в PowerShell - " ls env". Обратите внимание, что как в macOS, так и в Ubuntu(Linux) переменная окружения "HOME" вернет путь " ~/", который отображается на " /Users/" в macOS и " /home/ " в Ubuntu (Linux). ПРИМЕЧАНИЕ: Эта функция предназначена только для Windows, macOS и Ubuntu(Linux). Синтаксис: environment_get_variable(name); Аргумент Описание имя Имя (строка) переменной среды для проверки. Возвращается: Строка Пример: e_str = environment_get_variable("APPDATA"); Приведенный выше код вернет полный путь к переменной окружения " %appdata%", которая обычно имеет вид " C:\Users\{имя пользователя}\AppData\Roaming".
