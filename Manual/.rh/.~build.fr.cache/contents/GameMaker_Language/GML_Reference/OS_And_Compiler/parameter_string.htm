строка_параметра Параметры командной строки - это дополнительные команды, которые можно добавить в exe, чтобы изменить способ выполнения программы. С помощью этой функции вы можете получить выбранный параметр командной строки в виде строки. Вы можете узнать количество параметров для текущей игры с помощью функции parameter_count(), где первый параметр имеет индекс 1, а последний - индекс, возвращаемый функцией (значение 0 является особенным, так как это имя файла исполняемого файла игры, включая путь). Следует отметить, что эта функция будет работать на платформе HTML5, получая параметры url. Синтаксис: параметр_строки(n); Возвращается: Строка Пример: var p_num; p_num = parameter_count(); if p_num > 0 { var i; for (i = 0; i < p_num; i += 1) { p_string[i] = parameter_string(i + 1); } } Приведенный выше код получит количество параметров командной строки, и если их 1 или более, то пройдется по ним и сохранит их в виде строк в массиве.
