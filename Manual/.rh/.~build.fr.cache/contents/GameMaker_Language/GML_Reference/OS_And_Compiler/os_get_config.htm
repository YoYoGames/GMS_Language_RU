os_get_config Эта функция возвращает имя (в виде строки) текущей выбранной конфигурации для вашей игры. Более подробную информацию о конфигурациях см. в разделе Конфигурации. Синтаксис: os_get_config() Возвращается: Строка Пример: if os_get_config() = "Free_Version" { global.Ads = true; } else global.Ads = false; Приведенный выше код проверит, какая конфигурация используется, и если это конфигурация под названием "Free_Version", то реклама будет включена в игре.
