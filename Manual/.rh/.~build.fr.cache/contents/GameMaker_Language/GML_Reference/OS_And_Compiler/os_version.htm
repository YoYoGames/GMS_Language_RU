os_version Эта переменная сообщает вам номер версии ОС, на которой запущена ваша игра. Например, если вы запускаете ее на Windows 10, os_version будет равна 655360. В следующей таблице приведены значения, которые возвращаются для самых последних версий различных ОС: Операционная система Имя версии (номер версии) Возвращаемое значение (примеры) Android Nougat (7.0 - 7.11) Oreo (8.0 - 8.11) Pie (9.0) Android X (10.0) 24 - 25 26 - 27 28 29 iOS Возвращаемое значение вычисляется как: (major_version * 16777216) + (minor_version * 4096) + build_number iOS 10 (10.3) iOS 11 (11.4) iOS 12 (12.0) iOS 13 (13.0) iPhone iOS 13 (13.5) iPhone/Ipad 167784448 184565760 201326592 218103808 218124288 macOS X Возвращаемое значение вычисляется как: (major_version * 16777216) + (minor_version * 4096) + build_number El Capitan (10.11) Sierra (10.12) High Sierra (10.13) Mojave (10.14) Catalina (10.15) 167817216 167821312 167825408 167829504 167833600 Windows (возвращаемое значение вычисляется как: majorVersion * 65536 + minorVersion) Windows 7 (6.1) Windows 8 (6.2) Windows 8.1 (6.3) Windows 10 (10.0) 393217 393218 393219 655360 ПРИМЕЧАНИЕ: Если вам нужна дополнительная информация об ОС Windows, вы можете воспользоваться функцией environment_get_variable(). Синтаксис: os_version Возвращается: Реально Пример: if (os_type == os_android) && (os_version > 10) { global.GFX = 1; } Приведенный выше код проверяет тип операционной системы и номер версии, если они оба верны, то глобальная переменная устанавливается в 1.
