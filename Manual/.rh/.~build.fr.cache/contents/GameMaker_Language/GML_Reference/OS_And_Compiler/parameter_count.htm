счётчик параметров Параметры командной строки - это дополнительные команды, которые вы можете добавить в exe, чтобы изменить способ выполнения программы. Количество параметров для текущей игры можно узнать с помощью этой функции, где первый параметр имеет индекс 1, а последний имеет индекс, возвращаемый функцией (значение 0 является особенным, так как это имя файла исполняемого файла игры, включая путь). Следует отметить, что эта функция будет работать для платформы HTML5, получая параметры url. Синтаксис: parameter_count(); Возвращается: Реально Пример: p_num = parameter_count(); Приведенный выше код будет хранить количество параметров командной строки, которые были использованы для игры, в переменной "p_num".
