os_browser Эта переменная, доступная только для чтения, содержит одну из различных констант (перечисленных ниже), которые используются в GameMaker Studio 2, чтобы сообщить вам, в каком браузере вы в настоящее время запускаете игру (если таковой имеется). Синтаксис: os_browser Возвращается: Постоянная Постоянная Описание browser_not_a_browser Игра не воспроизводится в браузере браузер_неизвестен Неизвестный браузер browser_ie Internet Explorer browser_ie_mobile Internet Explorer на мобильном устройстве браузер_firefox Mozilla Firefox браузер_хром Google Chrome браузер_safari Сафари браузер_safari_mobile Safari на мобильном устройстве браузерная_опера Опера браузер_тизен Браузер для мобильных устройств Tizen магазин_окон_браузера Приложение дляWindows Пример: if (os_browser == browser_not_a_browser) { global.Config = 0; } else { global.Config = 1; } Приведенный выше код проверяет, запущена ли игра в браузере или нет, и устанавливает значение глобальной переменной в зависимости от результата проверки.
