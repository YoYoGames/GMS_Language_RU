game_save_id Эта переменная, доступная только для чтения, возвращает полный идентификатор пути к каталогу, который используется вашей игрой для сохранения файлов. Этот каталог может быть или не быть видимым для других приложений, в зависимости от платформы, и для HTML5-цели он вернет пустую строку. ПРИМЕЧАНИЕ: Эта переменная носит исключительно информационный характер и не должна использоваться для получения пути к файлу или чего-либо подобного, поскольку она не будет корректно работать на всех платформах. Синтаксис: game_save_id Возвращается: Строка Пример: save_dir = game_save_id; Это позволит сохранить каталог для сохранения файлов в переменной для дальнейшего использования.
