значение_высокой_оценки С помощью этой функции вы можете получить значение счета, которое было сохранено в списке высоких результатов в заданной позиции. Если счет не был введен, функция возвращает 0. Синтаксис: highscore_value(place); Аргумент Описание место Место на столе (1-10). Возвращается: Н/Д Пример: var i = 9; repeat(10) { scr[i] = highscore_value(i + 1); i -= 1; } Приведенный выше код будет циклически просматривать список высоких оценок и сохранять все оценки в массив.
