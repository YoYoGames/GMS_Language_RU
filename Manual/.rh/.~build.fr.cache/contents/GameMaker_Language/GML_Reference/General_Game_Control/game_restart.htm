перезапуск игры С помощью этой функции вы можете перезапустить игру. По сути, это то же самое, что и запустить игру в первый раз, поэтому будет вызвано событие Game Start, а также событие Game End. Однако следует отметить, что некоторые вещи не будут сброшены при вызове этой функции: Глобальные переменные не будут повторно инициализированы, если это не указано в явном виде - например, встроенная глобальная переменная score не будет начинаться с нуля после перезапуска игры, если она уже была изменена в игре. Состояние GPU не будет изменено (поэтому, если вы, например, установили цвет рисования или альфу, они останутся в измененном значении). Скорость игры останется такой, какой вы ее установили в коде игры (если вы ее изменили, это изменение будет сохранено). Любой актив из браузера активов, который был изменен во время выполнения игры - например, если вы измените начало координат для ресурса спрайта или сместите позицию ресурса пути - не будет сброшен. Динамические ресурсы, такие как буферы, поверхности, структуры данных или импортированные спрайты, также не будут очищены или удалены (хотя вы можете потерять ссылки на них, поэтому при использовании этой функции старайтесь либо использовать глобальные ссылки для динамического ресурса, либо очистить их до вызова функции). Синтаксис: game_restart(); Возвращается: Н/Д Пример: if keyboard_check_pressed(ord("R")) game_restart(); Это позволит перезапустить игру, когда игрок нажимает клавишу "R".
