конец игры С помощью этой функции вы можете завершить игру (при этом будет вызвано событие Game End Event ). Это произойдет не мгновенно, а скорее в конце текущего шага, поэтому любой код находящийся в том же шаге после вызова этой функции, будет по-прежнему выполняться. Обратите внимание, что эта функция имеет следующие ограничения: На устройствах Android вызов функции game_end() переводит приложение в фоновый режим, но не закрывает его. Это должен сделать пользователь. На iOS он ничего не сделает и молча провалится. На всех консолях (включая UWP при запуске на Xbox) это может привести к краху игры или в лучшем случае к молчаливому отказу, а также к отказу отправки, поэтому эту функцию нельзя использовать. На HTML5 это завершит игру, но оставит пользователя смотреть на чистый холст веб-страницы, и поэтому не рекомендуется использовать его на этой целевой платформе. В Windows (включая UWP на ПК), Linux и macOS функция просто завершает игру и закрывает окно игры (также будет вызвано событие Game End Event ). Синтаксис: game_end(); Возвращается: Н/Д Пример: if keyboard_check_pressed(vk_escape) game_end(); Это приведет к завершению игры, если игрок нажмет клавишу "escape".
