имя_высокой_оценки С помощью этой функции вы можете получить строку имени, которая была сохранена в списке высоких результатов на заданной позиции. Если имя не было введено, будет возвращена строка "Unknown". Синтаксис: highscore_name(place); Аргумент Описание место Место на столе (1-10). Возвращается: Строка Пример: var i = 9; repeat(10) { name[i] = highscore_name(i + 1); i -= 1; } Приведенный выше код будет циклически просматривать список высоких результатов и сохранять все имена в массив.
