имя_игрового_проекта Эта переменная, доступная только для чтения, возвращает отображаемое имя вашей игры для целевой платформы в "удобном для сохранения" формате для целевой платформы. Если отображаемое имя содержит символы, недопустимые для имени файла, они будут будут автоматически заменены на "_". Имя отображения можно задать в Параметрах игры. Обратите внимание, что поскольку для HTML5 нет ограничений на имена файлов, эта строка, вероятно, будет такой же что и та, которую возвращает game_display_name. Синтаксис: имя_игрового_проекта Возвращается: Строка Пример: var file = game_project_name + ".ini"; ini_open(file); ini_write_real("Scores", "Highscore", score); ini_close(); Приведенный выше код получает имя проекта и использует его для открытия (или создания) ini-файла, в который затем производится запись.
