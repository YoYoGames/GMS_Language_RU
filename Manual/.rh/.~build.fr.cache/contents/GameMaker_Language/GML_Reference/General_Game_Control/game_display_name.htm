имя_игрового_дисплея Эта переменная, доступная только для чтения, возвращает отображаемое имя вашей игры для целевой платформы, как установлено в параметрах игры. Синтаксис: имя_игрового_дисплея Возвращается: Строка Пример: var name = game_display_name; var ver = string(GM_version); draw_text(32, 32, name + ":" + ver); Приведенный выше код получает отображаемое имя и номер версии игры и рисует их.
