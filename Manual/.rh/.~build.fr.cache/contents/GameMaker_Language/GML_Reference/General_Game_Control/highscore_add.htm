highscore_add С помощью этой функции вы можете добавить имя и результат во внутренний глобальный список высоких результатов. Нет необходимости проверять значение, чтобы убедиться, что оно достаточно высоко для внесения в список, поскольку GameMaker Studio 2 будет сохранять только те значения, которые больше, чем десятая сохраненная позиция. Синтаксис: highscore_add(str, numb); Аргумент Описание str Строка (имя), к которой нужно приписать оценку numb Число (счет) для добавления. Возвращается: Н/Д Пример: if lives = 0 { highscore_add(global.Name, score); score = 0; room_goto(rm_MainMenu); } Приведенный выше код проверит жизни и, если они равны 0, добавит текущий счет и строку, хранящуюся в глобальной переменной "Name", в список высоких результатов, а затем установит счет на 0 и отправит игрока в комнату проиндексированную в "rm_MainMenu".
