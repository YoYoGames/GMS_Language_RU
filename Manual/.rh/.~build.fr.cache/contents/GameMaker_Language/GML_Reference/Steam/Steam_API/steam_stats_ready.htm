steam_stats_ready При использовании API Steam эту функцию можно вызвать, чтобы проверить, что API клиента Steam правильно инициализировал статистику для вашей игры. Синтаксис: steam_stats_ready(); Возвращается: Булево Пример: global.steam_api= false; if steam_initialised() { if steam_stats_ready() && steam_is_overlay_enabled() { global.steam_api= true; } } Приведенный выше код установит глобальную переменную в true, если API клиента Steam правильно инициализирован, вместе с функциями статистики и наложения Steam, или установит переменную в false в противном случае.
