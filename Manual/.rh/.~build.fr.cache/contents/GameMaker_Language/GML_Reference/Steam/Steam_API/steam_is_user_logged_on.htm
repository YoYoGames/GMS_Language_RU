steam_is_user_logged_on Эта функция возвращает true, если клиент Steam в настоящее время имеет активное соединение с серверами Steam. Если возвращается false, это означает, что активного соединения нет из-за сетевых проблем на локальной машине или из-за того, что сервер Steam сервер не работает или занят. Синтаксис: steam_is_user_logged_on(); Возвращается: Строка Пример: if steam_is_user_logged_on() { global.user_id = steam_get_user_account_id(); } Приведенный выше код проверит, вошел ли пользователь на сервер Steam, и если вошел, то сохранит идентификатор пользователя в глобальной переменной.
