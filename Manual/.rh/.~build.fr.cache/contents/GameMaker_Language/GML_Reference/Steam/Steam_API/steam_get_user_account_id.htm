steam_get_user_account_id Эта функция используется для получения уникального идентификатора пользователя, который Steam присваивает каждому пользователю и который необходим для использования некоторых функций User Generated Content. Синтаксис: steam_get_user_account_id(); Возвращается: Реально Пример: global.user_id = steam_get_user_account_id(); Приведенный выше код получает уникальный идентификатор пользователя для человека, владеющего игрой, и сохраняет его в глобальной переменной.
