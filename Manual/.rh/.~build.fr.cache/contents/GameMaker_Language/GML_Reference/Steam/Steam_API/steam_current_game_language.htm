steam_current_game_language Эта функция используется для получения текущего языка, который использует Steam (в виде строки), например, "english". Синтаксис: steam_current_game_language(); Возвращается: Строка Пример: язык = steam_current_game_language(); Приведенный выше код получает язык, используемый в текущей игре.
