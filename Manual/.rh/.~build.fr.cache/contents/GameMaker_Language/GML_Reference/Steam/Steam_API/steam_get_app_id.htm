steam_get_app_id Эта функция используется для получения уникального идентификатора приложения, который Steam присваивает вашей игре и который необходим для использования некоторых функций User Generated Content. Синтаксис: steam_get_app_id(); Возвращается: Реально Пример: global.app_id = steam_get_app_id(); Приведенный выше код получает уникальный идентификатор приложения для вашей игры в Steam и сохраняет его в глобальной переменной.
