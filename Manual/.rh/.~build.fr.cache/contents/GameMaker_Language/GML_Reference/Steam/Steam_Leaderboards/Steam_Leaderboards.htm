Пар API Steam поддерживает постоянные таблицы лидеров с автоматически упорядоченными записями. Эти таблицы лидеров можно использовать для отображения глобальных и дружеских таблиц лидеров в вашей игре и на веб-странице сообщества вашей игры. В каждой игре может быть до 10 000 таблиц лидеров, и каждая таблица лидеров может быть извлечена сразу после того, как в нее будет помещен результат игрока, но обратите внимание, что в каждой таблице лидеров у игрока может быть только одна запись, хотя количество игроков в каждой таблице не ограничено. игроков в каждой таблице лидеров. Каждая запись в таблице лидеров содержит имя, счет и ранг лидера, и эти данные будут заменяться при создании новой записи в таблице лидеров для пользователя, а для добавления и извлечения этих данных из таблицы лидеров можно использовать следующие функции для вашей игры: steam_create_leaderboard steam_upload_score steam_upload_score_buffer steam_download_scores steam_download_scores_around_user steam_download_friends_scores Назад: Пар Следующий: Достижения и статистика © Copyright YoYo Games Ltd. 2021 Все права защищены
