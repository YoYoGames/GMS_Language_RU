steam_file_persisted С помощью этой функции вы можете проверить заданный файл на предмет его синхронизации с Steam Cloud. Возвращаемое значение true означает, что это так, а false - что нет. Синтаксис: steam_file_persisted(filename); Аргумент Описание имя файла Имя файла для проверки. Возвращается: Булево Пример: if !steam_file_persisted("Save.txt") { steam_file_share("Save.txt"); } Приведенный выше код проверит, был ли файл сохранен в Steam Cloud, и если нет, то синхронизирует его.
