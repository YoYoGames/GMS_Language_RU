Облакопара Облако Steam Cloud обеспечивает простую и прозрачную систему удаленного хранения файлов для вашей игры. Все файлы, записанные на диск с помощью облачных функций GameMaker Studio 2 (перечисленных ниже), будут реплицированы на серверы Steam после выхода игры. Если пользователь сменит компьютер, файлы будут загружены на новый компьютер перед запуском игры, то есть игра сможет получить доступ к файлам, прочитав их с помощью соответствующих функций Steam. Клиент Steam выполняет работу обеспечивая синхронизацию файлов на всех компьютерах, к которым обращается пользователь. ПРИМЕЧАНИЕ: По умолчанию облако не включено для игры на Steamworks. его необходимо включить предварительно на вкладке "Облако" в админке игры Steamworks, где необходимо установить квоту байтов и файлов. В следующий раз, когда вы опубликуете свою игры конфигурацию Steamworks, облачное хранилище будет готово к использованию.
