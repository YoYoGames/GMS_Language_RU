steam_get_quota_free С помощью этой функции можно узнать, сколько свободного места осталось для пользователя квоты Steam Cloud. Возвращаемое значение - в байтах. Синтаксис: steam_get_quota_free(); Возвращается: Реально Пример: if steam_is_cloud_enabled_for_app() { quota = steam_get_quota_free(); } Приведенный выше код проверяет, включено ли облако steam для игры, и если да, то получает размер свободного места для хранения и сохраняет его в переменной.
