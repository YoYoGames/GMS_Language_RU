steam_is_cloud_enabled_for_account С помощью этой функции вы можете проверить, включен ли сервис Steam Cloud в настройках клиента Steam. Она вернет true, если это так, и false в противном случае. Обратите внимание, что это не означает, что вы автоматически можете сохранять данные в облако, так как оно также должно быть включено для вашей игры (вы можете проверить это с помощью функции steam_is_cloud_enabled_for_app()). Если облако Steam Cloud включено для вашей игры, но пользователь отключил его локально, вы все равно можете использовать функции облака для хранения и получения данных из локальной копии всех файлов, просто они не будут загружаться в облако при завершении игры или синхронизироваться при ее запуске. Синтаксис: steam_is_cloud_enabled_for_account(); Возвращается: Булево Пример: if steam_is_cloud_enabled_for_account { steam_file_share("Save.txt"); } Приведенный выше код проверяет, включен ли у пользователя Steam Cloud, и если он возвращает true, то синхронизирует заданный файл.
