steam_file_delete Эта функция удалит заданный файл из Steam Cloud. Функция вернет значение 0 в случае неудачи по какой-либо причине и значение больше 0 в случае успеха. Синтаксис: steam_file_delete(filename); Аргумент Описание имя файла Имя удаляемого файла. Возвращается: Реально Пример: if steam_file_exists("Save.txt") { steam_file_delete("Save.txt"); } Приведенный выше код проверит, существует ли файл, и если да, то удалит его из Steam Cloud.
