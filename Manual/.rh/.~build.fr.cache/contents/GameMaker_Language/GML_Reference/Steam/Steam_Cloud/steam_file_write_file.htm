steam_file_write_file С помощью этой функции вы можете скопировать содержимое локально сохраненного файла в файл, синхронизированный с Steam Cloud. Локальный файл должен существовать до использования этой функции, и она вернет значение 0 в случае неудачи по какой-либо причине и значение больше 0 в случае успеха. Синтаксис: steam_file_write_file(steam_filename, local_filename); Аргумент Описание steam_filename Файл Steam Cloud для копирования. локальное_имя Локальный файл, из которого будет производиться копирование. Возвращается: настоящий Пример: steam_file_write_file("rm_koala.png", "Koala2.png"); Приведенный выше код скопирует содержимое файла "Koala2.png" в файл Steam Cloud "rm_koala.png".
