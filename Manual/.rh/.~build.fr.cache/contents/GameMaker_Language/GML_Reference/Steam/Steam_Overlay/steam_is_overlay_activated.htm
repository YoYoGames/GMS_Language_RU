steam_is_overlay_activated Эта функция используется для определения того, активна ли у пользователя накладка Steam Overlay или нет. Если наложение активно и видно пользователю, то функция вернет true, а если нет, то false. Примером того, для чего эта функция может быть использована для опроса Steam API на предмет наличия оверлея, чтобы можно было приостановить игру во время показа оверлея. Синтаксис: steam_is_overlay_activated(); Возвращается: Булево Пример: if steam_is_overlay_activated() { global.Pause = true; } Приведенный выше код проверит, активен ли оверлей Steam, и если да, то установит глобальную переменную "Pause" в true.
