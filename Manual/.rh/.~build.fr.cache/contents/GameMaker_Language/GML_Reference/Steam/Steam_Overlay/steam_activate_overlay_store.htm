steam_activate_overlay_store С помощью этой функции вы можете открыть оверлей Steam на странице магазина для игры, чтобы пользователи могли купить или загрузить DLC (например). Вам необходимо предоставить уникальный идентификатор приложения для игры или DLC, который вы получите из панели Steam при настройке этой функции. вверх. Синтаксис: steam_activate_overlay_store(app_id); Аргумент Описание идентификатор приложения Уникальный идентификатор приложения для вашей игры. Возвращается: Н/Д Пример: if keyboard_check_pressed(vk_f1) { steam_activate_overlay_store(global.DLC_id); } Приведенный выше код опрашивает клавиатуру на предмет нажатия клавиши F1, и если она нажата, то на странице будет открыта накладка Steam для содержимого игры с использованием идентификатора приложения, хранящегося в глобальной переменной.
