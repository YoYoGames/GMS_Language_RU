steam_activate_overlay_browser С помощью этой функции вы можете открыть оверлей игры Steam в веб-браузере и загрузить указанный URL. Для правильного разрешения необходимо использовать полный URL в виде строки, например: "http://www. steamgames.com/". Синтаксис: steam_activate_overlay(url); Аргумент Описание url (Полный) URL-адрес оверлея для открытия. Возвращается: Н/Д Пример: if keyboard_check_pressed(vk_f1) { steam_activate_overlay_browser("http://www. steamgames.com/"); } Приведенный выше код опрашивает клавиатуру на предмет нажатия клавиши F1, и если она есть, то открывается оверлей Steam и переходит на заданный URL.
