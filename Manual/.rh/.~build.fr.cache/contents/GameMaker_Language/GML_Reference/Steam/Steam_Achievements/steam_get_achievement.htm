steam_get_achievement С помощью этой функции вы можете проверить Steam API, было ли получено определенное достижение. Достижение должно быть предварительно определено на странице учетных записей панели управления Steamworksдля вашей игры, а строка, передаваемая в функцию должна совпадать со строкой, используемой в качестве имени API на панели управления. Синтаксис: steam_get_achievement(ach_name); Аргумент Описание ач_имя Имя достижения, которое нужно получить (строка). Возвращается: Булево Пример: если hp <= 0 { global.Deaths += 1; if global.Deaths == 10 { if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times"); } } Приведенный выше код наградит игрока достижением, если глобальная переменная "Deaths" равна 10 и если достижение еще не было получено.
