steam_set_achievement С помощью этой функции вы можете сообщить Steam API о необходимости присвоить ("установить") игроку достижение. Эти достижения должны быть определены на странице учетных записей панели управления Steamworksдля вашей игры, а строка, передаваемая в функцию должна совпадать со строкой, используемой в качестве имени API на панели управления. Steam Game Overlay отобразит панель уведомления, информирующую пользователя о полученном достижении, если только достижение уже не было получено. в этом случае ничего не произойдет. Синтаксис: steam_set_achievement(ach_name); Аргумент Описание ач_имя Имя достижения, которое нужно установить (строка). Возвращается: Н/Д Пример: если hp <= 0 { global.Deaths += 1; if global.Deaths == 10 { if !steam_get_achievement("ach_Player_Dies_Ten_Times") steam_set_achievement("ach_Player_Dies_Ten_Times"); } } Приведенный выше код наградит игрока достижением, если глобальная переменная "Deaths" равна 10 и если достижение еще не было получено.
