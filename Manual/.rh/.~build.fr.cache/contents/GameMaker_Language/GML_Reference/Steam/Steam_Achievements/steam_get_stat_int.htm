steam_get_stat_int С помощью этой функции вы можете получить значение определенной целочисленной статистики со знаком. Статистика должна быть предварительно определена на странице учетных записей панели управления Steamworksдля вашей игры, а строка, передаваемая в функцию, должна совпадать со строкой, используемой в качестве API Name на панели управления. Синтаксис: steam_get_stat_int(stat_name); Аргумент Описание имя_статьи Имя статистики, которую нужно получить (строка). Возвращается: Реально Пример: xp += 100; steam_set_stat_int("Total_XP", steam_get_stat_int("Total_XP") + 100); if steam_get_stat_int( "Total_XP") > 1000 { if !steam_get_achievement("Ach_1000XP") steam_set_achievement("Ach_1000XP"); } Приведенный выше код устанавливает статистику, а затем проверяет ее конечное значение, чтобы решить, присуждать достижение или нет.
