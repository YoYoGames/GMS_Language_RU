steam_get_stat_avg_rate С помощью этой функции вы можете получить значение определенной средней статистики. Эта статистика должна быть предварительно определена на странице учетных записей панели управления Steamworksдля вашей игры, а строка, передаваемая в функцию, должна совпадать со строкой со строкой, используемой в качестве API Name на панели управления. Синтаксис: steam_get_stat_avg_rate(stat_name); Аргумент Описание имя_статьи Имя статистики, которую нужно получить (строка). Возвращается: Реально Пример: var avg = steam_get_stat_avg_rate("PointsPerHour"); draw_text(8, 8, "PPH = " + string(avg); Приведенный выше код получает текущее значение для средней статистики "PointsPerHour" и рисует его на экране.
