steam_set_stat_float С помощью этой функции вы можете установить определенную статистику в новое значение с плавающей точкой. Статистика должна быть предварительно определена на странице учетных записей панели управления Steamworksдля вашей игры, а строка, передаваемая в функцию, должна совпадать со строкой, используемой в качестве API Name на панели управления. Примером использования этой функции может быть отслеживание того, как далеко прошел игрок, или какой процент игры завершен. Синтаксис: steam_set_stat_float(stat_name, value); Аргумент Описание имя_статьи Имя статистики, которую нужно установить (строка). значение Значение (32-битное плавающее число), на которое нужно установить значение stat. Возвращается: Н/Д Пример: var dist_pc = (dist / dist_max) * 100; steam_set_stat_float("Travelled", dist_pc); Приведенный выше код вычисляет процент, основанный на переменной пройденного расстояния "dist" и максимальном расстоянии, которое вы можете проехать "dist_max", а затем устанавливает статистику "Travelled" в новое значение.
