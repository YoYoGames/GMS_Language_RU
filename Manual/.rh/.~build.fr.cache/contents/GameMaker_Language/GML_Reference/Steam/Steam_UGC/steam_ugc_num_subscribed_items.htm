steam_ugc_num_subscribed_items Эта функция может быть использована для получения количества элементов, на которые подписался текущий пользователь. Синтаксис: steam_ugc_num_subscribed_items(); Возвращается: Реально Пример: numSub = steam_ugc_num_subscribed_items(); Приведенный выше код будет хранить количество подписанных элементов в переменной.
