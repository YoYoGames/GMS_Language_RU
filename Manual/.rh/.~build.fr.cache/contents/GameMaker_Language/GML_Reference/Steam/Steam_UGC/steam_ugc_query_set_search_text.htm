steam_ugc_query_set_search_text Эта функция может использоваться для дальнейшей фильтрации любого заданного запроса пользовательского контента, в частности для поиска заданной строки в заголовке и описании элементов пользовательского контента. Ручка запроса - это значение, возвращенное при создании запроса (например, с помощью, steam_ugc_create_query_user()) а второй аргумент - это строка, которую вы хотите использовать в качестве поискового запроса. Функция вернет true, если фильтр запроса был установлен правильно, и false в противном случае. Синтаксис: steam_ugc_query_set_search_text(ugc_query_handle , search_text); Аргумент Описание ugc_query_handle Ручка запроса, которую следует использовать. поисковый_текст Текст для поиска, который будет использоваться (строка) для запроса. Возвращается: Булево Пример: var query_handle = steam_ugc_create_query_all(ugc_query_RankedByVote, ugc_match_Items, 1); steam_ugc_query_set_search_text(query_handle, "texture"); steam_ugc_query_set_return_long_description(query_handle, true); steam_ugc_query_set_allow_cached_response(query_handle, true); query_ID = steam_ugc_send_query(query_handle); Приведенный выше код создает запрос запроса и сохраняет его хэндл в локальной переменной для дальнейшего использования в остальных функциях, которые определяют запрос перед отправкой запроса.
