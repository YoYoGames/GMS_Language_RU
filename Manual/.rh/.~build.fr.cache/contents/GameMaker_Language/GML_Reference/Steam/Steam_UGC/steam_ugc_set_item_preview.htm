steam_ugc_set_item_preview Эта функция устанавливает изображение предварительного просмотра, которое будет использоваться для данного элемента. Изображение должно быть предоставлено в виде файла формата PNG, JPG или GIF с максимальным размером 1 МБ. Путь к изображению должен быть относительным путем в сохранении или в игре (т.е. включенный файл). Хендл обновления - это значение, возвращенное при вызове функции steam_ugc_start_item_update(). Функция вернет true, если обращение к API было успешным и false, если возникла проблема. Синтаксис: steam_ugc_set_item_preview(ugc_update_handle, preview); Аргумент Описание ugc_update_handle Уникальный идентификатор для обновляемого пользовательского центра. предварительный просмотр Изображение предварительного просмотра (JPG, GIF или PNG - максимальный размер 1 МБ), которое будет использоваться для элемента. Возвращается: Булево Пример: var app_id = steam_get_app_id(); var updateHandle = steam_ugc_start_item_update(app_id, global.Publish_ID); steam_ugc_set_item_title(updateHandle, "My workshop item(3)!"); steam_ugc_set_item_description( updateHandle, "testing мастерской..."); steam_ugc_set_item_visibility(updateHandle, ugc_visibility_public); var tagArray; tagArray[0] = "Test"; tagArray[1] = "New"; steam_ugc_set_item_tags( updateHandle, tagArray); steam_ugc_set_item_preview (updateHandle, "promo.jpg"); steam_ugc_set_item_content(updateHandle, "WorkshopContent1"); requestId = steam_ugc_submit_item_update(updateHandle, "Version 1.2"); Приведенный выше код получает ID игры, затем использует его вместе с ранее сохраненным ID опубликованного файла для создания хэндла обновления для элемента. Этот хэндл затем используется для обновления различных частей информации перед отправкой обновления на серверы Workshop сервера.
