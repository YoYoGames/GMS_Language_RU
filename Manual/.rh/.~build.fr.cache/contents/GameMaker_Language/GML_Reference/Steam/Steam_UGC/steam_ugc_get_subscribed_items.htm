steam_ugc_get_subscribed_items Эта функция заполнит список DS всеми опубликованными идентификаторами файлов для элементов, на которые пользователь в настоящее время подписан. Сначала вы должны создать список и сохранить индекс в переменной, а затем передать его в функцию. Функция вернет true, если все сделано правильно и Steam API инициализирован, или false, если произошла ошибка. Синтаксис: steam_ugc_get_subscribed_items(item_list); Аргумент Описание список элементов Индекс (ранее созданный) списка DS. Возвращается: Булево Пример: steam_list= ds_list_create(); steam_ugc_get_subscribed_items(steam_list); Приведенный выше код создаст пустой список DS, а затем заполнит его идентификаторами файлов для всех подписанных элементов для пользователя.
