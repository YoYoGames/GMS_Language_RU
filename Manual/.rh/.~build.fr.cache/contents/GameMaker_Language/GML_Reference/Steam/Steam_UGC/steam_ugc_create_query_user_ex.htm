steam_ugc_create_query_user_ex Эта функция может быть использована для запроса базы данных пользовательского контента. Функция требует значение ID для пользователя и ID игры, которая будет потреблять предмет, и/или ID игры, которая создала предмет. Также необходимо использовать следующие константы для типа данных для запроса, типа элемента для запроса и порядка, в котором будут отсортированы возвращаемые элементы, а также номер страницы - обратите внимание, что запрос вернет максимум 50 элементов. Тип списка Постоянная Описание ugc_list_Published См. документациюSteam. ugc_list_VotedOn - ugc_list_VotedUp - ugc_list_VotedDown - ugc_list_WillVoteLater - ugc_list_Favorited - ugc_list_Subscribed - ugc_list_UsedOrPlayed - ugc_list_Followed - Тип соответствия Постоянная Описание ugc_match_Items См. документациюSteam. ugc_match_Items_Mtx - ugc_match_Items_ReadyToUse - ugc_match_Collections - ugc_match_Artwork - ugc_match_Videos - ugc_match_Screenshots - ugc_match_AllGuides - ugc_match_WebGuides - ugc_match_IntegratedGuides - ugc_match_UsableInGame - ugc_match_ControllerBindings - Порядок сортировки Постоянная Описание ugc_sortorder_CreationOrderDesc См. документациюSteam. ugc_sortorder_CreationOrderAsc - ugc_sortorder_TitleAsc - ugc_sortorder_LastUpdatedDesc - ugc_sortorder_SubscriptionDateDesc - ugc_sortorder_VoteScoreDesc - ugc_sortorder_ForModeration - Функция возвращает уникальное значение хэндла запроса, которое должно быть сохранено в переменной для использования в других функциях запроса. Обратите внимание, что эта функция только подготавливает запрос, но не отправляет его - для этого вы должны вызвать функцию steam_ugc_send_query.htm">steam_ugc_send_query() - и вы можете использовать другие функции steam_ugc_query_* () для уточнения поискового запроса перед его отправкой.
