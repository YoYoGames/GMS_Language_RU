Двоичные файлы В редких ситуациях вам может понадобиться считывать данные из двоичных файлов. Эти функции могут быть использованы для этой цели, хотя в целом мы рекомендуем вам использовать специальные функции буфера для создания, сохранения и загрузки двоичных данных, а не эти функции. ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5. Для этого существуют следующие низкоуровневые процедуры: file_bin_open file_bin_rewrite file_bin_close размер файла_bin_size позиция файла_bin_position file_bin_seek файл_bin_запись_байт файл_bin_read_byte Назад: Работа с файлами Следующее: Файловые каталоги © Copyright YoYo Games Ltd. 2021 Все права защищены
