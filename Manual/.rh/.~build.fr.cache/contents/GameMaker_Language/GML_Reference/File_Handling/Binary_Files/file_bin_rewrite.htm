file_bin_rewrite Эта функция принимает имя файла, возвращенное функцией file_bin_open() и затем переписывает файл, очищая его от всех предыдущих данных, чтобы начать запись с начала файла. ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5. Синтаксис: file_bin_rewrite(binfile); Аргумент Описание binfile Идентификатор файла для перезаписи. Возвращается: Н/Д Пример: file = file_bin_open("myfile.bin", 2); file_bin_rewrite(file); Это позволит открыть файл из того же каталога, что и игра, и присвоить его индекс переменной "file".
