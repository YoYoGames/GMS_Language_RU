файл_bin_запись_байт Эта функция записывает байт данных в файл, идентифицированный идентификатором файла, в текущей позиции записи. Вы передаете значение идентификатора файла, возвращаемое функцией file_bin_open() и байт данных для записи. ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5. Синтаксис: file_bin_write_byte(binfile, byte); Аргумент Описание binfile Идентификатор файла для записи. байт Данные для записи. Возвращается: настоящий Пример: file_bin_write_byte(file, data); Это запишет байт в выбранный файл. Назад: Бинарные файлы Далее: file_bin_read_byte © Copyright YoYo Games Ltd. 2021 Все права защищены
