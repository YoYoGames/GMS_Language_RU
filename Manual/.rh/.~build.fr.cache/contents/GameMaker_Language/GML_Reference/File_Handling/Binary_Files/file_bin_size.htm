размер файла_bin_size Эта функция возвращает размер (в байтах) файла, который был открыт для чтения и/или записи. Идентификатором файла является значение, возвращаемое функцией file_bin_open(). ПРИМЕЧАНИЕ: Эти функции не работают, если целевым модулем является HTML5. Синтаксис: file_bin_size(binfile); Аргумент Описание binfile Идентификатор файла для получения размера. Возвращается: Реально Пример: file = file_bin_open("myfile.bin", 2); size = file_bin_size(file); file_bin_close(file); Это позволит открыть файл из локального каталога игры и присвоить его индекс переменной "file". Затем он получит размер этого файла и снова закроет его. Назад: Бинарные файлы Далее: file_bin_position © Copyright YoYo Games Ltd. 2021 Все права защищены
