ini_key_exists Эта функция проверяет, существует ли ключ в текущем открытом ini, и возвращает true, если существует, или false в противном случае. Эта проверка не является необходимой для предотвращения ошибок, так как, если ключ не существует, чтение из несуществующего ключа просто вернет значение по умолчанию. Однако она может быть полезна для проверки того, сохранил ли ini файл определенные данные, и для некоторых других целей. Синтаксис: ini_key_exists(section, key); Аргумент Описание раздел Раздел в открытом .ini файле для проверки ключа. ключ Ключ для проверки. Возвращается: Булево Пример: ini_open("savedata.ini"); if !ini_key_exists("save1", "name") { global.name = "Player1"; } ini_close(); Это установит переменную global.name в значение "Player1", если не существует такого ключа "name".
