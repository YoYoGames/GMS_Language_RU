ini_read_real Вы можете использовать эту функцию для чтения числа из файла данных ini. Ini файлы разбиваются на секции, а затем каждая секция разбивается на пары ключ - значение. Таким образом, типичный ini файл будет выглядеть примерно так: Синтаксис: ini_read_real(section, key, default); Аргумент Описание раздел Раздел .ini для чтения. ключ Ключ в соответствующем разделе .ini для чтения. по умолчанию Значение, которое возвращается, если значение не найдено в определенном месте (или файл .ini не существует). Должно быть действительным числом. Возвращается: Реально Пример: ini_open("savedata.ini"); score = ini_read_real("save1", "score", 0 ); ini_close(); Это откроет файл "savedata.ini" и установит значение score в значение в разделе "save1" > "score" в нем, затем снова закроет .ini. Если в разделе "save1" > "score" нет значения, или файл "savedata.ini" отсутствует. файла, счет будет установлен на 0.
