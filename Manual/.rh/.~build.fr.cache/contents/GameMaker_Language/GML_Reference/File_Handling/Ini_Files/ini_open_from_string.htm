ini_open_from_string Эта функция создаст ini файл из строки и откроет его для чтения/записи. Строка должна быть правильно отформатирована как ini файл (т.е. с секциями, ключами перевода строки и значениями), иначе ini файл не будет создан правильно. Обратите внимание что этот ini файл является временным и будет удален из памяти в момент закрытия, потеряв всю информацию, которая хранилась в нем, однако функция ini_close() функция возвращает строку полного ini-файла, который может быть сохранен на сервере или на диске. Синтаксис: ini_open_from_string(string); Аргумент Описание строка Строка, содержащая всю информацию ini. Возвращается: Н/Д Пример: ini_open_from_string(str) global.sound = ini_read_string("Options", "Sound", true); ini_close(); Приведенный выше код создает временный ini для хранения информации из строки "str", затем читает из ini, прежде чем снова закрыть его.
