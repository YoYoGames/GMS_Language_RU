ini_open Это открывает ini_файл для чтения и/записи. Если ini_файл не существует в проверяемом месте, GameMaker Studio 2 может создать его, но только если вы запишите в него данные. Если вы только считываете информацию из ini_файла, то будут возвращены значения по умолчанию для функции read, но файл ini фактически создан не будет. Обратите внимание, что одновременно у вас может быть открыт только один файл ini, и не забывайте использовать команду ini_close() после завершения чтения/записи из .ini файла, так как информация не сохраняется на диске до этого момента (она также хранится в памяти до тех пор, пока файл не будет закрыт). диск до этого момента (она также хранится в памяти до тех пор, пока файл не будет закрыт). ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации. Синтаксис: ini_open(name); Аргумент Описание имя Имя файла .ini. Возвращается: Н/Д Пример: ini_open("Settings/savedata.ini"); score = ini_read_real("save1", "score", 0); ini_close(); Это откроет файл 'savedata.ini' и прочитает значение очков в разделе "save1" с ключом "score" в нем, затем снова закройте .ini. Если в разделе "save1" нет значения "score" или отсутствует файл "savedata.ini" файла, счет будет установлен на 0 (значение по умолчанию). Обратите внимание, что файл ini был помещен в подкаталог "Settings", который является папкой, содержащей файл ini во включенных файлах Asset Browser.
