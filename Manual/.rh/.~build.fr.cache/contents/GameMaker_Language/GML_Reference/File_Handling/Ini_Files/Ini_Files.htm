Файлы Ини Файлы INI - это небольшие, легкие файлы, совместимые с большинством платформ. Они идеально подходят для хранения небольших фрагментов информации, например, предпочтений интерфейса, локальных высоких результатов, данных об уровнях и т.д... и как таковые являются отличный и простой способ начать сохранять информацию из своих игр. Однако стоит отметить, что у них есть ограничение на объем данных, которые они могут хранить, - максимум 64 КБ, поэтому если вы сохраняете такие вещи, как структуры данных, вам, возможно, стоит стоит присмотреться к другим файловым функциям, например, к текстовым файлам. ВАЖНО! Все проекты GameMaker Studio 2 имеют файл под названием "options.ini", поэтому вы не можете назвать ни один файл с таким именем (вы получите ошибку uid при попытке скомпилировать ваш игровой проект). Обратите внимание, что для игр с локализациями очень важно, чтобы буквы с ударениями могли быть прочитаны из внешних файлов. Это означает, что вам следует сначала создать ini файл в формате UTF8, а затем затем добавить его в GameMaker Studio 2 в качестве включенного файла, чтобы он экспортировался при запуске игры и использовался вместо стандартного ini-файла в формате ANSI, который создается функциями ini, если он не был задан ранее. не был предоставлен. Таким образом, вы сможете читать и писать в нем правильно, с сохранением всех ударений и нелатинских букв.
