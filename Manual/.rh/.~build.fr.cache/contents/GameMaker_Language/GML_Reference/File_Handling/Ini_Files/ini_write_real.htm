ini_write_real Эту функцию можно использовать для записи значения (числового) в файл данных ini. Ini файлы разбиваются на секции, а затем каждая секция разбивается на пары ключ - значение. Таким образом, типичный ini файл будет выглядеть примерно так: Синтаксис: ini_write_real(section, key, value); Аргумент Описание раздел Раздел .ini для записи. ключ Ключ в соответствующем разделе .ini для записи. значение Реальное значение для записи в соответствующее место назначения. Возвращается: Н/Д Пример: score = 1000; ini_open("savedata.ini"); ini_write_real("save1", "Score", score ); score2 = ini_read_real("save1", "Score", 0 ); ini_close(); Это установит score в 1000, затем откроет "savedata.ini" и запишет это значение в "save1" > "Score". Затем в "save1" > "score" будет установлено значение score2, равное 1000, и закроется файл .ini. снова. Если бы он не нашел это значение, счет2 в итоге был бы равен 0.
