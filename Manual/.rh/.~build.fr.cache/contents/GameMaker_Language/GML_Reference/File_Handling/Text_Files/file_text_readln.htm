file_text_readln С помощью этой функции вы можете пропустить остаток текущей строки из данного открытого текстового файла и перейти к началу следующей. Функция также возвращает всю строку в виде строки, что делает ее удобной для чтения полных "кусков" данных для последующего разбора. Синтаксис: file_text_readln(fileid); Аргумент Описание fileid Идентификатор файла для чтения. Возвращается: строка Пример: var file = file_text_open_read(working_directory + "hiscore.txt"); for (var i = 0; i < 10; ++i;) { scr[i] = file_text_read_real(file); file_text_readln(file); scr_name[i] = file_text_read_string(file); file_text_readln(file); } file_text_close(file); Приведенный выше код открывает файл для чтения, а затем циклически просматривает строки файла, считывая поочередно значение вещественного числа и строку в два разных массива для дальнейшего использования. После завершения цикла файл закрывается. Назад: Текстовые файлы Далее: file_text_write_real © Copyright YoYo Games Ltd. 2021 Все права защищены
