file_text_writeln С помощью этой функции можно записать новую строку в открытый текстовый файл. Таким образом, вы можете пропускать строки или записывать информацию построчно (см. пример кода ниже). Синтаксис: file_text_writeln(fileid); Аргумент Описание fileid Идентификатор файла для редактирования. Возвращается: Реально Пример: var file = file_text_open_write(working_directory + "hiscore.txt"); for (var i = 0; i < 10; ++i;) { file_text_write_real(file, scr[i]); file_text_writeln(file); file_text_write_string(file, scr_name[i]); file_text_writeln(file); } file_text_close(file); Приведенный выше код открывает файл для записи, а затем в цикле просматривает два массива, записывая каждое значение массива в новую строку файла. После завершения цикла файл закрывается. Назад: Текстовые файлы Далее: file_text_eoln © Copyright YoYo Games Ltd. 2021 Все права защищены
