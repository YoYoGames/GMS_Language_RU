file_text_write_real С помощью этой функции вы можете записать число в предварительно открытый текстовый файл. Обратите внимание, что поскольку записываемое значение может быть вещественным числом, все десятичные знаки будут записаны с точкой "." в качестве разделителя. Если файл уже содержит информацию, эта информация будет стерта, а строка будет записана в начало файла, если только вы не открыли файл с помощью команды file_text_open_append(). Вы также можете избежать этого, используя file_text_readln() вместе с функцией file_text_eof() чтобы просмотреть содержимое файла до конца, а затем начать запись. Важно отметить, что при записи очень больших чисел в текстовый файл с помощью этой функции они могут быть переведены в стандартный упрощенный формат, например, "6.6624e+003", который не может быть правильно считан обратно в GameMaker Studio 2. Чтобы избежать подобных проблем, вместо этого следует преобразовать значение в строку и использовать функцию file_text_write_string() вместо нее. Синтаксис: file_text_write_real(fileid, val); Аргумент Описание fileid Идентификатор файла для редактирования. val Реальное значение для записи в файл. Возвращается: Реально Пример: var file = file_text_open_write(working_directory + "hiscore.txt"); for (var i = 0; i < 10; ++i;) { file_text_write_real(file, scr[i]); file_text_writeln(file); file_text_write_string(file, scr_name[i]); file_text_writeln(file); } file_text_close(file); Приведенный выше код открывает файл для записи, а затем в цикле просматривает два массива, записывая каждое значение массива в новую строку файла.
