файл_текст_закрыть После завершения работы с файлом (чтения из него или записи в него) вы должны снова закрыть файл, иначе вы рискуете потерять содержащуюся в нем информацию. Это также предотвращает утечку памяти и гарантирует, что вы никогда не превысите лимит файлов, если открыто более 32 файлов. Синтаксис: file_text_close(fileid); Аргумент Описание fileid Идентификатор файла, который нужно закрыть. Возвращается: Н/Д Пример: var file = file_text_open_write(working_directory + "Game_Data.txt"); while (!file_text_eof(file)) { file_text_readln(file); } file_text_write_string(file, level_data); file_text_close(file); Приведенный выше код открывает файл для записи, затем циклически просматривает строки текста, уже записанные в файл, пока не дойдет до конца. В этот момент он записывает строку, а затем снова закрывает файл. Назад: Текстовые файлы Далее: file_text_open_read © Copyright YoYo Games Ltd. 2021 Все права защищены
