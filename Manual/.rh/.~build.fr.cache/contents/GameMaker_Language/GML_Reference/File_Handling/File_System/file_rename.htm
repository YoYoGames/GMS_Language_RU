переименование файла Эта функция переименует указанный файл с указанным именем. Функция вернет true, если файл был успешно переименован, или false при любых других обстоятельствах. ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации. Синтаксис: file_rename(oldname, newname); Аргумент Описание oldname Имя файла, который нужно изменить. новое имя Новое имя, которое будет присвоено файлу. Возвращается: Булево Пример: if file_exists("level1.txt") { file_rename("level1.txt", "level.txt"); } Это позволит проверить наличие файла и, если он существует, переименовать его.
