get_save_filename Эта функция открывает диалог и запрашивает у игрока имя файла для сохранения с заданным фильтром. Фильтр имеет вид " name1|mask1|name2|mask2|... ", где маска может содержать различные варианты с точкой с запятой между ними и вы также можете использовать "*" для обозначения любой строки. Например: " bitmaps|*.bmp;*.wmf", попросит пользователя выбрать имя растрового файла из системы, но только тех, которые имеют два указанных расширения (bmp и png). эта функция не сохраняет сам файл! Она только возвращает строку с полным именем и путем к файлу. Если пользователь нажимает кнопку "Отмена", возвращается пустая строка "". При использовании этой функции важно отметить, что она предоставляет вам определенные права доступа к этому файлу на время игры. Таким образом, получив путь к файлу от пользователя, вы можете обращаться к нему снова и снова без необходимости спрашивать. разрешение в равной степени распространяется на чтение и запись. ПРИМЕЧАНИЕ: Эта функция действует только для целей Windows и macOS. Синтаксис: get_save_filename(filter, fname); Аргумент Описание фильтр Фильтр типа файла и/или его имени. fname Предлагаемое имя файла для использования. Возвращается: Строка Пример: var file; file = get_save_filename("screenshot|*.png", ""); if file != "" { screen_save(file); } Приведенный выше код попросит пользователя выбрать имя файла изображения для сохранения, затем проверит возвращаемое имя файла и путь, и если это не пустая строка, то сделает снимок экрана и сохранит его, используя заданное имя.
