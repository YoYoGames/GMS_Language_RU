Файловая система В GameMaker Studio 2 существует ряд функций, которые позволяют манипулировать файлами всех типов различными способами: файл_существует файл_удалить переименование файла копирование_файла файл_найти_первым файл_найти_следующий найти_файл_закрыть атрибуты файла Для изменения имен файлов можно использовать следующие функции: ПРИМЕЧАНИЕ: Эти функции не работают с реальными файлами! Они работают только со строками, связанными с различными элементами пути, и должны использоваться вместе с другими файловыми функциями. имя_файла имя_файла имя_дира имя_диска имя_текста filename_change_ext Наконец, есть несколько специальных функций для получения имен файлов, которые можно использовать для загрузки и сохранения файлов с фильтрами типа файла. Эти функции открывают окно проводника и тем самым заставляют игру потерять фокус. Это означает, что любой ввод и поэтому мы рекомендуем (при использовании этих функций в событии клавиатуры) использовать событие " вверх", а не "вниз" или "нажато". ПРИМЕЧАНИЕ: Эти функции Windows и macOS только. ВНИМАНИЕ! Эти функции могут работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"!
