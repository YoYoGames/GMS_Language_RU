имя_дира Эта функция возвращает часть каталога указанного имени файла, которая обычно совпадает с путем, за исключением последней обратной косой черты. Синтаксис: filename_dir(fname); Аргумент Описание fname Файл для использования. Возвращается: Строка Пример: dir = filename_dir("Test.ini"); Приведенный выше код получает каталог (в виде строки) указанного файла.
