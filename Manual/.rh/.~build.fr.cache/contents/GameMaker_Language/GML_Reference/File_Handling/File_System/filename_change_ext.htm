filename_change_ext Эта функция возвращает указанное имя файла с расширением (включая точку), измененным на новое расширение. Используя пустую строку в качестве нового расширения, вы можете полностью удалить часть расширения. Синтаксис: filename_change_ext(fname, newext); Аргумент Описание fname Файл для использования. newext Новое расширение для использования. Возвращается: Строка Пример: ext = filename_change_ext(file_find_first(working_directory + "*.*", 0), ""); Приведенный выше код получает имя файла (в виде строки) с удаленной частью расширения.
