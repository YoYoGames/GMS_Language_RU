имя_файла Использование этой функции возвращает именную часть указанного файла с расширением, но без пути Синтаксис: filename_name(fname); Аргумент Описание fname Файл для использования. Возвращается: Строка Пример: name = filename_name(file_find_first("C:/Games/*.doc", 0)); Приведенный выше код получает имя (в виде строки) первого файла типа "doc", найденного в указанном каталоге.
