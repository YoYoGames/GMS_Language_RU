файл_существует Эта функция вернет true, если указанный файл существует, и false, если не существует. Обратите внимание, что функция может быть использована только для проверки локальных файлов, но не любых файлов, хранящихся на удаленном сервере. Синтаксис: file_exists(fname); Аргумент Описание fname Имя файла для проверки. Возвращается: Булево Пример: if file_exists("level.txt") { file = file_text_open_read("level.txt"); } Это позволит проверить наличие файла и, если он существует, открыть его для чтения.
