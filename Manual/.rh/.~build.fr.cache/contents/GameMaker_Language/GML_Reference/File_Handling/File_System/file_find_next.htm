файл_найти_следующий Эта функция возвращает имя следующего файла, который удовлетворяет заданной ранее маске и атрибутам (определенным через file_find_first()). Если такого файла не существует, возвращается пустая строка. ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации. ПРИМЕЧАНИЕ: Эта функция работает на всех целевых платформах C++(Windows, Mac, iOS, Android и WinPhone), НО флаги фильтрации работают только на Windows. Синтаксис: file_find_next(); Возвращается: Строка Пример: file = file_find_next(); Это вернет следующий файл, который соответствует ранее заданным атрибутам.
