имя_текста Эта функция возвращает часть расширения указанного имени файла, включая ведущую точку. Синтаксис: filename_ext(fname); Аргумент Описание fname Файл для использования. Возвращается: Строка Пример: ext = filename_ext(file_find_first("*.*", 0)); Приведенный выше код получает расширение (в виде строки) указанного файла.
