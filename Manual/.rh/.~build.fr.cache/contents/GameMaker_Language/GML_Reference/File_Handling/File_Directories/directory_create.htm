каталог_создать Эта функция создаст каталог с заданным именем в области сохранения. ПРИМЕЧАНИЕ: Эта функция не будет работать на HTML5, так как вы не можете создавать или удалять каталоги в локальном хранилище браузера. ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации. Синтаксис: directory_create(dname) Аргумент Описание dname Имя создаваемого каталога. Возвращается: Н/Д Пример: if !directory_exists("Games") { directory_create("Games"); } Это позволит проверить, существует ли указанный каталог в локальной папке данных, и, если нет, создать его для вас.
