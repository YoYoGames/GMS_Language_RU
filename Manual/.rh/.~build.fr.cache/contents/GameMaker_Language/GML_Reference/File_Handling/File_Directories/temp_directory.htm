каталог temp_directory Это можно использовать для возврата временного каталога, создаваемого для вашей игры при каждом запуске (корень не содержит конечного "\"). Эта директория будет содержать файлы и может быть доступна во время работы игры, но она будет удалена (вместе со всеми файлами, содержащимися в ней), когда игра будет закрыта. вместе со всеми файлами, которые он содержит), когда игра будет закрыта. ВНИМАНИЕ! Эта функция может работать не так, как вы ожидаете, из-за того, что GameMaker Studio 2 находится в "песочнице"! Пожалуйста, обратитесь к разделу о файловой системе для получения дополнительной информации. Синтаксис: каталог temp_directory Возвращается: Строка Пример: ini_open(temp_directory + "\temp_ini.ini"); Это откроет ini-файл во временном каталоге игры (создав его, если он еще не существует).
