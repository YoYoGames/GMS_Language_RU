zip_unzip Эта функция откроет сохраненный zip-файл и извлечет его содержимое в указанный каталог. Обратите внимание, что если вы не указали полный путь к ZIP-директории, то будет использован корень текущего диска, а если вы хотите поместить его в относительный путь к рабочему каталогу пакета игры, то следует использовать переменную рабочий_каталог как часть пути (относительные пути с использованием "." или "." не будут работать так, как ожидается поэтому их следует избегать). Обратите внимание, что zip-файл должен быть либо частью игрового пакета (т.е. включенным файлом), либо загружен в область хранения с помощью http_get_file(). Функция вернет значение, указывающее на количество извлеченных файлов, или вернет 0 или меньше, если извлечение не удалось. Синтаксис: zip_unzip(zip_file, target_directory) Аргумент Описание zip_file zip-файл для открытия целевой_каталог Целевой каталог для извлечения файлов Возвращается: Реально Пример: var num = zip_unzip("/downloads/level_data.zip", working_directory + "extracted/"); if num <= 0 { show_debug_message("Extraction Failed!"); } Приведенный выше код откроет zip-файл, хранящийся в каталоге "downloads", и извлечет его содержимое в каталог "extracted" (создав этот каталог, если он еще не существует), а затем проверит правильность извлечения. и покажет отладочное сообщение в случае неудачи.
