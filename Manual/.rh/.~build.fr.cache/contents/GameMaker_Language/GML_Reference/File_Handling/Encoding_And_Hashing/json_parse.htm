json_parse Эта функция может быть использована для разбора строки JSON (созданной ранее с помощью json_stringify или из любого корректного источника) и преобразования ее в коллекцию массивов или структур, где массив эквивалентен массиву JSON, а структура - объекту JSON. JSON-массив, а struct - эквивалент JSON-объекта. Вы предоставляете строку для разбора, и функция возвращает массив или структуру верхнего уровня, которые затем можно проверить в вашем коде. Если вы не уверены в содержимом JSON, вы можете использовать различные функции переменных (например typeof() или variable_struct_get_names()) для проверки возвращаемого содержимого. Обратите внимание, что при попытке разобрать недопустимое значение (т.е. не строку) возникнет ошибка исключения. Если предоставленная строка JSON содержит "undefined" в качестве значения для любого свойства, то при разборе она будет преобразована в pointer_null.
