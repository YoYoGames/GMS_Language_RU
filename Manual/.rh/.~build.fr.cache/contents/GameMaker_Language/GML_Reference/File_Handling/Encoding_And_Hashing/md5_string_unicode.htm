md5_string_unicode В криптографии MD5 (Message-Digest algorithm 5) - это широко используемая криптографическая хэш-функция со 128-битным хэш-значением, которая применяется в самых разных приложениях безопасности. Она также часто используется для проверки целостности файлов и строк. Эта функция принимает на вход строку unicode (16 бит для каждого символа) и возвращает 32-символьный шестнадцатеричный хэш MD5, уникальный для этой строки. Таким образом, вы можете сгенерировать безопасный ключ, который можно хранить и использовать для проверки целостности информации, отправляемой на внешний сервер (или получаемой от него) (например). ПРИМЕЧАНИЕ: Существует два формата кодировки MD5 - UTF-8 и unicode. Оба формата предусмотрены для облегчения взаимодействия с различными настройками сервера, но чаще всего используется unicode. ПРИМЕЧАНИЕ: MD5 не является полностью безопасным и может быть взломан. См. эту страницу для получения дополнительной информации. Синтаксис: md5_string_unicode(string) Аргумент Описание строка Строка для хэширования. Возвращается: Строка Пример: var hash, str; str = base64_encode(game_data); hash = md5_string_unicode(str); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?hash=" + hash); http_get("http://www.MacSweeneyGames.com/CatchTheHaggis/gamedata?data=" + str); Приведенный выше код кодирует строку в base64, а затем генерирует хэш MD5.
