sha1_file В криптографии SHA-1 - это криптографическая хэш-функция, разработанная Агентством национальной безопасности США и применяемая в нескольких широко используемых приложениях, популярных Git где она используется для проверки изменений файлов, а также в протоколах TLS и SSL, PGP, SSH, S/MIME и IPsec. Эта функция принимает входной файл и возвращает 160-битный дайджест сообщения в формате ASCII, уникальный для этого файла, который можно использовать для проверки целостности в любое время. ПРИМЕЧАНИЕ: SHA-1 не является полностью безопасным и может быть взломан. См. эту страницу для получения дополнительной информации. Синтаксис: sha1_file(filename) Аргумент Описание имя файла Файл, для которого нужно сгенерировать хэш sha1. Возвращается: Строка Пример: hash = sha1_file(working_directory + "game_data.ini") Приведенный выше код сгенерирует хэш sha1 для указанного файла и сохранит возвращаемое значение в переменной "hash".
