cloud_file_save Эта функция фиксирует файл в выбранной облачной службе для хранения. Функция возвращает уникальное значение id, которое затем должно быть использовано в соответствующем асинхронном событии для идентификации карты DS, возвращаемой в качестве "обратного вызова". от облачной службы. Файл должен содержать всю информацию, которую вам нужно сохранить для вашей игры, поскольку вы можете сохранить в облаке только один единственный "блок данных", и повторный запуск этой функции перезапишет любые ранее сохраненные значения (как и при использовании cloud_string_save() функция). Описание должно представлять собой короткую строку информации, описывающую сохранение, например: "Level2, Stage2". Для получения дополнительной информации о возвращаемых асинхронных данных, пожалуйста, обратитесь к функции cloud_synchronise(). Синтаксис: cloud_file_save(file, description); Аргумент Описание строка Файл для загрузки (в виде строки). описание Краткое описание хранимых данных. Возвращается: Реально Пример: var t_str = ""; for (var i = 0; i < 10; i++;) { t_str += string(global.Highscore[i]) + "|" } var file = file_text_open_write("Highscores.txt"); file_text_write_string(file, t_str); file_text_close(file); save_check = cloud_file_save("Highscores.txt", "Current Highscores"); Приведенный выше код создает строку из значений, хранящихся в глобальном массиве "Highscores", а затем записывает эту строку в файл для локального хранения.
