Facebook Facebook является одной из самых важных социальных сетей в Интернете, и многие игры и разработчики в настоящее время имеют свои собственные страницы для размещения игр и медиа. В GameMaker Studio 2 есть расширение, позволяющее подключиться к Facebook и выполнить определенные действия и задачи, которые позволяет выполнять их API. Перед началом использования расширения рекомендуется посетить страницу Facebook Developers, чтобы иметь представление о том, как работает Facebook Graph и доступные вызовы и задачи. Facebook Graph и доступные вызовы и разрешения необходимы для получения максимальной отдачи от этих функций. Вы можете узнать, как настроить и использовать расширение Facebook для всех доступных платформ по следующей ссылке: Android, iOS и HTML5: Интеграция Facebook Ниже перечислены устаревшие функции, которые были действительны только для устаревшего Facebook API. Поэтому вы больше не можете использовать эти функции в своих проектах, и они перечислены здесь на случай, если вы импортировали старый проект с этими функциями. Для текущей функциональности Facebook вам необходимо активировать Facebook API в соответствующих опциях игры для целевой платформы, что приведет к необходимости загрузить и установить расширение. Расширение поставляется с "мини-руководством" в формате *.pdf, объясняющим, как его использовать, а также показывающим все определения функций и примеры. Все ссылки на старые функции facebook_*() должны быть заменены (и обновить), чтобы использовать эквиваленты расширения fb_*().
