push_cancel_local_notification Эта функция отменяет данное push-уведомление. Вы можете получить идентификатор уведомления для отмены с помощью функций push_get_first_local_notification() и push_get_next_local_notification() и функция вернет true в случае успеха или false в противном случае (например, если уведомление не существует или уже было запущено). ПРИМЕЧАНИЕ: Эта функция ограничена iOS и Android целевыми модулями. Синтаксис: push_cancel_local_notification(ID); Аргумент Описание ID Идентификатор уведомления, которое необходимо отменить. Возвращается: Реально Пример var map = ds_map_create(); var ntf = push_get_first_local_notification(map); while(ntf >= 0) { var data = ds_map_find_value(map, "data"); if data == "Daily_Reward" { push_cancel_local_notification(ntf); } ntf = push_get_next_local_notification(map); } ds_map_destroy(map); Приведенный выше код создает карту DS Map, а затем запрашивает данные для первого доступного уведомления в очереди. Если уведомление существует, карта заполняется, а затем код проверяет ключ "data" на наличие определенной строки. Если строка найдена, уведомление отменяется и переходит к проверке следующего уведомления, пока не будут проверены все уведомления в очереди.
