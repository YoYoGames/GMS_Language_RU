прирост_достижений В играх могут быть достижения, не имеющие значения завершения, поэтому вы можете использовать эту функцию для увеличения этих достижений на заданную сумму. На Android вы отправляете ID достижения в виде строки (это уникальный ID достижения, который был присвоен, когда вы установили достижение), в то время как на всех других платформах вы передаете определенное имя достижения в виде строки, а затем указываете фактическое значение достижения для увеличения. Функция запустит социальное асинхронное событие, в котором возвращаемая async_load DS map будет содержать следующие пары ключ/значение: " type"- Это тип события, которое было запущено,для данной функции это будет строка "achievement_increment_result". " id" - Это идентификатор события, и для данной функции он возвращает GML-константу GooglePlayServices_EVENT_ID_INCREMENT_ACHIEVEMENT . "achievement_id" - Это уникальный ID достижения, которое было увеличено. "status" - Будет равно 1, если достижение было увеличено успешно, или 0, если запрос не был выполнен. "complete" - значение будет равно 1, если достижение было выполнено по данному инкременту, или 0, если запрос не был выполнен. Синтаксис: achievement_increment(name, value) Аргумент Описание имя Имя/идентификатор достижения. значение Значение, которое будет отправлено в качестве приращения. Возвращается: Н/Д Пример: achievement_increment(global.Achievement[0], score); Приведенный выше код добавляет текущий результат игрока в достижение с ID, сохраненным в глобальном массиве.
