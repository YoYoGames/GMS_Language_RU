достижение_поста Вы можете использовать эту функцию для отправки ваших достижений в выбранный сервис таблиц лидеров и достижений. На Android вам понадобится ID достижения в виде строки (это уникальный ID достижения, который был присвоен при установке достижения), а на всех остальных платформах - название достижения в виде строки, а также процент выполнения достижения (0 - нет, 100 - выполнено). Функция вызовет социальное асинхронное событие, в котором возвращаемая async_load DS map будут следующие пары ключ/значение: " type"- Это тип события, которое было инициировано,для данной функции это будет строка "achievement_post_result". " id" - Это идентификатор события, и для данной функции он возвращает GML-константу GooglePlayServices_EVENT_ID_POST_ACHIEVEMENT . "achievement_id" - Это уникальный идентификатор достижения, которое было обновлено. "status" - Значение будет равно 1, если достижение было обновлено успешно, или 0, если запрос не был выполнен. Синтаксис: achievement_post(achievement, percent) Аргумент Описание достижение Имя/идентификатор достижения. процент Процент выполнения достижения. Возвращается: Н/Д Пример: if achievement_available() { achievement_post(global.Achievement[0], 100); } Приведенный выше код проверяет, доступен ли выбранный сервис таблиц лидеров и достижений, и если да, то публикует достижение.
