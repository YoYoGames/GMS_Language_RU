сброс_достижений Эта функция вернет все достижения к их начальным значениям для данной игры. Эта функция предоставляется в качестве отладочной функции, и не рекомендуется разрешать конечному пользователю делать это в своих играх. Синтаксис: achievement_reset() Возвращается: Н/Д Пример: if debug_mode { if keyboard_check_pressed(vk_space) { achievement_reset() } } Приведенный выше код проверяет, находится ли игра в режиме отладки, и если да, то нажатие "пробела" сбросит все достижения.
