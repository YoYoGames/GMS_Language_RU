магазин_оставить_рейтинг Эта функция открывает диалог, зависящий от ОС, в котором вы можете попросить пользователя разместить оценку или комментарий к определенной странице. Вы можете задать текст, который будет отображаться в диалоге, а также текст, который будет отображаться на двух кнопках и URL-адресе. где должен быть размещен комментарий. ПРИМЕЧАНИЕ: Эта функция предназначена только для целей Android и iOS. ВАЖНО! На сайте iOS, при использовании TestFlight, в процессе разработки подсказка будет отображаться, но вы не сможете ничего сделать. Как только ваше приложение появится в App Store, эта кнопка станет активной, и пользователю будет предоставлена возможность написать отзыв или оставить оценку. Обратите внимание, что если вы распространяете игру через TestFlight, это окно может даже не отображаться. Синтаксис: shop_leave_rating(text, yes_string, no_string, url) Аргумент Описание текст Текст, который отображается в диалоговом окне. да_строка Текст, который появляется на кнопке "Да". no_string Текст, который появляется на кнопке "нет". url Полный URL-адрес, по которому должен быть отправлен комментарий. Возвращается: Н/Д Пример: if timer <= 0 { shop_leave_rating("Спасибо за игру! Почему бы не оставить комментарий?", "Хорошо!", "Не сегодня!", "http://MacSweeney/comments"); timer = 10000000; } else { таймер -= 1; } Приведенный выше код попросит пользователя оставить комментарий, если переменная "timer" досчитала до 0.
