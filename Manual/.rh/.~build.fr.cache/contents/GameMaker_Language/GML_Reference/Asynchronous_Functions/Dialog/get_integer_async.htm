get_integer_async Эта функция открывает окно и отображает сообщение, а также место для ввода пользователем значения (которое для начала будет содержать заданное по умолчанию значение). Это асинхронная функция, поэтому GameMaker Studio 2 не блокирует устройство, на котором она выполняется, в ожидании выполнения. устройство, на котором она выполняется, в ожидании ответа, а продолжает выполнять события в обычном режиме. После того как пользователь набрал свою строку и нажал кнопку "Ок", запускается асинхронное событие Диалог срабатывает асинхронное событие, которое только на время этого события будет иметь карту DS, хранящуюся в переменной async_load. Эта карта будет содержать три ключа: "id","status" и "value". "id" - этозначение, котороебыло возвращено функцией при вызове, "status" будет либо true, если кнопка "Ок" была нажата, либо false, если сообщение было отменено (где это применимо, поскольку не все целевые платформы позволяют отменять сообщения). Наконец, " value" вернет целое число, которое пользователь ввел в виде строки (пустая строка "" будет возвращена, если ввода не было). ПРИМЕЧАНИЕ: Эта функция предназначена только для отладки и тестированияи не должна использоваться в выпущенных играх. Для этих целей вы должны создать пользовательский интерфейс для получения ввода от игроков, чтобы иметь полный контроль над тем, как выглядят и ведут себя диалоги. Синтаксис: get_integer_async(string, default); Аргумент Описание Строка Сообщение, которое будет показано в диалоге. по умолчанию Значение по умолчанию.
