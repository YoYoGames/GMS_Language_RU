mp_grid_clear_cell Эта функция может быть использована для очистки определенной "ячейки" сетки MP. Ячейки рассчитываются не как координаты помещения, а как координаты сетки, где (0,0) - верхний левый угол сетки. это означает, что для очистки ячейки в определенном в комнате, мы должны изменить координаты x и y на координаты ячейки, разделив их на разрешение сетки MP. Пример кода ниже показывает, как это работает. Синтаксис: mp_grid_clear_cell(id, h, v); Аргумент Описание id Индекс mp_grid, который будет использоваться h Горизонтальное положение ячейки для очистки v вертикальное положение ячейки, которую нужно очистить Возвращается: Булево Пример: with (obj_Box) { mp_grid_clear_cell(grid, floor(x / 32), floor(y /32)); instance_destroy(); } Приведенный выше код заставит все "obj_Box" уничтожить себя и пометить ячейки, которые они занимали в mp_grid, проиндексированные в переменной "grid", как свободные. В этом примере мы находим соответствующую ячейку, взяв координаты x/y объекта и разделив их на разрешение сетки (используя floor, чтобы сохранить значения в виде целых чисел).
