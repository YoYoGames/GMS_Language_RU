mp_potential_step_object Эта функция позволяет экземпляру сделать шаг к определенной позиции, заданной xgoal/ygoal, стараясь при этом избежать препятствий. Когда объект столкнется с объектом, указанным в аргументе "obj", он изменит направление движения, чтобы попытаться обойти этот объект. направление движения, чтобы попытаться избежать этого объекта и двигаться вокруг него. Не гарантируется, что этот подход сработает, но в большинстве простых случаев он будет эффективно перемещать экземпляр к цели. Функция возвращает, была ли цель достигнута или нет. Синтаксис: mp_potential_step_object(xgoal, ygoal, stepize, obj) Аргумент Описание xgoal Положение цели x. ygoal Положение цели по оси y. размер шага Скорость перемещения экземпляра в пикселях за шаг. obj Объект, который должен блокировать путь экземпляра, выполняющего функцию. Это может быть индекс объекта, идентификатор экземпляра или специальное ключевое слово,all. Возвращается: Булево Пример: if instance_exists(obj_Enemy) { var inst; inst = instance_nearest(x, y, obj_Enemy); mp_potential_step_object(inst.x, inst.y, 5, obj_Wall); } Приведенный выше код сначала проверяет, есть ли в комнате экземпляры "obj_Enemy". Если таковые имеются, он находит ближайший из них и сохраняет его идентификатор в переменной. Эта переменная затем используется, чтобы сказать mp_potential_step_object переместить экземпляр с кодом в направлении x/y позиции объекта, который был найден, со скоростью 5 пикселей за шаг, избегая при этом только экземпляры объекта "obj_Wall".
