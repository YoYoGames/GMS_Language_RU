mp_linear_path Эта функция рассчитывает прямолинейный путь от текущей позиции экземпляра до точки, заданной значениями "xgoal" и "ygoal". Она использует указанный размер шага, такой же, как в функции mp_linear_step(). Указанный путь должен уже существовать и будет перезаписан новым путем, а функция вернет, был ли найден полный путь (true) или нет (false). Если возвращается false, то путь все равно будет сгенерирован, но он будет проходить только до позиции где путь был заблокирован. ПРИМЕЧАНИЕ: Эта функция не перемещает экземпляр. Она только задает путь, а для перемещения необходимо использовать функции Path. Синтаксис: mp_linear_path(path, xgoal, ygoal, stepize, checkall); Аргумент Описание путь Индекс пути, который будет использоваться. xgoal Положение цели x. ygoal Положение цели по оси y. размер шага Скорость перемещения экземпляра в пикселях за шаг. checkall Проверять ли столкновения со всеми экземплярами (true) или только со сплошными (false). Возвращается: Булево Пример: if mp_linear_path(path, obj_Player.x, obj_Player.y, 2, 0) { path_start(path, 2, 0, 0); } Это заставляет объект проверить, существует ли линейный путь от его текущей позиции до позиции "obj_Player".
