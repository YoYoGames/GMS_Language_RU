mp_grid_add_rectangle Эта функция просит вас определить прямоугольник в пределах комнаты, а затем помечает все ячейки сетки MP, "касающиеся" этого прямоугольника, как запрещенные, что означает, что функции поиска пути не могут пересечь их. На рисунке ниже показано, как это работает: Как вы видите, прямоугольник, определенный от (50, 90) до (200, 180), отмечает все эквивалентные ячейки сетки MP, которых он касается, как запрещенные. ячейки сетки MP, которых он касается, как запрещенные. Синтаксис: mp_grid_add_rectangle(id, x1, y1, x2, y2); Аргумент Описание id Индекс mp_grid, который будет использоваться x1 Координата x левой стороны прямоугольника для проверки. y1 Координата y верхней стороны прямоугольника для проверки. x2 Координата x правой стороны прямоугольника для проверки. y2 Координата x нижней стороны прямоугольника для проверки. Возвращается: Н/Д Пример: mp_grid_add_rectangle(grid, 0, 0, 100, 200); Приведенный выше код пометит как запрещенные все ячейки mp_grid, индексированные в переменной "grid", которые попадают в область от 0,0 до 100,200.
