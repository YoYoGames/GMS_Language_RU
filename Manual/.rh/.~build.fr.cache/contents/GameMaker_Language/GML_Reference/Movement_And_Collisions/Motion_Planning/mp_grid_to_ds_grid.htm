mp_grid_to_ds_grid С помощью этой функции вы можете скопировать заданную MP-сетку в DS-сетку. DS-сетка должна быть того же размера, что и MP-сетка, хотя это не обязательно (данные будут потеряны, если она будет меньше, и если она будет больше, то все лишние ячейки сетки будут равны 0). Ячейки сетки DS будут содержать значение -1, если ячейка сетки MP была отмечена как занятая, или 0, если нет. Синтаксис: mp_grid_to_ds_grid(source, destination); Аргумент Описание источник Индекс mp_grid, который будет использоваться направление Индекс ds_grid, в которую будут скопированы данные сетки. Возвращается: Булево Пример: motion_grid = ds_grid_create(room_width / 32, room_height / 32); mp_grid_to_ds_grid(mp_grid, motion_grid); Приведенный выше код создаст новую сетку DS, а затем скопирует данные сетки MP, содержащиеся в переменной "mp_grid", в новую сетку DS.
