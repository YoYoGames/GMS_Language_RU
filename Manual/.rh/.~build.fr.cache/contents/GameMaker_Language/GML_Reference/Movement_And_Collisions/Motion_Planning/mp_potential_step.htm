mp_potential_step Эта функция позволяет экземпляру сделать шаг к определенной позиции, заданной xgoal/ygoal, стараясь при этом избежать препятствий. Когда экземпляр столкнется с твердым объектом (или с любым объектом, когда checkall равен true), он изменит направление движения, чтобы попытаться обойти этот объект. движения, чтобы попытаться избежать этого объекта и двигаться вокруг него. Не гарантируется, что этот подход сработает, но в большинстве простых случаев он будет эффективно перемещать экземпляр к цели. Функция возвращает, была ли цель достигнута или нет. Синтаксис: mp_potential_step(xgoal, ygoal, stepize, checkall) Аргумент Описание xgoal Положение цели x. ygoal Положение цели по оси y. размер шага Скорость перемещения экземпляра в пикселях за шаг. checkall Проверять ли все экземпляры (true) или только твердые (false). Возвращается: Булево Пример: if instance_exists(obj_Enemy) { var inst; inst = instance_nearest(x, y, obj_Enemy); mp_potential_step(inst.x, inst.y, 5, false); } Приведенный выше код сначала проверяет, есть ли в комнате экземпляры "obj_Enemy". Если таковые имеются, он находит ближайший из них и сохраняет его идентификатор в переменной. Эта переменная затем используется для того, чтобы указать mp_potential_step переместить экземпляр с кодом в направлении x/y позиции объекта, который был найден, со скоростью 5 пикселей за шаг, избегая при этом только объектов, помеченных как твердые.
