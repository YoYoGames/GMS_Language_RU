motion_set Эта функция задает новое направление движения и новую скорость для экземпляра, выполняющего код. Обратите внимание, что она не добавляет к текущей скорости и направлению движения (для этого используется функция motion_add()) , а скорее принудительно устанавливает новые параметры. Синтаксис: motion_set(dir, speed); Аргумент Описание dir Новое направление. скорость Новая скорость.` Возвращается: Н/Д Пример: if irandom(9) = 1 { motion_set(random(360), 1 + random(3)); } Приведенный выше код заставит экземпляр менять скорость и направление движения через случайные промежутки времени.
