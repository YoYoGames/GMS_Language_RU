перемещение_вне_твердого С помощью этой функции можно приказать экземпляру выйти из столкновения с любым экземпляром, отмеченным как solid в любом направлении и на любое количество пикселей каждый шаг, при этом Значение -1 или 0 для maxdist по умолчанию равно 1000px, т.е. GameMaker Studio 2 будет постоянно перемещать экземпляр на 1000 пикселей вверх, пока он не выйдет из столкновения. Синтаксис: move_outside_solid(dir, maxdist); Аргумент Описание dir Направление, в котором нужно двигаться. maxdist Максимальное расстояние, которое может пройти объект (-1 или 0, значение по умолчанию - 1000 пикселей). Возвращается: Н/Д Пример: if other.solid { var pdir; pdir = point_direction(other.x, other.y, x, y); move_outside_solid(pdir, -1); } Приведенный выше код будет входить в событие столкновения и проверять, отмечен ли "другой" экземпляр в столкновении как "твердый". Если это так, то он переместит экземпляр за пределы столкновения. Назад: Движение Далее: move_random © Copyright YoYo Games Ltd. 2021 Все права защищены
