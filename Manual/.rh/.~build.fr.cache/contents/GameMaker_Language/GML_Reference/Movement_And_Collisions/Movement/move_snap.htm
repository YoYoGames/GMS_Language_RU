перемещение_защелки Эта функция используется для "привязки" экземпляра к сетке заданного размера. Он будет привязан к ближайшей соответствующей позиции на "невидимой" сетке, которую определяют значения hsnap и vsnap. Синтаксис: move_snap( hsnap, vsnap ); Аргумент Описание hsnap Горизонтальная привязка (размер в пикселях между "ячейками"). vsnap Вертикальная привязка (размер в пикселях между "ячейками"). Возвращается: Н/Д Пример: with (obj_Pieces) { if !place_snapped(32, 32) { move_snap(32, 32); } } Приведенный выше код проверяет все экземпляры "obj_Pieces" на то, привязаны ли они к сетке 32x32 пикселей, и если нет, то привязывает их к ближайшей позиции в этой сетке.
