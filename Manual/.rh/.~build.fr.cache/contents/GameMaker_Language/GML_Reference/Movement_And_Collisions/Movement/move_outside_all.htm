перемещение_вне_всех С помощью этой функции вы можете указать экземпляру выйти из столкновения в любом направлении и на любое количество пикселей за шаг, при этом значение -1 или 0 для maxdist по умолчанию равно 1000px, т.е.: GameMaker Studio 2 будет перемещать экземпляр постоянно вверх на 1000 пикселей, пока не выйдет из столкновения. Синтаксис: move_outside_all(dir, maxdist); Аргумент Описание dir Направление, в котором нужно двигаться. maxdist Максимальное расстояние, которое может пройти объект (-1 или 0, значение по умолчанию - 1000 пикселей). Возвращается: Н/Д Пример: if place_meeting(x, y, all) { move_outside_all(90, 1); } Приведенный выше код будет перемещать экземпляр на 1 пиксель вверх на каждом шаге, когда он столкнется с любым другим экземпляром.
