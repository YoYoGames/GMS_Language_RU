перемещение_отскока_всего С помощью этой функции вы можете приказать экземпляру отскакивать от всех экземпляров в комнате, за исключением тех, которым не назначен индекс спрайта или маски. Вы также можете указать ему использовать точную проверку столкновений, если она включена, но имейте в виду, что для этого необходимо, чтобы у всех экземпляров были включены точные маски, и это сильно замедлит игру, если в ней участвует много экземпляров, из-за большого количества обработки, которую необходимо выполнить. Обычно это должно происходить в событии шага для экземпляра, но может использоваться выборочно и в событии столкновения, как показано в примере кода ниже. Синтаксис: move_bounce_all( adv ); Аргумент Описание adv Включать ли расширенный отскок (true) или нет (false). Возвращается: Н/Д Пример: if other.visible { move_bounce_all(false); } Приведенный выше код будет помещен в событие столкновения с другим объектом. Сначала он проверяет, виден ли объект, и если виден, то выполняет действие move_bounce_all(). Обратите внимание, что в этом случае отскок является выборочным и будет вычислен только вычисляться только для этого столкновения, а не для каждого случая, каждого шага.
