точка_в_прямоугольнике При использовании этой функции вы задаете прямоугольную область, а GameMaker Studio 2 определяет, попадает ли заданная точка в ее пределы или нет. Если точка попадает в заданный прямоугольник, функция вернет true, в противном случае функция вернет false. Синтаксис: point_in_rectangle(px, py, x1, y1, x2, y2); Аргумент Описание px Координата x точки, которую нужно проверить. py Координата y точки для проверки. x1 Координата x левой стороны прямоугольника для проверки. y1 Координата y верхней стороны прямоугольника для проверки. x2 Координата x правой стороны прямоугольника для проверки. y2 Координата y нижней стороны прямоугольника для проверки. Возвращается: Булево Пример: if point_in_rectangle(mouse_x, mouse_y, x -10, y - 10, x + 10, y + 10) { audio_play_sound(snd_click, 0, false); } Этот короткий код проверяет положение мыши относительно заданной прямоугольной области и воспроизводит звук, если она попадает в границы.
