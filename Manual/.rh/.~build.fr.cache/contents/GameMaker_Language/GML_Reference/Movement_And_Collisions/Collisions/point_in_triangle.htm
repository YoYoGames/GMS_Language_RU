точка_в_треугольнике При использовании этой функции вы задаете треугольную область, а GameMaker Studio 2 определяет, попадает ли заданная точка в ее пределы или нет. Если точка попадает в заданный треугольник, функция вернет true, в противном случае функция вернет false. Синтаксис: point_in_triangle(px, py, x1, y1, x2, y2, x3, y3); Аргумент Описание px Координата x точки, которую нужно проверить. py Координата y точки для проверки. x1 Координата x первого угла треугольника для проверки. y1 Координата y первого угла треугольника для проверки. x2 Координата x второго угла треугольника для проверки. y2 Координата y второго угла треугольника для проверки. x3 Координата x третьего угла треугольника для проверки. y3 Координата y третьего угла треугольника для проверки. Возвращается: Булево Пример: var inst = instance_nearest(x, y, obj_Player); if instance_exists(inst) { var x1 = x + lengthdir_x(100, image_angle - 45); var y1 = y + lengthdir_y(100, image_angle - 45); var x2 = x + lengthdir_x(100, image_angle + 45); var y2 = y + lengthdir_y(100, image_angle + 45); if point_in_triangle(inst.x, inst.y, x, y, x1, y1, x2, y2) { can_see = true; } } Приведенный выше код использует функцию point_in_triangle как "конус зрения" для проверки экземпляра "obj_player", устанавливая переменную в true, если позиция объектов x/y попадает в определенный треугольник.
