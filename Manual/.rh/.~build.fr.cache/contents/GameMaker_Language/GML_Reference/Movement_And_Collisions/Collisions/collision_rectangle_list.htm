список_прямоугольников_коллизий Эта функция аналогична функции collision_rectangle() только вместо обнаружения одного экземпляра столкновения за один раз, она обнаруживает несколько экземпляров. Вы задаете позиции x/y верхнего левого и нижнего правого угла области для проверки вместе с объектом для проверки, а также можно задать точность проверки (в этом случае все проверяемые экземпляры должны иметь точные маски столкновений) и указать, должна ли проверка включать в себя вызывающий экземпляр или нет. Вы также предоставляете список DS, поэтому id значения всех экземпляров, которые сталкиваются с вызывающим экземпляром, будут будут добавлены в конец данного списка. Вы можете очистить список перед вызовом этой функции, чтобы он содержал только результаты этого вызова функции. У вас также есть возможность упорядочить на основании расстояния от центра прямоугольной области до места их возникновения. Функция возвращает количество найденных экземпляров или 0, если не найдено ни одного. Обратите внимание, что вместо индекса объекта можно использовать ключевое слово instance all, чтобы проверить все экземпляры в текущей комнате, которые могут включать экземпляр, выполняющий код (в зависимости от аргумента notme ). Синтаксис: collision_rectangle_list(x1, y1, x2, y2, obj, prec, notme, list, ordered); Аргумент Описание x1 Координата x левой стороны прямоугольника для проверки. y1 Координата y верхней стороны прямоугольника для проверки. x2 Координата x правой стороны прямоугольника для проверки. y2 Координата y нижней стороны прямоугольника для проверки. obj Объект для проверки столкновений экземпляров. prec Основана ли проверка на точных столкновениях(true, что медленнее) или на его граничной области в целом(false, быстрее). notme Должен ли вызывающий экземпляр, если он релевантен, быть исключен(true) или нет(false). список Список DS, используемый для хранения идентификаторов сталкивающихся экземпляров. заказал Должен ли список быть упорядочен по расстоянию(true) или нет(false). Возвращается: Integer (Число экземпляров, с которыми произошло столкновение) Пример: var _list = ds_list_create(); var _num = collision_rectangle_list(x - 100, y - 100, x + 100, y + 100, obj_Enemy, false, true, _list, false); if _num > 0 { for (var i = 0; i < _num; ++i;) { instance_destroy(_list[| i]); } } ds_list_destroy(_list); Код выше проверяет прямоугольную область в 100 пикселей вокруг позиции вызывающего экземпляра на наличие столкновений с экземплярами "obj_Enemy". Если есть какие-либо столкновения, то предварительно созданный список просматривается, и каждый экземпляр, который был в столкновении, уничтожается.
