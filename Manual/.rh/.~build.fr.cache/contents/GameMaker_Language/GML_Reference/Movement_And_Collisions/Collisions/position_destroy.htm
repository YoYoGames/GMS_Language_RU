уничтожение позиции Эта функция просто уничтожает все экземпляры, которые находятся в столкновении с указанной позицией. Столкновения основаны на маске экземпляров, и если какая-либо часть маски пересекается с целевой точкой, то функция уничтожит этот экземпляр. Экземпляры, уничтоженные таким образом, запускают свои события Destroy и Clean Up. Синтаксис: position_destroy(x, y); Аргумент Описание x Координата x места уничтожения сталкивающихся экземпляров. y Координата y места уничтожения сталкивающихся экземпляров. Возвращается: Н/Д Пример: if mouse_check_button_pressed(mb_left) { position_destroy(mouse_x, mouse_y); } Это уничтожит все экземпляры, которые сталкиваются с позицией курсора мыши при нажатии левой кнопки мыши.
