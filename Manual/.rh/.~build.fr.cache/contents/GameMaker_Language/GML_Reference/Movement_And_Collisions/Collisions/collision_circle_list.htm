список_кругов_коллизий Эта функция аналогична функции collision_circle() только вместо обнаружения одного экземпляра столкновения за один раз, она обнаруживает несколько экземпляров. Вы указываете x/y позицию центра круговой области для проверки, а также радиус и объект для проверки, и вы можете задать точность проверки (в этом случае все проверяемые экземпляры должны иметь точные маски столкновений) и указать, должна ли проверка включать в себя вызывающий экземпляр или нет. Вы также предоставляете список DS, поэтому id значения всех экземпляров, которые сталкиваются с вызывающим экземпляром, будут будут добавлены в конец данного списка. Вы можете очистить список перед вызовом этой функции, чтобы он содержал только результаты этого вызова функции. У вас также есть возможность упорядочить экземпляры на основе расстояний от начала круговой области до их начала. Функция возвращает количество найденных экземпляров или 0, если не найдено ни одного. Обратите внимание, что вместо индекса объекта можно использовать ключевое слово instance all, чтобы проверить все экземпляры в текущей комнате, которые могут включать экземпляр, выполняющий код (в зависимости от аргумента notme ). Синтаксис: collision_circle_list(x1, y1, rad, obj, prec, notme, list, ordered); Аргумент Описание x1 Координата x центра окружности для проверки. y1 Координата y центра круга для проверки. rad Радиус (расстояние в пикселях от центра до края). obj Объект для проверки столкновений экземпляров. prec Основана ли проверка на точных столкновениях(true, что медленнее) или на его граничной области в целом(false, быстрее). notme Должен ли вызывающий экземпляр, если он релевантен, быть исключен(true) или нет(false). список Список DS, используемый для хранения идентификаторов сталкивающихся экземпляров. заказал Должен ли список быть упорядочен по расстоянию(true) или нет(false). Возвращается: Int (Количество экземпляров, с которыми произошло столкновение) Пример: var _list = ds_list_create(); var _num = collision_circle_list(x, y, 100, obj_Enemy, false, true, _list, false); if (_num > 0) { for (var i = 0; i < _num; ++i;) { instance_destroy(_list[| i]); } } ds_list_destroy(_list); Приведенный выше код проверяет круговую область с радиусом 100 пикселей вокруг позиции вызывающего экземпляра на наличие столкновений с экземплярами "obj_Enemy". Если столкновения есть, то предварительно созданный список просматривается и каждый экземпляр который был в столкновении, уничтожается.
