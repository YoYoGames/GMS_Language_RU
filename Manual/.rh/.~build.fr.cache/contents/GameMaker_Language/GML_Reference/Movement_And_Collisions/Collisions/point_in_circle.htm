точка_в_круге При использовании этой функции вы задаете круглую область, а GameMaker Studio 2 определяет, попадает ли заданная точка в ее пределы или нет. Если точка попадает в заданную окружность, функция вернет true, в противном случае функция вернет false. функция вернет false. Синтаксис: point_in_circle(px, py, x1, y1, rad); Аргумент Описание px Координата x точки, которую нужно проверить. py Координата y точки для проверки. x1 Координата x центра окружности. y1 Координата y центра окружности. rad Радиус окружности. Возвращается: Булево Пример: if point_in_circle(mouse_x, mouse_y, x, y, 16) { over = true; } else { over = false; } Приведенный выше код использует функцию point_in_circle для проверки того, попадает ли позиция мыши в определенную круговую область, устанавливая переменную "over" в true, если попадает, или false в противном случае.
