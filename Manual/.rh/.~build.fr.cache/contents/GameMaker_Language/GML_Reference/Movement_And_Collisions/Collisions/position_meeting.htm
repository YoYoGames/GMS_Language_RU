позиция_совещания С помощью этой функции можно проверить позицию на столкновение с другим экземпляром или всеми экземплярами объекта. При использовании этой функции вы проверяете одну точку в комнате на наличие экземпляра или объекта. Проверка выполняется по ограничивающей рамки экземпляра или по маске экземпляра, если у этого экземпляра есть точная проверка столкновений. Если вам нужно получить уникальный экземпляр id объекта, с которым с которым происходит столкновение, следует использовать instance_position(). Эта функция также принимает специальные ключевые слова all и other. Синтаксис: position_meeting(x, y, obj); Аргумент Описание x Позиция x для проверки. y Позиция y для проверки. obj Объект (или идентификатор экземпляра, или ключевые слова "все" или "другие"), с которым нужно проверить столкновение. Возвращается: Булево Пример: if mouse_check_button(mb_left) { if !position_meeting(mouse_x, mouse_y, all) instance_create_layer(mouse_x, mouse_y, "Walls", obj_Wall); } Приведенный выше код проверяет наличие левой кнопки мыши, и если она нажата, то проверяет положение мыши x/y на предмет столкновения с каким-либо экземпляром.
