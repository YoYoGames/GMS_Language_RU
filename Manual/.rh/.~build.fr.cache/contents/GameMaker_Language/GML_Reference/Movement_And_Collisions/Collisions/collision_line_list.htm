список_линий_коллизий Эта функция аналогична функции collision_line() только вместо того, чтобы обнаруживать только один экземпляр столкновения за раз, она будет обнаруживать несколько экземпляров. Вы указываете x/y позиции начала и конец линии для проверки вместе с объектом для проверки, и вы можете задать точность проверки (в этом случае все проверяемые экземпляры должны иметь точные маски столкновений) и указать, должна ли проверка включать вызывающий экземпляр или нет. Вы также предоставляете список DS, поэтому id значения всех экземпляров, которые сталкиваются с вызывающим экземпляром, будут будут добавлены в конец данного списка. Вы можете очистить список перед вызовом этой функции, чтобы он содержал только результаты этого вызова функции. У вас также есть возможность упорядочить экземпляры на основе расстояния от центра линии до их начала. Функция возвращает количество найденных экземпляров или 0, если не найдено ни одного. Обратите внимание, что вместо индекса объекта можно использовать ключевое слово instance all, чтобы проверить все экземпляры в текущей комнате, которые могут включать экземпляр, выполняющий код (в зависимости от аргумента notme ). Синтаксис: collision_line_list(x1, y1, x2, y2, obj, prec, notme, list, ordered); Аргумент Описание x1 Координата x начала линии. y1 Координата y начала линии. x2 Координата x конца линии. y2 Координата y конца линии. obj Объект для проверки столкновений экземпляров. prec Основана ли проверка на точных столкновениях(true, что медленнее) или на его граничной области в целом(false, быстрее). notme Должен ли вызывающий экземпляр, если он релевантен, быть исключен(true) или нет(false). список Список DS, используемый для хранения идентификаторов сталкивающихся экземпляров. заказал Должен ли список быть упорядочен по расстоянию(true) или нет(false). Возвращается: Integer (Число экземпляров, с которыми произошло столкновение) Пример: var _list = ds_list_create(); var _num = collision_line_list(x - 100, y, x + 100, y, obj_Enemy, false, true, _list, false); if _num > 0 { for (var i = 0; i < _num; ++i;) { instance_destroy(_list[| i]); } } ds_list_destroy(_list); Приведенный выше код проверяет линию в 100 пикселей в обе стороны от позиции вызывающего экземпляра на наличие столкновений с экземплярами "obj_Enemy". Если столкновения есть, то предварительно созданный список просматривается по циклу и каждый экземпляр, который был в столкновении, уничтожается.
