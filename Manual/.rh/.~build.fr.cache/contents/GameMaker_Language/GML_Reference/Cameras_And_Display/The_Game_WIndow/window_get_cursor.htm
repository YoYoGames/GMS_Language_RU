window_get_cursor С помощью этой функции вы можете получить текущий курсор, используемый в окне игры, при этом возвращаемым значением будет любая из перечисленных ниже констант (для установки текущего курсора можно использовать функцию window_set_cursor() которая также принимает эти константы): Постоянная Курсор cr_none cr_default cr_arrow cr_cross cr_beam cr_size_nesw cr_size_ns cr_size_nwse cr_size_we cr_uparrow песочные часы cr_drag cr_appstart cr_handpoint cr_size_all Синтаксис: window_get_cursor(); Возвращается: Постоянная Пример: если mouse_check_button_pressed(mb_left) { if window_get_cursor() != cr_drag window_set_cursor(cr_drag); } Приведенный выше код изменит курсор окна на стандартный курсор перетаскивания окна, если была нажата левая кнопка мыши и он не был изменен ранее.
