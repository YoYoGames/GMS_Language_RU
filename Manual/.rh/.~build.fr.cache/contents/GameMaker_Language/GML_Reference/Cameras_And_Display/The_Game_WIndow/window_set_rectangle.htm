window_set_rectangle С помощью этой функции вы можете установить положение игрового окна в браузере (HTML5) или на экране(Windows, Ubuntu(Linux) или macOS) , а также задать масштаб окна. Для получения дополнительной информации о положении окна и его размере см. window_set_position() и window_set_size(). Синтаксис: window_set_rectangle(x, y, w, h); Аргумент Описание x Новая координата x окна. y Новая координата y окна. w Новая ширина окна. h Новая высота окна. Возвращается: Н/Д Пример: window_set_rectangle(0, 0, display_get_width(), display_get_height()); Приведенный выше код установит окно игры так, чтобы оно занимало всю область дисплея (либо браузер, либо экран, в зависимости от целевой платформы).
