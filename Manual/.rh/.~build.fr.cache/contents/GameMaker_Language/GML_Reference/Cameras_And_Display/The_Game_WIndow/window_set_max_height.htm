window_set_max_height Эта функция может быть использована для установки максимальной высоты окна вашей игры. Если вы включите опцию изменения размера окна в параметрах игры для целевой платформы, то игрок сможет изменить размер игрового окна до любого желаемого размера, однако с помощью этой функции вы можете ограничить максимальную высоту указанным вами размером. Если вы хотите вернуться к поведению по умолчанию (т.е. без минимума), то используйте значение -1. ПРИМЕЧАНИЕ: Эта функция доступна только на устройствах Windows, macOS и Linux платформах. Синтаксис: window_set_max_height(height); Аргумент Описание высота Максимальная высота в пикселях для игрового окна. Возвращается: Н/Д Пример: window_set_min_width(640); window_set_min_height(480); window_set_max_width(1280); window_set_max_height(960); Приведенный выше код установит минимальную и максимальную ширину и высоту для игрового окна.
