window_views_mouse_get_x Эта функция возвращает x-координату мыши относительно всех активных представлений и возвращает одно и то же значение mouse_x. ПРИМЕЧАНИЕ: Об обычных функциях мыши см. раздел " Ввод мыши". Синтаксис: window_views_mouse_get_x(); Возвращается: Реально Пример: myx = window_views_mouse_get_x(); Это установит myx в координату x мыши в окне относительно всех активных представлений.
