window_set_caption С помощью этой функции вы можете изменить или установить надпись окна для комнаты, в которой вы находитесь в данный момент. Эта надпись появляется в верхней части окна, рядом со значком игры, когда игра не находится в полноэкранном режиме. Синтаксис: window_set_caption(caption); Аргумент Описание надпись Новая надпись. Возвращается: Н/Д Пример: if window_get_caption() != "" { window_set_caption(""); } Приведенный выше код проверит надпись окна на наличие или отсутствие текста, и если текст есть, то установит его в пустую строку, чтобы надпись не отображалась.
