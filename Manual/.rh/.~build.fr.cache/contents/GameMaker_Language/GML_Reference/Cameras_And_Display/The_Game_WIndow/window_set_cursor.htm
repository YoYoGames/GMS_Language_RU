window_set_cursor С помощью этой функции вы можете установить курсор для игрового окна на любую из перечисленных ниже констант (для определения текущего используемого курсора вы можете использовать функцию window_get_cursor() которая также вернет одну из этих констант): Постоянная Курсор cr_none cr_default cr_arrow cr_cross cr_beam cr_size_nesw cr_size_ns cr_size_nwse cr_size_we cr_uparrow песочные часы cr_drag cr_appstart cr_handpoint cr_size_all Синтаксис: window_set_cursor(cursor); Аргумент Описание курсор Курсор, который нужно установить для игрового окна. Возвращается: Н/Д Пример: if mouse_check_button_pressed(mb_left) { window_set_cursor(cr_drag); } Приведенный выше код изменит курсор окна на стандартный курсор перетаскивания окна, если была нажата левая кнопка мыши.
