window_has_focus С помощью этой функции вы можете проверить состояние окна (или вкладки), и если оно потеряло фокус, функция вернет false, в противном случае она вернет true. В большинстве случаев вы можете просто использовать os_is_paused() функцию для проверки этого, но в некоторых специфических случаях (например, игры в Chrome Apps) эта функция не сработает, и в этом случае вместо нее следует использовать эту функцию. ПРИМЕЧАНИЕ: Эта функция действительна только для HTML5, Windows и MacOS. Синтаксис: window_has_focus(); Возвращается: Булево Пример: if !window_has_focus() { scr_PauseGame(); } Приведенный выше код будет проверять, находится ли окно игры в фокусе или нет, и если функция вернет false, будет вызван скрипт.
