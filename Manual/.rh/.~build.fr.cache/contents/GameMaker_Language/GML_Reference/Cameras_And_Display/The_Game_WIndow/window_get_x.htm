window_get_x С помощью этой функции вы можете получить позицию x (в пикселях) в браузере, если это игра HTML5, или на дисплее, если это игра для Windows, Ubuntu(Linux) или macOS. Синтаксис: window_get_x(); Возвращается: Реально Пример: wx = window_get_x(); Приведенный выше код сохраняет текущее положение окна по оси x в переменной "wx".
