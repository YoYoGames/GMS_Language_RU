рукоятка_окна С помощью этой функции вы можете получить внутреннее значение идентификатора Windows (HWND, указатель). Эта функция действительно полезна только для разработчиков расширений, которым нужен хэндл окна для вызова Windows API в коде DLL (возвращаемый указатель следует следует преобразовать в строку, а затем в C++ просто преобразовать его в HWND). В таблице ниже показаны поддерживаемые платформы и то, что они возвращают: Платформа Возвращает Windows Окно HWND macOS Класс NSWindow Ubuntu(Linux) XWindow handle HTML5 Идентификатор холста Синтаксис: window_handle(); Возвращается: Указатель Пример: win_id = windows_handle(); Приведенный выше код будет хранить идентификатор окна игры в переменной.
