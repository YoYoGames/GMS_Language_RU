window_get_fullscreen Эта функция возвращает, находится ли игровое окно в полноэкранном режиме(true) или нет(false). Синтаксис: window_get_fullscreen(); Возвращается: Булево Пример: if window_get_fullscreen() { draw_text(32, 32, "Fullscreen is ON"); } else { draw_text(32, 32, "Fullscreen is OFF"); } Приведенный выше код проверит, находится ли окно в полноэкранном режиме или нет, и нарисует текстовое сообщение в зависимости от возвращаемого значения.
