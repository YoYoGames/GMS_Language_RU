window_get_y С помощью этой функции вы можете получить позицию y (в пикселях) в браузере, если это игра HTML5, или на дисплее, если это игра для Windows, Ubuntu(Linux) или macOS. Синтаксис: window_get_y(); Возвращается: Реально Пример: wy = window_get_y(); Приведенный выше код сохраняет текущее положение окна оси y в переменной "wy".
