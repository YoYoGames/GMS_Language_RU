центр окна С помощью этой функции вы можете отцентрировать окно игры на дисплее, если целевой модуль - Windows, Ubuntu(Linux) или macOS, или отцентрировать его в браузере, если целевой модуль - HTML5. Эта функция не имеет эффекта на любом другом устройстве. ПРИМЕЧАНИЕ: Если вы изменили размер окна игры (например, переключившись с полноэкранного режима на оконный) и хотите отцентрировать новое окно, эта функция должна быть вызвана в alarm[0] по крайней мере на один шаг позже изменения размера окна, иначе иначе она не будет работать корректно. Синтаксис: window_center(); Возвращается: Н/Д Пример: if !window_get_fullscreen() { window_center(); } Приведенный выше код центрирует окно игры в пределах дисплея, если оно не находится в полноэкранном режиме.
