window_get_colour Эта функция возвращает цвет фона игрового окна. Этот цвет представляет собой цвет, который будет использоваться для тех областей игрового окна, которые не заняты никакими видами. Следующее изображение иллюстрирует это: Приведенное выше изображение имеет два вида с двумя портами обзора, каждый из которых нарисован в разных позициях. Это растягивает окно игры, чтобы вместить оба порта, и использует цвет окна для закрашивания фона, где вид не отображается. Синтаксис: window_get_colour(); Возвращается: Н/Д Пример: if window_get_colour() != c_black { window_set_colour(c_black); } Приведенный выше код проверит цвет окна, чтобы увидеть, установлен ли он как черный или нет, и если нет, то установит его черным.
