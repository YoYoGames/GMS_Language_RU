window_mouse_get_x С помощью этой функции вы можете получить x-позицию курсора мыши (в пикселях) в браузере, если это игра HTML5, или на дисплее, если это игра для Windows, Ubuntu(Linux) или macOS. ПРИМЕЧАНИЕ: Об обычных функциях мыши см. раздел " Ввод данных с помощью мыши". Синтаксис: window_mouse_get_x(); Возвращается: Реально Пример: wx = window_mouse_get_x(); Приведенный выше код сохраняет текущее положение мыши в окне по оси x в переменной "wx".
