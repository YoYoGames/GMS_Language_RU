window_view_mouse_get_x Эта функция возвращает положение мыши x относительно выбранного вида. ПРИМЕЧАНИЕ: Об обычных функциях мыши см. раздел " Ввод данных с помощью мыши". Синтаксис: window_view_mouse_get_x( id ); Аргумент Описание id Идентификатор представления, с которым нужно сравнить положение мыши. Возвращается: Реально Пример: if mouse_check_button_pressed(mb_left) { var xx, yy; xx = window_view_mouse_get_x(0); yy = window_view_mouse_get_y(0); if xx > 0 && xx < 32 && yy > 0 && yy < 32 { b_press[0] = true; } } Приведенный выше код проверяет, нажата ли кнопка мыши, и если да, то получает положение мыши относительно view[0] и сравнивает его, чтобы определить, нужно ли установить переменную в true.
