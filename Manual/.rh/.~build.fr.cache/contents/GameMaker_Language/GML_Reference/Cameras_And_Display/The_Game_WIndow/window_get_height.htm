window_get_height С помощью этой функции вы можете получить текущую высоту (в пикселях) окна игры. Синтаксис: window_get_height(); Возвращается: Реально Пример: hh = window_get_height(); Приведенный выше код будет хранить текущую высоту окна в переменной "hh".
