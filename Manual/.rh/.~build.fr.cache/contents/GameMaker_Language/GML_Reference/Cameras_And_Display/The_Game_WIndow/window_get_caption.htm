window_get_caption Эта функция возвращает надпись окна (это текст, который появляется в верхней части окна, рядом с его значком), по умолчанию она показывает надпись комнаты, в которой вы сейчас находитесь. Синтаксис: window_get_caption(); Возвращается: Строка Пример: if window_get_caption() != "" { window_set_caption(""); } Приведенный выше код проверит надпись окна на наличие или отсутствие текста, и если текст есть, то установит его в пустую строку, чтобы надпись не отображалась.
