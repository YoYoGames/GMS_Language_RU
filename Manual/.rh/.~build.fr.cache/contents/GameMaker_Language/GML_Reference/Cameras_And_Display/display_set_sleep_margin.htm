display_set_sleep_margin Эта функция может быть использована для установки значения маржи сна, используемого для синхронизации рендеринга вашей игры, и требует значения в миллисекундах. Значения по умолчанию следующие: Платформа Миллисекунды Windows 10 macOS 10 Ubuntu 10 HTML5 10 Android 4 iOS 4 Windows UWP 10 Xbox 10 PS4 10 Nintendo Switch 10 Обратите внимание, что даже если метод синхронизации установлен на использование синхронизации vsync, маржа сна будет иметь некоторое влияние на рендеринг игры, поэтому следует быть осторожным при установке этого значения. Для получения дополнительной информации о тайминге отображения см. display_set_timing_method(). ПРИМЕЧАНИЕ: В дополнение к Sleep Margin, вы можете дополнительно контролировать производительность и энергопотребление в Windows, регулируя разрешение планировщика потоков во время выполнения. Синтаксис: display_set_sleep_margin(milliseconds); Аргумент Описание миллисекунды Количество миллисекунд для запаса сна Возвращается: Н/Д Пример: if display_get_timing_method() != tm_sleep { display_set_timing_method(tm_sleep); if display_get_sleep_margin() != 20 { display_set_sleep_margin(20); } } Приведенный выше код проверит метод синхронизации, и если он не установлен на tm_sleep, то он будет установлен на это значение, а запас сна будет установлен на 20.
