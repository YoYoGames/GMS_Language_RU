display_set_gui_maximise Эта функция может быть использована для максимизации слоя GUI и установки его масштабирования и смещения относительно размеров экрана, а не положения и размеров поверхности приложения по умолчанию. По умолчанию слой GUI имеет соотношение 1:1 с поверхностью приложения и рисуется в позиции (0,0) поверхности. Однако это не всегда то, что вам нужно, поэтому вы можете использовать эту функцию, чтобы установить его рисование относительно абсолютной (0,0) позиции дисплея или игрового окна. Какие аргументы вы будут зависеть от того, какой эффект вы хотите получить от этой функции для слоя GUI. Если вызвать функцию без аргументов, слой GUI будет отрисован в позиции (0,0) экрана или игрового окна, а ширина и высота масштабируются, чтобы вместить всю область: display_set_gui_maximise(); Однако вы можете задать коэффициент масштабирования для слоя GUI, и ширина и высота будут масштабироваться на эту величину. Помните, что слой GUI всегда создается в соответствии с размером окна дисплея или игры, или поверхности приложения, поэтому установка значения это значение в любое значение, отличное от 1, в основном масштабирует количество пикселей по ширине и высоте. Если ваш дисплей 1024x768 и вы установили масштабный коэффициент на 0,5, то ваш слой GUI будет в два раза меньше размера дисплея, фактически удваивая размер пикселей. размер пикселей: display_set_gui_maximise(0.5, 0.5); Установка значений таким образом также установит позицию рисования в (0,0) окна дисплея или игры, поэтому вы можете предоставить значения смещения для "перемещения" позиции (0,0): var pos = application_get_position(); display_set_gui_maximise(0.5, 0.5, pos[0], pos[1]); Наконец, вы можете сбросить слой GUI с помощью этой функции, установив коэффициенты масштабирования на -1. В результате слой GUI снова будет иметь масштаб 1:1 и установит позицию рисования в положение (0,0) поверхности приложения, а не дисплея или окна. окно. display_set_gui_maximise(-1, -1); Синтаксис: display_set_gui_maximise(xscale, yscale, xoffset, yoffset); Аргумент Описание xscale <Опционально> Коэффициент горизонтального масштабирования (используйте -1, чтобы сбросить значение по умолчанию). yscale <Опционально> Коэффициент вертикального масштабирования (используйте -1, чтобы сбросить значение по умолчанию). xoffset <Опционально> По зиция смещения x для рисования. yoffset <Опционально> По зиция смещения по оси y для рисования. Возвращается: Н/Д Пример: display_set_gui_size(display_get_width() / 2, display_get_height() / 2); display_set_gui_maximise(2, 2, 0, 0); Приведенный выше код зафиксирует событие draw GUI на заданной ширине и высоте, масштабируя все компоненты для размещения на дисплее, используя эту пропорцию.
