display_reset Эта функция сбрасывает настройки дисплея на те, которые были установлены при запуске игры, а также позволяет изменить текущий уровень используемого полноэкранного сглаживания и использовать ли вертикальную синхронизацию. Доступные уровни сглаживания 0, 2, 4 и 8, при этом при запуске по умолчанию установлено значение 0, а для v-sync по умолчанию установлено значение false (выключено). Включениеv-sync может обеспечить более плавный игровой процесс, но при этом потребуется больше вычислительной мощности, поэтому его влияние следует учитывать То же самое относится и к сглаживанию, где чем выше число, тем больше требуется обработки. Поскольку не все целевые устройства одинаковы, некоторые могут не поддерживать, например, сглаживание 8x или 4x, поэтому существует переменная, доступная только для чтения, для получения различных уровней AA, которые может отображать устройство, на котором запущена игра: display_aa Эта переменная возвращает целочисленное значение, основанное на установке битов для различных уровней. Так, только для 2xAA она сообщит 2, для 2x и 4x доступности она сообщит 6. Для 8 и 4 - 12. Для всех трех уровней (2, 4 и 8) он сообщит сообщит 14. Синтаксис: display_reset(aa, vsync); Аргумент Описание aa Уровень фильтрации сглаживания (0, 2, 4 или 8). vsync Переключение вертикальной синхронизации на включение (true) или выключение (false). Возвращается: Реально Пример: if (display_aa > 12) { display_reset(8, true); } Приведенный выше код установит уровень сглаживания на 8, если поддерживается, и включит v-sync.
