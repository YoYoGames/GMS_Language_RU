display_get_gui_width С помощью этой функции вы можете получить высоту (в пикселях) графического интерфейса, используемую в событии Draw GUI Event. Синтаксис: display_get_gui_height(); Возвращается: Реально Пример: ads_move(0, display_get_gui_height() - ads_get_display_height(0), 0); Приведенный выше код установит рекламу для отображения в левом нижнем углу дисплея.
