display_mouse_get_x Эта функция возвращает положение мыши x на экране. Следует отметить, что эта функция работает правильно только при использовании для Windows. Она может быть использована и для HTML5, но вернет значение только относительно 0,0 и не будет возвращать никакого значения, пока мышь находится за пределами холста. Для других устройств он вернет 0, и вам следует использовать параметры device_mouse_raw_x()и device_mouse_raw_y() вместо этого. Синтаксис: display_mouse_get_x(); Возвращается: Реально Пример: my_x = display_mouse_get_x(); Это установит my_x в координату x мыши относительно дисплея.
