gif_add_surface С помощью этой функции вы можете сохранить кадр в GIF-файл с поверхности. Вы передаете идентификатор GIF-файла (возвращаемый функцией gif_open()) и идентификатор поверхности (возвращаемый функцией surface_create()), а также время задержки между кадрами. Время задержки рассчитывается в 1/100 секунды, но учтите, что после создания GIF-файла, если вы используете его, например, в рекламных целях, разные браузеры могут интерпретировать задержку между кадрами несколько иначе. по-разному - подробнее см. здесь. Вы можете (опционально) указать значения смещения по X и Y для поверхности, используемой для добавления в GIF. В этом случае часть, нарисованная в GIF, будет начинаться со смещения, а не с позиции по умолчанию (0,0), что позволяет выбрать только часть поверхности для добавления. Вы также можете задать значение квантования, которое представляет собой целое число от 0 и 3, где 0 - полное квантование, а 3 - нулевое квантование (при меньших значениях качество ухудшается, но создается меньший GIF). Обратите внимание, что нет большой разницы между качеством по умолчанию 2 и максимальным 3, а использование 3 значительно замедлит создание GIF, поэтому следует быть осторожным при использовании максимального значения. Обратите внимание, что нет никаких встроенных ограничений на количество кадров или на размер этих кадров в вашем GIF, но если вы используете слишком много памяти, то функция завершится неудачей и вернет -1, в противном случае она вернет 0. Синтаксис: gif_add_surface(gif_index, surface, delay_time, [xoffset], [yoffset]); Аргумент Описание gif_index Идентификатор gif для добавления данных о поверхности поверхность Идентификатор поверхности, которую следует использовать в качестве добавленной рамки время_задержки Время задержки для кадра (в сотых долях секунды) [xoffset] [Необязательно] Смещение вдоль оси X поверхности для добавления от [yoffset] [Необязательно] Смещение вдоль оси Y поверхности для добавления от [квантование]. [Необязательно] Величина квантования от 0 до 3 (0 - самое низкое качество, 3 - самое высокое качество, по умолчанию 2). Возвращается: Целое число Пример: if save_gif == true { if count == 0 { gif_image = gif_open(room_width, room_height); } else if count < 30 { gif_add_surface(gif_image, application_surface, 6/100); } else { gif_save(gif_image, "GameCapture.gif"); count = 0; save_gif = false; } count++; } Приведенный выше код создаст файл изображения GIF с 30 кадрами, взятыми с поверхности приложения, а затем сохранит его.
