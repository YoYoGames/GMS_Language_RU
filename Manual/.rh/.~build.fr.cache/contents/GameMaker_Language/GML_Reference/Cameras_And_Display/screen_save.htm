экран_сохранения С помощью этой функции вы можете сохранить скриншот игры в том виде, в котором он сейчас отображается в окне игры. Файл будет сохранен в рабочую директорию игры, которая в windows находится по адресу :/Users/YOURUSERNAME/AppData/Local/[Game Name]/, а на Mac это ~/Library/Application Support/[Game Name]/. Захваченное изображение будет окончательным рендером поверхности приложения игры, или (если поверхность приложения была отключена) это будет изображение всего дисплея или окна. Важно отметить, что эта функция предназначена для вызова в событии Draw GUI End Event, что должно дать согласованные результаты на всех платформах. Использование ее в любом другом событии может не сработать, как ожидалось, и дать разные результаты на на разных платформах и даже устройствах. Если вам нужно больше контроля над тем, что сохраняется, используйте функцию screen_save_part(). ПРИМЕЧАНИЕ: Эта функция не работает на цели HTML5. Синтаксис: screen_save(fname); Аргумент Описание fname Имя файла для сохранения снимка экрана Возвращается: Н/Д Пример: if keyboard_check_pressed(ord("S")) { screen_save(working_directory + "Screens\Screen_" + string(num++) + ".png") } Приведенный выше код делает снимок экрана игры, когда пользователь нажимает клавишу "S". Этот снимок экрана сохраняется в определенном каталоге и нумеруется с помощью переменной, которая затем увеличивается.
