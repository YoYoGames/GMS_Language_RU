view_set_hport Эта функция используется для установки высоты заданного порта просмотра. Вы указываете индекс порта просмотра (от 0 до 7) и новую высоту, которую нужно использовать. Синтаксис: view_set_hport(view_port, h) Аргумент Описание вид_порта Порт просмотра, на который будет нацелено представление (0 - 7) h Новая высота (в пикселях) Возвращается: Реально Пример: if view_get_hport(0) != (display_get_height () / 2) { view_set_hport(0, display_get_height() / 2); } Приведенный выше код проверяет высоту порта просмотра[0], и если она не равна половине высоты дисплея, то она устанавливается на это значение.
