camera_get_default Эта функция может быть использована для получения уникального значения ID камеры по умолчанию (камера, которую GameMaker Studio 2 использует, когда в игровой комнате не активны никакие виды камер или порты). Синтаксис: camera_get_default() Возвращается: Индекс камеры (реальный). Пример: var def = camera_get_default(); view_camera[0] = def; Приведенный выше код получает идентификатор камеры по умолчанию и устанавливает камеру просмотра для порта[0], чтобы использовать ее.
