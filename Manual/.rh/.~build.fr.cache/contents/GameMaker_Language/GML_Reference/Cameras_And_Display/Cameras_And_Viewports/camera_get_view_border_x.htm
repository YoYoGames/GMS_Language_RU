camera_get_view_border_x Эта функция может использоваться для получения значения границы для объекта/экземпляра, следующего за данной камерой, вдоль оси x (горизонтальная граница). Возвращаемое значение будет в пикселях. Синтаксис: camera_get_view_border_x(camera_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Реально Пример: var xb = camera_get_view_border_x(view_camera[0]); var yb = camera_get_view_border_y(view_camera[0]); if xb != 200 || yb != 200 { camera_set_view_border(view_camera[0], 200, 200); } Приведенный выше код извлекает значения xborder и yborder камеры, назначенной порту view port[0], а затем проверяет, совпадает ли оно с заданным значением. Если нет, то камера вида устанавливается на заданное значение. Назад: Камеры и порты для просмотра Далее: camera_get_view_border_y © Copyright YoYo Games Ltd. 2021 Все права защищены
