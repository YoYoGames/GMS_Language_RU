view_set_yport Эта функция используется для установки позиции y заданного порта просмотра. Вы указываете индекс порта просмотра (от 0 до 7) и новое положение, в которое его нужно поместить. Синтаксис: view_set_yport(view_port, y) Аргумент Описание вид_порта Порт просмотра, на который будет нацелено представление (0 - 7) y Новая позиция y Возвращается: Н/Д Пример: if view_get_yport(0) != (display_get_height() / 2) - (view_hport[0] / 2) { view_set_yport(0, (display_get_height() / 2) - (view_hport[0] / 2)); } Приведенный выше код проверит положение y порта просмотра[0], и если оно не там, где нужно, оно будет установлено в это положение.
