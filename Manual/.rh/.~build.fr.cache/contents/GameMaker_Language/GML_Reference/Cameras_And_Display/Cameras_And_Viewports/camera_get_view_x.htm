camera_get_view_x Эта функция может быть использована для получения положения x вида для данной камеры. Обратите внимание, что эта функция действительна только для камер, созданных с помощью функции camera_create_view() или для камер, добавленных в редакторе помещений. Синтаксис: camera_get_view_x(camera_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Реально Пример: var vx = camera_get_view_x(view_camera[0]); var vy = camera_get_view_y(view_camera[0]); draw_text(vx + 5, vy + 5, "SCORE" + string(score)); Приведенный выше код извлекает позицию камеры, назначенной порту view port[0], а затем рисует текст относительно этой позиции.
