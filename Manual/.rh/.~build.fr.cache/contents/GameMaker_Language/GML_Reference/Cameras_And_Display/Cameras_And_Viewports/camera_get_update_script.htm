camera_get_update_script Эта функция может использоваться для получения идентификатора функции сценария, назначенной в качестве сценария обновления для данной камеры. Если скрипт не назначен, то функция вернет -1. Синтаксис: camera_get_update_script(camera_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Идентификатор функции сценария или -1, если функция не назначена Пример: var scr = camera_get_update_script(camera_view[0]); if scr != scr_cutscene { camera_set_update_script(camera_view[0], scr_cutscene); } Приведенный выше код проверяет функцию сценария, назначенную в качестве функции обновления для камеры, назначенной на порт просмотра[0], и если это не "cutScene", то она устанавливается на этот сценарий.
