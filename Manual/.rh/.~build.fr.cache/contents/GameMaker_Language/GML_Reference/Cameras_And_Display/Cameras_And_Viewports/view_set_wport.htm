view_set_wport Эта функция используется для установки ширины заданного порта представления. Вы указываете индекс порта представления (от 0 до 7) и новую ширину, которую нужно использовать. Синтаксис: view_set_wport(view_port, w) Аргумент Описание вид_порта Порт просмотра, на который будет нацелено представление (0 - 7) w Новая ширина (в пикселях) Возвращается: Реально Пример: if view_get_wport(0) != (display_get_width () / 2) { view_set_wport(0, display_get_width() / 2); } Приведенный выше код проверяет ширину порта просмотра[0], и если она не равна половине ширины дисплея, то она устанавливается в это значение.
