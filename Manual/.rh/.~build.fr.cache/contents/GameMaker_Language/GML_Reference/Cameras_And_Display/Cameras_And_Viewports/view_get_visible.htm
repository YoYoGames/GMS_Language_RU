view_get_visible Эта функция может быть использована для проверки видимости заданного порта представления. Функция вернет true, если он виден, и false, если нет. Синтаксис: view_get_visible(view_port) Аргумент Описание вид_порта Порт просмотра, на который будет нацелено представление (0 - 7) Возвращается: Булево Пример: if !view_get_visible(0) { view_set_visible(0, true); } Приведенный выше код проверяет, является ли порт представления[0] видимым, и если нет, то он устанавливается в видимый.
