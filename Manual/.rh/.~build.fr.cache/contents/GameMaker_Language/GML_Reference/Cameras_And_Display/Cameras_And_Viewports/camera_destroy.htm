уничтожение камеры С помощью этой функции вы можете уничтожить любую камеру. При вызове функции вы передаете уникальное значение ID камеры, которое вы получаете из функций camera_create_* или из view_camera массива при использовании Room Editor для настройки порта просмотра и вида.
