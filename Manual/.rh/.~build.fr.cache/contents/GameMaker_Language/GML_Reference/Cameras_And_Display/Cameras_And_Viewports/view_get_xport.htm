view_get_xport Эта функция может быть использована для получения положения x заданного порта просмотра. Синтаксис: view_get_xport(view_port) Аргумент Описание вид_порта Порт просмотра, на который будет нацелено представление (0 - 7) Возвращается: Реально Пример: if view_get_xport(0) != (display_get_width() / 2) - (view_wport[0] / 2) { view_set_xport(0, (display_get_width() / 2) - (view_wport[0] / 2)); } Приведенный выше код проверит положение x порта представления[0], и если он находится не там, где должен быть, он будет установлен в это положение.
