camera_set_view_border Вы можете использовать эту функцию для установки границы следования камеры обзора в помещении. Вы задаете уникальное значение ID камеры (возвращаемое различными функциями camera_create ), а затем задаете размер границы x и размер границы y (горизонтальный и вертикальный). Эти значения будут использоваться только в том случае, если камере обзора назначена цель экземпляра для следования (либо в редакторе комнат, либо с помощью функции camera_set_view_target()) и относятся к тому, как далеко от границы вида должен находиться экземпляр, прежде чем вид обновит свою позицию для следования. Синтаксис: camera_set_view_border(camera_id, x_border, y_border) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры. x_border Граница (в пикселях) для камеры обзора вдоль горизонтальной оси y_border Граница (в пикселях) для камеры обзора вдоль вертикальной оси Возвращается: Н/Д Пример: camera_set_view_border(view_camera[0], 64, 64); Приведенный выше код установит границу камеры просмотра для камеры, назначенной порту просмотра[0].
