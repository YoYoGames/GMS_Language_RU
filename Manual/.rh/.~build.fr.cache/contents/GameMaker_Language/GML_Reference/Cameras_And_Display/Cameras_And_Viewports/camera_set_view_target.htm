camera_set_view_target Вы можете использовать эту функцию для установки цели следования камеры обзора в помещении. Вы задаете уникальное значение ID камеры (возвращаемое различными camera_create() функциями), а затем указываете экземпляр цели или ID объекта, на который вы хотите установить вид камеры. Обратите внимание, что если вы выбираете ID объекта, а в комнате находится более одного экземпляра этого объекта, GameMaker Studio 2 никак не сможет узнать, за каким экземпляром вы хотите следить, поэтому это может быть любой из них. поэтому это может быть любой из них. Синтаксис: camera_set_view_target(camera_id, instance_id/object_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры id_ экземпляра/id_объекта Экземпляр или объект, на который будет нацелена камера для слежения Возвращается: Н/Д Пример: camera_set_view_target(view_camera[0], id); Приведенный выше код установит цель камеры просмотра для камеры, назначенной порту просмотра[0], на цель экземпляра, выполняющего этот код.
