view_set_camera Эта функция устанавливает камеру на определенный порт просмотра. Вы указываете порт просмотра, который нужно установить (от 0 до 7), и предоставляете уникальное значение ID камеры (как возвращается функцией camera_create() функции или когда вы используете view_get_camera()). Если вы передаете значение -1 в качестве ID камеры, то вы удаляете камеру из порта просмотра, и обратите внимание, что если этот порт просмотра включен и виден, вы можете получить непредсказуемые результаты. Синтаксис: view_set_camera(view_port, camera_id) Аргумент Описание вид_порта Порт просмотра, на который будет нацелено представление (0 - 7) идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Н/Д Пример: var cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1); view_set_camera(0, cam); Приведенный выше код создаст новую камеру, а затем назначит ей порт просмотра[0].
