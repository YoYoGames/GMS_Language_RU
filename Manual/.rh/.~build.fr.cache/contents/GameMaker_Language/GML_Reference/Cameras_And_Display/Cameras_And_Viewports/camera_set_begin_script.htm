camera_set_begin_script Эта функция может быть использована для задания функции сценария, которая будет вызываться в начале каждого игрового кадра, когда камера назначена на видимый и активный порт просмотра. Вы задаете уникальное значение камеры ID значение (как возвращается различными camera_create() функции) и ID функции, которая будет вызываться. Порядок выполнения скриптов, прикрепленных к камерам, и фактического рендеринга вида камеры следующий: Камеры для всех видимых и активных портов просмотра имеют свою функцию обновления, вызываемую как Затем для каждого вида: Вызывается функция begin для камеры для этого представления Для этого вида выполняются события рисования Вызывается функция end для камеры Перейдите к следующему активному видимому виду и повторите... Синтаксис: camera_set_begin_script(camera_id, script) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры. скрипт Функция сценария для запуска каждого игрового кадра Возвращается: Н/Д Пример: camera_set_begin_script(view_camera[0], beginCamera); Приведенный выше код устанавливает функцию сценария begin для камеры, назначенной на порт просмотра [0].
