camera_get_view_mat Эта функция может быть использована для получения матрицы представления. Функция возвращает значение ID матрицы, которое затем может быть использовано в других матричных функциях или для установки матрицы вида для другой камеры (с помощью camera_set_view_mat()). Обратите внимание, что эта функция обычно используется для камер, созданных с помощью функции camera_create(), но она также может быть использована на камерах, созданных в редакторе помещений (или с помощью camera_create_view()) - в этом случае возвращаемая матрица будет действительна только после первого использования данной камеры в порте просмотра. Синтаксис: camera_get_view_mat(camera_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Идентификатор матрицы (реальный) Пример: mat = camera_get_view_mat(camera_view[0]); Приведенный выше код получает матрицу вида для камеры, назначенной на порт вида[0].
