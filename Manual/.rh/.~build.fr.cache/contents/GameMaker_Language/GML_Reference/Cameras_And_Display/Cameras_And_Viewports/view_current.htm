view_current Эта переменная, доступная только для чтения, действительна только в Событие рисования и возвращает текущий порт отрисовки представления. Возвращаемое значение будет меняться во время события draw когда у вас есть различные представления, поскольку событие draw вызывается один раз для каждого порта представления последовательно. Таким образом, когда (например) вы используете порт вида[0] и порт вида[1] в вашей игровой комнате, событие рисования для ВСЕХ экземпляров будет выполняться дважды, по одному разу для каждого порта, и с помощью этой переменной вы можете проверить, какой вид рисуется в данный момент. В общем, это используется для отрисовки определенных деталей только для одного порта, когда в комнате одновременно видно несколько портов просмотра. См. пример код ниже. Синтаксис: view_current; Возвращается: Целое число (индекс порта просмотра от 0 до 7) Пример: if view_current == 0 { var xx = camera_get_view_x(view_camera[0]); var yy = camera_get_view_y(view_camera[0]); draw_text(xx + 32, yy + 32, "Player 1"); } else { var xx = camera_get_view_x(view_camera[1]); var yy = camera_get_view_y(view_camera[1]); draw_text(xx + 32, yy + 32, "Player 2"); } Приведенный выше код проверяет, какой вид отображается в данный момент, а затем рисует разный текст для каждого вида в зависимости от возвращаемого значения.
