camera_get_view_target Эта функция может быть использована для получения цели следования для заданной камеры. Возвращаемое значение может быть индексом объекта, идентификатором экземпляра или -1, если цель следования не была установлена. Синтаксис: camera_get_view_target(camera_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Индекс объекта, ID экземпляра или -1, если цель не задана. Пример: var target = camera_get_view_target(view_camera[0]); if target != obj_Player { camera_set_view_target(view_camera[0], obj_Player); } Приведенный выше код извлекает цель следования камеры, назначенной порту вида[0], а затем проверяет, совпадает ли она с индексом объекта "obj_Player". Если нет, то камера вида устанавливается на следование за экземпляром этого объекта. Назад: Камеры и порты для просмотра Далее: camera_get_default © Copyright YoYo Games Ltd. 2021 Все права защищены
