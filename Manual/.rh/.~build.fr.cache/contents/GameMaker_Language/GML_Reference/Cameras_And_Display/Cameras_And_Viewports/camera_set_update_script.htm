camera_set_update_script Эта функция может быть использована для задания функции сценария, которая будет вызываться каждый игровой кадр, когда камера назначена на видимый и активный порт просмотра. Вы указываете уникальное значение ID камеры (возвращаемое различными функциями camera_create ) и ID функции, которая будет вызываться. Порядок вызова функций, прикрепленных к камерам, и фактического рендеринга вида камеры следующий: Камеры для всех видимых и активных портов просмотра имеют свою функцию обновления , вызываемую как Затем для каждого вида: Вызывается функция begin для камеры этого вида. Для этого вида выполняются события рисования Вызывается функция end для камеры Перейдите к следующему активному видимому виду и повторите... Синтаксис: camera_set_update_script(camera_id, script) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры. скрипт Функция сценария для запуска каждого игрового кадра Возвращается: Н/Д Пример: camera_set_update_script(view_camera[0], updateCamera); Приведенный выше код устанавливает функцию обновления для камеры, назначенной на порт просмотра [0].
