view_enabled Эта переменная управляет тем, включены или нет порты просмотра, видимые в комнате. Если для портов просмотра установлено значение видимые, а затем отключить этот параметр, вся комната будет отрисована на экране с масштабированием по размеру окна, вместо того чтобы вместо того, чтобы разные камеры рисовались через порты просмотра. Синтаксис: view_enabled; Возвращается: Булево(true: включено, false: выключено) Пример: if !view_enabled { view_visible[0] = true; view_enabled = true; } Приведенный выше код проверяет, включены ли представления для комнаты, и если нет, то убеждается, что порт представления[0] виден, а затем включает представления для комнаты.
