camera_get_view_angle Эта функция может быть использована для получения угла заданной камеры. Возвращаемое значение будет находиться в диапазоне от 0 до 360. Обратите внимание, что эта функция действительна только для камер, созданных с помощью функции camera_create_view() или для камер, добавленных в редакторе помещений. Синтаксис: camera_get_view_angle(camera_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Реально Пример: var ang = camera_get_view_angle(view_camera[0]); if ang != 0 { camera_set_view_angle(view_camera[0], 0); } Приведенный выше код извлекает угол камеры, назначенной порту view port[0], а затем проверяет, соответствует ли он заданному значению. Если нет, то камера вида устанавливается на этот угол. Назад: Камеры и порты для просмотра Далее: camera_get_view_target © Copyright YoYo Games Ltd. 2021 Все права защищены
