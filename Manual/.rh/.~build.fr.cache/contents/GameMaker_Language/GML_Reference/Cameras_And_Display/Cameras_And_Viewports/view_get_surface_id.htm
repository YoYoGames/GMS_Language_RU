view_get_surface_id Эта функция может быть использована для получения уникального значения ID для поверхности, назначенной для данного порта просмотра (возвращает -1, если поверхность не была назначена). Синтаксис: view_get_surface_id(view_port) Аргумент Описание вид_порта Порт просмотра, на который будет нацелено представление (0 - 7) Возвращается: Реально Пример: if view_get_surface_id(0) == -1 { view_set_surface_id(0, global.vSurf); } Приведенный выше код проверит, была ли поверхность назначена порту вида[0], и если нет, то она будет назначена.
