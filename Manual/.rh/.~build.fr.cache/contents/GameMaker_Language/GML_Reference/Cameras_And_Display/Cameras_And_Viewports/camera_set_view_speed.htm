camera_set_view_speed Вы можете использовать эту функцию для обновления скорости камеры обзора в помещении. Вы задаете уникальное значение ID камеры (как возвращается различными camera_create() функциями), а затем задаете x и y (горизонтальную и вертикальную) скорость, с которой она должна двигаться, если установлена для следования за данным экземпляром. Скорость рассчитывается как пиксели за шаг и может быть установлена на "-1", чтобы камера двигалась мгновенно, но если камера не настроена следовать за каким-либо объектом, то заданные здесь значения не будут иметь видимого эффекта. установленные здесь значения не будут иметь видимого эффекта. Синтаксис: camera_set_view_speed(camera_id, xspeed, yspeed) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры. xspeed Скорость (количество пикселей за игровой кадр), с которой вид должен двигаться по горизонтальной (x) оси. yspeed Скорость (количество пикселей за игровой кадр), с которой вид должен двигаться по вертикальной оси (y). Возвращается: Н/Д Пример: camera_set_view_speed(view_camera[0], 5, 5); Приведенный выше код установит скорость камеры просмотра для камеры, назначенной на порт просмотра[0].
