camera_get_end_script Эта функция может быть использована для получения идентификатора функции сценария, назначенной в качестве конечной функции для данной камеры. Если функция сценария не назначена, то функция вернет -1. Синтаксис: camera_get_end_script(camera_id) Аргумент Описание идентификатор камеры Уникальное значение ID камеры, полученное при создании камеры Возвращается: Идентификатор функции сценария или -1, если функция не назначена Пример: var scr = camera_get_end_script(camera_view[0]); if scr != cutScene { camera_set_update_script(camera_view[0], cutScene); } Приведенный выше код проверяет функцию сценария, назначенную в качестве начального сценария для камеры, назначенной на порт просмотра[0], и если это не "cutScene", то она устанавливается на эту функцию.
