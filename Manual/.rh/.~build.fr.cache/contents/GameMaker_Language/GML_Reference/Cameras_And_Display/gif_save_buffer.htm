gif_save_buffer С помощью этой функции вы можете сохранить анимацию в формате GIF. Вы передаете индекс GIF (возвращаемый функцией gif_open()), а функция вернет буфер роста с выравниванием в 1 байт, содержащий данные GIF. Обратите внимание, что конечный GIF-данные будут палетированы с помощью универсальной палитры 884 (более подробную информацию см. здесь ). Синтаксис: gif_save_buffer(gif_index); Аргумент Описание gif_index Идентификатор gif для сохранения Возвращается: Целое число (идентификатор буфера) Пример: if save_gif == true { if count == 0 { gif_image = gif_open(room_width, room_height); } else if count < 30 { gif_add_surface(gif_image, application_surface, 6/100); } else { global.capture_buff = gif_save_buffer(gif_image); count = 0; save_gif = false; } count++; } Приведенный выше код создаст файл изображения GIF с 30 кадрами, взятыми с поверхности приложения, а затем сохранит его в буфер.
