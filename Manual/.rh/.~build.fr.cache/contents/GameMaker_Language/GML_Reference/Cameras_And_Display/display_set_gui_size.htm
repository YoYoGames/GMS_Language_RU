display_set_gui_size Эта функция влияет только на то, что рисуется в событии Draw GUI, и может быть использована для установки определенной ширины и высоты для всех компонентов GUI в этом событии, и независимо от того, какой размер дисплея или окна используется для отображения GUI, он будет масштабирован для правильного размещения. Позиционирование GUI (т.е. позиция (0,0)) зависит от того, использовали ли вы функцию display_set_gui_maximise() или нет. Если вы не используете эту функцию (настройка по умолчанию), то слой GUI будет позиционирован с точкой (0,0) на той же позиции, что и поверхность приложения (область, где рисуется ваша игра), а ширина и высота будут например, если установить ширину GUI 256px с помощью этой функции, когда поверхность приложения равна 512px, то эти 256 пикселей будут растянуты до нужного размера, в результате чего пикселей станет в два раза больше. Это идеально подходит для тех игр в которых коррекция соотношения сторон установлена в глобальных настройках игры. Если вы использовали функцию display_set_gui_maximise(), то позиция (0,0) для слоя GUI будет по умолчанию соответствовать левому верхнему углу окна игры или дисплея (в полноэкранном режиме или на устройствах), с коэффициентом масштабирования, установленным этой функцией, а ширина и высота задаются этой функцией. Вы можете в любое время вернуть слой GUI к конфигурации по умолчанию, вызвав эту функцию, используя -1 в качестве ширины и высоты - это установит позицию (0, 0) на поверхности приложения, а ширина и высота будут сброшены на 1. и высота будут сброшены на 1:1 с поверхностью. Синтаксис: display_set_gui_size(width, height); Аргумент Описание ширина Ширина графического интерфейса пользователя высота Высота графического интерфейса пользователя Возвращается: Н/Д Пример: display_set_gui_size(768, 1024); Приведенный выше код зафиксирует событие draw GUI на заданной ширине и высоте, масштабируя все компоненты для размещения либо на поверхности приложения, либо на дисплее, используя эту пропорцию.
