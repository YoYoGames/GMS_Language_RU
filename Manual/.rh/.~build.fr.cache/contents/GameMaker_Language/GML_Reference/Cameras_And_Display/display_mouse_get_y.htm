display_mouse_get_y Эта функция возвращает положение мыши по оси y на экране. Следует отметить, что эта функция работает правильно только при использовании для Windows. Она может быть использована и для HTML5, но вернет значение только относительно 0,0 и не будет возвращать никакого значения, пока мышь находится за пределами холста. Для других устройств он вернет 0, и вам следует использовать параметры device_mouse_raw_x() и device_mouse_raw_y() вместо этого. Синтаксис: display_mouse_get_y(); Возвращается: Реально Пример: my_y = display_mouse_get_y(); Это установит my_y в координату y мыши относительно дисплея.
