display_set_ui_visibility Эта функция может быть использована для отображения или скрытия системного пользовательского интерфейса на Android только. Функция требует, чтобы вы предоставили один или несколько системных флагов в виде целочисленного значения. Если используется более одного флага, их необходимо объединить с помощью побитовой функции "или", как показано в примере ниже. Список системных флагов Android можно найти здесь. Синтаксис: display_set_ui_visibility(flags); Аргумент флаги флаги Флаги системы Android SYSTEM, которые необходимо использовать (целочисленное значение). Возвращается: Н/Д Пример: var flags = 1024|4096; display_set_ui_visibility(flags); Приведенный выше код будет использовать системные флаги Android "SYSTEM_UI_FLAG_IMMERSIVE_STICKY" и "SYSTEM_UI_FLAG_LAYOUT_FULLSCREEN" для установки видимости дисплея.
