display_get_sleep_margin Эта функция может быть использована для получения текущего значения маржи сна, используемого для тайминга рендеринга вашей игры, и возвращает значение в миллисекундах. Для получения дополнительной информации о тайминге отображения см. display_set_timing_method(). Синтаксис: display_get_sleep_margin(); Возвращается: Реально Пример: if display_get_timing_method() != tm_sleep { display_set_timing_method(tm_sleep); if display_get_sleep_margin() != 20 { display_set_sleep_margin(20); } } Приведенный выше код проверит метод синхронизации, и если он не установлен на tm_sleep, то он будет установлен на это значение, а запас сна будет установлен на 20.
