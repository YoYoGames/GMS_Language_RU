network_create_socket Эта функция используется для создания нового клиентского сокета для вашей игры для общения по сети. Вы должны определить тип сокета (см. список констант ниже), и функция вернет уникальный идентификатор для этого сокета, который должен использоваться во всех дальнейших вызовах функций для этого сокета, или значение меньше 0, если соединение не удалось. ПРИМЕЧАНИЕ: Вы можете использовать IP "127.0.0.1" для обратного подключения к тому же устройству, на котором запущена игра. Постоянная Описание network_socket_tcp Создайте сокет с помощью TCP. network_socket_udp Создайте сокет, используя UDP. network_socket_ws Создайте веб-сокет(только для проектов, работающих на HTML5), используя TCP. network_socket_bluetooth Создайте сокет bluetooth(в настоящее время недоступен!). Синтаксис: network_create_socket(type); Аргумент Описание тип Тип создаваемого сокетного соединения (см. константы, перечисленные ниже). Возвращается: Реально Пример: если os_browser == browser_not_a_browser { client = network_create_socket(network_socket_tcp); network_connect( client, "192.134.0.1", 6510 ); } else { client = network_create_socket(network_socket_ws); network_connect( client, "192.134.0.1", 6520 ); } Приведенный выше код проверит, запущена ли игра в браузере или нет, и создаст новый TCP или Web-сокет, после чего попытается подключиться через него к заданному IP-адресу на заданном порту.
