network_create_socket_ext Эта функция используется для создания нового клиентского сокета для вашей игры для общения по сети. Вы должны определить протокол сокета (см. список констант ниже) и указать порт, который будет использоваться, и функция вернет уникальный идентификатор, который будет использоваться во всех дальнейших вызовах функции, или значение меньше 0, если соединение не удалось. будет использоваться во всех дальнейших вызовах функций для этого сокета, или значение меньше 0, если соединение не удалось. Постоянная Описание network_socket_tcp Создайте сокет с помощью TCP. network_socket_udp Создайте сокет, используя UDP. network_socket_ws Создайте веб-сокет(только для проектов, работающих на HTML5), используя TCP. Синтаксис: network_create_socket_ext(protocol, port); Аргумент Описание протокол Сетевой протокол для использования порт Порт для использования Возвращается: Реально Пример: клиент = network_create_socket_ext(network_socket_udp, 6510); Приведенный выше код создаст новый UDP сокет на порту 6510.
