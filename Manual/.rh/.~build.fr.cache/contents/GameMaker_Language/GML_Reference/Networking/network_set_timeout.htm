network_set_timeout С помощью этой функции вы можете установить тайм-аут для чтения и записи данных на/от сервера через заданный сокет. Обратите внимание, что тайм-аут не генерирует никаких событий, поэтому вам придется самостоятельно бороться с тайм-аутами, используя, например, сигналы тревоги. Обратите внимание, что это значение влияет только на отправку и получение данных, и если вы хотите изменить значение таймаута соединения, то вам следует использовать функцию network_set_config(). Синтаксис: network_set_timeout(socket, read_timeout, write_timeout); Аргумент Описание розетка Идентификатор используемого сокета. таймаут чтения Миллисекунды, через которые произойдет тайм-аут при передаче данных с сервера. таймаут записи Миллисекунды, через которые передача данных на сервер будет прервана. Возвращается: Н/Д Пример: network_set_timeout(sock, 3000, 3000); Приведенный выше код установит тайм-аут для чтения и записи данных через сокет, индексированный в переменной "sock", равным 3 секундам.
