сетевой_коннект С помощью этой функции вы можете отправить запрос на подключение к серверу. Функция принимает идентификатор сокета для подключения (см. network_create_socket()) и требует, чтобы вы указали IP-адрес для подключения (строка), а также порт для подключения. В случае неудачи соединения будет возвращено значение меньше 0. Соединение использует специальный протокол, который обеспечивает соединение только игр GameMaker Studio 2, однако если вам необходимо подключиться к серверу, который не является игрой GameMaker Studio 2, вы можете воспользоваться командой network_connect_raw(). Обратите внимание, что по умолчанию функция является синхронной, а это означает, что ваша игра может "зависнуть". во время установления соединения. Вы можете установить значение тайм-аута для соединения или сделать создание асинхронным, используя функцию network_set_config()или альтернативно использовать функцию network_connect_async() вместо нее. Синтаксис: network_connect(socket, url, port); Аргумент Описание розетка Идентификатор используемого сокета. url URL или IP для подключения (строка). порт Порт, к которому необходимо подключиться. Возвращается: Реально Пример: client = network_create_socket(network_socket_tcp); network_connect(client, "192.134.0.1", 6510); Приведенный выше код создаст новый TCP сокет и попытается подключиться через него к заданному IP-адресу на порт 6510.
