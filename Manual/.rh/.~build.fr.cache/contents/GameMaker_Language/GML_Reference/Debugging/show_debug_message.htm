show_debug_message Эта функция создает пользовательское отладочное сообщение, которое отображается в окне компилятора во время выполнения программы. Обратите внимание, что сообщение может быть как строкой, так и вещественным числом, но если вам нужно и то, и другое, то число придется сначала преобразовать в строку с помощью функции string() (см. пример ниже) и что если число имеет более двух знаков после запятой, то следует использовать функцию string_format() чтобы показать их так как по умолчанию десятичные числа округляются до ближайших двух знаков после запятой (поэтому "1.2468" в окне вывода будет отображаться как "1.25"). Отладочные сообщения, показанные с помощью этой функции, будут отображаться в окне вывода компилятора в нижней части IDE, а также в представлении графика в отладчике при запуске игры в режиме отладки. отладчика при запуске игры в режиме отладки. Если вы хотите видеть сообщения только в режиме отладки, то вам, вероятно, следует использовать функцию debug_event() вместо этого. Синтаксис: show_debug_message(string); Аргумент Описание строка Пользовательское отладочное сообщение для показа. Возвращается: Н/Д Пример: if !instance_exists(obj_Exit) { show_debug_message("Exit not created!"); show_debug_message("Error Num: " + string(global.error)); game_end(); } Приведенный выше код проверяет, существует ли экземпляр, и если нет, то в окне компиляции появляется отладочное сообщение и игра завершается.
