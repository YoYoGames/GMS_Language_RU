показать_вопрос Эта функция создает всплывающее окно сообщения с двумя кнопками "Да" и "Нет". Она возвращает true или false в зависимости от того, какую из двух кнопок нажимает пользователь. ПРИМЕЧАНИЕ: Эта функция предназначена для отладки только для целей Desktop, а для всех остальных целей она устарела. Синтаксис: show_question(str); Аргумент Описание str Строка для отображения во всплывающем вопросе. Возвращается: Булево Пример: if (score > 500) && debug_mode { if show_question("Continue to next room?") { room_goto(rm_Level2); } else game_end(); } Приведенный выше код проверит счет, и если он больше 500, то спросит пользователя, хочет ли он продолжить или нет, и если будет нажата кнопка "да", то он перейдет в другую комнату, если же будет выбрана кнопка "нет", то игра закончится.
