code_is_compiled Эта функция возвращает true, если платформа компилируется вне виртуальной машины, например, для платформ YYC и JS. Синтаксис: code_is_compiled(); Возвращается: Булево Пример: if code_is_compiled() { show_debug_message("Compiler okay!"); } Приведенный выше код проверяет, была ли игра скомпилирована с помощью YoYo Compiler, и показывает отладочное сообщение, если это так.
