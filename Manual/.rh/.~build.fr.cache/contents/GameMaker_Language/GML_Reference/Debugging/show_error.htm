show_error Эта функция покажет пользовательскую строку в качестве сообщения об ошибке и даст возможность прервать игру после этого, если это необходимо. ПРИМЕЧАНИЕ: Эта функция предназначена только для отладки. Синтаксис: show_error(str, abort); Аргумент Описание str Строка для отображения во всплывающем сообщении. прервать Прервать (true) или нет (false). Возвращается: Н/Д Пример: if (room != rm_Dungeon) { show_error("Error: Went to wrong area. Aborting game.", true); } Приведенный выше код проверит, является ли комната, в которой находится игрок, "rm_Dungeon", и если нет, то покажет сообщение об ошибке и заставит пользователя прервать игру.
