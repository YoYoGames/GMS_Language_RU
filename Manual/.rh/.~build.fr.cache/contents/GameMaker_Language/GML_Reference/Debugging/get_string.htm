get_string Это создает всплывающее окно со стандартным сообщением и кнопкой "Ok", которая предлагает пользователю ввести строку. Функция возвращает введенную строку или значение по умолчанию, если ничего не было введено. ПРИМЕЧАНИЕ: Эта функция предназначена для отладки только для целей Desktop, а для всех остальных целей она устарела. Синтаксис: get_string(str, def); Аргумент Описание str Строка для отображения во всплывающем сообщении. def Строка по умолчанию в текстовом поле. Возвращается: Строка Пример: global.test_name = get_string("Test highscore name:", "Anonymous"); Приведенный выше код попросит пользователя указать имя, которое затем будет сохранено в глобальной переменной "test_name". Если ничего не будет введено и пользователь просто нажмет "Ok", то будет возвращено значение по умолчанию "Anonymous".
