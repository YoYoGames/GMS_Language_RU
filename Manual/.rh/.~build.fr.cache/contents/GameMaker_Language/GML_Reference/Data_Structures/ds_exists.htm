ds_exists С помощью этой функции вы можете проверить, существует ли структура данных заданного типа. Вы передаете значение "index" (хранящееся в переменной) и "тип" DS, который может быть любой из перечисленных ниже констант, и функция возвращает true, если структура данных существует, и false в противном случае. Постоянная Описание ds_type_map Структура данных карты ds_type_list Списочная структура данных ds_type_stack Структура данных стека ds_type_grid Структура данных сетки ds_type_queue Структура данных очереди ds_type_priority Структура данных о приоритетах Синтаксис: ds_exists(ind, type); Аргумент Описание инд Индекс переменной для проверки структуры данных тип Тип структуры данных для проверки (см. список констант ниже) Возвращается: Булево Пример: if !ds_exists(ai_grid, ds_type_grid) { ai_grid = ds_grid_create(room_width / 32, room_height / 32); } Приведенный выше код проверяет (предварительно инициализированную) переменную "ai_grid" на предмет индексации структуры данных типа DS grid, и если таковой нет, то создает ее и сохраняет ее индекс в переменной.
