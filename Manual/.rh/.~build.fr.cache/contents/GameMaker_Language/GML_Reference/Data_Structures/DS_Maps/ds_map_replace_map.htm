ds_map_replace_map С помощью этой функции вы можете заменить карту DS Map, которая была сохранена в заданном "ключе", на другую карту, созданную ранее. Эта функция предназначена для создания карт, совместимых с JSON, которые затем можно закодировать с помощью функции json_encode() и должна использоваться только в сочетании с этой функцией. Синтаксис: ds_map_replace_map(id, key, value) Аргумент Описание id Идентификатор используемой карты ds_map. ключ Ключ на замену. значение Идентификатор ds_map, который будет использоваться для замены ранее хранившегося в данном ключе. Возвращается: Н/Д Пример: var temp_map = ds_map_create(); ds_map_add_list(temp_map, "list", j_list); ds_map_add(temp_map, "array", j_array); ds_map_replace_map(j_map, "maps", temp_map); var j = json_encode(j_map); ds_map_destroy(temp_map); Приведенный выше код создаст DS Map и заполнит ее массивом и списком DS List перед заменой ранее сохраненной карты в DS Map "j_map".
