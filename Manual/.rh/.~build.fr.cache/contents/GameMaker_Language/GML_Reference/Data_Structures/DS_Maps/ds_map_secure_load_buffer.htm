ds_map_secure_load_buffer Эта функция загружает сохраненную карту DS из буфера. Вы должны предварительно загрузить буфер в память (используя buffer_load()), а затем передать его в эту функцию, которая вернет DS карту, заполненную содержимым буфера. Обратите внимание, что буфер должен быть создан с помощью функции ds_map_secure_save_buffer()для корректной работы, а также обратите внимание, что если DS map содержала массив, то при загрузке он будет преобразован в DS list. ВАЖНО! Одной из особенностей безопасного сохраненного файла является то, что он блокируется на устройстве, на котором был создан, поэтому вы не можете загрузить файл, сохраненный на одном устройстве, в проект, запущенный на другом устройстве. Синтаксис: ds_map_secure_load_buffer(buffer); Аргумент Описание имя файла Идентификатор буфера, из которого загружаются данные карты. Возвращается: Целое число (индекс карты DS) Пример: var buff = buffer_load("save.dat"); map = ds_map_secure_load_buffer(buff); buffer_delete(buff); Приведенный выше код загрузит сохраненную карту DS из буфера и сохранит значение ее индекса в переменной для дальнейшего использования.
