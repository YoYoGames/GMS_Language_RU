ds_map_secure_save Эта функция сохранит содержимое заданной карты DS в файл, который связан с устройством, на котором он был создан (это означает, что он не может быть прочитан при переносе на любое другое устройство). Сам файл может иметь практически любое расширение (например, *.dat, *.json, *.bin и т.д...) и будет обфусцирован и сохранен в локальном хранилище на целевой платформе. Затем вы можете повторно загрузить ds_map с помощью функции ds_map_secure_load(). Обратите внимание, что если сохраняемая карта DS содержит массив, то при сохранении этот массив будет преобразован в список DS. ВАЖНО! Одной из особенностей безопасного сохраненного файла является то, что он блокируется на устройстве, на котором был создан, поэтому вы не можете загрузить файл, сохраненный на одном устройстве, в проект, запущенный на другом устройстве. Синтаксис: ds_map_secure_save(map, filename); Аргумент Описание карта Идентификатор структуры данных для использования имя файла Имя файла, в который нужно сохранить карту Возвращается: Булево Пример: ds_map_secure_save(purchase_map, "p_data.dat"); Приведенный выше код сохранит карту DS, индексированную в переменной "p_data", в заданный файл для последующего извлечения.
