ds_map_find_last Эта функция возвращает последний ключ, хранящийся в карте DS. Это не последний ключ в том порядке, в котором вы их добавляли! Карты DS не хранятся в линейной форме, для этого используются DS-списки, поэтому все, что делает эта функция, это находит последний ключ, сохраненный компьютером. Это может быть полезно, если вам нужно итеративно просматривать DS-карту в поисках чего-либо, но по возможности этого следует избегать, так как это может быть медленным. Обратите внимание, что эта функция вернет неопределенное значение, если заданная карта DS пуста. Синтаксис: ds_map_find_last(id); Аргумент Описание id Идентификатор карты, которую нужно использовать. Возвращается: Вещественный, строковый или неопределенный Пример: var size = ds_map_size(inventory); var key = ds_map_find_last(inventory); for (var i = size; i > 0; i--;) { if (key != "gold") { key = ds_map_find_previous(inventory, key); } else break; } Приведенный выше код создает некоторые временные переменные, затем получает размер карты DS и находит последний ключ, сохраненный компьютером в карте. Затем с помощью цикла for выполняется итерация по карте DS в поисках значения ключа "gold". Если оно найдено, то цикл прерывается.
