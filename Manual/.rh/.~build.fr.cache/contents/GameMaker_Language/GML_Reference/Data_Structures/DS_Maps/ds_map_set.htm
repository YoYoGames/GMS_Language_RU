ds_map_set С помощью этой функции вы можете установить значение ключа в данной карте DS. Вы передаете значение ID карты DS (возвращаемое функцией ds_map_create()), затем указываете ключ, который вы хотите установить, и значение, которое нужно установить на которое его нужно установить. Ключи могут быть целыми числами или строками, и если заданный ключ не существует, то он будет создан для вас и установлен в значение. Эта функция аналогична использованию DS map accessor для установки/создания пары ключ/значение карты. Функция ничего не возвращает, поэтому если вам нужно проверить, было ли заменено значение ключа или создан новый ключ, используйте функцию ds_map_replace(). Синтаксис: ds_map_set(id, key, value) Аргумент Описание id Идентификатор карты, которую нужно использовать. ключ Ключ к установке. значение Значение, на которое нужно установить ключ. Возвращается: Н/Д Пример: if is_undefined(ds_map_find_value(map, "score")) { ds_map_set(map, "score", 0); } Приведенный выше код проверит, существует ли заданный ключ, и если нет, то он будет создан и установлен.
