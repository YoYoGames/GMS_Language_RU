ds_map_empty Эта функция просто вернет false, если указанная (ранее созданная) карта DS содержит какие-либо пары ключ/значение, или true, если не содержит. Синтаксис: ds_map_empty(id); Аргумент Описание id Идентификатор структуры данных для проверки. Возвращается: Булево Пример: if ds_map_empty(inventory) { вес = 0; } Приведенный выше код проверяет, есть ли в карте DS, индексируемой в переменной "inventory", пары ключ/значение, и если нет, то устанавливает переменную "weight" в 0.
