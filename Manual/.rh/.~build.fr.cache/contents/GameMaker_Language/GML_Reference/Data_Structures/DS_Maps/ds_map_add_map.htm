ds_map_add_map С помощью этой функции вы можете назначить (ранее созданную) карту DS ключу в заданной карте DS. Эта функция предназначена для создания JSON-совместимых карт, которые вы затем будете кодировать с использованием json_encode() и должна использоваться только в сочетании с этой функциональностью. Если в DS-карту добавлена другая карта, то уничтожение родительской карты приведет к уничтожению содержащихся в ней карт и освобождению их памяти, а вызов функции ds_map_clear() на родительской карты также уничтожит и очистит все помеченные карты. Синтаксис: ds_map_add_map(id, key, value) Аргумент Описание id Идентификатор карты, которую нужно использовать. ключ Ключ для добавленной карты. значение Идентификатор карты, которую нужно добавить. Возвращается: Н/Д Пример: var j_map = ds_map_create(); var j_list = ds_list_create(); var sub_map = ds_map_create(); ds_map_add_list(sub_map, "list", j_list); ds_map_add(sub_map, "array", j_array); ds_map_add_map(j_map, "map", sub_map); var j = json_encode(j_map); ds_map_destroy(j_map); Приведенный выше код создает две карты DS, затем заполняет одну из них списком и массивом, после чего добавляет их во вторую, которая затем кодируется в строку JSON. Затем карта уничтожается, чтобы удалить ее и любые другие карты или списки, которые она содержит, из памяти.
