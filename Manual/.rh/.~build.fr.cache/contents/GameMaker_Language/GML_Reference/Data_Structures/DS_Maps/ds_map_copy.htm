ds_map_copy Вы можете использовать эту функцию для копирования содержимого одной карты в другую, ранее созданную с помощью ds_map_create(). Если карта DS, в которую выполняется копирование, не пуста, то эта функция сначала очистит ее перед копированием. При этом исходная карта DS остается неизменной. Синтаксис: ds_map_copy(id, source); Аргумент Описание id Идентификатор карты, накоторую вы копируете источник Идентификатор карты, скоторой вы копируете. Возвращается: Н/Д Пример: inventory_2 = ds_map_create(); ds_map_copy(inventory_2, inventory_1); Приведенный выше код создаст новую карту и присвоит ее переменной "inventory_2". Затем в эту новую карту будет скопировано содержимое карты DS, проиндексированной в переменной "inventory_1". Назад: Карты DS Далее: ds_map_replace © Copyright YoYo Games Ltd. 2021 Все права защищены
