ds_map_size С помощью этой функции вы можете узнать, сколько пар ключ/значение содержит (ранее созданная) карта DS. Синтаксис: ds_map_size(id); Аргумент Описание id Идентификатор структуры данных для проверки. Возвращается: Реально Пример: if (ds_map_size(inventory) > 49) { full = true; } Приведенный выше код проверит размер карты DS (т.е. количество пар ключ/значение) и, если он больше 49, установит переменную "full" в true.
