ds_map_clear Эта функция очистит (ранее созданную) карту DS от всех пар ключ/значение, но не удалит саму карту DS из памяти. Для этого следует использовать функцию ds_map_destroy(). Обратите внимание, что очистка карты удалит ссылки на любые структуры данных, хранящиеся в карте, что приведет к утечке памяти, поэтому перед очисткой карты вам нужно будет пройтись по ней и уничтожить все элементы структуры данных вручную, чтобы предотвратить это. Единственный случай, когда это не требуется, это когда вы отметили какие-либо элементы в карте как список DS или как другую карту DS, в этом случае эти элементы будут уничтожены (не очищены!), а их память очищена автоматически при очистке родительская карта будет очищена. Синтаксис: ds_map_clear(id); Аргумент Описание id Идентификатор структуры данных, которую нужно очистить. Возвращается: Н/Д Пример: if global.new_game { ds_map_clear(inventory); } Приведенная выше функция проверит глобальную переменную на истинность и, если это так, то карта DS, индексируемая в переменной экземпляра "inventory", будет очищена от всех пар ключ/значение (но не уничтожена ).
