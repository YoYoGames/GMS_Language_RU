ds_map_secure_save_buffer Эта функция сохраняет ранее созданную карту DS в буфер. Вы передаете значение ID карты DS (возвращаемое функцией ds_map_create()) и идентификатор буфера для записи (возвращаемый функцией buffer_create()). Обратите внимание, что если сохраняемая карта DS содержит массив, то при сохранении он будет преобразован в список DS. ВАЖНО! Защищенный сохраненный файл карты DS можно загрузить только на то устройство, которое его создало, и если вы попытаетесь загрузить файл, сохраненный на другом устройстве, то он не будет работать. Синтаксис: ds_map_secure_save_buffer(filename); Аргумент Описание id Значение идентификатора карты DS. буфер Значение идентификатора буфера. Возвращается: Целое число (индекс карты DS) Пример: buff = buffer_create(128,buffer_grow,4); var map = ds_map_create(); ds_map_add(map, "bob", "ajob"); ds_map_add(map, "money",10); ds_map_secure_save_buffer(map, buff); ds_map_destroy(map); Приведенный выше код создаст буфер и карту DS, затем заполнит карту некоторыми значениями и запишет их в буфер перед удалением карты.
