ds_map_values_to_array С помощью этой функции вы можете получить все значения, которые содержит карта DS для каждого ключа в карте. Вы передаете идентификатор карты DS для получения ключей (как возвращается функцией ds_map_create()) и вернет массив, каждая запись в котором является значением из карты DS (например, если карта имеет 10 ключей, то будет возвращен массив из 10 записей, каждая из которых соответствует значению ключа). Функция имеет необязательный второй аргумент, в качестве которого вы можете указать созданный вами массив, в этом случае данные о значении ключа карты будут добавлены ко всем существующим данным в массиве. Обратите внимание, что функция будет изменять массив, но также возвращает ссылку на него (или ссылку на новый массив, если он не указан). Синтаксис: ds_map_values_to_array(id, [array]) Аргумент Описание id Идентификатор карты, которую нужно использовать. [массив] ВАРИАНТ! Массив для добавления значений карты DS. Возвращается: Массив Пример: var _values = ds_map_values_to_array(lvl_score); var _total = 0; for (var i = 0; i < array_lengh(values); ++i;) { _total += _values[i]; } draw_text(32, 32, "Общий балл для всех уровней = " + string(_total")); Вышеприведенный пример извлекает все значения из карты DS для каждого ключа в карте и сохраняет их в локальной переменной в виде массива. Затем выполняется циклический просмотр массива и добавление каждого значения в локальную переменную, которая затем используется для отображения общей суммы всех значений. значений.
