ds_map_is_list С помощью этой функции вы можете проверить, хранится ли список DS в заданном ключе карты. Если заданный ключ содержит идентификатор списка DS, то функция вернет true, в противном случае она вернет false. Синтаксис: ds_map_is_list(id, key) Аргумент Описание id Идентификатор используемой карты ds_map. ключ Ключ для проверки. Возвращается: Булево Пример: var size = ds_map_size(inventory) ; var key = ds_map_find_first(inventory); for (var i = 0; i < size; i++;) { if ds_map_is_list(inventory, key) { ds_list_destroy(key); } key = ds_map_find_next(inventory); } ds_map_destroy(inventory); Приведенный выше код выполняет цикл по карте DS и проверяет, не являются ли какие-либо ключи в ней ключами для списка DS. Если да, то список DS уничтожается, а в конце цикла уничтожается и карта DS. Назад: Карты DS Следующий: ds_map_is_map © Copyright YoYo Games Ltd. 2021 Все права защищены
