ds_map_replace_list С помощью этой функции вы можете заменить список DS, который был сохранен в заданном "ключе", на другой список, созданный ранее. Эта функция предназначена для создания JSON-совместимых карт, которые вы затем закодируете с помощью json_encode() и должна использоваться только в сочетании с этой функцией. Синтаксис: ds_map_replace_list(id, key, value) Аргумент Описание id Идентификатор карты, которую нужно использовать. ключ Ключ на замену. значение Идентификатор списка ds_list, который будет использоваться для замены списка, ранее хранившегося в данном ключе. Возвращается: Н/Д Пример: var j_list = ds_list_create(); ds_list_add(j_list, health); ds_list_add(j_list, lives); ds_list_add(j_list, score); ds_map_replace_list(j_map, "list", j_list); var j = json_encode(j_map); ds_list_destroy(j_list); Приведенный выше код создаст DS List и заполнит его значениями различных глобальных переменных, а затем заменит ранее сохраненный список в DS Map "j_map".
