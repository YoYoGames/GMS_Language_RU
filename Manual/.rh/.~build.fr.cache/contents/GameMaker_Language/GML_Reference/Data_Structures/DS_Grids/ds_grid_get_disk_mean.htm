ds_grid_get_disk_mean Эта функция может быть использована для нахождения среднего значения для всех ячеек, находящихся в пределах определенной области диска сетки (значения всех ячеек складываются, а затем делятся на общее количество ячеек, составляющих диск), как показано на рисунке ниже: Синтаксис: ds_grid_get_disk_mean(index, xm, ym, r); Аргумент Описание индекс Индекс сетки. xm Позиция x диска на сетке. ym Позиция y диска на сетке. r Радиус диска на сетке. Возвращается: Вещественный или строка Пример: val = ds_grid_get_disk_mean(grid, 5, 5, 2) Приведенный выше код установит переменную "val" в среднее значение, содержащееся в данном диске сетки DS, индексируемой в переменной "grid".
