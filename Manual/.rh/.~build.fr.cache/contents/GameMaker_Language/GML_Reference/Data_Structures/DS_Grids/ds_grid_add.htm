ds_grid_add Эта функция может быть использована для добавления заданного значения (вещественного или строкового) к значению заданной ячейки в сетке. Добавляемое значение должно быть того же типа, что и ячейка сетки, т.е. вы не можете добавить строку к вещественному значению или наоборот. строк это соответствует конкатенации. Синтаксис: ds_grid_add(index, x, y, val); Аргумент Описание индекс Индекс сетки. x Позиция x ячейки в сетке. y Положение y ячейки в сетке. val Значение для добавления в ячейку. Возвращается: Н/Д Пример: ds_grid_add(grid, 5, 5, 6) Это добавит 6 к данной ячейке в сетке DS, на которую ссылается переменная "grid".
