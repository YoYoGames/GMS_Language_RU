ds_grid_value_x С помощью этой функции вы можете получить координату x (в пределах заданной прямоугольной области сетки) искомого значения. Вы задаете индекс сетки DS (как возвращается функцией ds_grid_create()) вместе с позициями x/y для верхнего левого и нижнего правого углов области сетки для поиска, а затем вводите значение для поиска. Если искомое значение не существует, то функция вернет -1. Синтаксис: ds_grid_value_x(index, x1, y1, x2, y2, val); Аргумент Описание индекс Индекс сетки. x1 Положение x левой части области в сетке, от 0 до (ширина сетки - 1). y1 Положение y верхней части области в сетке, от 0 до (высота сетки - 1). x2 Положение x правой части области в сетке, от 0 до (ширина сетки - 1). y2 Положение y нижней части области в сетке, от 0 до (высота сетки - 1). val Значение, которое нужно найти. Возвращается: Реально Пример: if ds_grid_value_exists(grid, 0, 1, 5, 6, val) { xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val); ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val); } Приведенный выше код проверяет ds_grid на наличие определенного значения в прямоугольной области. Если оно найдено, то сохраняет x и y позицию значения в двух переменных для последующего использования. Назад: DS Grids Далее: ds_grid_value_y © Copyright YoYo Games Ltd. 2021 Все права защищены
