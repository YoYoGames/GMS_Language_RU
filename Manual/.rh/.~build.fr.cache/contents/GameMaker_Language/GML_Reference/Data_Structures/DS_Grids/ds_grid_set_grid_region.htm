ds_grid_set_grid_region Эта функция может быть использована для копирования содержимого прямоугольной области ячеек сетки из одной (ранее определенной) сетки DS в другую, или она может быть использована для копирования области внутри той же сетки. Следующие изображения иллюстрируют оба способа можно использовать эту функцию: Синтаксис: ds_grid_set_grid_region(index, source, x1, y1, x2, y2, xpos, ypos); Аргумент Описание индекс Индекс сетки назначения. источник Индекс исходной сетки. x1 Левая позиция области ячеек для копирования из исходной сетки. y1 Верхняя позиция области ячеек для копирования из исходной сетки. x2 Правая позиция области ячеек для копирования из исходной сетки. y2 Нижняя позиция области ячеек для копирования из исходной сетки. xpos Позиция x на сетке назначения для копирования исходного региона. ypos Позиция y на сетке назначения для копирования исходного региона. Возвращается: Н/Д Пример: ds_grid_set_grid_region(grid, t_grid, 0, 0, 5, 5, 5, 10, 10) Это скопирует область ячеек от (0,0) до (5,5) сетки, индексированной в переменной "t_grid", и скопирует их в позицию (10,10) в сетке, индексированной в переменной "grid".
