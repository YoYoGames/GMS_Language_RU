ds_grid_get_disk_min Эта функция может быть использована для нахождения минимального значения для всех ячеек, находящихся в пределах определенной области диска сетки, как показано на рисунке ниже: Синтаксис: ds_grid_get_disk_min(index, xm, ym, r); Аргумент Описание индекс Индекс сетки. xm Позиция x диска на сетке. ym Позиция y диска на сетке. r Радиус диска на сетке. Возвращается: Вещественный или строка Пример: val = ds_grid_get_disk_min(grid, 5, 5, 2) Приведенный выше код установит переменную "val" в минимальное значение, содержащееся в данном диске сетки DS, индексируемой в переменной "grid".
