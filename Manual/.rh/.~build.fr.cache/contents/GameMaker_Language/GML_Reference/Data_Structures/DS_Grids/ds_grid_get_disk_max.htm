ds_grid_get_disk_max Эта функция может быть использована для нахождения максимального значения для всех ячеек, находящихся в пределах определенной области диска сетки, как показано на рисунке ниже: Синтаксис: ds_grid_get_disk_max(index, xm, ym, r); Аргумент Описание индекс Индекс сетки. xm Позиция x центра диска в сетке. ym Положение y центра диска в сетке. r Радиус диска в сетке. Возвращается: Вещественный или строка Пример: val = ds_grid_get_disk_max(grid, 5, 5, 2) Приведенный выше код установит переменную "val" в максимальное значение, содержащееся в данном диске сетки DS, индексируемой в переменной "grid".
