ds_grid_create С помощью этой функции вы можете создать новую структуру данных сетки DS с указанной шириной и высотой ячеек. Эта функция возвращает идентификатор, который должен использоваться во всех последующих функциях, работающих с этой сеткой DS. ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целым числом, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, вам следует использовать значение индекса ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано впоследствии вновь созданной. Синтаксис: ds_grid_create(w, h); Аргумент Описание w Ширина создаваемой сетки. h Высота создаваемой сетки. Возвращается: Реально Пример: mygrid = ds_grid_create(10, 10) Это создает сетку высотой 10 ячеек и шириной 10 ячеек.
