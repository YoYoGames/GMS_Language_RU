ds_grid_add_region Эта функция может быть использована для добавления заданного значения (вещественного или строкового) ко всем значениям ячеек, находящихся в определенной области сетки. Добавляемое значение должно быть того же типа, что и значения в ячейках сетки, т.е. нельзя добавить строку к вещественному или наоборот, а для строк это соответствует конкатенации. Синтаксис: ds_grid_add_region(index, x1, y1, x2, y2, val); Аргумент Описание индекс Индекс сетки. x1 Положение x левой части области в сетке. y1 Положение y верхней части области в сетке. x2 Положение x правой части области в сетке. y2 Положение y нижней части области в сетке. val Значение для добавления в ячейки региона. Возвращается: Н/Д Пример: ds_grid_add_region(grid, 2, 4, 5, 5, ".") Это добавит "." ко всем строкам, хранящимся в ячейках внутри определенной области сетки DS, на которую ссылается переменная "grid".
