ds_grid_set_region Эта функция может быть использована для установки прямоугольной области заданной сетки на заданное значение (которое может быть вещественным или строковым), как показано на изображении ниже: Синтаксис: ds_grid_set_region(index, x1, y1, x2, y2, val); Аргумент Описание индекс Индекс сетки. x1 Положение x левой части области в сетке. y1 Положение y верхней части области в сетке. x2 Положение x правой части области в сетке. y2 Положение y нижней части области в сетке. val Значение, на которое нужно установить ячейки региона. Возвращается: Н/Д Пример: ds_grid_set_region(grid, 5, 5, 10, 10, 99) В результате все ячейки в области сетки, индексируемой переменной "grid" от (5,5) до (10,10), будут иметь значение 99.
