ds_grid_get_max Эта функция может быть использована для нахождения максимального значения для всех ячеек, находящихся в заданной области сетки, как показано на рисунке ниже: Синтаксис: ds_grid_get_max(index, x1, y1, x2, y2); Аргумент Описание индекс Индекс сетки. x1 Левый столбец ячеек региона. y1 Верхний ряд ячеек региона. x2 Правый столбец ячеек региона. y2 Нижний ряд ячеек региона. Возвращается: Вещественный или строка Пример: val = ds_grid_get_max(grid, 0, 0, 0, 5, 5) Приведенный выше код установит переменную "val" в максимальное значение, содержащееся в данной области сетки DS, индексируемой в переменной "grid".
