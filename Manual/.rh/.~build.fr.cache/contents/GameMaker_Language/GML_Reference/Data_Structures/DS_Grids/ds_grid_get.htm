ds_grid_get Эта функция может использоваться для получения значения (либо вещественного числа, либо строки) из любой ячейки в заданной сетке DS. Если вы передадите в функцию недопустимые координаты сетки, то возвращаемое значение будет неопределенным и будет показана ошибка в окне вывода. Синтаксис: ds_grid_get(index, x, y); Аргумент Описание индекс Индекс сетки. x Позиция x ячейки, значение которой вы хотите найти. y Позиция y ячейки, значение которой вы хотите найти. Возвращается: Вещественный или строка Пример: var xx = irandom(ds_grid_width(grid) - 1); var yy = irandom(ds_grid_height(grid) - 1); val = ds_grid_get(grid, xx, yy) Приведенный выше код выбирает случайную ячейку из сетки DS, проиндексированную в переменной "grid", и сохраняет ее значение в переменной "val".
