ds_grid_get_min Эта функция может быть использована для нахождения минимального значения для всех ячеек, находящихся в заданной области сетки, как показано на рисунке ниже: Синтаксис: ds_grid_get_min(index, x1, y1, x2, y2); Аргумент Описание индекс Индекс сетки. x1 Левый столбец ячеек региона. y1 Верхний ряд ячеек региона. x2 Правый столбец ячеек региона. y2 Нижний ряд ячеек региона. Возвращается: Вещественный или строка Пример: val = ds_grid_get_min(grid, 0, 0, 0, 5, 5) Приведенный выше код установит переменную "val" в минимальное значение, содержащееся в заданной области сетки DS, индексированной в переменной "grid".
