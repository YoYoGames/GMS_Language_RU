ds_grid_shuffle Эта функция может быть использована для рандомизации положения всех значений во всех ячейках сетки. ПРИМЕЧАНИЕ: Эта функция будет перемешивать ячейки сетки на одни и те же позиции при каждом новом запуске игры, поскольку GameMaker Studio 2 каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры. Это справедливо только при тестировании и отладке игры, так как конечный исполняемый пакет не будет демонстрировать такое поведение и будет случайным в каждой игре. Синтаксис: ds_grid_shuffle(index); Аргумент Описание индекс Индекс сетки для перетасовки. Возвращается: Н/Д Пример: ds_grid_shuffle(grid) Это позволит взять все значения в сетке DS, проиндексированные в переменной "grid", и перетасовать их, чтобы получить другой порядок.
