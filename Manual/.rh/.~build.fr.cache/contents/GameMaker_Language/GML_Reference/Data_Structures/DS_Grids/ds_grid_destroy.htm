ds_grid_destroy Эта функция удалит заданную структуру данных грида из памяти, освободив ресурсы, которые она использовала, и удалив все значения, которые она содержала. Эта функция всегда должна использоваться после завершения работы с сеткой DS, чтобы предотвратить утечку памяти. которые могут замедлить работу и привести к краху вашей игры. ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целым числом, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, вам следует использовать значение индекса ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано вновь созданной, поэтому мы рекомендуем всегда устанавливать переменную, в которой хранился индекс DS, в -1 после уничтожения. Синтаксис: ds_grid_destroy(index); Аргумент Описание индекс Индекс сетки для уничтожения. Возвращается: Н/Д Пример: if (lives == 0) { ds_grid_destroy(Wall_Grid); Wall_Grid = -1; room_goto(rm_Menu); } Приведенный выше код проверит значение встроенной глобальной переменной "lives" и, если оно равно 0, уничтожит сетку DS, индексированную в переменной "Wall_Grid", а затем поменяет комнаты.
