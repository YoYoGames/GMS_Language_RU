ds_grid_multiply_grid_region С помощью этой функции вы можете определить область в заданной сетке DS, затем взять эти значения и перемножить их со значениями, найденными в отдельной области либо той же сетки DS, либо другой (которая была создана ранее). Исходная область останется неизменным, в то время как область, с которой они были перемножены, будет хранить новые значения для каждой ячейки. Синтаксис: ds_grid_multiply_grid_region(index, source, x1, y1, x2, y2, xpos, ypos); Аргумент Описание индекс Индекс сетки назначения. источник Индекс исходной сетки. x1 Левая позиция области ячеек для копирования из исходной сетки. y1 Верхняя позиция области ячеек для копирования из исходной сетки. x2 Правая позиция области ячеек для копирования из исходной сетки. y2 Нижняя позиция области ячеек для копирования из исходной сетки. xpos Позиция x на сетке назначения для умножения исходного региона. ypos Позиция y на сетке назначения для умножения исходного региона. Возвращается: Н/Д Пример: ds_grid_multiply_grid_region(mygrid, mygrid, 0, 0, 5, 5, 0, 0) Для этого нужно взять область ячеек от (0,0) до (5,5) сетки DS "mygrid" и перемножить их с ячейками из позиции (0,0) той же сетки DS.
