ds_grid_value_exists С помощью этой функции вы можете проверить, присутствует ли определенное значение (вещественное или строковое) в прямоугольной области заданной сетки DS. Если оно присутствует, функция вернет true, в противном случае - false. Синтаксис: ds_grid_value_exists(index, x1, y1, x2, y2, val); Аргумент Описание индекс Индекс сетки. x1 Положение x левой части области в сетке. y1 Положение y верхней части области в сетке. x2 Положение x правой части области в сетке. y2 Положение y нижней части области в сетке. val Значение, которое нужно найти. Возвращается: Булево Пример: if ds_grid_value_exists(grid, 0, 1, 5, 6, val) { xpos = ds_grid_value_x(grid, 0, 1, 5, 6, val); ypos = ds_grid_value_y(grid, 0, 1, 5, 6, val); } Приведенный выше код проверяет сетку DS на наличие определенного значения в прямоугольной области. Если оно найдено, то сохраняет x и y позицию значения в двух переменных для последующего использования.
