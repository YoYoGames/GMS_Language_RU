ds_grid_value_disk_exists С помощью этой функции можно проверить, присутствует ли определенное значение (вещественное или строковое) в пределах круговой области заданной сетки DS. Если оно присутствует, функция вернет true, в противном случае - false. Синтаксис: ds_grid_value_disk_exists(index, xm, ym, r, val); Аргумент Описание индекс Индекс сетки. xm Позиция x диска на сетке. ym Позиция y диска на сетке. r Радиус диска на сетке. val Значение, которое нужно найти. Возвращается: Булево Пример: if ds_grid_value_disk_exists(grid, 5, 5, 5, 5, val) { xpos = ds_grid_value_disk_x(grid, 5, 5, 5, 5, val); ypos = ds_grid_value_disk_y(grid, 5, 5, 5, 5, val); } Приведенный выше код проверяет сетку DS на наличие определенного значения в области диска. Если оно найдено, то сохраняет x и y позицию значения в двух переменных для последующего использования. Назад: DS Grids Далее: ds_grid_value_x © Copyright YoYo Games Ltd. 2021 Все права защищены
