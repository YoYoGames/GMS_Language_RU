ds_grid_get_sum Эта функция может быть использована для суммирования всех значений всех ячеек, находящихся в определенной области сетки, как показано на рисунке ниже: Синтаксис: ds_grid_get_sum(index, x1, y1, x2, y2); Аргумент Описание индекс Индекс сетки. x1 Левый столбец ячеек региона. y1 Верхний ряд ячеек региона. x2 Правый столбец ячеек региона. y2 Нижний ряд ячеек региона. Возвращается: Реально Пример: val = ds_grid_get_sum(grid, 0, 0, 0, 5, 5) Приведенный выше код установит переменную "val" в сумму всех значений, содержащихся в данной области сетки DS, индексированной в переменной "grid".
