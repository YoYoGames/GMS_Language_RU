ds_priority_change_priority Эта функция принимает заданное значение и изменяет его приоритет в указанной очереди приоритетов. Синтаксис: ds_priority_change_priority(id, val, priority); Аргумент Описание id Идентификатор очереди приоритетов, который нужно изменить. val Значение для изменения приоритета. приоритет Новый приоритет значения. Возвращается: Н/Д Пример: if (global.Game_Time < 1000) { ds_priority_change(ai_priority, AI_Search, 1); } Приведенный выше код проверяет глобальную переменную, и если она ниже определенного значения, то изменяет приоритет индекса функции сценария, находящегося в очереди приоритетов.
