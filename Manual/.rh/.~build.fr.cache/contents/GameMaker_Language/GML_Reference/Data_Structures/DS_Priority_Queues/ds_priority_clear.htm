ds_priority_clear С помощью этой функции вы можете очистить все данные из заданной структуры данных очереди приоритетов. Это НЕ уничтожает структуру данных (для этого вы должны использовать функцию ds_priority_destroy()), а только стирает все данные из нее и возвращает пустую очередь приоритетов. Синтаксис: ds_priority_clear(id); Аргумент Описание id Идентификатор структуры данных, которую нужно очистить. Возвращается: Н/Д Пример: if (count = 15) && (!ds_priority_empty(command_queue)) { ds_priority_clear(command_queue); alarm[0] = room_speed; ai_count = 0; } Приведенный выше код проверяет переменную, достигла ли она определенного значения, и если достигла, то очищает очередь приоритетов DS, указанную в переменной "command_queue", устанавливает сигнал тревоги и сбрасывает переменную в 0.
