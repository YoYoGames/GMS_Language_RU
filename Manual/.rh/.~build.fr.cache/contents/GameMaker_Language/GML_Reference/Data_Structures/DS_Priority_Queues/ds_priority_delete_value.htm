ds_priority_delete_value Эта функция просто удалит заданное значение вместе с его приоритетом из индексированной очереди приоритетов. Синтаксис: ds_priority_delete_value(id,val); Аргумент Описание id Идентификатор используемой очереди приоритетов. val Значение для удаления из очереди приоритетов. Возвращается: Н/Д Пример: if (ai_move == false) { ds_priority_delete_value(ai_priority, AI_Move); } Приведенный выше код проверяет переменную экземпляра, и если она возвращает false, то удаляет индексированную функцию сценария из очереди приоритетов.
