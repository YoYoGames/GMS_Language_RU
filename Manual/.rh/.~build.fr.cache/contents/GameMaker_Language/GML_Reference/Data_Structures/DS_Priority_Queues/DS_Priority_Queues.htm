Очереди приоритетов DS Очередь приоритетов DS - это специальная структура данных, похожая на очередь приоритетов DS. Очередь DSтолько здесь очередь значений упорядочена по приоритетам (всегда вещественные числа), которые были присвоены значениям. Эта структура может быть очень полезна для построения таблиц лидеров или информационных списков, где важность каждой записи может быть взвешена. ПРИМЕЧАНИЕ: Как и все динамические ресурсы, структуры данных занимают память, поэтому их всегда следует уничтожать, когда они больше не нужны, чтобы предотвратить утечку памяти, которая замедлит и в конечном итоге приведет к краху вашей игры. ПРИМЕЧАНИЕ: Если вам нужно проверить, существует ли структура данных, вы можете использовать функцию ds_exists() функцию.
