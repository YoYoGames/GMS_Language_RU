ds_priority_find_priority С помощью этой функции вы можете получить приоритет любого заданного значения. Если значение не существует в очереди приоритетов, то будет возвращено значение undefined. Синтаксис: ds_priority_find_priority(id, val); Аргумент Описание id Идентификатор используемой очереди приоритетов. val Значение, для которого нужно найти приоритет. Возвращается: Реальный или неопределенный Пример: p = ds_priority_find_priority(ai_priority, "intelligence"); Приведенный выше код будет хранить возвращенный приоритет для заданного значения в переменной экземпляра "p".
