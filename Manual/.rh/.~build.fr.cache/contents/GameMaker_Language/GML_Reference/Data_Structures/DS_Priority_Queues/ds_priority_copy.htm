ds_priority_copy Эта функция может быть использована для копирования содержимого одной очереди приоритетов в другую. Обратите внимание, что при этом НЕ удаляется содержимое исходной очереди приоритетов и не уничтожается исходная очередь приоритетов. При использовании этой функции приоритетная очередь, в которую копируется, должна быть создана ранее, и если она содержала какие-либо элементы до копирования, то они будут очищены первыми (это означает, что информация будет потеряна). Синтаксис: ds_priority_copy(id, source); Аргумент Описание id Идентификатор приоритетной очереди для копирования . источник Идентификатор приоритетной очереди для копирования. Возвращается: Н/Д Пример: with (instance_create_layer(x, y, "Enemies", obj_Enemy)) { p_queue = ds_priority_create(); ds_priority_copy(p_queue, other.p_queue); } Приведенная выше функция создает новый экземпляр, а затем в этом экземпляре создает новую очередь приоритетов DS и копирует содержимое очереди приоритетов в экземпляре, выполняющем блок кода, во вновь созданную очередь приоритетов экземпляра.
