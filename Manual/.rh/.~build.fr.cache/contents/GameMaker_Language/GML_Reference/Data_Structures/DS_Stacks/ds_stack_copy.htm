ds_stack_copy Эта функция может использоваться для копирования содержимого одного стека в другой. Обратите внимание, что при этом НЕ удаляется содержимое из исходного стека и не уничтожается исходный стек. При использовании этой функции стек, в который производится копирование, должен быть был создан ранее, и если он содержал какие-либо элементы до копирования, то они будут очищены первыми (это означает, что информация будет потеряна). Синтаксис: ds_stack_copy(id, source); Аргумент Описание id Идентификатор нового стека. источник ОРИГИНАЛЬНЫЙ стек для копирования в id. Возвращается: Н/Д Пример: with (instance_create_layer(x, y, "Enemies", obj_Enemy)) { stack = ds_stack_create(); ds_stack_copy(stack, other.stack); } Приведенная выше функция создает новый экземпляр, а затем в этом экземпляре создает новый стек DS и копирует содержимое стека в экземпляре, выполняющем блок кода, во вновь созданный стек экземпляра.
