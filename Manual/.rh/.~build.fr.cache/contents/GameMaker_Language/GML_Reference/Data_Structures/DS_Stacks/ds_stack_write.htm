ds_stack_write Эта функция возвращает строку, которая затем может быть сохранена или передана в другую структуру данных с помощью функции ds_stack_read() функцию. ПРИМЕЧАНИЕ: Возвращаемая строка не является человекочитаемой строкой, а скорее дампом содержимого структуры данных. Синтаксис: ds_stack_write(id); Аргумент Описание id Идентификатор структуры данных для записи. Возвращается: Строка Пример: ini_open("save.ini"); var str = ds_stack_write(stack); ini_write_string("Stacks", "0", str); ds_stack_clear(stack); ini_close(); Приведенный выше код открывает ini-файл, а затем записывает строку, содержащую информацию, хранящуюся в стеке DS, индексированном в переменной "stack".
