ds_stack_size Эта функция возвращает "размер" стека, т.е. количество элементов, которые были на него помещены. Синтаксис: ds_stack_size(id); Аргумент Описание id Идентификатор структуры данных для проверки. Возвращается: Реально Пример: if !ds_stack_empty(control_stack) { num = ds_stack_size(control_stack); } Приведенный выше код проверяет стек DS на то, пуст он или нет, и если нет, то получает количество элементов, которые он содержит, и сохраняет это значение в переменной.
