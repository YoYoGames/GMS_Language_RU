ds_stack_pop Эта функция снимает верхнее значение со стека DS, удаляет его из стека и возвращает значение для хранения в переменной. Если стек пуст, то функция вернет константу undefined, в противном случае она вернет вещественное или строковое значение, содержащееся в стеке. Синтаксис: ds_stack_pop(id); Аргумент Описание id Идентификатор структуры данных, из которой нужно выполнить выборку. Возвращается: Значение типа данных, хранящееся в стеке или неопределенное Пример: if !ds_stack_empty(move_stack) { var xx = ds_stack_pop(move_stack); var yy = ds_stack_pop(move_stack); move_towards_point(xx, yy, 4); } Приведенный выше код проверяет стек DS, указанный в переменной "move_stack", на то, пуст ли он, и если нет, то извлекает два верхних значения из стека и использует их для задания направления движения.
