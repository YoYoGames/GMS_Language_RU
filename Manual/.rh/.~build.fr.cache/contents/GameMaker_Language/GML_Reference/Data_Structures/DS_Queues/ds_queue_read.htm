ds_queue_read С помощью этой функции можно воссоздать сохраненную очередь DS (ту, которая ранее была записана в виде строки с помощью функции ds_queue_write()). Сначала вы должны создать новую очередь DS для чтения строки, и если очередь DS уже существует и в ней хранится информация, то она будет очищена перед чтением. Эта функция имеет жизненно важное значение при создании механизмов сохранения/загрузки для вашей игры. Обратите внимание, что если структура данных была создана в предыдущих версиях программы GameMaker вам следует добавить необязательный аргумент "legacy", установив его в true, так как формат строк в этой версии изменился. Синтаксис: ds_queue_read(id, str [, legacy]); Аргумент Описание id Идентификатор структуры данных для чтения. str Строка для чтения. наследие(необязательно) Может быть как истинным, так и ложным или полностью отсутствовать. Возвращается: Н/Д Пример: queue = ds_queue_create(); ini_open("save.ini"); var str = ini_read_string("Queues", "0", ""); if (str != "") { ds_queue_read(queue, str); } ini_close(); Приведенный выше код создает очередь и сохраняет индекс в переменной "queue". Затем он открывает ini-файл и считывает из него строку, проверяя, что сначала строка не возвращается пустой. Затем эта строка считывается в только что созданную очередь DS.
