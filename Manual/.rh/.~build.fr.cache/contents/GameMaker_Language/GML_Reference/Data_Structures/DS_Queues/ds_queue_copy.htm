ds_queue_copy Эта функция может быть использована для копирования содержимого одной очереди в другую. Обратите внимание, что при этом НЕ удаляется содержимое исходной очереди и не уничтожается исходная очередь. При использовании этой функции очередь, в которую копируется содержимое, должна быть была создана ранее, и если она содержала какие-либо элементы до копирования, то они будут очищены первыми (это означает, что информация будет потеряна). Синтаксис: ds_queue_copy(id, source); Аргумент Описание id Идентификатор НОВОЙ очереди. источник ОРИГИНАЛЬНАЯ очередь для копирования в id. Возвращается: Н/Д Пример: with (instance_create_layer(x, y, "Enemies", obj_Enemy)) { queue = ds_queue_create(); ds_queue_copy(queue, other.queue); } Приведенная выше функция создает новый экземпляр, а затем в этом экземпляре создает новую очередь DS и копирует содержимое очереди в экземпляре, выполняющем блок кода, во вновь созданную очередь экземпляра.
