ds_queue_empty С помощью этой функции вы можете проверить заданную очередь DS на предмет того, пуста она (возвращается true) или нет (возвращается false). Синтаксис: ds_queue_empty(id); Аргумент Описание id Идентификатор структуры данных для проверки. Возвращается: Булево Пример: if (count == 15) && (!ds_queue_empty(command_queue)) { ds_queue_clear(command_queue); alarm[0] = room_speed; ai_count = 0; } Приведенный выше код проверяет переменную, достигла ли она определенного значения, и если достигла, то очищает очередь DS, указанную в переменной "command_queue", устанавливает сигнал тревоги и сбрасывает переменную в 0.
