ds_queue_tail Эта функция считывает только последнее значение очереди (то, которое находится "в хвосте"). Она не удаляет значение из очереди, что означает, что оно все еще может быть прочитано в будущем этой функцией или функцией ds_queue_dequeue(). Синтаксис: ds_queue_tail(id); Аргумент Описание id Идентификатор структуры данных для чтения. Возвращается: Значение типа данных, хранящееся в очереди Пример: num = ds_queue_tail(control_queue); Приведенный выше код прочитает значение из очереди, проиндексированной в переменной "control_queue", и сохранит возвращаемое значение в переменной "num".
