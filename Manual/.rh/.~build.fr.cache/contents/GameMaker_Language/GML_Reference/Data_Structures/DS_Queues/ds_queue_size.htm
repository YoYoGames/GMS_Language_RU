ds_queue_size Эта функция возвращает "размер" очереди, т.е. количество элементов, которые были поставлены в очередь. Синтаксис: ds_queue_size(id); Аргумент Описание id Идентификатор структуры данных для проверки. Возвращается: Реально Пример: if !ds_queue_empty(control_queue) { num = ds_queue_size(control_queue); } Приведенный выше код проверяет очередь DS на то, пуста она или нет, и если нет, то получает количество элементов, которые она содержит, и сохраняет это значение в переменной.
