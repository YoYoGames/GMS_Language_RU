ds_queue_enqueue Эта функция добавляет значение (вещественное или строковое) в хвост очереди DS. Функция может принимать еще 14 необязательных аргументов (всего 15 возможных добавлений), что позволяет добавлять несколько значений последовательно в хвост очереди за один вызов. Синтаксис: ds_queue_enqueue(id, val [, val2, ... val15]); Аргумент Описание id Идентификатор очереди для добавления. val Значение для добавления в очередь. [val2, ... val15] Необязательные значения для добавления в очередь. Возвращается: Н/Д Пример: move_queue = ds_queue_create(); ds_queue_enqueue(move_queue, x + 200); ds_queue_enqueue(move_queue, y); ds_queue_enqueue(move_queue, x + 200); ds_queue_enqueue(move_queue, y + 200); ds_queue_enqueue(move_queue, x); ds_queue_enqueue(move_queue, y + 200); ds_queue_enqueue(move_queue, x); ds_queue_enqueue(move_queue, y); Приведенный выше код создает новую очередь DS и сохраняет ее индекс в переменной "move_queue".
