ds_list_copy С помощью этой функции можно скопировать содержимое одного списка в другой. Оба списка должны быть созданы ранее, и если в списке, в который копируется информация, уже есть информация, этот список будет очищен первым. В результате вы получите два независимых списка, содержащие одну и ту же информацию. Синтаксис: ds_list_copy( id, source ); Аргумент Описание id Идентификатор списка, в который производится копирование. источник Идентификатор списка, из которого будет производиться копирование. Возвращается: Н/Д Пример: if !ds_list_empty(main_list) { old_list = ds_list_create(); ds_list_copy(old_list, main_list); ds_list_clear(main_list); } Приведенный выше код проверяет список DS на предмет его пустоты. Если он не пуст, он копируется в другой список DS (который был создан ранее), а затем исходный список очищается.
