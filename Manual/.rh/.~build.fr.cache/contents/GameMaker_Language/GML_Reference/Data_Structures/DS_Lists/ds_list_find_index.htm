ds_list_find_index С помощью этой функции вы можете проверить заданный список на наличие значения, после чего будет возвращена позиция в списке для этого значения. Обратите внимание, что если в списке несколько записей с одинаковым значением, то может быть возвращена позиция любой из них, и если значение не существует, то будет возвращено -1. Обратите внимание, что значение может быть массивом, что можно проверить с помощью функции is_array(). Синтаксис: ds_list_find_index(id, val); Аргумент Описание id Идентификатор списка для использования. val Значение, которое нужно найти. Возвращается: Реально Пример: pos = ds_list_find_index(list, "Player1"); Приведенный выше код проверяет список, проиндексированный в переменной "list", на наличие значения "Player1" и сохраняет возвращенную позицию в переменной "pos".
