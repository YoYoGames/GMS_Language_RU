ds_list_destroy Эта функция удалит заданную структуру данных списка из памяти, освободив используемые ею ресурсы и удалив все содержащиеся в ней значения. Эту функцию всегда следует использовать после завершения работы со списком ds_list, чтобы предотвратить утечку памяти. которые могут замедлить работу и привести к краху вашей игры. Обратите внимание, что при уничтожении списка все хранящиеся в нем структуры данных будут удалены, что приведет к утечке памяти, поэтому перед уничтожением списка необходимо пройтись по списку и уничтожить все элементы структуры данных вручную. чтобы предотвратить это. Единственный случай, когда это не требуется, это когда вы отметили элементы в списке как другой список DS или как карту DS, в этом случае эти элементы будут уничтожены и их память также будет очищена автоматически. ВАЖНО! Когда вы создаете структуру данных, значение индекса для ее идентификации является целым числом, начинающимся с 0. Это означает, что структуры данных разных типов могут иметь одинаковое значение индекса, поэтому, если вы сомневаетесь, вам следует использовать значение индекса ds_exists() функцию перед обращением к ним. Также обратите внимание, что индексы используются повторно, поэтому значение индекса уничтоженной структуры данных может быть использовано вновь созданной, поэтому мы рекомендуем всегда устанавливать переменную, в которой хранился индекс DS, в -1 после уничтожения. Синтаксис: ds_list_destroy(id); Аргумент Описание id Идентификатор структуры данных для удаления. Возвращается: Н/Д Пример: if (lives == 0) { ds_list_destroy(AI_list); AI_list = -1; room_goto(rm_Menu); } Приведенный выше код проверит значение встроенной глобальной переменной "lives" и, если оно равно 0, уничтожит список DS, индексированный в переменной "AI_list", а затем сменит комнаты.
