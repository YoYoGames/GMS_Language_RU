ds_list_find_value С помощью этой функции вы можете проверить заданную позицию списка и вернуть значение, хранящееся в списке для этой позиции. Обратите внимание, что если вы зададите позицию, которая находится за пределами заданного размера списка (например, позиция 11 в списке из 10 значений), то функция может вернуть неопределенное значение или 0. Это связано с тем, что при создании списка первые несколько записей в списке устанавливаются в 0, чтобы минимизировать проблемы с производительностью при первоначальном добавлении элементов в список (хотя функция ds_list_size() функция все равно вернет 0 для только что созданного списка). Если вы хотите убедиться, что список "действительно" пуст при создании, то вам следует вызвать функцию ds_list_clear() после создания списка, и тогда означает, что любые значения, возвращаемые для незаполненных слотов списка, будут неопределенными, что вы сможете последовательно проверить с помощью функции is_undefined() функцию. Синтаксис: ds_list_find_value(id, pos); Аргумент Описание id Идентификатор списка для использования. поз Позиция для просмотра, где 0 соответствует самому началу списка, а конечная позиция равна ds_list_size(id)-1. Возвращается: Вещественный, строковый или неопределенный Пример: val = ds_list_find_value(list, ds_list_size(list) - 1); Приведенный выше код проверяет список, проиндексированный в переменной "list", на последней позиции в списке и сохраняет возвращаемое значение в переменной "val".
