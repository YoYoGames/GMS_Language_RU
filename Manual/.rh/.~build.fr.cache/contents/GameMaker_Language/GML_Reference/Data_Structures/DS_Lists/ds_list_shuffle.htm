ds_list_shuffle С помощью этой функции вы можете перетасовать список, в результате чего все значения компонентов будут перестроены в случайные позиции по сравнению с теми, в которых они были первоначально добавлены в список. ПРИМЕЧАНИЕ: Эта функция будет перемешивать элементы списка на одни и те же позиции при каждом новом запуске игры, поскольку GameMaker Studio 2 каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры. Это справедливо только при тестировании и отладке игры, поскольку конечный исполняемый пакет не будет демонстрировать такое поведение и будет случайным в каждой игре. Синтаксис: ds_list_shuffle(id); Аргумент Описание id Идентификатор списка, который нужно перетасовать. Возвращается: Н/Д Пример: if restart { ds_list_shuffle(card_list); } Приведенный выше код перетасует список, индексированный в переменной "card_list", если переменная "restart" отмечена как true.
