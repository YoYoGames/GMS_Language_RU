ds_list_set Эта функция может использоваться для установки ранее добавленного элемента списка. Вы указываете ID списка (полученный при создании списка) и позицию в списке, которую нужно установить, а также значение, которое нужно установить. Обратите внимание, что если устанавливаемый элемент находится вне границ списка (т.е. вы задаете элемент списка 20, но текущий список содержит только 10 элементов), то список будет заполнен до заданной позиции, а каждый элемент будет установлен в 0. Эта функция аналогична использованию аксессора DS list. Синтаксис: ds_list_set(id, pos, val); Аргумент Описание id Идентификатор списка для добавления. поз Позиция в списке, которую нужно установить. [val2, ... max_val] Значение для установки в списке. Возвращается: Н/Д Пример: for (var i = 0; i < ds_list_size(list); i++;) { ds_list_set(list, i, -1); } Приведенный выше код добавит значение, хранящееся в переменной "score", в список, проиндексированный в переменной "sc_list".
