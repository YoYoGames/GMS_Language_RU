ds_list_is_map С помощью этой функции можно проверить, хранится ли карта DS в заданной позиции в списке DS. Если заданная позиция содержит идентификатор карты DS, то функция вернет true, в противном случае она вернет false. Синтаксис: ds_list_is_map(id, pos); Аргумент Описание id Идентификатор списка для проверки. поз Позиция в списке для проверки. Возвращается: Булево Пример: var size = ds_list_size(ships) ; for (var i = 0; i < size; i++;) { if ds_list_is_map(ships, i) { ds_map_destroy(i); } } ds_list_destroy(ships); Приведенный выше код выполняет циклический просмотр списка DS и проверяет, содержат ли какие-либо записи идентификаторы карт. Если да, то эти карты уничтожаются, а после завершения цикла уничтожается основной список. Назад: Списки DS Далее: ds_list_create © Copyright YoYo Games Ltd. 2021 Все права защищены
