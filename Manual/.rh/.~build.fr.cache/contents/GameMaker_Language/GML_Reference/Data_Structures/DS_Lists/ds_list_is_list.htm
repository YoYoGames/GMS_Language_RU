ds_list_is_list С помощью этой функции можно проверить, хранится ли другой список DS в заданной позиции внутри списка DS. Если заданная позиция содержит идентификатор списка DS, то функция вернет true, в противном случае она вернет false. Синтаксис: ds_list_is_list(id, pos); Аргумент Описание id Идентификатор списка для проверки. поз Позиция в списке для проверки. Возвращается: Булево Пример: var size = ds_list_size(ships) ; for (var i = 0; i < size; i++;) { if ds_list_is_list(ships, i) { ds_list_destroy(i); } } ds_list_destroy(ships); Приведенный выше код проходит по списку DS и проверяет, не содержат ли какие-либо записи идентификаторы других списков. Если да, то эти списки уничтожаются, а после завершения цикла уничтожается основной список. Назад: Списки DS Далее: ds_list_is_map © Copyright YoYo Games Ltd. 2021 Все права защищены
