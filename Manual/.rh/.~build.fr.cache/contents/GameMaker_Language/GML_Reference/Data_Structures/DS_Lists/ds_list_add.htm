ds_list_add Эта функция может использоваться для добавления нового значения (вещественного или строкового) в список, который будет добавлен в конец. Функция может принимать дополнительные необязательные аргументы (столько, сколько вам нужно), что позволяет добавлять в список последовательно несколько значений за один вызов. Синтаксис: ds_list_add(id, val1 [, val2, ... max_val]); Аргумент Описание id Идентификатор списка для добавления. val Значение для добавления в список. [val2, ... max_val] Необязательные значения для добавления в список. Возвращается: Н/Д Пример: ds_list_add(sc_list, score); Приведенный выше код добавит значение, хранящееся в переменной "score", в список, проиндексированный в переменной "sc_list".
