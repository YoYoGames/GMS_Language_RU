физика_применить_импульс С помощью физики в GameMaker Studio 2 к объекту можно применить не только силу и гравитацию, но и импульс. Он немного отличается от силы тем, что при его приложении он немедленно повлияет на скорость и, возможно, на крутящий момент (или "вращение") объекта, особенно если точка, выбранная для приложения импульса, не совпадает с центром масс (обратите внимание: центр масс не обязательно совпа дает с началом координат!). Вот иллюстрация: Как вы видите, игрок нажал на мяч, и это дало ему импульс двигаться в правый верхний угол, вращаясь по ходу движения. Точный импульс определяется вектором, который мы получаем из компонентов ximpulse/yimpulse по отношению к координатам xpos/ypos - это означает, что импульс рассчитывается как расстояние от xpos/ypos до ximpulse/yimpulse в Ньютонах, а направление - это угол, который мы получаем от xpos/ypos до ximpulse/yimpulse. ПРИМЕЧАНИЕ: Физический мир использует метры для всех своих расчетов, а соотношение пикселей на метр определяется в редакторе комнаты или с помощью функции physics_world_create(). Синтаксис: physics_apply_impulse(xpos, ypos, ximpulse, yimpulse) Аргумент Описание xpos Координата x в помещении, где будет применен импульс ypos Координата y в помещении, где будет применен импульс ximpulse x-компонент вектора импульсов yimpulse компонент y вектора импульсов Возвращается: Н/Д Пример: if mouse_check_button_pressed(mb_left) { with (place_meeting(mouse_x, mouse_y, all)) { physics_apply_impulse(mouse_x, mouse_y, -10 + irandom(20), -10 + irandom(20)); } } Приведенный выше код применит импульс со случайным вектором к экземпляру, который находится в позиции мыши при нажатии левой кнопки.
