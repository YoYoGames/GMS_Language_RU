physics_draw_debug При создании физической системы в GameMaker Studio 2 часто важно иметь визуальное представление того, что происходит с экземпляром. Этого можно достичь с помощью функции physics_draw_debug(), которая рисует схематичный контур физических свойств, связанных с экземпляром, выполняющим код. Вот изображение того, как это выглядит: Следует отметить, что для того, чтобы это сработало, оно должно быть в событии рисования экземпляра и он будет нарисован с использованием текущего цвета рисования. Синтаксис: physics_draw_debug() Возвращается: Н/Д Пример: draw_set_colour(c_red); physics_draw_debug(); Приведенный выше код нарисует представление физических свойств, связанных с этим экземпляром, используя красный цвет.
