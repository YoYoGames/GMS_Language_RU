physics_world_update_iterations Все в GameMaker Studio 2 основано на шагах, и, как правило, каждое событие происходит только один раз за шаг. Однако, чтобы добиться необходимой точности при использовании функций физики, они выполняют несколько вычислений на каждом шаге, которые называются "итерациями", количество которых контролируется этой функцией. По умолчанию для системы физики установлено число 10, но оно может быть изменено на большее или меньшее в зависимости от того, сколько раз вы хотите, чтобы физика рассчитывала каждый шаг, но рекомендуется рекомендуется устанавливать значение не ниже 5 и не выше 30. Вы также должны знать, что эта функция зависит от функции physics_world_update_speed() функция. Синтаксис: physics_world_update_iterations(iterations) Аргумент Описание y1 количество итераций (на шаг), которые будет выполнять физическая система Возвращается: Н/Д Пример: physics_world_update_iterations(20); Приведенный выше код устанавливает физический мир в текущей комнате для выполнения вычислений 20 раз за шаг.
