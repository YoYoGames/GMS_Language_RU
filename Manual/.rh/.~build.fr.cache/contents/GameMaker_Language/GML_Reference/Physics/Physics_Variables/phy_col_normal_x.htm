phy_col_normal_x Эта переменная, доступная только для чтения, возвращает x-компонент нормали столкновения, соответствующий phy_collision_x значению массива. Для каждой точки контакта существует соответствующая нормаль столкновения (которая обычно является одна и та же нормаль для всех точек контакта в столкновении). Эта контактная нормаль представляет собой единичный вектор, направленный от одной точки столкновения к другой, и может быть использована, например, для вычисления правильного направления "толчка" для разрешения столкновения ПРИМЕЧАНИЕ: Эта переменная доступна только в событии столкновения экземпляра с включенной физикой. Синтаксис: phy_col_normal_x; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику) Пример: repeat(5 + irandom(5)) { with (instance_create_layer(x, y, "Effects", obj_Debris) { physics_apply_local_impulse(0, 0, other.phy_col_normal_x[0], other.phy_col_normal_y[0]); } } Приведенный выше код использует нормаль контакта для задания направления движения для экземпляра, созданного в событии столкновения между двумя экземплярами с поддержкой физики.
