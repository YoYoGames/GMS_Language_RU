phy_angular_velocity Эта переменная может быть использована для установки угловой скорости экземпляра или для получения текущей угловой скорости в градусах в секунду, причем используемое значение может быть как положительным (для вращения по часовой стрелке), так и отрицательным (для вращения против часовой стрелки). Если задать это значение для экземпляра, который ранее был статичным (т.е. имел плотность 0), он станет кинематическим объектом и начнет вращаться. Синтаксис: phy_angular_velocity; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику) Пример: если abs(phy_angular_velocity) > 0 { phy_angular_velocity -= sign(phy_angular_velocity) * 0.01; } else { phy_angular_velocity = 0; } Приведенный выше код проверит угловую скорость экземпляра и, если она не равна 0, то будет добавлять (или вычитать) небольшую величину каждый шаг, пока значение не станет равным 0.
