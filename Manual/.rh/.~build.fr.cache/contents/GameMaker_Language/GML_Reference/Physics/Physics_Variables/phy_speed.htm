phy_speed Эта переменная, доступная только для чтения, возвращает текущую скорость экземпляра с включенной физикой, заданную в пикселях за шаг. Если вам нужно изменить это значение, вы должны сделать это, изменив векторы x и y с помощью переменных phy_speed_x и phy_speed_y. Синтаксис: phy_speed; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику) Пример: if phy_speed > 10 { phy_linear_damping += 0.01; } else { phy_linear_damping = 2; } Приведенный выше код проверяет скорость экземпляра с включенной физикой, а затем изменяет линейное демпфирование на основе возвращаемого значения.
