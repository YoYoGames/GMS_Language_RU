phy_position_xprevious Эта переменная может быть использована для получения (или установки) предыдущего положения x экземпляра в физическом мире игровой комнаты. Это положение экземпляра в физическом мире на предыдущем шаге по отношению к текущему. Синтаксис: phy_position_xprevious; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику) Пример: xx = phy_position_xprevious; yy = phy_position_yprevious; Приведенный выше код хранит предыдущую позицию x и y для экземпляра с включенной физикой в двух переменных.
