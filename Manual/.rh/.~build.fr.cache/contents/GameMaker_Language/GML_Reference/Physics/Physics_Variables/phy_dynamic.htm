фи_динамический Динамический экземпляр - это экземпляр, который полностью смоделирован в физическом мире, и эта переменная, доступная только для чтения, будет возвращать true, если проверяемый экземпляр полностью смоделирован, или false, если нет. Синтаксис: phy_dynamic; Возвращается: Булево (или неопределено, если экземпляр не включен в физику) Пример: если other.phy_dynamic { with (other) { var dir; dir = point_direction(x, y, other.x, other.y); physics_apply_impulse(x, y, x + lengthdir_x(100, dir), y + lengthdir_y(100, dir)); } } Приведенный выше код создает новый экземпляр и затем определяет его как "пулю" в мире физики, прежде чем придать ему мощный импульс вдоль оси y.
