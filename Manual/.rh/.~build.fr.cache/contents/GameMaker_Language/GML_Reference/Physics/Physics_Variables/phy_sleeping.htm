фи_спящий Эта переменная, доступная только для чтения, возвращает, "спит" ли данный экземпляр в данный момент (true) илинет (false)."Спящий" экземпляр - это тот, который не принимает активного участия в физической симуляции. GameMaker Studio 2 переводит объекты в спящий режим для экономии циклов моделирования, когда экземпляр находится в состоянии покоя и не сталкивается с другим экземпляром. Синтаксис: phy_sleeping; Возвращается: Булево (или неопределено, если экземпляр не включен в физику) Пример: if phy_sleeping { instance_destroy(); } Приведенный выше код проверяет, является ли объект активно моделируемым или нет, и если нет, то он уничтожается.
