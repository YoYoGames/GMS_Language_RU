phy_com_x Эта переменная, доступная только для чтения, возвращает положение x центра масс экземпляра. Она рассчитывается автоматически на основе плотности, инерции и массы экземпляра, как определено соответствующими функциями Синтаксис: phy_com_x; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику) Пример: xx = phy_com_x; yy = phy_com_y; Приведенный выше код устанавливает две переменные в положение x и y центра масс экземпляра.
