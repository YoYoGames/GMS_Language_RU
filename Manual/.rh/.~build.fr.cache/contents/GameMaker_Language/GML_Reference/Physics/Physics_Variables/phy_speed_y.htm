phy_speed_y Эта переменная может использоваться для получения или изменения компонента y вектора линейной скорости экземпляра и определяется в пикселях на шаг (для пикселей в секунду см. phy_linear_velocity_y). Изменение этого параметра для статического экземпляра (т.е. экземпляра с плотностью 0) превратит его в кинематический экземпляр. Синтаксис: phy_speed_y; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если в экземпляре не включена физика) Пример: if phy_speed_y != 0 { phy_speed_y = 0; } Приведенный выше код проверит компонент y вектора линейной скорости и, если он не равен 0, то установит его в 0.
