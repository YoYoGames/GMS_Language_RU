phy_collision_x Этот массив, доступный только для чтения, возвращает положение x всех точек, обнаруженных при столкновении между двумя экземплярами с поддержкой физики. ПРИМЕЧАНИЕ: Эта переменная доступна только в событии столкновения экземпляра с включенной физикой. Синтаксис: phy_collision_x; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если экземпляр не включен в физику) Пример: for(var i = 0; i < phy_collision_points; i += 1;) { part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5); } Приведенный выше код создает частицы во всех определенных точках столкновения между двумя экземплярами с поддержкой физики.
