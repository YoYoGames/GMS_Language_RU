phy_collision_y Этот массив, доступный только для чтения, возвращает положение y всех точек, обнаруженных при столкновении между двумя экземплярами с поддержкой физики. ПРИМЕЧАНИЕ: Эта переменная доступна только в событии столкновения экземпляра с включенной физикой. Синтаксис: phy_collision_y; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если в экземпляре не включена физика) Пример: for(var i = 0; i < phy_collision_points; i += 1;) { part_particles_create(global.Sname, phy_collision_x[i], phy_collision_y[i], global.Spark, 5); } Приведенный выше код создает частицы во всех определенных точках столкновения между двумя экземплярами с поддержкой физики.
