phy_position_yprevious Эта переменная может быть использована для получения (или установки) предыдущей позиции y экземпляра в физическом мире игровой комнаты. Это положение экземпляра в физическом мире на предыдущем шаге по отношению к текущему. Синтаксис: phy_position_yprevious; Возвращается: Real (значение с плавающей точкой одинарной точности, или неопределено, если в экземпляре не включена физика) Пример: xx = phy_position_xprevious; yy = phy_position_yprevious; Приведенный выше код хранит предыдущую позицию x и y для экземпляра с включенной физикой в двух переменных.
