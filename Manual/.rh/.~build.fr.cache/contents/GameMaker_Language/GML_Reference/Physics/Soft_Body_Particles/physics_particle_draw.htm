физика_частицы_рисовать Эта функция нарисует спрайт в позиции всех частиц, которые имеют одно и то же значение категории, определенное пользователем, и у которых флаги совпадают с флагами, установленными для маски. Значение маски задается так же, как и значение флаги при создании частицы (см. функцию physics_particle_create() для примера), т.е. с помощью побитовой комбина ции или "|" для маскирования соответствующих битов для каждого флага. ПРИМЕЧАНИЕ: Эта функция нарисует все частицы в физической симуляции с флагами, совпадающими с флагами маски, будь то отдельные частицы или группы, если категория пользователя одинакова. Нарисованная частица будет использовать цвет и альфа-значения, установленные при ее создании, и будет вращаться с угловым моментом частицы. Однако если вы хотите масштабировать спрайт или изменить другие его свойства, используйте функцию physics_particle_draw_ext(). Синтаксис: physics_particle_draw(typemask, category, sprite, subimg) Аргумент Описание typemask Маска, с которой нужно сравнить флаги частиц. категория Определенная пользователем категория для рисования (используйте 0 для рисования всех категорий). спрайт Спрайт, который будет использоваться для каждой частицы. subimg Индекс изображения спрайта для рисования. Возвращается: Н/Д Пример: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; physics_particle_draw(flags, 2, sprBlob, 0); Приведенный выше код нарисует спрайт для всех частиц в заданной пользователем категории "2", которые имеют те же флаги, что и заданная маска.
