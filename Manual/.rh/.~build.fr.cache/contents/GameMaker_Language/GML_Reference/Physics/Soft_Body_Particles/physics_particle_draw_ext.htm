physics_particle_draw_ext Эта функция нарисует спрайт в позиции всех частиц, которые имеют одно и то же значение категории, определенное пользователем, и у которых флаги совпадают с флагами, установленными для маски. Значение маски задается так же, как и значение флаги при создании частиц (см. функции physics_particle_create() для примера), т.е. используя побитовую комбинацию или "|" для маскирования соответствующих битов для каждого флага. ПРИМЕЧАНИЕ: Эта функция нарисует все частицы в физической симуляции с флагами, совпадающими с флагами маски, будь то отдельные частицы или группы, если категория пользователя одинакова. Нарисованная частица будет игнорировать значения цвета и альфа, установленные при ее создании, и вместо этого будет нарисована с использованием параметров масштаба, угла, цвета и альфа, заданных в этой функции. Обратите внимание, что эта функция предназначена исключительно для рисования спрайта в позиции частицы и никак не влияет на поведение частицы. Синтаксис: physics_particle_draw_ext(typemask, category, sprite, subimg, xscale, yscale, ang, col, alpha) Аргумент Описание typemask Маска для сравнения флагов частиц. категория Определенная пользователем категория для рисования (используйте 0 для рисования всех категорий). спрайт Спрайт, который будет использоваться для каждой частицы. subimg Индекс изображения спрайта для рисования. xscale Коэффициент масштабирования, применяемый к спрайту вдоль оси x. yscale Коэффициент масштабирования, применяемый к спрайту вдоль оси y. анг Угол (в градусах), под которым будет рисоваться спрайт. col Цвет для смешивания со спрайтом (по умолчанию c_white). альфа Значение альфа, используемое при рисовании (0 - 1). Возвращается: Н/Д Пример: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; physics_particle_draw_ext(flags, 2, sprBlob, 0, 2, 2, 0, c_white, 0.5); Приведенный выше код нарисует спрайт для всех частиц в заданной пользователем категории "2", которые имеют те же флаги, что и заданная маска, масштабированные на 2 по обеим осям и с альфой 0,5.
