physics_particle_group_box Эта функция задает форму создаваемой группы частиц. Сначала вы должны начать определение группы с помощью функции physics_particle_group_begin(), а затем вы использовать эту функцию для установки приблизительной полуширины и полувысоты группы в пикселях - приблизительной, потому что точные ширина и высота будут зависеть от размера базовых частиц, определенных функцией physics_particle_set_radius() поскольку физическое моделирование пытается "вписать" как можно больше частиц в заданную форму. Наконец, необходимо вызвать physics_particle_group_end() для создания группы частиц в комнате. Синтаксис: physics_particle_group_box(halfWidth, halfHeight) Аргумент Описание полуширина Половина ширины коробки. halfHeight Половина высоты коробки. Возвращается: Н/Д Пример: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; var groupflags = phy_particle_group_flag_solid; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, 0, c_white, 1, 1, 2); physics_particle_group_box(100, 100); mLastGroup = physics_particle_group_end(); Приведенный выше код сохраняет флаги для типа частиц и свойства группы частиц в переменных, а затем использует их для создания прямоугольной группы частиц со стороной 200px в позиции мыши.
