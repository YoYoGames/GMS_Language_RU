physics_particle_group_end Эта функция используется для завершения определения формы группы частиц. Вызов этой функции создает частицы в рамках заданных параметров формы, а также возвращает значение ID, которое можно сохранить и использовать в дальнейших функциях для взаимодействий с группой частиц. Синтаксис: physics_particle_group_end() Возвращается: Реально Пример: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; var groupflags = phy_particle_group_flag_solid; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, 0, c_white, 1, 1, 2); physics_particle_group_circle(100); mLastGroup = physics_particle_group_end(); Приведенный выше код сохраняет флаги типа частицы и свойства группы частиц в переменных, а затем использует их для создания круглой группы частиц с радиусом 100px в позиции мыши.
