physics_particle_set_flags С помощью этой функции вы можете изменить флаги частицы для отдельной частицы. Значение индекса - это идентификатор частицы, возвращаемый функцией physics_particle_create()а флаги - это возвращаемое значение комбинации следующих констант: Постоянная Описание phy_particle_flag_water Свойства по умолчанию для частицы мягкого тела. phy_particle_flag_zombie Частица-зомби - это частица, которая будет уничтожена за один шаг, при этом все остальные будут помечены таким образом. phy_particle_flag_wall Это определяет частицу как статическую, по существу создавая ее как неподвижный объект в физическом моделировании, поскольку она будет оставаться в фиксированном положении независимо от того, что с ней столкнется. Вы должны использовать этот флаг, а не устанавливать плотность в 0. phy_particle_flag_spring Пружинные частицы создают эффект прикрепления друг к другу, как будто пружиной. Частицы, созданные с помощью этого флага, "соединяются" попарно, причем каждая частица соединяется с той, которая была ближе всего к ней в момент создания. Попав в пару, частицы не меняют "партнеров", и чем дальше внешняя сила отталкивает их друг от друга, тем с большей силой они столкнутся, когда эта внешняя сила будет удалена. Обратите внимание, что независимо от того, насколько далеко парные частиц друг от друга, связь между ними не разрывается. phy_particle_flag_elastic Упругие частицы деформируются и могут также отскакивать при столкновении с другими жесткими телами в физическом моделировании. phy_particle_flag_viscous Вязкая частица - это частица, которая проявляет "цепкость" или "липкость", как масло. Вязкие частицы больше слипаются и склеиваются. phy_particle_flag_powder Частицы порошка создают эффект рассеивания, подобный тому, который можно наблюдать при использовании песка или пыли. phy_particle_flag_tensile Растягивающие частицы используются для создания эффекта поверхностного натяжения, или тугой кривизны на поверхности тела жидкости. Они могут использоваться, например, для создания поверхностного натяжения, которое можно наблюдать на капле воды. Как только натяжение частицы отскакивают, как если бы они были упругими, но при этом продолжают притягиваться друг к другу. В результате частицы, отскакивая, образуют скопления. phy_particle_flag_colourmixing Частицы, смешивающие цвета, принимают некоторые цвета других частиц, с которыми они сталкиваются. Обратите внимание, что если только одна из двух сталкивающихся частиц является смешивающей цвет, другая частица сохраняет свой цвет до столкновения. Эти флаги используют битовую маскировку для создания конечного выходного значения, которое затем проверяется для установки различных основных свойств частицы (при этом базовым свойством всегда является свойство phy_particle_flag_water). Например, если вы хотите смоделировать вязкую жидкость с поверхностным натяжением, вы можете использовать побитовый символ или "|" для маскировки соответствующих битов, как показано в примере ниже. Таким образом, вы можете установить различные свойства (отличные от глобальных) для каждой отдельной созданной частицы, если пожелаете. Синтаксис: physics_particle_set_flags(index, flags) Аргумент Описание индекс Индекс частицы. флаги Флаги для установки на частицу. Возвращается: Реально Пример: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; physics_particle_set_flags(p, flags); Приведенный выше код создаст переменную для хранения значения флагов, а затем использует ее для установки флагов на ранее созданной частице с индексом, хранящимся в переменной "p".
