physics_particle_group_get_x С помощью этой функции вы можете получить положение x в комнате группы частиц. Значение группы - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end(), и функция вернет значение, которое является комбинированным значением текущих установленных флагов. Синтаксис: physics_particle_group_get_x(group) Аргумент Описание группа Группа частиц, которую нужно получить. Возвращается: Реально Пример: xx = physics_particle_group_get_x(group1); yy = physics_particle_group_get_y(group1); Приведенный выше код получит x и y позиции группы частиц, индексированных в переменной "group1", и сохранит их в переменных.
