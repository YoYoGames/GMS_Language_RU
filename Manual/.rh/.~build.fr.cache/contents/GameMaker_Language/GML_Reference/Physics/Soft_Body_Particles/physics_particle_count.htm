счётчик_частиц Эта функция возвращает количество частиц, которые активны в комнате с включенной физикой. Синтаксис: physics_particle_count() Возвращается: Реально Пример: if (physics_particle_count() < physics_particle_get_max_count()) { physics_particle_create(0, x, y, 0, 0, 0, c_white, 1, 1) } Приведенный выше код проверит, есть ли в комнате меньше максимального количества разрешенных частиц, и если да, то создаст еще одну.
