physics_particle_delete_region_circle С помощью этой функции вы можете удалить все частицы, которые попадают в границы определенной круговой области, из физической симуляции в текущей комнате. Функция принимает положение x и y для центра удаляемой области, а также а также радиус (в пикселях), определяющий круглую область. Синтаксис: physics_particle_delete_region_circle(x, y, radius) Аргумент Описание x Положение x центра удаляемой области. y Положение y центра удаляемой области. радиус Радиус (в пикселях) круглой области для удаления. Возвращается: Н/Д Пример: physics_particle_delete_region_circle(mouse_x, mouse_y, 32); Приведенный выше код удалит все частицы, находящиеся в круговой области вокруг позиции мыши.
