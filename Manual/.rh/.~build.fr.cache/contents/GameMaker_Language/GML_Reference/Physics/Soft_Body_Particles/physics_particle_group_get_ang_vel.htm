physics_particle_group_get_ang_vel С помощью этой функции вы можете получить угловую скорость группы частиц. Значение группы - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end(), а функция вернет значение, которое является суммарным значением текущих установленных флагов. Синтаксис: physics_particle_group_get_ang_vel(group) Аргумент Описание группа Группа частиц, которую нужно получить. Возвращается: Реально Пример: ang_v = physics_particle_group_get_ang_vel(group1); Приведенный выше код получит угловую скорость группы частиц, индексированных в переменной "group1", и сохранит ее в переменной.
