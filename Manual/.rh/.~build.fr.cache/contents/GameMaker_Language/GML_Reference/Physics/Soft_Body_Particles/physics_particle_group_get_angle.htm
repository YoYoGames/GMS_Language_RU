physics_particle_group_get_angle С помощью этой функции вы можете получить вращение (угол) в комнате группы частиц. Значение группы - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end(), и функция вернет значение, которое является комбинированным значением текущих установленных флагов. Синтаксис: physics_particle_group_get_angle(group) Аргумент Описание группа Группа частиц, которую нужно получить. Возвращается: Реально Пример: ang = physics_particle_group_get_angle(group1); Приведенный выше код получит угол группы частиц, проиндексированных в переменной "group1", и сохранит его в переменной.
