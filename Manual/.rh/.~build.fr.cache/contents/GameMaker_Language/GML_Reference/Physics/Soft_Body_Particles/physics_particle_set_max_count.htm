physics_particle_set_max_count Эта функция устанавливает общее допустимое количество частиц в физической симуляции. Если вы установите это значение равным 0, то количество создаваемых частиц не будет ограничено, а установка любого другого значения остановит дальнейшее создание частиц, когда общее количество частиц будет равно этому значению. Обратите внимание, что минимальное количество частиц, которое можно создать в симуляции, равно 128, поэтому установка меньшего значения не будет иметь никакого эффекта (если не установлено значение 0). Синтаксис: physics_particle_set_max_count(count) Аргумент Описание считать Максимальное количество частиц, которое необходимо разрешить. Возвращается: Н/Д Пример: physics_particle_set_max_count(500); Приведенный выше код установит количество частиц в физическом симуляторе на 500.
