физика_частиц_группы_счет Эта функция возвращает количество частиц, активных в одной группе. Индекс группы (ID) - это значение, которое возвращается при вызове функции physics_particle_group_end(). Синтаксис: physics_particle_group_count(group) Аргумент Описание группа Индекс группы (ID) группы частиц для подсчета. Возвращается: Реально Пример: gp_num = physics_particle_group_count(group1); Приведенный выше код получит количество частиц, использованных для создания группы, индексированной в переменной "group1", и сохранит значение в переменной.
