физика_частицы_удалить С помощью этой функции вы можете удалить частицу из физической симуляции в текущей комнате. Функция принимает уникальный идентификатор частицы, которую нужно удалить, возвращаемый функцией physics_particle_create(). Синтаксис: physics_particle_group_delete(ind) Аргумент Описание инд Индекс (ID) удаляемой частицы. Возвращается: Н/Д Пример: physics_particle_delete(part); Приведенный выше код удалит из симуляции частицу с идентификатором, хранящимся в переменной "part".
