physics_particle_group_get_vel_y С помощью этой функции вы можете получить вертикальную скорость группы частиц. Значение группы - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end(), и функция вернет значение, которое является комбинированным значением текущих установленных флагов. Синтаксис: physics_particle_group_get_vel_y(group) Аргумент Описание группа Группа частиц, которую нужно получить. Возвращается: Реально Пример: vx = physics_particle_group_get_vel_x(group1); vy = physics_particle_group_get_vel_y(group1); Приведенный выше код получит горизонтальную и вертикальную скорость группы частиц, индексированных в переменной "group1", и сохранит их в переменных.
