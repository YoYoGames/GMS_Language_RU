physics_particle_group_get_inertia С помощью этой функции можно получить инерцию целой группы частиц. Значение группы - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end(), и функция вернет значение, которое является объединенным значением текущих установленных флагов. Синтаксис: physics_particle_group_get_inertia(group) Аргумент Описание группа Группа частиц, которую нужно получить. Возвращается: Реально Пример: i = physics_particle_group_get_inertia(group1); Приведенный выше код получит инерцию группы частиц, индексированных в переменной "group1", и сохранит ее в переменной.
