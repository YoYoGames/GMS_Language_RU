physics_particle_get_damping С помощью этой функции вы можете узнать, каково текущее линейное демпфирование для частиц в физическом моделировании (вы можете установить это значение с помощью функции physics_particle_set_damping()). Синтаксис: physics_particle_get_damping() Возвращается: Реально Пример: if (physics_particle_get_damping() < 1) { physics_particle_set_damping(physics_particle_get_damping() + 0.01); } Приведенный выше код проверит текущее значение демпфирования для всех частиц в системе и, если оно меньше 1, то добавит к нему 0,01.
