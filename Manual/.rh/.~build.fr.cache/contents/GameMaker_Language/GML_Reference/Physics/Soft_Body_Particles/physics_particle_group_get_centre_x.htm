physics_particle_group_get_centre_x С помощью этой функции вы можете получить x-компонент центра масс целой группы частиц. Значение группы - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end(), и функция вернет значение, которое является комбинированным значением текущих установленных флагов. Синтаксис: physics_particle_group_get_centre_x(group) Аргумент Описание группа Группа частиц, которую нужно получить. Возвращается: Реально Пример: xx = physics_particle_group_get_centre_x(group1); yy = physics_particle_group_get_centre_y(group1); Приведенный выше код получит положение x и y для центра масс группы частиц, индексированных в переменной "group1", и сохранит их в переменных.
