physics_particle_delete_region_box С помощью этой функции вы можете удалить все частицы, которые попадают в границы заданной прямоугольной области, из физической симуляции в текущей комнате. Функция принимает положение x и y для центра удаляемой области, а также а также половину ширины и высоты прямоугольника (в пикселях), определяющего эту область. Синтаксис: physics_particle_delete_region_box(x, y, halfWidth, halfHeight) Аргумент Описание x Положение x центра удаляемой области. y Положение y центра удаляемой области. полуширина Половина ширины прямоугольника. halfHeight Половина высоты прямоугольника. Возвращается: Н/Д Пример: physics_particle_delete_region_box(mouse_x, mouse_y, 32, 32); Приведенный выше код удалит все частицы, находящиеся в прямоугольной области вокруг позиции мыши.
