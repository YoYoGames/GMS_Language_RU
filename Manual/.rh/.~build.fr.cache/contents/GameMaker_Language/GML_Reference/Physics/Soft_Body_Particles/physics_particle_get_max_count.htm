physics_particle_get_max_count С помощью этой функции вы можете узнать, каково текущее значение максимального количества частиц, разрешенное в физическом моделировании (вы можете установить это значение с помощью функции physics_particle_set_max_count()). Синтаксис: physics_particle_get_max_count() Возвращается: Реально Пример: if (physics_particle_count() < physics_particle_get_max_count()) { physics_particle_create(0, x, y, 0, 0, 0, c_white, 1, 1) } Приведенный выше код проверит, есть ли в комнате меньше максимального количества разрешенных частиц, и если да, то создаст еще одну.
