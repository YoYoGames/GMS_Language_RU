physics_particle_group_get_vel_x С помощью этой функции можно получить горизонтальную скорость группы частиц. Значение группы - это значение, которое было возвращено при создании группы частиц с помощью функции physics_particle_group_end(), а функция вернет значение, которое является суммарным значением текущих установленных флагов. Синтаксис: physics_particle_group_get_vel_x(group) Аргумент Описание группа Группа частиц, которую нужно получить. Возвращается: Реально Пример: vx = physics_particle_group_get_vel_x(group1); vy = physics_particle_group_get_vel_y(group1); Приведенный выше код получит горизонтальную и вертикальную скорость группы частиц, индексированных в переменной "group1", и сохранит их в переменных.
