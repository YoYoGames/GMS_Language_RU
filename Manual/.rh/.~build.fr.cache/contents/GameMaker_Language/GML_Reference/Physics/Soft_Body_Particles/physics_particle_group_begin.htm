physics_particle_group_begin С помощью этой функции вы можете создать группу частиц в комнате. Группа частиц - это такая группа, в которой частицы, необходимые для создания определенной формы, создаются все сразу как группа, что позволяет создавать моделируемые мягкие тела с различными свойствами заданными используемыми флагами. Флаги" - это возвращаемое значение комбинации констант частиц (см. функцию physics_particle_create() для получения подробной информации о них), в то время как "флаги группы" это комбинация следующих групповых констант (или 0 для отсутствия флагов): Постоянная Описание phy_particle_group_flag_solid Группа твердых частиц не позволяет другим элементам задерживаться внутри нее. Если что-то проникает внутрь, группа твердых частиц выталкивает нарушителя обратно на свою поверхность, благодаря чему группа твердых частиц обладает особенно сильной силой отталкивания. отталкивающей силой. phy_particle_group_flag_rigid Жесткие группы частиц - это группы, форма которых не меняется, даже когда они сталкиваются с другими приспособлениями. Эти групповые флаги используют битовую маскировку для создания конечного выходного значения, которое затем проверяется для установки различных основных свойств группы, помимо тех, которые устанавливаются флагами частиц. Вы можете использовать ни одну, одну или обе эти константы, используя побитовую комбинацию или "|" для маскировки соответствующих битов (как показано в примере ниже). Кроме флагов, вы также можете задать положение в комнате, в котором будет создана группа частиц (указанное положение будет центром масс готовой группы), начальную горизонтальную и вертикальную скорость, а также угловую скорость группы. что позволяет создавать движущиеся и вращающиеся группы. Вы также можете установить начальный цвет и альфа всех частиц в группе, причем эти значения будут использоваться при рисовании частиц с помощью функции physics_particle_draw(). После этого вы задаете когезионную прочность частиц в группе, что является еще одним способом сказать, насколько сильно они "прилипают" друг к другу. Сила может быть значением от 0 до 1, где 0 - отсутствие когезии, а 1 - максимальная когезия. Наконец, вы должны указать категорию для вашей группы частиц. Это произвольное целочисленное значение, которое вы присваиваете каждой группе частиц, имеющей схожие свойства, и которое может быть использовано для последующей установки флагов для данной конкретной категории, а также для а также для ряда других вещей. ПРИМЕЧАНИЕ: Вы можете использовать любое целочисленное значение для категории, кроме 0, которое зарезервировано симуляцией и используется для выбора всех категорий в других функциях (например, для рисования). Важно отметить, что вызов этой функции не создает группу в физическом зале. Эта функция просто начинает процесс и устанавливает базовые свойства частиц для группы. Чтобы действительно создать группу частиц, необходимо задать ей форму (см. пример ниже), а затем вызвать функцию physics_particle_group_end(), после чего она будет создана. ПРИМЕЧАНИЕ: флаги типов частиц, которые вы установите, также будут влиять на сплоченность группы частиц, например, если вы отметите частицы как тип воды (например), то при столкновении с другим объектом группа распадется на части, в то время как если вы отметите частицу как пружинный тип, то при столкновении она сохранит свою сплоченность. Описание ниже). x Позиция x для создания группы частиц. y Позиция y для создания группы частиц. анг Угол (в градусах), на который следует повернуть группу при создании. xv Начальная горизонтальная скорость. yv Начальная вертикальная скорость. угловая_скорость Начальная угловая скорость. col Базовый цвет, который будет использоваться для группы частиц. альфа Базовая альфа, используемая для группы частиц. прочность Установите силу сцепления между частицами в группе. категория Определенная пользователем категория, к которой принадлежит группа.
