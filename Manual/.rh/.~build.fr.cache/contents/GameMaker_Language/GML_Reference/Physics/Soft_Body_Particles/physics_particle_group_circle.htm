physics_particle_group_circle Эта функция задает форму создаваемой группы частиц. Сначала вы должны начать определение группы с помощью функции physics_particle_group_begin(), а затем вы использовать эту функцию для установки приблизительного радиуса группы в пикселях - приблизительного, потому что точный радиус будет зависеть от размера базовых частиц, определенных функцией physics_particle_set_radius() поскольку физическая симуляция пытается "вписать" как можно больше частиц в заданную форму. Наконец, вам нужно вызвать physics_particle_group_end() для создания группы частиц в комнате. Синтаксис: physics_particle_group_circle(radius) Аргумент Описание радиус Радиус (в пикселях) создаваемого круга. Возвращается: Н/Д Пример: var flags = phy_particle_flag_water | phy_particle_flag_viscous | phy_particle_flag_tensile; var groupflags = phy_particle_group_flag_solid; physics_particle_group_begin(flags, groupflags, mouse_x, mouse_y, 0, 0, 0, 0, 0, c_white, 1, 1, 2); physics_particle_group_circle(100); mLastGroup = physics_particle_group_end(); Приведенный выше код сохраняет флаги для типа частиц и свойства группы частиц в переменных, а затем использует их для создания круглой группы частиц с радиусом 100px в позиции мыши.
