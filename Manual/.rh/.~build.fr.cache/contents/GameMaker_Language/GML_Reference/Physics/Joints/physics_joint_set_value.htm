физическое_соединение_установить_значение Некоторые свойства сустава могут быть изменены и заданы даже после его создания. Существует ряд констант, которые могут быть использованы в этой функции, и их можно найти здесь: Physics Joint Constants. Синтаксис: physics_joint_set_value(joint, field, value) Аргумент Описание совместный Индекс сустава, который вы хотите изменить поле Константа для совместной собственности, которую вы хотите изменить значение Новая стоимость совместного имущества Возвращается: Н/Д Пример: если physics_joint_get_value(revJoint, phy_joint_max_motor_torque) < 2 { physics_joint_set_value(revJoint, phy_joint_max_motor_torque, 2); } Приведенный выше код проверяет, установлен ли максимальный крутящий момент двигателя суставов меньше 2, и если да, то устанавливает его равным 2.
