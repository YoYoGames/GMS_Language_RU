physics_joint_weld_create Сварное соединение предназначено для прочного, но гибкого соединения двух крепежных деталей. Сварное соединение допускает изгиб между двумя соединенными приспособлениями, но без растяжения, связанного, например, с распорным соединением, и всегда будет и всегда будет пытаться "пружинить" обратно к исходному углу при любой нагрузке или напряжении. Вы определяете точку в помещении, где должно быть создано соединение, а также угол, который вы хотите, чтобы соединение пыталось постоянно поддерживать, как показано на рисунке ниже. как показано на рисунке ниже: Как вы видите, точки привязки задаются как координаты помещения, поэтому при их определении необходимо соблюдать осторожность. особенно если экземпляры создаются одновременно с суставами, а не помещаются в комнату через редактор комнат. Вы также должны понимать, что суставы создаются независимо от размера спрайта экземпляров или прикрепленных к ним приспособлений. Поэтому, если вы создадите сварной шов в другом месте, а не в начале координат экземпляра, он все равно будет действителен и будет ограничивать два экземпляра относительно позиции, в которой он был создан. Поскольку сварное соединение является гибким и будет сгибаться и разгибаться при любом напряжении, вы можете установить частоту колебаний (в Гц), а также коэффициент демпфирования для соединения, чтобы получить различные эффекты - вам может потребоваться поиграть с этими значениями для точной настройки. рекомендуется начинать с меньших значений и увеличивать их до получения желаемого эффекта. Если установить значение "col" в true, то два экземпляра могут взаимодействовать и сталкиваться друг с другом, но только если у них есть события столкновения, однако если установить значение false, то они не будут сталкиваться независимо ни от чего. Синтаксис: physics_joint_weld_create(inst1, inst2, anchor_x, anchor_y, ref_angle, freq_hz, damping_ratio, col) Аргумент Описание инст1 Первая инстанция, которая соединяется с суставом инст2 Вторая инстанция для соединения с суставом якорь_х Координата x для сустава в пределах игрового мира якорь_y Координата y для сустава в пределах игрового мира угол наклона Угол наклона сустава для поддержания частота_гц Это частота колебаний для сустава, в герцах коэффициент демпфирования Этот коэффициент демпфирования для сустава col Могут ли два экземпляра столкнуться (true) или нет (false) Возвращается: Целое число (индекс сустава) Пример: var i, fix, o_id, p_id; p_id = noone; o_id = noone; fix = physics_fixture_create(); physics_fixture_set_box_shape(fix, 64, 32); for (i = 0; i < 5; i++;) { o_id = instance_create_layer(x + (128 * i), y, "Instances", obj_BridgePart); physics_fixture_bind(fix, o_id); if i > 0 && i < 4 { physics_joint_weld_create(p_id, o_id, x + (128 * i) - 64, y, 0, 10, 12, true); } p_id = o_id; } physics-fixture_delete(fix); Приведенный выше код создает приспособление, затем с помощью цикла создает несколько экземпляров, привязывает приспособление к каждому новому экземпляру и затем соединяет их вместе с помощью сварного шва.
