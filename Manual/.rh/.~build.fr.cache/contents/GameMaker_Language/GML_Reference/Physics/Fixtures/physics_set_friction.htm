physics_set_friction Когда вы привязываете приспособление к экземпляру с помощью physics_fixture_bind() это возвращает "id" для связанного приспособления. Вы можете использовать этот идентификатор для установки значения трения связанного приспособления, а не "базового" приспособления, в любое время с помощью этой функции. Обратите внимание, что обычно трение устанавливается в значение между 0 и 1, но при необходимости вы можете использовать любое неотрицательное значение. Синтаксис: physics_set_friction(fixture, friction) Аргумент Описание приспособление идентификатор связанного приспособления трение новое значение трения для применения Возвращается: Н/Д Пример: var fric = physics_get_friction(fix_id); physics_set_friction(fix_id, fric + 0.1); Приведенный выше код получает текущее значение трения для связанных физических свойств экземпляра, а затем устанавливает их на другое значение.
