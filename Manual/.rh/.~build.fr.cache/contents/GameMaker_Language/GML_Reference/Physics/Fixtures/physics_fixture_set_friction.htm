physics_fixture_set_friction Трение - это сила, которая противодействует относительному движению материальных элементов, скользящих друг по другу, что в мире физики GameMaker Studio 2 означает потерю импульса, вызванную столкновением двух объектов с привязанными к ним креплениями. к ним. Таким образом, когда два экземпляра сталкиваются, на их движение влияет это значение, причем высокое значение трения приводит к большей потере импульса, чем низкое. Обратите внимание, что трение обычно устанавливается в значение между 0 и 1, но вы можете использовать любое неотрицательное значение. при необходимости. Синтаксис: physics_fixture_set_friction(fixture, friction) Аргумент Описание приспособление индекс приспособления трение трение приспособления Возвращается: Н/Д Пример: physics_fixture_set_friction(fix_Brick, 0.1); Приведенный выше код установит трение приспособления, проиндексированного в "fix_brick", равным 0,1.
