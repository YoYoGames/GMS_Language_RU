physics_fixture_set_polygon_shape Эта функция задает форму многоугольника для вашего приспособления, но вам нужно будет использовать функцию physics_fixture_add_point() для определения формы этого многоугольника относительно начала координат приспособления. Многоугольник замыкается, когда приспособление привязывается к экземпляру. Следует отметить, что эта функция должна быть вызвана до определения каких-либо точек, и вы также должны иметь по крайней мере три точки, определенные для вашего многоугольника, прежде чем привязать его к экземпляру, иначе иначе вы получите ошибку. Синтаксис: physics_fixture_set_polygon_shape(fixture) Аргумент Описание приспособление индекс приспособления Возвращается: Н/Д Пример: physics_fixture_set_polygon_shape(fix_Ship); physics_fixture_add_point(fix_Ship, 0,0); physics_fixture_add_point(fix_Ship, -40, 100); physics_fixture_add_point(fix_Ship, 40, 100); Приведенный выше код применит форму многоугольника к приспособлению, индексированному в переменной "fix_Ship", а затем определит три точки для создания треугольной формы.
