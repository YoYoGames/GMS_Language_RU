physics_get_restitution Когда вы привязываете приспособление к экземпляру с помощью physics_fixture_bind() это возвращает "id" для связанного приспособления. Вы можете использовать этот идентификатор для получения реституции (свойства "отскока") значение привязанного приспособления, а не "базового" приспособления, в любое время, используя эту функцию. Синтаксис: physics_get_restitution(fixture) Аргумент Описание приспособление идентификатор связанного приспособления Возвращается: Реально Пример: var rest = physics_get_restitution(fix_id); physics_set_restitution(fix_id, rest * 2); Приведенный выше код получает текущее значение реституции для связанных физических свойств экземпляра, а затем устанавливает их в другое значение.
