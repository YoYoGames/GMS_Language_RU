physics_fixture_set_chain_shape Эта функция определяет форму крепления "цепочка". Цепное приспособление состоит из нескольких точек, которые затем соединяются вместе с помощью краевых форм. Сама функция принимает индекс (ID) приспособления и может указать приспособлению зацикливаться или нет. Если установить значение true, то последняя точка соединится с первой точкой ребром, а если false, то нет. Обратите внимание, что эта функция подготавливает приспособление к приему точек, необходимых для создания цепочки, и они должны быть добавлены после вызова этой функции с помощью функции physics_fixture_add_point()так же, как и при построении многоугольного приспособления. При создании цепного приспособления вы должны дать ему не менее двух точек, но вы не ограничены в количестве последующих точек, которые вы можете добавить, чтобы составить готовое приспособление. Синтаксис: physics_fixture_set_chain_shape(fixture, loop) Аргумент Описание приспособление Индекс приспособления петля Замыкать ли цепочку(true) или нет(false). Возвращается: Н/Д Пример: var xx = -100; var yy = room_height / 2; var fix = physics_fixture_create(); physics_fixture_set_chain_shape(fix, false); for (var i = 0; i < 10; i++;) { physics_fixture_add_point(fix, xx, yy); xx += 50 + random(150); yy = (room_height / 2) - 250 + irandom(500); } physics_fixture_bind(fix, id); Приведенный выше код создаст линию "краевых" светильников с различной высотой по всей длине комнаты.
