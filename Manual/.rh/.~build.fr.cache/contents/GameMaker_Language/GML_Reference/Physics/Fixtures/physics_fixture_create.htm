physics_fixture_create Первым шагом в настройке приспособления является его создание с помощью этой функции. Возвращаемый индекс должен быть сохранен в переменной, которая будет использоваться во всех дальнейших функциях, используемых для определения и использования этого приспособления. Синтаксис: physics_fixture_create() Возвращается: Индекс приспособления Пример: fix_Ball = physics_fixture_create(); Приведенный выше код создаст приспособление и сохранит его индекс в переменной "fix_Ball".
