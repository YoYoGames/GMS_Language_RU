physics_fixture_set_circle_shape Эта функция определяет форму круга для вашего приспособления с радиусом, определяемым аргументом "rad". Синтаксис: physics_fixture_set_circle_shape(fixture, rad) Аргумент Описание приспособление индекс приспособления rad радиус окружности Возвращается: Н/Д Пример: physics_fixture_set_circle_shape(fix_Ball, sprite_get_width(spr_Ball) / 2); Код выше применит форму круга к фиксатору, индексированному в переменной "fix_Ball", с радиусом, равным ширине спрайта "spr_Ball", деленной на 2.
