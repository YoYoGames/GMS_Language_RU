physics_fixture_set_edge_shape Эта функция определяет "краевую" форму фикстуры. Форма края - это просто линия, которая будет генерировать столкновение, когда другие фикстуры пересекают ее, и может быть очень полезна для создания (например) рельефа или для создания границ вокруг комнаты. Положение края определяется с помощью локального пространства, т.е. позиция x/y экземпляра считается (0,0), поэтому это следует учитывать при их создании (в приведенном ниже примере кода экземпляр был бы размещен в точке (0,0) в комнате, чтобы избежать осложнений). Синтаксис: physics_fixture_set_edge_shape(fixture, local_x1, local_y1, local_x2, local_y2) Аргумент Описание приспособление индекс приспособления local_x1 начальное положение x для края локальный_y1 начальная позиция y для края local_x2 конечное положение x для края локальный_y2 конечное положение y для края Возвращается: Н/Д Пример: var xx = 0; var y1 = room_height - 100; var y2 = room_height - 50 - irandom(100); for (var i = 0; i < 10; i++;) { var fix = physics_fixture_create(); physics_fixture_set_edge_shape(fix, xx, y1, xx + 50, y2); physics_fixture_bind(fix, id); physics_fixture_delete(fix); xx += 50; y1 = y2; y2 = room_height - 50 - irandom(100); } Приведенный выше код создаст линию "краевых" светильников с различной высотой по всей длине комнаты.
