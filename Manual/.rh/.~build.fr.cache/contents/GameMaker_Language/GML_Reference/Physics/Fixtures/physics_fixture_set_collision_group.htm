physics_fixture_set_collision_group После создания приспособления можно определить его как часть группы столкновений. Это означает, что вы можете указать экземплярам с этим приспособлением всегда сталкиваться или никогда не сталкиваться с другими экземплярами, которые имеют привязанные к ним приспособления с той же группой столкновений. Значения, которые можно присвоить, основаны на положительных или отрицательных числах, при этом 0 является значением по умолчанию (это означает, что экземпляр с этой настройкой приспособления игнорируется, если для него не создано событие столкновения). Вот изображение для пояснения: Как вы видите, настройка групп столкновений может быть очень полезной для создания сложных физических тел, которые взаимодействуют с окружающим их физическим миром, но не друг с другом. Однако следует отметить, что из-за сложности вычисления столкновений на основе подобных групп, рекомендуется стараться свести к минимуму количество групп, используемых в одной комнате, до абсолютного минимума. Синтаксис: physics_fixture_set_collision_group(fixture, group) Аргумент Описание приспособление привязываемое приспособление группа группа, к которой относится данное приспособление (либо положительное значение, либо 0, либо отрицательное значение) Возвращается: Н/Д Пример: physics_fixture_set_collision_group(fix_Wheel, -1); Приведенный выше код установит приспособление, проиндексированное в переменной "fix_Wheel", в группу "-1".
