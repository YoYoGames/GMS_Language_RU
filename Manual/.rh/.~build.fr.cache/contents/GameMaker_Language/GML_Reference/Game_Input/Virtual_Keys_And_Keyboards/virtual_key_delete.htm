virtual_key_delete Если в вашей игре есть различные комнаты или инстансы для управления меню, игровым процессом и другими вещами, то вполне вероятно, что в какой-то момент вам понадобится изменить положение и карты клавиш ваших виртуальных клавиш. Для этого вы можете использовать эту функцию, чтобы удалить старые клавиши перед созданием новых (если это необходимо). Функция требует, чтобы вы указали ID виртуальной клавиши, которую нужно удалить - как возвращается функцией virtual_key_add(). Обратите внимание, что все виртуальные ключи будут автоматически удалены из комнаты, когда комната будет изменена, поэтому эта функция необходима только в том случае, если вы хотите вручную удалить ключи до завершения работы самой комнаты. Синтаксис virtual_key_delete(index); Аргумент Описание индекс Индекс виртуального ключа для удаления. Возвращается: Н/Д Пример: virtual_key_delete(global.Left); Приведенный выше код удаляет виртуальный ключ, индексированный в глобальной переменной "global.Left".
