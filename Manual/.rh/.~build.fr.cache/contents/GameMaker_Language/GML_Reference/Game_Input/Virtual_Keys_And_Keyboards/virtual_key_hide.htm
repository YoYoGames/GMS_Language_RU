virtual_key_hide После того, как вы создали виртуальный ключ для ваших устройств, вам может понадобиться отладка и проверка того, что он правильно расположен и работает должным образом, и для этого вы должны вызвать функцию virtual_key_show(). Однако, вам может понадобиться включать и выключать это представление, поэтому вы также можете скрыть виртуальный ключ с помощью этой функции, используя индекс виртуального ключа, который вы хотите скрыть (ранее созданный и сохраненный с помощью функции virtual_key_add()). Когда вы будете довольны, рекомендуется создать и разместить в его области свою собственную графику. Синтаксис virtual_key_hide(index); Аргумент Описание индекс Индекс виртуального ключа, который необходимо скрыть. Возвращается: Н/Д Пример: if global.Debug { virtual_key_show(global.Left); } else { virtual_key_hide(global.Left); } Приведенный выше код проверяет глобальную переменную "global.Debug" и если она истинна, то виртуальная клавиша, индексированная в переменной "global.Left", будет нарисована на экране, а если ложна, то клавиша будет скрыта.
