keyboard_virtual_hide Эта функция может быть использована для скрытия виртуальной клавиатуры на устройстве, на котором запущена игра. Вызов этой функции генерирует системное асинхронное событие, в котором карта async_load DS map будет заполнена следующими парами ключ/значение: " event_type" - это будет "статус виртуальнойклавиатуры", когда срабатывают функции виртуальной клавиатуры. "screen_height" - высота виртуальной клавиатуры (в пикселях). Это значение будет равно 0, если клавиатура невидима. "keyboard_status" - текущее состояние клавиатуры, возвращается в виде одной из следующих строк: "скрыт" "скрыт" "показывающий" "видимый" Синтаксис keyboard_virtual_hide(); Возвращается: Н/Д Пример: если input == true { input = false; keyboard_virtual_hide(); } Приведенный выше код скроет виртуальную клавиатуру ОС, если заданная переменная не установлена в false.
