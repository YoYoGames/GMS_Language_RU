состояние_виртуальной_клавиатуры Эта функция может быть использована для получения статуса виртуальной клавиатуры на устройстве, на котором запущена игра. Функция вернет true, если виртуальная клавиатура ОС видна/показывается, или false, если она скрыта/прячется. Синтаксис keyboard_virtual_status(); Возвращается: Булево Пример: var _status = keyboard_virtual_status(); if _status == false { keyboard_virtual_show(kbv_type_numbers, kbv_returnkey_default, kbv_autocapitalize_none, false); } Приведенный выше код покажет виртуальную клавиатуру ОС, если текущее состояние равно false.
