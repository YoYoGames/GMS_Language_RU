данные события Эта переменная является глобальной и используется для хранения карты DS при использовании в событиях жестов, и -1 во все остальное время. Фактическое содержимое карты DS будет зависеть от типа события жеста, которое его вызвало, поэтому обратитесь к отдельным разделам для этих событий. Синтаксис: данные события; Возвращается: Real (целочисленное значение) Пример: isFlick = event_data[? "isflick"]; if (isFlick) { flickVelX = event_data[? "diffX"]; flickVelY = event_data[? "diffY"]; } else { flickVelX = 0; flickVelY = 0; } Приведенный выше код берется из события жеста Drag End и проверяет, является ли это событие "flick", и если да, то извлекает необходимые данные из карты event_data DS и использует их для установки некоторых переменных. Если "flick" событие не обнаружено, то те же переменные устанавливаются в 0. Назад: Ввод с помощью жестов Далее: gesture_get_drag_time © Copyright YoYo Games Ltd. 2021 Все права защищены
