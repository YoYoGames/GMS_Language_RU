gesture_get_pinch_angle_away Эта функция используется для получения угла, в пределах которого прикосновение должно удаляться от другого прикосновения, прежде чем потенциально может быть запущен жест Pinch Out. Угол измеряется в градусах и имеет значение по умолчанию 45°. Синтаксис: gesture_get_pinch_angle_away(); Возвращается: Реальный (в дюймах) Пример: if gesture_get_pinch_angle_away() != 45 { gesture_pinch_angle_away(45); } Приведенный выше код проверяет, установлен ли угол отведения для жестов в 45°, и если нет, то устанавливает это значение.
