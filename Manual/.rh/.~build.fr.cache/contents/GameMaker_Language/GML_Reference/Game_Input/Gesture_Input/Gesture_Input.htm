Ввод с помощью жестов Функции ввода устройства хорошо подходят для работы с простыми прикосновениями к экрану или ориентации устройства, но иногда вам понадобится, чтобы ваши проекты определяли более сложные вещи, например, возможность перетаскивать или "листать" экземпляр по комнате. Это можно сделать с помощью функций устройства, но для облегчения работы GameMaker Studio 2 имеет полный набор событий жестов для обнаружения двойных касаний, перетаскивания, щелчков, щипков и вращений как на уровне экземпляра, так и на глобальном уровне, а также а также возвращать информацию о событии. Все перечисленные здесь функции предназначены для обеспечения большего контроля над тем, как и когда эти события будут вызваны. ВАЖНО! Эти функции не будут работать на целевой платформе HTML5. Они не выдадут ошибку при использовании, а просто будут проигнорированы.
