угол_поворота_жеста Эта функция используется для установки угла, который должна превысить пара касаний, чтобы вызвать жест начала поворота. Угол измеряется в градусах и имеет значение по умолчанию значение 5°. Синтаксис: gesture_rotate_angle(); Возвращается: Реальный (в дюймах) Пример: if gesture_get_rotate_angle() != 5 { gesture_rotate_angle(5); } Приведенный выше код проверяет, установлен ли угол поворота для жестов на 5°, и если нет, то устанавливает это значение.
