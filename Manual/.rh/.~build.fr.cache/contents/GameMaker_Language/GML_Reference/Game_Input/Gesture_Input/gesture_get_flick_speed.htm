gesture_get_flick_speed Эта функция используется для получения скорости, необходимой для срабатывания события Flick Gesture при нажатии или отпускании кнопки. Скорость измеряется в дюймах в секунду и по умолчанию имеет значение 2,0. Синтаксис: gesture_get_flick_speed(); Возвращается: Реальный (дюймы в секунду) Пример: if gesture_get_flick_speed() != 2 { gesture_flick_speed(2); } Приведенный выше код проверяет, установлена ли скорость щелчка для жестов на 2 дюйма в секунду, и если нет, то устанавливает это значение.
