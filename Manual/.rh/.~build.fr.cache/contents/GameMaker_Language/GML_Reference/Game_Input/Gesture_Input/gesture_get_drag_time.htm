gesture_get_drag_time Эта функция используется для получения времени, необходимого для того, чтобы событие жеста Drag Start было вызвано прикосновением или щелчком. Это время также влияет на то, как будет срабатывать событие касания поскольку касание/щелчок и отпускание до истечения этого времени будет считаться касанием. Время измеряется в секундах и по умолчанию имеет значение 0,16. Синтаксис: gesture_get_drag_time(); Возвращается: Real (время в секундах) Пример: if gesture_get_drag_time() != 0.16 { gesture_drag_time(0.16); } Приведенный выше код проверяет, установлено ли время перетаскивания для жестов на 0,16 секунды, и если нет, то устанавливает это значение.
