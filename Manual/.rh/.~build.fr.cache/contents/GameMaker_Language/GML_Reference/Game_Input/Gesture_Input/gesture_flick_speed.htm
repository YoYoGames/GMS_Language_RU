скорость жеста_щелчка Эта функция используется для установки скорости, необходимой для срабатывания события жеста Flick Gesture при нажатии или отпускании кнопки. Скорость измеряется в дюймах в секунду и по умолчанию имеет значение 2,0. Синтаксис: gesture_flick_speed(speed); Аргумент Описание скорость Скорость (в дюймах в секунду), которую необходимо установить для обнаружения события жеста щелчка. Возвращается: Н/Д Пример: if gesture_get_flick_speed() != 2 { gesture_flick_speed(2); } Приведенный выше код проверяет, установлена ли скорость щелчка для жестов на 2 дюйма в секунду, и если нет, то устанавливает это значение.
