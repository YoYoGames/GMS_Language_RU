расстояние_жеста_пинч Эта функция используется для установки расстояния, в пределах которого необходимо коснуться/нажать на экран и переместить два пальца, прежде чем сработает жест Pinch. Расстояние измеряется в дюймах и по умолчанию имеет значение 0,1. Синтаксис: gesture_pinch_distance(); Возвращается: Реальный (в дюймах) Пример: if gesture_get_pinch_distance() != 0.1 { gesture_pinch_distance(0.1); } Приведенный выше код проверяет, установлено ли расстояние прижатия для жестов в 0,1 дюйма, и если нет, то устанавливает это значение.
