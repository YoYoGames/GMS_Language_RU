gesture_get_double_tap_distance Эта функция используется для определения расстояния, на котором необходимо повторно коснуться/нажать на экран после одного касания, чтобы сработать жест двойного касания. Расстояние измеряется в дюймах и по умолчанию имеет значение 0,1. Синтаксис: gesture_get_double_tap_distance(); Возвращается: Реальный (в дюймах) Пример: if gesture_get_double_tap_distance() != 0.1 { gesture_double_tap_distance(0.1); } Приведенный выше код проверяет, установлено ли расстояние двойного касания для жестов в 0,1 дюйма, и если нет, то устанавливает это значение.
