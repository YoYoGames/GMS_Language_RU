расстояние между жестом и двойным касанием Эта функция используется для установки расстояния, на котором необходимо повторно коснуться/нажать на экран после одного касания, чтобы сработал жест двойного касания. Расстояние измеряется в дюймах и по умолчанию имеет значение 0,1. Синтаксис: gesture_double_tap_distance(distance); Аргумент Описание расстояние Расстояние (в дюймах), которое необходимо установить для обнаружения события жеста двойного касания. Возвращается: Н/Д Пример: if gesture_get_double_tap_distance() != 0.1 { gesture_get_double_tap_distance(0.1); } Приведенный выше код проверяет, установлено ли расстояние двойного касания для жестов в 0,1 дюйма, и если нет, то устанавливает это значение.
