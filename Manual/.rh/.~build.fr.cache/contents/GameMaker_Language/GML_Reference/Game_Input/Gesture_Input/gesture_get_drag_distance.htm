gesture_get_drag_distance Эта функция используется для получения расстояния, необходимого для срабатывания события жеста перетаскивания при касании или щелчке. Расстояние измеряется в дюймах и по умолчанию имеет значение 0,1. Синтаксис: gesture_get_drag_distance(); Возвращается: Реальный (в дюймах) Пример: if gesture_get_drag_distance() != 0.1 { gesture_drag_distance(0.1); } Приведенный выше код проверяет, установлено ли расстояние перетаскивания для жестов в 0,1 дюйма, и если нет, то устанавливает это значение.
