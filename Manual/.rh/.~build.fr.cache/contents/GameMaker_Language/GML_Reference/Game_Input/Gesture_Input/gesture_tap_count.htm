счетчик жестов Эта функция используется для установки того, включен или выключен отсчет касаний. При значении true счетчик включается, а при значении false - отключается, хотя вы можете проверить, в каком состоянии он находится, используя функцию gesture_get_tap_count(). Если функция включена, каждое событие касания будет иметь дополнительный элемент карты DS "tapcount", который будет регистрировать количество касаний, а если включена, это означает, что все события касания будут срабатывать, т.е. два касания будут срабатывать как при одинарном касании, так и при двойном, т.е. два касания будут срабатывать как при одинарном, так и при двойном касании. и двойное касание, при этом количество касаний при одиночном касании будет равно 1, а при двойном - 2. Значение счета переключений будет сброшено на 0 по истечении времени, установленного для обнаружения двойного переключения (см. функцию gesture_double_tap_time()). Если счетчик касаний отключен, то первое касание не будет зарегистрировано, пока не пройдет время двойного касания и не будет обнаружено второе касание. Обратите внимание, что по умолчанию эта функция включена. Синтаксис: gesture_tap_count(enable); Аргумент Описание включить Включить (установить значение true) или отключить (установить значение false) подсчет касаний. Возвращается: Н/Д Пример: if !gesture_get_tap_count() { getsure_tap_count(true); } Приведенный выше код проверяет, включен ли подсчет касаний, и если нет, то он включается.
