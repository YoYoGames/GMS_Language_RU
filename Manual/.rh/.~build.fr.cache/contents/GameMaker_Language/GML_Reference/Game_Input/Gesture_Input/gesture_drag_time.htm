время_перетягивания_жеста Эта функция используется для установки времени, в течение которого событие жеста перетаскивания будет вызвано прикосновением или щелчком. Это время также влияет на то, как будет срабатывать событие касания так как касание/щелчок и отпускание до истечения этого времени будет считаться касанием. Время измеряется в секундах и по умолчанию имеет значение 0,16. Синтаксис: gesture_drag_time(time); Аргумент Описание время Время (в секундах), которое необходимо установить для обнаружения события жеста перетаскивания. Возвращается: Н/Д Пример: if gesture_get_drag_time() != 0.16 { gesture_drag_time(0.16); } Приведенный выше код проверяет, установлено ли время перетаскивания для жестов на 0,16 секунды, и если нет, то устанавливает это значение.
