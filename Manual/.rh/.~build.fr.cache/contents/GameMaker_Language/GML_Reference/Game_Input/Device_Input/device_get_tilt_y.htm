device_get_tilt_y Эта функция возвращает значение от -1 до 1 в зависимости от угла "наклона" устройства. Фактическое соотношение между градусами наклона и возвращаемым значением зависит от устройства и используемой ОС, но обычно значение 1 или -1 - это то же самое, что +/-90°. На рисунке ниже показано, как каждая из доступных функций связана с устройством: Синтаксис: device_get_tilt_y() Возвращается: Реально Пример: if display_get_orientation() = display_landscape { x += sign(device_get_tilt_y()); } else { x += sign(device_get_tilt_x()); } Приведенный выше код проверяет ориентацию дисплея, а затем использует соответствующее значение наклона для перемещения игрока вдоль оси x.
