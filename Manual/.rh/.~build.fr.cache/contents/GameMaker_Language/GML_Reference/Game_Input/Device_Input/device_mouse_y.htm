device_mouse_y Эта функция возвращает положение y касания на устройстве. Если вы работаете на модулях HTML5 или PC и Mac, то это значение обновляется постоянно, пока устройство (обычно мышь) подключено, однако для мобильных устройств это значение будет обновляться только во время касания экрана. будет обновляться только во время касания экрана, и обратите внимание, что максимальное количество касаний, которое может быть обнаружено, будет сильно зависеть от используемого устройства и ОС, на которой оно работает. Также обратите внимание, что на HTML5 только "устройство 0" будет работать правильно, а другие устройства могут давать неправильные значения. Синтаксис: device_mouse_y(device); Аргумент Описание устройство Устройство (от 0 до n), которое проверяется. Возвращается: Реально Пример: if device_mouse_check_button(0, mb_left) { if device_mouse_y(0) > y-32 && device_mouse_y(0) < y+32 { pressed = true; } else { pressed = false; } } Приведенный выше код проверяет, нажимается ли устройство, и если да, то опрашивает положение y устройства, чтобы проверить, находится ли оно в пределах параметров.
