device_mouse_dbclick_enable Эта функция может быть использована для настройки устройства на определение двойного касания mb_left (левой кнопки мыши) как касания mb_right (правой кнопки мыши) или нет. По умолчанию установлено значение true, что означает, что каждый раз, когда пользователь дважды быстро и последовательно коснется экрана устройства. экрана устройства дважды быстро и последовательно, возвращаемое значение будет таким же, как если бы была нажата правая кнопка мыши. Когда этот параметр включен, первое касание будет определено как mb_left, а второе как mb_right, поэтому убедитесь, что что любой используемый вами код учитывает это. Синтаксис: device_mouse_dbclick_enable(bool); Аргумент Описание bool Включить(true) или выключить(false) обнаружение двойного щелчка. Возвращается: Булево Пример: if os_type == os_windows || os_type == os_mac { device_mouse_dbclick_enable(false); } Приведенный выше код проверяет, является ли устройство, на котором запущена игра, Windows PC или Mac, и если это одно из них, то отключает функцию двойного тапа.
