device_mouse_raw_x Эта функция возвращает необработанную x-позицию касания на устройстве. Это означает, что она возвращает фактическое определение устройства для позиции x, к которому производится прикосновение, а не определение GameMaker Studio 2, и поэтому игнорирует такие вещи, как такие как положение вида и масштабирование. Обратите внимание, что максимальное количество касаний, которое может быть обнаружено, будет сильно зависеть от используемого устройства и ОС, под управлением которой оно работает. ПРИМЕЧАНИЕ: Эта функция очень сильно зависит от устройства, и вам следует сначала поэкспериментировать с желаемым целевым модулем и устройством, чтобы увидеть, что именно будет возвращено. Синтаксис: device_mouse_raw_x(device); Аргумент Описание устройство Устройство (от 0 до n), которое проверяется. Возвращается: Реально Пример: if device_mouse_check_button(0, mb_left) && device_mouse_check_button(1, mb_left) { x_av = mean(device_mouse_raw_x(0), device_mouse_raw_x(1)); y_av = mean(device_mouse_raw_y(0), device_mouse_raw_y(1)); } Приведенный выше код проверяет, нажимаются ли устройства device1 и device2, и если да, то вычисляет среднее положение координат x/y между каждой точкой нажатия.
