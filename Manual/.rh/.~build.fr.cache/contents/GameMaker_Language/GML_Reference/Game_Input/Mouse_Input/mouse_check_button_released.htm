mouse_check_button_released Эта функция вернет true, если проверяемая кнопка мыши была отпущена, и false, если нет. Эта функция срабатывает только один раз для любой кнопки мыши, когда она отпущена, и для ее повторного срабатывания кнопка должна быть нажата и отпущена снова. Вы задаете кнопку мыши для проверки из одной из следующих констант: Постоянная Описание mb_left Левая кнопка мыши mb_middle Средняя кнопка мыши (это может быть актуально не для всех целевых платформ) mb_right Правая кнопка мыши mb_side1* Боковая кнопка мыши 1 mb_side2* Боковая кнопка мыши 2 mb_any Любая из кнопок мыши mb_none Отсутствие кнопки мыши * ПРИМЕЧАНИЕ: Кнопки mb_side1 и mb_side2 доступны только в Windows, macOS, Ubuntu и HTML5. Синтаксис: mouse_check_button_released(numb); Аргумент Описание numb Какую константу кнопки мыши проверять. Возвращается: Булево Пример: if mouse_check_button_released(mb_right) { скорость = point_distance(x, y, mouse_x, mouse_y) / 10; } Приведенный выше код проверит, была ли отпущена правая кнопка мыши, и если да, то установит скорость экземпляра на десятую часть расстояния между текущим положением x/y и положением мыши x/y.
