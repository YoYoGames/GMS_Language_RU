очистка_мыши Эта функция очистит текущее состояние заданной кнопки мыши. Это означает, что проверка на то, что она удерживается нажатой, не будет возвращать true до тех пор, пока игрок не отпустит кнопку и не нажмет ее снова (но состояние отпускания все равно будет обнаружено если очистка выполняется, пока кнопка мыши удерживается нажатой). Вы можете ввести одну из констант кнопок мыши, перечисленных на этой странице. Синтаксис: mouse_clear(button); Аргумент Описание кнопка Какую константу кнопки мыши проверять. Возвращается: Булево Пример: mouse_clear(mb_any); Приведенный выше код очистит состояние "вниз" всех кнопок мыши.
