мышь_x Эта переменная, доступная только для чтения, возвращает текущее положение мыши по оси x в пределах комнаты. Синтаксис: mouse_x; Возвращается: Реально Пример: x = median(64, mouse_x, room_width - 64); Приведенный выше код будет поддерживать экземпляр в позиции x мыши до тех пор, пока он находится в пределах 64 пикселей с любой стороны комнаты.
