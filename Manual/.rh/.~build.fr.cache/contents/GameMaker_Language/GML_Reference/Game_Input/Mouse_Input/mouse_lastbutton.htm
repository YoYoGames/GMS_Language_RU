кнопка_мыши Эта переменная возвращает последнюю нажатую кнопку мыши и может возвращать любую из специальных констант мыши, кроме mb_any (вы также можете установить эту переменную в одну из констант). Синтаксис: mouse_lastbutton; Возвращается: Постоянная Пример: if mouse_lastbutton = mb_left { x -= 1; } Это перемещает текущий экземпляр влево, если последней нажатой кнопкой была левая кнопка мыши.
