колесо_мыши_вверх Эта функция возвращает true, если колесо мыши вращается вверх, и false в противном случае. Синтаксис: mouse_wheel_up(); Возвращается: Булево Пример: if mouse_wheel_up() { y -= 10; } Это перемещает текущий экземпляр вверх по экрану, если колесико мыши повернуть вверх.
