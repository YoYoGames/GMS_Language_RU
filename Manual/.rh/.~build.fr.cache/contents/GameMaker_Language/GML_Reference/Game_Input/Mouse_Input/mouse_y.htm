мышь_y Эта переменная, доступная только для чтения, возвращает текущее положение мыши по оси y в пределах комнаты. Синтаксис: mouse_y; Возвращается: Реально Пример: y = median(64, mouse_y, room_height - 64); Приведенный выше код будет поддерживать экземпляр в позиции y мыши до тех пор, пока он находится в пределах 64 пикселей от верха и низа комнаты.
