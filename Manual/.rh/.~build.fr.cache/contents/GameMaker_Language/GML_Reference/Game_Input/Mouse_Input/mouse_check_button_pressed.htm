mouse_check_button_pressed Эта функция вернет true, если проверяемая кнопка мыши была нажата, и false, если нет. Эта функция срабатывает только один раз для любой кнопки мыши, когда она впервые нажата, и для ее повторного срабатывания кнопку необходимо должна быть отпущена и нажата снова. Обратите внимание, что она будет считаться сработавшей в течение всего шага и для всех экземпляров, в которых есть какие-либо события мыши или которые используют эту же функцию. Вы задаете кнопку мыши для проверки одной из следующих констант: Постоянная Описание mb_left Левая кнопка мыши mb_middle Средняя кнопка мыши (это может быть актуально не для всех целевых платформ) mb_right Правая кнопка мыши mb_side1* Боковая кнопка мыши 1 mb_side2* Боковая кнопка мыши 2 mb_any Любая из кнопок мыши mb_none Отсутствие кнопки мыши * ПРИМЕЧАНИЕ: Кнопки mb_side1 и mb_side2 доступны только в Windows, macOS, Ubuntu и HTML5. Синтаксис: mouse_check_button_pressed(numb); Аргумент Описание numb Какую константу кнопки мыши проверять. Возвращается: Булево Пример: if mouse_check_button_pressed(mb_left) { score += 50; } Приведенный выше код проверит, была ли нажата левая кнопка мыши, и если да, то добавит 50 к счету.
