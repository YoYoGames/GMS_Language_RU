клавиатура_последней клавиши Эта переменная ссылается на значение, которое клавиша_клавиатуры было в предыдущем кадре, возвращая код клавиши (возвращаются все стандартные константы кода клавиши). Эта переменная не является только для чтения, и вы можете изменить ее, например, установить значение -1, если вы его уже обработали. Синтаксис: keyboard_lastkey; Возвращается: Реально Пример: if (keyboard_lastkey != -1) { str += keyboard_lastchar; keyboard_lastkey = -1; } Приведенный выше код проверяет, не равна ли переменная lastkey значению -1, и если равна, то добавляет в переменную "str" все, что было последней клавишей в виде строки, а затем сбрасывает переменную keyboard_lastkey для дальнейшего ввода.
