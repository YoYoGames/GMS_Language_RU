keyboard_check_direct Эта функция вернет true, если клавиша с определенным кодом нажата, или false, если нет, проверяя аппаратное обеспечение напрямую. Она позволяет выполнить несколько дополнительных проверок, в частности, вы можете использовать коды клавиш vk_lshift, vk_lcontrol, vk_lalt, vk_rshift, vk_rcontrol и vk_ralt, чтобы проверить, нажата ли левая или правая клавиша shift, control или alt. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице " Ввод с клавиатуры ". ПРИМЕЧАНИЕ: Эта функция доступна только для стандартной цели Windows, и результат не зависит от того, какое приложение имеет фокус. Синтаксис: keyboard_check_direct(key); Аргумент Описание ключ Ключ для проверки состояния вниз. Возвращается: Булево Пример: if keyboard_check_direct(vk_ralt) || keyboard_check_direct(vk_lalt) { crouch = true; } Приведенный выше код проверяет, была ли нажата левая или правая клавиша alt, и если да, то устанавливает переменную "crouch" в true.
