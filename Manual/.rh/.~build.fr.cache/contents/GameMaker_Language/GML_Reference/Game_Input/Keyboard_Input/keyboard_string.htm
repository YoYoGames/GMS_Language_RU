строка_клавиатуры В этой переменной хранится строка, содержащая последние (максимум) 1024 символа, набранные на клавиатуре. Эта строка будет содержать только печатаемые символы, но она будет корректно реагировать на нажатие клавиши backspace, стирая последний символ. Эта переменная не предназначена только для чтения, и вы можете изменить ее, например, установить в "" (пустую строку), если вы уже работали с ней, и вы можете использовать функции String Functions для работы с ней. Обратите внимание, что при использовании экранной виртуальной клавиатуры, только эта переменная будет обновляться при вводе с клавиатуры. Синтаксис: keyboard_string; Возвращается: Строка Пример: if string_length(keyboard_string) > 15 { keyboard_string = string_copy(keyboard_string, 1, 15); } Приведенный выше код ограничивает длину строки клавиатуры 15 символами, удаляя те, которые превышают этот предел, копируя первые пятнадцать символов обратно в переменную.
