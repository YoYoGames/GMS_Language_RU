клавиатура_клавиша_отпустить С помощью этой функции вы можете имитировать нажатие любой клавиши на клавиатуре. Функция принимает значение кода клавиши, возвращаемое функцией ord() (только заглавные буквы от A-Z или цифры от 0-9), или любую из констант vk_*, перечисленных на главной странице " Ввод с клавиатуры ". Синтаксис: keyboard_key_release(key); Аргумент Описание ключ Ключ для имитации освобождения. Возвращается: Н/Д Пример: keyboard_key_release(vk_space); Это будет имитировать нажатие пробела.
