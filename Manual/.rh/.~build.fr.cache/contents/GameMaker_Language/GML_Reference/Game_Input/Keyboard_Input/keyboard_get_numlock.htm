keyboard_get_numlock Вы можете использовать эту функцию для определения состояния блокировки номера клавиатуры: true возвращается в случае включения, false - в случае выключения. ПРИМЕЧАНИЕ: Эта функция доступна только в сборках Windows exe и не будет работать на любом другом устройстве. Синтаксис: keyboard_get_numlock(); Возвращается: Булево Пример: if keyboard_get_numlock() { keyboard_set_numlock(false); } else { keyboard_set_numlock(true); } Приведенный выше пример кода получит состояние клавиши Numberlock и, если она включена (true), то установит ее в положение off (false) и наоборот.
