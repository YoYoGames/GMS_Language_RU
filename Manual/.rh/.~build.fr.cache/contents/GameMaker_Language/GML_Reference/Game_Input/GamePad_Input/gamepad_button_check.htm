проверка_кнопки_геймпада Эта функция возвращает true или false в зависимости от того, определена ли данная кнопка геймпада как удерживаемая или нет. Если вы проверяете аналоговую кнопку, то проверка будет учитывать пороговое значение и возвращать true только в том случае, если необработанное значение входа превышает заданный порог (вы можете получить это необработанное значение с помощью функции gamepad_button_value()). Синтаксис: gamepad_button_check(device, button); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. кнопка Какую константу кнопки геймпада следует проверить. Возвращается: Булево Пример: if gamepad_button_check(0, gp_face1) { if canshoot = true { audio_play_sound(snd_Shoot, 0, false); instance_create_layer(x, y, "Bullets", obj_Bullet) canshoot = false; alarm[0] = room_speed / 2; } } Приведенный выше код определит, удерживается ли нижняя кнопка верхней грани (кнопка "X" на контроллере ps3), и если да, то выстрелит "пулей" и подаст сигнал тревоги.
