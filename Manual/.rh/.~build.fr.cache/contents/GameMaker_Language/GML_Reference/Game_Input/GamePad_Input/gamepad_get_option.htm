При разблокировке вращение пульта изменит ввод, например, поворот на 90º влево заставит "правую" сторону сенсорной панели вернуться "вверх" и т.д... Поворот на 90º влево заставит "правую" сторону сенсорной панели вернуться "вверх", и т.д... таким образом, весь ввод будет осуществляться относительно ориентации устройства. Булево "dpad_absolute" tvOS Возвращает, принимает ли сенсорная панель Siri Remote абсолютный или относительный ввод. В абсолютном режиме центром сенсорной панели считается позиция (0,0) - т.е. место, где аналоговый стик находится в состоянии покоя, или место, где был бы центр дпада. - и движение вокруг этой позиции будет генерировать ввод. Если установлено относительное значение, то позиция (0,0) будет рассматриваться как место, где было сделано первоначальное касание сенсорной панели, и весь ввод будет генерироваться из этой позиции, а не из центра пэда.
