gamepad_get_axis_deadzone Эта функция может использоваться для получения значения "мертвой зоны" оси джойстика. Вы указываете слот устройства, который необходимо получить, и функция возвращает значение от 0 до 1, где значение отражает порог, при котором ось джойстика рассматривается как находится на уровне 0. Синтаксис: gamepad_get_axis_deadzone(device); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. Возвращается: Реально Пример: если gamepad_get_axis_deadzone(global.PadId) != 0.5 { gamepad_set_axis_deadzone(global.PadId, 0.5); } Приведенный выше код проверяет пороговое значение аналоговой оси геймпада, подключенного к устройству "slot", хранящееся в глобальной переменной, и если оно не равно значению по умолчанию 0,5, то оно устанавливается на это значение.
