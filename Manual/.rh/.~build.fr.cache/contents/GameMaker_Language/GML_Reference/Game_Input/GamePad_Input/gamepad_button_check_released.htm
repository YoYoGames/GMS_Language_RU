gamepad_button_check_released Эта функция возвращает true или false в зависимости от того, была ли данная кнопка геймпада определена как отпущенная или нет. Обратите внимание, что эта функция сработает только один раз для кнопки в момент ее отпускания. Чтобы она сработала снова, кнопку нужно сначала нажать, а затем снова отпустить. Если вы проверяете аналоговую кнопку, то при проверке будет учтена настройка порога срабатывания и только в том случае. вернет true, только когда необработанное значение входного сигнала окажется ниже заданного порога (необработанное значение можно получить с помощью функции gamepad_button_value()). Синтаксис: gamepad_button_check_released(device, button); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. кнопка Какую константу кнопки геймпада следует проверить. Возвращается: Булево Пример: if gamepad_button_check_released(0, gp_select) { audio_play_sound(snd_Button, 0, false); global.Pause = !global.Pause; } Приведенный выше код определит, была ли нажата кнопка "select" геймпада, подключенного к устройству "slot" 0, и переключит глобальную переменную "Pause".
