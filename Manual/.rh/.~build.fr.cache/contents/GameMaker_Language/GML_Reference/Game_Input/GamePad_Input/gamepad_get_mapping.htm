gamepad_get_mapping Эта функция используется для получения строки сопоставления для геймпада. Эта строка будет либо: Текущая строка карты для геймпада, например: "050000005e040000fd020000ffff3f00,Xbox Wireless Controller,a:b0,b:b1,start:b4,dpdown:h0.4,dpleft:h0.8,dpright:h0.2,dpup:h0.1,guide:b6,leftshoulder:b9,leftstick:b7,lefttrigger:a4,leftx:a0,lefty:a1,rightshoulder:b10,rightstick:b8,righttrigger:a5,rightx:a2,righty:a3,x:b2,y:b3,platform:android" Строка с "no mapping", если нет установленного отображения (это не то же самое, что когда отображение недоступно). Строка с "индекс устройства вне диапазона", если индекс слота не является допустимым индексом геймпада Пустая строка "", если для подключенного геймпада нет привязки. Для получения дополнительной информации о форматировании возвращаемой строки карты, пожалуйста, обратитесь к функции gamepad_test_mapping(). Синтаксис: gamepad_get_mapping(index); Аргумент Описание индекс Из какого "слота" с индексом геймпада брать отображение. Возвращается: Строка Пример: var _gpMap = gamepad_get_mapping(global.PadIndex); show_debug_message("Gamepad Mapping = " + _gpMap); Приведенный выше код получает строку карты для заданного слота геймпада, а затем выводит ее в консоль для отладки.
