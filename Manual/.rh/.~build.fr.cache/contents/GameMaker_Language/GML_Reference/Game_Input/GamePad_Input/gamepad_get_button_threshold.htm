gamepad_get_button_threshold Эта функция может использоваться для определения текущей пороговой настройки аналоговых кнопок для данного устройства. По умолчанию порог для всех аналоговых кнопок составляет 0,5, а диапазон - от 0 до 1. Порог определяет, в какой момент кнопка считается "нажатой" для игр, требующих, чтобы она действовала как цифровая кнопка. считается "нажатой" для игр, которые требуют, чтобы они действовали как цифровые кнопки. Синтаксис: gamepad_get_button_threshold(device); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. Возвращается: Реально Пример: if gamepad_get_button_threshold(0) != 0.5 { gamepad_set_button_threshold(0, 0.5); } Приведенный выше код проверяет пороговое значение аналоговой кнопки геймпада, подключенного к устройству "slot" 0, и если оно не равно значению по умолчанию 0,5, то оно устанавливается на это значение.
