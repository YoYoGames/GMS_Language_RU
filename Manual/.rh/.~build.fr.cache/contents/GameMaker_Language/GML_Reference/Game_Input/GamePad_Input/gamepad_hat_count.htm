gamepad_hat_count Эта функция возвращает общее количество шапок, доступных для геймпада, подключенного к заданному "слоту" устройства. Под шляпами обычно подразумеваются кнопки вверх/вниз/влево/вправо, и обратите внимание, что в целевой системе Windows шляпы доступны только для контроллеров DirectInput контроллеров (то есть, начиная со слота 4 и выше). Синтаксис: gamepad_hat_count(device); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. Возвращается: Реально Пример: h_num = gamepad_hat_count(4); Приведенный выше код вернет количество шапок, доступных на геймпаде, подключенном к устройству "slot" 4, и сохранит значение в переменной "h_num".
