gamepad_remove_mapping Эта функция используется для удаления текущего отображения устройства из заданного индекса слота геймпада. После вызова индекс слота нужно будет изменить с помощью функции gamepad_test_mapping() если вы хотите чтобы иметь возможность использовать константы геймпада для корректного определения ввода (прямой ввод всегда можно получить с помощью функций gamepad_axis/button/hat_count() и gamepad_axis/button/hat_value() вместе). Синтаксис: gamepad_remove_mapping(index); Аргумент Описание индекс Из какого "слота" индекса геймпада удалить отображение. Возвращается: Н/Д Пример: if remap == true { gamepad_remove_mapping(global.PadIndex); } Приведенный выше код удалит отображение из заданного слота индекса геймпада на основе значения переменной.
