gamepad_set_vibration С помощью этой функции вы можете настроить вибрацию моторов геймпада, причем для каждого мотора используется значение от 0 (без вибрации) до 1 (полная вибрация). Обратите внимание, что в этой функции нет ограничения по времени, поэтому вам нужно будет использовать переменную или сигнал тревоги, чтобы отключить вибрацию (установить моторы на 0) по истечении заданного времени. отключить вибрацию (установить моторы в 0) по истечении заданного времени, иначе геймпад будет продолжать вибрировать бесконечно. ПРИМЕЧАНИЕ: Эта функция в настоящее время доступна только для стандартных Windows, PS4 и Xbox One целевых модулей. Синтаксис: gamepad_set_vibration(device, left_motor, right_motor); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. левый двигатель Величина вибрации от левого двигателя от 0 до 1. правый двигатель Величина вибрации от правого двигателя от 0 до 1. Возвращается: Н/Д Пример: if lives = 0 { gamepad_set_vibration(0, 1, 1); alarm[0] = room_speed / 2; } Приведенный выше код можно использовать (например) в событии столкновения, чтобы заставить геймпад, подключенный к "гнезду" 0, вибрировать в течение половины секунды, а установленный сигнал тревоги будет использоваться для его повторного отключения по истечении этого времени.
