gamepad_is_connected Эта функция возвращает, подключен ли геймпад к данному "слоту" (возвращает true) или нет (возвращает false). Обычно вы используете эту функцию вместе с функцией gamepad_get_device_count() для получения правильного количества доступных геймпадов и/или "слотов" для геймпадов. Обратите внимание, что может возникнуть небольшая задержка между подключением геймпада пользователем и определением GameMaker Studio 2 как подключенного (это особенно актуально в тех случаях, когда при работе с контроллерами, подключенными через Bluetooth). ПРИМЕЧАНИЕ: На некоторых платформах - особенно на консолях - эта функция может вернуть false сразу после подключения/выбора геймпада и может потребовать проверки в тревоге несколькими кадрами (шагами) позже, чтобы она правильно определила геймпад. геймпад. Синтаксис: gamepad_is_connected(numb); Аргумент Описание устройство Какой "слот" геймпада нужно проверить. Возвращается: Булево Пример: var gp_num = gamepad_get_device_count(); for (var i = 0; i < gp_num; i++;) { if gamepad_is_connected(i) global.gp[i] = true else global.gp[i] = false; } Приведенный выше код перебирает доступные геймпады (или слоты для геймпадов) и затем проверяет каждый из них на наличие подключенного геймпада. возвращаемое значение затем используется для установки глобального массива в true или false для использования в будущих проверках.
