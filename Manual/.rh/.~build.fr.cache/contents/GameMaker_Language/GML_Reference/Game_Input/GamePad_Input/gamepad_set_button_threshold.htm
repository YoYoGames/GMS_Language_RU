gamepad_set_button_threshold С помощью этой функции можно установить текущее пороговое значение аналоговых кнопок для данного устройства. По умолчанию порог для всех аналоговых кнопок составляет 0,5, а диапазон - от 0 до 1. Порог определяет, в какой момент кнопка считается как "нажатая" для игр, которые требуют, чтобы они действовали как цифровые кнопки. Обратите внимание, что эта функция влияет на состояния проверки, нажатия и отпускания для аналоговых кнопок, но не влияет на значение, возвращаемое функцией gamepad_button_check()которая всегда будет возвращать необработанное значение для кнопки. Синтаксис: gamepad_set_button_threshold(device, threshold); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. порог Новое пороговое значение (от 0 до 1, по умолчанию 0,5). Возвращается: Н/Д Пример: if gamepad_get_button_threshold(0) != 0.5 { gamepad_set_button_threshold(0, 0.5); } Приведенный выше код проверяет пороговое значение аналоговой кнопки геймпада, подключенного к устройству "slot" 0, и если оно не равно значению по умолчанию 0,5, то оно устанавливается на это значение.
