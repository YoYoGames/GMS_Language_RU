gamepad_get_guid С помощью этой функции вы можете получить GUID для геймпада, подключенного к заданному индексу слота. Вы указываете слот геймпада для проверки (от 0 до 11), и функция возвращает одну из следующих строк: "none", если нет GUID или не подключен геймпад "индекс устройства вне диапазона", если индекс слота геймпада находится вне диапазона Строка GUID, что-то вроде "050000005e040000fd020000ffff3f00", если слоту назначен геймпад и для него можно получить GUID Эта функция обычно используется в сочетании с функцией gamepad_get_description() для генерации части строки SDL, необходимой для ремаппинга контроллеров на Windows, Android и macOS. Синтаксис: gamepad_get_guid(index); Аргумент Описание индекс Какой индекс "слота" геймпада нужно проверить Возвращается: Строка Пример: var _guid = gamepad_get_guid(global.PadIndex); var _desc = gamepad_get_description(global.PadIndex); global.GamepadID = _guid + "," + _desc; Приведенный выше код получает строки описания GUID и имени, затем объединяет их и сохраняет итоговую строку в глобальной переменной для дальнейшего использования.
