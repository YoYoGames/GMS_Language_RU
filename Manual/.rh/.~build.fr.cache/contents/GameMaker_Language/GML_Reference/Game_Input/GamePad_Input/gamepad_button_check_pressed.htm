gamepad_button_check_pressed Эта функция возвращает true или false в зависимости от того, была ли нажата заданная кнопка геймпада или нет. Обратите внимание, что эта функция сработает только один раз для кнопки при ее первом нажатии. Чтобы чтобы она сработала снова, кнопку нужно сначала отпустить, а затем нажать снова. Если вам нужно проверить непрерывное нажатие кнопки, следует использовать функцию gamepad_button_check(). Если вы проверяете аналоговую кнопку, то проверка будет учитывать установку порога и вернет значение true только тогда, когда необработанное значение входа превысит заданный порог. (это необработанное значение можно получить с помощью функции gamepad_button_value()). Синтаксис: gamepad_button_check_pressed(device, button); Аргумент Описание устройство Какой "слот" устройства геймпада нужно проверить. кнопка Какую константу кнопки геймпада следует проверить. Возвращается: Булево Пример: if gamepad_button_check_pressed(0, gp_start) { audio_play_sound(snd_Start, 0, false); room_goto(rm_Level_1); } Приведенный выше код определит, была ли нажата кнопка "start" геймпада, подключенного к устройству "slot" 0, или нет, и изменит комнату, если это произошло.
