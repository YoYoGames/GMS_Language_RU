дата_в_секунду С помощью этой функции вы можете увеличить заданное значение datetime на определенное количество секунд, и она вернет новое значение datetime. Синтаксис: дата_в_секунду(дата, сумма); Аргумент Описание дата Время, к которому нужно добавить дату. сумма Количество секунд (должно быть целым числом) для добавления. Возвращается: Реально Пример: mynewdatetime = date_inc_second(date_current_datetime(), 60); Это установит "mynewdatetime" в текущую дату, но с добавлением 60 секунд.
