дата_дата_от Возвращает значение даты для заданного времени. Синтаксис: дата_дата_от(дата); Аргумент Описание дата Время даты, из которого нужно извлечь дату. Возвращается: Реально Пример: today = date_date_of(date_current_datetime()); Это вернет только текущую дату и сохранит значение в переменной "today".
