дата_инк_месяц С помощью этой функции вы можете увеличить заданное значение времени на определенное количество месяцев, и она вернет новое значение времени. Синтаксис: date_inc_month( date, amount ); Аргумент Описание дата Время, к которому нужно добавить дату. сумма Количество месяцев (должно быть целым числом) для добавления. Возвращается: Реально Пример: mynewdatetime = date_inc_month(date_current_datetime(), 12); Это установит "mynewdatetime" на текущую дату, но с добавлением 12 месяцев.
