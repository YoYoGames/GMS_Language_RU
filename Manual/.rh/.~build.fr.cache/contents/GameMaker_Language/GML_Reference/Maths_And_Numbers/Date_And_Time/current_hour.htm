текущий_час Эта переменная, доступная только для чтения, возвращает час, соответствующий текущему моменту, на основе часового пояса по умолчанию для системы (т.е. местного времени). Вы можете изменить используемый базовый часовой пояс с помощью функции date_set_timezone(). Синтаксис: текущий_час; Возвращается: Реально Пример: draw_text(32, 32, "Время " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second)); Приведенный выше код выводит на экран текущее международное время.
