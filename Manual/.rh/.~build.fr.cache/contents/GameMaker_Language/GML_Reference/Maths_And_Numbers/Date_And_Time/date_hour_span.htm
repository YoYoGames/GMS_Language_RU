дата_часа_span С помощью этой функции вы можете получить количество часов между двумя датами. Это значение всегда положительное, а неполные часы будут возвращены в виде дроби. Синтаксис: date_hour_span(date1, date2); Аргумент Описание дата1 Первая дата времени. дата2 Время, с которым сравнивается первое значение. Возвращается: Реально Пример: diff = date_hour_span(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime()); Это установит "diff" как количество часов между 15 сентября 2011 года, 11:04.0 и текущей датой и временем.
