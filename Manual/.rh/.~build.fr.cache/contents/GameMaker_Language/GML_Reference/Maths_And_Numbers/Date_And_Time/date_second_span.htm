дата_секунда_span С помощью этой функции вы можете получить количество секунд между двумя датами. Возвращаемое значение всегда положительно и является целым числом. Синтаксис: дата_секунда_span(дата1, дата2); Аргумент Описание дата1 Первая дата времени. дата2 Время, с которым сравнивается дата. Возвращается: Реально Пример: diff = date_second_span(date_create_datetime(2011, 9, 15, 11, 4, 0 ), date_current_datetime()); Это установит "diff" как количество секунд между 15 сентября 2011 года, 11:04 и текущей датой и временем.
