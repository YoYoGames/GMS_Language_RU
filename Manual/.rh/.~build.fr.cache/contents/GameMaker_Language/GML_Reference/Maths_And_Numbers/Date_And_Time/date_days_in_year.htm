дата_дней_в_году С помощью этой функции вы можете получить количество дней, которое имеет данный год, возвращая 365 для обычного года и 366 для високосного. Синтаксис: дата_дней_в_году(дата); Аргумент Описание дата Дата, которую следует использовать. Возвращается: Реально Пример: days = date_days_in_year(date_current_datetime()); В этом случае значение "days" будет равно количеству дней в текущем году.
