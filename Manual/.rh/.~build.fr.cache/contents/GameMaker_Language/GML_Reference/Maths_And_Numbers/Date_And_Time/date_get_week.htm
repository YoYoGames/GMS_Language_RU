дата_получения_недели Эта функция возвращает неделю заданного значения времени в пределах года. Синтаксис: date_get_week(date); Аргумент Описание дата Время даты для проверки. Возвращается: Реально Пример: myweek = date_get_week(date_current_datetime()); Это установит значение myweek на текущую неделю.
