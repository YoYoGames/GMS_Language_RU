дата_время_строки С помощью этой функции можно создать строку, содержащую заданную дату, отформатированную для системы или устройства, на котором запущена игра в момент вызова функции. Синтаксис: дата_время_строки(дата); Аргумент Описание дата Дата, которую следует использовать. Возвращается: Строка Пример: str = date_datetime_string(date_current_datetime()); Это установит "str" на что-то вроде "15 сентября 2011, 11:33.00" в зависимости от настроек системы для отображения даты/времени и текущей даты и времени.
