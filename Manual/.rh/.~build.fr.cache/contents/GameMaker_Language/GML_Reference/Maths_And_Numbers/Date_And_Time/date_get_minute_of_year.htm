дата_получить_минуту_года Эта функция возвращает минуту заданного значения времени в пределах года (из общего количества минут за год, с учетом високосных лет). Синтаксис: date_get_minute_of_year(date); Аргумент Описание дата Дата для проверки. Возвращается: Реально Пример: myminuteyear = date_get_minute_of_year(date_current_datetime()); Это установит myminuteyear на текущую минуту года.
