дельта_время Цель дельта-тайминга - устранить эффект отставания или медлительности компьютеров, которые пытаются обрабатывать сложную графику или большое количество кода. Это величина, которую можно прибавить к скорости объектов, чтобы в конечном итоге они двигались с одинаковой скорость, независимо от запаздывания. Это достигается в GameMaker Studio 2 с помощью переменной delta_time, доступной только для чтения, которая измеряет время, прошедшее между одним шагом и следующим, в микросекундах (1 микросекунда - это 1 000 000 доля секунды). секунды). Поэтому переменную delta_time можно использовать для расчета того, насколько быстрее (например) должен двигаться игровой персонаж, чтобы компенсировать скачок задержки в игре. Синтаксис: дельта_время Возвращается: Вещественный (целое число) Пример: var _dt = delta_time / 1000000; скорость = spd * dt; Приведенный выше код установит скорость экземпляра, используя дельта-время для коррекции запаздывания. Переменная "_dt" будет использоваться для хранения ранее рассчитанного значения дельта-времени, чтобы это отношение можно было использовать для умножения базового значения скорости (хранящегося в переменной "spd") и таким образом получить согласованную скорость для экземпляра.
