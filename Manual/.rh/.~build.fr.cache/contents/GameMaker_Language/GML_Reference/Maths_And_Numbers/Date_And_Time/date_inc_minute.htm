дата_в_минуту С помощью этой функции вы можете увеличить заданное значение времени на определенное количество минут, и она вернет новое значение времени. Синтаксис: date_inc_minute( date, amount ); Аргумент Описание дата Время, к которому нужно добавить дату. сумма Количество минут (должно быть целым числом) для добавления. Возвращается: Реально Пример: mynewdatetime = date_inc_minute(date_current_datetime(), 60); Это установит "mynewdatetime" в текущую дату, но с добавлением 60 минут.
