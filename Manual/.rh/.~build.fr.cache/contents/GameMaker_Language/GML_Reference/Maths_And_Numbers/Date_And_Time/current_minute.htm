текущая_минута Эта переменная, доступная только для чтения, возвращает минуты, соответствующие текущему моменту. Синтаксис: текущая_минута; Возвращается: Реально Пример: draw_text(32, 32, "Время " + string(current_hour) + ":" + string(current_minute) + "." + string(current_second)); Приведенный выше код выводит на экран текущее международное время.
