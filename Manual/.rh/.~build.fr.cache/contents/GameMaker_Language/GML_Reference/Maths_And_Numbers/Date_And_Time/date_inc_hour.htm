дата_в_часах С помощью этой функции вы можете увеличить заданное значение времени на определенное количество часов, и она вернет новое значение времени. Синтаксис: date_inc_hour(date, amount); Аргумент Описание дата Время, к которому нужно добавить дату. сумма Количество часов (должно быть целым числом) для добавления. Возвращается: Реально Пример: mynewdatetime = date_inc_hour(date_current_datetime(), 24); Это установит mynewdatetime на текущую дату, но с добавлением 24 часов.
