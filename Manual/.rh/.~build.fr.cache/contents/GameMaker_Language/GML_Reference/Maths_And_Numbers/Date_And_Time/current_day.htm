текущий_день Эта переменная, доступная только для чтения, возвращает день в виде значения от 1 до 31, в зависимости от месяца. Синтаксис: текущий_день; Возвращается: Реально Пример: draw_text(32, 32, "Сегодня " + string(current_day) + "/" + string (current_month) + "/" + string(current_year) +"."); Приведенный выше код выводит день, месяц и год.
