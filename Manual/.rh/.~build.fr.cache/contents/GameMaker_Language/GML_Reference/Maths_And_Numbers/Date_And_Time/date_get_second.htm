дата_получить_секунду Эта функция возвращает секунду заданного значения времени. Синтаксис: date_get_second(date); Аргумент Описание дата Время даты для проверки. Возвращается: Реально Пример: mysecond = date_get_second(date_current_datetime()); Это установит значение mysecond в текущую секунду.
