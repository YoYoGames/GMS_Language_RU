По умолчанию эти функции возвращают местное время, как но вы можете использовать следующую функцию для переключения между местным и UTC временем: дата_установить_временную зону дата_get_timezone Следующие встроенные переменные могут быть использованы для получения определенных компонентов даты и времени: текущее_время текущая_секунда текущая_минута текущий_час текущий_день текущий_недельный день текущий_месяц текущий_год Следующие функции можно использовать для получения определенных компонентов даты и времени, а также для создания новых или для получения промежутка дней/часов между двумя заданными датами и т.д.... дата_создания_время дата_текущего_времени дата_сравнения_дат дата_сравнения_время дата_сравнения_времени дата_действительного_времени дата_дата_от дата_время_о дата_сегодня дата_високосного_года дата_дата_строка дата_время_строки дата_время_строка дата_секунда_span дата_минута_расстояния дата_часа_span дата_дня_расширения дата_недели_span дата_месяц_расширение дата_год_span дата_дней_в_месяце дата_дней_в_году дата_получить_секунду дата_get_минута дата_получения_часа дата_получения_дня дата_получения_недели дата_получения_недели дата_получения_месяца дата_получения_года дата_получить_вторую_ось_года дата_получить_минуту_года дата_получить_час_года дата_получения_дня_года дата_в_секунду дата_в_минуту дата_в_часах дата_инк_день дата_инк_неделя дата_инк_месяц дата_инк_год У нас также есть две специальные функции времени, которые обеспечивают более высокий уровень точности и контроля при ведении игры: get_timer дельта_время Назад: Математика и цифры Следующее: Функции чисел © Copyright YoYo Games Ltd. 2021 Все права защищены
