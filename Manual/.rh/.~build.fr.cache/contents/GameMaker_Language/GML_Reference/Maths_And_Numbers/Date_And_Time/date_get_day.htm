дата_получения_дня Эта функция возвращает день (от 1 до 31) заданного времени даты. Синтаксис: date_get_day(date); Аргумент Описание дата Дата для проверки. Возвращается: Реально Пример: myday = date_get_day( date_current_datetime() ); Это установит "myday" на текущий день.
