дата_получения_недели Эта функция возвращает значение дня недели для заданного времени. Это будет значение от 0 до 6. Синтаксис: date_get_weekday(date); Аргумент Описание дата Дата для проверки. Возвращается: Реально Пример: myweekday = date_get_weekday(date_current_datetime()); Это установит "myweekday" на текущий будний день.
