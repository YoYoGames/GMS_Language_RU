дата_создания_время Эта функция создаст значение datetime из компонента, заданного в качестве аргументов. Синтаксис: date_create_datetime(year, month, day, hour, minute, second); Аргумент Описание год Год, который нужно установить. месяц Месяц, который нужно установить. день День, который нужно установить. час Час для установки. минута Минута для установки. второй Второй для установки. Возвращается: Реально Пример: mydatetime = date_create_datetime(2011, 9, 15, 9, 43, 30); Это позволит установить "mydatetime" на заданную дату и время и сохранить возвращаемое значение в переменной "mydatetime".
