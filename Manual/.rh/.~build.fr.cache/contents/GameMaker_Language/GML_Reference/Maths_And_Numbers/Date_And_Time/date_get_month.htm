дата_получения_месяца Эта функция возвращает месяц заданного значения времени. Синтаксис: date_get_month(date); Аргумент Описание дата Время даты для проверки. Возвращается: Реально Пример: mymonth = date_get_month(date_current_datetime()); В этом случае mymonth будет установлен на текущий месяц.
