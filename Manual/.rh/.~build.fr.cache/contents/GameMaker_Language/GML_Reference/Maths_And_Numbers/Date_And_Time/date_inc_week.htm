дата_инк_неделя С помощью этой функции вы можете увеличить заданное значение времени на определенное количество недель, и она вернет новое значение времени. Синтаксис: date_inc_week(date, amount); Аргумент Описание дата Время, к которому нужно добавить дату. сумма Количество недель (должно быть целым числом) для добавления. Возвращается: Реально Пример: mynewdatetime = date_inc_week(date_current_datetime(), 52); Это установит mynewdatetime на текущую дату, но с добавлением 52 недель.
