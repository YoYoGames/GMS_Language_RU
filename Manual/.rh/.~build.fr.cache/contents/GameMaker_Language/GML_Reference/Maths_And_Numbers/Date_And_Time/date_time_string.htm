дата_время_строка С помощью этой функции вы можете создать строку, содержащую заданное время, отформатированное для системы или устройства, на котором запущена игра в момент вызова функции. Синтаксис: дата_время_строка( дата ); Аргумент Описание дата Используемое время даты. Возвращается: Строка Пример: str = date_time_string(date_current_datetime()); В зависимости от настроек системы для отображения даты/времени и текущей даты и времени, данная переменная будет установлена на что-то вроде "11:36.00".
