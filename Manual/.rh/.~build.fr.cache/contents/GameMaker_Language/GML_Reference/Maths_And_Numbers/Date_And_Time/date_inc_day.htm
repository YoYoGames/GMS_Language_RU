дата_инк_день С помощью этой функции вы можете увеличить заданное значение времени на определенное количество дней, и она вернет новое значение времени. Синтаксис: date_inc_day(date, amount); Аргумент Описание дата Время, к которому нужно добавить дату. сумма Количество дней (должно быть целым числом) для добавления. Возвращается: Реально Пример: mynewdatetime = date_inc_day(date_current_datetime(), 365); Это установит "mynewdatetime" на текущую дату, но с добавлением 365 дней.
