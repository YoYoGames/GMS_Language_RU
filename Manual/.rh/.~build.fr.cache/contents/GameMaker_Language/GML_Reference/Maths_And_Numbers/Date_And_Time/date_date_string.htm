дата_дата_строка С помощью этой функции вы можете создать строку, содержащую заданную дату, отформатированную как день/месяц/год. Синтаксис: дата_дата_строка(дата); Аргумент Описание дата Дата, которую следует использовать. Возвращается: Строка Пример: str = date_date_string(date_current_datetime()); draw_text(32, 32, str); В этом случае "str" будет содержать форматированную строку текущей даты и времени, отображаемую системой.
