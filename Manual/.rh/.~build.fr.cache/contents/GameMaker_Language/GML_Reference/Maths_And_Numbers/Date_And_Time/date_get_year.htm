дата_получения_года Эта функция возвращает год заданного времени даты. Синтаксис: date_get_year(date); Аргумент Описание дата Время даты для проверки. Возвращается: Реально Пример: myyear = date_get_year(date_current_datetime()); Это установит "myyear" на текущий год.
