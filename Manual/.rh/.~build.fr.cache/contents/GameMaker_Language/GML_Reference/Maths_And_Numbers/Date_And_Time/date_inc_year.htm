дата_инк_год С помощью этой функции вы можете увеличить заданное значение времени на определенное количество лет, и она вернет новое значение времени. Синтаксис: date_inc_year(date, amount); Аргумент Описание дата Время даты для добавления к сумма Количество лет (должно быть целым числом) для добавления. Возвращается: Реально Пример: mynewdatetime = date_inc_year(date_current_datetime(), 1000); Это установит mynewdatetime на текущую дату, но с добавлением 1000 лет.
