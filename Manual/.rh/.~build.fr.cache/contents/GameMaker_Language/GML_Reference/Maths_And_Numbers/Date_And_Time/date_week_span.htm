дата_недели_span С помощью этой функции вы можете получить количество недель между двумя датами. Это значение всегда положительное, а неполные недели будут возвращены в виде дроби. Синтаксис: date_week_span(date1, date2); Аргумент Описание дата1 Первая дата времени. дата2 Время, с которым сравнивается дата. Возвращается: Реально Пример: diff = date_week_span( date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime() ); В этом случае значение "diff" будет равно количеству недель между 15 сентября 2011 года, 11:04.0 и текущей датой и временем.
