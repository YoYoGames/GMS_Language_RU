lerp С помощью этой функции можно найти значение, которое соответствует позиции между двумя другими значениями для данного процента. Так что если вы, например: lerp(0, 10, 0.5) вы получите возвращаемое значение 5, что составляет 50% от входных значений. Вы также можете экстраполировать с помощью этой функции, задав положительное или отрицательное значение для суммы интерполяции, чтобы сделать что-то вроде: lerp(0, 10, 2) вернет значение 20. Синтаксис: lerp(a, b, amt) Аргумент Описание a Первое значение. b Второе значение. amt Количество для интерполяции. Возвращается: Реально Пример: xx = lerp(x, x + hspeed, room_speed); yy = lerp(y, y + vspeed, room_speed); Приведенный выше код использует функцию линейной интерполяции, чтобы предсказать, куда переместится экземпляр после одной секунды игрового времени.
