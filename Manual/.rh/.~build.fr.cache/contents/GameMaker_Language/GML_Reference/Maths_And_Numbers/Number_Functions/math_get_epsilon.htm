math_get_epsilon Эта функция возвращает текущее значение epsilon для целевой платформы. Для получения дополнительной информации об эпсилоне см. функцию math_set_epsilon(). Синтаксис: math_get_epsilon(); Возвращается: Реально Пример: var e = math_get_epsilon(); if e != 0.000001 { math_set_epsilon(0.000001); } Это позволит получить текущее значение эпсилона и сохранить его в локальной переменной, которая затем проверяется и при необходимости устанавливается новый эпсилон.
