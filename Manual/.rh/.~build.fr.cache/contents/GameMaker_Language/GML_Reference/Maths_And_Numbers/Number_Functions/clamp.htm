зажим С помощью этой функции вы можете поддерживать входное значение в заданном диапазоне. Синтаксис: clamp(val, min, max) Аргумент Описание val Значение для зажима. min Минимальное значение для зажима между ними. max Максимальное значение, между которым нужно зажать. Возвращается: Реально Пример: скорость = clamp(speed, 1, 10); Приведенный выше код будет зажимать скорость так, чтобы она никогда не падала ниже 1 и не превышала 10.
