случайный_диапазон Эта функция возвращает случайное число в указанном диапазоне, причем возвращаемое значение не обязательно должно быть целым числом. Например, random_range(20,50) вернет случайное число от 20 до 50, но значение может быть вещественным числом, например 38.65265. Вещественные числа также могут использоваться в качестве входных аргументов. ПРИМЕЧАНИЕ: Эта функция будет возвращать одно и то же значение при каждом новом запуске игры, поскольку GameMaker Studio 2 каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры. Синтаксис: random_range(n1, n2); Аргумент Описание n1 Нижняя граница диапазона, из которого будет выбрано случайное число. n2 Верхняя граница диапазона, из которого будет выбрано случайное число. Возвращается: Реально Пример: score += random_range(500, 600); Это добавит к общему счету от 500 до 600 баллов. Назад: Функции чисел Далее: ирандом © Copyright YoYo Games Ltd. 2021 Все права защищены
