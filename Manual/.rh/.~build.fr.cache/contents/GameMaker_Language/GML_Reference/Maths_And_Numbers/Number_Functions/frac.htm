ГРП Эта функция возвращает дробную часть n, то есть часть, стоящую за десятичной точкой. Она возвращает только десятичные дроби, стоящие за точкой, поэтому frac(3.125) возвращает 0.125, frac(6.921) возвращает 0.921 и т.д.. Синтаксис: frac(n); Аргумент Описание n Номер, который нужно изменить. Возвращается: Реально Пример: val = frac(3.4); Это позволит установить значение val на 0,4.
