sqrt Если вы умножаете число на себя, вы получаете квадрат этого числа, но иногда вы хотите сделать обратное и получить квадратный корень из числа. Поэтому, чтобы узнать, какое число было умножено на само себя для получения любого заданного положительного значения, мы используем эту функцию. Например: sqrt(9) вернет 3, так как 3*3=9. Синтаксис: sqrt(val); Аргумент Описание val Число, из которого нужно получить квадратный корень. Возвращается: Реально Пример: num = sqrt(exp); Приведенный выше код установит переменную "num" для хранения квадратного корня из значения, содержащегося в "exp".
