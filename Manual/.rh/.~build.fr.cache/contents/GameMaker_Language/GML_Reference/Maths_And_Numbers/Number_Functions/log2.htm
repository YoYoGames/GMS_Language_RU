log2 Эта функция в основном спрашивает: "Сколько двоек нужно перемножить, чтобы получить n?". Обычным примером использования в программировании является вычисление количества битов, необходимых для представления числа. Для этого мы вычисляем log2(x) числа, округляем его вниз и прибавляем 1 - например, log2(100) дает 6,643856, что округленное вниз число равно 6, к которому мы прибавляем единицу и получаем 7. Таким образом, нам нужно 7 бит, чтобы представить 100. Оно также может быть использовано для описания экспоненциального роста или и часто называется двоичным логарифмом. Синтаксис: log2(n); Аргумент Описание n Входное значение. Возвращается: Реально Пример: colorbits = floor(log2(color)) - 1; Приведенный выше код хранит количество битов, необходимых для представления значения, хранящегося в переменной "color".
