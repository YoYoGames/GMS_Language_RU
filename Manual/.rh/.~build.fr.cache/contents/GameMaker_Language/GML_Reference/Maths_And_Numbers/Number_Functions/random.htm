случайный Эта функция хороша для определения вероятностей, когда возвращение целого числа не является необходимым. Например, random(100) вернет значение от 0 до 99, но это значение может быть 22,56473! Вы также можете использовать не целые, а действительные числа в например, random(0.5), которая вернет значение от 0 до 0.4999999. ПРИМЕЧАНИЕ: Эта функция будет возвращать одно и то же значение при каждом новом запуске игры, поскольку GameMaker Studio 2 каждый раз генерирует одно и то же начальное случайное зерно, чтобы облегчить отладку кода. Чтобы избежать такого поведения, используйте randomise() в начале игры. Синтаксис: random(n); Аргумент Описание n Верхний диапазон, из которого будет выбрано случайное число. Возвращается: Реально Пример: if (random(10) >= 9) { score += 100; } Это даст примерно один шанс из десяти на добавление 100 к счету.
