logn Эта функция похожа на log2(n) и log10(n) только вы указываете значение логарифмической базы. Например, logn(5,25) вернет, сколько пятерок нужно перемножить. чтобы получить 25 (что равно 2). Синтаксис: logn(n, val); Аргумент Описание n Бревенчатая база. val Входное значение. Возвращается: Реально Пример: logval = logn(5, num); Приведенный выше код получает log5 значения, хранящегося в "num".
