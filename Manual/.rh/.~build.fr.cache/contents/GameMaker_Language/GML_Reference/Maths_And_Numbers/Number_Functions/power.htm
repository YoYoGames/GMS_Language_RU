мощность Это вернет значение числа, умноженного на себя "n" раз. Например, power(5,3) умножит 5 на себя 3 раза и вернет 125, что равносильно тому, что сказать 5*5*5=125. Обратите внимание, что значение "x" (число, которое нужно изменить) не может быть отрицательным значением. Синтаксис: мощность(x, n); Аргумент Описание x Номер, который нужно изменить. n Сколько раз умножить x на себя. Возвращается: Реально Пример: score += power(dmg, 3); Это добавит значение проведенного в переменной "dmg" до силы 3 к переменной score.
