потолок Эта функция берет любое вещественное число и округляет его до ближайшего целого числа. С этой функцией следует быть осторожным, так как одна из распространенных ошибок - использовать ее для округления случайного значения и ожидать, что оно всегда будет больше 1, т.е: int = ceil(random(5)); Теперь вы ожидаете, что этот код всегда будет давать целое число от 1 до 5, но это не всегда так, поскольку существует очень маленькая вероятность того, что функция random вернет 0, а округление до 0 все равно даст 0. Это очень малая вероятность, но ее следует учитывать при использовании этой функции. Синтаксис: ceil(val); Аргумент Описание x Номер, который нужно изменить. Возвращается: Целое число Пример: val = ceil( 3.4 ); Это установит значение val на 4. Назад: Функции чисел Далее: максимум © Copyright YoYo Games Ltd. 2021 Все права защищены
