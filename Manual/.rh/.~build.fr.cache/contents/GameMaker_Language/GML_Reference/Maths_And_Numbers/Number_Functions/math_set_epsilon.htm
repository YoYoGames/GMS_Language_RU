math_set_epsilon Эпсилон - это значение, используемое для определения того, достаточно ли близки два числа, подверженные ошибке округления, чтобы считаться "равными". Она полезна при работе с математикой с плавающей запятой, так как позволяет уменьшить "ошибки округления", из-за которых некоторые операции возвращают неверные значения. операции возвращают значения, которые кажутся неправильными или противоречат нашим ожиданиям. Например, мы можем добавить значение к индексу изображения спрайта и ожидать, что результатом будет одно целое число, но из-за природы математики с плавающей запятой фактическое конечное значение оказывается чем-то вроде 5.0000002, поэтому, когда мы проверяем следующее: if image_index == 5 { //делайте что-нибудь } Код ведет себя не так, как ожидалось, и возвращает false. Однако если мы установим значение эпсилон равным 0,000001, значение image_index будет округлено до ближайшего действительного числа, составляющего +/- 0,000001 от исходного значения, в результате чего приведенное выше сравнение вернется к истине. Значение эпсилон будет использоваться для всех следующих целочисленных операторов: < : Менее > : Больше, чем == : Равно <= : Меньше или равно >= : Больше или равно != : Не равно Обратите внимание, что установка значения epsilon, равного 0, отключает все округления, а использование значения 1 приводит к ошибке. Синтаксис: math_set_epsilon(epsilon); Аргумент Описание эпсилон Новое значение эпсилона (от 0 до 0,999999999). Возвращается: Реально Пример: math_set_epsilon(0.0001); Это установит значение эпсилон для всех дальнейших операций с плавающей запятой.
