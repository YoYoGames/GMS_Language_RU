exp Это функция power(e,n), где e - приблизительно 2,718281828 (также известное как число Эйлера), а n - количество раз, которое нужно умножить на себя. По сути, это означает: "Сколько прироста я получу через n единиц времени (и 100% непрерывного роста)?". Таким образом, exp(3) даст 20,08, что означает, что через 3 единицы времени у нас будет в 20,08 раз больше, чем в начале. Синтаксис: exp(n); Аргумент Описание n Сила е.
