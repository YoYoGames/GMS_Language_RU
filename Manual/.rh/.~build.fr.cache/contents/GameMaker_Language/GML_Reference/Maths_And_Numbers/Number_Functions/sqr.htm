sqr Умножает число на само себя и возвращает квадрат этого числа. Например: sqr(5) вернет 25, так как 5*5=25. Синтаксис: sqr(val); Аргумент Описание val Число для возведения в квадрат. Возвращается: Реально Пример: score += sqr(dmg); Приведенный выше код добавит к счету квадрат переменной "dmg". Назад: Функции чисел Далее: sqrt © Copyright YoYo Games Ltd. 2021 Все права защищены
