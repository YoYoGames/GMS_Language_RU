darcsin Возвращает обратный синус x, при этом если dsin(x)=n, darcsin(n)=x, то полученное число будет находиться между -90 и 90. ПРИМЕЧАНИЕ: Здесь принимается только число от -1 до 1 (все остальное приведет к ошибке). ПРИМЕЧАНИЕ: Эта функция возвращает значение в градусах, а не в радианах. Синтаксис: darcsin(x); Аргумент Описание val Значение, для которого возвращается обратный синус. Возвращается: Реально Пример: val = darcsin(-1); Это установит значение "val" на -90.
