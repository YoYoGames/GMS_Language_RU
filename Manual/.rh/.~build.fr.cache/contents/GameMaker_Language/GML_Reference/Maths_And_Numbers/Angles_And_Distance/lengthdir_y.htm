длина дир_у Эта функция используется для получения y-компоненты позиции "len" пикселей от начальной точки и в направлении "dir". Если представить круг вокруг вашего объекта, а затем представить точку в любом месте этого круга, то для перемещения в эту точку, нам нужно переместить объект на столько-то пикселей в этом направлении... поэтому эта функция (при использовании с функцией lengthdir_x()) получает положение этой точки на окружности для использования в коде экземпляра. См. рисунок ниже для деталей: Синтаксис: lengthdir_y(len, dir); Аргумент Описание len Длина расстояния до точки возврата. dir Направление точки возврата. Возвращается: Реально Пример: var _xx = x + lengthdir_x(64, image_angle); var _yy = y + lengthdir_y(64, image_angle); instance_create_layer(_xx, _yy, "Bullets", obj_bullet); Это создаст экземпляр пули в точке(_xx, _yy), которая будет находиться на расстоянии 64 пикселей от родительского экземпляра в направлении угла изображения.
