арктан2 Эта функция возвращает обратный тангенс угла y/x, где y = Opposite side of triangle и x = Adjacent side of triangle. В отличие от arctan() функция arctan2(y, x) действительна для всех углов и поэтому может быть использована для преобразования вектора в угол без риска деления на ноль, а также возвращает результат в правильном квадранте. ПРИМЕЧАНИЕ: Возвращаемое значение указывается в радианах, а не в градусах. Синтаксис: arctan2(y, x); Аргумент Описание y Координата y для расчета. x Координата x для расчета. Возвращается: Реально Пример: val = arctan2(1, 1); Это установит val на правильный угол, в данном случае 0,79. Назад: Углы и расстояние Далее: cos © Copyright YoYo Games Ltd. 2021 Все права защищены
