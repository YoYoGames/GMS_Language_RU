расстояние_до_точки Эта функция вычисляет расстояние от края ограничительной рамки вызывающего экземпляра до указанной позиции x/y в комнате, с возвращаемым значением в пикселях. Обратите внимание, что если у вызывающего объекта не определен спрайт или маска, результаты будут неверными. результаты будут неверными. Синтаксис: distance_to_point(x, y); Аргумент Описание x Позиция x для проверки. y Позиция y для проверки. Возвращается: Реально Пример: if (distance_to_point(obj_Player.x, obj_Player.y) < range) { canshoot = true; } Приведенный выше код проверяет расстояние до позиции x/y объекта игрока, и если оно меньше значения, хранящегося в переменной "range", то переменная "canshoot" устанавливается в true.
