направление точки Эта функция возвращает направление вектора, образованного указанными компонентами [x1,y1] и [x2,y2], относительно фиксированных координат x/y комнаты. Например, на изображении ниже, если мы хотим получить направление от позиции корабля игрока к позиции врага, чтобы мы могли выпустить по нему ракету, то мы используем эту функцию (точный код приведен в примере под изображением): Синтаксис: point_direction(x1, y1, x2, y2) Аргумент Описание x1 Координата x первого компонента вектора y1 Координата y первого компонента вектора x2 Координата x второй компоненты вектора y2 Координата y второй компоненты вектора Возвращается: Реально Пример: var ex, ey; ex = instance_nearest(x, y, enemy).x; ey = instance_nearest(x, y, enemy).y; with (instance_create_layer(x, y, "Bullets", obj_Missile)) { direction = point_direction(x, y, ex, ey); } Приведенный выше код получит координаты x и y ближайшего врага и затем передаст их объекту пули, чтобы использовать их в функции point_direction для правильной установки направления движения.
