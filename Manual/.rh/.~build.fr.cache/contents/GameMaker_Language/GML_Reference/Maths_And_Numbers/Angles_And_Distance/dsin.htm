dsin В прямоугольном треугольнике dsin(val) = Противоположность / Гипотенуза, где "val" - один из трех углов. ПРИМЕЧАНИЕ: Эта функция принимает значение в градусах, а не в радианах Синтаксис: dsin(val); Аргумент Описание val Угол (в градусах), синус которого нужно вернуть. Возвращается: Реально Пример: val = dsin(90); Это установит значение "val" равным 1. Назад: Углы и расстояние Далее: дтан © Copyright YoYo Games Ltd. 2021 Все права защищены
