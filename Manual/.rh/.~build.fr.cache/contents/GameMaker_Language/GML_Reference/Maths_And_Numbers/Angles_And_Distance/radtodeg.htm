radtodeg После того как вы произвели вычисления с помощью sin, cos и т.д., результат будет в радианах. Это не всегда то, что вам нужно, поэтому для преобразования радианов в градусы мы используем эту функцию. Например, radtodeg(sin(180)) вернет -45 градусов. Эта функция переводит радианы в градусы по формуле: угол_градусов = угол_радианов * 180 / пи; Синтаксис: radtodeg(rad); Аргумент Описание rad Радианы для преобразования. Возвращается: Реально Пример: val = radtodeg( pi ); Это установит значение val на 180. Назад: Углы и расстояние Далее: точка_направления © Copyright YoYo Games Ltd. 2021 Все права защищены
