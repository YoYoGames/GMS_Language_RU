дегторад В GM все тригонометрические функции работают в радианах, но большинство людей работают в градусах, и это означает, что для преобразования градусов в радианы нужно использовать эту функцию. Например, degtorad(180) возвращает 3,14159265 радианов. Эта функция переводит градусы в радианы по формуле: угол_радиан = угол_градусов * pi / 180 Синтаксис: degtorad(deg); Аргумент Описание deg Градусы для преобразования. Возвращается: Реально Пример: val = degtorad(90); Это установит значение val равным pi/2. Назад: Углы и расстояние Далее: radtodeg © Copyright YoYo Games Ltd. 2021 Все права защищены
