dtan В прямоугольном треугольнике dtan(val) = Противоположный / Смежный, где "val" - один из трех углов. ПРИМЕЧАНИЕ: Огромное количество значений (например, 90 или -90) будут ошибочными в этой функции из-за их возвращения в бесконечность, графическое представление этого даст асимптоты при этих значениях. ПРИМЕЧАНИЕ: Эта функция принимает значение в градусах, а не в радианах Синтаксис: dtan(val); Аргумент Описание val Угол (в градусах), для которого нужно вернуть тангенс. Возвращается: Реально Пример: val = dtan(45); Это установит значение "val" равным 1. Назад: Углы и расстояние Далее: дарккос © Copyright YoYo Games Ltd. 2021 Все права защищены
