darccos Возвращает обратный косинус x, причем если dcos(val)=n, darccos(n)=val, то полученное число будет находиться между 180 и 0. ПРИМЕЧАНИЕ: Здесь принимается только число от -1 до 1 (все остальное приведет к ошибке). ПРИМЕЧАНИЕ: Эта функция возвращает значение в градусах, а не в радианах. Синтаксис: darccos(val) Аргумент Описание x Значение, для которого возвращается обратный косинус. Возвращается: Реально Пример: val = arccos(-1); Это установит значение "val" равным 180.
