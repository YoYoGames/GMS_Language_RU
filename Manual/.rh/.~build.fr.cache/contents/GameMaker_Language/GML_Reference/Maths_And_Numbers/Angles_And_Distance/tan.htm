тан В прямоугольном треугольнике tan(val) = Opposite / Adjacent, где val - один из трех углов. ПРИМЕЧАНИЕ: Эта функция принимает значение в радианах, а не в градусах. ПРИМЕЧАНИЕ: Pi/2, (3 pi/2), -pi/2 и огромное количество других значений будут ошибочными с этой функцией из-за их возвращения в бесконечность, графическое представление этого даст асимптоты при этих значениях. Синтаксис: tan(val); Аргумент Описание val Угол (в радианах), для которого нужно вернуть тангенс. Возвращается: Реально Пример: val = tan( pi ); Это установит значение val равным 0.
