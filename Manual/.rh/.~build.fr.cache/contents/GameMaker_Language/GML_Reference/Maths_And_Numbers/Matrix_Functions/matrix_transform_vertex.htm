matrix_transform_vertex Эта функция преобразует вектор по матрице. Вы передаете идентификатор матрицы преобразования (возвращаемый функцией построить матрицу()), а также значения x, y и z для вектора, который нужно преобразовать. Функция вернет массив из 3 элементов, где: массив[0] = x массив[1] = y array[2] = z. Синтаксис: matrix_transform_vertex(matrix, x, y, z); Аргумент Описание матрица Матрица для использования x Компонент x вектора преобразования y Компонент y вектора преобразования z Компонента z вектора преобразования Возвращается: Массив (3 элемента) Пример: t_matrix = matrix_build(0, 0, 0, 0, 0, 90, 0, 1, 2, 1); verts = matrix_transform_vertex(t_matrix, x, y, z); Приведенный выше код преобразует заданные значения с помощью матрицы, хранящейся в переменной "t_matrix", и сохраняет их в массиве "verts".
