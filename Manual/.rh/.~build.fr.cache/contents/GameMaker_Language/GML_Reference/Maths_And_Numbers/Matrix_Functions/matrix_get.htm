matrix_get Эта функция возвращает массив из 16 значений, соответствующих заданному типу матрицы 4x4, где строка 1 - элементы [0 - 3], строка 2 - элементы [4 -7] и т.д.. (см. изображение на главной странице). Доступными матрицами являются view, projection и world, для которых можно использовать одну из следующих констант: Постоянная Описание матричный_просмотр Текущая матрица представления матричная_проекция Текущая матрица проекции матричный_мир Текущая мировая матрица Синтаксис: matrix_get(type); Аргумент Описание тип Тип матрицы для получения значений (см. константы, перечисленные выше) Возвращается: Массив (16 элементов) Пример: v_array = matrix_get(matrix_view); Приведенный выше код получит значения текущей матрицы представления и заполнит ими массив "v_array".
