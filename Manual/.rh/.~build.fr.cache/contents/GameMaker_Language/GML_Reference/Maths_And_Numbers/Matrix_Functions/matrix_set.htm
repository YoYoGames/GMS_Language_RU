matrix_set Эта функция принимает массив из 16 значений, соответствующих заданному типу матрицы 4x4, где элементы [0 - 3] будут строкой 1, элементы [4 -7] - строкой 2 и т.д.... (см. рисунок на главной странице). Вы можете создать такую матрицу, используя matrix_build() или matrix_get() или просто создать массив самостоятельно и передать его в функцию. Доступные типы матриц являются типы view, projection и world, для которых можно использовать одну из следующих констант: Постоянная Описание матричный_просмотр Текущая матрица представления матричная_проекция Текущая матрица проекции матричный_мир Текущая мировая матрица Синтаксис: matrix_set(type, matrix); Аргумент Описание тип Тип матрицы для получения значений (см. константы, перечисленные выше) матрица Данные матрицы в виде массива Возвращается: NA Пример: matrix_set(matrix_world, m_array); Приведенный выше код установит значения текущей мировой матрицы на значения, хранящиеся в матрице массива "m_array".
