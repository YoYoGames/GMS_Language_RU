matrix_build_projection_ortho Эта функция строит матрицу ортографической проекции на основе указанных ниже параметров (это метод проекции по умолчанию, используемый при создании комнаты в GameMaker Studio 2 без каких-либо изменений). Иногда необходимо переключиться с перспективной проекции на ортографическую, для чего и предназначена эта функция. Обычно она используется для рисования наложения, например, для отображения счета или других аспектов, поскольку это дает "плоский" вид нарисованных элементов (т.е. без перспективы) в 3D-игре: без перспективы) в 3D-игре. Смотрите рисунок ниже, чтобы понять разницу между ортографическим и перспективным видами. ПРИМЕЧАНИЕ: Вам также может понадобиться временно отключить удаление скрытых поверхностей, если вы хотите, чтобы информация рисовалась независимо от текущего значения глубины. Синтаксис: matrix_build_projection_ortho(width, height, znear, zfar); Аргумент Описание w Ширина проекции. h Высота проекции. znear Ближняя плоскость обрезания. zfar Дальняя плоскость обрезания. Возвращается: Матричный индекс Пример: viewmat = matrix_build_lookat(640, 240, -10, 640, 240, 0, 0, 1, 0); projmat = matrix_build_projection_ortho(640, 480, 1.0, 32000.0); camera_set_view_mat(view_camera[0], viewmat); camera_set_proj_mat(view_camera[0], projmat); Приведенный выше код создает новую матрицу взгляда и ортографическую матрицу, сохраняет их идентификаторы в локальных переменных, а затем использует их для установки матриц вида и проекции для камеры, назначенной на порт вида[0].
