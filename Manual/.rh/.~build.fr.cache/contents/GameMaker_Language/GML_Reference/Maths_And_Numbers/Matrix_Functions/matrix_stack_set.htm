matrix_stack_set Эта функция перезаписывает текущую вершину стека матриц указанной матрицей. Синтаксис: matrix_stack_set(matrix); Аргумент Описание матрица Используемый индекс матрицы. Возвращается: Н/Д Пример: var m = matrix_build(x, y, 0, 0, 0, 0, 0, 1, 1, 1); matrix_stack_set(m); Приведенный выше код построит новую матрицу и сохранит индекс полученной матрицы в переменной "m", после чего заменит ею вершину стека матриц.
