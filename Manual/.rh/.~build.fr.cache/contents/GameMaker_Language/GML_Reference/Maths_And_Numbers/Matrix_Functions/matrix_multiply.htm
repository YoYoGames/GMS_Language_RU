matrix_multiply С помощью этой функции вы можете перемножить два матричных массива вместе, чтобы создать новую матрицу преобразования. Функция возвращает индекс новой матрицы, который следует сохранить в переменной для дальнейшего использования. ПРИМЕЧАНИЕ: Вы не можете использовать константу матрицы в качестве аргумента в этой функции, поэтому если вы хотите умножить (например) матрицу вида на пользовательскую матрицу, которую вы построили, вы должны сначала вызвать функцию matrix_get() и присвоить значения матрицы вида переменной массива, затем умножить ее на вашу пользовательскую матрицу, а затем установить выбранную матрицу (вид, проекция или мир). Синтаксис: matrix_multiply(matrix1, matrix2); Аргумент Описание матрица1 Первый используемый индекс матрицы. матрица2 Второй используемый индекс матрицы. Возвращается: Матричный индекс (матрица 4x4) Пример: var v_matrix = matrix_get(matrix_view); var new_matrix = matrix_multiply(v_matrix, my_matrix); matrix_set(matrix_view, new_matrix); Приведенный выше код получит текущую матрицу представления, затем умножит ее на пользовательскую матрицу и затем использует индекс полученной матрицы для повторной установки матрицы представления.
