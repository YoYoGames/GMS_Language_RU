matrix_build_projection_perspective Эта функция строит матрицу перспективной проекции на основе размеров ближней плоскости, используя указанные параметры, перечисленные ниже. Синтаксис: matrix_build_projection_perspective(width, height, znear, zfar); Аргумент Описание w Ширина проекции. h Высота проекции. znear Ближняя плоскость обрезания. zfar Дальняя плоскость обрезания. Возвращается: Матричный индекс Пример: var projmat = matrix_build_projection_perspective(640, 480, 1.0, 32000.0); camera_set_proj_mat(view_camera[0], projmat); Приведенный выше код создает перспективную проекцию, а затем использует возвращенную матрицу для установки камеры, назначенной на порт просмотра[0].
