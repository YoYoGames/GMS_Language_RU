matrix_build_projection_perspective_fov Эта функция строит матрицу перспективной проекции на основе поля зрения, используя заданные параметры, перечисленные ниже. Синтаксис: matrix_build_projection_perspective_fov(fov_y, aspect, znear, zfar); Аргумент Описание fov Угол поля зрения. аспект Соотношение сторон поля зрения. znear Ближняя плоскость обрезания. zfar Дальняя плоскость обрезания. Возвращается: Матричный индекс Пример: projmat = matrix_build_projection_perspective_fov(60, 320/240, 1.0, 32000.0); camera_set_proj_mat(view_camera[0], projmat); Приведенный выше код создает матрицу проекции поля зрения, которая затем сохраняется в переменной.
