buffer_base64_decode С помощью этой функции вы можете декодировать строку в кодировке base64 (созданную с помощью функции buffer_base64_encode() ) в буфер. Эта функция создаст буфер (как выровненный по 1 байту "растущий" буфер) и возвращает уникальный индекс для буфера, который должен использоваться во всех дальнейших вызовах функции. ПРИМЕЧАНИЕ: Важно, чтобы вы удаляли из памяти любые динамически созданные ресурсы, подобные этому, когда они вам больше не нужны, чтобы предотвратить утечку памяти, поэтому, когда вы закончите работу с созданным буфером, вы должны освободить его снова использование buffer_delete(). Синтаксис: buffer_base64_decode(string); Аргумент Описание строка Строка в кодировке base64 для декодирования Возвращается: Реально Пример: ini_open("Save.ini"); buff = buffer_base64_decode(ini_read_string("Save", "Slot1", "")); ini_close(); Приведенный выше код откроет файл ini, а затем прочитает из него строку в функцию decode. Эта функция вернет индекс буфера, который хранится в переменной "buff", содержащей данные, ранее закодированные и сохраненные. Файл ini будет закрыт.
