buffer_base64_decode_ext С помощью этой функции вы можете декодировать строку в кодировке base64 (созданную с помощью функции buffer_base64_encode() ) в буфер. В отличие от функции buffer_base64_decode(), эта функция не будет создавать буфер для вас, скорее вы уже должны были создать буфер (см. buffer_create()), id которого вы затем будете использовать с помощью этой функции. Смещение" - это позиция в буфере для декодирования заданной строки (в байтах). Синтаксис: buffer_base64_decode_ext(buffer, string, offset); Аргумент Описание буфер Индекс буфера для декодирования строки. строка Строка в кодировке base64 для декодирования. смещение Значение смещения данных. Возвращается: Н/Д Пример: buff = buffer_create(16384, buffer_grow, 2); ini_open("Save.ini"); var str = ini_read_string("Save", "Slot1", ""); buffer_base64_decode_ext(buff, str, 0); ini_close(); Приведенный выше код создаст буфер и сохранит уникальный идентификатор для него в переменной "buff", затем откроет ini-файл и прочитает из него строку в локальную переменную "str". Затем эта строка декодируется во вновь созданный буфер перед тем, как закрытия ini-файла. Назад: Буферы Далее: buffer_set_used_size © Copyright YoYo Games Ltd. 2021 Все права защищены
