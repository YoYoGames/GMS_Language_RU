буферный_поиск Эта функция может использоваться для перемещения по буферу, находя начало, конец или позицию относительно той, которая была использована в последний раз при чтении или записи данных. Значение "offset" - это смещение (в байтах), которое нужно добавить к заданной позиции поиска, например, если "base" относительно и "offset" равно 4, то позиция буфера сдвинется на 4 байта от текущей позиции. Обратите внимание на следующее: Вы можете использовать отрицательные значения для смещения для поиска назад по буферу, а также положительные значения. Если буфер имеет тип "обертывания" и вы смещаетесь за конец буфера, позиция поиска также обернется. Если буфер не имеет типа "обертывания", поиск будет прижиматься к началу или концу буфера, даже если смещение выведет поиск за пределы буфера. В качестве "базового" аргумента для поиска принимаются следующие константы: Постоянная Описание buffer_seek_start Начало буфера буферный_поиск_относительный Позиция относительно текущей позиции чтения/записи буферный_поиск_конца Конец буфера Синтаксис: buffer_seek(buffer, base, offset); Аргумент Описание буфер Индекс используемого буфера. база Базовая позиция, к которой нужно стремиться. смещение Значение смещения данных. Возвращается: Н/Д Пример: buffer_seek(buff, buffer_seek_start, 0); buffer_write(buff, buffer_s16, 0); buffer_write(buff, buffer_s16, x); buffer_write(buff, buffer_s16, y); Приведенный выше код находит начало буфера с идентификатором, хранящимся в переменной "buff", и записывает в него серию знаковых 16-битных целочисленных значений.
