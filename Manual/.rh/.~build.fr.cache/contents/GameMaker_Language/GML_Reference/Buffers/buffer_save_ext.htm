buffer_save_ext С помощью этой функции вы можете сохранить часть содержимого буфера в файл, готовый к считыванию обратно в память с помощью функции buffer_load() функцию. Смещение" определяет начальную позицию в буфере для сохранения (в байтах), а "size" - размер сохраняемой области буфера, начиная с этого смещения и далее (также в байтах). Синтаксис: buffer_save_ext(buffer, filename, offset, size); Аргумент Описание буфер Индекс буфера для сохранения. имя файла Имя файла для сохранения. смещение Смещение в буфере для сохранения (в байтах). размер Размер области буфера для сохранения (в байтах). Возвращается: Н/Д Пример: buffer_save_ext(buff, "Player_Save.sav", 0, 16384); Сохраняет часть текущего содержимого буфера с идентификатором, хранящимся в переменной "buff", в файл.
