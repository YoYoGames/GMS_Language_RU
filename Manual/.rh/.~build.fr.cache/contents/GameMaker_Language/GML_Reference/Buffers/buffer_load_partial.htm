буферная_загрузка_частичная Эта функция загрузит некоторые данные из буфера, которые были ранее сохранены с помощью функции buffer_save() в уже созданный буфер. Вы указываете id ранее созданного буфера для загрузки, затем сохраненный файл буфера для загрузки, а затем смещение от начала буфера (в байтах), из которого вы хотите загрузить данные. Следующие аргументы предназначены для задания длины данных буфера (в байтах) от начальной точки смещения, с которой вы хотите загрузить данные, и точки смещения для загрузки данных в буфер (опять же, в байтах). Синтаксис: buffer_load_partial(buffer, filename, offset, src_len, dest_offset); Аргумент Описание буфер Индекс буфера для загрузки. имя файла Имя файла для загрузки. смещение Смещение в буфере для загрузки (в байтах). src_len Длина загружаемой части буфера (в байтах). dest_offset Смещение, с которого следует начать размещение частичных данных в новом буфере (в байтах). Возвращается: Реально Пример: buff = buffer_create(256, buffer_grow, 1); var _file = "save.dat"; var _so = 6; var _sl = 5; var _do = 0; buffer_load_partial(buff, _file, _so, _sl, _do); Приведенный выше код создаст новый буфер "grow", а затем загрузит в него часть данных, сохраненных в файле "save.dat".
