буфер_сохранения С помощью этой функции вы можете сохранить содержимое буфера в файл, готовый к повторному считыванию в память с помощью функции buffer_load() функцию. ПРИМЕЧАНИЕ: На HTML5 при использовании этой функции содержимое буфера будет сохранено в виде строк в кодировке base64. Синтаксис: buffer_save(buffer, filename); Аргумент Описание буфер Индекс буфера для сохранения. имя файла Имя файла для сохранения. Возвращается: Н/Д Пример: buffer_save(buff, "Player_Save.sav"); Сохраняет текущее содержимое буфера с идентификатором, хранящимся в переменной "buff", в файл.
