буферное_заполнение Эта функция может быть использована для заполнения ранее созданного буфера данными заданного типа и значения. Данные, которыми вы заполняете буфер, должны соответствовать аргументу "тип" этой функции, то есть вы не можете пытаться заполнить буфер, например, строкой, а использовать тип unsigned 16bit integer. строкой и использовать тип unsigned 16bit integer, например. Константы типов такие же, как и те, которые используются функциями buffer_read() и buffer_write() функции. "Размер" это размер буфера (в байтах), который вы хотите заполнить, а смещение - это значение смещения (также в байтах) от начала буфера для начала заполнения. Синтаксис: buffer_fill(buffer, offset, type, value, size); Аргумент Описание буфер Индекс буфера для заполнения. смещение Значение смещения данных (в байтах). тип Тип данных, которые должны быть записаны в буфер (см. список констант здесь). значение Данные для записи. размер Размер буфера (в байтах), который вы хотите заполнить. Возвращается: Н/Д Пример: map_buffer = buffer_create(16384, buffer_fixed, 0); buffer_fill(map_buffer, 0, buffer_u16, 0, 16384); Приведенный выше код находит начало буфера с идентификатором, хранящимся в переменной "buff", и записывает в него серию знаковых 16-битных целочисленных значений.
