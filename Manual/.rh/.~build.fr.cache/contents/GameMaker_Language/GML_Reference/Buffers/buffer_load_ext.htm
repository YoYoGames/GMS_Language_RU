buffer_load_ext Эта функция загрузит данные буфера, которые ранее были сохранены с помощью buffer_save() функции в уже созданный буфер. Вы указываете id ранее созданного буфера для загрузки, затем сохраненный файл буфера для загрузки, и, наконец, смещение от начала буфера (в байтах), в который вы хотите загрузить данные. Синтаксис: buffer_load_ext(buffer, filename, offset); Аргумент Описание буфер Индекс буфера для загрузки. имя файла Имя файла для загрузки. смещение Смещение в буфере для загрузки (в байтах). Возвращается: Н/Д Пример: var pos = buffer_seek(player_buffer, buffer_seek_end, 0); buffer_load(player_buffer, "Data_Save.sav", pos); Приведенный выше код сначала получит позицию конца буфера, индексированную в переменной "player_buffer", а затем загрузит данные из данного в эту позицию (обратите внимание, что этот пример будет работать только с "grow" или "wrap" типами буфера).
