размер буфера Эта функция возвращает размер (в байтах) любой из заданных констант данных (перечисленных здесь). Синтаксис: buffer_sizeof(type); Аргумент Описание тип Тип данных, подлежащих проверке (см. список констант здесь). Возвращается: Н/Д Пример: var b = 12 * buffer_sizeof(buffer_u8); buff = buffer_create(b, buffer_fixed, 1); Приведенный выше код сначала вычисляет размер создаваемого буфера, умножая тип данных unsigned 8bit на 12 (поскольку мы будем использовать буфер для хранения 12 частей данных), а затем использует это значение для установки фиксированного буфера нужного размера.
