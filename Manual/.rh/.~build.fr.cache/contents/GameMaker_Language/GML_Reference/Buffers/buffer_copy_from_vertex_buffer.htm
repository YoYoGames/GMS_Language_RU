буфер_копирование_из_вершинного_буфера Эта функция может использоваться для копирования некоторых (или всех) вершинных данных, хранящихся в одном вершинном буфере, в ранее созданный обычный буфер. При копировании из вершинного буфера в обычный буфер с помощью этой функции, оба буфера должны быть предварительно созданы (с помощью vertex_create_buffer() и buffer_create() например). Вы можете указать диапазон вершинных данных, которые вы где начальная вершина может быть в любом месте между 0 и числом вершин -1, и вы можете указать число вершин с этой точки для копирования. Вы можете использовать функцию vertex_get_number() для буфера вершин, чтобы получить общее количество хранящихся вершин. Наконец, вы указываете индекс буфера, в который нужно скопировать данные вершины, а также смещение данных для определения позиции, в которую нужно скопировать данные вершины в буфере назначения. Синтаксис: buffer_copy_from_vertex_buffer(vertex_buffer, start_vertex, num_vertices, dest_buffer, dest_offset); Аргумент Описание вершинный_буфер Индекс буфера вершин для копирования. начальная_вершина Начальная вершина. num_vertices Общее количество вершин для использования. dest_buffer Индекс буфера для копирования . dest_offset Позиция смещения для копирования данных (в байтах). Возвращается: Н/Д Пример: var v_num = vertex_get_number(model_buff); buffer_copy_from_vertex_buffer(model_buffer, 0, v_num - 1, player_buffer, 0); Приведенный выше код скопирует данные вершин, хранящиеся в буфере вершин, индексированном в переменной "model_buffer", а затем вставит их в буфер, индексированный в переменной "player_buffer".
