размер_буфера С помощью этой функции вы можете изменить размер буфера до указанного вами размера (в байтах). Синтаксис: buffer_resize(buffer, newsize); Аргумент Описание буфер Индекс буфера, размер которого нужно изменить. размер новости Новый размер буфера (в байтах). Возвращается: Н/Д Пример: if (buffer_get_size(buff) < 16384) { buffer_resize(buff, 16384); } Приведенный выше код проверит размер буфера, индексированного в переменной "buff", и если он меньше заданного значения, размер буфера будет изменен.
