buffer_async_group_option С помощью этой функции вы можете установить некоторые специфические для платформы параметры для сохраняемой группы буферов. Доступны следующие опции: " subtitle" или" slottitle" -Значением для этой опции будет строка, которая будет показана пользователю при управлении его сохраненными данными в ОС. Это важно только при сохранении данных, а не при повторной загрузке. " showdialog" - Значением этой опции является булева величина true или false. Если значение равно true, то при сохранении или загрузке GameMaker Studio 2 будет показывать системный пользовательский интерфейс, в противном случае будет выполняться квиксейв/квиклоад без отображения пользовательского интерфейса. Обычно этот параметр нужен только в том случае, если вы поддерживаете несколько слотов сохранения, чтобы пользователь мог выбрать слот, но если вы поддерживаете только один слот для пользователя, установите значение false. " savepadindex" - Для этой опции вы должны указать индекс пада игрока , который сохраняется или загружается, и система будет записывать данные в и читать данные из папки сохранения этого игрока. " saveslotsize" - Эта опция требует, чтобы вы указали фактический размер в байтах, который вы хотите сохранить (так что вы можете сделать buffer_seek() и buffer_tell() чтобы получить это, например). Обратите внимание, что указывать это значение не обязательно, так как для всех сохранений предварительно выделяется минимальное пространство, которое обычно составляет около 10 МБ или около того. ВАЖНО! В настоящее время эта функция действует только для целевых модулей PSVita, PS4 и XBox One. На всех остальных целевых модулях она ничего не даст. Синтаксис: buffer_async_group_option(option, value); Аргумент Описание вариант Возможность установки. значение Значение для установки (может быть строковым или вещественным, в зависимости от опции). Возвращается: Н/Д Пример: buffer_async_group_begin("save_folder_name"); buffer_async_group_option("showdialog", false); buffer_async_group_option("slottitle", "Catch The Haggis Save"); buffer_async_group_option("subtitle", "Все ваши хаггисы сохранены здесь!"); save = buffer_save_async(buff, "Player_Save.sav", 0, 16384); buffer_async_group_end(); Приведенный выше код начинает группу буферов, затем устанавливает параметры группы, прежде чем установить 4 файла для асинхронного сохранения.
