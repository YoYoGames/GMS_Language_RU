buffer_get_alignment С помощью этой функции можно получить выравнивание по байтам для заданного идентификатора буфера. Синтаксис: buffer_get_alignment(buffer); Аргумент Описание буфер Индекс буфера для проверки. Возвращается: Реально Пример: выравнивание = buffer_get_alignment(buff); Приведенный выше код получит выравнивание буфера из значения, проиндексированного в переменной "buff", и сохранит его в переменной.
