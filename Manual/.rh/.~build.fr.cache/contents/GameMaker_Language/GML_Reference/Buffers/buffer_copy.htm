буферное_копирование Эта функция может быть использована для копирования сегмента (или всех) данных, хранящихся в одном буфере, в другой. При использовании двух буферов и копировании из одного в другой оба буфера должны быть предварительно созданы с помощью функции buffer_create() (например), и вы можете указать смещение данных (в байтах) для начальной точки данных, копируемых из исходного буфера относительно начала буфера, а также другое смещение данных для определения позиции копирования данных в в буфер назначения. ПРИМЕЧАНИЕ: Копирование в один и тот же буфер невозможно. Синтаксис: buffer_copy(src_buffer, src_offset, size, dest_buffer, dest_offset); Аргумент Описание src_buffer Индекс буфера для копирования. src_offset Смещение данных, с которого следует начать копирование (в байтах). размер Размер данных для копирования (в байтах). dest_buffer Индекс буфера для копирования . dest_offset Позиция смещения для копирования данных (в байтах). Возвращается: Н/Д Пример: buffer_copy(buff1, 0, 2048, buff2, 2048); Приведенный выше код скопирует данные, хранящиеся в буфере, индексированном в переменной "buff1", а затем вставит их в буфер, индексированный в переменной "buff2", но со смещением на 2048 байт.
