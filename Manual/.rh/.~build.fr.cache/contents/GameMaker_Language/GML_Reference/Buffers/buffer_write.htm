буферная_запись Эта функция может быть использована для записи данных в ранее созданный буфер. Записываемые данные должны соответствовать аргументу "тип" этой функции, то есть нельзя, например, пытаться записать строку как беззнаковое 16-битное целое число. Для определения типа данных можно использовать следующие константы: Постоянная Описание буфер_u8 Беззнаковое 8-разрядное целое число. Это положительное значение от 0 до 255. буфер_с8 Знаковое 8-разрядное целое число. Это может быть положительное или отрицательное значение от -128 до 127 (0 считается положительным). буфер_u16 Беззнаковое 16-разрядное целое число. Это положительное значение от 0 до 65 535. буфер_с16 Подписанное 16-разрядное целое число. Это может быть положительное или отрицательное значение от -32,768 до 32,767 (0 считается положительным). буфер_u32 Беззнаковое 32-разрядное целое число. Это положительное значение от 0 до 4 294 967 295. буфер_с32 Подписанное 32-разрядное целое число. Это может быть положительное или отрицательное значение от -2,147,483,648 до 2,147,483,647 (0 считается положительным). буфер_u64 Беззнаковое 64-разрядное целое число. буфер_f16 16-битное плавающее число. Это может быть положительное или отрицательное значение в том же диапазоне, что и 16-битное знаковое целое число. буфер_f32 32-битное плавающее число. Это может быть положительное или отрицательное значение в том же диапазоне, что и 32-битное знаковое целое число. буфер_f64 64-разрядное плавающее число. буферный_баул Булево значение. Может быть только 1 или 0(true или false). буферная_строка Строка произвольного размера, завершаемая нулевым завершающим символом. буферный_текст Строка любого размера без завершающего нулевого символа. Функция вернет 0 в случае успеха или -1 в случае неудачи. Синтаксис: buffer_write(buffer, type, value) Аргумент Описание буфер Индекс буфера для записи. тип Тип данных, которые должны быть записаны в буфер (см. список констант выше). значение Данные для записи. Возвращается: 0 в случае успеха или -1 в случае неудачи Пример: buffer_seek(buff, buffer_seek_start, 0); buffer_write(buff, buffer_s16, 0); buffer_write(buff, buffer_s16, x); buffer_write(buff, buffer_s16, y); Приведенный выше код находит начало буфера с идентификатором, хранящимся в переменной "buff", и записывает в него серию знаковых 16-битных целочисленных значений.
