buffer_set_used_size Эта функция предназначена в основном для использования в расширениях и позволяет установить "использованный" размер данного буфера, то есть количество байт, которые были в него записаны. Когда вы записываете данные в буфер из расширения, GameMaker не знает, какая часть буфера была заполнена кодом расширения, и не может прочитать эти данные. Эта функция может быть вызвана расширением, чтобы сообщить движку, сколько байт данных было записано в буфер, чтобы движок мог прочитать эти данные. Синтаксис: buffer_set_used_size(buffer, size); Аргумент Описание буфер Индекс используемого буфера. размер Количество байт, которое следует установить в качестве "используемого" размера. Возвращается: Н/Д Пример: buffer_write(_bufferAddress, buffer_u8, 1); buffer_write(_bufferAddress, buffer_u8, 2); buffer_write(_bufferAddress, buffer_u16, 400); buffer_set_used_size(_bufferAddress, 4); Приведенные выше функции будут вызываться в расширении для записи некоторых данных в буфер GameMaker (через его адрес в памяти), а затем для установки количества байт, которые были записаны в него, чтобы движок мог прочитать эти данные.
