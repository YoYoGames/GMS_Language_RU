Когда одна из файловых функций не работает, вы можете вызвать эту функцию, чтобы узнать, в чем именно заключалась ошибка. была. Игра должна подтвердить пользователю, что он намерен продолжить игру без сохранения. Если пользователь все еще хочет иметь возможность сохраниться, игра должна повторить операцию сохранения. xboxlive_fileerror_quotaexceeded В этом случае игра должна уведомить пользователя, что он не может больше сохранять данные, пока не освободит место, а затем предложить ему способ сделать это. xboxlive_fileerror_noerror Функция сохранения или загрузки файлов выполнялась без проблем.
