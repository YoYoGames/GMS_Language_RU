Сохранение данных Следующие функции связаны с сохранением данных с помощью экспорта UWP для Xbox One и использованием функций XBox Live. Хотя данные сохранения, связанные с прогрессом игрока, всегда должны быть связаны с конкретным пользователем, вы можете сохранять общие данные в области "машинного хранилища", указав в качестве аргумента пользователя значение -1. Эта область зависит от названия, но не ассоциируется с конкретным пользователем. ПРИМЕЧАНИЕ: Размер сохраненного файла не может превышать 16 мегабайт!
