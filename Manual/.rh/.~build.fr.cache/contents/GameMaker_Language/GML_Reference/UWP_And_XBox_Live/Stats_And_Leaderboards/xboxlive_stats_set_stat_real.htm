xboxlive_stats_set_stat_real Эта функция может быть использована для установки значения stat для заданного ID пользователя. Вы вводите идентификатор пользователя, возвращаемый (например) функцией xboxlive_get_user(), затем строку stat для установки(если строка stat еще не существует, то будет создан новый stat и установлен на заданное значение) и значение (вещественное) для установки stat. Обратите внимание, что имя stat должно быть только буквенно-цифровым, без символов и пробелов. При установке значения стата любое предыдущее значение будет отменено, поэтому вы сами должны определить, следует ли обновить значение стата или нет (т.е. проверить, что высокий результат действительно самый высокий), сравнив текущее значение стата с новым, прежде чем устанавливать его. новым, прежде чем установить его. Функция вернет -1, если произошла ошибка или идентификатор пользователя недействителен, или любое другое значение, если функция была вызвана успешно. Синтаксис: xboxlive_stats_set_stat_real(user_id, stat, value); Аргумент Описание идентификатор пользователя Указатель идентификатора пользователя для установки статистики статистика Статистика, которую нужно установить (строка) значение Значение, на которое нужно установить стат (реальное) Возвращается: Реально Пример: если (game_over == true) { if (xboxlive_stats_get_stat(p_user_id, "PercentDone") < 100) { var _val = (global.LevelsFinished / global.LevelsTotal) * 100; xboxlive_stats_set_stat_real(p_user_id, "PercentDone", _val); } } Приведенный выше код проверяет переменную, и если она истинна , то проверяется значение статистики "PercentDone". Если это значение меньше 100, генерируется значение, и статистика устанавливается на это значение.
