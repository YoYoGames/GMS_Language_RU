xboxlive_stats_set_stat_string Эта функция может быть использована для установки stat в строку для заданного ID пользователя. Вы передаете идентификатор пользователя, возвращаемый (например) функцией xboxlive_get_user()а затем строку статистики для установки(если строка stat еще не существует, то будет создан новый stat и установлен в заданное значение), а затем строку, в которую нужно установить stat. Обратите внимание, что имя stat должно быть только буквенно-цифровым, без символов и пробелов, а в строке значение, которое вы задаете, может содержать пробелы и специальные символы. При установке строки статистики любая предыдущая строка будет отменена, поэтому вы сами должны определить, следует ли обновить строку статистики или нет, сравнив текущую строку статистики с новой перед ее установкой. Функция вернет -1, если произошла ошибка или идентификатор пользователя недействителен, или любое другое значение, если функция была вызвана успешно. Синтаксис: xboxlive_stats_set_stat_string(user_id, stat, value); Аргумент Описание идентификатор пользователя Указатель идентификатора пользователя для установки статистики статистика Статистика, которую нужно установить (строка) значение Значение, на которое нужно установить статистику (строка) Возвращается: Реально Пример: xboxlive_stats_set_stat_string(p_user_id, "LevelName", global.LevelName); Приведенный выше код устанавливает статистику "LevelName" в строку, хранящуюся в данной глобальной переменной.
