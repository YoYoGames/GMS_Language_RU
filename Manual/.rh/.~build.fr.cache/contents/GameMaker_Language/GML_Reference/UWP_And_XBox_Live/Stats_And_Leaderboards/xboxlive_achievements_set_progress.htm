xboxlive_achievements_set_progress Эта функция может быть использована для обновления прогресса достижения. Вы вводите идентификатор пользователя, возвращаемый, например, функцией xboxlive_get_user(), а затем строку достижения (это идентификатор достижения, присвоенный в XDP/Windows Dev Center при его создании). Наконец, вы устанавливаете значение прогресса в диапазоне от 0 до 100. Обратите внимание, что система достижений будет отклонять обновления, которые ниже текущего значения прогресса. значение. Синтаксис: xboxlive_achievements_set_progress(user_id, достижение, прогресс); Аргумент Описание идентификатор пользователя Идентификатор пользователя, для которого нужно установить достижение достижение Достижение для установки (строка) прогресс Значение прогресса для установки (от 0 до 100) Возвращается: Н/Д Расширенный пример: Ниже приведен расширенный пример использования этой функции. Для начала вы должны вызвать ее в каком-либо событии, например, в событии Room Start или Create: xboxlive_stats_get_social_leaderboard(user_id, "GlobalTime", 20, 1, false, false); Приведенный выше код будет вызван для получения списка всех позиций в глобальной таблице лидеров в игре и сгенерирует обратный вызов Social Asynchronous Event, с которым мы поступим следующим образом: var _progress = (global.Level / global.LevelMax) * 100; xboxlive_achievements_set_progress(user_id, "Game_Completed", _progress); Приведенный выше код создает процентное значение на основе некоторых глобальных переменных, а затем использует его для установки прогресса достижения для достижения "Game_Completed".
