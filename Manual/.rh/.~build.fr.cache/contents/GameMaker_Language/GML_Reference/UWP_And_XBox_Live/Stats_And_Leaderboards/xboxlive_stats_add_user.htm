xboxlive_stats_add_user Эта функция может быть использована для добавления заданного указателя ID пользователя в менеджер статистики. Это должно быть сделано до использования любой другой функции статистики, чтобы автоматически синхронизировать игру с сервером XBox Live и получить последние значения. Вы предоставляете идентификатор пользователя, возвращенный (например) функцией xboxlive_get_user(), и функция вернет -1, если произошла ошибка или если ID пользователя недействителен, или любое другое значение, если функция была успешно вызвана. Эта функция генерирует обратный вызов, который вызовет системное асинхронное событие. Это событие будет иметь специальную карту DS async_load которая затем должна быть разобрана на наличие следующих ключей: " id" - Будет содержать константу achievement_stat_event " eventname" - Будет содержать строку " LocalUserAdded". " userid" - Идентификатор пользователя, связанный с запросом " error" - 0 в случае успеха, другое значение, если произошла ошибка " errormessage" - Строка с сообщением об ошибке, если таковое имеется Синтаксис: xboxlive_stats_add_user(user_id); Аргумент Описание идентификатор пользователя Идентификатор пользователя (указатель) для добавления Возвращается: Реально Пример: for(var i = 0; i < xboxlive_get_user_count(); ++i;) { user_id[i] = xboxlive_get_user(i); xboxlive_stats_add_user(user_id[i]); } Приведенный выше код получит количество доступных учетных записей пользователей, затем пройдется по ним и присвоит идентификатор учетной записи массиву, а также зарегистрирует пользователя в менеджере статистики.
