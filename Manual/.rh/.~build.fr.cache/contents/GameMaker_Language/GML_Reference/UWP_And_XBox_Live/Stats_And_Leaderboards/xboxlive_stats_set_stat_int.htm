xboxlive_stats_set_stat_int Эта функция может быть использована для установки значения статистики для заданного ID пользователя. Вы вводите ID пользователя, возвращенный (например) функцией xboxlive_get_user(), затем строку статистики для установки(если строка stat еще не существует, то будет создана новая stat и установлено заданное значение) и значение (целое число) для установки stat. Обратите внимание, что имя stat должно быть только буквенно-цифровым, без символов и пробелов. При установке значения стата любое предыдущее значение будет отменено, поэтому вы сами должны определить, следует ли обновить значение стата или нет (т.е. проверить, что высокий результат действительно самый высокий), сравнив текущее значение стата с новым, прежде чем устанавливать его. новым, прежде чем установить его. Функция вернет -1, если произошла ошибка или идентификатор пользователя недействителен, или любое другое значение, если функция была вызвана успешно. Синтаксис: xboxlive_stats_set_stat_int(user_id, stat, value); Аргумент Описание идентификатор пользователя Указатель идентификатора пользователя для установки статистики статистика Статистика, которую нужно установить (строка) значение Значение, на которое нужно установить статистику (целое число) Возвращается: Реально Пример: var _val = xboxlive_stats_get_stat(p_user_id, "GamesPlayed"); xboxlive_stats_set_stat_int(p_user_id, "GamesPlayed", ++_val); Приведенный выше код извлекает значение для статистики "GamesPlayed", затем устанавливает эту же статистику в это значение плюс 1.
