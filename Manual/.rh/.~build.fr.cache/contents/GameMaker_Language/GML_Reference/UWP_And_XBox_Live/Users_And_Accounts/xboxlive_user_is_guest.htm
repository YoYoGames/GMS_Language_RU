xboxlive_user_is_guest Эта функция вернет true, если заданный указатель ID пользователя является гостевым пользователем, и false, если нет. Вы можете получить указатель идентификатора пользователя с помощью функции xboxlive_get_user(). Синтаксис: xboxlive_user_is_guest(user_id); Аргумент Описание идентификатор пользователя Указатель ID пользователя для проверки. Возвращается: Булево Пример: if xboxlive_user_is_guest(user_id[0]) { global.user_name[0] = "GUEST"; } else { global.user_name[0] = xboxlive_gamedisplayname_for_user(user_id[0]); } Вышеприведенные действия сохраняют указатель идентификатора активирующего пользователя в глобальной переменной.
