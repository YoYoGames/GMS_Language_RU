xboxlive_user_is_signing_in Функция проверяет, находится ли пользователь в процессе регистрации, и возвращает true, если это так, и false в противном случае. Синтаксис: xboxlive_user_is_signing_in(); Возвращается: Булево Пример: если !xboxlive_user_is_signed_in() { if !xboxlive_user_is_signing_in() { xboxlive_show_account_picker(); } } else { global.GamerTag = xboxlive_gamertag_for_user(); } Приведенный выше код проверяет, зарегистрирован ли пользователь в XBox Live, и если он зарегистрирован, то код получает его геймертаг и сохраняет его в глобальной переменной для дальнейшего использования. Если пользователь не зарегистрирован, то код проверяет, находится ли он в процессе регистрации, в этом случае ничего не произойдет, а если нет, то откроется окно выбора учетной записи, предлагающее зарегистрироваться.
