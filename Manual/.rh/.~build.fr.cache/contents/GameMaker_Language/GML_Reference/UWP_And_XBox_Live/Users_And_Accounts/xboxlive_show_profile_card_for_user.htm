xboxlive_show_profile_card_for_user С помощью этой функции вы можете запросить, чтобы XBox показал карточку профиля для целевого указателя ID пользователя. Функция требует как ID-указатель пользователя, запрашивающего информацию, так и ID-указатель пользователя. для получения карты профиля. Синтаксис: xboxlive_show_profile_card_for_user(requesting_user_id, target_user_id); Аргумент Описание запрашивающий_идентификатор_пользователя Идентификатор пользователя (указатель) запрашивающего пользователя идентификатор_пользователя Идентификатор пользователя (указатель) пользователя для получения карточки профиля Возвращается: Н/Д Пример: if (gamepad(0, gp_start)) { xboxlive_show_profile_card_for_user(user[0], user[1]); } Приведенный выше код проверяет нажатие кнопки геймпада и, если таковое обнаружено, показывает карточку профиля для данного пользователя.
