xboxlive_user_for_pad Эта функция возвращает указатель ID пользователя, связанный с заданным значением индекса геймпада, или указатель_null, если ни один пользователь не связан с ним. Обратите внимание, что эта функция должна использоваться только с индексами геймпадов, а не с индексами xboxlive_get_user_count(). Синтаксис: xboxlive_user_for_pad(pad_index); Аргумент Описание pad_index Индекс (целое число) слота геймпада для проверки Возвращается: Указатель Пример: for(var i = 0; i < 11; ++i;) { var u_id = xboxlive_user_for_pad(i); if u_id != pointer_null { pad_name[i] = xboxlive_gamedisplayname_for_user(u_id); } else { pad_name[i] = "Press Play"; } } Приведенный выше код перебирает все доступные слоты пэдов и проверяет, есть ли указатель идентификатора пользователя, связанный с ним. Если есть, то в массив устанавливается отображаемое имя пользователя, в противном случае в массив устанавливается текст по умолчанию.
