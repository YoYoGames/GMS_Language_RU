xboxlive_user_id_for_user Эта функция преобразует указатель User ID в строку. Синтаксис: xboxlive_user_id_for_user(user_id); Аргумент Описание идентификатор пользователя Указатель ID пользователя для проверки. Возвращается: Строка Пример: var _uid = xboxlive_user_for_pad(0); uid_string = xboxlive_user_id_for_user(_uid); Приведенный выше код получает указатель ID пользователя, назначенного на слот [0] геймпада, а затем сохраняет этот ID пользователя в виде строки в переменной экземпляра.
