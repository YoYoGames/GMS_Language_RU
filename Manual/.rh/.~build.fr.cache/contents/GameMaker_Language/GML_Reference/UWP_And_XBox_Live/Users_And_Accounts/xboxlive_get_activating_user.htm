xboxlive_get_activating_user С помощью этой функции вы можете получить указатель идентификатора пользователя, который запустил игру с консоли. ПРИМЕЧАНИЕ: Использование этого указателя зависит от того, что вы хотите делать в игре, но не стоит полагаться на то, что это будет текущий ID пользователя, так как активирующий пользователь может выйти из игры во время ее работы. Синтаксис: xboxlive_get_activating_user(); Возвращается: Указатель Пример: global.activUser = xboxlive_get_activating_user(); Вышеуказанное сохраняет указатель идентификатора активирующего пользователя в глобальной переменной Назад: Пользователи и учетные записи Далее: xboxlive_user_is_guest © Copyright YoYo Games Ltd. 2021 Все права защищены
