xboxlive_pad_count_for_user С помощью этой функции вы можете узнать, сколько "слотов" пэда активно для текущего пользователя (см. xboxlive_pad_for_user() для более подробной информации). Синтаксис: xboxlive_pad_count_for_user(user_id, slot); Аргумент Описание идентификатор пользователя Идентификатор пользователя (указатель) для проверки Возвращается: Целое число Пример: global.slots[0] = xboxlive_pad_count_for_user(user_id[0]) Приведенный выше код сохраняет количество колодок, связанных с заданным указателем идентификатора пользователя.
