xboxlive_set_rich_presence Эта функция устанавливает строку богатого присутствия для данного пользователя. Строка богатого присутствия показывает активность пользователя в игре после названия игры, в которую играет пользователь, через дефис. Эта строка отображается под Gamertag в списке "Друзья и клубы", а также в профиле пользователя Xbox Live. При использовании этой функции вам необходимо предоставить указатель User ID для пользователя, а затем вы можете отметить пользователя как активного в данный момент в игре или нет (используя true/false). Если установлено значение false, то к строке присутствия будет будет добавлено "/afk" или что-нибудь подходящее. Следующим аргументом является ID строки присутствия, которую нужно показать, и, наконец, вы можете (опционально) указать строку service_config_id. Обратите внимание, что это необязательный аргумент, так как если вы вызвали xboxlive_stats_setup() вам не нужно передавать здесь service_config_id. Для получения дополнительной информации о богатом присутствии и о том, как настроить строки для использования в партнерском центре, обратитесь к следующей документации Microsoft: Обзор строк присутствия Синтаксис: xboxlive_set_rich_presence(user_id, is_user_active, rich_presence_string, [service_config_id]) Аргумент Описание идентификатор пользователя Указатель ID пользователя для проверки. is_user_active Пометить пользователя как активного или нет. rich_presence_string Идентификатор строки присутствия, который необходимо использовать (как определено в Центре партнеров - не более 50 символов). service_config_id [OPTIONAL] Это строка config_id для игры. Возвращается: Н/Д Пример: var pad_uid = xboxlive_user_for_pad(0); xboxlive_set_rich_presence(pad_uid, true, "Playing_Challenge"); Приведенный выше код получает указатель идентификатора пользователя, назначенного на слот [0] геймпада, а затем устанавливает строку присутствия для этого пользователя.
