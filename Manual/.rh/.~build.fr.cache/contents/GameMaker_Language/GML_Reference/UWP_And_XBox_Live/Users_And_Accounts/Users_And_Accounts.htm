Пользователи и учетные записи Этот раздел предназначен для пользователей платформ UWP или Xbox One и описывает функции XBox Live, доступные для работы с пользователями и учетными записями пользователей. Xbox One позволяет нескольким пользователям одновременно входить в консоль. Каждый пользователь имеет определенный контроллер, связанный с ним, и все данные сохраненных игр также зависят от конкретного пользователя. Чтобы максимально использовать возможности этой системы, GameMaker Studio 2 вводит ряд функций, связанных с ассоциацией между контроллерами и пользователями только для XBox One, позволяющих запрашивать информацию о пользователе, а также управлять обработкой сохранений для каждого пользователя. Вы можете найти дополнительную информацию о разработчиках этой системе в официальном документе на сайте разработчиков Microsoft. ПРИМЕЧАНИЕ: Обратите внимание, что ID пользователя уникален для каждого человека и сохраняется на разных консолях, поэтому если один и тот же пользователь входит в систему на двух разных устройствах, его ID будет одинаковым.
