uwp_license_trial_time_remaining Эта функция возвращает приблизительное количество секунд до истечения срока испытания, основанного на времени. Значение, возвращаемое этой функцией, не имеет смысла, если испытание не основано на времени или если игра не запущена в режиме испытания. Синтаксис: uwp_license_trial_time_remaining(); Возвращается: Реально Пример: if uwp_license_trial_version() { var secs = uwp_license_trial_time_remaining() mod 60; var mins = uwp_license_trial_time_remaining() div 60; var hours = mins div 60; if secs < 10 secs = "0" + string(secs) else secs = string(secs); if mins > 60 mins -= (hours * 60); if mins < 10 mins = "0" + string(mins) else mins = string(mins); if hours > 9 hours = "9" else hours = string(hours); time_string = hours + ":" + mins + ":" + secs; } Приведенный выше код проверяет, запущена ли игра с пробной лицензией, и если да, то создает строку с временем до истечения срока действия лицензии.
