uwp_license_trial_user Эта функция возвращает указатель ID пользователя, чья пробная лицензия используется в настоящее время, или возвращает указатель_null, если пользователь в настоящее время не активен или если игра в настоящее время не находится в пробном режиме. Синтаксис: uwp_license_trial_user(); Возвращается: Указатель Пример: if gamepad_button_check_pressed(0, gp_start) { if uwp_license_trial_version() { var _uid = uwp_license_trial_user(); uwp_show_help(_uid); } } Приведенный выше код проверяет, нажата ли кнопка, а затем проверяет, запущено ли приложение с пробной лицензией. Если это так, то он получает идентификатор пользователя для лицензии и открывает файл справки для этого пользователя.
