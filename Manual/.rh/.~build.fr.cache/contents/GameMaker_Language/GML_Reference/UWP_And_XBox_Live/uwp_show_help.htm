uwp_show_help Эта функция запускает приложение помощи Xbox One, которое, в свою очередь, отобразит файл помощи вашей игры (который представляет собой файл в формате HTML, загруженный вами ранее на XBox Developer Portal). Обратите внимание, что при вызове этой функции указывается идентификатор пользователя чтобы при отображении использовалась информация о локали пользователя. Идентификатор пользователя можно получить с помощью функций xboxlive_get_user(), xboxlive_get_activating_user(), или uwp_license_trial_user(). Синтаксис: uwp_show_help(user_id); Аргумент Описание идентификатор пользователя Указатель идентификатора пользователя, для которого нужно открыть справочный файл. Возвращается: Булево Пример: if gamepad_button_check_pressed(0, gp_start) { if uwp_license_trial_version() { var _uid = uwp_license_trial_user; uwp_show_help(_uid); } } Приведенный выше код проверяет, нажата ли кнопка, а затем проверяет, запущено ли приложение с пробной лицензией. Если это так, то он получает идентификатор пользователя для лицензии и открывает файл справки для этого пользователя.
