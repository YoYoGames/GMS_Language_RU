xboxlive_matchmaking_stop Эта функция может быть использована для завершения сеанса подбора для данного пользователя. Синтаксис: xboxlive_matchmaking_stop(user_id); Аргумент Описание идентификатор пользователя Указатель идентификатора пользователя для использования Возвращается: Н/Д Пример: if !xboxlive_user_is_signed_in(user_id[0]) { xboxlive_matchmaking_stop(user_id[0]); } Приведенный выше код завершит сеанс подбора для данного пользователя.
