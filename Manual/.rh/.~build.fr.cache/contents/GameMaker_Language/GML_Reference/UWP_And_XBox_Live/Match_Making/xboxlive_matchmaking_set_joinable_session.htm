xboxlive_matchmaking_set_joinable_session Эта функция устанавливает (ранее созданную) сессию доступной для присоединения других пользователей через пользовательский интерфейс системы. У пользователя есть только одна сессия, к которой можно присоединиться одновременно, и когда он покидает сессию (или сессия завершается), это значение будет очищено, однако вы можете очистить его вручную, передав -1 в качестве аргумента session. Синтаксис: xboxlive_matchmaking_set_joinable_session(local_user, session_that_is_joinable); Аргумент Описание локальный_пользователь Указатель локального идентификатора пользователя. session_that_is_joinable Идентификатор сеанса, который нужно сделать присоединяемым, или -1. Возвращается: Н/Д Пример: if (global.session_ID != -1) { xboxlive_matchmaking_set_joinable_session(xboxlive_user_for_pad(0), global.session_ID); } Приведенный выше код проверяет наличие действительного значения ID сессии, и если оно обнаружено, то устанавливает сессию как присоединяемую.
