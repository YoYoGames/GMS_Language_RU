копирование_строки С помощью этой функции вы можете легко выбрать несколько символов из строки для копирования в другую строку. Первый символ в строке всегда индексируется как 1, а не 0, как можно было бы ожидать, поэтому для копирования (например) первых пяти символов строки будет string_copy(str, 1, 5). Еще один пример приведен ниже. Синтаксис: string_copy(str, index, count); Аргумент Описание str Строка для копирования. индекс Позиция первого символа в строке для копирования (нумеруется от 1) считать Количество символов, начиная с позиции первого, для копирования Возвращается: Строка Пример: name = keyboard_string; if (string_length(name) > 15) { keyboard_string = string_copy(name, 1, 15); } Приведенный выше код позволяет игроку ввести строку с клавиатуры, которая затем сохраняется в переменной name. Затем эта переменная проверяется, не превышает ли она максимальную длину 15 символов, и если да, то строка keyboard_string заменяется 15-символьной копией строки name. Это эффективно ограничивает ввод игрока 15 символами. Назад: Струны Далее: string_count © Copyright YoYo Games Ltd. 2021 Все права защищены
