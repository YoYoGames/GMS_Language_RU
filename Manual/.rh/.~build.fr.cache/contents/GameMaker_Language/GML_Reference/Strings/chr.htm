chr Эта функция возвращает строку, содержащую символ, который относится к входному коду Unicode для отображения. Этот символ зависит от кодовой страницы набора символов текущего чертежного шрифта, и если шрифт не задан, то используется кодовая страница по умолчанию для машины. Синтаксис: chr(val); Аргумент Описание val Значение кода Юникода, из которого нужно получить строку Возвращается: Строка Пример: mystring = chr(53) + chr(48); Это установит mystring в "50" (строка, а не целое число), поскольку chr(53) - это "5", а chr(48) - "0".
