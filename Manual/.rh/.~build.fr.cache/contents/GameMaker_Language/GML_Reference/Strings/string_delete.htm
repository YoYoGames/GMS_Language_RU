string_delete Вы можете использовать эту функцию для удаления определенной части строки. Итак, вы вводите входную строку и начальную и конечную позицию в этой строке для удаления символов (индекс начинается с 1), и функция вернет новую строку без этого участка в ней. Синтаксис: string_delete(str, index, count); Аргумент Описание str Строка для копирования и удаления. индекс Позиция первого символа для удаления. считать Количество символов для удаления. Возвращается: Строка Пример: str1 = "Helloo World"; str2 = string_delete(str1, 5, 1); Это установит str2 в "Hello World", так как удалит лишнее "o" из "Hello", т.е. строка отсчитывает от первой буквы 5 мест, затем удаляет 1 символ).
