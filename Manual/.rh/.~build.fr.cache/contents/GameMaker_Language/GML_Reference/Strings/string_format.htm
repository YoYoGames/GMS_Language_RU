строковый_формат Превращает вещественное число в строку, используя собственное форматирование, где можно выбрать, сколько "знаков" сохраняется в строке и сколько десятичных знаков также сохраняется. Оба варианта могут быть очень удобны, некоторые игры предпочитают отображать счет в виде заданного числа цифр, а контроль над десятичными знаками может быть полезен для высокой точности, которую два знака после запятой string() не может обеспечить. Если количество указанных знаков больше, чем отображаемое значение и/или число плюс указанные десятичные знаки меньше общего количества знаков, то перед значением будут добавлены пробелы, чтобы компенсировать разницу (см. пример ниже). Справа от десятичной точки будут добавлены нули, если указанное значение меньше, чем общее значение и количество знаков после запятой. Формат по умолчанию - без дополнительных пробелов слева и только два десятичных знака справа, например, "265,73". Синтаксис: string_format(val, tot, dec); Аргумент Описание val Вещественное число, которое должно быть преобразовано в строку. tot Общее количество мест основного числа, которое должно быть показано. Нули или пробелы будут вставлены в соответствии с этим. дек Количество десятичных знаков, которые должны быть включены. Возвращается: Строка Пример: str1 = string_format(1234, 8, 0); str2 = string_format(pi, 1, 10); str3 = string_format(pi, 5, 5); Это установит str1 в " 1234", str2 в "3 .1415926535 " и str3 в " 3.14159 ".
