строка_верхняя С помощью этой функции вы можете заставить строку содержать только символы верхнего регистра. ПРИМЕЧАНИЕ: Эта функция распознает только 26 букв английского алфавита от A до Z. Синтаксис: string_upper(string); Аргумент Описание строка Строка для преобразования в верхний регистр. Возвращается: Строка Пример: str1 = "Hello World"; str2 = string_upper(str1); Приведенный выше код установит переменную str2 в значение "HELLO WORLD".
