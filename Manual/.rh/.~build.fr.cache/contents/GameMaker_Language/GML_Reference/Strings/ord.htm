орден Эта функция принимает строку ввода одного символа и возвращает значение Unicode (UTF8) для этого символа. Обратите внимание, что при использовании с keyboard_check* функции, входная строка может быть длиной только в один символ и может быть только числом от 0 до 9 или заглавным римским символом от A до Z. Синтаксис: ord(string); Аргумент Описание строка Строка, с помощью которой можно найти код Unicode. Возвращается: Реально Пример: if keyboard_check(ord("W")) { y -= 4; } Это переместит вызывающий экземпляр на четыре пикселя вверх, если удерживать клавишу W.
