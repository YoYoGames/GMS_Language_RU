буфер обмена_получить_текст Эта функция возвращает строку текста, содержащегося в буфере обмена. если текст не сохранен, она возвращает пустую строку "". ПРИМЕЧАНИЕ: Эта функция действительна только для Windows цели. Синтаксис: clipboard_get_text(); Возвращается: Строка Пример: if clipboard_has_text() { str = clipboard_get_text(); clipboard_set_text(""); } Приведенный выше код проверяет буфер обмена на наличие текста и, если он есть, считывает его как строку в переменную "str". Наконец, буфер обмена очищается путем установки в него пустой строки. Назад: Струны Далее: clipboard_set_text © Copyright YoYo Games Ltd. 2021 Все права защищены
