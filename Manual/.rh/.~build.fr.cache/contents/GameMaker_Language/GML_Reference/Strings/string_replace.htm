строка_замены Вы можете использовать эту функцию для разбора строки в поисках определенной части, которая затем может быть заменена новой строкой, которую вы указали. Синтаксис: string_replace(str, substr, newstr); Аргумент Описание str Строка, которую нужно скопировать. субстрат Подстрока в строке, которую нужно заменить. newstr Новая подстрока для замены предыдущей. Возвращается: Строка Пример: str1 = "Hello Earth"; str2 = string_replace(str1, "Earth", "World"); Это установит str2 в str1, но при этом экземпляр "Earth" будет заменен на "World", в результате чего str2 будет "Hello World".
