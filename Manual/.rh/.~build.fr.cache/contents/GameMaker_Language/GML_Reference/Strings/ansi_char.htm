ansi_char Эта функция возвращает строку, содержащую символ с необработанным набором значений BYTE. Он не будет и не должен отображаться, но он будет правильно сохранен на диске для использования при кодировании. Синтаксис: ansi_char(val); Аргумент Описание val Необработанное значение байта. Возвращается: Строка (один символ) Пример: var str1 = ansi_char($EF); var str2 = ansi_char($BB); var str3 = ansi_char($BF); file_text_write_string(global.saveFile, str1 + str2 + str3); Приведенный выше код создает строку из необработанных байтовых данных и записывает ее в (ранее открытый) файл.
