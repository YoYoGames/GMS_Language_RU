ширина строки Эта функция возвращает ширину (в пикселях) введенной строки с учетом возможных разрывов строк. Это очень удобно для вычисления расстояний между элементами текста на основе общей ширины букв, составляющих строку как она была бы нарисована с draw_text() с использованием определенного в данный момент шрифта. Синтаксис: string_width(string); Аргумент Описание строка Строка для измерения ширины. Возвращается: Реально Пример: var ww = string_width(str_Name + " "); draw_text(32, 32, str_Name)); draw_text(32 + ww, 32, "выиграл игру!"); Приведенный выше код получит ширину заданной строки, а затем нарисует две строки текста, используя возвращенную ширину строки в качестве разделителя.
