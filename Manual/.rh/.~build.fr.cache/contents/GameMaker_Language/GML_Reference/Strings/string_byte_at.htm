строка_байт_ат Возвращает необработанное значение байта в виде вещественного значения в заданной позиции в заданной строке. Синтаксис: string_byte_at(str, index); Аргумент Описание str Строка для проверки. индекс Позиция, из которой нужно получить байт. Возвращается: Реально Пример: newbyte = string_byte_at("Hello World", 5); Это установит newbyte в необработанное байтовое значение шестой буквы слова "Hello World".
