string_replace_all Вы можете использовать эту функцию для разбора строки в поисках определенных частей, которые затем могут быть заменены новой строкой, указанной вами, во всех местах, где они встречаются. Синтаксис: string_replace_all(str, substr, newstr); Аргумент Описание str Строка, которую нужно скопировать. субстрат Подстрока в строке, которая должна быть заменена во всех случаях. newstr Новая подстрока для замены предыдущей(их). Возвращается: Строка Пример: str1 = "Hexxo Worxd"; str2 = string_replace_all(str1, "x", "l"); Это установит str2 в str1, но с экземплярами "x", замененными на "l", в результате чего str2 будет "Hello World".
