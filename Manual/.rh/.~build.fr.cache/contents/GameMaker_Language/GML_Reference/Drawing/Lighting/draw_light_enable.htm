draw_light_enable Эта функция используется для включения определенного освещения. Когда вы определяете позиционный или направленный свет, вы должны присвоить ему индексный номер, который затем используется этой функцией для включения или выключения света. По умолчанию отключено(false). ПРИМЕЧАНИЕ: Имеется только 8 аппаратных светильников, поэтому одновременно могут быть включены только 8 определенных светильников (хотя можно определить и больше). Синтаксис: draw_light_enable(ind, enable); Аргумент Описание инд Индексный номер светильника (от 0 до 7) включить Включить или выключить все освещение(true или false) Возвращается: Н/Д Пример: draw_set_lighting(true); draw_light_define_direction(1, 0, 1, 0, c_white); draw_light_enable(1, true); Приведенный выше код включит освещение для всей сцены, затем определит белый направленный свет в пространстве комнаты и, наконец, включит этот свет.
