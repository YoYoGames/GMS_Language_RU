draw_set_lighting Эта функция используется для включения всех эффектов освещения. По умолчанию отключено(false). Синтаксис: draw_set_lighting(enable); Аргумент Описание включить включить или выключить все освещение(true или false) Возвращается: Н/Д Пример: draw_set_lighting(true); draw_light_define_direction(1, 0, 1, 0, c_white); draw_light_enable(1, true); Приведенный выше код включит освещение для всей сцены, затем определит белый направленный свет в пространстве комнаты и, наконец, включит этот свет.
