color_get_value Эта функция возвращает значение (светимость) заданного цвета. Это количество "света", которое примешивается к конечному цвету, и является частью метода определения цвета по оттенку, насыщенности и значению. Следующее изображение иллюстрирует как это значение соответствует цветовой шкале HSV: Синтаксис: color_get_value(col); Аргумент Описание col Цвет для проверки Возвращается: Int Пример: col = make_colour_hsv(random(255), 255, color_get_value(c_teal)); Приведенный выше код получает значение, используемое для создания цветовой константы "c_teal", а затем использует это значение для установки случайного цвета с той же светлотой, сохраняя полученный цвет в переменной "col".
