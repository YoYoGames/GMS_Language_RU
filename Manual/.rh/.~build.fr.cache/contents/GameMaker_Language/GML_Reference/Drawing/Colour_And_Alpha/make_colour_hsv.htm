make_colour_hsv GameMaker Studio 2 предоставляет эту функцию (а также другие) для того, чтобы пользователь мог создавать свои собственные цвета. Эта конкретная функция использует три компонента: оттенок, насыщенность и значение (также известное как "светимость"). для создания желаемого цвета. Эти значения находятся в диапазоне от 0 до 255, поэтому с помощью этой функции можно создать 16 777 216 (256*256*256) цветов! Ниже показано изображение того, как выглядят эти компоненты в разделенном виде: Изображение слева представляет собой разбивку отдельных компонентов функции, а справа - иллюстрацию того. того, как изменение этих компонентов влияет на конечный цвет. Как видите, даже не изменяя оттенок (он остается на уровне 0, красный), мы можем значительно изменить цвет с помощью двух других параметров, при этом насыщенность заставляет цветовой тон перейти от серого (0) до чистого тона (255), а значение - от более темного (0) до более светлого (255). Синтаксис: make_colour_hsv(hue, sat, val); Аргумент Описание оттенок Оттенок цвета сб Насколько насыщен цвет val Насколько темный цвет Возвращается: Целое число Пример: col = make_colour_hsv(100, 145, 255); Приведенный выше код использует функцию для создания цвета и сохранения его значения в переменной "col" для последующего использования.
