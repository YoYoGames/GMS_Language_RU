draw_getpixel С помощью этой функции вы можете получить значение цвета любого пикселя, который рисуется на текущей цели рендеринга. Это означает, что результаты будут зависеть от события, в котором вызывается функция, а также от используемой целевой поверхности. Обратите внимание, что эта функция не возвращает альфа-значения, для этого следует использовать функцию draw_getpixel_ext(). ПРИМЕЧАНИЕ: Эта функция приведет к значительному снижению производительности, поэтому ее следует использовать только в случае крайней необходимости. Синтаксис: draw_getpixel(x, y); Аргумент Описание x Координата x пикселя для проверки y Координата y пикселя для проверки Возвращается: Реально Пример: draw_set_colour(draw_getpixel(500,200)); Приведенный выше код установит цвет рисунка для всего текста, форм и примитивов на цвет пикселя в точке (500,200) на игровом экране.
