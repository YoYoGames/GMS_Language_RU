draw_clear Эта функция может быть использована для очистки всего экрана (без альфа-смешения) до заданного цвета и предназначена только для использования в событии draw экземпляра (при использовании в любом другом событии она не будет отображаться). Она также может быть полезна для очистки поверхностей при их новом создании. Синтаксис: draw_clear(col); Аргумент Описание col Цвет, которым будет очищен экран Возвращается: Н/Д Пример: draw_clear(c_blue); Это очистит экран от синего цвета.
