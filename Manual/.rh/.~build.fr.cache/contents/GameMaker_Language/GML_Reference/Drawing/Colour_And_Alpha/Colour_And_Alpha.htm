Цвет и альфа GameMaker Studio 2 имеет обширный набор функций для манипулирования цветом и альфа-каналом рисуемого на экране изображения (о смешивании изображений с использованием специальных режимов наложения и других специфических для PGU функций см. см. раздел о GPU Control. Используя эти функции, вы можете разделять цвета на составные части, создавать новые цвета и манипулировать альфой отрисовки, чтобы изменить, как различные различные объекты в вашей игре рисуются вместе. Помимо этих функций, существует также ряд констант, которые можно использовать в качестве базовых цветов в этих (и других) функциях: Постоянная Цвет Значение c_aqua 16776960 c_black 0 c_blue 16711680 c_dkgray 4210752 c_fuchsia 16711935 c_gray 8421504 c_зелёный 32768 c_lime 65280 c_ltgray 12632256 c_maroon 128 c_navy 8388608 c_olive 32896 c_оранжевый 4235519 c_purple 8388736 c_red 255 c_серебро 12632256 c_teal 8421376 c_white 16777215 жёлтый 65535 Обратите внимание, что вы также можете создавать цвета из их шестнадцатеричного значения, используя предварительно символ "$", и эти значения также могут быть использованы в цветовых функциях для получения оттенков компонентов или для смешивания и т.д.. Шестнадцатеричное значение для этих цветов разбивается на три части в формате Синий / Зеленый / Красный, поэтому, например, для определения пурпурного цвета можно использовать: col = $983c95 Где 98 - синий компонент, 3c - зеленый компонент и 95 - красный компонент. Следующие функции можно использовать для получения компонентных оттенков, составных оттенков, насыщенности и светимости выбранного цвета, а также альфа и других свойств: color_get_blue color_get_green color_get_red цвет_получить_оттенок color_get_saturation color_get_value draw_getpixel draw_getpixel_ext draw_get_colour draw_get_alpha Эти функции можно использовать для создания цветов из необработанных входных значений цветовых компонентов: make_colour_hsv make_colour_rgb слияние_цветов Следующие функции можно использовать для настройки различных параметров рисования на экране, включая альфа, цвет и смешивание: draw_clear draw_clear_alpha draw_set_alpha draw_set_colour Назад: Рисунок Следующее: Управление графическим процессором © Copyright YoYo Games Ltd. 2021 Все права защищены
