слияние_цветов С помощью этой функции вы можете взять два цвета и затем объединить их вместе, чтобы получить новый цвет. Количество каждого из составляющих цветов может быть определено путем изменения аргумента "amount", при этом значение 0 возвращает первый цвет (col1), значение 1 вернет второй цвет (col2), а значение между ними вернет соответствующую смесь. Например, значение 0,5 смешает два цвета поровну. На следующем изображении показано, как это работает на примере слияния красного и синий: Синтаксис: merge_colour(col1, col2, amount); Аргумент Описание кол1 Первый цвет, который сливается кол2 Второй цвет для слияния сумма Сколько каждого цвета должно быть объединено. Например, 0 вернет col1, 1 вернет col2, а 0,5 вернет слияние обоих цветов поровну. Возвращается: Int Пример: col = merge_colour(c_lime, c_orange, 0.5); Приведенный выше код использует функцию для создания цвета путем слияния лайма и оранжевого 50/50, а затем сохраняет его значение в переменной "col" для последующего использования.
