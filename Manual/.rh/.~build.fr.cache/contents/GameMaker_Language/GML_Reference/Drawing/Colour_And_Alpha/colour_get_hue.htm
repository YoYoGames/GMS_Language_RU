цвет_получить_оттенок Эта функция возвращает оттенок заданного цвета. Это "чистый" цветовой тон, который является частью метода оттенка, насыщенности и значения (светлоты) для определения цвета. Следующее изображение иллюстрирует, как это значение соответствует HSV шкале цвета: Синтаксис: color_get_hue(col); Аргумент Описание col Цвет для проверки Возвращается: Int Пример: col = merge_colour(color_get_hue(c_teal), c_white, 0.5); Приведенный выше код получает оттенок, используемый для создания цветовой константы "c_teal", а затем объединяет его с белым в соотношении 50:50, сохраняя полученный цвет в переменной "col".
