gpu_set_alphatestenable Эта функция включает или отключает альфа-тестирование для вашей игры (по умолчанию оно отключено). Включив альфа-тестирование, вы можете использовать сопутствующую функцию gpu_set_alphatestref() для установки "отсечки" На изображении ниже показана разница, которую можно увидеть при включении и выключении альфа-тестирования: ПРИМЕЧАНИЕ: Эта функция может негативно повлиять на производительность устройств iOS и Android. Синтаксис: gpu_set_alphatestenable(enable); Аргумент Описание включить Включить или отключить альфа-тестирование(true / false) Возвращается: Н/Д Пример: if !gpu_get_alphatestenable() { gpu_set_alphatestenable(true); gpu_set_alphatestref(128); } Приведенный выше код проверит, включено ли альфа-тестирование, и если нет, то включит альфа-тестирование и установит порог теста на 128 (будут отрисованы только пиксели с альфой более 0,5).
