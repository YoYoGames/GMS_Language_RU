gpu_set_colourwriteenable С помощью этой функции можно включить или выключить цветовые каналы и альфа-канал для всего дальнейшего рисования. Например, вы можете использовать эту функцию для создания поверхностей только с альфа-каналом (отключите запись RGB перед использованием поверхности) или для создания специальных эффектов при рисовании на экране. Значение по умолчанию для каждого из компонентов равно true, и они могут быть заданы либо как четыре уникальных аргумента, либо как четырехэлементный одномерный массив со следующими элементами в нем, которые будут либо true (включено) или false (отключено): [0] = Красный канал включен/выключен [1] = Зеленый канал включен/выключен [2] = Синий канал включен/выключен [3] = Альфа-канал включен/выключен Синтаксис: gpu_set_colorwriteenable(red, green, blue, alpha); Аргумент Описание красный Включить/выключить красный канал зеленый Включить/выключить зеленый канал синий Включение/выключение синего канала альфа Включить/выключить альфа-канал Возвращается: Н/Д Пример: var cw = gpu_get_colorwriteenable(); cw[3] = false; gpu_set_colorwriteenable(cw); Приведенный выше код получает текущие значения записи цвета, а затем устанавливает компонент alpha в false.
