gpu_get_blendenable Эта функция может быть использована для получения состояния альфа-клинки. Если она возвращает true, то альфа-клинки включены, а если false, то отключены. По умолчанию она включена, поэтому функция возвращает true. Синтаксис: gpu_get_blendenable(); Возвращается: Булево Пример: if gpu_get_blendenable() == false { gpu_set_blendenable(true); } Приведенный выше код проверяет состояние альфа-смешивания, и если оно отключено, то снова включается.
