gpu_set_texrepeat Эта функция может быть использована для указания GameMaker Studio 2, должны ли текстуры повторяться(true) или нет(false) при использовании для моделей и примитивов. Обычно ширина и высота текстуры лежат в диапазоне 0 - 1, но если вы укажете значение больше 1 по умолчанию остальная часть не отрисовывается (вся текстура будет отрисована с "пустым" участком, соответствующим проценту больше 1, который используется. Таким образом, значение 2 для ширины или высоты текстуры будет рисовать текстуру в половину пространства и оставит другую половину пустой). Однако при использовании этой функции для установки параметра repeat в true текстура автоматически повторяется для всех моделей и примитивов. Скорее всего, эти функции используются для повторения текстуры в 3D, но для того, чтобы она работала и не вытягивала изображения с остальной страницы текстуры, используемый спрайт должен быть помечен как "отдельная страница текстуры" в редакторе спрайтов. ПРИМЕЧАНИЕ: Спрайты и фоны всегда рисуются без повторения, поэтому после рисования спрайта или фона это значение сбрасывается в false. Это поведение по умолчанию не влияет на настройки текстурной сцены (т.е. вызов функции повлияет, но внутренняя установка значения false не повлияет). ПРИМЕЧАНИЕ: Эта настройка будет перекрывать все повторы текстурных стадий, установленные для шейдеров при использовании функции gpu_set_texrepeat_ext(). Синтаксис: gpu_set_texrepeat(enable); Аргумент Описание включить Включить или отключить фильтрацию текстур(true / false) Возвращается: Н/Д Пример: if gpu_get_texrepeat() { gpu_set_texrepeat(false); } else { gpu_set_texrepeat(true); } Приведенный выше код проверяет, включено или выключено повторение текстуры, и переключает его соответствующим образом.
