gpu_get_alphatestref Вы можете использовать эту функцию, чтобы найти текущее значение для ссылки альфа-теста (по умолчанию 0, но вы можете использовать функцию gpu_set_alphatestref() чтобы установить другое значение). Синтаксис: gpu_get_alphatestref(); Возвращается: Реальный (от 0 до 255) Пример: если gpu_get_alphatestenable() { if gpu_get_alphatestref() < 254 { gpu_set_alphatestref(254); } } Приведенный выше код проверяет, включено ли альфа-тестирование, и если включено, то проверяет текущее опорное значение альфа-теста и устанавливает его, если оно меньше 254.
