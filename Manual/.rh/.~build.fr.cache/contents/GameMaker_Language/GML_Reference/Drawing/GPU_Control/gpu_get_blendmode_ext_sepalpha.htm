gpu_get_blendmode_ext_sepalpha Эта функция может использоваться для получения текущего расширенного режима наложения, используемого для рисования, разделяя режимы наложения на две части - компонент RGB и компонент Alpha. Функция возвращает 4-элементный одномерный массив со следующими элементами в нем: [0] = режим наложения источника за вычетом альфа-компонента (по умолчанию bm_src_alpha (As, As, As)) [1] = Режим наложения назначения за вычетом альфа-компонента (по умолчанию bm_inv_src_alpha (1-As, 1-As, 1-As)). [2] = исходный альфа-компонент (по умолчанию bm_src_alpha (As)) [3] = Альфа-компонент назначения (по умолчанию bm_inv_src_alpha (1-As)) Значениями массива будут одни из следующих констант ("s" обозначает значение, взятое из источника, а "d" - значение из пункта назначения): Постоянная Коэффициент смешивания (красный, зеленый, синий, альфа) bm_zero (0, 0, 0, 0) bm_one (1, 1, 1, 1) bm_src_colour (Rs, Gs, Bs, As) bm_inv_src_colour (1-Rs, 1-Gs, 1-Bs, 1-As) bm_src_alpha (As, As, As, As, As) bm_inv_src_alpha (1-Ас, 1-Ас, 1-Ас, 1-Ас, 1-Ас) bm_dest_alpha (Ad, Ad, Ad, Ad, Ad) bm_inv_dest_alpha (1-Ad, 1-Ad, 1-Ad, 1-Ad, 1-Ad) bm_dest_colour (Rd, Gd, Bd, Ad) bm_inv_dest_colour (1-Rd, 1-Gd, 1-Bd, 1-Ad) bm_src_alpha_sat (f, f, f, 1), где f = min(As, 1-Ad) При создании расширенных режимов наложения таким образом компоненты RGB будут взяты из одного режима наложения, а компоненты Alpha - из другого, так что, например, если вы используете bm_zero (0, 0, 0, 0, 0) в качестве режима наложения RGB и bm_one в качестве режима наложения alpha, вы получите составной режим наложения (0,0,0,0,1). Обратите внимание, что вы можете изменить эти значения непосредственно в массиве и передать полный массив в функцию в качестве единственного аргумента (как показано в примере ниже). Синтаксис: gpu_get_blendmode_ext_sepalpha(); Возвращается: 4-элементный одномерный массив (константы см. выше) Пример: var bm = gpu_get_blendmode_ext_sepalpha(); bm[0] = bm_src_alpha; bm[1] = bm_inv_src_alpha; gpu_set_blendmode_ext_sepalpha(bm); Приведенный выше код создает локальный массив и получает текущий расширенный режим наложения с отдельными альфа-значениями. Затем этот массив обрабатывается и возвращается для установки режима наложения.
