gpu_get_colourwriteenable Эта функция может быть использована для получения текущих значений разрешения записи цветов. Функция возвращает 4-элементный массив 1D со следующими элементами, которые будут либо true (включено), либо false (отключено). По умолчанию все цвета запись установлена на true: [0] = Красный канал включен/выключен [1] = Зеленый канал включен/выключен [2] = Синий канал включен/выключен [3] = Альфа-канал включен/выключен Синтаксис: gpu_get_colorwriteenable(); Возвращается: 4-элементный одномерный массив (подробности см. выше) Пример: var cw = gpu_get_colorwriteenable(); cw[3] = false; gpu_set_colorwriteenable(cw); Приведенный выше код получает текущие значения записи цвета, а затем устанавливает компонент alpha в false.
