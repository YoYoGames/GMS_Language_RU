gpu_get_state Эта функция получает текущее состояние GPU, возвращая его в виде карты DS Map. Этой картой можно манипулировать или даже сохранить, а затем вернуть эту карту в GPU с помощью функции gpu_set_state(). Синтаксис: gpu_get_state(); Возвращается: Целое число (идентификатор карты DS) Пример: gpu_map = gpu_get_state(); Приведенный выше код сохраняет текущее состояние GPU в переменной.
