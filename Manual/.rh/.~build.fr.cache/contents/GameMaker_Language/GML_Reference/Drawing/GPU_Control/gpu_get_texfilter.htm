gpu_get_texfilter С помощью этой функции вы можете проверить, включена ли фильтрация текстуры (линейная интерполяция) (возвращается true) или нет (возвращается false). Для получения дополнительной информации о фильтрации текстур см. функцию gpu_set_texfilter(). ПРИМЕЧАНИЕ: На цели HTML5 эта функция будет работать только при включенном WebGL. Синтаксис: gpu_get_texfilter(); Возвращается: Булево Пример: if gpu_get_texfilter() { gpu_set_texfilter(false); } else { gpu_set_texfilter(true); } Приведенный выше код проверяет, включена или выключена фильтрация текстур, и переключает ее соответствующим образом.
