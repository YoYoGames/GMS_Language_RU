gpu_set_blendenable Эта функция может использоваться для включения и выключения альфа-смешения. В принципе, если установить значение false, все рисуемые изображения будут рисоваться на 100% непрозрачными, то есть все прозрачные или полупрозрачные области спрайта или фона будут будут видны как сплошной цвет. Хорошей идеей является отключение альфа-смешивания, когда это возможно (особенно при разработке для мобильных устройств), так как это значительно увеличивает скорость отрисовки. Синтаксис: gpu_set_blendenable(enable); Аргумент Описание включить Включение или отключение значения альфа-смешивания(true или false). Возвращается: Н/Д Пример: gpu_set_blendenable(false); draw_sprite(spr_Background, 0, 0, 0, 0); gpu_set_blendenable(true); Приведенный выше код отключает альфа-смешивание для рисования фонового спрайта, а затем снова включает его для продолжения рисования.
