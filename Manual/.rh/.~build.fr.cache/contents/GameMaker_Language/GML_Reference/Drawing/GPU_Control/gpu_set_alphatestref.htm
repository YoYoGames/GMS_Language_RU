gpu_set_alphatestref Эта функция устанавливает опорное значение для альфа-тестирования (если оно было включено с помощью функции gpu_set_alphatestenable()). Это порог "отсечения", при котором пиксели с альфой не будут отрисовываться. В принципе, если у вас это значение установлено в 0, и у вас включена линейная интерполяция (либо в Опциях игры, либо с помощью функции gpu_set_texfilter()), вы все равно можете получить визуальные ошибки на краях перекрывающихся объектов. Это происходит потому, что у вас могут быть почти полностью прозрачные области, которые блокируют то, что нарисовано за ними, но если вы установите это опорное значение, например, 254 (т.е. на единицу ниже максимального), это гарантирует, что будут отрисованы только 100% непрозрачные пиксели. будут нарисованы. Вы также можете отключить интерполяцию текстур, что обычно позволяет использовать значения по умолчанию без проблем и без необходимости использовать эту функцию вообще, однако многое зависит от исходного произведения искусства. ПРИМЕЧАНИЕ: Это расширенная функция рисования, поэтому если вы не уверены, нужна ли она вам или нет, или не знаете точно, что она делает, вам, вероятно, лучше не использовать ее, поскольку она может серьезно испортить то, как рисуется ваша игра. Синтаксис: gpu_set_alphatestref(val); Аргумент Описание val Устанавливает опорное значение для альфа-тестирования от 0 до 255 (значение по умолчанию 0). Возвращается: Н/Д Пример: if !gpu_get_alphatestenable() { gpu_set_alphatestenable(true); gpu_set_alphatestref(128); } Приведенный выше код проверит, включено ли альфа-тестирование, и если нет, то включит альфа-тестирование и установит порог теста на 128 (будут отрисованы только пиксели с альфой более 0,5).
