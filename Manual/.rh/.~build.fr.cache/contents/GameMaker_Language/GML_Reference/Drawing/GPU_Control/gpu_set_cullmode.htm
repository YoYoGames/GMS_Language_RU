gpu_set_cullmode Эта функция используется для установки режима отбраковки задней стороны. Многоугольник (примитив) имеет переднюю и заднюю стороны. Передней считается сторона, на которой вершины определены в порядке против часовой стрелки. Обычно рисуются обе стороны рисуются, но если вы создаете замкнутую форму (например, пирамиду), то это пустая трата вычислительной мощности, поскольку обратная сторона треугольника никогда не будет видна. Именно в таких случаях можно включить отбраковку обратной стороны, так как это экономит примерно половину времени рисования. время рисования, хотя это означает, что вы должны определить полигоны правильным образом, чтобы избежать проблем. Используемое значение режима выбраковки должно быть одной из следующих констант (значение по умолчанию - cull_noculling): Постоянная Описание выбраковка Выбраковка не будет проводиться cull_clockwise Все треугольники по часовой стрелке будут удалены cull_counterclockwise Все треугольники против часовой стрелки будут удалены Синтаксис: gpu_set_cullmode(cullmode); Аргумент Описание cullmode Используемый режим выбраковки (см. список выше) Возвращается: Н/Д Пример: if gpu_get_cullmode() != cull_clockwise { gpu_set_cullmode(cull_clockwise); } Приведенный выше код получает текущий режим cull и, если он не cull_clockwise, то устанавливается на эту константу.
