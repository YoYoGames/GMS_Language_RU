gpu_get_alphatestenable С помощью этой функции вы можете проверить, включено ли альфа-тестирование (возвращается true) или нет (возвращается false). Для получения дополнительной информации об альфа-тестировании см. функцию gpu_set_alphatestref(). Синтаксис: gpu_get_alphatestenable(); Возвращается: Булево Пример: if !gpu_get_alphatestenable() { gpu_set_alphatestenable(true); gpu_set_alphatestref(128); } Приведенный выше код проверит, включено ли альфа-тестирование, и если нет, то включит альфа-тестирование и установит порог теста на 128 (будут отрисованы только пиксели с альфой более 0,5).
