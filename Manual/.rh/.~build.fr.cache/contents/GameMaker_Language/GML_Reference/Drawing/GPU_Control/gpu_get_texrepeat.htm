gpu_get_texrepeat С помощью этой функции вы можете проверить, включено ли повторение текстуры (возвращается true) или нет (возвращается false). Для получения дополнительной информации о повторении текстур см. функцию gpu_set_texrepeat(). Синтаксис: gpu_get_texrepeat(); Возвращается: Булево Пример: if gpu_get_texrepeat() { gpu_set_texrepeat(false); } else { gpu_set_texrepeat(true); } Приведенный выше код проверяет, включено или выключено повторение текстуры, и переключает его соответствующим образом.
