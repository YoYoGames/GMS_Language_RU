gpu_get_ztestenable Эта функция может быть использована для получения информации о том, включено ли z-тестирование (функция возвращает true) или нет (функция возвращает false). По умолчанию z-тестирование отключено, поэтому функция возвращает false. Синтаксис: gpu_get_ztestenable(); Возвращается: Булево Пример: if gpu_get_ztestenable() == false { gpu_set_ztestenable(true); } Приведенный выше код проверяет, включено или нет z-тестирование, и если оно отключено, то снова включается.
