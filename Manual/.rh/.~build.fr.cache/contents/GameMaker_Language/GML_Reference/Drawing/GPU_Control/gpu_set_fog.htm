gpu_set_fog Эта функция может использоваться для включения или отключения рисования тумана. Туман можно использовать в 3D-играх, чтобы объекты вдали выглядели размытыми или даже исчезали, что помогает создать атмосферу, а также скрыть тот факт, что вы не рисуете объекты которые находятся далеко. Вы устанавливаете, включен ли туман(true) или отключен(false), цвет, который туман должен использовать для смешивания, а также начальное и конечное расстояния рисования. Обратите внимание, что начало тумана указывает радиус относительно а конец тумана указывает, при каком радиусе (также относительном) экземпляры будут полностью невидимы. Функция может принимать четыре отдельных аргумента (приведенных выше) или массив со следующей структурой (в примере кода ниже показан этот метод): [0] = включенный тумблер (булево значение, либо истина, либо ложь), по умолчанию false [1] = цвет (реальный), по умолчанию c_black [2] = начальное расстояние (реальное), по умолчанию 0 [3] = конечное расстояние (реальное), по умолчанию 1 Синтаксис: gpu_set_fog(enable, color, start, end); Аргумент Описание включить Включить или отключить туман цвет Цвет тумана запустить Расстояние, с которого начинается наложение тумана конец Расстояние до конца тумана Возвращается: Н/Д Пример: var fog_a = gpu_get_fog(); fog_a[1] = c_red; gpu_set_fog(fog_a); Приведенный выше код получает текущие настройки тумана, затем устанавливает элемент color массива в c_red, после чего снова устанавливает туман, используя измененный массив.
