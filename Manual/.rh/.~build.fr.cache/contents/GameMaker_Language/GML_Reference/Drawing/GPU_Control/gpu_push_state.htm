gpu_push_state С помощью этой функции вы можете поместить текущее состояние GPU в стек, чтобы использовать его позже. Обычно вы используете эту функцию, если хотите "сохранить" текущее состояние GPU (такие вещи, как режим наложения, запись альфа-канала, свертывание и т.д. будут перемещены в стек), затем нарисовать что-то другое и сбросить стек GPU. стек), затем нарисовать что-то с другими настройками, а затем сбросить стек GPU к тому, что было раньше (вызовом gpu_pop_state()). Синтаксис: gpu_push_state(); Возвращается: Н/Д Пример: gpu_push_state(); gpu_set_blendmode(bm_add); gpu_set_blendenable(false); gpu_set_cullmode(true); with (obj_Effect_Parent) { draw_self(); } gpu_pop_state(); Приведенный выше код "сохранит" текущее состояние GPU в стеке, затем изменит определенные настройки GPU и отрисует группу экземпляров, после чего сбросит состояние GPU на прежнее.
