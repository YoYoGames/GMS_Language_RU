draw_flush С помощью этой функции вы можете очистить весь конвейер рисования. Эта функция предназначена только для отладки, и в целом ее не следует использовать, если на это не укажет сотрудник службы поддержки YoYo Games. так как беспорядочное использование приведет к серьезным проблемам с производительностью вашей игры. Синтаксис: draw_flush(); Возвращается: Н/Д Пример: draw_flush(); Приведенный выше код промывает конвейер рисования.
