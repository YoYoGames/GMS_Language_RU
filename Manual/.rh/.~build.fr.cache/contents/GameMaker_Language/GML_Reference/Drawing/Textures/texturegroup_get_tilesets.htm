texturegroup_get_tilesets С помощью этой функции вы можете получить индекс тайлсета каждого из тайлсетов, назначенных текстурным страницам в данной группе текстур. Вы вводите строку ID текстурной группы (как определено в редакторе текстурных групп), и функция возвращает одномерный массив, в котором каждый элемент содержит индекс тайлсета. массив, каждая запись которого содержит индекс тайлсета для ресурса тайлсета. Если функция не работает - т.е. указана неверная группа, или группа не имеет текстуры, назначенной ей - то массив будет пуст (длина 0). Синтаксис: texturegroup_get_tilesets(tex_id); Аргумент Описание tex Имя группы текстур для проверки (строка). Возвращается: одномерный массив Пример: var _tex_array = texturegroup_get_tilesets("MainMenu"); for (var i = 0; i < array_length(_tex_array); ++i;) { show_debug_message("Tileset " + string(i) + " Index:" + string(tex_array[i])); } Приведенный выше код получит индексы тайлсета для группы текстур "MainMenu", а затем отобразит эти идентификаторы в окне консольного вывода.
