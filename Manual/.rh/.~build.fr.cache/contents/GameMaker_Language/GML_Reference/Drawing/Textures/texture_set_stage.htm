texture_set_stage Эта функция задаст заданному "слоту" сцены текстуру, которая будет использоваться. Количество доступных "слотов" сцены зависит от платформы, под которую вы компилируете, максимальное количество - 8 для Windows, Mac и Linux, но на более дешевых устройствах Android (например. Стоит также отметить, что первый "слот" сцены всегда автоматически используется GameMaker Studio 2. ПРИМЕЧАНИЕ: Эта функция ничего не делает вне контекста запущенного шейдера! См. Шейдеры для получения дополнительной информации. Синтаксис: texture_set_stage(stage, tex); Аргумент Описание сцена Текстурный "слот" для использования. tex Текстура для использования. Возвращается: Н/Д Пример: glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler") spr = sprite_get_texture(sprite_index, 0); shader_set (GlassShader); texture_set_stage(glassshader_bgsampler, spr); shader_reset(); Приведенный выше код получит хэндл для сэмплера в шейдере с индексом "GlassShader", а затем установит константу этого шейдера на заданную текстуру спрайта.
