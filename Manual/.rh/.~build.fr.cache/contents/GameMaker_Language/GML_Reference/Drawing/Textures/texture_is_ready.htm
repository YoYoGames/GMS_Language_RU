texture_is_ready Эта функция может быть использована для проверки, была ли распакована и готова к использованию определенная текстурная страница, или группа текстурных страниц была распакована и готова к использованию. Вы передаете уникальный идентификатор текстурной страницы (возвращаемый функциями texturegroup_* ) или строку идентификатора текстурной группы (как определено в редакторе текстурных групп), и функция возвращает true, если они были распакованы, или false в противном случае. Синтаксис: texture_is_ready(tex_id); Аргумент Описание tex_id Идентификатор страницы текстуры (целое число) или идентификатор группы текстур (строка) Возвращается: Булево Пример: var _tex_array = texturegroup_get_textures("MainMenu"); for (var i = 0; i < array_length(_tex_array); ++i;) { if !texture_is_ready(_tex_array[i]) { texture_prefetch(_tex_array[i]); } } Приведенный выше код получит идентификаторы текстурных страниц для группы текстур "MainMenu", затем проверит, распакованы ли они, и если нет, то они будут префетишированы в VRAM.
