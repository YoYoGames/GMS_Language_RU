текстурная_выборка Эта функция может использоваться для "предварительной выборки" текстурной страницы или группы текстурных страниц, то есть для загрузки их в VRAM при необходимости. Вы предоставляете уникальный идентификатор текстурной страницы (возвращаемый функциями texturegroup_* ) для предварительной выборки одной страницы, или вы можете предоставить имя группы текстур (как определено в функции Texture Group). или имя группы текстур (как определено в редакторе групп текстур) для предварительной выборки всех страниц текстур в группе. Синтаксис: texture_prefetch(tex_id); Аргумент Описание tex_id Идентификатор страницы текстуры (целое число) или идентификатор группы текстур (строка) Возвращается: Н/Д Пример: var _tex_array = texturegroup_get_textures("MainMenu"); for (var i = 0; i < array_length(_tex_array); ++i;) { texture_prefetch(_tex_array[i]); } Приведенный выше код будет предварительно получать все страницы текстур под группой текстур "MainMenu".
