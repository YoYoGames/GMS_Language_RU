draw_texture_flush С помощью этой функции вы можете удалить все текстуры из видеопамяти, после чего они будут перезагружены при первом использовании. Это единственный эффективный способ управления видеопамятью, когда у вас есть несколько текстурных страниц для игры, и вы должны промывать текстурную память между уровнями игры и организовывать графику с помощью функции группы текстур, чтобы обеспечить использование минимального количества текстур. Стоит отметить, что по умолчанию на всех объектах страницы текстур будут загружаться только по мере необходимости, и вы можете использовать различные функции предварительной выборки и промывки для спрайтов (находятся здесь) для управления вещами, а также эту функцию. Синтаксис: draw_texture_flush(); Возвращается: Н/Д Пример: draw_texture_flush(); Приведенный выше код очищает видеопамять от текстурных данных и, вероятно, будет помещен в событие create первого экземпляра объекта, размещенного в комнате.
