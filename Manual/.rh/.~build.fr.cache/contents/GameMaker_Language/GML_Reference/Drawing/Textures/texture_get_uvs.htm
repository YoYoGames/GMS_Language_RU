texture_get_uvs Эта функция возвращает одномерный массив с 4 элементами, представляющими УФ-координаты для изображения на странице текстуры, заполняя массив следующими значениями: [0] = левый [1] = верх [2] = право [3] = низ [4] = количество пикселей, которые компилятор активов обрезал с левой стороны спрайта (только для активов спрайта) [5] = количество пикселей, которые компилятор активов обрезал с верхней стороны спрайта (только для активов спрайта) [6] = нормализованный процент пиксельных данных от ширины исходного спрайта, который был сохранен на странице текстуры (только для спрайтов). [7] = нормализованный процент пиксельных данных от высоты исходного спрайта, который был сохранен на странице текстуры (только для спрайтов). Затем это значение можно использовать в других функциях рисования, в частности, в общих функциях 3D и некоторых функциях 2D примитивов, а также в функциях шейдера. Если вам нужен UVS для спрайта, вы можете использовать функции sprite_get_uvs() и для шрифта, font_get_uvs(). Синтаксис: texture_get_uvs(texid) Аргумент Описание texid Актив изображения для получения текстуры Возвращается: Массив 1D (4 - 8 элементов) Пример: var tex = texture_get_uvs(surf_back); tex_left = tex[0]; tex_top = tex[1]; tex_right = tex[2]; tex_left = tex[3]; Приведенный выше код будет хранить UV-координаты для заданной текстуры поверхности в локальном массиве, а затем присваивать значения переменным экземпляра.
