текстурная_промывка Эта функция может использоваться для "промывки" текстурной страницы или группы текстурных страниц, то есть для удаления их из VRAM, когда они больше не нужны. Вы передаете уникальный идентификатор текстурной страницы (возвращаемый функциями texturegroup_* ), чтобы прошить или имя группы текстур (как определено в редакторе групп текстур), чтобы прошить все страницы текстур в группе. Синтаксис: texture_flush(tex_id); Аргумент Описание tex_id Идентификатор страницы текстуры (целое число) или идентификатор группы текстур (строка) Возвращается: Н/Д Пример: var _tex_array = texturegroup_get_textures("MainMenu"); for (var i = 0; i < array_length(_tex_array); ++i;) { texture_flush(_tex_array[i]); } Приведенный выше код прошьет все страницы текстур под группой текстур "MainMenu".
