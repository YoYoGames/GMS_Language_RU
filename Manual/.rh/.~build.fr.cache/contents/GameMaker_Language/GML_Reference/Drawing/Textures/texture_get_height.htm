texture_get_height Возвращает высоту текстуры с заданным id, который всегда является значением в диапазоне 0 - 1. Это значение можно использовать при отображении текстур на модели или примитивы. Синтаксис: texture_get_height(tex); Аргумент Описание tex Указатель актива страницы текстуры для использования Возвращается: Реально Пример: tex_h = texture_get_height(surface_get_texture(global.Surf)); Приведенный выше код получит высоту текстуры, взятой из ранее созданной поверхности.
