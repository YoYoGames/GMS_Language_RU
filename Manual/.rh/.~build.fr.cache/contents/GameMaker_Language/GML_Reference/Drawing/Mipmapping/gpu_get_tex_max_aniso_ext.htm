gpu_get_tex_max_aniso_ext С помощью этой функции можно получить текущий максимальный уровень анизотропии при использовании режима фильтра tf_anisotropic (см. gpu_get_tex_mip_filter() более подробную информацию) в шейдерном сэмплере. Вы вводите значение индекса шейдерного сэмплера (возвращаемое функцией shader_get_sampler_index(), и функция вернет значение в диапазоне от 1 до 16. Синтаксис: gpu_get_tex_max_aniso_ext(sampler_index); Аргумент Описание sampler_index Индекс шейдерного сэмплера для получения Возвращается: Величина (по умолчанию: 16) Пример: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set (shd_Glass); if gpu_get_tex_max_aniso_ext(_sampleIndex) != 8 { gpu_set_tex_max_aniso_ext(_sampleIndex, 8); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Приведенный выше код устанавливает максимальный уровень анизотропии на 8 для данного шейдерного текстурного сэмплера, если он еще не установлен на 8.
