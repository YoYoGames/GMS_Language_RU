gpu_set_tex_min_mip_ext С помощью этой функции вы можете установить минимальный уровень mipmap, который в настоящее время используется для данного шейдерного сэмплера. Вы передаете значение индекса для шейдерного сэмплера (возвращаемое функцией shader_get_sampler_index(), и затем задаете значение, где 0 - для полного разрешения, 1 - для первого мипмапа, 2 - для второго и т.д.... Синтаксис: gpu_set_tex_min_mip_ext(sampler_index, minmip); Аргумент Описание sampler_index Индекс шейдерного сэмплера для получения minmip Минимальный уровень мипмапа для использования Возвращается: Н/Д Пример: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set (shd_Glass); if gpu_get_tex_min_mip_ext(_sampleIndex) != 0 { gpu_set_tex_min_mip_ext(_sampleIndex, 0); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Приведенный выше код устанавливает минимальный уровень mipmap в 0 для данного шейдерного текстурного сэмплера, если он еще не установлен в 0.
