gpu_get_tex_max_aniso С помощью этой функции вы можете получить текущий максимальный уровень анизотропии при использовании режима tf_anisotropic filter (см. gpu_get_tex_mip_filter() для дополнительной информации). Возвращаемое значение будет в диапазоне от 1 до 16. Синтаксис: gpu_get_tex_max_aniso(); Возвращается: Величина (по умолчанию: 16) Пример: if gpu_get_tex_max_aniso() != 8 { gpu_set_tex_max_aniso(8); } Приведенный выше код проверит текущий максимальный уровень анизотропной фильтрации, и если он не равен 8, то будет установлен на 8.
