gpu_set_tex_mip_enable_ext С помощью этой функции вы можете установить, выключен ли mipmapping, включен ли он для всего или включен только для групп текстур, выбранных в менеджере групп текстур на шейдерном сэмплере. Вы вводите значение индекса шейдерного сэмплера (возвращаемое функцией shader_get_sampler_index(), а затем одну из перечисленных ниже констант. Постоянная Описание mip_off Mipmapping отключен. mip_on Mipmapping для всех текстур включен. mip_markedonly Mipmapping включен для текстур, у которых он включен в опциях группы текстур (по умолчанию). Синтаксис: gpu_set_tex_mip_enable_ext(sampler_index, setting); Аргумент Описание sampler_index Индекс шейдерного сэмплера для получения настройка Настройка mipmap (константа, по умолчанию: mip_markedonly). Возвращается: Н/Д Пример: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set (shd_Glass); if gpu_get_tex_mip_enable_ext(_sampleIndex) != mip_on { gpu_set_tex_mip_enable_ext(_sampleIndex, mip_on); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Приведенный выше код включает mipmapping для данного шейдерного текстурного сэмплера, если он еще не включен.
