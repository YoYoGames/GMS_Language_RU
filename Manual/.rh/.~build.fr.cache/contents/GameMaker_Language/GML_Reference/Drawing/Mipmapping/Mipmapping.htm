Mipmapping В этом разделе содержатся все функции, связанные с использованием функций mipmapping. Перед использованием этих функций вы должны включить mipmapping для необходимых текстурных страниц в Texture Group Editor и/или включить mipmapping для тех спрайтов, которые были настроены на использование только одной текстурной страницы в General Game Options. На изображении ниже показана разница, которую может внести mipmapping при рендеринге вашего проекта: После того как вы включили мипмаппинг для проекта или страницы текстуры, вы можете использовать следующие функции для получения и установки значений, которые изменят внешний вид проекта при запуске.
