gpu_get_tex_min_mip_ext С помощью этой функции вы можете получить минимальный уровень mipmap, который в настоящее время используется для данного шейдерного сэмплера. Вы предоставляете значение индекса для шейдерного сэмплера (как возвращается функцией shader_get_sampler_index(), и функция вернет значение 0 для полного разрешения, 1 для первого мипмапа, 2 для второго и т.д.... Синтаксис: gpu_get_tex_min_mip_ext(sampler_index); Аргумент Описание sampler_index Индекс шейдерного сэмплера для получения Возвращается: Реальный (по умолчанию: 0) Пример: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set (shd_Glass); if gpu_get_tex_min_mip_ext(_sampleIndex) != 0 { gpu_set_tex_min_mip_ext(_sampleIndex, 0); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Приведенный выше код устанавливает минимальный уровень mipmap в 0 для данного шейдерного текстурного сэмплера, если он еще не установлен в 0.
