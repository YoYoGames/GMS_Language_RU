gpu_set_tex_mip_filter С помощью этой функции вы можете установить текущий режим фильтрации mip на один из трех поддерживаемых типов. Вы задаете константу, которая ссылается на требуемый режим фильтрации mip, где вы можете выбрать между точечной фильтрацией (настройка по умолчанию, означающая нет фильтрации), линейной фильтрацией (она также известна как трилинейная фильтрация) или аниостропической фильтрацией (см. таблицу констант ниже). Постоянная Описание tf_point Это отключит фильтрацию между уровнями mipmap, что может привести к заметным переходам текстуры, но дает наилучшую производительность. tf_linear Это позволит включить линейную фильтрацию между уровнями mipmap (это также известно как трилинейная фильтрация), которая сглаживает переходы текстур, но это незначительно снижает производительность. tf_anisotropic Это позволит включить анизотропную фильтрацию между уровнями mipmap, что значительно улучшает качество перехода текстуры и может уменьшить размытие, видимое при других режимах фильтрации, но это сильно сказывается на производительности. Синтаксис: gpu_get_tex_mip_filter(filter); Аргумент Описание фильтр Используемый режим фильтрации mip (константа, по умолчанию: tf_point). Возвращается: Н/Д Пример: if keyboard_check(ck_enter) { switch(gpu_get_tex_mip_filter()) { case tf_point: gpu_set_tex_mip_filter(tf_linear); break; case tf_linear: gpu_set_tex_mip_filter(tf_anisotropic); break; case tf_anisotropic: gpu_set_tex_mip_filter(tf_point); break; } } Приведенный выше код проверяет клавиатуру, и если указанная клавиша нажата, то он получит текущий mip-фильтр и переключит значение на следующий, циклически перебирая различные режимы.
