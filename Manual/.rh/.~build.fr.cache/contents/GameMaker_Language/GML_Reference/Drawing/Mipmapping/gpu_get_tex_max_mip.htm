gpu_get_tex_max_mip С помощью этой функции вы можете получить установленный на данный момент максимальный уровень мипмапа, который должен быть использован, где значение 0 - самое высокое разрешение, 1 - использование первого мипмапа, 2 - второго и т.д.. Обратите внимание, что это может быть весьма полезно для предотвращения артефактов растекания При рендеринге текстур, например, установив границу страницы текстуры на 8px и затем установив максимальный уровень mipmap на 3, вы не получите проблем с растеканием на больших расстояниях рендеринга. Синтаксис: gpu_get_tex_max_mip(); Возвращается: Величина (по умолчанию: 16) Пример: if gpu_get_tex_max_mip() != 4 { gpu_set_tex_max_mip(4); } Приведенный выше код проверит текущий максимальный уровень mipmap, и если он не равен 4, то будет установлен на 4.
