gpu_set_tex_mip_bias_ext С помощью этой функции вы можете установить значение смещения mipmap для данного шейдерного сэмплера. Вы предоставляете значение индекса для шейдерного сэмплера (как возвращается функцией shader_get_sampler_index(), и затем значение смещения, где 0 - без смещения, 1 - для первого mipmap, 2 - для второго mipmap и т.д.. Это контролирует скорость смены мип-карт и, как правило, делает шейдерные текстуры более размытыми, чем выше значение и чем чем больше "расстояние" до объекта. Обратите внимание, что эта функция может принимать и отрицательные значения, в этом случае шейдерные текстуры будут более четкими на большем расстоянии, чем меньше значение. Синтаксис: gpu_set_tex_mip_bias_ext(sampler_index, bias); Аргумент Описание sampler_index Индекс шейдерного сэмплера для получения смещение Значение смещения mipmap для использования (по умолчанию: 0). Возвращается: Н/Д Пример: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set (shd_Glass); if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0 { gpu_set_tex_mip_bias_ext(_sampleIndex, 0); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Приведенный выше код устанавливает смещение фильтра mip в 0 для данного шейдерного текстурного сэмплера, если оно еще не установлено в 0.
