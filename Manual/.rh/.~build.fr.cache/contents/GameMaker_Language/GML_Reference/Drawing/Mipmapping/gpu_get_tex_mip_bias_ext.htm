gpu_get_tex_mip_bias_ext С помощью этой функции вы можете получить значение смещения mipmap для заданного шейдерного сэмплера. Вы передаете значение индекса для шейдерного сэмплера (как возвращается функцией shader_get_sampler_index(), и функция вернет значение 0.0 для отсутствия смещения, или большее значение, где 1 соответствует первому мипмапу, 2 - второму мипмапу и так далее. Это регулирует скорость смены карт mip и, как правило, делает текстуры шейдера более размытыми. чем выше значение, тем больше "расстояние", на котором происходит просмотр. Обратите внимание, что этот параметр может возвращать и отрицательные значения, в этом случае шейдерные текстуры будут более четкими на большем расстоянии, чем меньше значение. Синтаксис: gpu_get_tex_mip_bias_ext(sampler_index); Аргумент Описание sampler_index Индекс шейдерного сэмплера для получения Возвращается: Реальный (по умолчанию: 0) Пример: var _sampleIndex = shader_get_sampler_index(shd_Glass, "s_Background"); var _spriteTex = sprite_get_texture(sprite_index, 0); shader_set (shd_Glass); if gpu_get_tex_mip_bias_ext(_sampleIndex) != 0 { gpu_set_tex_mip_bias_ext(_sampleIndex, 0); } texture_set_stage(_sampleIndex , _spriteTex); shader_reset(); Приведенный выше код устанавливает смещение фильтра mip в 0 для данного шейдерного текстурного сэмплера, если оно еще не установлено в 0.
