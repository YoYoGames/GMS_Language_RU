gpu_set_tex_max_aniso С помощью этой функции вы можете установить текущий максимальный уровень анизотропии при использовании режима tf_anisotropic filter (см. gpu_get_tex_mip_filter() для дополнительной информации). Вводимое значение должно находиться в диапазоне от 1 до 16. Синтаксис: gpu_set_tex_max_aniso(maxaniso); Аргумент Описание maxaniso Максимальный уровень для анизотропной фильтрации (по умолчанию: 16) Возвращается: Н/Д Пример: if gpu_get_tex_max_aniso() != 8 { gpu_set_tex_max_aniso(8); } Приведенный выше код проверит текущий максимальный уровень анизотропной фильтрации, и если он не равен 8, то будет установлен на 8.
