gpu_get_tex_mip_enable С помощью этой функции можно узнать, выключен ли mipmapping, включен ли он для всего или включен только для групп текстур, выбранных в менеджере групп текстур. Функция возвращает одну из перечисленных ниже констант, по умолчанию mip_markedonly. Синтаксис: gpu_get_tex_mip_enable(); Возвращается: Постоянный: Постоянная Описание mip_off Mipmapping отключен. mip_on Mipmapping для всех текстур включен. mip_markedonly Mipmapping включен для текстур, у которых он включен в опциях группы текстур (по умолчанию). Пример: if gpu_get_tex_mip_enable != mip_on { gpu_set_tex_mip_enable(mip_on); } Приведенный выше код проверит, включен ли mipmapping, и если нет, то включит его.
