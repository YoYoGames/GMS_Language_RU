поверхность_копирования_части Эта функция просто берет изображение с одной поверхности и копирует его на другую поверхность в указанной локальной позиции внутри этой поверхности (где позиция (0,0) - левый верхний угол целевой поверхности). Вы можете указать локальную позицию x и y для копирования, а также ширину и высоту секции. Обратите внимание, что это координаты, основанные на размере поверхности, а не на положении, в котором поверхность рисуется в комнате. Если поверхность назначения уже содержит информацию она будет перезаписана копией, и функция никак не изменит исходную поверхность. ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней. Синтаксис: surface_copy_part(destination, x, y, source, xs, ys, ws, hs); Аргумент Описание направление Идентификатор поверхности, на которую нужно скопировать другую поверхность. x Позиция x для копирования. y Позиция y для копирования. источник Идентификатор поверхности, которую нужно скопировать. xs Позиция x в исходной поверхности для копирования. ys Позиция y в исходной поверхности для копирования. ws Ширина области в исходной поверхности для копирования. hs Высота области в исходной поверхности, с которой нужно копировать. Возвращается: Н/Д Пример: if view_current == 0 { surface_copy_part(surf, 0, 0, temp_surf, 0, 0, 0, view_xview[1] - mouse_x, view_yview[1] - mouse_y); } else { draw_surface(surf, 0, 0); } Приведенный выше код проверяет текущий вид, который рисуется, и если это view[0], то копирует поверхность, указанную в переменной "temp_surf", на поверхность, указанную в переменной "surf". Скопированная область соответствует прямоугольнику, образованному относительным положением мыши в пределах поверхности, как она была бы нарисована в представлении[1]. Если текущий вид не является видом[0], то поверхность "surf" рисуется на экране. Назад: Поверхности Далее: surface_depth_disable © Copyright YoYo Games Ltd. 2021 Все права защищены
