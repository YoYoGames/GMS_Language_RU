поверхность_сохранить_часть Эта функция сохранит часть поверхности на диск, используя заданное имя файла. Поверхность должна быть сохранена как файл формата *.png, а положение (x,y) должно быть задано как локальные координаты поверхности, учитывая, что левый верхний левый верхний угол поверхности всегда (0,0). Синтаксис: surface_save_part(surface_id, fname, x, y, width, height); Аргумент Описание идентификатор поверхности Идентификатор поверхности, которую нужно установить в качестве цели рисования. fname Имя сохраненного файла изображения. x Начальное положение x в пределах поверхности. y Начальная позиция y в пределах поверхности. ширина Ширина детали для сохранения. высота Высота детали для сохранения. Возвращается: Н/Д Пример: if keyboard_check_pressed(ord("S") { surface_save_part(surf, "test.png", 0, 0, 100, 100); } Приведенный выше код проверит, нажимает ли пользователь клавишу "S" на клавиатуре, и если нажимает, то сохранит часть поверхности, проиндексированной в переменной "surf", на диск.
