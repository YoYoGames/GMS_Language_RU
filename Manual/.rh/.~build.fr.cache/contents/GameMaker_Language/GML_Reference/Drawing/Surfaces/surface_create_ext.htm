поверхность_создать_экст Эта функция позволяет прикрепить поверхность к элементу холста, который уже существует на вашей веб-странице, что означает, что вы можете эффективно разделить части вашей игры для рисования в различных местах страницы. Для этого вы должны правильно определить элемент canvas в *.html-странице вашей игры, используя правильные размеры и имена, соответствующие поверхностям, которые вы хотите создать. Таким образом, у вас есть "главный" холст, а затем вторичные поверхностные элементы холста, которые будут назначены с помощью этой функции правильным поверхностям. Следующее изображение является примером того, как будет настроена страница с тремя элементами холста: Макет страницы может быть непростым, особенно если вы хотите, чтобы все элементы выстраивались правильно, но как только вы можете добавить этот html-файл в качестве файла страницы по умолчанию для игры, используя вкладку HTML5 в опциях игры. Следующее, что вы должны сделать, это настроить комнату и виды, поскольку каждая поверхность должна быть связана с определенным видом для "захвата" игровых изображений (см. переменную вида view_surface_id[0...7]). Функция вернет индекс поверхности, который следует сохранить в переменной для последующих вызовов функции. Когда поверхность создается впервые, она может содержать "шум", так как по сути это просто область памяти, отведенная для этой цели (и эта память все еще может содержать информацию), поэтому вы можете захотеть очистить поверхность перед использованием с помощью функции типа draw_clear_alpha(). ПРИМЕЧАНИЕ: Эта функция доступна только для использования с модулем HTML5. Синтаксис: surface_create_ext(name, w, h); Аргумент Описание имя Имя элемента холста, к которому нужно привязать поверхность. w Ширина создаваемой поверхности. h Высота создаваемой поверхности. Возвращается: Реально Пример: s1 = surface_create_ext("surface1", 192, 550); s2 = surface_create_ext("surface2", 608, 186); view_surface_id[1] = s1; view_surface_id[2] = s2; Приведенный выше код создает две поверхности разных размеров, назначая каждую из них отдельному элементу холста, а затем эти поверхности назначаются двум видам, чтобы была захвачена нужная часть комнаты.
