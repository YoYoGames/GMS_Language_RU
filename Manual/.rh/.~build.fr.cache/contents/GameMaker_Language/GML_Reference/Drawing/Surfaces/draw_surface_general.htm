рисовать_поверхность_общая Эта функция объединяет функцию draw_surface_ext() с функцией draw_surface_part()и добавляет некоторые дополнительные параметры смешивания, чтобы каждый угол чтобы каждый угол конечной части поверхности можно было смешивать с отдельным цветом. ПРИМЕЧАНИЕ: Градиентное смешивание недоступно для цели HTML5, если не включен WebGL. ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней. Синтаксис: draw_surface_general(id, left, top, w, h, x, y, xscale, yscale, rot, c1, c2, c3, c4, alpha); Аргумент Описание id Уникальное значение ID поверхности для рисования. слева Левая позиция на поверхности рисуемой детали. топ Положение вершины на поверхности рисуемой детали. w Ширина рисуемой части, слева направо. h Высота рисуемой детали сверху. x Позиция x в месте рисования поверхности. y Положение по оси y места рисования поверхности. xscale Горизонтальное масштабирование для рисования поверхности. yscale Вертикальный масштаб для рисования поверхности. гниение Поворот или угол, под которым будет рисоваться поверхность. c1 Цвет левого верхнего угла поверхности. c2 Цвет правого верхнего угла поверхности. c3 Цвет правого нижнего угла поверхности. c4 Цвет левого нижнего угла поверхности. альфа Альфа-прозрачность для рисования поверхности. Возвращается: Н/Д Пример: draw_surface_general(surf, 8, 8, 32, 32, x, y, 2, 0.5, 180, c_white, c_white, c_black, c_black, 1); Это нарисует на поверхности область размером 32x32 пикселя из 8x8 пикселей. Она будет растянута в два раза по ширине, но в два раза по высоте. Она будет непрозрачной и перевернутой. Верхняя область поверхности будет смешанной белой и, следовательно, нормальной, но нижняя область будет черной, то есть поверхность будет переходить от нормали к силуэту вниз в виде плавного градиента.
