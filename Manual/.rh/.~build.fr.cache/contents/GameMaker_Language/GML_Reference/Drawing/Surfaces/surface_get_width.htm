surface_get_width Эта функция просто возвращает ширину, в пикселях, индексированной поверхности. Следует отметить, что если вы вызовете эту функцию для проверки поверхности приложения_поверхность сразу после изменения ее размера с помощью функции surface_resize() она не вернет новое значение, так как для полной обработки изменения требуется шаг или два. После ожидания одного шага он должен корректно вернуть новый размер. ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней. Синтаксис: surface_get_width(surface_id); Аргумент Описание идентификатор поверхности Идентификатор поверхности для получения ширины. Возвращается: Реально Пример: sw = surface_get_width(surf); Приведенный выше код будет хранить ширину поверхности, проиндексированной в переменной "surf", в переменной "sw".
