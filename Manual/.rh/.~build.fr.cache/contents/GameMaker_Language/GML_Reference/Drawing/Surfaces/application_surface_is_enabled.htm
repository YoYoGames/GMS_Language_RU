приложение_поверхность_является_включенным Эта функция возвращает true, если для рисования используется поверхность приложения, или false, если используется буфер экрана. Синтаксис: application_surface_is_enabled(); Возвращается: Булево Пример: if keyboard_check_pressed(vk_space) { if application_surface_is_enabled() { application_surface_enable(false); } else { application_surface_enable(true); } } Приведенный выше код проверяет нажатие клавиши и включает или выключает поверхность приложения в зависимости от ее состояния (как в меню опций).
