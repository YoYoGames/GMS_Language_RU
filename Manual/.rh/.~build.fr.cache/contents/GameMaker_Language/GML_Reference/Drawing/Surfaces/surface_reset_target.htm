поверхность_сброса_цели С помощью этой функции вы сбрасываете все дальнейшее рисование с целевой поверхности обратно на экран. Обратите внимание, что для начала рисования на поверхности вы должны сначала вызвать функцию surface_set_target() и а затем эту функцию для каждой заданной поверхности, иначе на экране ничего не будет нарисовано, так как все дальнейшее рисование (даже в других экземплярах) будет происходить на поверхности. Вы также должны понимать, что ничего не будет видно, если сама поверхность не будет нарисована на экране в событии draw экземпляра. ПРИМЕЧАНИЕ: Если вы ранее не устанавливали цель рендеринга с помощью функции surface_set_target()то использование этой функции беззвучно (т.е. без без сообщений об ошибках) завершит дальнейшее выполнение кода для этого события. Синтаксис: surface_reset_target(); Возвращается: Н/Д Пример: if view_current = 0 { surface_set_target(surf); with (obj_Effect) { draw_self(); } surface_reset_target(); } else { draw_surface(surf, 0, 0); } Приведенный выше код проверяет, какой вид в данный момент рисуется, и если это вид[0], то устанавливает цель рисования на поверхность и рисует все экземпляры объекта "obj_Effect", после чего снова сбрасывает цель рисования.
