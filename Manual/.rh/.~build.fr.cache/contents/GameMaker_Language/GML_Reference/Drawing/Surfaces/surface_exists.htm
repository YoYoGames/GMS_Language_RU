поверхность_существует Эта функция необходима при работе с поверхностями из-за их изменчивой природы. Поверхности всегда хранятся в памяти текстур, что означает, что они могут быть уничтожены в один момент времени (например, при запуске заставки на окнах или при сворачивании на устройстве Android ), поэтому всегда следует проверять существование поверхности, прежде чем что-либо с ней делать (в том числе рисовать ее на экране). В приведенном ниже примере кода показано типичное использование этой команды в событии рисования экземпляра для проверки наличия поверхности и ее повторного создания, если она была удалена (обратите внимание, что поверхность была первоначально создана в событии create объекта). Синтаксис: surface_exists(surface_id); Аргумент Описание идентификатор поверхности Идентификатор поверхности для проверки. Возвращается: Булево Пример: if !surface_exists(surf) { surf = surface_create(1024, 1024); } else { if view_current = 1 { draw_surface(surf,0,0); } } Приведенный выше код проверит, существует ли поверхность, индексированная в переменной "surf", и если нет, то создаст ее заново. Если она существует, то проверяется, какой вид в данный момент рисуется, и если это вид[1], то рисуется поверхность.
