нарисовать_поверхность_части С помощью этой функции можно нарисовать часть любой поверхности в заданном положении в помещении. Как и в случае с нарисовать_поверхность() вы можете указать поверхность, но затем вам нужно указать относительные координаты в пределах поверхности области, которую нужно выбрать для рисования. Это означает, что левая позиция 0 и верхняя позиция 0 будут левым верхним углом поверхности, и все дальнейшие координаты должны быть взяты из этой позиции. ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() прежде чем обращаться к ней напрямую. Синтаксис: draw_surface_part(id, left, top, w, h, x, y); Аргумент Описание id Уникальное значение ID поверхности для рисования. слева Левая позиция на поверхности рисуемой детали. топ Положение вершины на поверхности рисуемой детали. w Ширина рисуемой части, слева направо. h Высота рисуемой детали сверху. x Позиция x в месте рисования поверхности. y Положение по оси y места рисования поверхности. Возвращается: Н/Д Пример: draw_surface_part(surf, 8, 8, 32, 32, x, y); Это нарисует область 32x32 размером 8px на 8px слева вверху от поверхности, индексированной в "surf", в позиции экземпляра (x,y).
