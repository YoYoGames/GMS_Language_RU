surface_set_target_ext Эта функция предназначена для использования с функциямишейдеров и устанавливает MRT (0 - 3) для шейдеров родного уровня (DX9, DX11, OpenGL). ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстуры. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() прежде чем обращаться к ней напрямую. Синтаксис: surface_set_target_ext(index, surface_id); Аргумент Описание индекс Используемый индекс цели рендеринга (от 0 до 3). идентификатор поверхности Идентификатор поверхности, которую необходимо использовать. Возвращается: Булево Пример: surface_set_target_ext(0, surf); Приведенный выше код установит цель рендеринга 0 на идентификатор поверхности, указанный в переменной "surf".
