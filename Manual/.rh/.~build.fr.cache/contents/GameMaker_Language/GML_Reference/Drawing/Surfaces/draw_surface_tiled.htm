нарисовать_поверхность_плиткой Эта функция возьмет поверхность и многократно проложит плитку по всей комнате, начиная с координат, которые вы укажете в функции. ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней. Синтаксис: draw_surface_tiled(id, x, y); Аргумент Описание id Уникальное значение ID поверхности для рисования. x Позиция x в месте рисования поверхности. y Положение по оси y места рисования поверхности. Возвращается: Н/Д Пример: draw_surface_tiled(surf, x, y); Это нарисует поверхность, индексированную в "surf", в собственном положении экземпляра по x и y, а также плитку во всех направлениях в комнате.
