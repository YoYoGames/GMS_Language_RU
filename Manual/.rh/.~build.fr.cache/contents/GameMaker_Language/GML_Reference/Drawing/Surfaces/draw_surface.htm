нарисовать_поверхность С помощью этой функции можно нарисовать поверхность в заданном положении в помещении, при этом левый верхний угол изображения будет нарисован в заданном положении x/y. ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() перед непосредственным обращением к ней. Синтаксис: draw_surface(id, x, y); Аргумент Описание id Уникальное значение ID поверхности для рисования. x Позиция x в месте рисования поверхности. y Положение по оси y места рисования поверхности. Возвращается: Н/Д Пример: var _vx = camera_get_view_x(view_camera[0]); var _vy = camera_get_view_y(view_camera[0]); draw_surface(surf, _vx, _vy); Приведенный выше код рисует поверхность, индексированную в "surf", в той же позиции, что и камера view[0].
