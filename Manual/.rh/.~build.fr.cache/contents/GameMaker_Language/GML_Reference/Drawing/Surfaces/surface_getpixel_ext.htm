surface_getpixel_ext С помощью этой функции можно получить полное abgr 32-битное значение любого пикселя (ранее созданной) поверхности. ПРИМЕЧАНИЕ: Эта функция сильно снижает производительность, поэтому ее следует использовать только в случае крайней необходимости. Синтаксис: surface_getpixel_ext(surface_id, x, y); Аргумент Описание идентификатор поверхности Идентификатор поверхности, которую необходимо использовать. x Координата x пикселя для проверки y Координата y пикселя для проверки Возвращается: Реально Пример: col = surface_getpixel_ext(mouse_x, mouse_y); alpha = (col >> 24) & 255; blue = (col >> 16) & 255; green = (col >> 8) & 255; red = col & 255; Приведенный выше код получит 32-битное значение цвета в позиции мыши, а затем разделит его на составляющие значения, сохранив их в переменных.
