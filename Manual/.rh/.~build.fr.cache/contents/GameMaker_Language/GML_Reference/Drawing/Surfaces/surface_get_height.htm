surface_get_height Эта функция просто возвращает высоту, в пикселях, индексированной поверхности. Следует отметить, что если вы вызовете эту функцию для проверки поверхности приложения_поверхность сразу после изменения ее размера с помощью функции surface_resize() она не вернет новое значение, так как для полной обработки изменения требуется шаг или два. После ожидания одного шага он должен корректно вернуть новый размер. ПРИМЕЧАНИЕ: При работе с поверхностями существует вероятность того, что они могут перестать существовать в любой момент, поскольку хранятся в памяти текстур. Вы должны ВСЕГДА проверять существование поверхности с помощью функции surface_exists() прежде чем обращаться к ней напрямую. Синтаксис: surface_get_height(surface_id); Аргумент Описание идентификатор поверхности Идентификатор поверхности для получения высоты. Возвращается: Реально Пример: sh = surface_get_height(surf); Приведенный выше код будет хранить высоту поверхности, проиндексированной в переменной "surf", в переменной "sh".
