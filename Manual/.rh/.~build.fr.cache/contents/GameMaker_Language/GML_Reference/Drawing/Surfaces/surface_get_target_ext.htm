surface_get_target_ext Эта функция извлекает идентификатор поверхности, назначенный одной из 4 целей рендеринга, доступных для поверхностей. Вы указываете индекс цели рендеринга для проверки, и функция вернет -1, если поверхность не назначена, или целое значение >= 0, представляющее собой ID назначенной поверхности (как возвращено функцией surface_create()). Синтаксис: surface_get_target_ext(index); Аргумент Описание индекс Индекс цели рендеринга для проверки (от 0 до 3). Возвращается: Целое число (-1 для отсутствия цели на поверхности, или значение идентификатора поверхности) Пример: if surface_get_target_ext(0) == -1 { surface_set_target_ext(0, global.Surf); } Приведенный выше код сначала проверит, была ли шейдер render target 0 установлен на поверхность, и если нет, то назначит ее.
