Рисование Этот раздел содержит все функции, связанные с рисованием в игровой комнате, а также для управления тем, как все будет нарисовано (смешивание, альфа, выравнивание и т.д...). Существует большое количество различных функций для рисования, и они разделены на следующие категории, чтобы было легче найти то, что вам нужно: Цвет и альфа Управление графическим процессором Mipmapping Основные формы Спрайты и плитки Текст Примитивы и форматы вершин Поверхности Освещение Частицы Текстуры Шейдеры Также доступны две специальные функции, одна из которых используется для включения или отключения события Draw Event для вашего проекта, а другая - для промывки всего конвейера рисования: draw_enable_drawevent draw_flush Назад: Ссылка наGML Следующий: Камеры и дисплей © Copyright YoYo Games Ltd. 2021 Все права защищены
