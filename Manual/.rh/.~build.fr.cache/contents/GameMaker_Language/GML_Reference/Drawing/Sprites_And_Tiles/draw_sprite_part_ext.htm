draw_sprite_part_ext Эта функция нарисует часть выбранного спрайта в заданной позиции, следуя тем же правилам, что и функция draw_sprite_part()только теперь вы можете масштабировать часть, смешивать с ней цвет или изменять ее альфа-коэффициент при (так же, как при рисовании спрайта с помощью функции draw_sprite_ext()). Следует отметить, что если страница текстуры допускает автоматическое кадрирование, то эта функция может работать не так, как ожидается, поскольку дополнительное "пустое" пространство вокруг спрайта будет удалено при создании страницы текстуры. Чтобы решить эту проблему, необходимо в настройках страницы текстуры (в редакторе группы текстур) отключить автоматическое кадрирование. ПРИМЕЧАНИЕ: При рисовании с помощью этой функции, смещение спрайта по x и y игнорируется, и часть спрайта будет нарисована с левым верхним углом в заданной позиции x / y в комнате. ПРИМЕЧАНИЕ: Эта функция полезна только для растровых спрайтов и не работает со спрайтами SWF или JSON (Spine). ПРИМЕЧАНИЕ: Смешивание цветов рекомендуется только для HTML5-цели, когда включен WebGL, хотя вы можете установить цвет смешивания, если он не включен, и он будет смешивать спрайт как обычно. Однако при таком смешивании создается дубликат спрайта, который затем сохраняется в кэше и используется при необходимости. Это далеко не оптимально, и если вы используете многократное изменение цвета, это замедлит производительность вашей игры, если вы не активируете WebGL. если вы не хотите использовать WebGL, то вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это, если потребуется, используя функцию sprite_set_cache_size(). Синтаксис: draw_sprite_part_ext(sprite, subimg, left, top, width, height, x, y, xscale, yscale, color, alpha); Аргумент Описание спрайт Индекс спрайта для рисования. subimg Подimg (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте). слева Позиция x на спрайте левого верхнего угла области для рисования. топ Позиция y на спрайте левого верхнего угла области для рисования. ширина Ширина области для рисования. высота Высота области для рисования. x Координата x места рисования спрайта. y Координата y места рисования спрайта. xscale Горизонтальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... yscale Вертикальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... цвет Цвет, с которым будет смешиваться спрайт. c_white - отображать его в обычном режиме. альфа Альфа спрайта (от 0 до 1, где 0 - прозрачный, а 1 - непрозрачный). Возвращается: Н/Д Пример: draw_sprite_part_ext(sprite_index, image_index, 8, 8, sprite_width-16, sprite_height-16, x, y, 2, 0.5, c_black, 1); Это нарисует назначенный экземпляр спрайта (sprite_index) и его текущий кадр анимации (image_index), но при этом сбреет 8px поля со всех четырех сторон спрайта. Затем он будет растянут в два раза по ширине, но в два раза по высоте. и, хотя альфа все еще равна 1, он будет закрашен черным цветом (превращаясь в силуэт).
