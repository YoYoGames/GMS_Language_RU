нарисовать_спрайт_общий Эта функция объединяет функцию draw_sprite_ext() с функцией draw_sprite_part()но добавляет некоторые дополнительные параметры смешивания, чтобы каждый угол конечной части спрайта может быть смешана с отдельным цветом. ПРИМЕЧАНИЕ: Смешивание цветов рекомендуется только для HTML5-цели, когда включен WebGL, хотя вы можете установить цвет смешивания, если он не включен, и он будет смешивать спрайт как обычно. Однако при таком смешивании создается дубликат спрайта, который затем сохраняется в кэше и используется при необходимости. Это далеко не оптимально, и если вы используете многократное изменение цвета, это замедлит производительность вашей игры, если вы не активируете WebGL. если вы не хотите использовать WebGL, то вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это, если потребуется, используя функцию sprite_set_cache_size(). ПРИМЕЧАНИЕ: Эта функция может работать не так, как ожидается, при использовании спрайтов со скелетной анимацией, и вы можете обнаружить, что функция рисует только первый кадр позы по умолчанию. Вместо нее следует использовать функции draw_skeleton_*. Синтаксис: draw_sprite_general(sprite, subimg, left, top, width, height, x, y, xscale, yscale, rot, c1, c2, c3, c4, alpha); Аргумент Описание спрайт Индекс спрайта для рисования. subimg Подimg (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте). слева Позиция x на спрайте левого верхнего угла области для рисования. топ Позиция y на спрайте левого верхнего угла области для рисования. ширина Ширина области для рисования. высота Высота области для рисования. x Координата x места рисования спрайта. y Координата y места рисования спрайта. xscale Горизонтальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... yscale Вертикальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... гниение Поворот спрайта. 0=нормальный, 90=повернутый на 90 градусов против часовой стрелки и т.д. c1 Цвет, которым будет смешиваться левая верхняя область спрайта. c2 Цвет, которым будет смешиваться правая верхняя область спрайта. c3 Цвет, которым будет смешиваться правая нижняя область спрайта. c4 Цвет, которым будет смешиваться левая нижняя область спрайта. альфа Альфа спрайта (от 0 до 1, где 0 - прозрачный, а 1 - непрозрачный). Возвращается: Н/Д Пример: draw_sprite_general(sprite_index, image_index, 8, 8, sprite_width-16, sprite_height-16, x, y, 2, 0.5, 180, c_white, c_white, c_black, c_black, 1); При этом будет нарисован назначенный экземпляр спрайта (sprite_index) и его текущий кадр анимации (image_index), но при этом будет сбрито 8px поле со всех четырех сторон спрайта. Он будет растянут на двойную ширину, но половину своей обычной высоты. Он будет непрозрачным и перевернутым. Верхняя область спрайта будет белой и, следовательно, нормальной, но нижняя область будет черной, то есть спрайт будет переходить от нормального силуэта к силуэту вниз в плавном градиенте.
