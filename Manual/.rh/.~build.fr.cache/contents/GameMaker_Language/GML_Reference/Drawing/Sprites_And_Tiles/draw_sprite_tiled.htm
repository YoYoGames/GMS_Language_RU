нарисовать_прямую_плитку Эта функция берет спрайт и многократно чередует его по всему виду (или комнате, если вид не определен), начиная с координат, которые вы задаете в функции. Плитка основывается на ширине и высоте спрайта, которые определяются переменными sprite переменных экземпляра, выполняющего код. Эта функция предназначена только для 2D (ортографических) проекций и не будет работать корректно, если используется 3D проекции камеры. Синтаксис: draw_sprite_tiled(sprite, subimg, x, y); Аргумент Описание спрайт Индекс спрайта для рисования. subimg Подimg (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте). x Координата x места рисования спрайта. y Координата y места рисования спрайта. Возвращается: Н/Д Пример: draw_sprite_tiled(sprite_index, image_index, x, y); Это нарисует назначенный экземпляру спрайт (sprite_index) и его текущий кадр анимации (image_index) в собственной позиции x и y экземпляра, а также черепицу по горизонтали и вертикали по всему представлению.
