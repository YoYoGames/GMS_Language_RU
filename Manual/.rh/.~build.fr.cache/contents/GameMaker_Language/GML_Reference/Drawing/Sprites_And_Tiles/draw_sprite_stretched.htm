нарисовать_спрайт_растянутый Эта функция просто берет ресурс спрайта и растягивает его по заданной ширине и высоте так, чтобы он занимал эту область. Как и в случае с draw_sprite() вы можете указать спрайт и суб-изображение для рисования, затем позицию x / y позицию в комнате, в которой будет рисоваться спрайт, и, наконец, ширину и высоту (которые должны быть пиксельными). На рисунке ниже показан результат работы этой функции с разными наборами аргументов: ПРИМЕЧАНИЕ: При рисовании с помощью этой функции, смещение спрайта по x и y игнорируется, и спрайт будет нарисован с левым верхним углом в заданной позиции x / y в комнате. Синтаксис: draw_sprite_stretched(sprite, subimg, x, y, w, h); Аргумент Описание спрайт Индекс спрайта для рисования. subimg Подimg (кадр) спрайта для рисования (image_index или -1 соотносятся с текущим кадром анимации в объекте). x Координата x места рисования спрайта. y Координата y места рисования спрайта. w Ширина области, которую будет занимать растянутый спрайт. h Высота области, которую будет занимать растянутый спрайт. Возвращается: Н/Д Пример: draw_sprite_stretched(sprite_index, image_index, x, y, sprite_width, sprite_height / 2); Это нарисует назначенный спрайт экземпляра и его подизображение с левым углом в позиции x/y экземпляра. Его ширина устанавливается равной ширине спрайта, а высота - высоте спрайта, деленной на два.
