нарисовать_картинку Эта функция может быть использована для рисования заданного tilemap в любом месте комнаты. Вы передаете идентификатор элемента tilemap (который вы получаете при создании элемента tilemap с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), затем указываете позицию x/y для рисования (в комнате). Обратите внимание, что это просто нарисует тайлмап в указанной точке, используя глубину слоя экземпляра, вызывающего функцию. Это не перемещает тайлмап, не изменяет его каким-либо образом, и не имеет значения, отмечен ли тайлмап как видимый или нет. Синтаксис: draw_tilemap(tilemap_element_id, x, y); Аргумент Описание tilemap_element_id Уникальное значение ID элемента tilemap для получения позиции y ячейки x Позиция x в помещении для рисования y Позиция y в помещении для рисования Возвращается: Н/Д Пример: var lay_id = layer_get_id("Tiles_Effects"); var map_id = layer_tilemap_get_id(lay_id); draw_tilemap(map_id, mouse_x, mouse_y); Приведенный выше код получает идентификатор слоя, затем использует его для получения определенного идентификатора тайлмапа, который, в свою очередь, используется для рисования тайлмапа в позиции мыши.
