draw_set_swf_aa_level Эта функция может использоваться для установки уровня сглаживания (AA) для векторных спрайтов формата SWF. Это может быть вещественное значение от 0 до 1, которое будет "сглаживать" края этих спрайтов. Обратите внимание, что для того, чтобы увидеть этот эффект, вы должны сначала включить AA с помощью функции функцию draw_enable_swf_aa(). Синтаксис: draw_set_swf_aa_level(AA); Аргумент Описание AA Используемое значение сглаживания от 0 до 1. Возвращается: Н/Д Пример: if draw_get_swf_aa_level() == 0 { draw_enable_swf_aa(true); draw_set_swf_aa_level(0.5); } Приведенный выше код проверит значение AA для спрайтов формата SWF, и если оно равно 0, то включит AA и установит значение 0,5.
