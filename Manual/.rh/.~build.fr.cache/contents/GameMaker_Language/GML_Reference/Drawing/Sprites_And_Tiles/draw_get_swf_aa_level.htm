draw_get_swf_aa_level Эта функция может использоваться для получения уровня сглаживания (AA) для векторных спрайтов формата SWF. Возвращаемое значение находится в диапазоне от 0 до 1 и показывает, насколько "гладкими" будут края этих спрайтов. Вы можете установить уровень AA с помощью функции draw_set_swf_aa_level(). Синтаксис: draw_get_swf_aa_level(); Возвращается: Реально Пример: if draw_get_swf_aa_level() == 0 { draw_enable_swf_aa(true); draw_set_swf_aa_level(0.5); } Приведенный выше код проверит значение AA для спрайтов формата SWF, и если оно равно 0, то включит AA и установит значение 0,5.
