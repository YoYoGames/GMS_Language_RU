часть_частиц_создать Эта функция идеально подходит для тех эффектов, которые не требуют функциональности, предлагаемой эмиттерами частиц (например, для создания дыма от ракеты или простого эффекта взрыва), поскольку она позволяет быстро и легко создавать частицы в любом месте игровой комнаты. Вы также можете окрашивать частицы "на лету", когда они создаются с помощью этой функции, и этот цвет будет перекрывать предопределенный цвет частицы, но он не будет смешиваться с цветом частицы в течение всего времени ее жизни. Обратите внимание, что для использования этой функции необходимо предварительно создать систему частиц и тип частиц. Синтаксис: part_particles_create_colour(ind, x, y, parttype, color, number); Аргумент Описание инд Индекс системы частиц. x Координата x места создания частиц. y Координата y места создания частиц. тип детали Индекс (тип) создаваемых частиц. цвет Смешивание цветов для частиц. номер Количество частиц для создания. Возвращается: Н/Д Пример: if (speed > 1) { var _c = choose(c_aqua, c_lime, c_fuschia, c_orange); part_particles_create_colour(sname, x, y, p_Smoke, _c, 1); } Приведенный выше код будет генерировать одну частицу на каждом шаге, когда экземпляр с кодом имеет скорость больше единицы. Эти частицы будут иметь случайный цвет. Назад: Системы частиц Далее: часть_системы_существует © Copyright YoYo Games Ltd. 2021 Все права защищены
