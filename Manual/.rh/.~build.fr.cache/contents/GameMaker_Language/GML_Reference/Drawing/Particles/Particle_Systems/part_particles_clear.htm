часть_частиц_чистая С помощью этой функции вы можете удалить из комнаты все частицы, созданные системой в данный момент. Она не сбрасывает и не удаляет сами типы частиц, только их визуальное представление, и если у вас есть какой-либо объект, излучающий частицы от эмиттера, эти частицы исчезнут, но начнут появляться снова на следующем шаге после вызова этого кода. Синтаксис: part_particles_clear(ind); Аргумент Описание инд Индекс системы частиц. Возвращается: Н/Д Пример: if (lives <= 0) { part_particles_clear(global.Sname); room_goto(rm_intro); } Приведенный выше код проверит значение переменной "lives" и, если оно равно 0, очистит все частицы из системы, а затем сменит комнату.
