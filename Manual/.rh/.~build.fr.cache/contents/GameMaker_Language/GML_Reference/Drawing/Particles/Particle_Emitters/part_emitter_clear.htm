часть_излучателя_чистая С помощью этой функции вы можете очистить данный эмиттер от указанной системы частиц и вернуть его в состояние по умолчанию. Это также остановит все частицы, которые в данный момент передаются из эмиттера, и если вы захотите использовать эмиттер снова, вам нужно будет необходимо установить позицию региона и тип частицы с помощью функции part_emitter_region() функцию. Синтаксис: part_emitter_clear(ps, ind); Аргумент Описание ps Система частиц, в которой находится эмиттер. инд Индекс эмиттера, который нужно очистить. Возвращается: Н/Д Пример: part_emitter_clear(Sname, p_emit1 ); Приведенный выше код очистит эмиттер частиц, индексированный в переменной "p_emit1".
