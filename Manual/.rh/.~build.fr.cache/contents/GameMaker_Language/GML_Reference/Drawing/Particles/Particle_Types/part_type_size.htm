тип_части_размер Эта функция используется для определения размера частицы при ее создании, а также может быть использована для увеличения или уменьшения размера частицы в течение ее жизни. Минимальное и максимальное значения размера по умолчанию равны 1 (это означает, что размер равен соотношение с базовой частицей после part_type_scale() и если они установлены в одно и то же значение, начальный размер для всех частиц будет одинаковым, но если они установлены в разные значения, каждая частица начнет со случайным размером из диапазона, например, если у вас минимальное значение 0,5, а максимальное 2, все частицы будут созданы с размером между половиной и удвоенной базовой шкалой. Вы также можете задать значение приращения, которое заставит спрайт частицы увеличиваться (если это положительное число) или уменьшаться (если это отрицательное число) в течение времени жизни частицы, причем заданное значение будет добавляться к размеру частицы на каждом шаге. Это значение может быть минимум (+/-)0,01. Наконец, вы можете задать коэффициент "вихляния". Это значение, которое будет добавляться или вычитаться случайным образом к размеру на каждом шаге времени жизни частиц. Очевидно, что большие значения более выражены, чем меньшие, и это значение может быть даже отрицательным с максимальным диапазоном от -20 до 20. Синтаксис: part_type_size(ind, size_min, size_max, size_incr, size_wiggle); Аргумент Описание инд Индекс типа частицы, который необходимо изменить. размер_min Минимальный размер частицы, с которого она может стартовать. size_max Максимальный размер частицы, с которого она может стартовать. размер_incr На сколько частица должна увеличиваться или уменьшаться за шаг. размер_колебаний Сколько нужно произвольно прибавлять или вычитать из размера частиц на каждом шаге. Возвращается: Н/Д Пример: part_type_shape(particle1, pt_shape_explosion); part_type_size(particle1, 2, 2, 0.10, 0.01); part_type_scale(particle1, 1, 1); part_type_colour2(particle1, 16777215, 12632256); part_type_alpha3(particle1, 0, 0.20, 0); part_type_speed(particle1, 0.20, 0.20, 0, 0); part_type_direction(particle1, 0, 359, 0, 20); part_type_orientation(particle1, 0, 359, 0, 0, 1); part_type_blend(particle1, 0); part_type_life(particle1, 40, 60); Приведенный выше код установит различные значения частиц, включая размер, который при создании каждой частицы устанавливается в двойном размере, а затем увеличивает размер на 0,01 каждый шаг с колебанием в 0,1.
