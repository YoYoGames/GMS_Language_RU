часть_излучателя_потока Эта функция позволяет настроить эмиттер на поток частиц определенного типа и обычно используется в событии create, поскольку это одноразовый код, который устанавливает эмиттер на генерацию указанного количества частиц на каждом последующем шаге игры. частицы испускаются в соответствии с распределением, формой и положением, заданными функцией part_emitter_region(). Вы можете установить количество излучаемых частиц на ноль и снова вызвать эту функцию. функция также может принимать отрицательные числа для количества частиц, в этом случае эмиттер будет испускать частицы по случайному принципу. Например, если вы установили количество частиц равным -5, то вероятность того, что на каждом шаге будет произведена частица, будет 1:5. Если вам нужно, чтобы частицы появлялись все сразу, а не на каждом шаге, используйте функцию part_emitter_burst(). Синтаксис: part_emitter_stream(ps, ind, parttype, number); Аргумент Описание ps Система частиц, в которой находится эмиттер. инд Индекс эмиттера, из которого будет происходить взрыв. тип детали Индекс (тип) создаваемых частиц. номер Количество частиц, создаваемых за шаг. Возвращается: Н/Д Пример: part_emitter_stream(global. Sname, p_emit1, p1, 1); Приведенный выше код будет передавать по 1 частице на каждом шаге игры, пока эмиттер не будет уничтожен или поток не установится на 0.
