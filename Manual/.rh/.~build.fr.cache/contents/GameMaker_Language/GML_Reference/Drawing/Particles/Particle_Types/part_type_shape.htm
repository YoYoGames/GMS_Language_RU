тип_части_формы С помощью этой функции можно задать форму спрайта, используемую для типа частиц, при этом допустимы следующие константы: Постоянная Описание pt_shape_pixel Пиксель 1x1. (Это настройка по умолчанию). pt_shape_disk Заполненный круг. pt_shape_square Заполненный квадрат. pt_shape_line Горизонтальная линия шириной 8px. pt_shape_star Пятиконечная заполненная звезда. pt_shape_circle Очерченный круг размером 3px. pt_shape_ring Круг с внутренним свечением (выглядит как пузырь). pt_shape_sphere Круг с исходящим свечением ' твердый в центре, светящийся наружу. pt_shape_flare Резко светящаяся точка (выглядит как настоящая звезда в ночи). pt_shape_spark Эффект искры ' как звезда с несколькими точками, угасающая. pt_shape_explosion Квадратное облако дыма" требует нескольких цветов, чтобы напоминать взрыв. pt_shape_cloud Тонкое облако требует увеличения масштаба и множества частиц, чтобы походить на облако. pt_shape_smoke Сглаженная версия эффекта взрыва. Используйте несколько для создания дымового облака. pt_shape_snow Обычная форма снежинки. Синтаксис: part_type_shape(ind, shape); Аргумент Описание инд Индекс типа частицы, который необходимо установить. форма Выбор формы. Возвращается: Н/Д Пример: global.p1 = part_type_create(); part_type_shape(global.p1, pt_shape_square); part_type_size(global.p1, 1, 3, 0, 0); part_type_scale(global.p1, 1, 1); part_type_colour1(global.p1, c_white); part_type_alpha2(global.p1, 1, 0); part_type_speed(global.p1, 2, 4, 0, 0); part_type_direction(global.p1, 0, 180, 0, 0); part_type_gravity(global.p1, 0.20, 270); part_type_orientation(global.p1, 0, 0, 0, 0, 1); part_type_blend(global.p1, 1); part_type_life(global.p1, 15, 60); Приведенный выше код устанавливает тип частицы, указанный в глобальной переменной "p1", в форму квадрата, а затем устанавливает различные другие свойства частицы.
