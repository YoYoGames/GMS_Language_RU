тип_детали_масштаб Эта функция устанавливает горизонтальный и вертикальный масштаб частицы до того, как будут произведены другие изменения размера. Эта функция работает на соотношении размеров спрайта, поэтому значение 1 (значение по умолчанию) соответствует соотношению 1:1, что означает, что 0,5 будет вдвое меньше, а 2 - вдвое больше. а 2 - в два раза больше. Синтаксис: part_type_scale(ind, xscale, yscale); Аргумент Описание инд Индекс типа частицы, который необходимо изменить. xscale Горизонтальное масштабирование базовой частицы. yscale Вертикальный масштаб базовой частицы. Возвращается: Н/Д Пример: part_type_shape(particle1, pt_shape_flare); part_type_size(particle1, 2, 2, 0.10, 1); part_type_scale(particle1, 8, 0.50); part_type_colour3(particle1, 8388608, 4227327, 8454143); part_type_alpha3(particle1, 0.10, 0.10, 0); part_type_speed(particle1, 0, 1, 0.18, 20); part_type_direction(particle1, 0, 359, 0, 0); part_type_orientation(particle1, 0, 0, 0, 0, 1); part_type_blend(particle1, 1); part_type_life(particle1, 10, 40); Приведенный выше код установит различные значения частиц, включая масштаб, который устанавливает xscale в восемь раз больше размера, а yscale в половину размера.
