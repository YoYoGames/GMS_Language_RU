тип_детали_цвет2 Эта функция может быть использована для установки двухцветного градиента для каждой созданной частицы заданного типа. Первый цвет - это цвет, с которого будут начинаться все частицы, а второй цвет - это цвет, на котором частица будет заканчиваться, и плавное градиентное изменение цвета будет происходить на протяжении всего времени жизни частицы от одного цвета к другому. Синтаксис: part_type_colour2(ind, color1, color2); Аргумент Описание инд Индекс типа частицы, который необходимо изменить. цвет1 Цвет, с которого будет начинаться частица. цвет2 Цвет, на котором будет заканчиваться частица. Возвращается: Н/Д Пример: part_type_colour2( part_Health, c_red, c_white); Приведенный выше код заставит все созданные частицы типа, указанного в переменной "part_Health", менять цвет от красного к белому в течение их жизни.
