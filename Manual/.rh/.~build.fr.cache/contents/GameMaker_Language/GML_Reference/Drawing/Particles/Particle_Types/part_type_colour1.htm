тип_детали_цвет1 Эта функция используется для установки типа частицы, чтобы она была одного цвета в течение всего времени жизни каждой отдельной частицы. Синтаксис: part_type_colour1(ind, color1); Аргумент Описание инд Индекс типа частицы, который необходимо изменить. цвет1 Единственный цвет, которым будет окрашен тип частицы. Возвращается: Н/Д Пример: part_type_colour1(global.Snow_Part, c_white); Приведенный выше код установит, что все частицы, созданные по типу частиц, индексированных в глобальной переменной "Snow_Part", будут только белыми.
