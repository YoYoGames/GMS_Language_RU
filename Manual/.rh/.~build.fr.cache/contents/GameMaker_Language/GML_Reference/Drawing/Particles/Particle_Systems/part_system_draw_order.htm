порядок_системы_черчения С помощью этой функции можно задать способ отрисовки частиц при создании на экране. Система по умолчанию использует старый > новый вид (функция установлена в true), где старые частицы рисуются на большей глубине, чем новые, и поэтому появляются "под" новыми частицами. Таким образом, новые частицы оказываются "под" ними, но, установив эту функцию в false, вы можете изменить этот порядок, и новые частицы будут рисоваться выше и, таким образом, окажутся "под" старыми. Изображения ниже иллюстрируют это, на изображении слева по умолчанию установлено значение true, а на изображении справа - false: ПРИМЕЧАНИЕ: Когда частицы рисуются с аддитивным режимом наложения, эффект от этой функции не всегда очевиден. Синтаксис: part_system_draw_order(ind, oldtonew); Аргумент Описание инд Индекс системы частиц для изменения. oldtonew Должны ли старые частицы рисоваться за более новыми (true) или наоборот (false). Возвращается: Н/Д Пример: mysystem = part_system_create(); part_system_draw_order(mysystem, true); Это создаст новую систему частиц с индексом " mysystem", а затем установит частицы для рисования новых частиц поверх старых.
