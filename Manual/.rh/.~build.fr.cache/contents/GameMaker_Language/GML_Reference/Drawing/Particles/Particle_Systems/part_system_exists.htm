часть_системы_существует С помощью этой функции вы можете проверить, существует ли данная система частиц в игре или нет. Обратите внимание, что если проверяемая переменная является неинициализированной переменной (которой система частиц в противном случае присвоила бы свой индекс), эта функция будет произойдет ошибка. Синтаксис: part_system_exists(ind); Аргумент Описание инд Индекс системы частиц для проверки. Возвращается: Булево Пример: if !part_system_exists(global.Sname) { global.Sname = part_system_create(); } Приведенный выше код проверяет, существует ли система частиц, на которую ссылается глобальная переменная, и если нет, то она создается.
