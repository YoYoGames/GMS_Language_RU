часть_системы_создать Эта функция используется для создания новой системы частиц и возвращает уникальный индексный номер, который должен быть сохранен и использован во всех дальнейших функциях, относящихся к этой системе. Системе будет присвоен управляемый слой и установлена глубина 0. Управляемые слои недоступны для пользователя и используются только для внутреннего управления, когда вместо слоев используется глубина. Обычно вы используете функцию part_system_create_layer() вместо вместо этой. Синтаксис: part_system_create(); Возвращается: Реально Пример: global.Sname = part_system_create(); Это создаст новую систему частиц и сохранит индекс в глобальной переменной "Sname".
