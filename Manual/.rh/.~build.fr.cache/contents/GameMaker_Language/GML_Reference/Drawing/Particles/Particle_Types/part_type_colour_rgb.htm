тип_части_цвета_rgb С помощью этой функции вы можете задать сочетание красного, зеленого и синего цветов для всех создаваемых частиц данного типа. Вы задаете минимальное значение и максимальное значение для каждого из трех компонентов, и созданные частицы будут иметь случайный цвет, основанный на на основе заданного диапазона параметров. Все значения должны быть в диапазоне от 0 до 255. Синтаксис: part_type_colour_rgb(ind, rmin, rmax, gmin, gmax, bmin, bmax p> Аргумент Описание инд Индекс типа частицы, который необходимо изменить. rmin Минимальное значение красной составляющей конечного цвета. rmax Максимальное значение красной составляющей конечного цвета. gmin Минимальное значение зеленого компонента конечного цвета. gmax Максимальное значение зеленого компонента конечного цвета. bmin Минимальное значение синей составляющей конечного цвета. bmax Максимальное значение синей составляющей конечного цвета. Возвращается: Н/Д Пример: part_type_rgb(global.Blood_Part, 0, 255, 0, 0, 0, 0, 0); Приведенный выше код устанавливает, что каждая частица, испускаемая типом частицы, проиндексированным в глобальной переменной "Blood_Part", имеет только различные оттенки красного цвета.
