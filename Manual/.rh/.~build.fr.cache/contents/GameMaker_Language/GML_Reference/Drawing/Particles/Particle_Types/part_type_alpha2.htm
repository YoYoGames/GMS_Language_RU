тип_части_альфа2 Эта функция может быть использована для установки градиента альфа-значения (прозрачности) для каждой созданной частицы данного типа. Первая альфа - это та, с которой все частицы будут начинаться, а вторая альфа - та, с которой частица будет заканчиваться, и плавное градиентное изменение альфа будет происходить на протяжении жизни частиц от одной к другой. Это значение может быть от 0 (прозрачный) до 1 (непрозрачный). Синтаксис: part_type_alpha2(ind, alpha1, alpha2); Аргумент Описание инд Индекс типа частицы, который необходимо изменить. альфа1 Стартовая альфа частицы. альфа2 Конечная альфа-частица. Возвращается: Н/Д Пример: part_type_alpha2(global.Snow_Part, 0, 1); Приведенный выше код установит для всех созданных частиц типа частиц, индексированных в глобальной переменной "Snow_Part", альфа-значение 0 (прозрачный), а в конце их жизни альфа-значение станет равным 1 (непрозрачный).
