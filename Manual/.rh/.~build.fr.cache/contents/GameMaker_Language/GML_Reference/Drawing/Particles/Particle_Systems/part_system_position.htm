часть_системы_позиция С помощью этой функции вы можете установить базовую позицию для системы частиц относительно позиции (0,0) комнаты, что означает, что все последующие функции частиц, относящиеся к этой системе, теперь будут рисоваться относительно новой позиции. По умолчанию это всегда (0,0), но в некоторых особых обстоятельствах вы можете захотеть изменить это положение на другое. ПРИМЕЧАНИЕ: Эта функция изменит все в системе частиц, поэтому если у вас есть эмиттер в позиции (100,100), а затем установите позицию системы частиц на (0,100), эмиттер теперь будет рисовать в позиции (100,200). То же самое произойдет, если вы сместите систему, а затем создать эмиттер, так как даже если вы создадите его в позиции (100,100), он будет нарисован в позиции (100,200). Синтаксис: part_system_position(ind, x, y); Аргумент Описание инд Индекс системы частиц для изменения. x Новая координата x системы частиц. y Новая координата y системы частиц. Возвращается: Н/Д Пример: if mouse_check_button_pressed(mb_left) { part_system_position(global.Sname, mouse_x, mouse_y); } Приведенный выше код будет проверять нажатие кнопки мыши, и если он обнаружит таковое, система частиц, индексированная в глобальной переменной "Sname", будет смещена в позицию x/y мыши.
