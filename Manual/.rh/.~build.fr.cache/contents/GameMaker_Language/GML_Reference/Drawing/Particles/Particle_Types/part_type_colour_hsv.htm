тип_части_цвета_hsv С помощью этой функции вы можете задать оттенок, насыщенность и диапазон значений для всех частиц данного типа. Вы задаете минимальное значение и максимальное значение для каждого из трех компонентов, и созданные частицы будут иметь случайный цвет, основанный на заданном диапазона параметров. Таким образом, вы можете создавать частицы одного оттенка, но разной насыщенности, или разных оттенков, но одинаковой величины (светимость) и т.д.. Все значения должны быть в диапазоне от 0 до 255. Синтаксис: part_type_colour_hsv(ind, hmin, hmax, smin, smax, vmin, vmax); Аргумент Описание инд Индекс типа частицы, который необходимо изменить. hmin Минимальный компонент оттенка конечного цвета. hmax Максимальное значение компонента оттенка конечного цвета. smin Минимальный компонент насыщенности конечного цвета. smax Максимальный компонент насыщенности конечного цвета. vmin Минимальное значение компонента значения конечного цвета. vmax Максимальный компонент значения конечного цвета. Возвращается: Н/Д Пример: part_type_hsv(global.Stars, 0, 255, 0, 255, 255, 255 ); Приведенный выше код устанавливает для каждой испускаемой частицы типа частицы, индексируемой в глобальной переменной "Stars", разные цвета и насыщенность, но одинаковое значение (светимость).
