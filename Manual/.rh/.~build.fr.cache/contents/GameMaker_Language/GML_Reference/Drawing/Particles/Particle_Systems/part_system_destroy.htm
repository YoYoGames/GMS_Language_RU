часть_системы_уничтожить С помощью этой функции вы можете уничтожить данную систему частиц и удалить ее из памяти. Эта функция всегда должна вызываться, когда система больше не нужна, например, в конце комнаты или в событии уничтожения экземпляра, иначе вы можете оказаться с частицы появляются в последующих комнатах, а удалить их нет возможности, а также произойдет утечка памяти, что в конечном итоге приведет к краху игры. ПРИМЕЧАНИЕ: Эта функция также уничтожит и удалит все эмиттеры, которые могли быть созданы и связаны с уничтожаемой системой. Синтаксис: part_system_destroy(ind); Аргумент Описание инд Индекс системы частиц для уничтожения. Возвращается: Н/Д Пример: if (global.player_hp <= 0) { part_system_destroy(p_sys); room_goto_next(); } Приведенный выше код проверяет, меньше или равно нулю значение глобальной переменной, и если оно равно нулю, то уничтожает систему частиц, на которую ссылается данная переменная, а затем переходит в следующую комнату.
