draw_set_font Эта функция устанавливает шрифт, который будет использоваться для дальнейшего рисования текста. Этот шрифт должен быть добавлен в шрифтовые активы игры или создан с помощью функции font_add() или font_add_sprite(). Синтаксис: draw_set_font(font); Аргумент Описание шрифт Имя используемого шрифта. Возвращается: Н/Д Пример: draw_set_colour(c_blue); draw_set_font(fnt_Game); draw_text(200, 200, "Hello World" ); Приведенный выше код нарисует заданный текст, используя шрифт, индексированный в переменной "fnt_Game" и окрашенный в синий цвет.
