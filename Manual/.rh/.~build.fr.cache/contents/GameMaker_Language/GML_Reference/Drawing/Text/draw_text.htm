рисовать_текст С помощью этой функции вы можете нарисовать любую строку в любой позиции в пределах комнаты (для рисования вещественных чисел следует использовать функцию строка() для преобразования их в текст). Для объединения строк вы можете использовать + (см. пример ниже), а также использовать \n в строке для добавления разрыва строки, чтобы она рисовалась в несколько строк (информацию о том, как правильно форматировать строку и какие управляющие символы можно использовать, смотрите здесь). Цвет текста и альфа определяются текущими базовыми значениями альфа и цвета, заданными в параметрах draw_set_alpha() и draw_set_colour(). ПРИМЕЧАНИЕ:Фактическое положение текста будет зависеть от значений выравнивания, заданных параметрами draw_set_halign() и draw_set_valign(). Синтаксис: draw_text(x, y, string); Аргумент Описание x Координата x нарисованной строки. y Координата y нарисованной строки. строка Нить, которую нужно натянуть. Возвращается: Н/Д Пример: draw_text(x, y, "Здравствуйте, " + global.Name + "!\nНадеюсь, у вас все хорошо!"); Приведенный выше код нарисует строку в позиции экземпляра x/y, которая будет использовать строку, хранящуюся в глобальной переменной "Name", и разделит ее на две строки.
