нарисовать_текст_текста Эта функция будет рисовать текст аналогично функции нарисовать_текст() только теперь вы можете задать пространство между каждой строкой текста - если текст занимает более одной строки - и ограничить ширину (в пикселях) строки в строке, так что если любая строка превысит это значение, GameMaker Studio 2 автоматически разделит текст на следующую строку в ближайшем доступном месте (если в тексте нет свободного места, он превысит это максимальное значение). значение ширины). Обратите внимание, что любое белое пространство в начале строки будет вычеркнуто перед разбором для отрисовки. Также обратите внимание, что при значении -1 для аргумента разделения строк по умолчанию будет установлено разделение, основанное на значении высоте символа "M" в выбранном шрифте. Синтаксис: draw_text_ext(x, y, string, sep, w); Аргумент Описание x Координата x нарисованной строки. y Координата y нарисованной строки. строка Нить, которую нужно натянуть. сеп Расстояние в пикселях между строками текста. w Максимальная ширина строки в пикселях перед переводом строки. Возвращается: Н/Д Пример: draw_text_ext(100, 50, keyboard_string, 3, 300); Приведенный выше код будет рисовать текст, который пользователь набирает на клавиатуре, разбивая его на новые строки каждый раз, когда длина строки превышает 300 пикселей. Код также будет поддерживать разделение в 3 пикселя между строками, если это произойдет. Назад: Текст Далее: draw_text_colour © Copyright YoYo Games Ltd. 2021 Все права защищены
