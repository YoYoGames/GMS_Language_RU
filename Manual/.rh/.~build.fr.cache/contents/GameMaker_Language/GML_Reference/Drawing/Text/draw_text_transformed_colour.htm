рисовать_текст_трансформированный_цвет Эта функция представляет собой комбинацию базовой draw_text() функции с draw_text_transformed() и нарисовать_текст_цвет() позволяющая масштабировать и поворачивать текст, а также окрашивать его градиентной заливкой и изменять значение альфа, игнорируя базовые настройки альфа и цвета для рисования. ПРИМЕЧАНИЕ: Градиентное смешивание недоступно для HTML5, если не включен WebGL, хотя вы все равно можете задать цвета смешивания, и шрифт будет смешиваться с первым заданным цветом. Однако при таком смешивании создается дубликат шрифта, который затем сохраняется в кэше и используется по мере необходимости, что далеко не оптимально, а если вы используете многократное изменение цвета, это замедлит производительность вашей игры. Вы можете установить размер кэша шрифтов, чтобы попытаться ограничить это в случае необходимости используя функцию font_set_cache_size(). Синтаксис: draw_text_transformed_colour(x, y, string, xscale, yscale, angle, c1, c2, c3, c4, alpha); Аргумент Описание x Координата x нарисованной строки. y Координата y нарисованной строки. строка Нить, которую нужно натянуть. xscale Горизонтальная шкала. yscale Вертикальная шкала. угол Угол наклона текста. c1 Цвет для верхней левой части нарисованного текста. c2 Цвет для верхней правой части нарисованного текста. c3 Цвет для правой нижней части нарисованного текста. c4 Цвет для левой нижней части нарисованного текста. альфа Альфа для текста. Возвращается: Н/Д Пример: draw_set_halign(fa_center); draw_set_valign(fa_middle); image_angle += 1; draw_text_transformed_colour(room_width / 2, room_height / 2, keyboard_string, 2, 2, image_angle, c_red, c_red, c_yellow, c_yellow, 0.5); Приведенный выше код нарисует заданный текст в центре комнаты, вращаясь по кругу и масштабируя его до удвоенного исходного размера, с градиентом цвета от желтого к красному, а также с альфой 0,5.
