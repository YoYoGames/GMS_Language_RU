draw_get_font Эта функция получает шрифт, назначенный в настоящее время для рисования текста. Функция вернет -1, если шрифт не установлен, или значение ID (целое положительное число) для назначенного ресурса шрифта. Синтаксис: draw_get_font(); Возвращается: Целое число (-1 или значение идентификатора шрифта) Пример: if draw_get_font() != fnt_MainMenu { draw_set_colour(c_blue); draw_set_font(fnt_MainMenu); } draw_text(room_width / 2, 200, "MENU" ); Приведенный выше код проверит, какой шрифт используется для рисования, и если это не fnt_MainMenu , то шрифт и цвет рисования будут установлены до того, как текст будет нарисован на экране.
