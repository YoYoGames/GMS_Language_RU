draw_get_valign Эта функция используется для получения настройки выравнивания текста по вертикальной оси и возвращает одну из перечисленных ниже констант. Синтаксис: draw_get_valign(); Возвращается: Постоянная Постоянная Выравнивание fa_top fa_middle fa_bottom Пример: if draw_get_valign() != fa_top { draw_set_valign(fa_top); } draw_text(100, 32, "Score: " + string(score)); Приведенный выше код проверит, является ли текущее выравнивание текста выравниванием по верху, и если нет, то оно будет установлено, после чего будет нарисован текст.
