нарисовать_вершину Эта функция определяет положение вершины для примитива. Окончательный вид примитива будет зависеть от типа примитива, выбранного для рисования, и порядка добавления в него вершин. См. draw_primitive_begin() для более подробную информацию. Чтобы завершить и нарисовать примитив, необходимо вызвать команду draw_primitive_end(). Синтаксис: draw_vertex(x, y) Аргумент Описание x Координата x вершины. y Координата y вершины. Возвращается: Н/Д Пример: draw_primitive_begin(pr_trianglelist); draw_vertex(100, 100); draw_vertex(100, 200); draw_vertex(150, 150); draw_primitive_end(); Приведенный выше код нарисует простой примитив треугольника.
