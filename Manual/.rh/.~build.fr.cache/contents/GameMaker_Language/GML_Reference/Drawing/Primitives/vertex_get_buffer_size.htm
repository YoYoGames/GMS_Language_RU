размер вершины_get_buffer_size С помощью этой функции вы можете получить размер заданного вершинного буфера в байтах. Синтаксис: vertex_get_buffer_size(buffer); Аргумент Описание буфер Буфер для получения размера. Возвращается: Реально Пример: bufferBytes = vertex_get_buffer_size(buff); Приведенный выше код получит количество байт, используемых данным вершинным буфером, и сохранит это значение в переменной "bufferBytes".
