vertex_format_add_position Скажите GameMaker Studio 2, чтобы она принимала 2D позиционные данные (x и y) как часть создаваемого нового формата вершин. Синтаксис: vertex_format_add_position(); Возвращается: Н/Д Пример: vertex_format_begin(); vertex_format_add_colour(); vertex_format_add_position(); my_format = vertex_format_end(); Приведенный выше код создаст новый формат вершин, содержащий только значения цвета и позиции, а затем сохранит идентификатор формата в переменной "my_format".
