vertex_create_buffer_from_buffer_ext Как и в случае с функцией vertex_create_buffer()эта функция создает новый буфер вершин, только теперь данные вершин, хранящиеся в нем, копируются из обычного буфера, указанного в качестве источника. Буфер предварительно отформатирован в вершинном формате для построения примитивов для использования (например) в шейдерах, и вы также можете указать смещение в исходном буфере для копирования и количество вершин, которое должно быть в конечном буфере. Обратите внимание, что если количество вершин не совпадает с копируемыми, вы можете получить поврежденные вершинные данные. Синтаксис: vertex_create_buffer_from_buffer_ext(buffer, format, src_offset, vert_num); Аргумент Описание буфер Буфер для создания буфера вершин. формат Используемый формат примитивных вершин. src_offset Смещение в исходном буфере для копирования из него. vert_num Количество вершин, которые должен иметь буфер. Возвращается: Указатель Пример: vertex_format_begin(); vertex_format_add_position_3d(); vertex_format_add_colour(); vertex_format_add_textcoord(); var my_format = vertex_format_end(); v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat, 0, 512); Приведенный выше код создаст новый вершинный формат, затем создаст новый вершинный буфер из ранее созданного обычного буфера, применив к нему пользовательский вершинный формат со смещением 0.
