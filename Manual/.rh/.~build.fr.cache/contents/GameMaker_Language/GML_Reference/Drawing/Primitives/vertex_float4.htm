вершина_плоская4 Эта функция добавит четыре значения с плавающей точкой к данным вершины. Вершина должна быть правильно отформатирована, чтобы принять это с помощью функции vertex_format_add_custom() функцию. Синтаксис: vertex_float4(buffer, float, float, float, float); Аргумент Описание буфер Буфер для записи информации. float Первое входное значение. float Второе входное значение. float Третье входное значение. float Четвертое входное значение. Возвращается: Н/Д Пример: vertex_float4(buff, 0.05, 0.01, room_width / x, room_height / y); Приведенный выше код добавит четыре значения с плавающей точкой к определяемым данным вершин.
