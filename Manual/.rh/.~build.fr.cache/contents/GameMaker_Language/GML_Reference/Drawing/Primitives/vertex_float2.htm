вершина_плоская2 Эта функция добавит два значения с плавающей точкой к данным вершины. Вершина должна быть правильно отформатирована, чтобы принять ее с помощью функции vertex_format_add_custom() функцию. Синтаксис: vertex_float2(buffer, float, float); Аргумент Описание буфер Буфер для записи информации. float Первое входное значение. float Второе входное значение. Возвращается: Н/Д Пример: vertex_float2(buff, 0.05, 0.01); Приведенный выше код добавит два значения с плавающей точкой к определяемым данным вершин.
