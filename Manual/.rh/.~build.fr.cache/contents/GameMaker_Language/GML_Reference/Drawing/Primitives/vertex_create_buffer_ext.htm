vertex_create_buffer_ext Как и в случае с функцией vertex_create_buffer()эта функция создаст новый буфер вершин. Это специальный растущий буфер, созданный GameMaker Studio 2, который предварительно отформатирован для использования при создания примитивов для использования с шейдерами. Вы можете указать начальный размер буфера (в байтах), после чего будет возвращен "хэндл" (индекс) для буфера, который будет использоваться во всех последующих обращениях к буферу. Синтаксис: vertex_create_buffer_ext(size); Аргумент Описание размер Начальный размер буфера (в байтах). Возвращается: Указатель Пример: v_buff = vertex_create_buffer_ext(1024 * 1024); Приведенный выше код создаст новый буфер вершин, первоначально размером 1 МБ, и сохранит его хэндл в переменной "v_buff".
