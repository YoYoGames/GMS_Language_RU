vertex_create_buffer_from_buffer Как и в случае с функцией vertex_create_buffer()эта функция создает новый буфер вершин, только теперь данные вершин, хранящиеся в нем, копируются из обычного буфера, указанного в качестве источника. Буфер создается специальный растущий буфер, который предварительно отформатирован в вершинном формате для построения примитивов для использования (например) в шейдерах. Эта функция требует, чтобы вы указали указатель на ранее созданный обычный буфер, а также вершинный формат, который должен быть применен к копируемому буферу. формат, который должен быть применен к скопированным данным. Синтаксис: vertex_create_buffer_from_buffer(buffer, format); Аргумент Описание буфер Буфер для создания буфера вершин. формат Используемый формат примитивных вершин. Возвращается: Указатель Пример: vertex_format_begin(); vertex_format_add_position_3d(); vertex_format_add_colour(); vertex_format_add_textcoord(); var my_format = vertex_format_end(); v_buff = vertex_create_buffer_from_buffer(global.modelBuff, myFormat); Приведенный выше код создаст новый формат вершин, затем создаст новый буфер вершин из ранее созданного обычного буфера, применив к нему пользовательский формат вершин.
