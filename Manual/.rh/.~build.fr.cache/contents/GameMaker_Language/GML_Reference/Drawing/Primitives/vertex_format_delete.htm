вершинный_формат_удалить Эта функция должна вызываться каждый раз, когда вы закончили использовать созданные векторные форматы. Вы предоставляете значение ID формата (возвращаемое функцией vector_format_end()), и эта функция освободит память, связанную с ним. Обратите внимание, что если после вызова этой функции вы попытаетесь снова использовать этот формат, вы получите ошибку. Синтаксис: vertex_format_delete(formatID); Возвращается: Н/Д Пример: vertex_format_delete(myFormat); Приведенный выше код удалит из памяти формат вершин, созданный в переменной "myFormat".
