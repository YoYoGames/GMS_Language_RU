вершинный_создать_буфер С помощью этой функции вы можете создать новый буфер вершин. Это специальный растущий буфер, созданный GameMaker Studio 2, который предварительно отформатирован для использования при построении примитивов (например, для использования с шейдерами). Функция возвращает "Pointer" (индекс) для буфера, который затем должен использоваться во всех дальнейших обращениях к нему. При использовании буфера вершин, созданного с помощью этой функции, вы просто вызываете функцию vertex_begin() чтобы начать присваивать ему вершинные данные, чтобы начать определять ваш пользовательский примитив, который затем будет храниться в буфере готовым для передачи в шейдер. Буфер может быть использован повторно при необходимости (если вы не использовали функцию vertex_freeze() ), при этом каждый вызов vertex_begin() стирает предыдущие данные буфера, готового принять новые данные. Синтаксис: vertex_create_buffer(); Возвращается: Указатель Пример: v_buff = vertex_create_buffer(); Приведенный выше код создаст новый вершинный буфер и сохранит его хэндл в переменной "v_buff".
