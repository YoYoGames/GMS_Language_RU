нарисовать_примитивный_конец Эта функция должна быть вызвана, когда вы закончили определять точки вашего примитива. Если вы не вызовете эту функцию, ничего не будет нарисовано, так как она фактически сообщает GameMaker Studio 2, что вы закончили и что теперь она может рисовать определенный примитив. Синтаксис: draw_primitive_end() Возвращается: Н/Д Пример: draw_set_colour(c_white); var tex = sprite_get_texture(spr_Background, 0); draw_primitive_begin_texture(pr_trianglestrip, tex); draw_vertex_texture(0, 0, 0, 0); draw_vertex_texture(640, 0, 1, 0); draw_vertex_texture(0, 480, 0, 1); draw_vertex_texture(640, 480, 1, 1); draw_primitive_end(); Приведенный выше код нарисует полосу треугольника с 4 вершинами, текстурированную текстурой, хранящейся в переменной "tex".
