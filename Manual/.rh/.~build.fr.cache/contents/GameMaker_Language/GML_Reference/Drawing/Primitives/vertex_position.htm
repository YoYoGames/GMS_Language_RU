вершинная_позиция Эта функция добавит данные о двумерной позиции к вершине, которая в данный момент определена для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позиции x и y для рисования. Синтаксис: vertex_position(buffer, x, y); Аргумент Описание буфер Буфер для записи информации. x Положение x. y Положение по оси y. Возвращается: Н/Д Пример: vertex_position(buff, x - 100, y - 125); Приведенный выше код установит позицию определяемой вершины.
