вершинный_коорд Эта функция устанавливает координаты текстуры для вершины, которая в данный момент определена для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позицию UV в текстуре, которую необходимо использовать. Синтаксис: vertex_texcoord(buffer, u, v); Аргумент Описание буфер Буфер для записи информации. u Первая координата текстуры, которую следует использовать (0 - 1). v Вторая координата текстуры для использования (0 - 1). Возвращается: Н/Д Пример: vertex_texcoord(buff, 0, 0); Приведенный выше код установит значения UV для определяемой вершины.
