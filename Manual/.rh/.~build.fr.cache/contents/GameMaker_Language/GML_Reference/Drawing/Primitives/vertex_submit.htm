вершина_подтвердить Вы можете использовать эту функцию для передачи содержимого вершинного буфера в графический конвейер для использования в шейдере. Вы указываете индекс буфера, который будет использоваться, тип базового примитива (см. константы ниже) и текстуру, которая будет использоваться. базовый тип примитива используется только для назначения порядка, в котором вершины, сохраненные в буфере, рисуются и соединяются, но фактические данные, используемые для каждой из вершин, будут теми, которые вы определили при создании буфера вершин. ПРИМЕЧАНИЕ: Эта функция может быть использована только в событии Draw Event. В качестве наглядного примера различных доступных базовых примитивов см. рисунок ниже: Постоянная Описание pr_pointlist Примитив, состоящий из списка точек. pr_linelist Примитив, состоящий из отдельных строк в списке. pr_linestrip Примитив, состоящий из последовательной полосы линий. pr_trianglelist Примитив, состоящий из отдельных треугольников в списке. pr_trianglestrip Примитив, состоящий из последовательной полосы треугольников. Синтаксис: vertex_submit(buffer, primitive, texture); Аргумент Описание буфер Буфер для использования. примитивный Примитивный базовый тип. текстура Текстура, которую следует использовать (или -1 для отсутствия текстуры). Возвращается: Н/Д Пример: shader_set(shader_prim); vertex_submit(buff, pr_trianglelist, sprite_get_texture(sprite_index)); shader_reset(); Приведенный выше код передаст буфер вершин, индексированный в переменной "buff", для отрисовки шейдером " shader_prim ",используя список треугольников в качестве базового примитива и текстуру спрайта для экземпляра, выполняющего код.
