вершина_плоская1 Эта функция добавит значение с плавающей точкой к данным вершины. Вершина должна быть правильно отформатирована, чтобы принять это значение с помощью функции vertex_format_add_custom() функция. Синтаксис: vertex_float1(buffer, float); Аргумент Описание буфер Буфер для записи информации. float Входное значение. Возвращается: Н/Д Пример: vertex_float1(buff, 0.05); Приведенный выше код добавит значение с плавающей точкой к определяемым вершинным данным.
