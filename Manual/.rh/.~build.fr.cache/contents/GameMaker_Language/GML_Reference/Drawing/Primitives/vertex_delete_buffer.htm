вершинный_удаляемый_буфер Эта функция может быть использована для удаления ранее созданного вершинного буфера (см. vertex_create_buffer()) из системной памяти. При этом также удаляются все вершинные данные для любых пользовательских примитивов, которые в нем содержались. Синтаксис: vertex_delete_buffer(buffer); Аргумент Описание буфер Буфер, который необходимо удалить. Возвращается: Н/Д Пример: vertex_delete_buffer(v_buff); Приведенный выше код удалит из памяти буфер, индексированный в переменной "v_buff".
