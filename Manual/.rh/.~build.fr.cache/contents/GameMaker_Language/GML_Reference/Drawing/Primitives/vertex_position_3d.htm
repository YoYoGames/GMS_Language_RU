вершинная_позиция_3d Эта функция добавит данные о трехмерной позиции к вершине, которая в данный момент определена для пользовательского примитива. Вы предоставляете буфер для записи данных, а также позиции x, y и z для рисования. Синтаксис: vertex_position_3d(buffer, x, y, z); Аргумент Описание буфер Буфер для записи информации. x Положение x. y Положение по оси y. z Положение z. Возвращается: Н/Д Пример: vertex_position_3d(buff, x - 100, y - 125, 0); Приведенный выше код установит позицию определяемой вершины.
