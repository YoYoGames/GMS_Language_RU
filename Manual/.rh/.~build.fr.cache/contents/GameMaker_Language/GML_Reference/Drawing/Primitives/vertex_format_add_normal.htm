vertex_format_add_normal Скажите GameMaker Studio 2 принимать данные о нормалях поверхности (nx, ny и nz) как часть создаваемого нового формата вершин. Синтаксис: vertex_format_add_normal(); Возвращается: Н/Д Пример: vertex_format_begin(); vertex_format_add_textcoord(); vertex_format_add_normal(); my_format = vertex_format_end(); Приведенный выше код создаст новый формат вершин, содержащий только значения текстуры и нормали поверхности, а затем сохранит идентификатор формата в переменной "my_format".
