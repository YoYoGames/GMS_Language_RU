точка_рисунка С помощью этой функции вы можете нарисовать один пиксель в любом месте экрана, используя установленные в данный момент цвет рисунка и альфу. Синтаксис: draw_point(x, y); Аргумент Описание x Координата x точки, которая будет построена. y Координата y точки, которая будет построена. Возвращается: Н/Д Пример: draw_set_colour(c_yellow); draw_point(100,100); Это нарисует желтый пиксель в позиции (100,100).
