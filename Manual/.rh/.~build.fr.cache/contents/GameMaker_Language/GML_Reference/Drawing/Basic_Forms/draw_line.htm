нарисовать_линию С помощью этой функции вы можете нарисовать линию шириной 1 пиксель между любыми двумя точками в игровой комнате. Обратите внимание, что для того, чтобы нарисованная линия имела нужные размеры, могут потребоваться разные значения (+/-1 по x, y) из-за различий на различных поддерживаемых платформах. ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_line(x1, y1, x2, y2); Аргумент Описание x1 Координата x начала линии. y1 Координата y начала линии. x2 Координата x конца линии. y2 Координата y конца линии. Возвращается: Н/Д Пример: draw_set_colour(c_lime); draw_line(50,50,150,50); Это позволит провести светло-зеленую горизонтальную линию от точки (50,50) до точки (150,50).
