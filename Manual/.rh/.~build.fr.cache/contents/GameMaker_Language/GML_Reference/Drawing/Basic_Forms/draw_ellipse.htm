нарисовать_эллипс С помощью этой функции можно нарисовать либо контур эллипса, либо заполненный эллипс, определив прямоугольную область, под которую затем будет создан эллипс. Вы можете определить, насколько точным будет рисунок, с помощью функции draw_set_circle_precision(). ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_ellipse(x1, y1, x2, y2, контур); Аргумент Описание x1 Координата x левой части эллипса. y1 Координата y вершины эллипса. x2 Координата x правой части эллипса. y2 Координата y нижней части эллипса. конспект Будет ли эллипс рисоваться заполненным (false) или в виде контура шириной в один пиксель (true). Возвращается: Н/Д Пример: draw_set_colour(c_white); draw_ellipse(100, 100, 300, 200, false); Это позволит нарисовать заполненный эллипс в пределах заданной прямоугольной области.
