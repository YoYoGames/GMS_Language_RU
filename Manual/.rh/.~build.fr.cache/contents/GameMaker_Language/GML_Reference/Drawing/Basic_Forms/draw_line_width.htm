ширина_линии С помощью этой функции можно нарисовать линию заданной ширины между любыми двумя точками в игровой комнате. Обратите внимание, что для того, чтобы нарисованная линия имела нужные размеры, могут потребоваться различные значения (+/-1 по x, y) в связи с различиями между различных поддерживаемых платформах. ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_line_width(x1, y1, x2, y2, w); Аргумент Описание x1 Координата x начала линии. y1 Координата y начала линии. x2 Координата x конца линии. y2 Координата y конца линии. w Ширина линии в пикселях. Возвращается: Н/Д Пример: draw_set_colour(c_red); draw_line_width(100, 100, 200, 200, 6); Это позволит нарисовать красную диагональную линию шириной 6 пикселей от точки (100,100) до точки (200,200).
