draw_set_circle_precision Чтобы оптимизировать производительность при рисовании окружностей, GameMaker Studio 2 в основном рисует многоугольную форму с достаточным количеством сторон, чтобы она казалась круглой. Однако в зависимости от того, насколько большими или маленькими должны быть ваши круги, вы можете обнаружить, что изменение значения этого значения может помочь увеличить производительность вашей игры или сделать круги более привлекательными. Точное значение, которое вы можете ввести в эту функцию, - это количество сторон многоугольника окружности, которое может быть любым от 4 до 64, но должно быть числом, кратным 4, по умолчанию значение равно 24. ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_set_circle_precision(precision); Аргумент Описание точность Точность всех окружностей, нарисованных после вызова этой функции. Возвращается: Н/Д Пример: draw_set_circle_precision(12); Приведенный выше код нарисует все окружности, используя 12-сторонний многоугольник.
