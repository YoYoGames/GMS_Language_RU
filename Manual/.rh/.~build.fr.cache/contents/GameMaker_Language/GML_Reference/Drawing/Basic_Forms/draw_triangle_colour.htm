нарисовать_треугольник_цветом С помощью этой функции можно нарисовать либо контур треугольника, либо заполненный треугольник. В случае заливки можно задать индивидуальные цвета для каждой угловой точки, и если эти цвета не совпадают, то получится эффект градиента от одного к другому (настройки цвета будут перекрывать базовый цвет, заданный с помощью функции draw_set_colour()). ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_triangle_colour(x1, y1, x2, y2, x3, y3, col1, col2, col3, outline); Аргумент Описание x1 Координата x первого угла треугольника. y1 Координата y первого угла треугольника. x2 Координата x второго угла треугольника. y2 Координата y второго угла треугольника. x3 Координата x третьего угла треугольника. y3 Координата y третьего угла треугольника. кол1 Цвет первого угла. кол2 Цвет второго угла. col3 Цвет третьего угла. конспект Является ли треугольник контуром (true) или заполненным (false). Возвращается: Н/Д Пример: draw_triangle_colour(200, 200, 300, 200, 200, 300, c_red, c_blue, c_blue, false); Это позволит нарисовать заполненный равнобедренный прямоугольный треугольник с красным цветом на прямом углу и синим цветом на двух других углах.
