нарисовать_эллипс_цвет С помощью этой функции можно нарисовать либо контур эллипса, либо заполненный эллипс, определив прямоугольную область, под которую будет создан эллипс. В случае заливки можно определить внутренний и внешний цвета заливки. Если эти цвета не совпадают, вы получите эффект градиента от одного к другому (настройки цвета будут перекрывать базовый цвет, заданный с помощью функции draw_set_colour()). Вы можете определить, насколько насколько точным будет рисунок с помощью функции draw_set_circle_precision(). ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_ellipse_colour(x1, y1, x2, y2, col1, col2, outline); Аргумент Описание x1 Координата x левой части эллипса. y1 Координата y вершины эллипса. x2 Координата x правой части эллипса. y2 Координата y нижней части эллипса. кол1 Цвет центра. кол2 Цвет края. конспект Является ли эллипс контуром (true) или заполненным (false). Возвращается: Н/Д Пример: draw_ellipse_colour(50, 50, 200, 100, c_black, c_white, false); Это нарисует заполненный, горизонтально вытянутый эллипс с черным центром и белым краем.
