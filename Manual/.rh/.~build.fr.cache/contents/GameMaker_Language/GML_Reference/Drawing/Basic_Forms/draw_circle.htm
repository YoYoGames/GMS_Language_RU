нарисовать_круг С помощью этой функции можно нарисовать либо контур круга, либо заполненный круг. Вы можете определить, насколько точным будет рисунок, с помощью функции draw_set_circle_precision(). ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_circle(x, y, r, outline); Аргумент Описание x Координата x центра окружности. y Координата y центра окружности. r Радиус окружности (длина от центра до края) конспект Будет ли круг нарисован заполненным(false) или в виде контура шириной в один пиксель(true). Возвращается: Н/Д Пример: draw_set_colour(c_white); draw_circle(100, 100, 50, true); Это нарисует контур белого круга шириной один пиксель и радиусом 50 пикселей.
