нарисовать_круглый_правильный_цвет С помощью этой функции можно нарисовать либо контур закругленного прямоугольника, либо заполненный закругленный прямоугольник, где позиция (x1,y1) - левый верхний угол, а позиция (x2,y2) - правый нижний угол. Если прямоугольник заполнен, вы можете определить отдельные цвета для центра и краев, и если эти цвета не совпадают, вы получите эффект градиента от одного к другому (настройки цвета будут перекрывать базовый цвет, заданный с помощью функции draw_set_colour()).Вы Вы можете определить, насколько точным будет рисование углов с помощью функции draw_set_circle_precision()но углы всегда рисуются с фиксированным радиусом. Если вам нужно изменить радиус угла воспользуйтесь функцией draw_roundrect_colour_ext(). Обратите внимание, что для рисования прямоугольника с нужными размерами могут потребоваться разные значения (+/-1 по x, y, ширине или высоте). из-за различий между различными поддерживаемыми платформами. ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_roundrect_colour(x1, y1, x2, y2, col1, col2, outline); Аргумент Описание x1 Координата x левой части закругленного прямоугольника. y1 Координата y вершины скругленного прямоугольника. x2 Координата x правой части закругленного прямоугольника. y2 Координата y нижней части скругленного прямоугольника. кол1 Центральный цвет. кол2 Цвет внешнего края. конспект Является ли прямоугольник контуром (true) или заполненным (false). Возвращается: Н/Д Пример: draw_roundrect_colour(50, 50, 200, 200, c_black, c_white, false); Это нарисует заполненный квадрат с закругленными углами и плавным черно-белым градиентом от центра к краям.
