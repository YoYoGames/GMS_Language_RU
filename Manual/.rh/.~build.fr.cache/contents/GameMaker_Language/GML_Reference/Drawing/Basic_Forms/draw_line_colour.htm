нарисовать_линию_цвета С помощью этой функции можно нарисовать линию шириной 1 пиксель со смешением цвета между цветом 1 в первой точке и цветом 2 во второй точке. Настройки цвета отменяют базовый цвет, установленный с помощью функции draw_set_colour(). Обратите внимание, что для рисования линии с нужными размерами могут потребоваться другие значения (+/-1 по x, y) из-за различий в различных поддерживаемых платформах. ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_line_colour(x1, y1, x2, y2, col1, col2); Аргумент Описание x1 Координата x начала линии. y1 Координата y начала линии. x2 Координата x конца линии. y2 Координата y конца линии. кол1 Цвет начала линии. кол2 Цвет конца строки. Возвращается: Н/Д Пример: draw_line_colour(50, 50, 300, 50, c_red, c_blue); Это позволит нарисовать горизонтальную линию от (50,50) до (300,50) с плавным градиентом от красного к синему.
