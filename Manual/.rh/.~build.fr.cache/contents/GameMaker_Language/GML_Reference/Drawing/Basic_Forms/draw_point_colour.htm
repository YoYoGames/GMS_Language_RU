нарисовать_точку_цвета С помощью этой функции вы можете нарисовать один пиксель в любом месте экрана определенным вами цветом. Настройки цвета будут перекрывать базовый цвет, заданный с помощью функции draw_set_colour(). ПРИМЕЧАНИЕ: Если вы хотите нарисовать фигуру с помощью шейдера, вы должны знать, что большинство шейдеров ожидают следующих входных данных: вершина, текстура, цвет. Однако при использовании этой функции передаются только данные вершин и цвета, поэтому шейдер может ничего не нарисовать (или нарисовать, но не отрисовать). шейдер может ничего не нарисовать (или нарисовать, но неправильно). Если вам нужно рисовать фигуры таким образом, то шейдер должен быть настроен с учетом этого. Синтаксис: draw_point_colour(x, y, col1); Аргумент Описание x Координата x точки. y Координата y точки. кол1 Цвет точки. Возвращается: Н/Д Пример: draw_point_colour(50, 50, c_red); Это нарисует красный пиксель в точке (50,50).
