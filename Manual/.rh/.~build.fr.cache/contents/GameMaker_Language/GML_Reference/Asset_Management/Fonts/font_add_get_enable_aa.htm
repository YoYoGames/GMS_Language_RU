font_add_get_enable_aa Эта функция может быть использована для проверки того, включено ли сглаживание (AA) для шрифтов, добавленных с помощью функции font_add(). Функция вернет true, если AA включено, и false, если если нет. Обратите внимание, что AA включено по умолчанию, но вы можете изменить состояние AA для добавленных шрифтов с помощью функции font_add_enable_aa()при условии, что она будет вызвана до добавления шрифта. Синтаксис: font_add_get_enable_aa(); Возвращается: Булево Пример: if !font_add_get_enable_aa() { font_add_enable_aa(true); } Приведенный выше код проверяет состояние сглаживания для добавленных шрифтов, и если оно не включено, то мы включаем его.
