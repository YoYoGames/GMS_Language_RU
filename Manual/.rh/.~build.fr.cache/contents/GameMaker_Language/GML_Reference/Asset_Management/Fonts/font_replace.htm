font_replace Эта функция может быть использована для замены шрифта в вашей игре из тех шрифтов, которые установлены на системе, на которой она запущена. Обратите внимание, что заменяемый шрифт не может быть шрифтом из браузера активов, а должен быть шрифтом, который был добавлен ранее с помощью функции font_add(). При замене шрифта таким образом вы можете определить размер шрифта (в пунктах), а также то, должен ли шрифт быть жирным или курсивным, а также определить диапазон символов, которые необходимо включить. Если функция не сработает, то она и заданный шрифт не будет заменен. Для целевого модуля HTML5 эта функция может использоваться только для замены веб-шрифтов и будет работать только в том случае, если WebGL не включен - если WebGL включен, вы должны использовать общий ресурс шрифта (т.е. использовать draw_set_font() вместе со шрифтом из Asset Browser) или растровые шрифты (см. font_add_sprite()). ВАЖНО! При использовании HTML5 курсив и полужирный текст поддерживаются только в том случае, если в активе шрифта или в самом веб-шрифте есть эти символы. Они не будут синтезированы из базового шрифта. В следующей таблице приведены шрифты, которые являются стандартными для всех браузеров и должны быть доступны для использования без проблем. Любой другой шрифт может существовать или не существовать на компьютере или устройстве, на котором запущена ваша игра, поэтому использование этой функции должно быть ограничено этими стандартными шрифтами: Имя шрифтаWindows Имя шрифта Mac Семейство шрифтов Arial Arial / Helvetica sans-serif Arial Black Arial Black / Gadget sans-serif Comic Sans MS Comic Sans MS скоропись Курьерская новинка Курьерская новинка моношрифт Грузия Грузия засечка Влияние Удар / Древесный уголь sans-serif Консоль Lucida Монако моношрифт Lucida Sans Unicode Lucida Grande sans-serif Palatino Linotype / Book Antiqua Палатино засечка Tahoma Женева sans-serif Times New Roman Times New Roman / Times засечка Требушет МС Требушет МС sans-serif Verdana Вердана / Женева sans-serif Грузия Грузия засечка Символ Символ Н/Д Webdings Webdings Н/Д Wingdings Zapf Dingbats Н/Д MS Sans Serif Женева sans-serif MS Serif Нью-Йорк sans-serif Для всех других (не веб-) целей можно использовать эту функцию для замены шрифта из файла. Файл должен быть включен в пакет игры с помощью функции Included Files программы GameMaker Studio 2, и это должен быть файл шрифта формата *.ttf или *.otf, что удобно для добавления нестандартных шрифтов, например, азиатских или арабских. ВНИМАНИЕ!!! Если вы включаете такой файл шрифта в игру, он должен быть лицензирован для распространения вместе с игрой. При загрузке шрифта из файла таким способом размер шрифта указывается в пикселях, а первое и последнее значения игнорируются, то есть все глифы шрифта будут добавлены. Шрифты, добавленные таким образом, назначаются своей собственной текстуре странице, поэтому при использовании этой функции следует соблюдать осторожность, так как она увеличивает количество смен текстур при рисовании. Также стоит отметить, что шрифты могут выглядеть немного больше при отрисовке, так как глифы могут иметь части, которые рисуются за пределами граничного поля, определенного для шрифта. Также следует помнить, что существует ограничение на 200 глифов, которые могут быть отрисованы в одном кадре для определенного шрифта. ПРИМЕЧАНИЕ: Когда вы загружаете шрифт в GameMaker Studio 2: Studio, вы должны не забыть удалить его снова (с помощью кнопки font_delete()), когда он больше не нужен, иначе существует риск утечки памяти, что приведет к замедлению и в конечном итоге к краху вашей игры. Синтаксис: font_replace(ind, name, size, bold, italic, first, last); Аргумент Описание инд Индекс (ранее добавленного) шрифта, который необходимо заменить. имя Имя шрифта, который будет использоваться (например, 'Arial'), или путь к файлу, если шрифт является включенным *. ttf или *.otf файлом. размер Размер шрифта - пункты для веб-шрифтов, пиксели для файловых шрифтов. полужирный Является ли шрифт полужирным (true) или нет (false). курсив Является ли шрифт курсивным (true) или нет (false). первый Первый символ для включения (если вы не уверены, выбирайте 32). последний Последний символ, который нужно включить (если вы не уверены, выбирайте 128). Возвращается: Реально Пример: font_replace(myfont, "Arial", 24, true, true, 32, 128); Это заменит шрифт, указанный в переменной "myfont", на новый шрифт размером 24pt, использующий шрифт "Arial", полужирный и курсивный.
