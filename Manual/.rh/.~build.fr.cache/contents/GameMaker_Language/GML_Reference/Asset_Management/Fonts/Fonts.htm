Шрифты GameMaker Studio 2 имеет ряд функций, которые можно использовать для получения информации о включенных шрифтовых активах, а также дополнительный набор функций, которые можно использовать для изменения, добавления или удаления шрифтов из вашей игры во время выполнения. В частности, функция font_add() может быть использована для добавления нестандартных шрифтов, например, азиатских глифов, из файла *.ttf или *.otf. Стоит отметить, что если вы добавите в игру шрифт, который не имеет глифов, необходимых для текста, который вы хотите написать, то символ Unicode 9647 (▯) используется для замены недостающих глифов при рендеринге в событии draw. Таким образом, если в вашем шрифте нет, например, символа º, то при написании 90º в действительности получится 90▯. Для получения информации о ресурсах шрифта можно использовать следующие функции: font_get_name font_get_fontname font_get_first font_get_last font_get_italic font_get_bold font_get_size font_get_texture font_get_uvs font_get_info Следующие функции можно использовать для изменения и манипулирования шрифтами во время игры: font_exists font_set_cache_size font_add_enable_aa font_add_get_enable_aa font_add font_add_sprite font_add_sprite_ext font_replace font_replace_sprite font_replace_sprite_ext font_delete размер_текстуры_страницы Назад: Управление активами Следующее: Временные шкалы © Copyright YoYo Games Ltd. 2021 Все права защищены
