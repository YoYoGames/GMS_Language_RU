font_add_sprite_ext С помощью этой функции вы можете использовать "полосу спрайта" (сам спрайт должен быть активом спрайта из браузера активов или спрайтом, который вы добавили в ресурсы игры с помощью sprite_add ) для создания нового актива шрифта, где каждое подизображение будет отдельным символом или буквой. В отличие от обычного шрифта font_add_sprite() которая имеет определенный порядок для суб-изображений спрайта, эта функция отображает суб-изображения спрайта на основе аргумента "string_map" функции. Этот аргумент представляет собой строку, которую вы можете использовать, чтобы сообщить GameMaker Studio 2, в каком порядке расположены суб-изображения шрифта спрайта, и она будет автоматически отображать их при написании текста. Так, например, если у вас есть карта строк "AaBbCcDdEeFfGgHh", ваш спрайтовый шрифт должен иметь суб-изображения, упорядоченные таким образом. Вы можете определить "пробел" как любой символ, например, одну строку такого размера, какого вы хотите, чтобы был пробел, и все пробелы в тексте будут отображаться с такой шириной (но выбранное изображение никогда не будет отображаться), или, если вы не указали спрайт для пробела, то вместо него будет использоваться ширина самого широкого символа в шрифте спрайта. Вы также можете установить пропорциональное(true) или непропорциональное(false) расстояние между буквами шрифта. При пропорциональном шрифте расстояние между буквами определяется их фактической шириной (например, буква "i" занимает меньше места, чем буква "w", в то время как непропорциональный шрифт будет размещаться на основе ширины суб-изображения, как моноширинный шрифт. Наконец, вы можете определить пространство, которое нужно оставить между каждой буквой при написании, и это может быть любое целое значение, при этом 0 означает отсутствие пространства (буквы будут "соприкасаться", если шрифт пропорциональный). буквы будут "соприкасаться", если они пропорциональны). Функция возвращает значение индекса, которое должно быть сохранено в переменной, так как оно будет необходимо во всех дальнейших кодах, которые будут ссылаться на этот шрифт, или возвращает -1, если функция не работает по какой-либо причине. причине. Стоит отметить, что для работы функций выравнивания шрифта (например, таких как draw_set_halign()), у спрайта шрифта должно быть установлено начало координат в левом верхнем углу. Если вы используете другие значения то при рисовании текста с использованием шрифта необходимо учитывать смещение начала координат. ПРИМЕЧАНИЕ: Когда вы загружаете шрифт в GameMaker Studio 2, вы должны не забыть удалить его снова (с помощью функции font_delete()), когда он больше не нужен, иначе существует риск утечки памяти, что приведет к замедлению и в конечном итоге к краху вашей игры. Синтаксис: font_add_sprite_ext(spr, string_map, prop, sep); Аргумент Описание spr Спрайт, на основе которого будет добавлен шрифт. карта строк Строка, из которой берется порядок суб-изображений спрайта. реквизит Установить пропорциональный шрифт или нет. сеп Пробел, который нужно оставить между каждой буквой. Возвращается: Реально Пример: global.Font = font_add_sprite_ext(spr_CalcFont, "0123456789+-*/=", true, 2); Приведенный выше код создаст новый актив шрифта из спрайта, индексированного в переменной "spr_CalcFont", и сохранит индекс нового шрифта в переменной "global.Font". Вложенные изображения шрифта спрайта будут отображены на строку указанной. Назад: Шрифты Далее: font_replace © Copyright YoYo Games Ltd. 2021 Все права защищены
