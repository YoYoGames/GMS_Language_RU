font_get_bold С помощью этой функции вы можете проверить любой актив шрифта на наличие или отсутствие у него флага полужирного начертания. Если есть, то функция вернет true, в противном случае - false. Синтаксис: font_get_bold(ind); Аргумент Описание инд Индекс шрифта для проверки. Возвращается: Булево Пример: if font_get_bold(fnt_Main) { draw_set_font(fnt_Main); } Это позволит установить активный шрифт рисунка на fnt_Main, если в свойствах шрифта он установлен как полужирный.
