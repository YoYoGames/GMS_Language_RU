font_delete С помощью этой функции вы можете удалить актив шрифта из игры. Это постоянное удаление, и смена комнаты или перезапуск игры не вернет удаленный шрифт обратно. Для этого игроку придется выйти из игры и перезапустить игру, поэтому будьте осторожны при использовании этой функции. В целом, она нужна только для освобождения памяти, которая использовалась шрифтом, добавленным в игру с помощью функций font_add() или font_add_sprite(). Синтаксис: font_delete(ind); Аргумент Описание инд Индекс шрифта для удаления. Возвращается: Н/Д Пример: font_delete(global.Font); Приведенный выше код удалит шрифт, индексированный в глобальной переменной "Font".
