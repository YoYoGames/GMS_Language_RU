font_add_enable_aa Эта функция может быть использована для включения или отключения сглаживания (AA) для шрифтов, добавленных с помощью функции font_add(). Эта функция должна быть вызвана до добавления шрифтов, и установка значения true включит AA, а установка false отключит его. По умолчанию AA включено. Обратите внимание, что это не повлияет на шрифты, которые были добавлены до вызова функции и функция должна вызываться только один раз при добавлении шрифта, а не на каждом отрисовке/шаге, когда шрифт используется. Синтаксис: font_add_enable_aa(enable); Аргумент Описание включить Включать(true) или выключать(false) АА для добавленных шрифтов. Возвращается: Н/Д Пример: if !font_add_get_enable_aa() { font_add_enable_aa(true); } Приведенный выше код проверяет состояние сглаживания для добавленных шрифтов, и если оно не включено, то мы включаем его.
