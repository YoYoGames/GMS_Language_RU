font_get_italic С помощью этой функции вы можете проверить любой актив шрифта на наличие или отсутствие флага курсива. Если есть, то функция вернет true, в противном случае - false. Синтаксис: font_get_italic(ind); Аргумент Описание инд Индекс шрифта для проверки. Возвращается: Булево Пример: if font_get_italic(fnt_Main) { draw_set_font(fnt_Main); } Это позволит установить активный шрифт рисунка на fnt_Main, если в свойствах шрифта он установлен как курсив.
