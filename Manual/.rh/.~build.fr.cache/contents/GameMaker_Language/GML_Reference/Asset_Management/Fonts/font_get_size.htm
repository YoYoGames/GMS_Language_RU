font_get_size С помощью этой функции можно получить размер любого ресурса шрифта, который представляет собой значение пункта, отображаемое в диалоге ресурса шрифта. Синтаксис: font_get_size(ind); Аргумент Описание инд Индекс шрифта для получения размера. Возвращается: Реально Пример: sz = font_get_size(font0); Это позволит получить размер шрифта, индексируемого переменной "font0", и сохранить его в переменной "sz".
