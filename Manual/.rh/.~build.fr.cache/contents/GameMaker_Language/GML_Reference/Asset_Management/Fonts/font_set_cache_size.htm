font_set_cache_size При выполнении смешивания и раскрашивания изображений HTML5 не может делать это динамически так же, как это может делать исполняемый файл. Поэтому GameMaker Studio 2 приходится временно сохранять копию смешивания изображений и загружать ее при необходимости. и загружать их по мере необходимости. Эта функция устанавливает, сколько смешанных копий данного шрифта можно кэшировать, прежде чем старые копии будут перезаписаны. Значение по умолчанию равно 4. ПРИМЕЧАНИЕ: для спрайтовых шрифтов вы должны использовать эквивалентную функцию для спрайтов, sprite_set_cache_size(). Синтаксис: font_set_cache_size(ind, max); Аргумент Описание инд Индекс шрифта для изменения размера кэша. max Максимальное количество кэшированных копий шрифта, которое может быть сохранено. Возвращается: Н/Д Пример: font_set_cache_size(fnt_MainMenu, 2); Это установит кэш шрифта, индексированного в переменной "fnt_MainMenu", в 2 копии.
