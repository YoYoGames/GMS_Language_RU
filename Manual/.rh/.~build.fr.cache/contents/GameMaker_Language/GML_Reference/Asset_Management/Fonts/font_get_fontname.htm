font_get_fontname С помощью этой функции можно получить фактическое системное имя выбранного актива шрифта. Эта функция возвращает строку, а не индекс, и возвращаемое имя будет зависеть от используемого шрифта. Например, у вас может быть актив шрифта под названием "fnt_Main" в GameMaker Studio 2 Asset Browser, а сам шрифт может быть системным шрифтом Arial в Windows. В этом случае функция вернет значение "Arial". поскольку это системное название шрифта. Если вам нужно название в том виде, в котором оно отображается в браузере ассетов, используйте font_get_name(). Синтаксис: font_get_fontname(ind); Аргумент Описание инд Индекс шрифта для проверки. Возвращается: Строка Пример: fnt_Name = font_get_fontname(font0); Приведенный выше код получит системное имя ресурса шрифта и сохранит его в виде строки в переменной "fnt_Name".
