font_get_uvs Эта функция возвращает массив с UV-координатами для текстуры шрифта на странице текстуры, заполняя массив следующими значениями: [0] = левый [1] = верх [2] = право [3] = низ Это значение затем может быть использовано в других функциях рисования, особенно в общем рисовании при использовании примитивов, а также функций Шейдер функции. Синтаксис: font_get_uvs(font); Аргумент Описание шрифт Индекс используемого шрифта. Возвращается: Массив Пример: var tex = font_get_uvs(fnt_Main); tex_left = tex[0]; tex_top = tex[1]; tex_right = tex[2]; tex_left = tex[3]; Приведенный выше код сохранит УФ-координаты для заданного фона в локальном массиве, а затем присвоит значения переменным экземпляра.
