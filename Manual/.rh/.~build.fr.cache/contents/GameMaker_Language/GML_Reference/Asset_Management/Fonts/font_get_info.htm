font_get_info Эта функция используется для получения информации о заданном шрифте. Вы задаете индекс шрифта (это может быть актив, добавленный через Asset Browser, или шрифт, добавленный во время выполнения), и функция возвращает структуру со следующими переменными: Имя переменной Тип данных Описание ascenderOffset настоящий Максимальное смещение от базовой линии к вершине шрифта (в пикселях). размер настоящий Приблизительный размер шрифта (в пикселях) spriteIndex актив (реальный) Индекс спрайта для шрифта, если он был создан из спрайта, иначе -1 текстура актив (реальный) -1, если шрифт был создан из спрайта, иначе - идентификатор текстуры шрифта имя строка Название шрифта полужирный булево true, если шрифт полужирный, иначе false (см. ПРИМЕЧАНИЕ ниже) курсив булево true, если шрифт курсивный, иначе false (см. ПРИМЕЧАНИЕ ниже) глифы struct Структура, содержащая информацию о каждом глифе в шрифте (более подробная информация приведена ниже) ПРИМЕЧАНИЕ: Переменные полужирный шрифт икурсив отражают только настройки пользователя для шрифта, поэтому они могут быть неточными для шрифтов, которые по умолчанию являются жирными или курсивными. Если указанный шрифт не существует, функция вернет неопределенное значение . Также обратите внимание, что изменение значений любой из этих переменных или любых переменных, содержащихся в структуре glyphs, не изменит отображение шрифта, поэтому эту информацию следует рассматривать как информацию только для чтения. Структура глифов Переменная glyphs в возвращаемой структуре будет самостоятельной структурой, содержащей информацию о каждом глифе, включенном в шрифт. Каждая переменная в этой структуре будет именем символа глифа, который будет содержать структуру, содержащую информацию для этого глифа. Вы можете представить эти вложенные структуры в следующем формате (начиная с главной структуры): { глифы: { A: {}, B: {}, C: {}, // ...другие глифы }, // ...другая информация о шрифте } Вы можете использовать имя символа глифа для получения информации о нем из структуры, как, например, в следующем коде: var _font_info = font_get_info(Font1); var _info_A = _font_info.glyphs[$ "A"]; // $ - это аксессор структуры show_debug_message(_info_A); Приведенный выше код получает глиф-структуру для символа "A" и печатает ее в выходной журнал. Вы также можете заменить строку ("A") переменной, содержащей строку, что позволит вам получить информацию для любого глифа через эту переменную во время выполнения. Структура, возвращаемая для глифа, будет содержать следующие переменные: ПРИМЕЧАНИЕ: В HTML5 структура glyphs будет пустой для любых загруженных файловых шрифтов (таких как *.ttf или *.otf), поскольку среда выполнения не будет иметь информации об этих глифах. Имя переменной Тип данных Описание char настоящий Если шрифт был создан из спрайта, это будет индекс изображения глифа из этого спрайта, в противном случае это будет номер символа Unicode. Примечание: Все переменные ниже этой не будут присутствовать в структуре, если шрифт был создан из спрайта. x настоящий X позиция глифа на странице текстуры (в текселях). y настоящий Положение глифа по оси Y на странице текстуры (в текселях) w настоящий Ширина глифа на странице текстуры (в текселях). h настоящий Высота глифа на странице текстуры (в текселях) смена настоящий Количество пикселей для сдвига вправо при переходе к следующему символу (может быть отрицательным для сдвига влево). смещение настоящий Количество пикселей для горизонтального смещения рендеринга этого глифа (может быть положительным или отрицательным). кернинг массив Массив целых чисел, содержащий информацию о кернинге в парах (или группах по 2). Первое целое число в паре - это значение Unicode для символа, а второе целое число - сумма, которую нужно добавить к значению сдвига этого символа (может быть положительным или отрицательное), если он предшествует символу данного глифа. ПРИМЕЧАНИЕ: Для файловых шрифтов, загружаемых во время выполнения, значения x и y для глифа могут быть равны -1, если он еще не был отрисован, поскольку у него не будет позиции на странице текстуры, используемой для кэширования. Синтаксис: font_get_info(font); Аргумент Описание шрифт Шрифт, для которого необходимо получить информацию. Возвращается: Структура (или неопределенная) Пример: var _info = font_get_info(my_font); if (_info != undefined && _info.bold) { image_xscale *= 2; image_yscale *= 2; } Этот код извлекает информацию о шрифте, хранящемся в переменной my_font. Если возвращаемое значение не являетсянеопределенным и данный шрифт является полужирным, масштаб экземпляра удваивается.
