shader_set_uniform_f_array С помощью этой функции можно установить константу шейдера для хранения массива значений. Вы должны предварительно получить "хэндл" константы с помощью функции shader_get_uniform()и вы должны предварительно инициализировать массив. ПРИМЕЧАНИЕ: Все формы должны быть установлены после вызова функции shader_set(), и до вызова shader_reset(). Синтаксис: shader_set_uniform_f_array(handle, array); Аргумент Описание ручка Ручка константы шейдера, которую нужно установить. массив Предварительно инициализированный массив значений с плавающей точкой. Возвращается: Н/Д Пример: shader_set(shader_Glass); tex_array[0] = 0.5; tex_array[1] = 0.1; tex_array[2] = 0.25; shader_params= shader_get_uniform(shader_glass, "u_vParams"); shader_set_uniform_f_array(shader_params, tex_array); draw_self(); shader_reset(); Приведенный выше код получит хэндл константы шейдера "u_vParams", а затем установит эту константу в заданный массив.
