shader_get_name С помощью этой функции можно получить имя ресурса шейдера. Вы вводите уникальное значение ID шейдера, имя которого нужно получить, и функция возвращает имя ресурса в виде строки. Синтаксис: shader_get_name(shader); Аргумент Описание шейдер Индекс шейдера, имя которого нужно получить. Возвращается: Строка Пример: var _shader = shader_current(); var _name = shader_get_name(_shader); draw_text(32, 32, "Debug - Currently Rendering = " + _name); Приведенный выше код получит имя заданного шейдера и нарисует его на экране.
