shader_set_uniform_i С помощью этой функции вы можете установить константу шейдера для хранения массива значений. Предварительно вы должны получить "хэндл" константы с помощью функции shader_get_uniform()и необходимо предварительно инициализировать массив. ПРИМЕЧАНИЕ: Все формы должны быть установлены после вызова функции shader_set()и до вызова shader_reset(). Синтаксис: shader_set_uniform_i_array(handle, array); Аргумент Описание ручка Ручка константы шейдера, которую нужно установить. массив Предварительно инициализированный массив целочисленных значений. Возвращается: Н/Д Пример: shader_set(shader_Glass); col_array[0] = 255; col_array[2] = 255; col_array[3] = 64; col_array[4] = 128; shader_params= shader_get_uniform(shader_tint, "cColourArray"); shader_set_uniform_i_array(shader_params, col_array); draw_self(); shader_reset(); Приведенный выше код получит хэндл константы шейдера "cColourArray", а затем установит эту константу в заданный массив.
