shader_get_sampler_index Поскольку значение шейдерного сэмплера нельзя изменить в самом шейдере, его нужно задать до вызова шейдера с помощью одной из доступных функций uniform set. Однако, чтобы сделать это, вы необходимо сначала вызвать эту функцию, чтобы получить "хэндл" шейдерного сэмплера, который вы хотите установить. ПРИМЕЧАНИЕ: Хотя шейдер состоит из двух отдельных программ (вершинной и фрагментной), эта функция не делает различий между ними и возвращает хэндл образца шейдера из любой из них. Синтаксис: shader_get_sampler_index(shader, uniform); Аргумент Описание шейдер Индекс шейдера, который нужно использовать. униформа Сэмплер шейдера для получения ручки (строка). Возвращается: Реально Пример: glassshader_bgsampler = shader_get_sampler_index(GlassShader, "s_BackgroundSampler") spr = sprite_get_texture(sprite_index, 0); shader_set (GlassShader); texture_set_stage(glassshader_bgsampler, spr); shader_reset(); Приведенный выше код получит хэндл для сэмплера в шейдере с индексом "GlassShader", а затем установит константу этого шейдера на заданную текстуру спрайта.
