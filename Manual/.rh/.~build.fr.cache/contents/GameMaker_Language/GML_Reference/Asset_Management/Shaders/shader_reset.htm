shader_set_uniform_matrix_array Эта функция сбрасывает цель рисования и должна вызываться, когда вы больше не хотите использовать текущий шейдер (установленный с помощью функции shader_set()). Синтаксис: shader_reset(); Возвращается: Н/Д Пример: shader_set(shader_Glass); draw_self(); shader_reset(); Приведенный выше код установит вершину шейдера, используемую для отрисовки, а затем отрисует текущий спрайт, используемый для экземпляра с помощью этого шейдера.
