shader_set_uniform_matrix С помощью этой функции вы можете установить значение (или значения) константы шейдера на текущую матрицу преобразования (заданную с помощью матричных функций). Предварительно вы должны получить "хэндл" константы с помощью функции shader_get_uniform(). ПРИМЕЧАНИЕ: Все формы должны быть установлены после вызова функции shader_set()и до вызова shader_reset(). Синтаксис: shader_set_uniform_matrix(handle); Аргумент Описание ручка Ручка константы шейдера, которую нужно установить. Возвращается: Н/Д Пример: shader_set(shader_Glass); shader_matrix= shader_get_uniform(shader_glass, "u_vMatrix"); shader_set_uniform_matrix(shader_matrix); draw_self(); shader_reset(); Приведенный выше код получит хэндл константы шейдера "u_vMatrix", а затем установит эту константу в текущую матрицу преобразования.
