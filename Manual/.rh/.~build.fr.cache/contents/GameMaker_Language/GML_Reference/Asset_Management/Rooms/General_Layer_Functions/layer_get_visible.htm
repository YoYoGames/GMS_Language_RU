layer_get_visible С помощью этой функции можно проверить, является ли слой видимым или нет. Вы передаете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция вернет true, если слой виден. и функция вернет true, если слой виден, и false в противном случае. Синтаксис: layer_get_visible(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). Возвращается: Булево Пример: var lay_id = layer_get_id("Instances"); if layer_get_visible(lay_id) { layer_set_visible(lay_id, false); } else { layer_set_visible(lay_id, true); } Приведенный выше код получает значение ID для слоя с именем "Instances" в редакторе помещений, затем использует ID для проверки видимости слоя, переключая видимость слоя в зависимости от возвращаемого значения.
