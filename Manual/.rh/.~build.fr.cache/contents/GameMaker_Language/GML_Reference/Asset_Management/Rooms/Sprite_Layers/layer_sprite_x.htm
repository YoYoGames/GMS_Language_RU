слой_спрайта_x Эта функция управляет положением вдоль оси x комнаты элемента спрайта актива на слое. Вы передаете ID элемента спрайта (который вы получаете, когда создаете элемент спрайта с помощью команды layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете используемое значение x (на основе координат комнаты). Синтаксис: layer_sprite_x(sprite_element_id, x); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для изменения x Позиция x для спрайта Возвращается: Н/Д Пример: var lay_id = layer_get_id("Asset_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); layer_sprite_x(spr_id, irandom(room_width)); Приведенный выше код получает значение ID спрайта актива "Clouds", назначенного слою "Asset_sky", а затем устанавливает его x-позицию в случайное значение между 0 и шириной комнаты.
