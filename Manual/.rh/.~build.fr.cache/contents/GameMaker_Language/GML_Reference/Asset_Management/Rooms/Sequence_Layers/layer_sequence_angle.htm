угол_последовательности_слоя С помощью этой функции можно установить угол поворота для заданного элемента последовательности. Вы передаете идентификатор элемента последовательности, возвращаемый функцией layer_sequence_create() или одной из функций элемента слоя вместе с новым углом, который нужно задать, и последовательность будет повернута вокруг своего начала в новое положение. Углы в GameMaker отсчитываются от 0º вправо и идут против часовой стрелки - 90º вверх, 180º влево и 270º вниз. для последовательности будет 0º. Обратите внимание, что вы не должны изменять переменные image_xscale \ image_yscale \ image_angle \ x \ y для любых экземпляров в последовательности, использующей эту функцию, так как они будут перезаписаны, как только последовательность обновится после вызова функции. Синтаксис: layer_sequence_angle(sequence_element_id, angle) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться угол Новый угол, на который нужно повернуть элемент последовательности Возвращается: Н/Д Пример: if current_angle < 90 { current_angle += 1; layer_sequence_angle(my_seq, current_angle); } Приведенный выше код проверяет значение, хранящееся в переменной current_angle, и если оно меньше 90, то прибавляет к нему значение, а затем использует его для установки угла элемента последовательности, на который ссылается переменная my_seq.
