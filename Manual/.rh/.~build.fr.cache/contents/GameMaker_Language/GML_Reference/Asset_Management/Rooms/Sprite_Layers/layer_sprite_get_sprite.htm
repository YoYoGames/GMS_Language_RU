layer_sprite_get_sprite Эта функция может быть использована для получения текущего индекса спрайта элемента спрайта. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или когда вы используете функция layer_sprite_get_id()), и функция вернет вещественное значение, представляющее индекс отображаемого спрайта. Если элементу не назначен спрайт, функция вернет -1. Синтаксис: layer_sprite_get_sprite(sprite_element_id); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию Возвращается: Индекс спрайта (реальный) или -1 Пример: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_sprite(spr_id) != spr_Clouds { layer_sprite_change(spr_id, spr_Clouds); } Приведенный выше код получит идентификатор слоя для слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот ID затем используется для проверки спрайта, назначенного этому элементу, и установки его на спрайт "spr_Clouds". если он еще не установлен.
