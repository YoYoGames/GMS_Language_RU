layer_get_id_at_depth Вы можете использовать эту функцию для получения идентификаторов всех слоев, которым присвоена определенная глубина. Вы указываете глубину, которую нужно проверить, и функция возвращает массив с 1 или более элементами в зависимости от того, существуют ли слои на заданной глубине или нет. Если на заданной глубине нет слоев, то в массиве будет одна запись в позиции [0] со значением -1, но если на глубине есть слои, то в массиве будет сделана запись для каждого найденного слоя. слоя - значение записи будет уникальным значением ID для слоя. Синтаксис: layer_get_id_at_depth(depth) Аргумент Описание глубина Глубина для проверки и извлечения идентификаторов слоев. Возвращается: Массив Пример: var a = layer_get_id_at_depth(0); if a[0] != -1 { for (var i = 0; i < array_length(a); i++;) { layer_destroy(a[i]); } } Приведенный выше код извлекает данные о слоях с глубиной 0. Выполняется проверка, существуют ли слои с такой глубиной, и если существуют, то возвращаемый массив разбирается и каждый из найденных слоев уничтожается.
