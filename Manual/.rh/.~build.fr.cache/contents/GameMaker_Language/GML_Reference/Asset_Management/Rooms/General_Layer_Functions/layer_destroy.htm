уничтожение слоя Эта функция уничтожит заданный слой. Вы передаете ID слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и эта функция удалит его из текущей комнаты. Если слой был создан в редакторе комнат, то при следующем выходе из комнаты и последующем возвращении слой будет создан заново с исходным содержимым, однако если комната постоянна, слой будет удален, если только постоянство помещения не будет снова отключено. При уничтожении слоя таким образом все его содержимое также будет удалено, поэтому любые идентификаторы ссылок для фонов, карт плиток и т.д. больше не будут действительны, а все экземпляры, назначенные слою, будут уничтожены. назначенные слою, будут уничтожены (одновременно с этим будет выполнено событие Destroy, если оно у них есть, а также событие Clean Up). Синтаксис: layer_destroy(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя для уничтожения (или имя слоя в виде строки) Возвращается: Н/Д Пример: if !instance_exists(obj_Bullet_Parent) { layer_destroy(global.Bullet_Layer); } Приведенный выше код проверит, существуют ли экземпляры объекта "obj_Bullet_Parent", и если нет, то уничтожит слой с идентификатором, сохраненным в глобальной переменной.
