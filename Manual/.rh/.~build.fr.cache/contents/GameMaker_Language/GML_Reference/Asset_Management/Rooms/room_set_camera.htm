комната_установить_камеру С помощью этой функции вы можете назначить камеру на определенный видовой экран в помещении, отличном от текущего. Вы вводите индекс помещения, индекс вида (от 0 до 7), а затем индекс камеры, которую нужно использовать. Синтаксис: room_set_camera(rm, vind, camera); Аргумент Описание rm Индекс комнаты для установки камеры просмотра vind Индекс порта просмотра, на который назначается камера камера Индекс камеры для назначения Возвращается: Н/Д Пример: global.myroom = room_add(); room_set_camera(global.myroom, 0, global.MainCam); Приведенный выше код назначает камеру во вновь созданной комнате на порт просмотра [0].
