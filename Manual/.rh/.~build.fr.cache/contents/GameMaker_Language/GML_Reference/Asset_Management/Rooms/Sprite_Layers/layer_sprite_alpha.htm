слой_спрайта_альфа Эта функция управляет альфа (прозрачностью) спрайта на слое актива. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функция layer_sprite_get_id()), а затем задаете используемое значение альфа. Альфа может быть от 0 (полностью прозрачный) до 1 (полностью непрозрачный), по умолчанию значение альфа для элемента спрайта равно 1. Обратите внимание что если слой, которому назначен элемент спрайта, не виден, или сам элемент был сделан невидимым, вы не увидите никакой разницы с помощью этой функции, пока слой или элемент снова не станут видимыми. Синтаксис: layer_sprite_alpha(sprite_element_id, alpha); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для изменения альфа Альфа для спрайта спрайта, от 0 до 1 (по умолчанию 1) Возвращается: Н/Д Пример: var lay_id = layer_get_id("Asset_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); layer_sprite_alpha(spr_id, random(1)); Приведенный выше код получает значение ID актива спрайта с именем "Clouds", назначенного слою "Asset_sky", а затем устанавливает его альфа-фактор на случайное значение между 0 и 1.
