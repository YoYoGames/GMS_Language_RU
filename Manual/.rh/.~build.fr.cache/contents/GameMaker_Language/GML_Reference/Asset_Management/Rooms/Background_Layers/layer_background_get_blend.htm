layer_background_get_blend Эта функция может быть использована для получения цвета смешивания фонового элемента. Вы передаете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет реальное значение, представляющее назначенный цвет. Синтаксис: layer_background_get_blend(background_element_id); Аргумент Описание background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию Возвращается: Значение цвета (реальное) Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_blend(back_id) == c_white { layer_background_blend(back_id, make_colour_rgb(random(255), random(255), 255)); } Приведенный выше код получит идентификатор слоя для слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки цвета наложения элемента, и если он равен константе c_white, то наложение слоя устанавливается на случайный цвет.
