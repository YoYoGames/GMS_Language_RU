layer_sprite_get_xscale Эта функция может быть использована для получения текущего множителя масштаба по оси x для элемента спрайта. Вы передаете идентификатор элемента спрайта (который вы получаете при создании элемента спрайта с помощью команды layer_sprite_create() или при использовании функции layer_sprite_get_id()), и функция вернет реальное значение для масштаба x, где 1 - нет масштабирования. Синтаксис: layer_sprite_get_xscale(sprite_element_id); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию Возвращается: Реально Пример: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_xscale(spr_id) != 1 { layer_sprite_xscale(spr_id, 1); } Приведенный выше код получит идентификатор слоя для слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот идентификатор затем используется для проверки значения xscale, и если оно не равно 1, то устанавливается равным 1.
