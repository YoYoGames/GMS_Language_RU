имя_слоя Вы можете использовать эту функцию для получения имени заданного слоя. Вы передаете уникальное значение ID слоя, и если слой является одним из именованных слоев, созданных в редакторе помещений, то функция вернет строку с именем слоя. Если слой не является одним из слоев, созданных в редакторе помещений (т.е. он был создан с помощью функции layer_create()), то будет возвращена пустая строка. Синтаксис: layer_get_name(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, имя которого необходимо получить Возвращается: Строка Пример: var a = layer_get_all(); var layer_list = ds_list_create(); for (var i = 0; i Alt; array_length(a); i++;) { if layer_get_name(a[i]) != "" { ds_list_add(layer_list, a[i]) } } Приведенный выше код получает идентификаторы всех слоев в комнате и затем циклически проверяет, являются ли они именованными слоями.
