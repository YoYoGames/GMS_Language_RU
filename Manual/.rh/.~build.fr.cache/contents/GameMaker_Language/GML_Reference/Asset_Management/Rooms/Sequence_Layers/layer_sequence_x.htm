layer_sequence_x С помощью этой функции можно задать положение вдоль оси X (горизонтальной) комнаты для данного элемента последовательности. Вы передаете идентификатор элемента последовательности, возвращаемый функцией layer_sequence_create() или одной из функции элемента слоя вместе с заданной позицией X, и последовательность будет перемещена в новую позицию. Синтаксис: layer_sequence_x(sequence_element_id, pos_x) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться позиция_x Позиция X для перемещения элемента последовательности Возвращается: Н/Д Пример: if layer_sequence_exists(my_seq) { layer_sequence_x(my_seq, x); layer_sequence_y(my_seq, y); } Приведенный выше код проверяет, существует ли элемент последовательности, на который ссылается переменная "my_seq", и если да, то он устанавливает позицию x/y на позицию экземпляра, выполняющего код.
