room_instance_add С помощью этой функции вы можете добавить экземпляр в любую комнату, кроме текущей, и в любую позицию внутри этой комнаты. Функция возвращает уникальный id экземпляра, который затем может использовать в дальнейших вызовах функций для установки свойств и т.д... этого экземпляра, но только после того, как игра войдет в указанную комнату. Если вы хотите создать экземпляр в текущей комнате, вы должны использовать функцию instance_create_layer(). Обратите внимание что вызов этой функции на активе комнаты, созданном в браузере активов, навсегда добавит экземпляр в комнату, и даже вызов game_restart() не вернет комнату в исходное состояние. (только завершение игры и ее повторное открытие приведет к тому, что комната снова окажется в исходном состоянии). Синтаксис: room_instance_add(index, x, y, obj); Аргумент Описание индекс Индекс комнаты, в которую нужно добавить экземпляр объекта. x Позиция x нового экземпляра. y Позиция y нового экземпляра. obj Объект для добавления экземпляра. Возвращается: Идентификатор экземпляра Пример: global.rm = room_add(); room_assign(rm_Base, global.rm); room_instance_add(global.rm, 100, 100, obj_Player); Приведенный выше код добавит в игру новую комнату, а затем скопирует в нее содержимое комнаты, проиндексированной как "rm_Base". Затем он добавит экземпляр объекта "obj_player" в позицию 100,100 этой новой комнаты. Назад: Номера Далее: room_instance_clear © Copyright YoYo Games Ltd. 2021 Все права защищены
