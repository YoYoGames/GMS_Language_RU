layer_sequence_get_instance С помощью этой функции вы передаете идентификатор элемента последовательности - как возвращено функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет структуру экземпляра последовательности. Подробнее о формате структуры экземпляра последовательности вы можете узнать на этой странице. Синтаксис: layer_sequence_get_instance(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Структура Пример: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); var _struct = layer_sequence_get_instance(_seq); _struct.speedScale = 0.5; Приведенный выше код создает новый элемент последовательности, а затем извлекает идентификатор экземпляра последовательности. Затем он используется для изменения свойства шкалы скорости воспроизведения экземпляра последовательности.
