layer_get_x Вы можете использовать эту функцию для получения x-позиции слоя в текущем масштабируемом помещении. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (как строка - это влияет на производительность), и функция возвращает вещественное число для x-положения слоя относительно (0,0) положения комнаты. По умолчанию - 0. Синтаксис: layer_get_x(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя для получения положения x Возвращается: Реально Пример: var lay_id = layer_get_id("Sprites"); if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0 { layer_x(lay_id, 0); layer_y(lay_id, 0); } Приведенный выше код проверяет заданную позицию слоя, и если она не установлена в (0, 0), то она устанавливается в эту позицию.
