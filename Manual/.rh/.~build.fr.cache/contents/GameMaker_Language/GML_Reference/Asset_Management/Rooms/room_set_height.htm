высота_комнаты С помощью этой функции вы можете изменить (или установить) высоту любой комнаты в вашей игре, кроме текущей. Синтаксис: room_set_height(index, h); Аргумент Описание индекс Индекс помещения для установки высоты. h Новая высота помещения в пикселях. Возвращается: Н/Д Пример: global.myroom = room_add(); room_set_width(global.myroom, 640); room_set_height(global.myroom, 480); room_set_persistent(global.myroom, false); Это создаст новую комнату и сохранит ее индекс в переменной "global.myroom". Затем она установит ее ширину 640 пикселей, высоту 480 пикселей, надпись 'Game Room' и постоянство 'false'. Назад: Номера Далее: room_set_width © Copyright YoYo Games Ltd. 2021 Все права защищены
