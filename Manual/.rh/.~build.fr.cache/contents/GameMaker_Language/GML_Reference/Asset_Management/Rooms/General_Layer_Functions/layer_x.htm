слой_x Вы можете использовать эту функцию для установки x-позиции слоя в пределах текущего масштабируемого помещения. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это повлияет на производительность), и функция переместит слой на заданное количество пикселей вдоль горизонтальной оси комнаты. Синтаксис: layer_x(layer_id, x) Аргумент Описание идентификатор слоя Уникальное значение ID слоя для установки положения x x Позиция x в помещении, на которую нужно установить слой Возвращается: Н/Д Пример: var lay_id = layer_get_id("Sprites"); if layer_get_x(lay_id) != 0 || layer_get_y(lay_id) != 0 { layer_x(lay_id, 0); layer_y(lay_id, 0); } Приведенный выше код проверяет заданную позицию слоя, и если она не установлена в (0, 0), то она устанавливается в эту позицию.
