layer_background_exists С помощью этой функции можно проверить, существует ли элемент фона на данном слое. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функция layer_background_get_id()), и функция вернет либо true, если элемент существует, либо false, если его нет. ПРИМЕЧАНИЕ: Эта функция работает в пределах текущей целевой комнаты - по умолчанию это комната, в которой вызывается функция, - которую можно установить с помощью функции layer_set_target_room(). Синтаксис: layer_background_exists(layer_id, background_element_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). background_element_id Уникальное значение ID элемента фона для проверки Возвращается: Булево Пример: var lay_id = layer_get_id("Background_trees"); if layer_background_exists(lay_id, global.TreesBackground) { layer_background_destroy(lay_id, global.TreesBackground); } Приведенный выше код проверяет слой "Background_trees" на предмет существования данного фонового элемента, и если он существует, то он уничтожается (но не слой).
