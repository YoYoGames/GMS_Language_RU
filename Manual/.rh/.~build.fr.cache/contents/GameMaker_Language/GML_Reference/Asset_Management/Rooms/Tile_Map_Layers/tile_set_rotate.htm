tile_set_rotate Эта функция может быть использована для установки заданного набора данных tile-data для поворота плитки на 90 градусов или нет. Вы передаете данные тайла, которые можно получить с помощью функции tilemap_get()и затем задаете аргумент rotate либо true, если вы хотите, чтобы плитка была повернута, либо false, если вы хотите, чтобы плитка была в своем стандартном, не повернутом состоянии. Функция вернет модифицированный набор данных плитки, который затем можно применить с помощью функции tilemap_set() функцию. Синтаксис: tile_set_rotate(tiledata, rotate) Аргумент Описание tiledata данные плитки для установки вращать Установите значение true для поворота и false, чтобы оставить все как есть. Возвращается: Реально Пример: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_rotate(data); data = tile_set_rotate(data, bool); tilemap_set(map_id, data, mx, my); Приведенный выше код получает ID карты плитки из заданного слоя, а затем получает позицию x и y ячейки для плитки под мышью. Эта позиция затем используется для получения данных плитки, которые поворачиваются и затем используются для установки плитки снова.
