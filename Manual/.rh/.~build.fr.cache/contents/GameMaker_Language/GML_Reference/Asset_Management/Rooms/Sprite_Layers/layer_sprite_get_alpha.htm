layer_sprite_get_alpha Эта функция может быть использована для получения значения альфа-эффекта элемента спрайта. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), и функция вернет значение от 0 (полностью прозрачный) до 1 (полностью непрозрачный). Синтаксис: layer_sprite_get_alpha(sprite_element_id); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию Возвращается: Реальный (от 0 до 1) Пример: var lay_id = layer_get_id("sprite_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_alpha(spr_id) < 0.1 { layer_sprite_destroy(spr_id); } Приведенный выше код получит идентификатор слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот идентификатор затем используется для проверки альфа-элемента, и если он меньше 0,1, то элемент слоя будет уничтожен.
