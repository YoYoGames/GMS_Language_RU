layer_sequence_get_headdir В этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она вернет текущее направление головки воспроизведения, которое будет одной из констант, перечисленных ниже. Синтаксис: layer_sequence_get_headdir(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Постоянная Постоянная Описание Значение seq_dir_right Последовательность воспроизводит кадры в инкрементном порядке слева направо 1 seq_dir_left Последовательность воспроизводит кадры в убывающем порядке справа налево -1 Пример: if layer_sequence_get_headdir(title_sequence) != seq_dir_left { layer_sequence_headdir(title_sequence, seq_dir_left); } Приведенный выше код проверяет текущее направление воспроизведения ID элемента последовательности, хранящегося в переменной "title_sequence", и если оно не установлено в seq_dir_left, то оно устанавливается в это значение.
