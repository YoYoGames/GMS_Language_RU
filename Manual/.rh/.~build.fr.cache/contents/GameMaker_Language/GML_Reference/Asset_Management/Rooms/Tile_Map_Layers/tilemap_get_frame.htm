tilemap_get_frame Поскольку плитки могут быть анимированными, иногда бывает полезно знать, какой кадр в данный момент рисуется, и реагировать соответствующим образом, поэтому с помощью этой функции можно получить индекс текущего кадра для данной карты плиток. Вы задаете ID элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и функция вернет индекс кадра. Синтаксис: tilemap_get_frame(tilemap_element_id) Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки для получения индекса кадра Возвращается: Величина (от 0 (включительно) до максимального количества кадров анимации (исключение)) Пример: var lay_id = layer_get_id("Tiles_Traps"); var map_id = layer_tilemap_get_id(lay_id); if tilemap_get_frame(map_id) >= 2 && tilemap_get_frame(map_id) < 4 { global.activate = true; } else { global.activate = false; } Приведенный выше код проверяет текущий кадр анимации для заданной карты тайлов на заданном слое и устанавливает глобальную переменную на основе возвращаемого значения.
