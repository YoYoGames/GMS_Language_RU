layer_get_depth Вы можете использовать эту функцию для получения значения глубины, связанного с данным слоем. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция вернет глубину слоя в виде вещественного числа. Обратите внимание, что глубина определяется как большая, чем "дальше" от камеры, и меньшая, чем "ближе" к камере, поэтому если у вас есть три слоя с глубинами -100, 0, 100, слои будут рисоваться в порядке 100, 0, -100, так что "верхним" слоем (т.е. самым близким к виду камеры и поэтому нарисованным поверх всех остальных) будет слой с глубиной -100. На следующем изображении показана схема работы глубины для слоев: Обратите внимание, что если вы предоставите недопустимое значение ID слоя, то получите ошибку. Синтаксис: layer_get_depth(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя для получения глубины (или имя слоя в виде строки). Возвращается: Реально Пример: if layer_get_depth(global.temp_layer) != -100 { layer_destroy(global.temp_layer); global.temp_layer = layer_create(-100); } Приведенный выше код проверяет глубину ID слоя, хранящегося в глобальной переменной, и если она не равна -100, то уничтожает слой и создает его заново на глубине -100.
