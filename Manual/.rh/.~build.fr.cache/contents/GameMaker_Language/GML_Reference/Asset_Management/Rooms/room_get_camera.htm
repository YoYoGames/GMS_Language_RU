room_get_camera С помощью этой функции вы можете получить уникальный индекс ID камеры, назначенной на определенный вид в комнате, отличной от текущей. Вы указываете комнату, которую нужно использовать, порт вида, который нужно использовать (от 0 до 7), и функция возвращает индекс камеры. Синтаксис: room_get_camera(rm, vind); Аргумент Описание rm Индекс комнаты для получения камеры просмотра vind Индекс порта представления для получения камеры Возвращается: Реальный (индекс камеры) Пример: var cam = room_get_camera(rm_Game, 0); if cam != global.MainCam { room_set_camera(rm_Game, 0, global.MainCam); } Приведенный выше код назначает камеру во вновь созданной комнате на порт просмотра [0].
