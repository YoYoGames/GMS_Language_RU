layer_sequence_create С помощью этой функции вы можете создать экземпляр актива последовательности на заданном слое. Вы передаете идентификатор слоя, который может быть строкой имени слоя - как определено в редакторе помещений - или уникальным идентификатором слоя - как возвращается функцией layer_get_id(), а также позиции X и Y в комнате, в которых нужно создать последовательность, и, наконец, ID создаваемой последовательности. Идентификатор последовательности - это константа имени, которую вы определили в браузере активов для последовательности. Функция вернет уникальный идентификатор элемента последовательности, который затем можно использовать во всех дальнейших функциях слоя для последовательностей, или его можно использовать для получения структуры экземпляра последовательности с помощью функции layer_sequence_get_instance(). Синтаксис: layer_sequence_create(layer_id, x, y, sequence_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя экземпляра, на который нужно нацелиться x Позиция x в комнате для создания последовательности в y Положение y в комнате для создания последовательности в sequence_id Идентификатор последовательности для создания, как определено в браузере активов Возвращается: Целое число (идентификатор элемента) Пример: var _s = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); layer_sequence_pause(_s); Приведенный выше код создает новую последовательность на слое "Background", а затем приостанавливает ее.
