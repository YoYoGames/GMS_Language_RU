layer_sprite_yscale С помощью этой функции вы можете изменить, должен ли данный элемент спрайта на слое масштабироваться по оси y или нет. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью команды layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете значение масштаба. Масштаб 1 означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться вниз (0,5, например, уменьшит высоту используемого спрайта вдвое), большие значения будут масштабироваться вниз (0,5, например, уменьшит высоту используемого спрайта вдвое). спрайта), большие значения будут масштабироваться вверх, а отрицательные значения будут зеркально отражать спрайт и масштабировать его, за исключением случая, когда используемое значение равно -1 (в этом случае используемый спрайт будет просто зеркально отражен сверху вниз относительно своей позиции (0, 0) без масштабирования). Синтаксис: layer_sprite_yscale(sprite_element_id, yscale); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для изменения yscale Значение шкалы yscale (по умолчанию равно 1) Возвращается: Н/Д Пример: var asset_sprite = layer_sprite_get_id(layer, "gfc_Trees"); if layer_sprite_get_xscale(asset_sprite) != 1 || layer_sprite_get_yscale(asset_sprite) != 1 { layer_sprite_xscale(asset_sprite, 1); layer_sprite_yscale(asset_sprite, 1); } Приведенный выше код проверит элемент спрайта, назначенный слою, на котором находится экземпляр, выполняющий код, и если он масштабируется в любом направлении, то установит масштаб по оси x и масштаб по оси y равным 1.
