layer_background_get_visible Эта функция может быть использована для получения видимого состояния фонового элемента. Вы передаете идентификатор фонового элемента (который вы получаете при создании фонового элемента, используя layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет либо true, если элемент в данный момент виден, либо false, если нет. Обратите внимание, что на возвращаемое значение не зависит от того, является ли слой, на котором находится элемент, видимым или нет. Синтаксис: layer_background_get_visible(background_element_id); Аргумент Описание background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию Возвращается: Булево Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_visible(back_id) { layer_background_visible(back_id, false); } Приведенный выше код получит идентификатор слоя для слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки видимости элемента, и если он виден, то это свойство устанавливается в false.
