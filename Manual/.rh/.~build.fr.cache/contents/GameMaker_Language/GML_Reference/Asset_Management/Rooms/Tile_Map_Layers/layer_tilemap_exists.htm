layer_tilemap_exists С помощью этой функции можно проверить, существует ли элемент карты плиток на данном слое. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и ID элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()) и функция вернет либо true, если элемент существует, либо false, если его нет. ПРИМЕЧАНИЕ: Эта функция работает в пределах текущей целевой комнаты - по умолчанию это комната, в которой вызывается функция, - которую можно установить с помощью функции layer_set_target_room(). Синтаксис: layer_tilemap_exists(layer_id, tilemap_element_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). tilemap_element_id Уникальное значение ID элемента карты плитки для проверки Возвращается: Булево Пример: var lay_id = layer_get_id("tilemap_trees"); if layer_tilemap_exists(lay_id, global.Treestilemap) { layer_tilemap_destroy(lay_id, global.Treestilemap); } Приведенный выше код проверяет слой "tilemap_trees" на предмет существования данного элемента карты плиток, и если он существует, то уничтожает его (но не слой).
