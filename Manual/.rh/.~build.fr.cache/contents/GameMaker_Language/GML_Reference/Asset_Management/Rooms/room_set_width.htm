ширина_комнаты С помощью этой функции вы можете изменить (или установить) ширину любой комнаты в вашей игре, кроме текущей. Синтаксис: room_set_width(index, w); Аргумент Описание индекс Индекс комнаты для установки ширины. w Новая ширина комнаты в пикселях. Возвращается: Н/Д Пример: global.myroom = room_add(); room_set_width(global.myroom, 640); room_set_height(global.myroom, 480); room_set_persistent(global.myroom, false); Это создаст новую комнату и сохранит ее индекс в переменной "global.myroom". Затем она установит ее ширину 640 пикселей, высоту 480 пикселей, надпись 'Game Room' и постоянство 'false'. Назад: Номера Далее: room_set_persistent © Copyright YoYo Games Ltd. 2021 Все права защищены
