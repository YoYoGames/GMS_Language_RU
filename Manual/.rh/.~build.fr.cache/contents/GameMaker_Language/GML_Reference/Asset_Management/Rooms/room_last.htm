номер_последний Эта переменная, доступная только для чтения, возвращает индекс самой последней комнаты в игре (он определяется порядком появления комнат в Менеджере комнат, а не порядком их создания). они были созданы). Обратите внимание, что эта переменная не распознает и не учитывает комнаты, которые были добавлены динамически с помощью room_add() или room_duplicate(). Синтаксис: room_last; Возвращается: Индекс Пример: if keyboard_check_pressed(ord("Q")) { room_goto(room_last); } Приведенный выше код проверяет, была ли нажата клавиша, и если да, то переходит в последнюю комнату в игре.
