layer_has_instance Эта функция может быть использована для проверки того, назначен ли данный экземпляр на данный момент данному слою. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это повлияет на производительность) и ID экземпляра, который нужно проверить. Можно также указать object_index (т.е. имя объекта в браузере активов), и функция проверит, есть ли экземпляры этого объекта на данном слое. заданном слое. Функция вернет true, если экземпляр находится на слое, и false, если нет. Синтаксис: layer_has_instance(layer_id, instance_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). идентификатор экземпляра Уникальный идентификатор экземпляра или индекс объекта экземпляра для проверки Возвращается: Булево Пример: if !layer_has_instance(global.Bullet_Layer, obj_Bullet_Parent) { layer_destroy(global.Bullet_Layer); } Приведенный выше код проверит, содержит ли данный слой какие-либо экземпляры объекта "obj_Bullet_Parent", и если нет, то уничтожит слой.
