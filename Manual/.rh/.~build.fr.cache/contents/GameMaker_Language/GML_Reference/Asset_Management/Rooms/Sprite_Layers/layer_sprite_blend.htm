слой_спрайта_смесь Эта функция управляет смешиванием (или "тонированием") спрайта, и значение по умолчанию равно -1 (что представляет константу c_white, которую также можно использовать). Любое другое значение (включая внутренние цветовые константы, такие как c_red, или c_aqua) будет смешивать указанный цвет с исходным спрайтом. Вы указываете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью команды layer_sprite_create() или при использовании функция layer_sprite_get_id()), а затем задаете цвет смешивания. Ниже показан пример спрайта, на который были наложены разные цвета: Обратите внимание, что на платформе HTML5 следует ограничить смешивание (если не используется WebGL), поскольку каждый смешиваемый спрайт должен кэшироваться отдельно, и поэтому большое количество смешиваемых спрайтов может отрицательно сказаться на производительности (вы также можете установить размер кэша с помощью функции sprite_set_cache_size()). Синтаксис: layer_sprite_blend(sprite_element_id, blend); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для изменения смесь Цвет для смешивания со спрайтом (по умолчанию c_white) Возвращается: Н/Д Пример: var lay_id = layer_get_id("Asset_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); layer_sprite_blend(spr_id, c_gray); Приведенный выше код получает значение ID спрайта под названием "Clouds", назначенного слою "Asset_sky", и затем окрашивает его в определенный цвет.
