tilemap_get_at_pixel С помощью этой функции вы можете получить данные плитки из позиции (в пределах комнаты) элемента карты плитки. Вы задаете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а также позицию x и y в комнате, из которой нужно получить данные плитки, и функция вернет "сгусток" данных плитки. Эти данные по сути, является битовой маской, содержащей индекс плитки, булевы flip/rotate/mirror и любые примененные данные маски (см. tilemap_set_mask() подробнее), и полученное значение данных может быть использовано можно использовать в функциях tile_* для изменения свойств плитки. Если вам нужно получить данные плитки из конкретной ячейки, используйте функцию tilemap_get() вместо нее. ВАЖНО! Если плитки в карте плиток были неизменны (т.е. не повернуты, не перевернуты и т.д.), то возвращаемое значение данных набора плиток будет точно равно индексу плитки в наборе плиток. Таким образом, вы можете создавать "карты столкновений карты" плиток, используя одну плитку с индексом 1 в наборе плиток - например - затем использовать эту функцию для проверки 1 или 0 (пустая плитка) для расчета столкновений. Синтаксис: tilemap_get_at_pixel(tilemap_element_id, x, y); Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки для получения данных о плитке x Положение по оси x для получения данных о плитке (в пикселях комнаты). y Положение по оси y для получения данных о плитке (в пикселях комнаты). Возвращается: Real (-1, если произошла ошибка) Пример: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var data = tilemap_get_at_pixel(map_id, 64, 128); data = tile_set_flip(data, true); tilemap_set_at_pixel(map_id, data, 64, 128); Приведенный выше код получает идентификатор карты плиток "Clouds" на слое "Tiles_Sky", а затем использует его для получения данных из плитки в позиции (64, 128). Эти данные плитки затем переворачиваются перед использованием для установки плитки на карте плиток снова.
