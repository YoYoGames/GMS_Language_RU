tile_get_index Эта функция может быть использована для получения индекса плитки (положение плитки в изображении набора плиток) из набора данных плитки. Вы передаете данные тайлов, которые могут быть получены с помощью функции tilemap_get()и функция вернет целочисленное значение индекса или -1 в случае ошибки. Синтаксис: tile_get_index(tiledata) Аргумент Описание tiledata данные плитки для проверки Возвращается: Целое число Пример: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var ind = tile_get_index(data); data = tile_set_index(data, irandom(23)); tilemap_set(map_id, data, mx, my); Приведенный выше код получает ID карты плитки из заданного слоя и затем использует его для получения данных плитки для ячейки под позицией мыши. Эти данные затем используются для установки индекса плитки на случайное число. Назад: Плитка Слои карты Далее: tile_get_flip © Copyright YoYo Games Ltd. 2021 Все права защищены
