layer_sequence_get_angle С помощью этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет текущий угол элемента последовательности в игровой комнате. Обратите внимание, что углы возвращаются в градусах, причем 0º - это вправо, 90º - вверх, 180º - влево и 270º - вниз. Синтаксис: layer_sequence_get_angle(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Реально Пример: var _ang = layer_sequence_get_angle(title_sequence) if _ang > 0 { _ang -= 1; layer_sequence_angle(title_sequence, _ang); } Приведенный выше код извлекает текущий угол элемента последовательности с идентификатором, хранящимся в переменной "title_sequence", и если он не равен 0, то из текущего угла вычитается 1.
