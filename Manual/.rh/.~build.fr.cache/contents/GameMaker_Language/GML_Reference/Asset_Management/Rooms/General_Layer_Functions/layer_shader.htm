layer_shader С помощью этой функции вы можете назначить ресурс шейдера любому заданному слою, после чего слой будет отрисован с использованием этого шейдера. Вы передаете либо уникальное значение ID слоя, либо имя слоя (в виде строки - это влияет на производительность) , а также ID шейдера, который необходимо использовать. Шейдер должен быть создан ранее в Asset Browser, а индекс шейдера (имя ресурса шейдера ) передается в эту функцию. Если к назначенному слою добавлены экземпляры, то шейдер будет применен ко всем событиям рисования, которые использует данный экземпляр - например, если у экземпляра есть событие Draw GUI Begin, то шейдер будет применен к нему автоматически. Шейдер также будет влиять на любые другие графические элементы, нарисованные на этом слое, например, спрайтовые активы или карты плиток. Обратите внимание, что функция не предназначена для вызова в каких-либо событиях рисования или событиях шага, а должна вызываться только в начале комнаты в коде создания комнаты или в событии Create Event / Room Start Event экземпляра. Синтаксис: layer_shader(layer_id, shader) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). шейдер Индекс шейдера, который нужно присвоить слою Возвращается: Н/Д Пример: var lay_id = layer_get_id("Instances"); layer_shader(lay_id, shd_Sepia); Приведенный выше код назначит шейдерный ресурс "shd_Sepia" данному слою для всей отрисовки.
