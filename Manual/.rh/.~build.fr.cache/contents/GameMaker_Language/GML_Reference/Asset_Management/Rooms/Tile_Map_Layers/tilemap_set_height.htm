tilemap_set_height Эта функция может использоваться для изменения размера элемента карты плиток. Вы передаете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и новую высоту карты тайлов в ячейках тайлов. Синтаксис: tilemap_set_height(tilemap_element_id, height) Аргумент Описание tilemap_element_id Уникальное значение идентификатора элемента карты плитки для установки высоты высота Значение высоты (в "ячейках") Возвращается: Н/Д Пример: var lay_id = layer_get_id("Tiles_Walls"); var map_id = layer_tilemap_get_id(lay_id); if tilemap_get_height(map_id) != room_height div 16 { tilemap_set_height(map_id, room_height div 16); } Приведенный выше код проверяет ширину конкретной карты плиток, и если она не соответствует размеру, то ширина устанавливается.
