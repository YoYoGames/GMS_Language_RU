layer_background_alpha Эта функция управляет альфа (прозрачностью) фонового спрайта. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или когда вы используете функцию layer_background_get_id()), а затем задаете используемое значение альфа. Альфа может быть в диапазоне от 0 (полностью прозрачный) до 1 (полностью непрозрачный), а значение альфа по умолчанию для элемента фона элемента по умолчанию равно 1. Обратите внимание, что если слой, которому был назначен фоновый элемент, не виден, или сам элемент был сделан невидимым, вы не увидите никакой разницы с помощью этой функции, пока слой или элемент не станут видимыми. снова. Синтаксис: layer_background_alpha(background_element_id, alpha); Аргумент Описание background_element_id Уникальное значение ID элемента фона для изменения альфа Альфа для фонового спрайта, от 0 до 1 (по умолчанию 1) Возвращается: Н/Д Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); layer_background_alpha(back_id, random(1)); Приведенный выше код получает значение ID фона, назначенного слою "Background_sky", а затем устанавливает его альфа-фактор на случайное значение между 0 и 1.
