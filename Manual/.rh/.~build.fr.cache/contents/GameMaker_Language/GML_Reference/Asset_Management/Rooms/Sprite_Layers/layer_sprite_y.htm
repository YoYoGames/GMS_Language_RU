слой_спрайта_y Эта функция управляет положением по оси y комнаты элемента спрайта актива на слое. Вы задаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете значение y, которое нужно использовать (на основе координат комнаты). Синтаксис: layer_sprite_y(sprite_element_id, y); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для изменения y Положение по оси y для спрайта Возвращается: Н/Д Пример: var lay_id = layer_get_id("Asset_sky"); var spr_id = layer_sprite_get_id(lay_id, "Clouds"); layer_sprite_y(spr_id, 50 + irandom(room_height - 200)); Приведенный выше код получает значение ID спрайта актива "Clouds", назначенного слою "Asset_sky", а затем устанавливает его позицию y на случайное значение между 50 и высотой комнаты минус 200.
