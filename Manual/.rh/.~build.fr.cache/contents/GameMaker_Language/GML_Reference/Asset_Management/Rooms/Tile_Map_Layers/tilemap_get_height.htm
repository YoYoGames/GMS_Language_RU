tilemap_get_height С помощью этой функции вы можете получить высоту (в ячейках) элемента карты плитки. Вы передаете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или когда вы используете функцию layer_tilemap_get_id()), а функция вернет высоту (количество клеток в ширину). Синтаксис: tilemap_get_height(tilemap_element_id); Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки для получения высоты Возвращается: Целое число Пример: var lay_id = layer_get_id("espadrilles"); var map_id = layer_tilemap_get_id(lay_id); var _w = tilemap_get_width(map_id); var _h = tilemap_get_height(map_id); total_tiles = (_w * _h); Приведенный выше код использует полученный идентификатор карты плиток для получения ширины и высоты карты плиток, а затем использует эти значения для установки переменной экземпляра.
