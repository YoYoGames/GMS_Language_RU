индекс_слоя Эта функция может быть использована для установки индекса изображения спрайтового актива, который имеет несколько суб-изображений на слое. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете индекс изображения, который будет использоваться. Если вы зададите значение вне диапазона суб-изображений, то индекс изображения будет зацикливаться. Синтаксис: layer_sprite_index(sprite_element_id, image_index); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для установки индекс Индекс изображения, используемый для спрайта Возвращается: Н/Д Пример: var lay_id = layer_get_id("sprite_trees"); var spr_id = layer_sprite_get_id(lay_id, "gfc_trees"); layer_sprite_index(spr_id, 1); Приведенный выше код получит идентификатор слоя для слоя с именем "sprite_trees", а затем использует его для получения идентификатора заданного элемента спрайта на этом слое. Этот идентификатор затем используется для изменения индекса изображения элемента. Назад: Слои спрайта Далее: слой_спрайта_скорость © Copyright YoYo Games Ltd. 2021 Все права защищены
