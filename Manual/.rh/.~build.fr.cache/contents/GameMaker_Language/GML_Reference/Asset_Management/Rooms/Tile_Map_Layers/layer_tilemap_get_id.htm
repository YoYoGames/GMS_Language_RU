layer_tilemap_get_id Эта функция может использоваться для получения уникального значения ID элемента карты плиток на слое. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция вернет значение ID, связанное с элементом карты плиток на слое. Обратите внимание, что эта функция специально разработана для использования с картами плиток, которые были добавленных в IDE, так как если вы добавите карту плиток в слой с помощью кода, используя функцию layer_tilemap_create()то она вернет уникальный ID для добавленного элемента карты плиток. Если заданный идентификатор карты плиток неверен или карта плиток не существует, функция вернет -1. Синтаксис: layer_tilemap_get_id(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). Возвращается: значение ID элемента карты плитки (Real) или -1 Пример: var lay_id = layer_get_id("tilemap_trees"); var tile_id = layer_tilemap_get_id(lay_id); layer_tilemap_destroy(tile_id); Приведенный выше код получит идентификатор слоя для слоя с именем "tilemap_trees", а затем использует его для получения идентификатора элемента карты плиток на этом слое. Этот идентификатор затем используется для удаления карты плитки из слоя. Назад: Плитка Слои карты Далее: layer_tilemap_exists © Copyright YoYo Games Ltd. 2021 Все права защищены
