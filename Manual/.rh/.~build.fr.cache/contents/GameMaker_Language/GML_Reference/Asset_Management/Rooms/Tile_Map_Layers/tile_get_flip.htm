tile_get_flip Эта функция может быть использована для проверки того, была ли в заданном наборе данных плитки перевернута плитка или нет. Вы передаете данные плитки, которые можно получить с помощью функции tilemap_get(), и функция вернет true, если плитка перевернута, или false, если нет. Синтаксис: tile_get_flip(tiledata) Аргумент Описание tiledata данные плитки для проверки Возвращается: Булево Пример: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_flip(data); data = tile_set_flip(data, bool); tilemap_set(map_id, data, mx, my); Приведенный выше код получает ID карты плитки из заданного слоя, а затем получает позицию x и y ячейки для плитки под мышью. Эта позиция затем используется для получения данных плитки, которые переворачиваются и затем снова используются для установки плитки.
