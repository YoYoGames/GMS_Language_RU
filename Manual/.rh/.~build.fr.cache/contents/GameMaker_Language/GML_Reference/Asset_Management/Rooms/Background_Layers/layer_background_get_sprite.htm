layer_background_get_sprite Эта функция может быть использована для получения текущего значения индекса спрайта фонового элемента. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет реальное значение, представляющее индекс отображаемого спрайта. Если элемент не имеет назначенного спрайта, функция вернет -1. Синтаксис: layer_background_get_sprite(background_element_id); Аргумент Описание background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию Возвращается: Индекс спрайта (реальный) или -1 Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_sprite(back_id) != spr_Clouds { layer_background_sprite(back_id, spr_Clouds); } Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот ID затем используется для проверки спрайта, назначенного этому элементу, и установки его на спрайт "spr_Clouds". если он еще не установлен.
