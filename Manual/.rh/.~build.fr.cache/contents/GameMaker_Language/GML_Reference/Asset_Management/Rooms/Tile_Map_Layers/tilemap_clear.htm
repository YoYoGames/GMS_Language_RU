tilemap_clear С помощью этой функции вы можете очистить/установить все плитки на заданной карте плиток. Вы передаете ID элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функция layer_tilemap_get_id()), а затем предоставляете данные плитки, которую вы хотите очистить. Значение по умолчанию 0 очистит все плитки из слоя (по сути, сделает все плитки "пустыми"), В то время как вы можете использовать специальные функции tile_* для создания собственных данных тайлов для очистки карты тайлов. Синтаксис: tilemap_clear(tilemap_element_id, tiledata) Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки, которое необходимо изменить tiledata Данные плитки, используемые для очистки слоя Возвращается: Н/Д Пример: var layer_id = layer_get_id("Forest"); var tile_id = layer_tilemap_get_id(layer_id); tilemap_clear(tile_id, 0); Приведенный выше код получает значение ID карты плиток, созданной в редакторе комнат, а затем очищает ее, используя "пустые" плитки.
