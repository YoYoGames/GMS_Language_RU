слой_создать Эта функция используется для создания нового слоя в текущей комнате. Вы указываете глубину, которую должен занять новый слой, и функция возвращает уникальное значение Layer ID, которое затем может быть использовано для доступа к слою в дальнейших функциях. Вы также можете указать необязательный аргумент "имя", который позволит вам дать слою определенное имя (в виде строки), и вместо ID слоя для доступа к слою вы можете использовать эту строку имени, хотя следует отметить, что использование строк имен будет иметь большее влияние на производительность, чем использование только значения ID. Если вы не укажете имя слоя, то созданному слою будет присвоено имя в формате " _layer_XXX", где "XXX" шестнадцатеричное значение, используемое для присвоения слою уникального имени. Когда задается значение глубины, оно будет тем выше, чем "дальше" от камеры находится слой, и тем ниже, чем "ближе" к камере находится слой, поэтому если у вас есть три слоя с глубинами -100, 0, 100, то слои будут рисоваться в следующем порядке 100, 0, -100, так что "верхним" слоем (т.е. самым близким к виду камеры и поэтому нарисованным поверх всех остальных) будет слой с глубиной -100. На следующем изображении показана схема работы глубины для слоев: Стоит отметить, что хотя вы можете назначить более одного слоя на одну глубину, вы никак не можете гарантировать, что слои на одной и той же глубине будут отрисовываться в определенном порядке, поэтому лучше всего назначать только один слой на любую глубину. Также обратите внимание, что минимальная и максимальная глубина слоя составляет от -16000 до 16000, и все, что находится за пределами этих глубин, не будет отрисовано. Если вам нужна глубина за пределами этого диапазона то вам необходимо использовать функцию layer_force_draw_depth(). Синтаксис: layer_create(depth, [name]) Аргумент Описание глубина Глубина, на которой должен быть создан слой. имя Имя, которое нужно дать слою (строка - необязательно). Возвращается: Идентификатор слоя (реальный) Пример: global.Bullet_Layer = layer_create(-100); Приведенный выше код создаст новый слой со значением глубины -100 и присвоит его ID значение глобальной переменной.
