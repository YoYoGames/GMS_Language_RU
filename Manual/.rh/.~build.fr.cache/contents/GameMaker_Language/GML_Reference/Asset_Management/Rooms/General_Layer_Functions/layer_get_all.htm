layer_get_all Эта функция возвращает массив, заполненный уникальными значениями ID каждого слоя в комнате. Синтаксис: layer_get_all() Возвращается: Массив (1D, заполненный идентификаторами слоев) Пример: var a = layer_get_all(); for (var i = 0; i < array_length(a); i++;) { layer_destroy(a[i]); } Приведенный выше код извлекает все слои в комнате и добавляет их значения ID в массив.
