layer_background_speed Эта функция может быть использована для установки скорости анимирующего спрайта, который был назначен элементу фона. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете скорость анимации. Если в IDE задана скорость анимации для используемого спрайта, рассчитываемая как Frames Per Game Frame, то заданное значение скорости является множителем, причем по умолчанию используется значение 1, а при установке значения 0,5 скорость анимации уменьшится вдвое, а при установке значения 2 - удвоится. Однако, если в IDE установлена скорость анимации для используемого спрайта, которая будет рассчитываться как Frames Per Second, то заданное значение скорости является буквальным значением кадров в секунду, поэтому, например, установка значения 5 приведет к тому, что анимация будет повторяться по 5 кадров каждую секунду, пока идет игра. Если элемент фона не имеет привязанного спрайта или используемый спрайт не имеет вложенных изображений, эта функция не будет иметь никакого эффекта. Синтаксис: layer_background_speed(background_element_id, speed); Аргумент Описание background_element_id Уникальное значение ID элемента фона для изменения скорость Множитель/значение скорости, используемое для элемента Возвращается: Н/Д Пример: var lay_id = layer_get_id("Background_water"); var back_id = layer_background_get_id(lay_id); layer_background_speed(back_id, 0.5); Приведенный выше код получит идентификатор слоя для слоя с именем "Background_water", а затем использует его для получения идентификатора фонового элемента на этом слое.
