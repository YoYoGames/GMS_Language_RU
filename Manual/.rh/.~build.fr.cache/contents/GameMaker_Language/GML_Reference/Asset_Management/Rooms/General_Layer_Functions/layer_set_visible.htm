layer_set_visible С помощью этой функции можно переключить видимость слоя. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), а также значение переключения для слоя, где видимый и невидимый а также значение переключения для слоя, где visible - true, а invisible - false. Когда слой невидим, ничто, назначенное слою, не будет нарисовано, а если слою назначены какие-либо экземпляры, то они даже не запустят свое событие рисования (точно так же, как если бы вы установили переменную экземпляра visible в false). Обратите внимание, что любые экземпляры, которые уже помечены как невидимые, не будут помечены как видимые, если для слоя, на котором они находятся, установлено значение visible, поскольку это две независимые настройки, хотя их поведение одинаково. Синтаксис: layer_set_visible(layer_id, visible) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). видимый Должен ли слой быть видимым(true) или нет(false). Возвращается: Н/Д Пример: var lay_id = layer_get_id("Instances"); if layer_get_visible(lay_id) { layer_set_visible(lay_id, false); } else { layer_set_visible(lay_id, true); } Приведенный выше код получает значение ID для слоя с именем "Instances" в редакторе помещений, затем использует ID для проверки видимости слоя, переключая видимость слоя в зависимости от возвращаемого значения.
