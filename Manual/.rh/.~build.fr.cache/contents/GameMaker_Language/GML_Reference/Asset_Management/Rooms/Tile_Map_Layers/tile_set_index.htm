tile_set_index Эта функция используется для установки индекса плитки (положение плитки в изображении набора плиток) для набора данных плиток. Вы передаете данные тайлов, которые могут быть получены с помощью функции tilemap_get(), и затем целочисленное значение, которое представляет позицию плитки (ее индекс) в наборе плиток. Функция вернет модифицированный набор плиточных данных, который затем можно применить с помощью функции tilemap_set() функцию. Синтаксис: tile_set_index(tiledata, index) Аргумент Описание tiledata данные плитки для установки индекс Целочисленное значение индекса набора плиток для использования Возвращается: Реальные (плиточные) Пример: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var ind = tile_get_index(data); data = tile_set_index(data, irandom(23)); tilemap_set(map_id, data, mx, my); Приведенный выше код получает ID карты плитки из заданного слоя и затем использует его для получения данных плитки для ячейки под позицией мыши. Эти данные затем используются для установки индекса плитки на случайное число. Назад: Плитка Слои карты Далее: tile_set_flip © Copyright YoYo Games Ltd. 2021 Все права защищены
