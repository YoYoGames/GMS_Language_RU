layer_background_create С помощью этой функции вы можете назначить ресурс спрайта слою для использования в качестве фона в вашем проекте. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность) и индекс спрайта (это имя спрайта, отображаемое в Asset Browser), и он будет добавлен в слой. Функция возвращает уникальное значение ID для элемента, который затем может быть использован в дальнейших функциях слоя для фона. Синтаксис: layer_background_create(layer_id, sprite) Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). спрайт Индекс спрайта, который будет использоваться Возвращается: Идентификатор фонового элемента (реальный) Пример: global.back_layer = layer_create(10000); global.back_trees = layer_background_create(global.back_layer, spr_Trees); Приведенный выше код создает новый слой, а затем добавляет к нему новый элемент фона, устанавливая спрайт в качестве используемого фонового изображения.
