layer_sequence_get_sequence С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращено функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет структуру объекта последовательности. Эта функция позволяет обойти необходимость сначала получить структуру экземпляра последовательности и получить прямой доступ к данным последовательности. Вы можете узнать больше о формате структуры объекта последовательности на этой странице. Синтаксис: layer_sequence_get_sequence(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Структура Пример: var _seq = layer_sequence_create("Background", 0, 0, 0, seq_AnimatedBackground); var _struct = layer_sequence_get_sequence(_seq); _struct.playbackSpeedType = spritespeed_framespersecond; _struct.playbackSpeed = 30; Приведенный выше код создает новый элемент последовательности, а затем извлекает его структуру данных последовательности. Затем он используется для изменения скорости воспроизведения и типа последовательности.
