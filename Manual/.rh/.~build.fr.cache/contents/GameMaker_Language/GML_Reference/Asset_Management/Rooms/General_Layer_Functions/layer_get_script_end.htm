layer_get_script_end Эта функция возвращает индекс функции сценария, назначенной для запуска в конце рендеринга заданного слоя, или возвращает -1, если функция не назначена. Вы передаете слой ID (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность). Вы можете назначить функции сценария слою с помощью layer_script_begin() и layer_script_end() . Синтаксис: layer_get_script_end(layer_id); Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). Возвращается: Индекс функции сценария (вещественный) или -1, если сценарий не назначен Пример: if layer_get_script_end(layer) == -1 { layer_script_end(layer, scr_ResetShaderValues); } Приведенный выше код проверяет, назначена ли слою, на котором находится экземпляр, выполняющий код, функция сценария, и если нет, то назначается.
