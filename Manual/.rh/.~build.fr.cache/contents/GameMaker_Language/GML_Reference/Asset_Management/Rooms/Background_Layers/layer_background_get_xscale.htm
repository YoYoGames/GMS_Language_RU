layer_background_get_xscale Эта функция может использоваться для получения текущего значения множителя масштаба фонового элемента. Вы передаете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет реальное значение, которое представляет собой множитель масштаба, используемый для отрисовки спрайта. Значение по умолчанию равно 1. Синтаксис: layer_background_get_xscale(background_element_id); Аргумент Описание background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию Возвращается: Реально Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_xscale(back_id) != -1 { layer_background_xscale(back_id, -1); } Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки масштаба x для элемента, и если он не равен -1, то устанавливается значение -1.
