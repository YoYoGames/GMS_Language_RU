layer_destroy_instances Эта функция может быть использована для уничтожения всех экземпляров, назначенных данному слою. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это повлияет на производительность), после чего все экземпляры, находящиеся на слое, будут удалены из игры, вызывая события Destroy и Clean Up. Синтаксис: layer_destroy_instances(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя для установки глубины (или имя слоя в виде строки). Возвращается: Real (Уникальное значение идентификатора для экземпляра) Пример: if global.game_over { layer_destroy_instances(layer); } Приведенный выше код проверит глобальную переменную, и если она истинна, то все экземпляры, находящиеся на уровне вызывающего экземпляра, будут уничтожены (включая вызывающий экземпляр).
