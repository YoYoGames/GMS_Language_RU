layer_sequence_speedscale С помощью этой функции вы можете изменить скорость воспроизведения заданной последовательности. Вы передаете идентификатор элемента последовательности, возвращаемый layer_sequence_create() или одной из функций элемента слоя, а затем значение шкалы скорости. Это значение представляет собой множитель, где 1 - скорость воспроизведения по умолчанию, значения меньше 1 замедляют воспроизведение, а значения больше 1 ускоряют его, например: значение 0,5 означает половинную скорость воспроизведения, в то время как значение 2 - удвоенная скорость воспроизведения. Синтаксис: layer_sequence_speedscale(sequence_element_id, speedscale) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться шкала скорости Используемая шкала скорости (множитель), по умолчанию 1 Возвращается: Н/Д Пример: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); layer_sequence_speedscale(_seq, 0.75); Приведенный выше код создает новую последовательность на слое вызывающего экземпляра, а затем устанавливает значение шкалы скорости 0,75 (три четверти скорости воспроизведения по умолчанию).
