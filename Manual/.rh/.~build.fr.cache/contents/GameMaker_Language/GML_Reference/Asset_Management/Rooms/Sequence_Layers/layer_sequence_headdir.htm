layer_sequence_headdir С помощью этой функции вы можете задать направление для заданной головки воспроизведения последовательности. Вы передаете идентификатор элемента последовательности, возвращаемый layer_sequence_create() или одной из функций элемента слоя, а затем задаете направление головки воспроизведения, которое должно быть одной из следующих констант: Постоянная Описание Значение seq_dir_right Последовательность будет воспроизводить кадры в инкрементном порядке слева направо 1 seq_dir_left Последовательность будет воспроизводить кадры в убывающем порядке справа налево -1 Синтаксис: layer_sequence_headdir(sequence_element_id, direction) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться направление Направление головки воспроизведения, константа, указанная выше Возвращается: Н/Д Пример: var _seq = layer_sequence_create("Background", 0, 0, seq_AnimatedBackground); layer_sequence_headdir(_seq, seq_dir_left); Приведенный выше код создает новую последовательность и сохраняет ее ID в локальной переменной. Этот идентификатор затем используется для установки направления движения головки воспроизведения для уменьшения кадров (воспроизведение справа налево).
