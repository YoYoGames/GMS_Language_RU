tilemap_get_x С помощью этой функции вы можете получить x-позицию (в пределах помещения) элемента карты плиток. Вы передаете ID элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_get_id() или при использовании функции layer_tilemap_get_id()). layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), и функция вернет положение по оси x. Синтаксис: tilemap_get_x(tilemap_element_id); Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки для получения x-позиции Возвращается: Реально Пример: var lay_id = layer_get_id("Tiles_Walls"); var map_id = layer_tilemap_get_id(lay_id); var _x = tilemap_get_x(map_id); var _y = tilemap_get_y(map_id); tilemap_x(map_id, _x + 10); tilemap_y(map_id, _y + 10); Приведенный выше код использует полученный идентификатор карты плитки для получения позиции x и y карты плитки, а затем использует эти значения для ее перемещения.
