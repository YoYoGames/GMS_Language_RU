layer_sprite_get_id Эта функция может использоваться для получения уникального значения ID элемента спрайта на слое. Вы передаете ID слоя (который вы получаете при создании слоя с помощью функции layer_create() или когда вы используете имя слоя вместе с layer_get_id()) и имя элемента спрайта, заданное в редакторе комнат. Функция вернет значение ID, связанное с этим спрайт-элементом на слое. элемента на слое. Обратите внимание, что эта функция специально предназначена для использования со спрайтами, которые были добавлены в IDE на слой актива, и если вы добавили спрайт на слой через код с помощью функции layer_sprite_create(), то для всех дальнейших ссылок следует использовать ID, возвращаемый этой функцией (поскольку у этого элемента спрайта не будет имени, которое можно передать в эту функцию). Если указанный слой не существует, или заданный элемент спрайта не может быть найден, функция вернет -1. Синтаксис: layer_sprite_get_id(layer_id, sprite_element_name) Аргумент Описание идентификатор слоя Уникальное значение идентификатора слоя, на который нужно нацелиться имя_элемента_ спрайта Уникальное имя элемента спрайта на слое, определенное в редакторе помещений Возвращается: Значение идентификатора элемента спрайта (Real) Пример: var lay_id = layer_get_id("Assets_trees"); var back_id = layer_sprite_get_id(lay_id, "graphic_254367CB"); layer_sprite_change(back_id, spr_Trees_Winter); Приведенный выше код получит идентификатор слоя "Assets_trees" и затем использует его для получения идентификатора элемента спрайта "graphic_254367CB" на этом слое. Полученный идентификатор элемента спрайта затем используется для изменения его спрайта. Назад: Слои спрайта Далее: слой_спрайта_существует © Copyright YoYo Games Ltd. 2021 Все права защищены
