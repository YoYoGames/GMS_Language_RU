layer_background_blend Эта функция управляет смешиванием (или "тонированием") фонового спрайта. Значение по умолчанию равно -1 (что представляет константу c_white, которая также может быть использована). Любое другое значение (включая внутренние цветовые константы, такие как c_red, или c_aqua) будет смешивать указанный цвет с исходным спрайтом. Вы задаете ID элемента фона (который вы получаете при создании элемента фона с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), а затем задаете цвет смешивания. Ниже показан пример спрайта, который был смешан с разными цветами: Обратите внимание, что на платформе HTML5 следует ограничить смешивание (если только не используется WebGL), так как каждый смешиваемый спрайт должен кэшироваться отдельно, поэтому большое количество смешиваемых спрайтов может негативно сказаться на производительности (вы также можете установить размер кэша с помощью функции sprite_set_cache_size()). Синтаксис: layer_background_blend(background_element_id, blend); Аргумент Описание background_element_id Уникальное значение ID элемента фона для изменения смесь Цвет, который будет смешиваться с фоновым спрайтом (по умолчанию c_white). Возвращается: Н/Д Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); layer_background_blend(back_id, c_aqua); Приведенный выше код получает значение ID фона, назначенного слою "Background_sky", и затем оттеняет его цветом.
