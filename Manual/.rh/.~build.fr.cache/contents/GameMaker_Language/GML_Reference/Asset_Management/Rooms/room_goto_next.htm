номер_следующий С помощью этой функции вы можете заставить вашу игру перейти в следующую комнату, указанную в Менеджере комнат на момент компиляции игры. Если такой комнаты не существует, будет выдана ошибка и игра будет принудительно закрыта. Обратите внимание, что комната не изменится до конца события, в котором была вызвана функция, поэтому любой код после ее вызова будет выполняться, если он находится в том же событии. Оговорка заключается в том, что любой код, создающий экземпляры который будет вызван после этого, не создаст экземпляр и поэтому должен быть вызван до смены комнаты. Эта функция также вызовет событие Room End. Синтаксис: room_goto_next(); Возвращается: Н/Д Пример: if room_exists(room_next(room)) { room_goto_next(); } Приведенный выше код проверит, есть ли еще одна комната после текущей, и если да, то перейдет в эту комнату.
