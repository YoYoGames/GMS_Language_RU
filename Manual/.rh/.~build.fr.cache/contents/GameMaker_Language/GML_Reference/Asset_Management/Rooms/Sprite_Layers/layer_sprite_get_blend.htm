layer_sprite_get_blend Эта функция может быть использована для получения цвета смешивания элемента спрайта. Вы передаете ID элемента спрайта (который вы получаете, когда создаете элемент спрайта с помощью команды layer_sprite_create() или при использовании функции layer_sprite_get_id()), и функция вернет реальное значение, представляющее назначенный цвет. Синтаксис: layer_sprite_get_blend(sprite_element_id); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта, из которого нужно получить информацию Возвращается: Значение цвета (реальное) Пример: var lay_id = layer_get_id("sprite_sky"); var back_id = layer_sprite_get_id(lay_id, "Clouds"); if layer_sprite_get_blend(back_id) == c_white { layer_sprite_blend(back_id, make_colour_rgb(random(255), random(255), 255)); } Приведенный выше код получит идентификатор слоя для слоя с именем "sprite_sky" и затем использует его для получения идентификатора элемента спрайта на этом слое. Этот ID затем используется для проверки цвета наложения элемента, и если он равен константе c_white, то наложение элемента устанавливается на случайный цвет.
