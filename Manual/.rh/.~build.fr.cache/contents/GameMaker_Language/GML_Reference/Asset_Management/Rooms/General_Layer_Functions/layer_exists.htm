слой_существует Эта функция может быть использована для проверки существования заданного слоя. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция вернет булево значение true, если слой существует. ), и функция вернет булево значение true, если он существует, или false, если не существует. ПРИМЕЧАНИЕ: Эта функция работает в пределах текущей целевой комнаты - по умолчанию это комната, в которой вызывается функция, - которую можно установить с помощью функции layer_set_target_room(). Синтаксис: layer_exists(имя_слоя) Аргумент Описание имя_слоя Название слоя (строка или значение ID) Возвращается: Булево Пример: if !layer_exists(global.tileLayer) { global.tileLayer = layer_create(1000); } Приведенный выше код проверит, существует ли слой, хранящийся в глобальной переменной, и если нет, то он будет создан.
