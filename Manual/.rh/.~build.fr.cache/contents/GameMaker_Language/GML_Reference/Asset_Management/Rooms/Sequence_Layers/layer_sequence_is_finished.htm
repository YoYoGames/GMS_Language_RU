layer_sequence_is_finished В этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она будет проверять, закончила ли последовательность воспроизведение или нет, возвращая true, если да, и false, если нет. Обратите внимание, что это применимо только в том случае, если последовательность не настроена на цикл или пинг-понг в режиме воспроизведения. Синтаксис: layer_sequence_is_finished(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Булево Пример: if layer_sequence_is_finished(title_sequence) != 0 { alarm[0] = room_speed * 3; layer_sequence_play(title_sequence); } Приведенный выше код проверяет, завершен ли идентификатор элемента последовательности, хранящийся в переменной "title_sequence", и если да, то запускает его воспроизведение снова и устанавливает сигнал тревоги.
