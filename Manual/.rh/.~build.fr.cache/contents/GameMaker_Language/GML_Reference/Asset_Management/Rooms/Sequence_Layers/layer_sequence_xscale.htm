layer_sequence_xscale С помощью этой функции можно установить масштаб X для заданного элемента последовательности. Вы передаете идентификатор элемента последовательности, возвращенный функцией layer_sequence_create() или одной из функций элемента слоя вместе с новым масштабом, который нужно установить по оси X, и последовательность будет масштабирована на эту величину. Масштаб 1 означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться вниз (0,5, например, уменьшит ширину последовательности в два раза), большие значения будут масштабироваться вверх, а отрицательные значения будут зеркально отображать последовательность. вверх, а отрицательные значения зеркально отражают последовательность относительно ее начала и масштабируют ее, если только используемое значение не равно -1 (в этом случае последовательность просто зеркально отражается относительно ее начала без масштабирования). Очень важно отметить, что применение неравномерного масштабирования (например, масштабирование оси X на 3, а оси Y на 2) к элементам последовательности, которые содержат любой экземпляр, использующий вращение, может вызвать проблемы с отрисовкой экземпляра, коллизии, выбраковку и многое другое. В общем, если ваша последовательность зависит от каких-либо свойств экземпляра, мы не рекомендуем использовать неравномерное масштабирование вместе с вращением экземпляра. Также обратите внимание, что вы не должны изменять переменные image_xscale \ image_yscale \ image_angle \ x \ y для всех экземпляров, использующих эту функцию, так как они будут перезаписаны, как только последовательность обновится после вызова функции. экземпляров в последовательности, использующей эту функцию, так как они будут перезаписаны, как только последовательность обновится после вызова функции. Синтаксис: layer_sequence_xscale(sequence_element_id, xscale) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться xscale Новое значение масштаба оси X для применения к элементу последовательности Возвращается: Н/Д Пример: if seq_scale < 2 { seq_scale += 0.01; layer_sequence_xscale(my_seq, seq_scale); layer_sequence_yscale(my_seq, seq_scale); } Приведенный выше код проверяет значение, хранящееся в переменной seq_scale, и если оно меньше 2, то прибавляет его, а затем использует это значение для установки масштаба X и Y элемента последовательности, на который ссылается переменная my_seq.
