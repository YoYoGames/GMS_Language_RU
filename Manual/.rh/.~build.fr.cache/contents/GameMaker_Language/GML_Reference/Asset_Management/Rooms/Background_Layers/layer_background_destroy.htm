layer_background_destroy Эта функция уничтожит заданный элемент фона. Вы передаете идентификатор фона (который вы получаете при создании фона с помощью команды layer_background_create() или когда вы используете идентификатор слоя вместе с с layer_get_background_id()), и эта функция удалит его. Обратите внимание, что при этом не удаляется слой, а только фон, и если фон был добавлен в редакторе комнаты то в следующий раз, когда вы покинете комнату, а затем вернетесь, фон будет создан заново. Однако если комната постоянна, фон будет удален, пока постоянство комнаты не будет снова отключено. Синтаксис: layer_background_destroy(background_element_id) Аргумент Описание background_element_id Уникальное значение ID фона, который должен быть уничтожен Возвращается: Н/Д Пример: var lay_id = layer_get_id("Background_trees"); var bck_id = layer_background_get_id("Forrest"); if layer_background_exists(lay_id, bck_id) { layer_background_destroy(bck_id); } Приведенный выше код проверяет слой "Background_trees" на предмет существования данного фонового элемента, и если он существует, то он уничтожается (но не слой).
