следующая комната С помощью этой функции вы можете получить индекс комнаты после ввода комнаты в функцию. Например, вы можете использовать переменную комната для получения индекса текущей комнаты, а затем использовать эту функцию для поиска комнату, которая следует за ней, как указано в диспетчере комнат. Если после введенной вами комнаты нет комнаты, то возвращается -1. Обратите внимание, что эта функция не распознает и не принимает во внимание комнаты, которые были добавлены динамически с помощью room_add() или room_duplicate(). Синтаксис: room_next(numb); Аргумент Описание numb Индекс комнаты, после которой нужно получить следующую. Возвращается: Индекс Пример: if room_next(room) != -1 { room_goto_next()); } Приведенный выше код проверит, существует ли следующая комната, и если да, то перейдет в нее.
