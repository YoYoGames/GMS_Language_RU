layer_sequence_is_paused В этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она проверит, приостановлена ли последовательность в данный момент или нет, возвращая true, если приостановлена, и false, если нет. Синтаксис: layer_sequence_is_paused(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Булево Пример: if layer_sequence_is_paused(title_sequence) != 0 { layer_sequence_play(title_sequence); } Приведенный выше код проверяет, находится ли идентификатор элемента последовательности, хранящийся в переменной "title_sequence", на паузе, и если да, то запускает его воспроизведение.
