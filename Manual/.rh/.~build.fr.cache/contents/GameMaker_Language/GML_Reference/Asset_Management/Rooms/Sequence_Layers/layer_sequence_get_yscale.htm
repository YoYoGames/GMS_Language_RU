layer_sequence_get_yscale С помощью этой функции вы предоставляете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она вернет текущий масштаб по оси Y элемента последовательности в игровой комнате. Синтаксис: layer_sequence_get_yscale(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Реально Пример: var _ys = layer_sequence_get_yscale(title_sequence) if _ys < 1 { _ys += 0.01; layer_sequence_yscale(title_sequence, _ys); } Приведенный выше код извлекает текущий масштаб по оси Y элемента последовательности с идентификатором, хранящимся в переменной "title_sequence", и если он меньше 1, то к текущему масштабу по оси Y добавляется 0,01.
