layer_sequence_get_length С помощью этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элементов слоя - и она вернет длину последовательности. Это количество кадров, в течение которых будет выполняться последовательность. Синтаксис: layer_sequence_get_length(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Целое число Пример: var _frames = layer_sequence_get_length(my_seq); alarm[0] = frames; Приведенный выше код извлекает количество кадров, которые будет выполнять последовательность, а затем использует это значение для установки сигнала тревоги.
