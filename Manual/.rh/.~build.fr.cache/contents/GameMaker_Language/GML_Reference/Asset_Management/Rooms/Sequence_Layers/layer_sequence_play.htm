layer_sequence_play С помощью этой функции вы можете запустить воспроизведение заданной последовательности. Вы передаете идентификатор элемента последовательности, возвращенный layer_sequence_create() или одной из функций элементов слоя, и функция начнет воспроизведение последовательности, которую при необходимости можно приостановить с помощью функции layer_sequence_pause(). ВАЖНО! Если в вашей последовательности есть экземпляры, эти экземпляры не должны менять свои image_xscale / image_yscale /. угол изображения / x / y переменные, так как они будут перезаписаны, когда последовательность обновляет каждый шаг после начала воспроизведения. Синтаксис: layer_sequence_play(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Н/Д Пример: if keyboard_check_pressed(ord("P")) { global.Pause = !global.Pause; var a = layer_get_all_elements(layer); for (var i = 0; i < array_length(a); i++;) { if layer_get_element_type(a[i]) == layerelementtype_sequence { if global.Pause { layer_sequence_pause(a[i]); } else { layer_sequence_play(a[i]); } } } } Приведенный выше код проверяет, была ли игра поставлена на паузу или нет, когда нажата клавиша. Если игра поставлена на паузу, то он перебирает все элементы последовательности на текущем слое (слое вызывающего экземпляра) и приостанавливает их воспроизведение, а если игра не приостановлена, то цикл снова запустит их воспроизведение.
