layer_sequence_headpos С помощью этой функции вы можете установить положение головки воспроизведения элемента последовательности на определенный кадр. Вы передаете идентификатор элемента последовательности, возвращенный layer_sequence_create() или одной из функций элемента слоя вместе с новой позицией, которую нужно установить. Обратите внимание, что позиция указывается в кадрах, и если вы зададите значение больше, чем общее количество кадров (или меньше 0), то фактическая конечная позиция головки воспроизведения будет зависеть от типа воспроизведения последовательности, который был выбран, в соответствии с этими правилами: Нет зацикливания: Головка воспроизведения будет прижиматься к концу цикла (или к началу, если заданное значение отрицательно). Зацикливание: Головка воспроизведения будет возвращаться к началу последовательности плюс дополнительные кадры, например: если вы установите головку воспроизведения в последовательности из 30 кадров на позицию кадра 45, фактическая позиция головки воспроизведения будет равна 15. Если вы зададите отрицательное значение, то головка воспроизведения последовательности будет разворачиваться в обратном направлении. Пинг-понговое зацикливание: Головка воспроизведения продвигается до конца, а затем возвращается назад на дополнительное количество кадров, превышающее общую длину кадра, меняя направление воспроизведения, например: Если последовательность состоит из 30 кадров, а вы установили головку воспроизведения на 40, конечная позиция головки воспроизведения будет 20, а направление будет обратным (вниз к 0). Обратите внимание, что никогда не следует задавать отрицательное значение для пинг-понг воспроизведения последовательности, так как это может привести к неопределенным результатам. Установка положения головки таким образом не остановит воспроизведение, и последовательность просто продолжится с нового положения головки воспроизведения, если она не приостановлена. Синтаксис: layer_sequence_headpos(sequence_element_id, position) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться позиция Позиция в последовательности (в кадрах), на которую нужно установить положение головки воспроизведения (может быть десятичным значением). Возвращается: Н/Д Пример: var a = layer_get_all_elements(layer); for (var i = 0; i < array_length(a); i++;) { if layer_get_element_type(a[i]) == layerelementtype_sequence { layer_sequence_headpos(a[i], 0) } } Приведенный выше код получает идентификаторы всех элементов, назначенных слою экземпляра, на котором выполняется код. Затем код проверяет, являются ли какие-либо из возвращенных элементов активами последовательности, и если да, то их положение головки воспроизведения устанавливается на 0.
