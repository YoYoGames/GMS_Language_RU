layer_get_script_begin Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и эта функция вернет индекс функции сценария, назначенной для запуска в начале рендеринга для этого слоя, или вернет -1, если функция не назначена. Вы можете назначить скриптовые функции слою с помощью layer_script_begin() и layer_script_end(). Синтаксис: layer_get_script_begin(layer_id); Аргумент Описание идентификатор слоя Уникальное значение ID слоя, на который нужно нацелиться (или имя слоя в виде строки). Возвращается: Индекс функции сценария (вещественный) или -1, если сценарий не назначен Пример: if layer_get_script_begin(layer) == -1 { layer_script_begin(layer, scr_SetShaderValues); } Приведенный выше код проверяет, назначена ли слою, на котором находится экземпляр, выполняющий код, функция сценария, и если нет, то назначается.
