tile_get_mirror Эта функция может быть использована для проверки того, была ли в заданном наборе тайл-данных зеркально отображена плитка или нет. Вы передаете данные тайла, которые могут быть получены с помощью функции tilemap_get(), и функция вернет true, если плитка зеркально отражена, или false, если нет. Синтаксис: tile_get_mirror(tiledata) Аргумент Описание tiledata данные плитки для проверки Возвращается: Булево Пример: var lay_id = layer_get_id("Tiles_sky"); var map_id = layer_tilemap_get_id(lay_id); var mx = tilemap_get_cell_x_at_pixel(map_id, mouse_x, mouse_y); var my = tilemap_get_cell_y_at_pixel(map_id, mouse_x, mouse_y); var data = tilemap_get(map_id, mx, my); var bool = !tile_get_mirror(data); data = tile_set_mirror(data, bool); tilemap_set(map_id, data, mx, my); Приведенный выше код получает ID карты плитки из заданного слоя, а затем получает позицию x и y ячейки для плитки под мышью. Эта позиция затем используется для получения данных плитки, которые зеркально отображаются и затем используются для повторной установки плитки.
