комната_существует С помощью этой функции вы можете проверить, существует ли указанная вами комната или нет. Эта функция принимает индекс комнаты (вещественное число), а не имя комнаты (строка). Эта функция наиболее полезна, когда вы создаете комнаты динамически с помощью функции room_add()но вы также можете использовать переменные, доступные только для чтения комната_первая и комната_последний или функции room_next() и комната_предыдущая() чтобы получить определенный индекс комнаты, или предоставить переменную, в которой хранится индекс любой другой комнаты в вашей игре. Синтаксис: room_exists(index); Аргумент Описание индекс Индекс помещения для проверки. Возвращается: Булево Пример: if room_exists(global.rm[0]) { room_goto(global.rm[0]); } Приведенный выше код проверяет, существует ли комната, указанная в массиве "global.rm[]", и если существует, то переходит в эту комнату.
