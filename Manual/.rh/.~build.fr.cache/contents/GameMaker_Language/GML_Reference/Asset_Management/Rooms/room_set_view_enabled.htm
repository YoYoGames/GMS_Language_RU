room_set_view_enabled С помощью этой функции вы можете включить(true) или выключить(false) просмотр любой комнаты в вашей игре, кроме текущей. Синтаксис: room_set_view_enabled(index, val); Аргумент Описание индекс Индекс комнаты, которую нужно установить. val Включить(true) или выключить(false) просмотр в данной комнате. Возвращается: Н/Д Пример: global.myroom = room_add(); room_set_view_enabled(global.myroom, true); Это позволит включить просмотры в комнате, проиндексированной в "global.myroom".
