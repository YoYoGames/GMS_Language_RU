скорость_слоя Эта функция может быть использована для установки скорости анимированного изображения, которое было назначено элементу спрайта. Вы передаете ID элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем задаете скорость анимации. Если в IDE задана скорость анимации для используемого спрайта, рассчитываемая как Frames Per Game Frame, то значение скорости является множителем, причем по умолчанию используется значение 1, а при установке значения 0,5 скорость анимации уменьшится вдвое, а при установке значения 1,5 - вдвое. Однако если в IDE установлена скорость анимации для используемого спрайта, которая рассчитывается как Frames Per Second, то указанное значение скорости является буквальным значением кадров в секунду, поэтому, например, установка значения 5 приведет к тому, что анимация будет повторяться по 5 кадров каждую секунду, пока идет игра. Если используемый спрайт не имеет вложенных изображений, эта функция не будет иметь никакого эффекта. Синтаксис: layer_sprite_speed(sprite_element_id, speed); Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для изменения скорость Множитель/значение скорости, используемое для элемента Возвращается: Н/Д Пример: var lay_id = layer_get_id("Assets_water"); var spr_id = layer_sprite_get_id(lay_id, "gfc_water"); layer_sprite_speed(spr_id, 0.5); Приведенный выше код получит идентификатор слоя для слоя с именем "Assets_water", а затем использует его для получения идентификатора заданного элемента спрайта на этом слое. Этот идентификатор затем используется для изменения скорости изображения элемента. Назад: Слои спрайта Далее: layer_sprite_xscale © Copyright YoYo Games Ltd. 2021 Все права защищены
