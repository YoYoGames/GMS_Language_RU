layer_background_get_index Эта функция может использоваться для получения текущего значения индекса изображения фонового элемента. Вы передаете идентификатор фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет реальное значение, которое представляет индекс изображения, отображаемого для спрайта. Функция вернет -1, если либо фоновый элемент не существует, либо элемент не имеет действительного спрайта, назначенного ему. Синтаксис: layer_background_get_index(background_element_id); Аргумент Описание background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию Возвращается: Real (индекс текущего изображения спрайта или -1) Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_index(back_id) < 4 { layer_background_index(back_id, 4); } Приведенный выше код получит идентификатор слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки, является ли индекс изображения для элемента меньше 4, и если да, то он устанавливается значение 4.
