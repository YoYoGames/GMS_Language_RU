layer_background_get_id Эта функция может использоваться для получения уникального значения ID элемента фона на слое. Вы передаете идентификатор слоя (который вы получаете, когда используете имя слоя вместе с параметром layer_get_id()) и функция вернет значение ID, связанное с фоновым элементом на слое. Обратите внимание, что эта функция предназначена для работы с фоновыми элементами, добавленными в IDE, так как если вы добавите фон к слою через код с помощью функции layer_background_create()то она вернет уникальный ID для добавленного элемента фона. Синтаксис: layer_background_get_id(layer_id) Аргумент Описание идентификатор слоя Уникальное значение идентификатора слоя, на который нужно нацелиться Возвращается: Идентификационное значение фонового элемента (Real) Пример: var lay_id = layer_get_id("Background_trees"); var back_id = layer_background_get_id(lay_id); layer_background_sprite(back_id, bck_Trees_Winter); Приведенный выше код получит идентификатор слоя для слоя с именем "Background_trees", а затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для изменения спрайта элемента. Назад: Фоновые слои Далее: layer_background_exists © Copyright YoYo Games Ltd. 2021 Все права защищены
