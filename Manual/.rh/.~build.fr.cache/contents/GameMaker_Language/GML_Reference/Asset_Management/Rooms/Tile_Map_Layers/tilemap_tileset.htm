набор плиток С помощью этой функции можно изменить ресурс набора плиток, назначенный данному элементу карты плиток на слое. Вы передаете идентификатор элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а затем указываете индекс набора плиток, и карте плиток будет присвоен новый спрайт. Синтаксис: tilemap_tileset(tilemap_element_id, tileset_index) Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки, которое необходимо изменить индекс_набора_плиток Новый индекс набора плиток для использования Возвращается: Н/Д Пример: var lay_id = layer_get_id("Tiles_trees"); var tile_id = layer_tilemap_get_id(lay_id); if tilemap_get_tileset(tile_id) != ts_Nighttime { tilemap_tileset(tile_id, ts_Nighttime); } Приведенный выше код проверяет текущий набор плиток, назначенный карте плиток на слое "Tiles_trees", и если он не "ts_Nighttime", то этот набор плиток назначается карте плиток.
