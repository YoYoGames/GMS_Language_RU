layer_set_target_room Вызывая эту функцию, вы сообщаете GameMaker Studio 2, что все дальнейшие функции слоев должны применяться к данной комнате. Таким образом можно процедурно изменять или генерировать слои и содержимое слоев в комнате, которая не является текущей. текущей комнатой. Когда вы закончите добавлять слои или элементы слоев в комнату, вызовите функцию layer_reset_target_room() для сброса цели помещения (или вызовите эту функцию снова с аргументом помещения равным -1). Обратите внимание, что эта функция может использоваться только для комнат, отличных от текущей комнаты, и предназначена для того, чтобы вы могли добавлять/удалять слои и элементы слоя в комнаты, отличные от текущей комнаты. ВНИМАНИЕ! При нацеливании на другую комнату вы можете использовать все обычные функции слоя, за исключением того, что вы не можете создавать экземпляры, используя instance_create_layer() или instance_create_depth(), также не будет доступна функция слоя layer_add_instance() не будет доступна. Синтаксис: layer_set_target_room(room) Аргумент Описание номер Помещение, предназначенное для всех дальнейших функций слоя Возвращается: Н/Д Пример: layer_set_target_room(rm_Game); var l = layer_get_id("SpriteAssets"); repeat(50) { var _x = random(room_get_width(rm_Game)); var _y = random(room_get_height(rm_Game)); layer_sprite_create(l, _x, _y, spr_Trees); } layer_reset_target_room(); Приведенный выше код устанавливает целевую комнату на комнату "rm_Game", а затем получает идентификатор слоя под названием "SpriteAssets" в этой комнате. Этот идентификатор слоя затем используется для добавления 50 случайных спрайтовых активов в слой, прежде чем цель слоя сбрасывается на текущую комнату.
