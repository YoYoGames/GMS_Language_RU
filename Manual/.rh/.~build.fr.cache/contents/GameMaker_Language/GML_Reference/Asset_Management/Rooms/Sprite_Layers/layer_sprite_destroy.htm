слой_спрайта_уничтожить Эта функция уничтожит заданный элемент спрайта. Вы передаете идентификатор спрайта (который вы получаете при создании спрайта с помощью команды layer_sprite_create() или когда вы используете идентификатор слоя вместе с layer_get_sprite_id()) и эта функция удалит его. Обратите внимание, что при этом удаляется не слой, а только спрайт из него, и если спрайт был добавлен в редакторе комнаты, то при следующем выходе из комнаты и последующем возвращении спрайт будет создан заново. снова. Однако если комната постоянна, спрайт будет удален, пока постоянство комнаты не будет снова отключено. Синтаксис: layer_sprite_destroy(sprite_element_id) Аргумент Описание sprite_element_id Уникальное значение ID уничтожаемого спрайта Возвращается: Н/Д Пример: var lay_id = layer_get_id("Asset_Trees"); if layer_sprite_exists(lay_id, global.Asset_spr_trees) { layer_sprite_destroy(lay_id); } Приведенный выше код проверяет слой "Asset_Trees" на существование заданного элемента спрайта, и если он существует, то уничтожает его (но не слой).
