имя_комнаты Эта функция может быть использована для возврата названия указанной комнаты в виде строки. Обратите внимание, что это только строка, и ее нельзя использовать для прямой ссылки на комнату - для этого вам понадобится индекс комнаты. Однако вы можете использовать эту строку для получения индекса комнаты, используя возвращенную строку вместе с функцией asset_get_index(). Синтаксис: room_get_name(index); Аргумент Описание индекс Индекс комнаты, название которой необходимо проверить. Возвращается: Строка Пример: var roomname = room_get_name(room); draw_text(32, 32, roomname); Приведенный выше код получит название текущей комнаты и нарисует его на экране.
