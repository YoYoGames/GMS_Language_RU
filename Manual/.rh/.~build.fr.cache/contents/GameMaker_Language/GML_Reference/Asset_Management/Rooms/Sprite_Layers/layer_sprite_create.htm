слой_спрайта_создать С помощью этой функции вы можете назначить ресурс спрайта слою, который будет использоваться в вашем проекте. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create() или когда вы используете имя слоя вместе с layer_get_id()), позицию в комнате и индекс спрайта (это имя спрайта, отображаемое в браузере активов), и он будет будет добавлен в слой. Функция возвращает уникальное значение ID для элемента, которое затем можно использовать в дальнейших функциях слоя для спрайтов. Синтаксис: layer_sprite_create(layer_id, x, y, sprite) Аргумент Описание идентификатор слоя Уникальное значение идентификатора слоя, на который нужно нацелиться x Позиция x для использования y Позиция y для использования спрайт Индекс спрайта, который будет использоваться Возвращается: Идентификатор элемента спрайта (реальный) Пример: global.asset_layer = layer_create(10000); for (var i = 0; i< 10; i++;) { var _x = random(room_width); var _y = room_height - 100; global.asset_spr_trees[i] = layer_sprite_create(global.asset_layer, _x, _y, spr_Trees); } Приведенный выше код создает новый слой, а затем добавляет в него 10 новых элементов спрайта, сохраняя ID каждого элемента в массив.
