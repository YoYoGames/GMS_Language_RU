tilemap_get_width С помощью этой функции можно получить ширину (в ячейках) элемента карты плиток. Вы передаете ID элемента карты плиток (который вы получаете при создании элемента карты плиток с помощью команды layer_tilemap_create() или когда вы используете функцию layer_tilemap_get_id()), а функция вернет ширину (количество ячеек в ширину). Синтаксис: tilemap_get_width(tilemap_element_id); Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки для получения ширины Возвращается: Целое число Пример: var lay_id = layer_get_id("Tiles_Walls"); var map_id = layer_tilemap_get_id(lay_id); var _w = tilemap_get_width(map_id); var _h = tilemap_get_height(map_id); total_tiles = (_w * _h); Приведенный выше код использует полученный идентификатор карты плиток для получения ширины и высоты карты плиток, а затем использует эти значения для установки переменной экземпляра.
