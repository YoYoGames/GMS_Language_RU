layer_background_get_speed Эта функция может использоваться для получения текущего значения множителя скорости фонового элемента. Вы передаете ID фонового элемента (который вы получаете при создании фонового элемента с помощью функции layer_background_create() или при использовании функции layer_background_get_id()), и функция вернет реальное значение, которое представляет собой множитель скорости, используемый для анимации спрайта. Значение по умолчанию равно 1. Синтаксис: layer_background_get_speed(background_element_id); Аргумент Описание background_element_id Уникальное значение ID элемента фона, из которого нужно получить информацию Возвращается: Реально Пример: var lay_id = layer_get_id("Background_sky"); var back_id = layer_background_get_id(lay_id); if layer_background_get_speed(back_id) > 0 { layer_background_speed(back_id, 0); } Приведенный выше код получит идентификатор слоя для слоя с именем "Background_sky" и затем использует его для получения идентификатора элемента фона на этом слое. Этот идентификатор затем используется для проверки скорости анимации элемента, и если он больше 0, он устанавливается на 0.
