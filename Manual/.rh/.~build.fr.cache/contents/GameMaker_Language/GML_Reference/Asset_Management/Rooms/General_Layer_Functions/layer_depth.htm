глубина слоя Эта функция может быть использована для изменения глубины заданного слоя, изменяя порядок вывода его содержимого на экран. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), а затем задаете новую глубину слоя (целочисленное значение от -16000 до 16000). Глубина слоя определяется как большая, если он находится "дальше" от камеры, и меньшая, если он находится "ближе" к камере. поэтому если у вас есть три слоя с глубиной -100, 0, 100, слои будут рисоваться в порядке 100, 0, -100, так что "верхним" слоем (т.е. самым близким к виду камеры и поэтому нарисованным поверх всех остальных) будет будет слой с глубиной -100. На следующем изображении показана схема работы глубины для слоев: Обратите внимание, что вы можете проверить глубину слоя в любое время с помощью функции layer_get_depth(). Также обратите внимание, что минимальная и максимальная глубина слоя составляет от -16000 до 16000, и все, что находится за пределами этих значений, не будет отрисовано. Если вам нужна глубина за пределами этого диапазона, то вам нужно использовать функцию layer_force_draw_depth(). Синтаксис: layer_depth(layer_id, depth) Аргумент Описание идентификатор слоя Уникальное значение ID слоя для установки глубины (или имя слоя в виде строки). глубина Новая глубина для слоя Возвращается: Н/Д Пример: if layer_get_depth(layer) != -100 { layer_depth(layer, -100); } Приведенный выше код получает глубину слоя, на котором находится экземпляр, выполняющий код, и если она не равна -100, то глубина устанавливается равной -100.
