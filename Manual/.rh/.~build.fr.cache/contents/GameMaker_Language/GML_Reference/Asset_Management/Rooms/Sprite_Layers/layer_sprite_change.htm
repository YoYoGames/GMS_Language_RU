слой_изменения С помощью этой функции вы можете изменить ресурс спрайта, назначенный данному элементу спрайта на слое. Вы передаете идентификатор элемента спрайта (который вы получаете при создании элемента спрайта с помощью функции layer_sprite_create() или при использовании функции layer_sprite_get_id()), а затем указать индекс элемента спрайта, ресурс которого будет изменен. Обратите внимание, что если спрайт имеет вложенные изображения, то они тоже будут анимироваться (это можно контролировать с помощью layer_sprite_index() и layer_sprite_speed() ). Обратите внимание, что вы можете присвоить значение -1 в качестве нового индекса спрайта, и тогда спрайт не будет показан, хотя элемент все еще будет существовать и может быть изменен позже. Синтаксис: layer_sprite_change(sprite_element_id, sprite_index) Аргумент Описание sprite_element_id Уникальное значение ID элемента спрайта для изменения sprite_index Новый индекс спрайта для использования Возвращается: Н/Д Пример: var s = layer_sprite_get_id(layer, global.Asset_sprite); if layer_sprite_get_sprite(s) != spr_Nighttime { layer_sprite_change(s, spr_nighttime); } Приведенный выше код проверяет индекс спрайта элемента с ID, хранящимся в глобальной переменной global.Asset_sprite, и если он не "spr_Nighttime", то этот спрайт назначается элементу.
