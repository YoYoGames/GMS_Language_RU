layer_get_vspeed Вы можете использовать эту функцию для получения вертикальной скорости (в пикселях на игровой кадр) слоя в текущей комнате. Вы передаете идентификатор слоя (который вы получаете при создании слоя с помощью функции layer_create()) или имя слоя (в виде строки - это влияет на производительность), и функция возвращает действительное число для вертикальной скорости, где положительное значение - вниз, а отрицательное - вверх. По умолчанию 0 (если не задано в редакторе помещений). Синтаксис: layer_get_vspeed(layer_id) Аргумент Описание идентификатор слоя Уникальное значение ID слоя для получения вертикальной скорости Возвращается: Н/Д Пример: var lay_id = layer_get_id("Sprites"); if layer_get_hspeed(lay_id) != 0 || layer_get_vspeed(lay_id) != 0 { layer_hspeed(lay_id, 0); layer_vspeed(lay_id, 0); } Приведенный выше код проверяет заданные горизонтальную и вертикальную скорости слоя, и если они обе не равны 0, то устанавливает их в 0.
