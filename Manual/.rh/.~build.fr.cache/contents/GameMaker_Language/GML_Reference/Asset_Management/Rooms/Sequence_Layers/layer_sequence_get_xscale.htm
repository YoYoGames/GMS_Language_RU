layer_sequence_get_xscale С помощью этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она вернет текущий масштаб по оси X элемента последовательности в игровой комнате. Синтаксис: layer_sequence_get_xscale(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Реально Пример: var _xs = layer_sequence_get_xscale(title_sequence) if _xs < 1 { _xs += 0.01; layer_sequence_xscale(title_sequence, _xs); } Приведенный выше код извлекает текущий масштаб по оси X элемента последовательности с идентификатором, хранящимся в переменной "title_sequence", и если он меньше 1, то к текущему масштабу X добавляется 0,01.
