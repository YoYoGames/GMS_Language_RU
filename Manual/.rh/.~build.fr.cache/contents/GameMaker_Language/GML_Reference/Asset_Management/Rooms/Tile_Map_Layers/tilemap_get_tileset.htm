tilemap_get_tileset С помощью этой функции можно получить значение индекса ресурса набора плиток, назначенного данному элементу карты плиток на слое. Вы передаете идентификатор элемента карты плитки (который вы получаете при создании элемента карты плитки с помощью функции layer_tilemap_create() или при использовании функции layer_tilemap_get_id()), а функция вернет значение индекса набора плиток. Синтаксис: tilemap_get_tileset(tilemap_element_id); Аргумент Описание tilemap_element_id Уникальное значение ID элемента карты плитки, из которого нужно получить набор плиток Возвращается: Плитка Установленное значение индекса (Real) Пример: var lay_id = layer_get_id("Tiles_trees"); var map_id = layer_tilemap_get_id(lay_id); if tilemap_get_tileset(map_id) != ts_Nighttime { tilemap_tileset(map_id, ts_Nighttime); } Приведенный выше код проверяет текущий набор плиток, назначенный слою "Tiles_trees", и если он не "ts_Nighttime", то этот набор плиток назначается на карту плиток.
