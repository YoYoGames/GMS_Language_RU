layer_sequence_get_y С помощью этой функции вы передаете идентификатор элемента последовательности - как возвращается функцией layer_sequence_create() или одной из функций элемента слоя - и она вернет текущую позицию Y в игровой комнате для последовательности. Синтаксис: layer_sequence_get_y(sequence_element_id) Аргумент Описание sequence_element_id Уникальное значение ID элемента последовательности, на который нужно нацелиться Возвращается: Реально Пример: if layer_sequence_get_y(title_sequence) != room_height / 2 { layer_sequence_y(title_sequence, room_height / 2); } Приведенный выше код проверяет положение Y идентификатора элемента последовательности, хранящегося в переменной "title_sequence", и если оно не установлено на половину высоты комнаты, то оно устанавливается на это значение.
