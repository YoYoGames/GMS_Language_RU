animcurve_destroy Эта функция может быть использована для уничтожения любых анимационных кривых, которые были ранее созданы с помощью функции animcurve_create(). Вызов этой функции удалит анимационную кривую из памяти и очистит все каналы или точки, которые она содержит (их никогда не нужно очищать вручную). Обратите внимание, что вы не можете уничтожить анимационные кривые, созданные в браузере активов, только те, которые были созданы во время выполнения. Синтаксис: animcurve_destroy(curve_struct); Аргумент Описание кривая_структура Указатель на структуру кривой, которую нужно уничтожить. Возвращается: Н/Д Пример: animcurve_destroy(my_curve); Приведенный выше код уничтожит (ранее созданную) структуру кривой анимации, индексированную в переменной "my_curve".
