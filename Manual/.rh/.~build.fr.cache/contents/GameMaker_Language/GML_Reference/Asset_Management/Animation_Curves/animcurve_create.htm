animcurve_create Эта функция создает пустую структуру анимационной кривой, готовую к заполнению данными каналов. Функция возвращает указатель на структуру, которая затем используется для добавления каналов и других данных к кривой анимации. Вы также можете передать его таким функциям, как animcurve_get() для последующего получения данных из кривой. Когда вы используете эту функцию, созданная структура не будет содержать никаких данных, и чтобы использовать ее, вы должны добавить хотя бы один канал, а в канал должны быть добавлены точки. Чтобы добавить канал, см. функцию animcurve_channel_new(), а чтобы добавить точки в канал, см. функцию animcurve_point_new(). Кроме того, вы можете дать структуре анимационной кривой имя, задав переменную "name" (как показано в примере кода, приведенном ниже). Каналы, добавленные в новую кривую анимации, должны быть добавлены в виде массива, где каждый элемент массива ссылается на структуру канала, а точки в канале также должны быть добавлены в виде массива, где каждый элемент массива ссылается на структуру точки. Обратите внимание, что созданные таким образом кривые анимации должны быть удалены, если они больше не нужны, так как в противном случае они будут занимать место в памяти, что может привести к утечке памяти, замедлению и, в конечном счете, краху вашей игры. Вы можете удалить кривую анимации, если она больше не нужна, с помощью функции animcurve_destroy(). Вам не нужно очищать данные канала или точки внутри кривой, так как это будет сделано сборщиком мусора автоматически при уничтожении самой кривой. Синтаксис: animcurve_create(); Возвращается: Ссылка на структуру Пример: my_curve = animcurve_create(); my_curve.name = "My_Curve"; var _channels = array_create(1); _channels[0] = animcurve_channel_new(); _channels[0].name = "alpha"; _channels[0].type = animcurvetype_catmullrom; _channels[0].iterations = 8; var _points = array_create(3); _points[0] = animcurve_point_new(); _points[0].posx = 0; _points[0].value = 0; _points[1] = animcurve_point_new(); _points[1].posx = 0.5; _points[1].value = 1; _points[2] = animcurve_point_new(); _points[2].posx = 1; _points[2].value = 0; _channels[0].points = _points; my_curve.channels = _channels; Приведенный выше код создает новую структуру анимационной кривой и сохраняет ее в переменной "my_curve". Затем эта структура заполняется именем и массивом каналов. Массив каналов содержит один канал с тремя точками.
