Анимационные кривые Кривые анимации можно настроить в браузере активов как активы с помощью редактора кривых анимации, а затем их можно использовать в редакторе последовательностей для управления дорожками последовательности. Редактор последовательностей для управления дорожками в последовательности. Однако их также можно использовать в коде для изменения значений во времени в играх - от эффектов переключения кнопок до движения, изменения цвета и многого другого. Вы также можете создавать свои собственные кривые анимации во время выполнения, используя перечисленные здесь функции, что делает их очень гибким инструментом для ваших игр.
