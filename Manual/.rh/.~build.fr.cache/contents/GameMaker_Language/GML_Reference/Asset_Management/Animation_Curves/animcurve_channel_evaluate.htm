animcurve_channel_evaluate Эта функция может быть использована для получения значения в определенный момент времени из структуры канала. Вы передаете указатель struct для канала (как возвращается функцией animcurve_get_channel(), или как возвращается в структуре анимационной кривой из функции animcurve_get()) и позицию "x" (время) вдоль канала кривой для оценки. Эта позиция должна быть между 0 и 1, и функция вернет значение кривой в этой позиции, или вернет 0, если предоставленная структура канала недействительна. Синтаксис: animcurve_channel_evaluate(channel_struct, posx); Аргумент Описание структура канала Указатель struct для оцениваемого канала. posx Положение во времени для проверки (от 0 до 1). Возвращается: Реально Пример: var _channel = animcurve_get_channel(ac_AlphaCurve, 0); var _val = animcurve_channel_evaluate(_channel, sin(current_time/1000)); image_alpha = _val; Приведенный выше код получает структуру канала для актива кривой анимации "ac_AlphaCurve". Затем он использует возвращаемое значение оценки для установки альфа-канала изображения для данного экземпляра.
