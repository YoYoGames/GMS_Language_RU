audio_get_listener_mask Эта функция возвращает данные битовой маски, определяющей текущую маску по умолчанию (глобальную) для аудиослушателей. "mask"]); global.Audio_Mask = global.Audio_Mask | m; ds_map_destroy(info); } audio_set_listener_mask(mask); } Приведенный выше код получает данные маски текущего слушателя и сравнивает их с данными, хранящимися в глобальной переменной. Если они не совпадают, код проверяет количество доступных слушателей, затем перебирает их и получает биты их масок, которые затем объединяются для создания единой маски, которая применяется к аудиосистеме для определения глобальных слушателей.
