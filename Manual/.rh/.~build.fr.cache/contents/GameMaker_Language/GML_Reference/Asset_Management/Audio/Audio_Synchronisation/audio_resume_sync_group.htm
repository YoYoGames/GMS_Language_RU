audio_resume_sync_group Эта функция возобновит заданную группу синхронизации, если она воспроизводится и ранее была приостановлена (с помощью функции audio_pause_sync_group). Индексом группы является значение, возвращенное при создании группы группу с помощью функции audio_create_sync_group(). ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5. Синтаксис: audio_resume_sync_group(group_index); Аргумент Описание групповой_индекс Индекс группы для возобновления. Возвращается: Н/Д Пример: if (keyboard_check_pressed((ord) "P")) { global.Pause = !global.Pause if global.Pause { audio_pause_sync_group(sg); } else { audio_resume_sync_group(sg); } } Приведенный выше код проверяет нажатие клавиши "P", и если оно обнаружено, то переключает переменную "global.Pause", затем проверяет ее на приостановку или возобновление группы синхронизации, индексированной в переменной "sg".
