audio_listener_velocity Эта функция может быть использована для придания слушателю эффекта Доплера и имитации движения звука на основе вектора, который определяется из заданных относительных положений x, y и z. Если сам слушатель никогда не будет двигаться, или движение будет или движение не является постоянным, вам обычно не нужно задавать эти значения, но, например, если вы создаете скроллинговую игру, где игрок имеет постоянное движение снизу вверх, а враги - сверху вниз, вы должны установить скорости слушателя и излучателя (для излучателей вы бы использовали audio_emitter_velocity()) на соответствующие векторы, чтобы смоделировать правильный эффект Доплера. эффект Доплера при движении мимо экземпляра игрока. ПРИМЕЧАНИЕ: если у вас несколько слушателей, вам следует использовать функцию audio_listener_set_velocity(). На рисунке ниже показано, как это можно настроить для приведенного выше примера игры: Синтаксис: audio_listener_velocity(vx, vy, vz); Аргумент Описание vx Компонент скорости x слушателя (по умолчанию 0). vy Компонент скорости y слушателя (по умолчанию 0). vz Компонент скорости z слушателя (по умолчанию 0). Возвращается: Н/Д Пример: если скорость > 0 { audio_listener_velocity(abs(hspeed), abs(vspeed), 0); } Приведенный выше код проверяет, не превышает ли скорость экземпляра игрока 0, и если да, то использует соответствующие абсолютные компоненты hspeed и vspeed для установки скорости слушателя.
