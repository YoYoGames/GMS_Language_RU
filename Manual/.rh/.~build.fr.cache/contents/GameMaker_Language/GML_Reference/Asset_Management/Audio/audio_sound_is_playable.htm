audio_sound_is_playable Эта функция может быть использована для проверки того, может ли заданный индекс звука воспроизводиться в данный момент. Это необходимо из-за различных способов воспроизведения потокового и непотокового звука на целевой платформе HTML5. возвращает true, если звук может быть воспроизведен, и false, если не может. Обратите внимание, что на всех других платформах, кроме HTML5, функция всегда будет возвращать true. Синтаксис: audio_sound_is_playable(index); Аргумент Описание индекс Индекс звука для проверки. Возвращается: Реально Пример: if audio_sound_get_pitch(snd_Explode) != 1 { audio_sound_pitch(snd_Explode, 1); } Приведенный выше код изменит высоту тона звука, воспроизводимого из звука, проиндексированного как "snd_Explode", если его высота тона не равна 1.
