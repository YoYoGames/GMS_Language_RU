audio_free_play_queue Эта функция используется для освобождения памяти, связанной с заданной аудиоочередью. Индекс очереди - это значение, возвращенное при создании очереди с помощью функции audio_create_play_queue()и эту функцию следует вызывать, когда очередь больше не нужна, чтобы предотвратить утечку памяти. Освобождение очереди остановит любой воспроизводимый звук, и вы не сможете удалить буфер, из которого передается звук, пока не освободится очередь, которой он назначен. пока не будет освобождена очередь. Эта функция вызовет асинхронное событие воспроизведения звука, и в этом событии будет создана специальная карта DS будет создана в переменной async_load со следующими парами ключ/значение: " queue_id" - очередь, которая закончила воспроизведение данных " buffer_id" - буфер, воспроизведение из которого прекращено " queue_shutdown" - устанавливается в 0 при обычном воспроизведении и в 1 при получении события, поскольку вы освободили очередь с помощью этой функции. Когда это значение равно 1, вы не хотите больше ставить данные в очередь. Синтаксис: audio_free_play_queue(queueIndex); Аргумент Описание queueIndex Индекс очереди, которую нужно освободить.
