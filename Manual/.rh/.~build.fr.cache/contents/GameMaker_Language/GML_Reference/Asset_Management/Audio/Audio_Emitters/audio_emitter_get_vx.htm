audio_emitter_get_vx Эта функция возвращает текущую скорость вдоль оси x для данного звукового излучателя. Синтаксис: audio_emitter_get_vx(emitter); Аргумент Описание эмиттер Индекс эмиттера, который необходимо использовать. Возвращается: Реально Пример: if audio_emitter_get_vx(emitter_player) != 0 { audio_emitter_velocity(emitter_player, 0, 0, 0, 0); } Приведенный выше код проверяет текущую x-скорость данного эмиттера, и если она не равна 0, то она устанавливается в 0.
