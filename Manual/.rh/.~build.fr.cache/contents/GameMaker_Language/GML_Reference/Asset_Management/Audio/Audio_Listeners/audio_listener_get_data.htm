audio_listener_get_data Эта функция создаст карту DS и заполнит ее значениями позиции, скорости и ориентации для заданного слушателя. По умолчанию индекс слушателя равен 0, но вы можете использовать функцию audio_get_listener_info() для получения различных индексов, доступных для целевой платформы. Если вы укажете неправильный индекс слушателя, функция вернет -1. ПРИМЕЧАНИЕ: Вы несете ответственность за уничтожение возвращенной карты DS с помощью соответствующей функции. Карта DS будет содержать следующие ключи: " x" - Позиция x слушателя " y" - Положение слушателя по оси y " z" - Положение слушателя по оси z " vx" - скорость слушателя по оси x " vy" - скорость слушателя по оси y " vz" - скорость слушателя по оси z " lookat_x" - x-компонент вектора взгляда слушателя " lookat_y" - Компонент y вектора взгляда слушателя " lookat_z" - Компонент z вектора взгляда слушателя. " up_x"- Компонент x вектора вверх слушателя " up_y"- Компонент y вектора вверх слушателя " up_z"- компонент z вектора вверх слушателя Синтаксис: audio_listener_get_data(index); Аргумент Описание индекс Слушатель, для которого необходимо получить данные (по умолчанию 0). Возвращается: ds_map Пример: var num = audio_get_listener_count(); for(var i = 0; i < num; ++i;) { var info = audio_get_listener_info(i); var data = audio_listener_get_data(info[? "index"]); if data[? "x"] != 0 { audio_listener_set_position(info[? "index"], 0, 0, 0, 0); } ds_map_destroy(info); ds_map_destroy(data); } Приведенный выше код проверяет количество доступных слушателей, затем перебирает их и, если их положение x не равно 0, устанавливает их положение в 0, 0, 0.
