audio_get_listener_count Некоторые целевые платформы позволяют использовать более одного слушателя, поэтому важно знать, сколько их у цели, прежде чем менять или использовать различные слушатели. Эта функция возвращает количество доступных слушателей. Синтаксис: audio_get_listener_count(); Возвращается: Целое число Пример: global.listener_num= audio_get_listener_count(); Приведенный выше код получает количество доступных слушателей и сохраняет возвращаемое значение в глобальной переменной.
