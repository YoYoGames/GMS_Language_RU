audio_group_set_gain С помощью этой функции можно приглушать или затухать группу звуков в течение заданного времени или мгновенно устанавливать усиление группы. Время измеряется в миллисекундах, и функция требует, чтобы вы ввели конечный уровень усиления для группа должна достичь к концу этого времени. Этот уровень усиления может быть от 0 (тихий) до 1 (полная громкость), шкала линейная, так что значение 0,5 означает половину громкости. Чтобы мгновенно изменить коэффициент усиления, просто установите аргумент time в 0. Обратите внимание, что на некоторых платформах коэффициент усиления может быть больше 1, хотя значение 1 считается "полной громкостью", а большее значение может привести к обрыву звука. Синтаксис: audio_group_set_gain(groupID, volume, time); Аргумент Описание groupID Индекс аудиогруппы, которую необходимо остановить (как определено в окне Аудиогруппы). объем Окончательное значение для группового объема. время Длительность изменения усиления в миллисекундах. Возвращается: Н/Д Пример: if keyboard_check_pressed(vk_space) { audio_group_set_gain(audiogroup1, 0, 5000); } Приведенный выше код проверяет наличие клавиши "пробел" и затем затухает все аудио для "audiogroup1" до 0 в течение 5 секунд.
