audio_emitter_get_vy Эта функция возвращает текущую скорость вдоль оси y для данного звукового излучателя. Синтаксис: audio_emitter_get_vy(emitter); Аргумент Описание эмиттер Индекс эмиттера, который необходимо использовать. Возвращается: Реально Пример: if audio_emitter_get_vy(emitter_player) != 0 { audio_emitter_velocity(emitter_player, 0, 0, 0, 0); } Приведенный выше код проверяет текущую скорость y данного эмиттера, и если она не равна 0, то она устанавливается равной 0.
