audio_sync_group_is_playing Эта функция может быть использована для проверки того, воспроизводится ли аудио в синхронизированной группе. Вам необходимо предоставить идентификатор группы синхронизации, возвращаемый функцией audio_create_sync_group(). ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5. Синтаксис: audio_sync_group_is_playing(group_index); Аргумент Описание групповой_индекс Групповой индекс для проверки. Возвращается: Н/Д Пример: if audio_sync_group_is_playing(group_one) { audio_stop_sync_group(group_one); } Приведенный выше код проверяет, играет ли в данный момент группа синхронизации group_one, и если да, то группа останавливается.
