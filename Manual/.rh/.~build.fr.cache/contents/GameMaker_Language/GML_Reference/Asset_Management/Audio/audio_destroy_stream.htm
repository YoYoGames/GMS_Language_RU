audio_destroy_stream Если вы ранее создали звуковой поток из файла с помощью функции audio_create_stream() и вам больше не нужен этот звук, вы можете вызвать эту функцию, чтобы удалить его из памяти. Любые дальнейшие обращения к индексу звука после его уничтожения приведут к ошибке. Следует отметить, что это освободит поток, но на целевой платформе это может не отобразиться в менеджере памяти. Это происходит потому, что GameMaker Studio 2 объединяет ресурсы памяти для предотвращения накладных расходов на выделение памяти, поэтому память будет оставаться выделенной до тех пор, пока не понадобится для чего-то другого или повторного использования для нового потока. ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5. Синтаксис: audio_destroy_stream(filename); Аргумент Описание имя файла Файл, из которого будет передаваться аудио. Возвращается: NA Пример: audio_destroy_stream(snd); Приведенный выше код удаляет из памяти звук, индексированный в переменной "snd".
