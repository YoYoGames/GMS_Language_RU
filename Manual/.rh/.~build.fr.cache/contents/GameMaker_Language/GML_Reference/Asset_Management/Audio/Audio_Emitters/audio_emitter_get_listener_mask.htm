audio_emitter_get_listener_mask Эта функция возвращает данные битовой маски, определяющие, с каких аудиослушателей эмиттер должен воспроизводить звуки. Для получения дополнительной информации см. раздел " Аудиопрослушиватели". Синтаксис: audio_emitter_get_listener_mask(emitterID); Аргумент Описание emitterID Уникальный идентификатор излучателя для получения маски Возвращается: Реально Пример: s_emit = audio_emitter_create(); if audio_emitter_get_listener_mask(s_emit) != global.PlayerMask { audio_emitter_set_listener_mask(snd, global.PlayerMask); } Приведенный выше код создает эмиттер, затем проверяет данные маски слушателя для него, и если они не совпадают с данными, хранящимися в глобальной переменной, то устанавливает слушателя(ей) для воспроизведения, используя данные маски, хранящиеся в глобальной переменной.
