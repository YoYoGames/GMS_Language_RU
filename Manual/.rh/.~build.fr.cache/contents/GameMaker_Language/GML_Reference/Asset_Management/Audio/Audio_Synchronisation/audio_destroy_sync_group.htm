audio_destroy_sync_group Группы синхронизации звука должны быть уничтожены, если они не используются, чтобы освободить память и звуковые ресурсы, связанные с ними, с помощью этой функции. Она принимает индекс группы, который был возвращен при создании группы с помощью функции audio_create_sync_group(), и освобождает все ресурсы, используемые группой. ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5. Синтаксис: audio_destroy_sync_group(group_index); Аргумент Описание групповой_индекс Групповой индекс должен быть уничтожен. Возвращается: Н/Д Пример: audio_destroy_sync_group(sg); Приведенный выше код уничтожает группу синхронизации, индексированную в переменной "sg", и, вероятно, будет использоваться в событиях destroy или Room End.
