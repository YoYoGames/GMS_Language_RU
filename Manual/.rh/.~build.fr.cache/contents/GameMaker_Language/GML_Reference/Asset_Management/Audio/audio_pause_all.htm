audio_pause_all С помощью этой функции вы можете приостановить все воспроизводимые в данный момент звуки. Синтаксис: audio_pause_all(); Возвращается: Н/Д Пример: if keyboard_check_pressed(ord("P")) { global.Pause = !global.Pause; if global.Pause { audio_pause_all(); } else { audio_resume_all(); } } Приведенный выше код проверяет нажатие клавиши "P" и, если обнаруживает ее, устанавливает глобальную переменную "Pause" в true или false, а затем либо приостанавливает все звуки, либо перезапускает все ранее приостанавливает все звуки, либо перезапускает все ранее приостановленные звуки.
