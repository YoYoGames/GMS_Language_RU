audio_emitter_position С помощью этой функции вы можете изменить положение аудиоизлучателя в трехмерном аудиопространстве. Положение будет по-разному влиять на звук в зависимости от того, где находится слушатель в аудиопространстве (по умолчанию положение равно (0, 0, 0). См. audio_listener_position() дополнительную информацию), поэтому, например, если позиция эмиттера установлена на (100, 0, 0) и текущий слушатель находится на (200, 0, 0), то звук, передаваемый от эмиттера, будет отображаться слева от звукового поля. На рисунке ниже показано визуальное представление эмиттеров и их относительных позиций по отношению к слушателю: Синтаксис: audio_emitter_position(emitter, x, y, z); Аргумент Описание эмиттер Индекс излучателя, который нужно изменить. x Положение x. y Положение по оси y. z Положение z. Возвращается: Н/Д Пример: if speed > 0 { audio_emitter_position(s_emit, x, y, 0); } Приведенный выше код проверяет, превышает ли скорость экземпляра 0, и если да, то обновляет звуковой эмиттер, индексированный в переменной "s_emit", до текущей позиции x/y.
