audio_sound_set_track_position Эта функция устанавливает позицию (в секундах) в звуковом файле для воспроизведения звука. Звук может быть либо одним экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() ) или звуковой актив. Если используется уникальный идентификатор звука, то позиция немедленно изменится на новое заданное время, а если это актив Asset Browser, то все последующие проигрывания данного звука будут начинаться с нового времени. Функция будет игнорировать любое заданное значение, которое больше, чем длина звука, заданного для значения "index". значение. Синтаксис: audio_sound_set_track_position(index, time); Аргумент Описание индекс Индекс звука, который нужно изменить. время Время (в секундах) для установки начальной точки. Возвращается: Н/Д Пример: var snd = audio_play_sound(snd_MainTrack, 0, false); audio_sound_set_track_position(snd, 32); Приведенный выше код воспроизводит звук, а затем использует возвращаемое значение ID для установки начальной позиции звука на 32 секунды.
