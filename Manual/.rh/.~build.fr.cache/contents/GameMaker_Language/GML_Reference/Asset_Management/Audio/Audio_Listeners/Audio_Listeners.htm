Слушатели аудио При работе со сложными аудиоситуациями бывает полезно определить, где в аудиосреде должен находиться слушатель. При работе с простыми аудиофункциями обычно нет необходимости изменять слушателя каким-либо образом, Однако при использовании эмиттеров для создания реалистичного звука в трехмерном аудиопространстве важно, чтобы слушатель был изменен и перемещен правильно, иначе звук будет находиться в неправильном положении для человека, играющего в вашу игру. GameMaker Studio 2 предоставляет следующие функции слушателей: audio_listener_position audio_listener_velocity ориентация аудио_слушателя audio_listener_get_data audio_listener_set_position audio_listener_set_velocity audio_listener_set_orientation Некоторые целевые платформы могут допускать более одного слушателя, каждый из которых соответствует отдельному устройству вывода, т.е.: ваша целевая платформа может допускать два слушателя, один для наушников и один для монитора. Поэтому GameMaker Studio 2 предоставляет вам ряд дополнительных функций, с помощью которых вы можете найти доступные слушатели, а также установить их для воспроизводимых звуков: audio_get_listener_count audio_get_listener_info audio_get_listener_mask audio_set_listener_mask Также есть функции для получения и установки данных маски для отдельных звуков: audio_sound_get_listener_mask audio_sound_set_listener_mask А также для получения и установки данных маски для эмиттеров: audio_emitter_get_listener_mask audio_emitter_set_listener_mask Назад: Аудио Следующее: Аудиогруппы © Copyright YoYo Games Ltd. 2021 Все права защищены
