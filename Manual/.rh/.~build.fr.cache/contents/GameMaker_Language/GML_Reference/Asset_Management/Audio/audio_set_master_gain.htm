audio_set_master_gain С помощью этой функции вы можете установить абсолютное значение глобальной громкости всех звуков и музыки для определенного слушателя. По умолчанию индекс слушателя равен 0, но вы можете использовать функцию audio_get_listener_info() чтобы получить различные индексы, доступные для целевой платформы. Значение усиления основано на линейной шкале от 0 (тишина) до 1 (полная громкость) и влияет на относительную громкость всех звуков и музыки, воспроизводимых в игре через этот слушатель. Синтаксис: audio_set_master_gain(listenerIndex, gain); Аргумент Описание listenerIndex Индекс слушателя для установки усиления. усиление Значение для глобального объема (от 0 до 1). Возвращается: Н/Д Пример: var num = audio_get_listener_count(); for( var i = 0; i < num; i++;) { var info = audio_get_listener_info(i); audio_set_master_gain(info[? "index"], 0.75); ds_map_destroy(info); } Приведенный выше код перебирает доступные слушатели, а затем устанавливает их главный коэффициент усиления на 0,75.
