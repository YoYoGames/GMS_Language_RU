audio_group_unload Эта функция выгружает в память все звуки, отмеченные как принадлежащие данной аудиогруппе. Функция вернет true, если выгрузка начата, и false, если ID группы недействителен, или она уже была помечена для выгрузки. Обратите внимание, что все звуки, воспроизводимые в данный момент, при вызове этой функции будут остановлены. Синтаксис: audio_group_unload(groupID); Аргумент Описание groupID Индекс аудиогруппы для выгрузки (как определено в окне Аудиогруппы). Возвращается: Булево Пример: if audio_group_is_loaded(audiogroup_level1) { audio_group_unload(audiogroup_level1); } Приведенный выше код проверяет, была ли загружена аудиогруппа, и если была, то выгружает ее.
