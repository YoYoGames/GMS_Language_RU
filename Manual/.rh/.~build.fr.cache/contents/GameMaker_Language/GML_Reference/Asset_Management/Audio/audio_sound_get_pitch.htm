audio_sound_get_pitch Эта функция может быть использована для получения высоты тона заданного звука. Звук может быть либо ссылкой на индекс отдельного воспроизводимого звука, который был сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() или фактический звуковой актив из браузера активов. Синтаксис: audio_sound_get_pitch(index); Аргумент Описание индекс Индекс звука для получения высоты тона. Возвращается: Реально Пример: if audio_sound_get_pitch(snd_Explode) != 1 { audio_sound_pitch(snd_Explode, 1); } Приведенный выше код изменит высоту тона звука, воспроизводимого из звука, проиндексированного как "snd_Explode", если его высота тона не равна 1.
