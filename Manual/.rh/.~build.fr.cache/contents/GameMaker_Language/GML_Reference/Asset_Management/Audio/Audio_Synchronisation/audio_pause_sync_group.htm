audio_pause_sync_group эта функция приостановит заданную группу синхронизации, если она играет, при этом индекс группы будет значением, возвращенным при создании группы с помощью функции audio_create_sync_group(). Это не останавливает звук, и вызов функции audio_resume_sync_group()начнет его воспроизведение с той же позиции, на которой он был приостановлен. ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5. Синтаксис: audio_pause_sync_group(group_index); Аргумент Описание групповой_индекс Группа сделала паузу. Возвращается: Н/Д Пример: if (keyboard_check_pressed((ord) "P")) { global.Pause = !global.Pause if global.Pause { audio_pause_sync_group(sg); } else { audio_resume_sync_group(sg); } } Приведенный выше код проверяет нажатие клавиши "P", и если оно обнаружено, то переключает переменную "global.Pause", затем проверяет ее на приостановку или возобновление группы синхронизации, индексированной в переменной "sg".
