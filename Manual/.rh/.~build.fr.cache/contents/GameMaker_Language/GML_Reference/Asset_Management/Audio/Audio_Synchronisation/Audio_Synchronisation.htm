Синхронизация звука Функции группы синхронизации звука позволяют воспроизводить несколько звуков, которые начинаются в одно и то же время и остаются синхронизированными (на уровне каждого сэмпла). Это позволяет (например) добавлять различные дорожки для одной песни и затем включать и выключать секции без не беспокоясь о том, будет ли между ними задержка. Обратите внимание, что в настоящее время эти функции работают только со сжатым аудио, т.е. с файлами формата *.ogg и *.mp3. ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5.
