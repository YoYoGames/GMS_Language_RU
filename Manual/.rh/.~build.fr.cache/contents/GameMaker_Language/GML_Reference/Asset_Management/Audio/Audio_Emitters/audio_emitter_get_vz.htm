audio_emitter_get_vz Эта функция возвращает текущую скорость вдоль оси z для данного звукового излучателя. Синтаксис: audio_emitter_get_vz(emitter); Аргумент Описание эмиттер Индекс эмиттера, который необходимо использовать. Возвращается: Реально Пример: if audio_emitter_get_vz(emitter_player) != 0 { audio_emitter_velocity(emitter_player, 0, 0, 0, 0); } Приведенный выше код проверяет текущую скорость z данного эмиттера, и если она не равна 0, то она устанавливается равной 0.
