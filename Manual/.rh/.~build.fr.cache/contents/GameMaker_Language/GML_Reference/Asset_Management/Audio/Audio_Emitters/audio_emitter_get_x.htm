audio_emitter_get_x Эта функция возвращает текущее положение x заданного звукового излучателя. Синтаксис: audio_emitter_get_x(emitter); Аргумент Описание эмиттер Индекс эмиттера, который необходимо использовать. Возвращается: Реально Пример: if audio_emitter_get_x(emitter_player) != x { audio_emitter_position(emitter_player, x, y, 0); } Приведенный выше код проверяет текущее положение x данного эмиттера, и если оно не равно положению x экземпляра, то оно устанавливается в положение экземпляра.
