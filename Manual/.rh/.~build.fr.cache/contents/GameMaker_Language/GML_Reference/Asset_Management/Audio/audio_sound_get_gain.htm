audio_sound_get_gain Эта функция возвращает текущее значение усиления для заданного звука. Звук может быть либо одним, ссылающимся на индекс отдельного воспроизводимого звука, который был сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() или фактический звуковой актив из браузера активов. Усиление обычно рассчитывается как значение от 0 до 1, но на некоторых платформах вы можете иметь усиление больше 1, хотя значение 1 считается "полная громкость", а большее значение может привести к обрыву звука. Синтаксис: audio_sound_get_gain(index); Аргумент Описание индекс Индекс звука для получения коэффициента усиления. Возвращается: Реально Пример: if audio_sound_get_gain(snd_Music) != 1 { audio_sound_gain(snd_Music, 1, 0); } Приведенный выше код изменит усиление звука, воспроизводимого из звука, проиндексированного как "snd_Music", если его усиление не равно 1.
