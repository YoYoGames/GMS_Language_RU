audio_exists Эта функция возвращает, существует ли звук(true) или нет(false). Обратите внимание, что если искомый индекс не был инициализирован ранее, эта функция вызовет ошибку, так как она ищет несуществующие индексы активов. звук для проверки может быть либо одним экземпляром звука (индекс для отдельных воспроизводимых звуков может быть сохранен в переменной при использовании функции audio_play_sound() или audio_play_sound_at() функции) либо звуковой актив. Синтаксис: audio_exists(index); Аргумент Описание индекс Индекс звука, существование которого необходимо проверить. Возвращается: Булево Пример: if audio_exists(global.Music) { audio_play_sound(global.Music, 0, true); } Приведенный выше код проверяет, существует ли звук.
