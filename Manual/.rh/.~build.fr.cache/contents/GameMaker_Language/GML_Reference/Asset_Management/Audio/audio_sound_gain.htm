аудио_звук_усиление С помощью этой функции можно приглушить или затушить звук на определенный промежуток времени, а также мгновенно настроить усиление звука. Время измеряется в миллисекундах, и функция требует, чтобы вы ввели конечный уровень усиления звука, который должен быть достигнут к концу этого времени. достигнут к концу этого времени. Этот уровень усиления может быть от 0 (без звука) до любого значения, превышающего 0, хотя обычно максимальный уровень громкости считается равным 1. вы можете получить искажения или клиппинг при воспроизведении звука. Обратите внимание, что шкала усиления линейна, и чтобы мгновенно изменить усиление, просто установите аргумент времени равным 0. Эта функция будет воздействовать на все экземпляры звука, воспроизводимые в данный момент в комнате, если индекс является звуковым ресурсом, и конечная громкость будет громкостью, на которой будут воспроизводиться все последующие экземпляры звука. воспроизводиться. Однако если вы использовали индекс, возвращенный из функции типа audio_play_sound() это повлияет только на один экземпляр звука. Синтаксис: audio_sound_gain(index, volume, time); Аргумент Описание индекс Индекс звука, для которого нужно установить усиление. объем Значение громкости музыки. время Длительность изменения усиления в миллисекундах. Возвращается: Н/Д Пример: if val { var snd = audio_play_sound(snd_fountain); audio_sound_gain(snd, 0, 0); audio_sound_gain(snd, 1, 5000); } Приведенный выше код проверяет переменную и, если он возвращает true, то присваивает индекс воспроизводимого звука локальной переменной "snd". Эта переменная затем используется для уменьшения громкости этого конкретного звука до 0 и увеличивается до полной громкости в течение 5 секунд.
