audio_start_sync_group С помощью этой функции вы можете начать воспроизведение ранее созданной группы синхронизации. Вам необходимо предоставить индекс группы, который является значением, возвращенным при создании группы с помощью функции audio_create_sync_group(). ПРИМЕЧАНИЕ: Эта функциональность недоступна для целевой платформы HTML5. Синтаксис: audio_start_sync_group(group_index); Аргумент Описание групповой_индекс Групповой индекс для игры. Возвращается: Н/Д Пример: sg = audio_create_sync_group(true); audio_play_in_sync_group(sg, sound1); audio_play_in_sync_group(sg, sound2); audio_sound_gain(sound2, 0, 0); audio_play_in_sync_group(sg, sound3); audio_sound_gain(sound3, 0, 0); audio_play_in_sync_group(sg, sound4); audio_sound_gain(sound4, 0, 0); audio_start_sync_group(sg); Вышеописанное создает новую группу синхронизации и присваивает индекс группы переменной "sg". Затем в группу добавляются четыре звука, причем усиление для трех из них устанавливается на 0. Наконец, группа синхронизации воспроизводится.
