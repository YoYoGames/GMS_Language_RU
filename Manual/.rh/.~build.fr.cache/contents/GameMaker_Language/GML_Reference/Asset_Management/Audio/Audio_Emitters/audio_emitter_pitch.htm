audio_emitter_pitch Эта функция может быть использована для изменения высоты тона всех звуков, издаваемых данным излучателем. Это множитель высоты тона, то есть вводимое значение умножает текущую высоту тона на данное значение, поэтому значение по умолчанию 1 не изменяет высоту тона, Значение меньше 1 понижает высоту тона, а больше 1 - повышает. Для этой функции лучше всего использовать небольшие приращения, поскольку любое значение меньше 0 или больше 5 может быть не слышно. ПРИМЕЧАНИЕ: Если звук проходит через излучатель по циклу, изменение высоты тона не будет обнаружено, пока звук не будет остановлен и снова пропущен по циклу! Синтаксис: audio_emitter_pitch(emitter, pitch); Аргумент Описание эмиттер Индекс излучателя, который нужно изменить. шаг Множитель шага (по умолчанию 1). Возвращается: Н/Д Пример: switch (gear) { case 1: audio_emitter_pitch(s_emit, 0.8); break; case 2: audio_emitter_pitch(s_emit, 0.9); break; case 3: audio_emitter_pitch(s_emit, 0.95); break; case 4: audio_emitter_pitch(s_emit, 1); break; case 5: audio_emitter_pitch(s_emit, 1.2); break; } Приведенный выше код изменит высоту тона звука, воспроизводимого эмиттером, индексированным в переменной "s_emit", в зависимости от значения переменной "gear".
