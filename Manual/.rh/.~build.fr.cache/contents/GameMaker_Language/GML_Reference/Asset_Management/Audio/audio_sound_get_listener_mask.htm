audio_sound_get_listener_mask Эта функция возвращает данные битовой маски, определяющие, с каких аудиопрослушивателей должен воспроизводиться звук. Дополнительную информацию см. в разделе " Слушатели звука ". Синтаксис: audio_sound_get_listener_mask(soundID); Аргумент Описание soundID Уникальный идентификатор звука, маску которого необходимо получить Возвращается: Реально Пример: var snd = audio_play_sound(snd_PlayerDead, 10, false); if audio_sound_get_listener_mask(snd) != global.PlayerMask { audio_sound_set_listener_mask(snd, global.PlayerMask); } Приведенный выше код воспроизводит звук, затем проверяет данные маски слушателя для этого звука, и если они не совпадают с данными, хранящимися в глобальной переменной, то устанавливает слушателя(ей) для воспроизведения, используя данные маски, хранящиеся в глобальной переменной.
