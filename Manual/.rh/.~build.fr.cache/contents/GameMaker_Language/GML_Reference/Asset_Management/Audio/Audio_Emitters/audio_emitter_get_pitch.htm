audio_emitter_get_pitch Эта функция возвращает текущее значение высоты тона, установленное для данного звукового излучателя. Синтаксис: audio_emitter_get_pitch(emitter); Аргумент Описание эмиттер Индекс эмиттера, который необходимо использовать. Возвращается: Реально Пример: if audio_emitter_get_pitch(emitter_player) != 1 { audio_emitter_pitch(emitter_player, 1); } Приведенный выше код проверяет текущий шаг данного эмиттера, и если он не равен 1, то он устанавливается в 1.
