audio_master_gain С помощью этой функции можно установить абсолютное значение глобальной громкости всех звуков и музыки. Она основана на линейной шкале от 0 (тишина) до любого значения больше 0, хотя обычно максимальная громкость считается равной 1. Все, что больше 1 можно использовать, но в зависимости от используемого звука и платформы, под которую он компилируется, при воспроизведении звука могут возникнуть искажения или обрывы. Эта функция влияет на относительную громкость всех звуков и музыки, воспроизводимых в игре. Синтаксис: audio_master_gain(gain); Аргумент Описание усиление Значение для глобального объема (от 0 до 1). Возвращается: Н/Д Пример: if keyboard_check(vk_up) { if vol < 1 vol += 0.05; audio_master_gain(vol); } if keyboard_check(vk_down) { if vol > 0 vol -= 0.05; audio_master_gain(vol); } Приведенный выше код проверяет нажатия клавиш со стрелками вверх и вниз, которые увеличивают или уменьшают переменную "vol". Эта переменная затем используется для установки глобального усиления всех звуков и музыки в игре. Назад: Аудио Далее: audio_set_master_gain © Copyright YoYo Games Ltd. 2021 Все права защищены
