audio_emitter_get_y Эта функция возвращает текущее положение y заданного звукового излучателя. Синтаксис: audio_emitter_get_y(emitter); Аргумент Описание эмиттер Индекс эмиттера, который необходимо использовать. Возвращается: Реально Пример: if audio_emitter_get_y(emitter_player) != x { audio_emitter_position(emitter_player, x, y, 0); } Приведенный выше код проверяет текущую позицию y данного эмиттера, и если она не равна позиции y экземпляра, то она устанавливается в позицию экземпляра.
