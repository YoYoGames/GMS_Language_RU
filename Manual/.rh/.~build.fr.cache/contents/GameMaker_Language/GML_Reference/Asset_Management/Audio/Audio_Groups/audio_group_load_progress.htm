audio_group_load_progress Эта функция проверяет прогресс загрузки для аудиогруппы и возвращает (приблизительное) значение от 0 до 100. Синтаксис: audio_group_load_progress(groupID); Аргумент Описание groupID Индекс аудиогруппы, которую необходимо проверить (как определено в окне Аудиогруппы). Возвращается: Реально Пример: if load { var pc = audio_group_load_progress(audiogroup_level1); draw_text(32, 32, "Loading: " + string(pc)); } Приведенный выше код проверяет переменную, и если она возвращает true, то будет нарисован текст, показывающий прогресс загрузки аудио.
