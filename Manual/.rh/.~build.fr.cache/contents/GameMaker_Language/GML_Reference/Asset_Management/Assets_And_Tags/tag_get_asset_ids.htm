tag_get_asset_ids С помощью этой функции вы можете получить все активы заданного типа, которым присвоены заданные теги. Вы передаете либо один тег (в виде строки), либо массив, где каждый элемент массива является тегом (в виде строки), а также тип актива для проверки. Тип актива должен быть одной из следующих констант: Постоянная Описание актив_объекта Данное имя относится к объекту. актив_спрайт Данное имя относится к спрайту. актив_звук Данное имя обозначает звук. комната_активов Данное имя относится к помещению. актив_плитки Данное название относится к набору плиток. путь_актива Данное имя относится к пути. актив_скрипт Данное имя относится к сценарию. asset_font Данное название относится к шрифту. временная шкала активов Данное имя относится к временной линии. asset_shader Данное имя относится к шейдер. asset_animationcurve Данное название относится к кривой анимации. последовательность активов Данное имя относится к последовательности. Функция вернет массив, где каждый элемент массива будет одним значением индекса актива. Если не существует активов данного типа, имеющих заданный тег(ы), будет возвращен пустой массив. Синтаксис: tag_get_asset_ids(tags, asset_type); Аргумент Описание теги Одна строка тега актива или массив с различными тегами актива. тип актива Константа типа актива (перечислены выше) Возвращается: Массив Пример: var _paths = tag_get_asset_ids("enemy", asset_path); var _num = irandom(array_length(_paths) - 1); path_start(_paths[_num], 1, path_action_reverse, false); Приведенный выше код использует тег "enemy" для поиска всех активов пути с этим тегом, затем выбирает один случайным образом и назначает его экземпляру, выполняющему код.
