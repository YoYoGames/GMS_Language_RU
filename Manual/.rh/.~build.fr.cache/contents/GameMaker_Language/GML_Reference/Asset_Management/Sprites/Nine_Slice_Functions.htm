Девять функций нарезки В дополнение к разделу Nine Slice в редакторе спрайтов, GameMaker Studio 2 предоставляет функции для установки свойств Nine Slice для спрайтов во время выполнения. Прежде чем читать об этих функциях, рекомендуется прочитать эту страницу, чтобы получить представление о технике Nine Slice. Следующие функции можно использовать для получения и изменения свойств Nine Slice для актива спрайта во время выполнения: sprite_nineslice_create sprite_get_nineslice sprite_set_nineslice Эти функции используют структуру Nine Slice, о содержимом которой вы можете прочитать на этой странице.
