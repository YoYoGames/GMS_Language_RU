sprite_get_speed Эта функция может использоваться для получения скорости спрайта, определенной для ресурса спрайта в редакторе спрайтов. Возвращаемое значение может быть использовано, например, для расчета того, сколько сколько кадров может быть нарисовано для различных скорость изображения. Обратите внимание, что возвращаемое значение будет сильно отличаться в зависимости от типа скорости которая была применена в редакторе спрайтов, либо Frames Per Second, либо Frames Per Game Frame. Следующие два примера иллюстрируют это: Если у вас есть спрайт, который рисует 1 кадр в секунду, а скорость изображения установлена на 0,5, он будет рисоваться со скоростью 0,5 кадра в секунду. Если частота кадров в игре составляет 60 кадров в секунду, спрайт будет рисовать 1 кадр на каждые 120 игровых кадров. Если у вас есть спрайт, который рисует 1 кадр на игровой кадр, и вы установили скорость изображения на 0,5, он будет рисовать 0,5 кадра на игровой кадр. Если частота кадров в игре составляет 60 кадров в секунду, спрайт будет рисовать 30 кадров на каждые 60 игровых кадров. Узнать, какой тип скорости анимации был использован при определении спрайта, можно с помощью функции sprite_get_speed_type(), а установить скорость и тип анимации можно с помощью функции sprite_set_speed(). Синтаксис: sprite_get_speed(index) Аргумент Описание индекс Индекс спрайта для получения скорости Возвращается: Реально Пример: s_speed = sprite_get_speed(sprite_index); s_type = sprite_get_speed_type(sprite_index); Приведенный выше код получает скорость спрайта и тип анимации спрайта и сохраняет их в переменных экземпляра для дальнейшего использования.
