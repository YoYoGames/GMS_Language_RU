skeleton_animation_clear Эта функция очистит указанную дорожку анимации от всех анимаций, готовых к повторному назначению. Синтаксис: skeleton_animation_clear(track); Аргумент Описание трек Анимационная дорожка очищается. Возвращается: Н/Д Пример: if mouse_check_button(mb_right) { skeleton_animation_clear(1); } Приведенный выше код очистит дорожку анимации 1 при нажатии правой кнопки мыши.
