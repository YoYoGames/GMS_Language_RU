скорость изображения Эта переменная определяет скорость, с которой GameMaker Studio 2 будет перебирать суб-изображения для текущего экземпляра спрайта. Значение скорости является множителем, по умолчанию используется значение 1, а при установке значения 0,5 скорость анимации уменьшится вдвое. скорость анимации - как установлено в редакторе спрайтов или редакторе изображений, - в то время как при установке значения 2 она удваивается. Если используемый спрайт не имеет нет вложенных изображений, эта переменная не будет иметь никакого эффекта. Синтаксис: скорость изображения; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: with (instance_create_layer(x, y, "Effects", obj_Explosion)) { image_speed = 0.5; } Приведенный выше код создает экземпляр объекта "obj_Explosion" и устанавливает его image_speed равным 0,5 (фактически уменьшая скорость бега вдвое).
