skeleton_attachment_create Когда вы создаете скелетный анимационный спрайт, вы можете назначить слоты для вложений и вложения, которые будут в них находиться. Это просто изображения, которые находятся отдельно от анимации, но при прикреплении будут двигаться вместе с костью, к которой они прикреплены. Обычно вы назначаете эти вложения в программе анимации (Spine), но с помощью этой функции вы можете создавать свои собственные вложения во время выполнения, используя актив спрайта из вашей игры. чтобы вы дали вложению имя (в виде строки), затем задали используемые значения sprite_index и image_index, а также x и y origin (которые могут отличаться от тех, что заданы для спрайта в свойствах спрайта), и вы также можете задать любые преобразования. которые вы хотите применить к изображению при вложении. Если вложение было успешно создано, функция вернет 1, а если нет (вы указали неверный индекс спрайта, или базовый спрайт не является спрайтом Spine спрайт), то функция вернет -1. Синтаксис: skeleton_attachment_create(name, sprite, ind, xorigin, yorigin, xscale, yscale, rot); Аргумент Описание имя Имя (в виде строки) создаваемого вложения. спрайт Индекс sprite_index для получения изображения вложения. инд Индекс_изображения для получения изображения вложения. xorigin Начало координат x для используемого изображения. yorigin Начало координат y для используемого изображения. xscale Горизонтальное масштабирование изображения в виде множителя: 1 = нормальное масштабирование, 0,5 - половина и т.д... yscale Вертикальное масштабирование изображения в виде множителя: 1 = нормальное масштабирование, 0,5 - половина и т.д.... гниение Поворот изображения. 0=нормальное, 90=поворот на 90 градусов против часовой стрелки и т.д. Возвращается: 1 - если успешно, -1 - если нет Пример: skeleton_attachment_create("меч", spr_Weapons, 0, 0, 80, 1, 1, 90); skeleton_attachment_create("нож", spr_Weapons, 1, 0, 45, 1, 1, 90); skeleton_attachment_create("арбалет", spr_Weapons, 0, 10, 30, 1, 1, 0); skeleton_attachment_set("slot_leftHand", choose("sword", "knife", "crossbow")); Приведенный выше код проверит текущее назначенное имя вложения для слота с именем "slot_leftHand", и если будет возвращена пустая строка, то будет прикреплен новый спрайт.
