Рисование и разное Следующие функции являются дополнительными инструментами для получения различных наборов данных о скелетном анимационном спрайте: skeleton_get_minmax skeleton_get_num_bounds skeleton_get_bounds Вы также можете использовать различные функции рисования, разработанные специально для этих типов спрайтов (помимо обычных draw_sprite() функций).
