bbox_left Эта переменная, доступная только для чтения, возвращает положение (вдоль оси x) в пределах комнаты левой граничной области для данного экземпляра, где граничная область определяется максимальной шириной и высотой маски для данного экземпляра (как задано sprite_index или mask_index). Даже если спрайт имеет точную маску столкновений, ограничительная область существует и используется для определенных целей. и используется для определенных действий, поэтому вы можете использовать эту переменную для ее поиска. Обратите внимание, что если экземпляр не имеет назначенного спрайта, возвращаемое значение будет равно позиции X экземпляра. Синтаксис: bbox_left; Возвращается: Вещественный (целое число) Пример: if bbox_left < 0 { x = sprite_xoffset; } Приведенный выше код проверяет, находится ли ограничивающая рамка экземпляра за пределами комнаты, и если да, то устанавливает экземпляр в новое положение.
