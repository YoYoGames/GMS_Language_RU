угол изображения Это значение задает угол (поворот) спрайта и измеряется в градусах: вправо - 0º, вверх - 90º, влево - 180º и вниз - 270º. Установите эту переменную в 0, чтобы вернуть спрайт к тому виду, который был задан в редакторе спрайтов. Пожалуйста, обратите внимание, что для того, чтобы изменения в этой переменной были заметны, экземпляр должен либо не иметь события рисования (и тогда GameMaker Studio 2 будет рисовать спрайт по умолчанию), либо рисоваться с помощью одной из расширенных функций рисования, таких как draw_self() или draw_sprite_ext(). Синтаксис: угол_изображения; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: image_angle = point_direction(x, y, mouse_x, mouse_y); Приведенный выше код повернет спрайт экземпляра так, чтобы он всегда указывал на позицию мыши.
