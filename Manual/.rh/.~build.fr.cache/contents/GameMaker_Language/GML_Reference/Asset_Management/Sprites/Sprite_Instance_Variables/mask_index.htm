индекс_маски Когда вы определяете объект в GameMaker Studio 2, вы можете назначить ему маску, которая будет использоваться для столкновений, а не ту, которая соответствует определенному спрайту. Эта переменная может быть использована для поиска sprite_index этой маски (или она вернет -1, если спрайт не был назначен) или для установки маски для экземпляра на выбранный спрайт. Установка индекса маски означает, что вы можете иметь, например, спрайт для экземпляра с неправильной формой, но дать ему круговую маску столкновений, полученную из другого спрайта. Синтаксис: mask_index; Возвращается: Real (целое число - значение индекса ресурса) Пример: mask_index = spr_Round; Приведенный выше код устанавливает маску экземпляра на маску спрайта "spr_Round".
