sprite_get_nineslice Эта функция возвращает структуру Nine Slice для спрайта, содержащую все его свойства Nine Slice, установленные в редакторе спрайтов или с помощью функции sprite_set_nineslice(). Содержимое этой структуры подробно описано на этой странице. Если предоставленный спрайт не имеет назначенной ему структуры Nine Slice, для него будет создана и возвращена новая структура со свойствами Nine Slice по умолчанию. Изменение любых значений в этой структуре изменит свойства Nine Slice исходного спрайта, что повлияет на все последующие вызовы рисования, сделанные с этим спрайтом. Синтаксис: sprite_get_nineslice(ind); Аргумент Описание инд Индекс спрайта, из которого будет извлечена структура Nine Slice. Возвращается: Struct (или -1, если спрайт не существует) Пример: var _box_nineslice = sprite_get_nineslice(spr_box_0); _box_nineslice.enabled = true; _box_nineslice.left = 10; _box_nineslice.right = 10; _box_nineslice.top = 10; _box_nineslice.bottom = 10; Приведенный выше код извлекает структуру Nine Slice struct из спрайта, включает для него Nine Slicing и устанавливает смещения направляющих.
