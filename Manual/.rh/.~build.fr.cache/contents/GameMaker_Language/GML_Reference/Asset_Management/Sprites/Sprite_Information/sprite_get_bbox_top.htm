sprite_get_bbox_top Эта функция возвращает относительное положение вершины ограничивающей рамки спрайта. Это значение дается как относительная величина, основанная на левом верхнем углу базового объекта спрайта (0,0). это то же значение, которое можно найти в редакторе спрайтов для свойств маски столкновений. На рисунке ниже показано, как оно вычисляется: Синтаксис: sprite_get_bbox_top(ind); Аргумент Описание инд Индекс спрайта для проверки. Возвращает Реально Пример: var ww, hh; ww = sprite_get_bbox_left(sprite_index) - sprite_get_bbox_right(sprite_index); hh = sprite_get_bbox_bottom(sprite_index) - sprite_get_bbox_top(sprite_index); Приведенный выше код вычисляет ширину и высоту маски столкновений на основе относительных положений сторон ограничивающей рамки.
