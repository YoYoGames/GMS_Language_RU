sprite_exists Эта функция возвращает, существует или нет спрайт с указанным индексом в выполняемом проекте. Синтаксис: sprite_exists(index); Аргумент Описание индекс Индекс спрайта для проверки. Возвращает Н/Д Пример: if sprite_exists(spr_array[0]) { sprite_index = spr_array[0]; } else { sprite_index = spr_BaseSprite; } Приведенный выше код проверяет массив на наличие в нем допустимого индекса спрайта, и если да, то присваивает этот спрайт экземпляру, а если нет, то присваивает спрайт из включенных ресурсов.
