sprite_get_height С помощью этой функции можно узнать высоту базового спрайта без преобразований в пикселях. Синтаксис: sprite_get_height(index); Аргумент Описание индекс Индекс спрайта, для которого нужно найти высоту. Возвращает Реально Пример: if sprite_height != sprite_get_height(sprite_index) { image_yscale = 1; } Приведенный выше код проверяет высоту спрайта в текущем экземпляре и, если она отличается от высоты исходного базового спрайта, сбрасывает масштаб оси y.
