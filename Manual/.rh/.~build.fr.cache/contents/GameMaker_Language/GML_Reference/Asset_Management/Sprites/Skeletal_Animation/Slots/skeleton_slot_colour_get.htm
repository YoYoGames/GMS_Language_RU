skeleton_slot_colour_get С помощью этой функции можно получить цвет слота вложения. Введите имя слота (строку), и функция вернет целочисленное значение цвета. Синтаксис: skeleton_slot_colour_get(slot); Аргумент Описание слот Имя слота для проверки, строка Возвращается: Целое число Пример: slot_list = ds_list_create(); skeleton_find_slot(mouse_x, mouse_y, slot_list); if !ds_list_empty(slot_list) { for (var i = 0; i < ds_list_size(slot_list); ++i;) { if skeleton_slot_colour_get(slot_list[| i]) != c_white { skeleton_slot_colour_set(slot_list[| i], c_white, 1); } } } Приведенный выше код создает список DS, затем заполняет его именами слотов, найденных по положению мыши в комнате. Затем он проходит по списку слотов и сбрасывает цвет для найденных слотов на белый с альфой 1, если цвет не является уже белый.
