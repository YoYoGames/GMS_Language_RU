sprite_flush С помощью этой функции вы можете удалить из текстурной памяти заданную страницу текстуры для заданного спрайта. Вы указываете индекс спрайта (как определено при создании ресурса спрайта), и текстурная страница, которой он назначен, будет удалена из текстурной памяти. Обратите внимание, что если страница текстуры используется в другом месте комнаты (другим экземпляром спрайта или фоном и т.д.), вы можете получить небольшое снижение производительности, поскольку страница будет немедленно перезагружена, поэтому при использовании этой функции следует соблюдать осторожность. Обратите внимание, что вернет -1, если flush не поддерживается для выбранного ресурса, или вернет 0, если все работает правильно. Синтаксис: sprite_flush(ind) Аргумент Описание инд Индекс (имя ресурса) актива спрайта для промывки Возвращается: Int (-1 или 0) Пример: sprite_flush(spr_Player_Aura); Приведенный выше код стирает спрайт "spr_Player_Aura" из памяти.
