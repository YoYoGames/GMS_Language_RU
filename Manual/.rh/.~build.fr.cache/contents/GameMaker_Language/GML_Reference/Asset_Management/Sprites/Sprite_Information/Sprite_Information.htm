Информация о спрайте Следующие функции можно использовать для получения информации об активе спрайта. Обратите внимание, что эти функции получают информацию из базового спрайта (т. е. актива спрайта, созданного в браузере активов). а не из спрайта, используемого в экземпляре. Например, вы можете использовать спрайт переменная экземпляра sprite_width для получения ширины спрайта, который используется в экземпляре, и эта ширина будет учитывать любое масштабирование, которое могло быть применено. Однако если вам нужно узнать ширину спрайта в обычном виде, без применения преобразований, вам нужно использовать функцию sprite_get_width(), приведенную ниже.
