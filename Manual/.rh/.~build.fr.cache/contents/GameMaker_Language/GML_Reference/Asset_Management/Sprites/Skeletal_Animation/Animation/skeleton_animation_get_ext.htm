skeleton_animation_get_ext Один скелетный анимационный спрайт может иметь различные наборы анимации, и этим наборам могут быть назначены различные дорожки, так что вы можете "смешивать и сопоставлять" наборы анимации. Эта функция возвращает имя набора анимации, используемого в данный момент на заданном номере дорожки (заданном функцией skeleton_animation_set_ext). Синтаксис: skeleton_animation_get_ext(track); Аргумент Описание трек Номер дорожки для получения имени набора анимации. Возвращается: Строка Пример: if skeleton_animation_get_ext(1) != "bodyfight" { skeleton_animation_set_ext("bodyfight", 1); } Приведенный выше код изменит набор анимации, используемый дорожкой 1, на "bodyfight", если он еще не является таковым.
