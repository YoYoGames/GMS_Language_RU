sprite_flush_multi С помощью этой функции вы можете удалить из текстурной памяти любое количество текстурных страниц для заданных спрайтов. Вы передаете индексы спрайтов в виде массива, и текстурные страницы, которым они назначены, будут удалены из текстурной памяти. Обратите внимание, что если одна из текстурных страниц используется в другом месте комнаты (другим экземпляром спрайта или фоном и т.д.), вы можете получить небольшое снижение производительности, так как страница будет немедленно перезагружена обратно в память, поэтому при использовании этой функции следует соблюдать осторожность. Обратите внимание, что функция вернет -1, если flush не поддерживается для выбранных ресурсов, или вернет 0, если все работает правильно. Синтаксис: sprite_flush_multi(array); Аргумент Описание массив Массив с индексами спрайтов для промывки Возвращается: Int (-1 или 0) Пример: spr_a[0] = spr_Player_Aura1; spr_a[1] = spr_Player_Aura2; spr_a[2] = spr_Player_Aura3; spr_a[3] = spr_Player_Aura4; sprite_flush_multi(spr_a); Приведенный выше код создает массив, каждый элемент которого содержит индекс спрайта. Этот массив затем используется для очистки текстур спрайтов из памяти. Назад: Манипуляция спрайтом Далее: sprite_prefetch © Copyright YoYo Games Ltd. 2021 Все права защищены
