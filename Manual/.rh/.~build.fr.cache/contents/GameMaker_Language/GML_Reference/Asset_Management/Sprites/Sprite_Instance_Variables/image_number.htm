номер_изображения Эта переменная, доступная только для чтения, может быть использована для получения количества суб-изображений в спрайте, который был назначен экземпляру (если вам нужно количество суб-изображений для спрайта, отличного от того, который был назначен экземпляру, вам следует следует использовать sprite_get_number()). Обратите внимание, что при наличии (например) 1 субизображения эта переменная вернет "1", но image_index этого субизображения равен 0. Синтаксис: номер_изображения; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: если image_speed > 0 { if image_index > image_number - 1 instance_destroy(); } Приведенный выше код проверяет, анимирован ли спрайт, и если да, то проверяет, превышает ли текущий image_index количество вложенных изображений, и если да, то уничтожает экземпляр.
