skeleton_attachment_get Скелетный анимационный спрайт может иметь другие спрайты, добавленные в качестве вложений, причем эти спрайты добавляются в именованный слот (имя задается при создании слота вложения в программе анимации), и они будут рисоваться вместе с анимацией текущего спрайта. С помощью этой функции вы можете получить имя (в виде строки) вложения для данного слота текущего прикрепленного спрайта. Обратите внимание, что ссылки на прикрепленные спрайты даются через их строку имени, назначенную в Spine, или когда вы вызвали skeleton_attachment_create(). Синтаксис: skeleton_attachment_get(slot); Аргумент Описание слот Имя слота (строка) для получения вложения. Возвращается: Строка Пример: if skeleton_attachment_get("slot_leftHand") == "" { skeleton_attachment_set("slot_leftHand", choose("sword", "spear", "knife")); } Приведенный выше код проверит текущее назначенное имя вложения для слота с именем "slot_leftHand", и если будет возвращена пустая строка, то будет прикреплен новый спрайт.
