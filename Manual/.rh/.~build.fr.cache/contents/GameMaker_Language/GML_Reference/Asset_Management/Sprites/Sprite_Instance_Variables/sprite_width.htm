ширина спрайта Эта переменная, доступная только для чтения, возвращает ширину спрайта, который был назначен данному экземпляру. Ширина возвращается в пикселях и зависит от значения параметра image_xscale. Если вам нужна немасштабированная ширина вам следует использовать sprite_get_width(). Синтаксис: sprite_width; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if sprite_width != sprite_get_width(sprite_index) { image_xscale = 1; } Приведенный выше код сверяет ширину спрайта, назначенного экземпляру, выполняющему код, с шириной ресурса спрайта, и если они не совпадают, то сбрасывает значение image_xscale до 1, Назад: Переменные экземпляра спрайта Далее: sprite_height © Copyright YoYo Games Ltd. 2021 Все права защищены
