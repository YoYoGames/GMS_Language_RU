sprite_prefetch Эта функция может использоваться для предварительной выборки (размещения в памяти текстуры) страницы текстуры с заданным спрайтом. Вы указываете индекс спрайта, определенный при создании актива спрайта, и страница текстуры, на которой он находится, будет загружена в память. Обратите внимание, что вернет -1, если prefetch не поддерживается для выбранного ресурса или целевая платформа - HTML5, или вернет 0, если все работает правильно. ПРИМЕЧАНИЕ: На большинстве устройств происходит снижение производительности при загрузке текстуры в память текстуры, поэтому рекомендуется вызывать команду sprite_prefetch на любую необходимую графику в начале уровня, чтобы избежать задержек. Синтаксис: sprite_prefetch(ind) Аргумент Описание инд Индекс спрайта для получения Возвращается: Int (-1 или 0) Пример: sprite_prefetch(spr_Player_Aura); Приведенный выше код поместит ссылаемый спрайт в память текстуры, готовую к использованию.
