skeleton_slot_data_instance С помощью этой функции вы можете заполнить (предварительно созданный) список DS данными для каждого из доступных слотов крепления для спрайта Spine, связанного с текущим экземпляром (включая любые модификации вложений). Эти данные возвращаются в виде карты DS, которая содержит следующие пары ключ/значение: "имя": Название слота. "кость": Название кости. "attachment": Имя вложения, если оно используется, или "(none)", если оно не назначено. Значения для каждого ключа будут строками и затем могут быть использованы в других функциях прикрепления скелета для этих типов спрайтов. Обратите внимание, что созданные карты DS не уничтожаются, поэтому вам нужно будет пройтись по списку DS и уничтожить каждую из созданных карт самостоятельно. Синтаксис: skeleton_slot_data_instance(list); Аргумент Описание список Идентификатор списка DS для заполнения картами DS. Возвращается: Н/Д Пример: var list = ds_list_create(); var open = true; slot_name = ""; skeleton_slot_data_instance(list); for (var i = 0; i < ds_list_size(list); i++;) { var map = list[| i]; if open { if map[? "attachment"] == "(none)" { open = false; имя_слота = map[? "имя"]; } } ds_map_destroy(map); } ds_list_destroy(list); Приведенный выше код создает список DS, а затем заполняет его данными слота для экземпляра спрайта. Затем эти данные анализируются для извлечения отдельных карт DS с данными слота. Затем проверяется, есть ли пустой слот, и если да, то переменной "имя_слота" присваивается имя пустого слота перед уничтожением карты DS.
