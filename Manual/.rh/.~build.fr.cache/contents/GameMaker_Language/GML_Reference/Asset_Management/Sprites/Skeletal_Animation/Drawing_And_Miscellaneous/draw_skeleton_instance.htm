draw_skeleton_instance Эта функция используется только для экземпляров, которым назначены спрайты, созданные с помощью программы скелетной анимации Spine. Хотя вы можете рисовать эти спрайты с помощью обычных функций рисования (таких как нарисовать_себя()), бывают случаи, когда нужно нарисовать один кадр или показать изменение текстуры кожи без изменения свойств спрайта экземпляра. В таких случаях можно нарисовать спрайт, назначенный экземпляру, с помощью этой функции, где вы указываете уникальный ID экземпляра, затем имя анимации, из которой нужно получить кадр (строка, заданная в программе, используемой для создания спрайта). Затем указывается имя скина, который нужно использовать (опять же, заданное при создании спрайта), и значение кадра. Значение кадра указывает программе GameMaker Studio 2, какую часть анимации нужно показать, и представляет собой значение от 0 до номер_изображения. Остальные аргументы такие же, как и при обычном рисовании спрайта, и будут изменять масштаб, угол, цвет наложения и альфа-код конечного рисуемого кадра анимации. ВНИМАНИЕ! Из-за того, что скелетная анимация интерполируется между различными "ключевыми" кадрами, с этим вызовом связаны дополнительные накладные расходы, поэтому рекомендуется избегать рисования скелетных спрайтов с помощью этого метода, если это не необходимо. этот метод, если в этом нет крайней необходимости. Синтаксис: draw_skeleton_instance(instance, animname, skinname, frame, x, y, xscale, yscale, rot, color, alpha); Аргумент Описание экземпляр Индекс экземпляра, из которого будет рисоваться спрайт. animname Имя анимации, из которой нужно получить кадр (строка). skinname Имя кожи для использования (строка). рама Кадр анимации для рисования (от 0 до image_number - 1). x Координата x места рисования спрайта. y Координата y места рисования спрайта. xscale Горизонтальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... yscale Вертикальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... гниение Поворот спрайта. 0=нормальный, 90=повернутый на 90 градусов против часовой стрелки и т.д. цвет Цвет, с которым будет смешиваться спрайт. альфа Альфа спрайта (от 0 до 1, где 0 - прозрачный, а 1 - непрозрачный). Возвращается: Н/Д Пример: draw_skeleton_instance(obj_Player, "jump", "skin1", 0, x, y, 1, 1, 0, c_white, 1); Приведенный выше код нарисует первый кадр скелетного анимационного спрайта, назначенного данному экземпляру, используя набор анимации "jump" и скин, определенный как "skin1".
