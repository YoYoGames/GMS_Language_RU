sprite_set_offset Эта функция может использоваться для установки начала координат x и y спрайта и принимает относительные значения, основываясь на позиции (0,0), являющейся левым верхним углом спрайта. Следующее изображение иллюстрирует это: ПРИМЕЧАНИЕ: Эта функция влияет на актив спрайта так, что все последующие экземпляры с этим спрайтом будут иметь одинаковое смещение. Синтаксис: sprite_set_offset(ind, xoff, yoff); Аргумент Описание инд Индекс спрайта для изменения смещения. xoff Положение x начала координат. yoff Позиция y относительно начала координат. Возвращает Н/Д Пример: sprite_assign(spr_Custom, spr_Base); sprite_set_offset(spr_Custom, sprite_get_xoffset(spr_Base), sprite_get_yoffset(spr_Base)); Приведенный выше код присваивает спрайт с индексом "spr_Base" спрайту с индексом "spr_Custom", а затем использует значения x и y смещения "spr_Base" для установки начала нового спрайта.
