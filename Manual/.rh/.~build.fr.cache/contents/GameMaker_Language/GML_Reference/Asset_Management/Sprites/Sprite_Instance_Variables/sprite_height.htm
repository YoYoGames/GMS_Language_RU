высота спрайта Эта переменная, доступная только для чтения, возвращает высоту спрайта, который был назначен данному экземпляру. Эта высота возвращается в пикселях и зависит от значения параметра image_yscale. Если вам нужна немасштабированная высота, используйте sprite_get_height(). Синтаксис: sprite_height; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if sprite_height != sprite_get_height(sprite_index) { image_yscale = 1; } Приведенный выше код сверяет высоту спрайта, назначенного экземпляру, выполняющему код, с высотой ресурса спрайта, и если они не совпадают, то сбрасывает значение image_yscale до 1, Назад: Переменные экземпляра спрайта Далее: sprite_xoffset © Copyright YoYo Games Ltd. 2021 Все права защищены
