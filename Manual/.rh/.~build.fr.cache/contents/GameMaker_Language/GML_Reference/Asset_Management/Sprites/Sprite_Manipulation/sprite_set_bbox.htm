sprite_set_bbox Эта функция может использоваться для установки значений ограничивающей рамки для спрайта. Вы указываете индекс спрайта, а затем значения left, top, right и bottom для позиций ограничивающей рамки. Позиции являются абсолютными значениями, где позиция (0, 0) соответствует левому верхнему углу спрайта, независимо от смещения спрайта, наличия у него "пустых" пикселей или места рисования в комнате. ПРИМЕЧАНИЕ: Эта функция влияет на актив спрайта так, что все последующие экземпляры с этим спрайтом будут иметь одинаковые ограничительные рамки. Синтаксис: sprite_set_bbox(ind, left, top, right, bottom); Аргумент Описание инд Индекс спрайта, на который нужно установить ограничивающую рамку. слева Левая сторона ограничительной рамки топ Верхняя часть ограничивающей рамки. правильно Правая сторона ограничительной рамки дно Нижняя часть ограничивающей рамки. Возвращает Н/Д Пример: if sprite_get_bbox_mode(sprite_index) == 2 { left = irandom(sprite_width / 2); right = irandom((sprite_width / 2) + irandom(sprite_width / 2)); top = irandom(sprite_height / 2); bottom = irandom((sprite_height / 2) + irandom(sprite_height / 2)); sprite_set_bbox(sprite_index, left, top, right, bottom); } Приведенный выше код проверит режим bounding box спрайта, назначенного на sprite_index, и если он установлен на manual, то его bounding box будет изменен.
