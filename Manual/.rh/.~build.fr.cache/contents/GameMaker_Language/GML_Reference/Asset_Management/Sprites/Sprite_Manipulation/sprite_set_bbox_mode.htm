sprite_set_bbox_mode Эта функция используется для установки режима ограничивающей рамки для спрайта. Вы передаете индекс спрайта и используемый режим, который должен быть одной из следующих констант: Постоянная Описание bboxmode_automatic Автоматически - Ограничительная рамка будет рассчитана автоматически, на основе настройки допуска для спрайта. bboxmode_fullimage Полное изображение - ограничивающая рамка будет установлена так, чтобы использовать полную ширину и высоту спрайта, независимо от допусков и "пустых" пикселей. bboxmode_manual Вручную - Ограничительная рамка устанавливается вручную по заданным пользователем значениям, что требует использования функции sprite_set_bbox() ПРИМЕЧАНИЕ: Эта функция влияет на актив спрайта так, что все последующие экземпляры с этим спрайтом будут иметь тот же режим ограничивающей рамки. Синтаксис: sprite_set_bbox_mode(ind, mode); Аргумент Описание инд Индекс спрайта для изменения режима. режим Режим для установки (константа). Возвращает Н/Д Пример: if sprite_get_bbox_mode(sprite_index) != 0 { sprite_set_bbox_mode(sprite_index, bboxmode_automatic); } Выше проверяется режим bbox для текущего спрайта, и если он не автоматический, то устанавливается это значение.
