sprite_get_yoffset Когда вы определяете спрайт в редакторе спрайтов, вам необходимо задать его начало координат. Это точка, в которой спрайт будет "прикреплен" или "нарисован" при использовании с экземпляром. Эта функция возвращает относительное смещение для оси y начала координат, исходя из того, что левый верхний угол является позицией 0,0, при этом +y означает "вниз", а -y - "вверх". Следующее изображение иллюстрирует это: Синтаксис: sprite_get_yoffset(index); Аргумент Описание индекс Индекс спрайта для поиска смещения по оси y. Возвращает Реально Пример: if y - sprite_get_yoffset(sprite_index) < 0 { y = sprite_get_yoffset(sprite_index); } Приведенный выше код обеспечит сохранение экземпляра в комнате на основе спрайта (так что спрайт всегда будет виден).
