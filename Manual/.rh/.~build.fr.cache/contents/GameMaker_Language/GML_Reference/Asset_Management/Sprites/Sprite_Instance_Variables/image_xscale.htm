масштаб_изображения Это значение задает горизонтальное масштабирование, применяемое к спрайту, который был назначен текущему экземпляру. Значение 1 означает отсутствие масштабирования (1:1), меньшие значения будут масштабироваться вниз (0,5, например, уменьшит ширину спрайта вдвое), большие значения увеличивает масштаб, а отрицательные значения переворачивают спрайт и масштабируют его, если только используемое значение не равно -1 (в этом случае спрайт просто переворачивается относительно начала координат без масштабирования). Синтаксис: image_xscale; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if image_xscale < 5 { image_xscale += 0.2; image_yscale = image_xscale; } else { instance_create_layer(x, y, "Effects", obj_Explosion); instance_destroy(); } Приведенный выше код масштабирует спрайт, а затем, когда он увеличивается в 5 раз до исходного размера, создается новый экземпляр другого объекта и этот экземпляр уничтожается.
