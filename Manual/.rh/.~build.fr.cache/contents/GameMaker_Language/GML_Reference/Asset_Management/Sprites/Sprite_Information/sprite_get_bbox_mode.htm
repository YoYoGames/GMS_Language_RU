sprite_get_bbox_mode Эта функция возвращает текущий "режим" для расчетов ограничивающей рамки. Вы указываете индекс спрайта, который нужно проверить, и функция возвращает одну из констант, показанных ниже. Синтаксис: sprite_get_bbox_mode(ind); Аргумент Описание инд Индекс спрайта для проверки. Возвращает Постоянная Постоянная Описание bboxmode_automatic Автоматически - Ограничительная рамка будет рассчитана автоматически, на основе настройки допуска для спрайта. bboxmode_fullimage Полное изображение - ограничивающая рамка будет установлена так, чтобы использовать всю ширину и высоту спрайта, независимо от допусков и "пустых" пикселей. bboxmode_manual Manual - Ограничительная рамка была установлена вручную на значения, определенные пользователем (либо в редакторе спрайтов, либо с помощью функции sprite_set_bbox()) Пример: if sprite_get_bbox_mode(sprite_index) != 0 { sprite_set_bbox_mode(sprite_index, bboxmode_automatic); } Приведенное выше действие проверяет режим bbox для текущего спрайта и, если он не автоматический, то устанавливает его в это значение.
