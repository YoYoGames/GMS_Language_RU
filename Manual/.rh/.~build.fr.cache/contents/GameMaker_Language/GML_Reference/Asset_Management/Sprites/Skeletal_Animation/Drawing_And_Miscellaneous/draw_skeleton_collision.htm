draw_skeleton_collision Эта функция рисует маски столкновений, связанные с заданной скелетной анимацией. Вы передаете базовый спрайт, набор анимации, который нужно использовать, и кадр, из которого нужно получить информацию, а также можете задать свойства трансформации. Синтаксис: draw_skeleton_collision(sprite, animname, frame, x, y, xscale, yscale, rot, color) Аргумент Описание спрайт Индекс спрайта для рисования. animname Имя анимации, из которой нужно получить кадр (строка). рама Кадр анимации для рисования (от 0 до image_number - 1). x Координата x места рисования спрайта. y Координата y места рисования спрайта. xscale Горизонтальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... yscale Вертикальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... гниение Поворот спрайта. 0=нормальный, 90=повернутый на 90 градусов против часовой стрелки и т.д. цвет Цвет, с которым будет смешиваться спрайт. Возвращается: Н/Д Пример: draw_skeleton_collision(sprite_index, "jump", image_index, x, y, image_xscale, image_yscale, image_angle, c_white); Приведенный выше код нарисует данные маски столкновений для текущего спрайта, используя текущие преобразования, для анимации с набором "jump".
