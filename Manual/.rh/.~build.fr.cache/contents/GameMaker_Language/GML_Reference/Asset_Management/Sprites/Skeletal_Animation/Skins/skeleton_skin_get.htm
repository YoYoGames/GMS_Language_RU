skeleton_skin_get С помощью спрайтов скелетной анимации вы можете назначить анимации отдельные текстуры (называемые "скинами"), тем самым используя одну анимацию для нескольких разных вещей. Эта функция возвращает имя скина (в виде строки), который в настоящее время присвоено скелетному анимационному спрайту, который использует ваш экземпляр. Возвращаемое имя - это имя, которое вы задали при создании спрайта в вашей программе анимации. Синтаксис: skeleton_skin_get(); Возвращается: Строка Пример: if skeleton_skin_get() == "skin_Enemy1" { skeleton_skin_set(choose("skin_Enemy1", "skin_Enemy2", "skin_Enemy3"); } Приведенный выше код проверяет текущий назначенный скин для анимации, и если это "skin_Enemy1", он выбирает и устанавливает новый скин из одного из трех вариантов.
