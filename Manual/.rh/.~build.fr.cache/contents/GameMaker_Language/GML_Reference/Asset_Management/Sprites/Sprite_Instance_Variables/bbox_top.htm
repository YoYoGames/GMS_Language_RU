bbox_top Эта переменная, доступная только для чтения, возвращает положение в пределах комнаты (вдоль оси y) вершины ограничительной рамки для данного экземпляра, где ограничительная рамка определяется максимальной шириной и высотой маски для данного экземпляра (как установлено на сайте sprite_index или mask_index). Даже если спрайт имеет точную маску столкновений, ограничивающая область существует и используется для определенных целей. и используется для определенных действий, поэтому вы можете использовать эту переменную для ее поиска. Обратите внимание, что когда экземпляр не имеет назначенного спрайта, возвращаемое значение будет равно позиции Y экземпляра. Синтаксис: bbox_top; Возвращается: Вещественный (целое число) Пример: if bbox_top < 0 { y = sprite_yoffset; } Приведенный выше код проверяет, находится ли ограничивающая рамка экземпляра за пределами комнаты, и если да, то устанавливает экземпляр в новое положение.
