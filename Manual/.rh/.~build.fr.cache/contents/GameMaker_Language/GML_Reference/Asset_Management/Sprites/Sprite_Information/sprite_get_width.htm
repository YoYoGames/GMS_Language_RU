sprite_get_width С помощью этой функции можно узнать ширину базового ресурсного спрайта без преобразований в пикселях. Синтаксис: sprite_get_width(index); Аргумент Описание индекс Индекс спрайта для определения ширины. Возвращает Реально Пример: if sprite_width != sprite_get_width(sprite_index) { image_xscale = 1; } Приведенный выше код проверяет ширину спрайта в текущем экземпляре и, если она отличается от ширины исходного базового спрайта, сбрасывает масштаб оси x.
