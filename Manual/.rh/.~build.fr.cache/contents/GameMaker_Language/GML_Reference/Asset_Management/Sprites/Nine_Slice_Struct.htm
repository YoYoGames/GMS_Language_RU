Структура девяти ломтиков Структура Nine Slice может быть создана с помощью sprite_nineslice_create()или получить из спрайта с помощью sprite_get_nineslice(). Эта структура будет содержать свойства, которые могут быть изменены и применены к спрайту с помощью sprite_set_nineslice(). При извлечении из спрайта изменение значений в этой структуре повлияет на исходный спрайт . В этой структуре будут доступны следующие свойства: enabled: Это булево значение будет истинным , если для спрайта включена функция Nine Slicing, и ложным , если она отключена. отключено. Вы можете изменить этот параметр, чтобы включить или отключить Nine Slicing для спрайта. левая: Это смещение левой направляющей по оси x относительно левого края спрайта. правая: Это смещение правой направляющей по оси x относительно правого края спрайта. top: Это смещение верхней направляющей по оси y относительно верхнего края спрайта. нижняя: Это смещение нижней направляющей по оси y относительно нижнего края спрайта. tilemode: Возвращает массив, содержащий режимы плитки для краевого и центрального срезов. Режим Tile Mode каждого фрагмента можно прочитать или изменить, используя эти константы в качестве индекса массива: Постоянная Описание nineslice_left Левый краевой срез nineslice_top Верхний краевой срез nineslice_right Правый краевой срез дно Нижний краевой срез девятиликий_центр Центральный ломтик Режим плитки для среза будет (или может быть установлен) одной из следующих констант: Постоянная Описание nineslice_stretch Ломтик будет растянут nineslice_repeat Нарезка будет повторяться nineslice_mirror Срез будет повторяться путем зеркального отображения nineslice_blank Ломтик не будет растягиваться или повторяться, в результате чего после него останется пустая область nineslice_hide Ломтик не появится вообще Поскольку tilemode является массивом, для изменения режима плитки среза необходимо использовать синтаксис массива. Приведенный ниже пример кода устанавливает режим черепицы верхнего фрагмента на Repeat: Пример: var _nineslice = sprite_get_nineslice(spr_box); _nineslice.tilemode[nineslice_top] = nineslice_repeat; Более подробную информацию о технике "Девять ломтиков" см. на этой странице.
