sprite_get_uvs Эта функция возвращает массив с UV-координатами и другими данными для текстуры подизображения спрайта на странице текстуры. Функция возвращает массив со следующими 8 элементами: [0] = левый [1] = верх [2] = право [3] = низ [4] = количество пикселей, которые компилятор активов обрезал с левой стороны спрайта [5] = количество пикселей, которые компилятор активов обрезал с верхней стороны спрайта [6] = нормализованный процент пиксельных данных от исходной ширины спрайта, который был сохранен на странице текстуры [7] = нормализованный процент пиксельных данных от высоты исходного спрайта, который был сохранен на странице текстуры Этот массив затем можно использовать в других функциях рисования, особенно при работе в 3D или при использовании функций 2D-примитивов, а также при работе с функциями Шейдер функциями. ПРИМЕЧАНИЕ: Эта функция не работает с векторными спрайтами или спрайтами скелетной анимации. Синтаксис: sprite_get_uvs(sprite, subimage); Аргумент Описание спрайт Индекс используемого спрайта. субизображение Вложенное изображение спрайта для использования. Возвращает Массив (1D, 8 элементов) Пример: var tex = sprite_get_uvs(sprite, 0); tex_left = tex[0]; tex_top = tex[1]; tex_right = tex[2]; tex_bottom = tex[3]; Приведенный выше код сохранит UV-координаты для заданного спрайта в локальном массиве, а затем присвоит значения переменным экземпляра.
