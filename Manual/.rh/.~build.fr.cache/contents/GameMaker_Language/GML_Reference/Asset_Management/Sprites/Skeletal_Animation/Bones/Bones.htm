Кости Как следует из названия, скелетные анимационные спрайты состоят из ряда "костей", поверх которых добавляется кожа для создания окончательной анимации персонажа. Обычно вы хотите установить эти кости в анимационной программе (Spine), а затем экспортировать спрайт с уже заданным набором анимаций, чтобы вы могли просто переключаться между ними по мере необходимости.
