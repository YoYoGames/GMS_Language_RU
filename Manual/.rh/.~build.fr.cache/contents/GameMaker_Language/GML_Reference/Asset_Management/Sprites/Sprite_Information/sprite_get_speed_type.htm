sprite_get_speed_type Эта функция может быть использована для получения типа анимации спрайта, определенного для ресурса спрайта в редакторе спрайтов. Возвращаемое значение будет одной из следующих констант: Постоянная Описание spritespeed_framespersecond Спрайт был определен с анимацией в кадрах в секунду. spritespeed_framespergameframe Спрайт был определен с анимацией в кадрах на один игровой кадр. Следующие два примера иллюстрируют разницу, которую это может внести при расчете скорости анимации спрайта: Если у вас есть спрайт, который рисует 1 кадр в секунду, а скорость изображения установлена на 0,5, он будет рисоваться со скоростью 0,5 кадра в секунду. Если частота кадров в игре составляет 60 кадров в секунду, спрайт будет рисовать 1 кадр на каждые 120 игровых кадров. Если у вас есть спрайт, который рисует 1 кадр на игровой кадр, и вы установили скорость изображения на 0,5, он будет рисовать 0,5 кадра на игровой кадр. Если частота кадров в игре составляет 60 кадров в секунду, спрайт будет рисовать 30 кадров на каждые 60 игровых кадров. Узнать, какое значение скорости использовалось для анимации спрайта, можно с помощью функции sprite_get_speed(), а установить скорость и тип анимации можно с помощью функции sprite_set_speed(). Синтаксис: sprite_get_speed_type(index) Аргумент Описание индекс Индекс спрайта для получения типа анимации Возвращается: Постоянная (см. выше) Пример: s_speed = sprite_get_speed(sprite_index); s_type = sprite_get_speed_type(sprite_index); Приведенный выше код получает скорость спрайта и тип анимации спрайта и сохраняет их в переменных экземпляра для дальнейшего использования.
