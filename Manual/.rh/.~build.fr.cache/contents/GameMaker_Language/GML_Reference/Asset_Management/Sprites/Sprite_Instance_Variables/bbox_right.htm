bbox_right Эта переменная, доступная только для чтения, возвращает положение в пределах комнаты (вдоль оси x) правой стороны ограничительной рамки для данного экземпляра, где ограничительная рамка определяется максимальной шириной и высотой маски для экземпляра (как задано на сайте sprite_index или mask_index). Даже когда спрайт имеет точную маску столкновений, ограничивающая область существует и используется для определенных вещей, поэтому вы можете использовать эту переменную для ее поиска. Обратите внимание, что если экземпляр не имеет назначенного спрайта, возвращаемое значение будет равно позиции X экземпляра. Синтаксис: bbox_right; Возвращается: Вещественный (целое число) Пример: if bbox_right > room_width { x = room_width - sprite_xoffset; } Приведенный выше код проверяет, находится ли ограничивающая рамка экземпляра за пределами комнаты, и если да, то устанавливает экземпляр в новое положение.
