список_анимации_скелета С помощью этой функции вы можете заполнить (предварительно созданный) список DS всеми именами анимаций, включенных в скелетный анимационный спрайт. Эти имена будут строками и могут быть использовать в других функциях анимации для этих типов спрайтов. Синтаксис: skeleton_animation_list(sprite, list); Аргумент Описание спрайт Индекс спрайта скелетной анимации Spine для получения списка. список Идентификатор списка DS для заполнения именами анимаций. Возвращается: Н/Д Пример: var list = ds_list_create(); skeleton_animation_list(sprite_index, list); for (var i = 1; i < ds_list_size(list); i++;) { skeleton_animation_mix(list[| 0], list[| i], 0.5); } ds_list_destroy(list); Приведенный выше код создает список DS всех имен анимаций для спрайта, используемого данным экземпляром. Затем он перебирает их и устанавливает значение mix для всех анимаций с первой анимацией равным 0,5. Назад: Анимация Далее: skeleton_animation_clear © Copyright YoYo Games Ltd. 2021 Все права защищены
