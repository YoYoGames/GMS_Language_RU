sprite_prefetch_multi Эта функция может быть использована для предварительной выборки (размещения в памяти текстур) ряда текстурных страниц, содержащих заданные спрайты. Вы передаете массив, заполненный индексами спрайтов (как определено при создании актива спрайтов), и страницы текстуры, на которых они находятся, будут загружены в память. на которых они находятся, будут загружены в память. Обратите внимание, что функция вернет -1, если prefetch не поддерживается для выбранного ресурса или целевой платформой является HTML5, или вернет 0, если все работает правильно. ПРИМЕЧАНИЕ: На большинстве устройств происходит снижение производительности при загрузке текстуры в память текстуры, поэтому рекомендуется вызывать команду sprite_prefetch_multi() на любую необходимую графику в начале уровня, чтобы избежать задержек. Синтаксис: sprite_prefetch_multi Аргумент Описание массив Массив с индексами спрайтов для выборки Возвращается: Int (-1 или 0) Пример: spr_a[0] = spr_Player_Aura1; spr_a[1] = spr_Player_Aura2; spr_a[2] = spr_Player_Aura3; spr_a[3] = spr_Player_Aura4; sprite_prefetch_multi(spr_a); Приведенный выше код создает массив, каждый элемент которого содержит индекс спрайта.
