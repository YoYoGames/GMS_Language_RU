нарисовать_скелет Эта функция предназначена только для спрайтов, созданных с помощью программы скелетной анимации, такой как Spine. Хотя вы можете рисовать эти спрайты с помощью обычных функций рисования (таких как нарисовать_себя()), бывают случаи, когда нужно нарисовать один кадр или показать изменение текстуры кожи без изменения свойств спрайта. В таких случаях можно нарисовать спрайт с помощью этой функции, где вы указываете индекс спрайта, а затем имя анимации, из которой нужно взять кадр (строка, заданная в программе, используемой для создания спрайта). Затем вы указываете имя скина, который нужно использовать (опять же, заданное при создании спрайта), и значение кадра. Значение кадра указывает программе GameMaker Studio 2, какую часть анимации показывать, и представляет собой значение между 0 и номер_изображения. Остальные аргументы аналогичны тем, которые используются при обычном рисовании спрайта, и изменяют масштаб, угол, цвет наложения и альфа-код конечного рисуемого кадра анимации. ВНИМАНИЕ! Из-за того, что скелетная анимация интерполируется между различными "ключевыми" кадрами, с этим вызовом связаны дополнительные накладные расходы, поэтому рекомендуется избегать рисования скелетных спрайтов с помощью этого метода, если это не необходимо. этот метод, если в этом нет крайней необходимости. Синтаксис: draw_skeleton(sprite, animname, skinname, frame, x, y, xscale, yscale, rot, color, alpha); Аргумент Описание спрайт Индекс спрайта для рисования. animname Имя анимации, из которой нужно получить кадр (строка). skinname Имя кожи для использования (строка). рама Кадр анимации для рисования (от 0 до image_number - 1). x Координата x места рисования спрайта. y Координата y места рисования спрайта. xscale Горизонтальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... yscale Вертикальное масштабирование спрайта в виде множителя: 1 = нормальное масштабирование, 0.5 - половина и т.д.... гниение Поворот спрайта. 0=нормальный, 90=повернутый на 90 градусов против часовой стрелки и т.д. цвет Цвет, с которым будет смешиваться спрайт. альфа Альфа спрайта (от 0 до 1, где 0 - прозрачный, а 1 - непрозрачный). Возвращается: Н/Д Пример: draw_skeleton(spr_Jump, "jump", "skin1", 0, x, y, image_xscale, image_yscale, image_angle, c_white, 0.5); Приведенный выше код нарисует первый кадр заданного скелетного анимационного спрайта, используя набор анимации "прыжок", скин, определенный как "skin1", в том же масштабе и положении, как определено в свойствах экземпляра, но с альфой 0,5.
