Манипуляция путями В GameMaker Studio 2 вы не ограничены только контурами, которые вы создаете с помощью редактора контуров. Существует ряд функций, которые можно использовать для манипулирования и изменения этих путей во время выполнения, или даже для создания новых путей и последующего изменения и манипулирования ими. Для изменения путей используются следующие функции: путь_существует путь_добавления точка_добавления_пути точка_изменения_пути точка_вставки_пути путь_удаления_точки путь_очистить_точки путь_приложения путь_назначения путь_удаления путь_дубликат путь_флип путь_зеркала обратный путь вращение_пути path_rescale путь_закрыт вид_пути path_set_precision сдвиг_пути Назад: Пути Следующий: Переменные пути © Copyright YoYo Games Ltd. 2021 Все права защищены
