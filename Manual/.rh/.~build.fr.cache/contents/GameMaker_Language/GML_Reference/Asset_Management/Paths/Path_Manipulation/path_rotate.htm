вращение_пути Вы можете использовать эту функцию для поворота заданного пути вокруг его центра. Помните, что в GameMaker Studio 2 (если вы не используете физику) углы рассчитываются против часовой стрелки, поэтому поворот пути на 90 градусов будет повернет путь влево. Эта функция изменяет фактический ресурс пути, поэтому она будет постоянно влиять на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. игры. Если это не то, что вам нужно, то вам следует использовать функцию типа path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать на активе). не забудьте вызвать path_delete() на активе, когда он больше не нужен). ПРИМЕЧАНИЕ: После того, как путь был назначен экземпляру с помощью функции path_start() функцию, вы можете использовать переменную path_orientation для изменения поворота пути. Синтаксис: path_rotate(index, angle); Аргумент Описание индекс Индекс пути, который нужно перевернуть. угол Угол поворота в градусах. Возвращается: Н/Д Пример: path_rotate(pth_Patrol, 90); Это повернет заданную величину на 90 градусов против часовой стрелки.
