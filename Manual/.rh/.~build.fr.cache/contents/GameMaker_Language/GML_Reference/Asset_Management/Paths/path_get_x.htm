path_get_x С помощью этой функции вы можете получить координату x позиции на любом заданном пути. 0 - это начало пути, 1 - конец пути, а все, что находится между ними, приравнивается к этому расстоянию на пути. Это не обязательно должна быть определяющая точка пути. это может быть любое место на пути. Синтаксис: path_get_x(ind, pos); Аргумент Описание индекс Индекс пути для проверки. поз Как далеко по пути нужно пройти для проверки. От 0 (начало) до 1 (конец). Возвращается: Реально Пример: x = path_get_x(pth_WallWalk, 0.5); Это установит x вызывающего экземпляра в координату x точки, расположенной ровно на половине заданного пути.
