ориентация_пути Эта переменная хранит текущую ориентацию пути, который был назначен экземпляру при вызове функции path_start(). При создании пути его ориентация по умолчанию равна 0 градусов, но с помощью этой переменной можно установить любое значение. Помните, что в GameMaker Studio 2 (если вы не используете физику) углы рассчитываются против часовой стрелки, поэтому установка ориентации пути на 90° приведет к повороту пути влево. Синтаксис: ориентация_пути; Возвращается: Реально Пример: mypath = path_duplicate(choose(path_1, path_2, path_3, path_4)); path_start(path, 4, path_action_reverse, 0); path_orientation = 90; Приведенный выше код дублирует случайный, предварительно созданный актив пути в переменную "mypath". Затем этот новый путь запускается и поворачивается на 90°. Назад: Переменные пути Далее: path_endaction © Copyright YoYo Games Ltd. 2021 Все права защищены
