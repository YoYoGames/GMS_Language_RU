индекс_пути Все ресурсы в GameMaker Studio 2 имеют уникальный идентификационный номер, а path_index - это переменная, доступная только для чтения, которая хранит значение индекса для данного path ресурса, который был назначен экземпляру с помощью функции path_start(). Если экземпляр не имеет назначенного пути, переменная будет установлена в -1. Синтаксис: path_index; Возвращается: целое число (идентификатор пути) Пример: if (path_index == -1) { path_start(pth_enemy3, 4, path_action_reverse, 0); } Приведенный выше код проверяет, был ли назначен путь данному экземпляру, и если нет, то начинает новый путь (одновременно назначая его на path_index ).
