скорость движения Вы можете использовать эту функцию для получения или задания скорости пути после того, как он был запущен с помощью функции path_start(). Вы можете использовать отрицательные значения для обозначения того, что экземпляр должен следовать путь в обратном направлении. Синтаксис: скорость пути; Возвращается: Реально Пример: скорость_пути = -1 + random(2); Приведенный выше код задаст экземпляру прохождение пути со случайной скоростью в диапазоне от -1 до 1.
