путь_дубликат Эта функция берет путь и копирует его в новый путь. Новый путь создается в процессе, а индекс возвращается для использования во всех дальнейших вызовах для использования этого нового пути. Синтаксис: path_duplicate(index); Аргумент Описание индекс Индекс существующего пути для дублирования. Возвращается: Реально Пример: mypath = path_duplicate(choose(pth_1, pth_2, pth_3, pth_4)); Приведенный выше код выбирает один из четырех ресурсов пути и дублирует его, сохраняя индекс нового пути в переменной "mypath".
