путь_get_closed Эта функция может быть использована для возврата информации о том, отмечен ли путь как закрытый (true) или открытый (false), т.е. зацикливается ли путь или имеет ли он определенное начало и конец. Синтаксис: path_get_closed(index); Аргумент Описание индекс Индекс пути для проверки. Возвращается: Булево Пример: state = path_get_closed(pth_Patrol); Это установит "state" в true или false в зависимости от закрытого состояния пути, индексированного в "pth_Patrol".
