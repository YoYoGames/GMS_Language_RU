path_set_precision С помощью этой функции можно изменить "гладкость" контура. Это значение должно быть от 1 до 8, при этом низкое значение создает более прямые края с более резкими изгибами между точками, а более высокое значение округляет точки и делает контур более "кривым". более "кривым". Обратите внимание, что эта функция не будет иметь видимого эффекта, если путь не был установлен на сглаживание в редакторе путей или с помощью функции path_set_kind(). Эта функция изменяет фактический актив пути, и, следовательно, навсегда повлияет на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. Если это не то, что вам нужно, то следует использовать функцию path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен). Синтаксис: path_set_precision(index, prec); Аргумент Описание индекс Индекс пути, который нужно изменить. prec Точность пути. Должно быть целым числом от 1 до 8. Возвращается: Н/Д Пример: path_set_precision(pth_Patrol, 2); Это установит точность заданного пути равной 2.
