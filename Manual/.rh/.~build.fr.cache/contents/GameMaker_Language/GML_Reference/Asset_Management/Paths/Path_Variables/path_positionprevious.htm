путь_позиция_предыдущий Эта переменная может быть использована для получения или установки положения экземпляра на его текущем пути в предыдущем шаге и представляет собой нормализованное значение между 0 и 1, то есть: 0 - это начальное положение пути, а 1 - конечное. Он аналогичен параметру xprevious и yprevious по принципу работы, только она специфична для путей. Она может быть полезна для например, для временной остановки последователя пути, если что-то стоит на пути (см. пример кода ниже). Синтаксис: path_positionprevious; Возвращается: Реально Пример: var _x = x + lengthdir_x(16, direction); var _y = y + lengthdir_y(16, direction); if collision_circle(xx, yy, 16, obj_Player, false,true) { path_position = path_positionprevious; } Приведенный выше код проверяет область перед экземпляром на наличие столкновения с объектом "obj_Player" и, если таковое имеется, возвращает экземпляр в предыдущую позицию, которую он занимал на текущем пути, как назначено функцией path_start().
