path_get_point_y Эта функция возвращает положение y (в комнатных координатах) точки, которую вы ввели для указанного пути. Если точка находится за пределами диапазона пути (т.е. путь имеет 8 точек, а вы запрашиваете положение y точки 10), то будет возвращено значение будет возвращено значение 0. Синтаксис: path_get_point_y(index, n); Аргумент Описание индекс Индекс пути для проверки. n Идентификация точки для проверки. Возвращается: Реально Пример: if (path_position == 1) { var _num = path_get_number(pth_Patrol); var _pos = floor(random(_num)); x = path_get_point_x(pth_Patrol, _pos); y = path_get_point_y(pth_Patrol, _pos); path_position = (1 / _num) * _pos; } Приведенный выше код проверяет, находится ли экземпляр в конце пути. Если да, то он выберет случайную точку на пути и переместит экземпляр в эту точку. Назад: Пути Далее: path_get_precision © Copyright YoYo Games Ltd. 2021 Все права защищены
