Пути ВGameMaker Studio 2 имеется обширная коллекция функций и встроенных переменных для использования путей в вашей игре. В этом разделе рассматриваются основные из них, которые используются при работе с путями, ранее созданными в редакторе путей, Однако существуют и другие функции, позволяющие создавать собственные пути с помощью кода и манипулировать ими различными способами. Существуют следующие основные функции, работающие с путями: путь_начало конец_пути путь_get_closed path_get_kind длина_пути имя_пути path_get_number path_get_point_speed path_get_point_x path_get_point_y path_get_precision путь_получить_скорость path_get_x path_get_y Чтобы облегчить отладку путей, есть функция, которая также рисует его на экране: нарисовать_путь Каждый экземпляр объекта также имеет ряд встроенных переменных, предназначенных для облегчения использования путей, и эти переменные перечислены в следующем разделе: Переменные пути Наконец, можно манипулировать содержимым пути, а также создавать новые пути во время работы игры.
