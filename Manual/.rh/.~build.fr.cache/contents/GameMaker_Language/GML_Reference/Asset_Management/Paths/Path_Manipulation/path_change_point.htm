точка_изменения_пути С помощью этой функции можно изменить положение и/или коэффициент скорости любой точки, ранее определенной для пути (путь может быть создан в редакторе путей или в коде с помощью функции path_add()). При использовании функции на пути из браузера активов, обратите внимание, что функция изменит фактический актив и, таким образом, навсегда повлияет на то, как этот путь используется всеми экземплярами в игре с момента использования функции и до конца игры. Если это не то, что вам нужно, то следует использовать функцию типа path_duplicate() для создания копии пути, а затем вызвать эту функцию на дублированном активе (не забудьте вызвать path_delete() на активе, когда он больше не нужен). Синтаксис: path_change_point(index, n, x, y, speed); Аргумент Описание индекс Индекс пути для изменения точки. n Определяющая точка для изменения атрибутов. x Новая координата x (относительно пути) точки. y Новая координата y (относительно пути) точки. скорость Новый коэффициент скорости точки. Возвращается: Н/Д Пример: for (var i = 0; i < path_get_number(path0); ++i;) { var px = path_get_point_x(pth_Patrol, i) + 64 - random(128); var py = path_get_point_y(pth_Patrol, i) + 64 - random(128); path_change_point(pth_Patrol, i, px, py, 100); } Приведенный выше код перебирает все точки пути, индексированные как "path0", и переставляет все точки в случайное положение в области размером 128x128 пикселей.
