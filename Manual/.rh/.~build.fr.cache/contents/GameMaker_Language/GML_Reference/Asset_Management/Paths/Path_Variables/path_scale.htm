масштаб_пути Это значение можно использовать для получения или установки масштаба текущего назначенного пути для экземпляра (как установлено функцией path_start()) со значением по умолчанию 1. Это скалярное значение, так что 1 - это масштаб 1:1, в то время как установка значения 2, например, удвоит масштаб, а установка значения 0.5 уменьшит масштаб вдвое. Синтаксис: path_scale; Возвращается: Реально Пример: path_scale = 1 + random(2); Приведенный выше код установит масштаб текущего пути на случайный размер между 1 (исходный размер) и 3 (в 3 раза больше исходного размера).
