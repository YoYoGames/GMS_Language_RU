script_exists Эта функция возвращает true или false в зависимости от того, существует ли сценарий или функция сценария с заданным индексом. Обратите внимание, что это не строка, а имя актива, который содержит уникальный индекс для каждого скрипта (как он будет отображаться в IDE) или переменная имени функции скрипта, как определено в активе скрипта (обратите внимание, что это не работает для переменных метода). Для получения дополнительной информации о скриптах см. раздел Редактор скриптов. Синтаксис: script_exists(scr); Аргумент Описание scr Имя сценария, который вы хотите проверить. Возвращается: Булево Пример: script[0] = -1; script[1] = AI_Left; script[2] = AI_Right; var script_num = choose(0, 1, 2); if script_exists(script[script_num]) { script_execute(script[script_num]); } Приведенный выше пример добавляет две функции сценария и значение в массив, затем получает случайное число и использует его для выбора функции сценария для запуска, если не выбрано значение -1, в этом случае ничего не произойдет.
