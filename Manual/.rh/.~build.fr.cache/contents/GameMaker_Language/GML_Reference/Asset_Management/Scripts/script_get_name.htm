имя_скрипта Эта функция возвращает имя в виде строки указанного скрипта. Это имя - то, которое было указано для скрипта в Asset Browser главного окна GameMaker Studio 2. Дополнительные сведения о скриптах см. в разделе Редактор скриптов. Синтаксис: script_get_name(scr); Аргумент Описание scr Индекс сценария, имя которого вы хотите получить. Возвращается: Строка Пример: scr_name = script_get_name(Help_File); Приведенный выше код примера будет хранить имя указанного индекса скрипта в переменной "scr_name".
