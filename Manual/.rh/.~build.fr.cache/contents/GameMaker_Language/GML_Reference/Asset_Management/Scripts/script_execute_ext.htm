script_execute_ext Эта функция работает аналогично функции script_execute(), только вы можете предоставить массив, содержащий аргументы, необходимые для вызываемой функции/скрипта. Вы также можете передать два необязательных аргумента, чтобы указать смещение в массиве для получения аргументов, а также количество аргументов из массива (оно должно быть не более array_length - offset). Синтаксис: script_execute(scr, array_args, [offset], [num_args]); Аргумент Описание scr Имя функции/скрипта, который вы хотите вызвать array_args Массив, содержащий аргументы для функции/скрипта [смещение]. ВАРИАНТ! Смещение в массиве аргументов [num_args] ВАРИАНТ! Количество аргументов для (начиная со смещения и далее) Возвращается: Будет зависеть от возвращаемого значения вызываемой функции/скрипта Пример: script_execute_ext(move_inst, move_array, floor(random(4)), 1); Приведенный выше код примера будет использовать script_execute_ext для вызова заданной функции, предоставляя массив из четырех аргументов, но используя только один из них случайным образом.
