имя_объекта Эта функция возвращает имя в виде строки указанного объекта. Это имя - то, которое было указано для объекта в Asset Browser главного окна GameMaker Studio 2. Обратите внимание, что это только строка и не может быть использована для прямой ссылки на объект - для этого вам понадобится индекс объекта. Однако вы можете использовать эту строку для получения индекса объекта, используя возвращаемую строку вместе с функцией asset_get_index(). Синтаксис: object_get_name(obj); Аргумент Описание obj Индекс объекта для проверки. Возвращается: Строка Пример: str = object_get_name(object_index); Приведенный выше код получит имя индекса объекта для экземпляра, выполняющего код, и сохранит возвращаемое значение в переменной "str".
