объект_события Эта переменная, доступная только для чтения, возвращает индекс объекта экземпляра, в котором выполняется проверяемое событие. Синтаксис: event_object; Возвращается: Реально Пример: global.obj = event_object; Приведенный выше код хранит индекс объекта экземпляра, выполняющего событие, в глобальной переменной.
