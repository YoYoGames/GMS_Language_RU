object_set_visible С помощью этой функции вы можете отметить определенный объект как видимый или нет (более подробную информацию о флаге "видимый" см. в разделе Редактор объектов ). Это означает, что все экземпляры этого объекта, созданные после изменения параметра "видимый ", будут созданы с этим новым состоянием, в то время как экземпляры, уже находящиеся в комнате, могут быть не затронуты. Обратите внимание, что это не функция экземпляра! Вы можете установить флаг видимости отдельных экземпляров с помощью переменной visible и таким образом десять экземпляров будут отмечены как видимые, даже если в IDE флаг объекта установлен false. Даже если вы измените флаг видимости объекта на true с помощью этой функции, все экземпляры, которые в данный момент находятся в комнате, останутся такими, какими они были, и только экземпляры, созданные после вызова функции, начнут работать с флагом видимости true. вызовом этой функции, начнут работать с флагом visible, установленным как true. Синтаксис: object_set_visible( index, vis ); Аргумент Описание индекс Индекс объекта, который нужно изменить. vis Новая видимость объекта (true=видимый, false=невидимый). Возвращается: Н/Д Пример: if !object_get_visible(obj_Block) { object_set_visible(obj_Block, true); } Приведенный выше код проверяет флаг видимости объекта "obj_Block", и если он ложный, то он изменяется на истинный.
