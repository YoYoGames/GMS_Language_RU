объект_существует Эта функция возвращает, существует ли объект с указанным индексом или нет. Обратите внимание, что здесь проверяется, присутствует ли объект в браузере активов, а не в игровой комнате. Для этого следует использовать функцию instance_exists(). Синтаксис: object_exists(obj); Аргумент Описание obj Индекс объекта для проверки. Возвращается: Булево Пример: if object_exists(obj_temp) { with (obj_Temp) scr_add_event(); } Приведенный выше код проверяет переменную "obj_temp", чтобы убедиться, что в ней хранится индекс объекта, а затем, если это так, заставляет объект выполнить сценарий.
