object_set_sprite С помощью этой функции вы можете установить индекс спрайта определенного объекта. Это означает, что все экземпляры этого объекта, созданные после изменения sprite_index, будут созданы с этим новым sprite_index, в то время как экземпляры, уже находящиеся в комнате, могут быть не затронуты. Обратите внимание, что это не функция экземпляра! Вы можете установить индекс спрайта отдельных экземпляров с помощью переменной sprite_index переменную и таким образом иметь десять экземпляров, каждый из которых будет иметь и даже если вы измените индекс спрайта объекта с помощью этой функции, все экземпляры, которые находятся в комнате в данный момент, останутся такими, какими они были, и только экземпляры, созданные после вызова функции будут начинаться с нового спрайта. Синтаксис: object_set_sprite( index, spr ); Аргумент Описание индекс Индекс объекта, который нужно изменить. spr Спрайт, который нужно присвоить объекту. Возвращается: Н/Д Пример: switch (room) { case rm_start: object_set_sprite(obj_Player, spr_uniform); break; case rm_middle: object_set_sprite(obj_Player, spr_swimsuit); break; case rm_end: object_set_sprite(obj_Player, spr_casual); break; } instance_create_layer(32, 32, "Instances", obj_Player); Приведенный выше код установит индекс спрайта объекта "obj_Player" на разные значения в зависимости от комнаты, в которой в данный момент находится экземпляр, выполняющий код.
