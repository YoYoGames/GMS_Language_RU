object_get_mask Эта функция сообщит вам, имеет ли проверяемый объект индекс маски или нет, и если имеет, то вернет индекс этой маски, или -1, если не имеет. Обратите внимание, что это не функция экземпляра! Вы можете иметь объект без маски, а экземпляр того же объекта может иметь маску, и наоборот, или даже иметь разные маски. Вы можете установить индекс маски отдельного экземпляра с помощью переменной mask_index переменной instance. Синтаксис: object_get_mask(obj); Аргумент Описание obj Индекс объекта для проверки Возвращается: настоящий Пример: if (mask_index != object_get_mask(object_index)) { mask_index = object_get_mask(object_index); } В приведенном выше примере индекс маски экземпляра сверяется с маской индекса object_index экземпляра. Если они не совпадают, то экземпляру будет присвоена та же маска, что и у индекса объекта. Назад: Объекты Далее: object_get_parent © Copyright YoYo Games Ltd. 2021 Все права защищены
