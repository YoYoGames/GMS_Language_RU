object_get_physics Эта функция сообщит вам, был ли проверяемый объект помечен как "физика включена" - в этом случае она вернет true, - или нет - в этом случае она вернет false. Синтаксис: object_get_physics(obj); Аргумент Описание obj Индекс объекта для проверки. Возвращается: Булево Пример: if object_get_physics(object_index) { phy_active = true; } Приведенный выше код проверит запущенный экземпляр на предмет того, включен ли в объект, из которого он создан, физический режим, и если включен, то активирует симуляцию физики для данного экземпляра.
