Структура ключевого кадра Каждая дорожка (как определено на странице, описывающей структуры дорожек) будет иметь одну или более структур ключевых кадров, назначенных ей (которые вы можете получить из свойства ключевые кадры свойство track struct), и каждый из них будет иметь следующие свойства: кадр: Позиция (в кадрах) на временной шкале для ключевого кадра. Значение по умолчанию равно 0. длина: Длина ключевого кадра. Значение по умолчанию равно 1, при установке больших значений свойство дорожки, на которое ссылается ключевой кадр, будет поддерживаться на начальном значении в течение заданной длины. Обратите внимание, что свойство stretch отменяет это свойство, если оно установлено в true. stretch: Если это свойство установлено в true, то ключевой кадр растягивается либо до следующего ключевого кадра дорожки, либо до конца дорожки, если это последний ключевой кадр . Вы можете получить или установить это значение, а значение по умолчанию равно false. каналы: Это свойство позволяет получить доступ к списку структур данныхключевых кадров для каналов дорожки. При получении этого свойства возвращается массив keyframe data structs возвращается, а при установке этого свойства должен быть указан массив keyframe data structs.
