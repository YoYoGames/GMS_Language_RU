Структура данных ключевого кадра Когда вы определяете структуруключевых кадров для дорожки, необходимо также определить данные, которые будут с ними связаны. Эти данные состоят из различных каналов, и каждому каналу может быть присвоен определенный тип данных в зависимости от того, для какого типа дорожки вы устанавливаете данные ключевого кадра. Этими свойствами являются: канал: Это канал, к которому должны быть применены данные ключевого кадра. Это целое положительное значение, начинающееся с 0. Стоит отметить, что при создании дорожек параметров для ключевых кадров "положение" или "масштаб", необходимо использовать очень специфические значения каналов. Это: канал 0 - положение X или масштаб X, канал 1 - положение Y или масштаб Y. spriteIndex: Индекс актива спрайта, используемого для дорожки. Это свойство доступно только для дорожек типа seqtracktype_graphic, и вы можете получить или установить его. soundIndex: Индекс аудиоактива, используемого для дорожки. Это свойство доступно только для дорожек типа seqtracktype_audio, и вы можете получить или установить его. playbackMode: Режим воспроизведения для звука. Это свойство доступно только для дорожек типа seqtracktype_audio, и вы можете получить или установить его. Свойство должно быть одной из следующих констант: Постоянная Описание Значение seqaudiokey_loop При воспроизведении звук будет зацикливаться. 0 seqaudiokey_oneshot Звук будет воспроизводиться только один раз, а затем прекратится. 1 кривая: Это свойство требует наличия структуры кривой анимации (см. здесь для получения дополнительной информации) и доступно только для дорожек типа seqtracktype_real. Если структура кривой не то значение этого свойства будет равно -1. значение: Это свойство - просто значение, связанное с каналом данных ключевых кадров, и доступно только для дорожек типа seqtracktype_real, когда не задана структура кривой. не предоставлена. color: Это свойство возвращает (или требует, если установлено) массив для значения цвета ключевого кадра с форматом [A, R, G, B]. Это свойство доступно только для дорожек типа seqtracktype_colour. Обратите внимание, что значения для каждого компонента должны быть выражены между 0 и 1, где 0 соответствует HEX-значению #00, а 1 - HEX-значению #FF (0 - 255, как показано в окне выбора цвета для цветных дорожек в редакторе последовательностей). последовательность: Это свойство возвращает (или требует, если установлено) объект sequence struct и доступно только для дорожек типа seqtracktype_sequence. objectIndex: Это свойство возвращает (или требует, если установлено) индекс объекта и доступно только для дорожек типа seqtracktype_instance. события: Это свойство позволяет получить доступ к событиям и широковещательным сообщениям, связанным со структурой данных ключевых кадров. Вы можете получить или установить это свойство, причем при получении возвращается массив возвращается массив строк, а при установке необходимо указать массив строк. Более подробную информацию о событиях можно найти в разделе События и моменты последовательности. Это свойство доступно только для дорожек типа seqtracktype_message. событие: Это свойство возвращает (или может быть установлено на) метод, связанный со структурой данных ключевого кадра. Если метод не был указан или вы хотите удалить Это свойство доступно только для треков типа seqtracktype_moment.
