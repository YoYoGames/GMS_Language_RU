Структура экземпляра последовательности Последовательность, когда она помещается в комнату, размещается как элемент на слое, и этот элемент содержит экземпляр последовательности. Этот экземпляр последовательности будет иметь следующие свойства в своей структуре: последовательность: Это идентификатор структуры объекта последовательности, на основе которой был создан экземпляр последовательности (подробнее см. здесь). Вы можете получить или установить эту структуру, и таким образом изменить базовую последовательность, на которую ссылается данный экземпляр. headPosition: Это текущая позиция головки воспроизведения (в кадрах) для экземпляра последовательности. Вы можете получить или установить это значение, но учтите, что установка его в значение больше или меньше, чем текущей определенной области воспроизведения будет иметь различные эффекты в зависимости от типа воспроизведения, установленного для последовательности. См. layer_sequence_headpos() для получения дополнительной информации. headDirection: Это текущее направление воспроизведения для экземпляра последовательности. Вы можете получить или установить это значение с помощью следующих констант (обратите внимание, что вы можете использовать функции слоя layer_sequence_get_headdir() и layer_sequence_headdir() для получения или установки этого значения без прямого доступа к struct): Постоянная Описание Значение seqdir_right Последовательность будет воспроизводить кадры в инкрементном порядке слева направо 1 seqdir_left Последовательность будет воспроизводить кадры в убывающем порядке справа налево -1 speedScale: Это свойство можно использовать для получения или установки шкалы скорости воспроизведения. Шкала скорости - это множитель, где 1 - скорость воспроизведения по умолчанию, значения меньше 1 замедляют воспроизведение, а значения больше 1 - ускоряют. Например: значение 0,5 - это половина скорости воспроизведения, а значение 2 - удвоенная скорость. Обратите внимание, что вы можете использовать функции слоя layer_sequence_get_speedscale() и layer_sequence_speedscale() для получения или установки этого значения без прямого доступа к структуре. paused: Вы можете проверить это свойство, чтобы узнать, была ли последовательность приостановлена или нет, и оно будет истинным, если была, или ложным в противном случае. Это свойство доступно только для чтения, но вы можете использовать функцию layer layer_sequence_pause() чтобы приостановить воспроизведение и layer_sequence_play() возобновить его при необходимости. Вы также можете проверить это свойство с помощью функции layer layer_sequence_is_paused() скорее чем проверять свойство непосредственно в структуре. закончен: Вы можете проверить это свойство, чтобы узнать, закончила ли последовательность играть или нет, возвращая true, если она закончила играть, и false в противном случае. Это свойство доступно только для чтения и возвращает true только для тех дорожек, которые не установлены на цикл или пинг-понг. Вы также можете проверить это свойство с помощью функции слоя layer_sequence_is_finished() вместо того, чтобы проверять свойство в структуре напрямую. elementID: Это свойство содержит идентификатор элемента последовательности. Этот идентификатор является простым идентифицирующим значением, которое связано со слоем в редакторе помещений, и его можно использовать с соответствующими функциями слоя, чтобы найти слой, которому была назначена последовательность, или изменить определенные свойства последовательности без необходимости работать со структурой. activeTracks: Это свойство будет содержать массив структур "evaluation", содержащих информацию о текущем состоянии каждой дорожки актива в последовательности (графика, последовательность, аудио и т.д...). Это только дорожки верхнего уровня, и возвращаемый массив доступен только для чтения. О содержимом структуры дорожки актива, возвращаемой в каждом элементе массива, см. раздел ниже. Следующий список содержит свойства, которые могут быть доступны вам при доступе к структуре дорожки (как возвращено свойством activeTracks родительской структуры последовательности, объяснено выше). Каждая структура дорожки актива может иметь некоторые или все из следующие свойства, в зависимости от типа актива, на который ссылается дорожка (обратите внимание, что вы можете получить или установить все эти свойства, кроме тех, которые указаны только для чтения), и все возвращаемые значения являются значениями для текущей позиции головки воспроизведения: матрица: Это матрица преобразования, используемая для рисования дорожки (дорожки спрайта, экземпляра и последовательности). posx: Положение актива в последовательности вдоль оси X для дорожки в текущей позиции головки воспроизведения (все активы). posy: Положение актива в последовательности вдоль оси Y для дорожки в текущей позиции головки воспроизведения (все активы). scalex: масштаб актива в последовательности вдоль оси X для дорожки в текущей позиции головки воспроизведения (дорожки спрайта, экземпляра и последовательности). scaley: Масштаб актива в последовательности по оси Y для дорожки в текущей позиции головки воспроизведения (дорожки спрайта, экземпляра и последовательности). xorigin: X-образное начало актива для дорожки (все активы). yorigin: Y происхождения актива для дорожки (все активы). width: Ширина (в пикселях) актива для дорожки (дорожки спрайта, экземпляра и последовательности). высота: Высота (в пикселях) актива для дорожки (дорожки спрайта, экземпляра и последовательности). imageindex: Индекс изображения для актива на дорожке в последовательности в текущей позиции головки воспроизведения (дорожки спрайта и экземпляра). imagespeed: Скорость изображения для актива на дорожке в последовательности в текущей позиции головки воспроизведения (дорожки спрайта и экземпляра). colorMultiply: Значение умножения цвета для актива на дорожке в последовательности в текущей позиции головки воспроизведения (дорожки спрайта, экземпляра и последовательности). Это значение будет представлять собой массив из четырех значений ARGB с форматом [A, R, G, B]. Обратите внимание, что значения для каждого компонента выражаются в диапазоне от 0 до 1, где 0 соответствует HEX-значению #00, а 1 - HEX-значению #FF (0 - 255 в формате 0 - 255). значению #FF (0 - 255, как показано в окне выбора цвета для дорожек с цветовым умножением в редакторе последовательностей). усиление: Усиление (громкость) дорожки в текущем положении головки воспроизведения (аудиодорожки). питч: Высота тона дорожки в текущем положении головки воспроизведения (аудиодорожки). трек: Структура данных трека, на которой основан этот трек (все активы, значение только для чтения ). parent: ID экземпляра родительской последовательности для дорожки (все активы, значение только для чтения ). activeTracks: Это массив оценочных структур для каждой дорожки параметров, которую содержит дорожка актива (все активы, значение только для чтения ).
