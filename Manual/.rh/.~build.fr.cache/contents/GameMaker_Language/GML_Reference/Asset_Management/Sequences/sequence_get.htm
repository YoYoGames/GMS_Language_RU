последовательность_get С помощью этой функции вы можете получить структуру объекта последовательности из индекса актива последовательности. Вы указываете индекс актива для последовательности, как определено в браузере активов, и функция вернет структуру объекта последовательности, к которой можно получить доступ, или -1, если последовательность не существует или указанный индекс актива не является последовательностью. Синтаксис: sequence_get(sequence_index); Аргумент Описание индекс_последовательности Индекс последовательности, определенный в обозревателе активов. Возвращается: структура или -1 Пример: var _seq = sequence_get(seq_Logo); seq.loopmode = seqplay_pingpong; Приведенный выше код извлекает структуру объекта последовательности из объекта последовательности "seq_Loop", а затем устанавливает режим цикла для последовательности (все последующие созданные экземпляры последовательности теперь будут использовать этот режим цикла).
