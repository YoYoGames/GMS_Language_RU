sequence_instance_override_object С помощью этой функции вы можете переопределить (заменить) все экземпляры объекта, используемые в последовательности, на другой. Вы предоставляете идентификатор структуры экземпляра последовательности (полученный при создании экземпляра последовательности в помещении или с помощью одной из функций слоя помещения), а также индекс объекта (определенный в браузере активов) для объекта. - см. здесь), а также индекс объекта (определенный в браузере активов) для объекта, который вы хотите переопределить. Наконец, вы указываете индекс объекта или идентификатор экземпляра для использования в качестве объекта, который будет переопределять последовательность (при указании ID экземпляра в качестве переопределяемого объекта будет использоваться объект, на основе которого был создан экземпляр). Обратите внимание, что это может быть сделано только для экземпляров последовательности (не для объектов последовательности) и должно быть сделано до начала воспроизведения последовательности, иначе она не будет работать. Синтаксис: sequence_instance_override_object(sequence_instance_struct, object_id, instance_or_object_id); Аргумент Описание структура_объекта_последовательности Структура экземпляра последовательности для изменения. идентификатор объекта Индекс объекта в последовательности для переопределения. идентификатор экземпляра_или_объекта Индекс объекта или идентификатор экземпляра, используемый для переопределения объектов последовательности. Возвращается: Н/Д Пример: var _seq = layer_sequence_create("Background", 0, 0, 0, seq_AnimatedBackground); var _seq_inst = layer_sequence_get_instance(_seq); sequence_instance_override_object(_seq_inst, obj_Trees_Winter, obj_Trees_Summer); Приведенный выше код создает новый экземпляр последовательности на данном слое, а затем изменяет его таким образом, что все экземпляры объекта "obj_Trees_Winter" заменяются экземплярами объекта "obj_Trees_Summer".
