Структура трека Структура track struct может использоваться либо длятрековактивовверхнего уровня , либо для трековпараметров подтрека , и поведение трека определяется двумя вещами, его именем и типом . Структуры треков создаются с помощью функции sequence_track_new() и могут быть получены из активов последовательности с помощью свойства activeTracks из структуры экземпляра последовательности или свойства tracks из структуры объекта последовательности. Для структуры трека доступны следующие свойства: тип: Следующие константы могут быть использованы для определения типа дорожки, создаваемой в последовательности (эти константы также используются при генерации ключевых кадров и данных ключевых кадров, о чем говорится в разделах ниже): Постоянная Описание Значение seqtracktype_graphic Это дорожка графических (спрайтовых) активов. 1 seqtracktype_audio Это дорожка аудиоактивов. 2 seqtracktype_instance Это трек актива экземпляра. 14 seqtracktype_sequence Это трек актива последовательности. 7 seqtracktype_clipmask Это дорожка активов группы масок клипов. 8 seqtracktype_clipmask_mask Это дорожка актива спрайта маски клипа, используемая для генерации маски клипа. 9 seqtracktype_clipmask_subject Это дорожка актива спрайта маски клипа, который маскируется. 10 seqtracktype_group Эта групповая папка отслеживает активы. 11 seqtracktype_colour Это дорожка параметров цветовых данных. 4 seqtracktype_real Это трек параметра значения реального числа. 3 seqtracktype_message Это дорожка широковещательных сообщений. 15 seqtracktype_moment Это трек события/момента. 16 seqtracktype_bool В настоящее время не используется. 5 seqtracktype_string В настоящее время не используется. 6 seqtracktype_spriteframes В настоящее время не используется. 13 seqtracktype_empty В настоящее время не используется. 12 имя: При создании дорожки активов имя, которое вы здесь задаете, может быть любой строкой, необходимой для идентификации дорожки, однако для дорожек параметров необходимо указать специальные строки, чтобы сообщить GameMaker Studio 2, какую дорожку параметров (эти строки применимы только к дорожкам параметров типов seqtracktype_real и seqtracktype_colour ): "position" - при установке на дорожке типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться как позиционные данные. " scale" - при установке на дорожке типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться как данные масштабирования. "rotation" - при установке на дорожке типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться как данные о вращении. " blend_multiply" - при установке для дорожки типа seqtracktype_colour указывает, что содержимое этой дорожки должно использоваться как данные умножения цвета. Данные ключевого кадра для такой дорожки должны представлять собой массив из четырех значений в формате [A, R, G, B]. Обратите внимание, что значения для каждого компонента выражаются между 0 и 1, где 0 соответствует HEX-значению #00, а 1 - HEX-значению #FF (0 - 255, как показано в цветовом редакторе для дорожек наложения изображений в разделе "Цветовая палитра"). для дорожек смешивания изображений в редакторе последовательностей). " image_speed" - при установке на дорожку типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве значения скорости изображения. " image_index" - при установке на дорожке типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве значения индекса изображения. " image_angle" - при установке на дорожке типа seqtracktype_real указывает, что содержимое этой дорожки должно использоваться в качестве значения угла изображения. " image_blend" - при установке для дорожки типа seqtracktype_colour указывает, что содержимое этой дорожки должно использоваться в качестве значения наложения изображения. Данные ключевого кадра для такой дорожки должны представлять собой массив из четырех значений в формате [A, R, G, B]. Обратите внимание, что значения для каждого компонента выражаются между 0 и 1, где 0 соответствует HEX-значению #00, а 1 - HEX-значению #FF (0 - 255, как показано в подборщике цветов для дорожек с цветовым умножением в редакторе последовательностей). треки: Это свойство позволяет получить доступ к списку дорожек, которые являются дочерними для данной дорожки. При получении этого свойства возвращается массив структур дорожек, а при установке этого свойства должен быть указан массив структур дорожек. visible: Указывает, является ли эта дорожка видимой (значение true) или нет (значение false). Вы можете получить или установить это значение, и если дорожка не видна, то ни одна из ее дочерних дорожек не будет отрисована. ключевые кадры: Это свойство позволяет получить доступ к списку структурключевых кадров для дорожки. При получении этого свойства возвращается массив структур ключевых кадров, а при установке этого свойства необходимо указать массив структур ключевых кадров. должен быть указан массив структур ключевых кадров.
