уничтожение последовательности С помощью этой функции вы можете уничтожить объект последовательности, который был создан динамически. Вы передаете либо структуру объекта последовательности (возвращаемую функцией sequence_create()) или идентификатор последовательности (возвращаемый функцией layer_sequence_get_sequence() или из свойства sequence struct экземпляра последовательности). Эту функцию следует использовать всякий раз, когда динамически созданная последовательность больше не нужна, чтобы освободить связанную с ней память. Синтаксис: sequence_destroy(sequence_struct_or_id); Аргумент Описание sequence_struct_or_id Структура или идентификатор объекта последовательности для уничтожения Возвращается: Н/Д Пример: if sequence_exists(my_seq) { sequence_destroy(my_seq); } Приведенный выше код проверяет, существует ли данный объект последовательности, и если да, то он уничтожается.
