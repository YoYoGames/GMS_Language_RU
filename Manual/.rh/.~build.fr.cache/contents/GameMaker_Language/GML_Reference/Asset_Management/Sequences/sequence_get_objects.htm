sequence_get_objects С помощью этой функции вы можете получить массив всех индексов объектов, экземпляры которых создаются в данной последовательности. Вы передаете либо структуру объекта последовательности (как возвращается функцией sequence_create() или sequence_get()) или идентификатор последовательности (возвращаемый функцией layer_sequence_get_sequence() или из свойства sequence struct экземпляра последовательности), и функция вернет массив, где каждый элемент массива является идентификатором объектный_индекс для различных объектов, используемых последовательностью для создания экземпляров. Синтаксис: sequence_get_objects(sequence_struct_or_id); Аргумент Описание sequence_struct_or_id Структура или идентификатор объекта последовательности для получения объектов. Возвращается: Массив Пример: var _seq = sequence_get(seq_Logo); obj_array = sequence_get_objects(_seq); Приведенный выше код получает структуру для объекта последовательности, затем извлекает объекты, которые она использует, и сохраняет массив в переменной.
