идентификатор экземпляра В этом массиве, доступном только для чтения, хранятся идентификаторы всех активных экземпляров в комнате. Это означает, что если вы использовали какую-либо из функций Instance Deactivate, те экземпляры, которые были деактивированы, не будут включены в этот массив (если вы использовали значение из этого массива ранее, теперь оно вернет ключевое слово noone). Синтаксис: instance_id[num]); Возвращается: Real (значение идентификатора экземпляра) Пример: for (var i = 0; i < instance_count; i ++;) { with (instance_id[i]) speed += 0.1; } Приведенный выше код пройдется по всем экземплярам в комнате и добавит 0,1 к их скорости.
