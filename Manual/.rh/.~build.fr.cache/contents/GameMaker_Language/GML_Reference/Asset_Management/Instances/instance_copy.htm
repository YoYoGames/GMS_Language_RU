экземпляр_копии С помощью этой функции вы можете "клонировать" экземпляр, так как при этом будет создана новая версия экземпляра, выполняющая код на той же позиции. Аргумент "perf" используется для указания новому экземпляру выполнить событие create или нет. Эта функция возвращает значение id нового экземпляра, который затем может быть сохранен в переменной или использован для доступа к этому экземпляру. ПРИМЕЧАНИЕ: Если вы решите не выполнять событие create, вы можете столкнуться с ошибками, если экземпляр зависит от каких-либо переменных, инициализированных в этом событии. Синтаксис: instance_copy(perf); Аргумент Описание перф Выполнять ли событие Create нового экземпляра (true) или нет (false). Возвращается: Real (значение идентификатора экземпляра) Пример: var inst = instance_number(object_index); if inst < 10 { instance_copy(true); } Приведенный выше код создает локальную переменную и использует ее для хранения количества экземпляров объекта, выполняющего код в комнате.
