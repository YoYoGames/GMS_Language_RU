экземпляр_найти Все экземпляры имеют уникальный идентификатор (id), который можно использовать для изменения и манипулирования ими во время игры, но вы не всегда можете знать, какой идентификатор у конкретного экземпляра, поэтому эта функция может помочь, так как с ее помощью вы можете перебрать их все, чтобы найти то, что вам нужно. Вы указываете объект, экземпляр которого хотите найти, и число, и если в списке экземпляров есть экземпляр в данной позиции, то функция возвращает идентификатор этого экземпляра, а если нет, то возвращает специальное ключевое слово noone. Вы также можете использовать ключевое слово all для перебора все экземпляры в комнате, а также родительский объект для итерации по всем экземплярам, которые являются частью иерархии родитель/ребенок, и вы даже можете указать экземпляр (если у вас есть его id) в качестве проверки, существует ли он в текущей комнате. он действительно существует в текущей комнате. Обратите внимание, что поскольку экземпляры сортируются произвольным образом, нет определенного порядка проверки экземпляров этой функцией, и любой экземпляр может находиться в любой позиции. Максимальное значение для "n" в этой функции будет равно Для ключевого слова all: количество экземпляров - 1 Для индекса объекта: instance_number(OBJ) - 1 Синтаксис: instance_find(obj, n); Аргумент Описание obj Объект для поиска n-го экземпляра n Номер экземпляра, который необходимо найти. Возвращается: Real (либо значение ID экземпляра, либо ключевое слово noone) Пример: for (var i = 0; i < instance_number(obj_Enemy); ++i;) { enemy[i] = instance_find(obj_Enemy,i); } Приведенный выше код будет использовать цикл for для перебора всех экземпляров "obj_Enemy" и сохранения их id в массиве "enemy[]".
