счётчик экземпляров С помощью этой переменной, доступной только для чтения, можно получить подсчет всех активных экземпляров, находящихся в комнате. Сюда входит экземпляр, выполняющий код, но не входят те экземпляры, которые были деактивированы с помощью функций деактивации экземпляра. Обратите внимание, что эта переменная дает только количество экземпляров на начало шага, поэтому любые изменения экземпляров в комнате, сделанные после начала шага, не будут учтены. Синтаксис: instance_count; Возвращается: Реально Пример: if (instance_count < 100) { var dif = 100 - instance_count; while (--dif > 0) { instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star); } } Приведенный выше код будет создавать несколько экземпляров объекта "obj_Star", пока общее количество экземпляров не достигнет 100.
