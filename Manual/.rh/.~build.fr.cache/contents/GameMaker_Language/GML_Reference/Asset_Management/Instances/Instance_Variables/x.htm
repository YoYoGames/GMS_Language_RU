x Значение x экземпляра - это горизонтальное положение в текущей комнате, измеряемое в пикселях. Это значение может быть 0, положительным или отрицательным, где 0 - это левая сторона комнаты, движение вправо увеличивает x, движение влево уменьшает x (отрицательное значение x означает, что экземпляр вышел за пределы левой стороны комнаты). означает, что экземпляр вышел за пределы левой стороны комнаты). Можно даже задать x реальное значение, например 12,345, но поскольку вы не можете получить .345 пикселя, может оказаться, что экземпляр не переместился, хотя эти значения будут сохранены и использоваться внутри GameMaker Studio 2 и являются совершенно правильными. Задавая значения x (и y), вы можете заставить объект прыгать по комнате в нужную вам позицию, или вы можете прибавлять и отнимать к ним меньшие значения, чтобы создать иллюзию движения без использования встроенных функций скорость и направление функции. Это следует учитывать при создании игр, поскольку часто один тип движения больше подходит для определенного стиля игры, чем другой. Синтаксис: x; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if keyboard_check(vk_left) x += 5; if keyboard_check(vk_right) x -= 5; Приведенный выше код будет использовать клавиши со стрелками для добавления или вычитания к позиции x и таким образом создавать иллюзию движения.
