экземпляр_активировать_слой С помощью этой функции можно активировать слой, который ранее был деактивирован. Вам необходимо указать ID слоя, который может быть либо именем слоя, записанным в редакторе кода (в виде строки), либо фактическим значением ID слоя, возвращаемым программой layer_create() и все деактивированные экземпляры на этом слое снова активируются. Обратите внимание, что если вы деактивировали слой, который имеет постоянные экземпляры, вам необходимо необходимо снова активировать слой с помощью этой функции перед сменой комнаты, иначе все постоянные экземпляры не будут перенесены и будут отброшены. Обратите также внимание, что активация не является мгновенной, и экземпляр, который был активирован не будет считаться активным до окончания события, в котором была вызвана эта функция. Синтаксис: instance_activate_layer(layer_id); Аргумент Описание идентификатор слоя Строка (или значение) идентификатора слоя, который будет использоваться Возвращается: Н/Д Пример: instance_activate_all(); var _vx = camera_get_view_x(view_camera[0]); var _vy = camera_get_view_y(view_camera[0]); instance_deactivate_region(view_xview[0] - 64, view_yview - 64, _vx + 128, _vy + 128, false, false); instance_activate_layer("Player_Layer"); Приведенный выше код активирует все экземпляры в пределах комнаты, а затем деактивирует те, которые находятся за пределами текущего обзора камеры, за исключением экземпляров на слое "Player_Layer", которые снова активируются в конце.
