список_мест_экземпляров С помощью этой функции вы можете проверить позицию на столкновение со всеми экземплярами объекта, используя маску столкновения того экземпляра, который запускает код проверки. При использовании этой функции вы фактически просите GameMaker Studio 2 переместить экземпляр в новую позицию, проверить наличие столкновения, затем вернуться назад и сообщить вам, было ли обнаружено столкновение или нет. Это будет работать для точных столкновений, но только если и у экземпляра, и у экземпляров проверяемого объекта выбраны точные маски столкновений. в противном случае применяются только столкновения с ограничивающей рамкой. Функция заполнит (предварительно созданный) список DS уникальными id значениями всех экземпляров объекта, которые считаются находящимися в столкновении с вызывающим экземпляром, и у вас есть возможность упорядочить список на основе расстояния от начала координат экземпляра, выполняющего проверку, до начала координат экземпляров, найденных в столкновении найденных столкнувшимися. Обратите внимание, что функция также принимает специальное ключевое слово всев этом случае будут перечислены все экземпляры, найденные в столкновении. Функция возвращает количество найденных экземпляров, или 0, если не найдено ни одного. Обратите внимание, что заданные координаты x/y будут округлены до ближайшего целого числа перед выполнением проверки, поэтому, если это не то, что вам нужно (или вы используете устаревший продукт GameMaker ), вам следует заполнить координаты в параметре проверить: instance_place_list(floor(x), floor(y), obj_Enemy, list, true). Синтаксис: instance_place_list(x, y, obj, list, ordered); Аргумент Описание x Позиция x для проверки экземпляров. y Позиция y для проверки экземпляров. obj Объект для проверки экземпляров. список Список DS, используемый для хранения идентификаторов сталкивающихся экземпляров. заказал Должен ли список быть упорядочен по расстоянию(true) или нет(false). Возвращается: Int (Количество экземпляров, с которыми произошло столкновение) Пример: var _list = ds_list_create(); var _num = instance_place_list(x, y, obj_Enemy, _list, false); if _num > 0 { for (var i = 0; i < _num; ++i;) { instance_destroy(_list[| i]); } } ds_list_destroy(_list); Приведенный выше код проверит на столкновение все экземпляры, найденные в позиции вызывающего экземпляра. Они будут добавлены в список DS, который затем будет пройден по циклу для уничтожения каждого из столкнувшихся экземпляров.
