скорость Все экземпляры в GameMaker Studio 2 имеют определенные "встроенные" свойства, которые можно использовать и устанавливать для управления их внешним видом и поведением. Скорость является одним из таких свойств и определяет, на сколько пикселей экземпляр будет перемещаться каждый шаг. В отличие от hspeed и vspeed скорость не связана с направлением, так как оно зависит от направление значением экземпляра, но она может иметь отрицательное значение, в этом случае экземпляр будет двигаться в направлении, противоположном тому, которое задано функцией direction (т.е. direction - 180°). Обратите внимание, что установка параметров speed и/или direction, также изменит значения переменных hspeed и vspeed, и что gravity, gravity_direction и friction могут изменять значение этой переменной, когда они используются в ваших играх. Синтаксис: скорость; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if keyboard_check(vk_up) speed = 2; if keyboard_check(vk_left) direction += 5; if keyboard_check(vk_right) direction -= 5; Приведенный выше код будет использовать клавиши со стрелками для установки скорости и направления движения экземпляра.
