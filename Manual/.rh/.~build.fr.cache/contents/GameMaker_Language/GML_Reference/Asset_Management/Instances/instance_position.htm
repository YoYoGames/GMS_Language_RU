позиция экземпляра С помощью этой функции можно проверить позицию на столкновение с другим экземпляром или всеми экземплярами объекта. При использовании этой функции вы проверяете одну точку в комнате на наличие экземпляра или объекта. Проверка выполняется по ограничивающей или по маске объекта, если у этого объекта есть точная проверка столкновений, и возвращает уникальный экземпляр. id. Если вам не нужен идентификатор столкнувшегося экземпляра, вам следует следует использовать позиция_встречи() вместо этого. Эта функция также принимает специальные ключевые слова все и other и возвращает ключевое слово noone, если столкновения не произошло, или уникальное значение ID найденного экземпляра, если столкновение произошло. Синтаксис: instance_position( x, y, obj ); Аргумент Описание x Позиция x для проверки экземпляров. y Позиция y для проверки экземпляров. obj Объект для проверки экземпляров. Возвращается: Real (либо значение ID экземпляра, либо ключевое слово noone) Пример: var inst; inst = instance_position(mouse_x, mouse_y, obj_Pause_Button); if (inst != noone) { with (inst) image_index=1; instance_create_layer(room_width / 2, 0, "Controllers", obj_Menu); } Приведенный выше код проверит наличие столкновения с экземпляром "obj_Pause_Button" в позиции мыши, и если оно есть, то использует возвращенный id для установки его image_index на новое значение перед созданием нового экземпляра объекта "obj_Menu".
