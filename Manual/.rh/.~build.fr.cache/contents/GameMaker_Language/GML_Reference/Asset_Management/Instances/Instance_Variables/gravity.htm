гравитация гравитация - это одна из "встроенных" переменных, которую имеют все экземпляры и которая, когда она установлена, прикладывает постоянную силу к направление_гравитации экземпляра, оказывая влияние как на экземпляр скорость и direction . Обратите внимание, что гравитация является кумулятивной силой и будет ускорять объект, если вы решите не ограничивать конечную скорость, и обычно вы устанавливаете эту переменную на небольшие десятичные значения, например 0.01. например, 0,01. Если вы установите gravity в 0, то гравитация не будет применяться к экземпляру (это значение по умолчанию). Синтаксис: гравитация; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if !place_meeting(x, y + 1, obj_Ground) { gravity = 0.01; } else { gravity = 0; } Приведенный выше код будет применять гравитацию, только если экземпляр не найдет под собой ни одного экземпляра "obj_Ground".
