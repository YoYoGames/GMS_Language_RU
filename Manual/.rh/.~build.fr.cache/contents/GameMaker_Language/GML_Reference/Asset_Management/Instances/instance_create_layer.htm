экземпляр_создания_слоя С помощью этой функции можно создать новый экземпляр указанного объекта в любой заданной точке помещения и на указанном слое. Слой может быть идентифицирован с помощью значения ID слоя (возвращаемого функцией layer_create()) или по имени слоя (в виде строки, например, "instance_layer"), заданному в редакторе помещения . Эта функция возвращает id нового экземпляра, который затем может быть сохранен в переменной или использован для доступа к этому экземпляру. Обратите внимание, что эта функция также вызовет событие Create Event создаваемого экземпляра, прежде чем продолжить код или действия для события, вызвавшего функцию. ВАЖНО! Существует минимальная и максимальная глубина слоя от -16000 до 16000. Все, что размещено на слое вне этого диапазона, не будет нарисовано, хотя все события будут проходить в обычном режиме. Синтаксис: instance_create_layer(x, y, layer_id, obj); Аргумент Описание x Позиция x, в которой будет создан объект y Позиция y, в которой будет создан объект идентификатор слоя Идентификатор слоя (или имя), которому следует присвоить созданный экземпляр obj Индекс объекта для создания экземпляра объекта Возвращается: Real (значение идентификатора экземпляра) Пример: var inst = instance_create_layer(x, y, "instance_layer", obj_Bullet); with (inst) { speed = other.shoot_speed; direction = other.image_angle; } Приведенный выше код создает новый экземпляр объекта "obj_Bullet" и сохраняет идентификатор экземпляра в переменной.
