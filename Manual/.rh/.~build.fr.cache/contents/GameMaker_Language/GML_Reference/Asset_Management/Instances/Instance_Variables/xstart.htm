xstart Эта переменная хранит начальную x-позицию экземпляра при его первом создании в комнате. Это не переменная только для чтения, и ее можно как установить, так и прочитать. Синтаксис: xstart; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if place_meeting(x, y, obj_spike) { score -= 100; x = xstart; y = ystart; } Приведенный выше код проверит столкновение с экземпляром "obj_spike", и если оно есть, то вычтет 100 из счета и переместит экземпляр обратно в начальную позицию.
