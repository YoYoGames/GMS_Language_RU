место_объекта С помощью этой функции вы можете проверить позицию на столкновение с другим экземпляром или со всеми экземплярами объекта, используя маску столкновения того экземпляра, который запускает код проверки. При использовании этой функции вы фактически просите GameMaker Studio 2 переместить экземпляр в новую позицию, проверить наличие столкновения, вернуться назад и сообщить вам, было ли обнаружено столкновение или нет. Это будет работать для точных столкновений, но только если и у экземпляра, и у проверяемого объекта выбраны маски точных столкновений. в противном случае применяются только столкновения с ограничивающей рамкой. эта функция вернет уникальный экземпляр id объекта, с которым произошло столкновение, но если это не нужно, то немного быстрее использовать функцию функцию место_встречи(). Эта функция также принимает специальные ключевые слова all и other и возвращает ключевое слово , если столкновения не произошло. возвращает ключевое слово noone, если столкновения не произошло, или уникальное значение ID найденного экземпляра, если столкновение произошло. Обратите внимание, что заданные координаты x/y будут округлены до ближайшего целого числа перед выполнением проверки, поэтому, если это не то, что вам нужно (или вы используете устаревший продукт GameMaker ), вам следует заполнить координаты в параметре проверить: instance_place_list(floor(x), floor(y), obj_Enemy, list, true). Синтаксис: instance_place(x, y, obj); Аргумент Описание x Позиция x для проверки экземпляров. y Позиция y для проверки экземпляров. obj Объект для проверки экземпляров. Возвращается: Real (либо значение ID экземпляра, либо ключевое слово noone) Пример: var _inst = instance_place(x, y, obj_Enemy); if _inst != noone { hp -= _inst.dmg; instance_destroy(_inst); } Приведенный выше код проверит наличие столкновения с экземплярами "obj_Enemy" и, если таковое произойдет, уменьшит переменную "hp" на величину, хранящуюся в переменной "dmg" столкнувшегося экземпляра, а затем уничтожит столкнувшийся экземпляр. экземпляр.
