Переменные экземпляра Когда вы создаете новый объект, он будет поставляться с определенными переменными, уже инициализированными значениями по умолчанию. Эти переменные мы называем переменными экземпляра, поскольку они будут создаваться для каждого экземпляра объекта, который вы помещаете в игровые комнаты, и после создания экземпляра значения этих переменных уникальны для этого и только этого экземпляра. Некоторые из этих переменных вы будете использовать часто, например x и y, в то время как другие нужны меньше, в зависимости от того, что вы хотите сделать. но в целом они очень полезны, и по возможности вы должны использовать их, а не создавать свои собственные переменные. Большинство переменных экземпляра можно как устанавливать, так и считывать, что позволяет вам изменять свойства и поведение экземпляра, просто изменяя значение определенной переменной - например, вы можете запретить экземпляру рисовать, просто установив переменную visible встроенной переменной false. Ниже приведены различные переменные, которые инициализируются для всех экземпляров всех объектов в вашей игре. ВАЖНО! Если экземпляр находится в последовательности, то некоторые из этих переменных - например, image_xscale / image_yscale / image_angle / x / y - будут перезаписаны, когда последовательность обновляет каждый шаг после начала воспроизведения.
