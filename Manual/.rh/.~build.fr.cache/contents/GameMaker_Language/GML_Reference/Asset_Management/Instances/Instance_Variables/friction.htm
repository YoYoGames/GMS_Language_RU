трение Трение является одним из "встроенных" свойств, которыми могут обладать экземпляры, и может быть использовано для замедления экземпляра со временем, если параметр скорость отлична от нуля. Оно работает просто путем вычитания суммы из скорости на каждом шаге, пока скорость объекта не станет равной 0. Если трение установлено, например, на 0,1, а скорость экземпляра равна 1 (1 пиксель за шаг), он замедлится и остановится после 10 шагов. Обратите внимание, что трение применяется к положительным и отрицательным скоростям одинаково, в результате чего через определенное время скорость объекта всегда будет равна 0. Синтаксис: трение; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: если abs(speed) > 0 { трение = 0.05; } else { трение = 0; } Приведенный выше код будет применять трение, только если абсолютная скорость экземпляра больше 0.
