экземпляр_ближайший Эта функция проверяет все экземпляры заданного объекта, чтобы узнать, какой из них находится ближе всего к заданной точке x/y. Все проверки будут проводиться от заданной точки x/y к началу координат (точка x/y) экземпляров указанного объекта. Если ни одного экземпляра объекта, функция вернет ключевое слово noone, но если экземпляры существуют, то функция вернет значение id найденного экземпляра. Обратите внимание, что если экземпляр, выполняющий код, имеет тот же индекс объекта, что и проверяемый объект, то он будет включен в проверку (это включает в себя проверку родительских объектов, если вызывающий экземпляр также является дочерним объектом родительского объекта). Синтаксис: instance_nearest(x, y, obj); Аргумент Описание x Позиция x для проверки. y Положение y для проверки. obj Объект для проверки экземпляров. Возвращается: Real (либо значение ID экземпляра, либо ключевое слово noone) Пример: var inst, xx; xx = x; x -= 10000; inst = instance_nearest(xx, y, object_index); if inst != id { draw_line(x, y, inst.x, inst.y); } x += 10000; Приведенный выше код перемещает текущий экземпляр на 10000 пикселей, затем проверяет его предыдущую позицию, чтобы найти ближайший экземпляр объекта того же типа. Если этот экземпляр окажется тем же самым, то он не будет делать ничего другого, кроме как переместится в исходное положение, но если найденный экземпляр будет другим, то будет проведена линия между ними. если найденный экземпляр будет другим, он проведет линию между ними.
