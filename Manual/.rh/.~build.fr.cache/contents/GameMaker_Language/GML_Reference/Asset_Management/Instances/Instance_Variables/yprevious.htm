yprevious Эта встроенная переменная возвращает предыдущую позицию y для данного экземпляра. Эта переменная будет установлена в текущее положение x непосредственно перед началом события begin step, но она также может быть установлена через код в любое время, то есть вы можете придать ей собственное значение, если в этом возникнет необходимость. Синтаксис: yprevious; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if x != xprevious || y != yprevious { moved = true; } Приведенный выше код проверяет, есть ли разница между предыдущими значениями xprevious и yprevious и текущими значениями x и y. Если есть, то переменная устанавливается в true. Назад: Переменные экземпляра Далее: id © Copyright YoYo Games Ltd. 2021 Все права защищены
