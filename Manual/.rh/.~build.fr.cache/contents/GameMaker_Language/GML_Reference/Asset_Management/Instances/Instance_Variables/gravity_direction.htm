направление силы тяжести gravity_direction - это одно из "встроенных" свойств всех экземпляров, которое можно использовать для задания направления движения, когда значение экземпляра гравитация Обратите внимание, что направления в GameMaker Studio 2 обычно рассчитываются как 0° - вправо, 90° - вверх, 180° - влево и 270° - вниз. Синтаксис: направление гравитации; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if place_meeting(x, y, obj_switch) { gravity_direction += 180; } Приведенный выше код изменит направление гравитации, если объект игрока встретит экземпляр "obj_switch".
