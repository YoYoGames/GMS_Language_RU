hspeed hspeed - это одно из "встроенных" свойств, которым обладают все экземпляры. Оно определяет скорость горизонтального перемещения (вдоль оси x) экземпляра в пикселях за шаг. Так, hspeed, равная 3, означает движение на 3 пикселя вправо (+x) на каждый шаг, а hspeed -3 означает движение на 3 пикселя влево (-x) на каждый шаг. Обратите внимание, что если вы установите переменные speed и/или direction, то значение hspeed будет обновлено автоматически, чтобы и аналогично, изменение значения hspeed также повлияет на значения speed и direction. Также обратите внимание, что gravity, gravity_direction и friction могут изменять значение этой переменной, когда они используются в ваших играх. Синтаксис: hspeed; Возвращается: Real (значение с плавающей точкой одинарной точности) Пример: if keyboard_check(vk_left) hspeed = -5; if keyboard_check(vk_right) hspeed = 5; Приведенный выше код будет изменять горизонтальную скорость в зависимости от того, какие клавиши нажаты.
