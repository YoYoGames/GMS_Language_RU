установка_тревоги Эта функция может использоваться для установки сигнала тревоги. Вы вводите номер сигнала тревоги от 0 до 11, а затем значение, на которое нужно установить сигнал тревоги. Значение должно быть целым, и вы можете установить его на -1, чтобы остановить сигнал тревоги (нецелые значения будут округлены до ближайшего целого числа). целому числу). Это альтернативный метод непосредственной установки массива сигналов тревоги. Синтаксис: alarm_set(index, value); Аргумент Описание индекс Индекс сигнализации, который необходимо установить, от 0 до 11. значение Значение (целое число), на которое нужно установить сигнал тревоги. Возвращается: Н/Д Пример: for (var i = 0; i < 12; i++) { if alarm_get(i) > 0 alarm_set(i, -1); } Приведенный выше код проверяет все сигналы тревоги в вызывающем экземпляре и, если они больше 0, устанавливает их в -1, останавливая дальнейший отсчет.
