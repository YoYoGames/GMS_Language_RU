список_позиций_экземпляров С помощью этой функции можно проверить позицию на столкновение со всеми экземплярами другого объекта. При использовании этой функции вы проверяете одну точку в комнате на наличие всех экземпляров объекта, и проверка будет выполняться по ограничивающей рамке всех экземпляров или по маске экземпляров, если у этого объекта отмечены точные столкновения. или по маске экземпляров, если у этого объекта отмечены точные столкновения. Функция заполнит (ранее созданный) список DS уникальными id значениями всех экземпляров объекта, которые считаются находящимися в и у вас есть возможность упорядочить список на основе расстояния от проверяемой позиции до начала координат всех экземпляров, которые считаются находящимися в столкновении. Обратите внимание, что функция также принимает специальное ключевое слово GML_Overview/Instance_Keywords.htm">всев этом случае будут перечислены все экземпляры, найденные в столкновении. Функция возвращает количество найденных экземпляров или 0, если не найдено ни одного. Синтаксис: instance_position_list(x, y, obj, list, ordered); Аргумент Описание x Позиция x для проверки экземпляров. y Позиция y для проверки экземпляров. obj Объект для проверки экземпляров. список Список DS, используемый для хранения идентификаторов сталкивающихся экземпляров. заказал Должен ли список быть упорядочен по расстоянию(true) или нет(false). Возвращается: Int (Количество экземпляров, с которыми произошло столкновение) Пример: var _list = ds_list_create(); var _num = instance_position_list(mouse_x, mouse_y, all, _list, false); if _num > 0 { for (var i = 0; i < _num; ++i;) { instance_destroy(_list[| i]); } } ds_list_destroy(_list); Приведенный выше код проверит на столкновение все экземпляры, найденные в позиции мыши. Они будут добавлены в список DS, который затем будет пройден по циклу для уничтожения каждого из столкнувшихся экземпляров.
