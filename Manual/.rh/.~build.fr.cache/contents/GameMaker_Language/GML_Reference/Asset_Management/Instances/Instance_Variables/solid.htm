твердый Экземпляр может быть помечен как твердый через свойства объекта в редакторе объектов или путем изменения значения этой встроенной переменной. Если solid имеет значение true, то, при обнаружении столкновения столкнувшийся экземпляр автоматически возвращается в положение, в котором он находился на шаге, предшествующем столкновению (а затем выполняется любой код или действия в событии столкновения). Если установлено значение false, то все позиционирование должно выполняться через событие столкновения. Синтаксис: твердый; Возвращается: Булево Пример: if other.solid = true { x = xprevious; y = yprevious; } Приведенный выше код проверит флаг solid экземпляра "другого" экземпляра при столкновении и, если он установлен в true, переместит этот экземпляр в его предыдущую позицию.
