временная шкала_удаления С помощью этой функции вы можете удалить любую временную линию из вашей игры. Если эта временная линия была создана в GameMaker Studio 2 Asset Browser, обратите внимание, что она полностью удаляется из игры, и попытка вызвать индексированную временную линию после использования этой функции приведет к ошибке. таймлайн после использования этой функции приведет к ошибке. Если линия времени была создана динамически с помощью функции timeline_add() функцию, то эта функция должна быть использована в тот момент, когда чтобы предотвратить утечку памяти, которая может замедлить работу (и в конечном итоге привести к краху) вашей игры. Синтаксис: timeline_delete(ind); Аргумент Описание инд Индекс временной линии для удаления. Возвращается: Н/Д Пример: if timeline_exists(global.tl) { timeline_delete(global.tl); } Приведенный выше код проверяет, существует ли временная линия и индексируется ли она в глобальной переменной "tl", и если да, то удаляет эту временную линию.
