временная шкала_существует С помощью этой функции вы можете проверить, существует ли линия времени (возвращается true) или нет (возвращается false). Это особенно полезно при динамическом создании временных линий с помощью функции timeline_add() функцию, но следует иметь в виду, что при поиске неинициализированной переменной (которая в противном случае была бы присвоена индексу временной линии) будет выдана ошибка. Синтаксис: timeline_exists(ind); Аргумент Описание инд Индекс временной линии для проверки. Возвращается: Булево Пример: if timeline_exists(global.tl) { timeline_delete(global.tl); } Приведенный выше код проверяет, существует ли временная линия и индексируется ли она в глобальной переменной "tl", и если да, то удаляет эту временную линию.
