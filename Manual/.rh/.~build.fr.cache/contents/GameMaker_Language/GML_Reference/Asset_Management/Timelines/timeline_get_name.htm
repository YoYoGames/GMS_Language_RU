имя_временной_линии Эта функция может быть использована для получения имени временной линии в виде строки. если временная линия была создана динамически с помощью функции timeline_add() функции, возвращаемое имя будет иметь формат "__newtimelineN", где " N" -номер временной линии (начиная с 0). Обратите внимание, что это только строка и не может быть использована для прямой ссылки на временную шкалу - для этого вам понадобится индекс временной шкалы. для этого вам понадобится индекс временной шкалы. Однако вы можете использовать эту строку для получения индекса временной шкалы, используя возвращаемую строку вместе с функцией asset_get_index(). Синтаксис: timeline_get_name(ind); Аргумент Описание инд Индекс линии времени, название которой необходимо проверить. Возвращается: Строка Пример: var _str = timeline_get_name(timeline_index); show_debug_message("Current timeline = " + _str); Приведенный выше код получает имя текущей назначенной временной шкалы и выводит его на консоль.
