таймлайн_момент_чистый С помощью этой функции вы можете очистить конкретный момент любой ранее определенной временной линии от всех кодов и действий. Синтаксис: timeline_moment_clear(ind, step); Аргумент Описание инд Индекс временной линии, которую нужно очистить. шаг Момент для очистки. Возвращается: Н/Д Пример: timeline_moment_clear(global.tl, room_speed * 30); Приведенный выше код очистит указанный момент временной линии, индексируемой переменной "global.tl".
