ширина браузера В этой переменной хранится ширина (в пикселях) браузера, в котором запущена игра. Если браузер отсутствует, то возвращается размер окна. на следующем рисунке показано, как эта переменная связана с остальными функциями позиционирования, доступными через функции окна. Синтаксис: ширина браузера Возвращается: Реально Пример: if (browser_height > window_get_height()) || (browser_width > window_get_width()) { var xx, yy; if (browser_width > window_get_width()) { xx = (browser_width - window_get_width()) / 2; } else { xx = 0; } if (browser_height > window_get_height()) { yy = (browser_height - window_get_height()) / 2; } else { yy = 0; } window_set_position(xx, yy); } Приведенный выше код проверяет ширину и высоту браузера на предмет того, не превышает ли они ширину и высоту текущего окна.
