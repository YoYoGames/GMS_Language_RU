clickable_add_ext С помощью этой функции вы можете создать пользовательскую иконку с возможностью клика, которая действует как "кнопка" вне игровой канвы, но в самом окне браузера. Функция возвращает значение id, которое должно быть сохранено в переменной для использования во всех последующих функциях кнопок, которые ссылаются на новую иконку. Эта функция очень полезна для предотвращения создания браузером всплывающего окна при нажатии на кнопку, поскольку все это делается в самой GameMaker Studio 2. Эта функция работает точно так же, как и clickable_add() где вы должны задать позицию относительно (0,0) окна браузера, указать текстуру страницы для спрайта, который вы хотите, чтобы кнопка имела (см: sprite_get_tpe()), указать URL для открытия или скрипт (из браузера активов GameMaker Studio 2) для открытия или запуска, а затем задайте цель и параметры для использования. Однако эта функция также имеет две дополнительные опции, которые позволяют установить альфа-фактор значка от 0 (полностью прозрачный) до 1 (полностью непрозрачный) и масштаб значка, который представляет собой множитель, применяемый к используемым используемому спрайту. Таким образом, значение 1 - это масштаб по умолчанию, в то время как (например) 0,5 - это половинный масштаб, а 2 - двойной. Также стоит отметить, что если передаваемый вами аргумент URL является именем функции скрипта (т.е. не начинается с "http://") И скрипт вызывается gmcallback_* то вы можете получить прямую передачу в html5-событие. Имена скриптовых функций GameMaker Studio 2 не обфусцируются, если они начинаются с gmcallback_. Синтаксис: clickable_add_ext(x, y, tpe, url, target, params, alpha, scale); Аргумент Описание x Позиция x в окне. y Позиция y в окне. tpe Запись страницы текстуры для используемого спрайта. url URL (адрес веб-сайта) для ссылки или сценарий, который необходимо запустить. цель Это целевая область, в которой нужно открыть URL (см. описание). params Различные параметры для значка (см. описание). альфа Альфа изображения значка (по умолчанию 1). шкала Масштаб значка (по умолчанию 1). Возвращается: Реально Пример: home_but = clickable_add_ext(32, 32, sprite_get_tpe(spr_MS_Home, 0), "http://macsweeney_games.com", "_blank", "width=700, height=500, menubar=0, toolbar=0, scrollbars=0", 1, 2); Приведенный выше код создает кликабельную DOM-иконку в позиции (32, 32) страницы, на которой запущен игровой холст. Иконка использует спрайт, ссылающийся на страницу текстуры как "spr_MS_Home", и при нажатии на иконку откроется новое окно для указанного URL и определенных свойств этого окна. для указанного URL и с определенными свойствами для этого окна. Кнопка также будет масштабирована до удвоенного размера оригинального спрайта и половины альфа изображения, равного 1.
