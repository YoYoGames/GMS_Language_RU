Функции совместимости При импорте проекта из предыдущей версии GameMaker вы можете увидеть, что в проект добавлены сценарии совместимости. В целом, эти скрипты используют обновленные функции языка GameMaker Language(GML) для преобразования устаревших функций в новые, и это отличный способ узнать, как все должно быть сделано в GameMaker Studio 2. Однако есть несколько функций, используемых только для совместимости, которые не должны использоваться нигде больше, и уж точно не в ваших собственных проектах.
