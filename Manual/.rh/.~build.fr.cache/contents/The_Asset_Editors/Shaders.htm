Редактор шейдеров Шейдеры - это очень мощный инструмент, который можно использовать для манипулирования графикой, выводимой на экран вашей игры, позволяя невероятно быстро создавать эффекты, которые могут быть самыми разными. от, например, добавления тонкого цветового оттенка к спрайту до эффектов искажения всего экрана. Но что такое шейдер? Шейдер - это программа, состоящая из двух частей, которая выполняется непосредственно на самой видеокарте, что делает ее очень быстрой, поскольку всю работу выполняет GPU, и освобождает циклы CPU для вашего игрового кода. Полный шейдер состоит из программы вершинного шейдера и фрагментного шейдера (также известного как пиксельный шейдер). Обе эти маленькие программы работают вместе, чтобы манипулировать тем, что графическая карта отображает на экране. Это позволяет в реальном времени управлять положением, цветом и альфа-каналом. которые на самом деле отображаются в буфере дисплея. Вершинный шейдер Вершинный шейдер - это программируемый шейдерный этап в конвейере рендеринга, который обрабатывает отдельные вершины (точки треугольников, используемых для рендеринга любого изображения). При рендеринге геометрии - например, спрайта или поверхности - GameMaker Studio 2 создает поток вершин, называемый буфером вершин (Vertex Buffer), который определяет геометрию этих треугольников. - GameMaker Studio 2 создает поток вершин - так называемый буфер вершин - который определяет геометрию этих треугольников. Например, геометрия спрайта состоит из двух треугольников (обычно называемых полигонами), которые отображаются вместе, образуя "квадрат". вместе, чтобы сформировать "квадрат". Этот поток вершин из буфера вершин подается на вход вершинного шейдера, который может обрабатывать данные вершин программируемым образом. Выходной сигнал шейдера вершин используется графическим процессором для сборки треугольников, которые затем правильно обрезаются и обрезаются для порта обзора и камеры обзора, а затем отправляются в блок растеризации GPU, который генерирует новый выходной поток, состоящий из того, что называется фрагментами. Это крошечные структуры данных структуры данных, каждая из которых относится к одному пикселю, появляющемуся на экране. Фрагментный шейдер Фрагментный шейдер - это программируемый шейдерный этап конвейера рендеринга, который работает с "фрагментами" - интерполированными пикселями, используемыми для текстурирования любого данного полигона - и отвечает за вывод окончательного цвета пикселя каждого пикселей каждого треугольника. В основном это работает следующим образом: шейдер фрагментов получает на вход все фрагменты (отдельные пиксели треугольника), которые были переданы по конвейеру вершинным шейдером. Затем он может обрабатывать эти фрагменты, чтобы изменить цвет и альфа-коэффициент конечного пикселя, который будет выведен на экран.
