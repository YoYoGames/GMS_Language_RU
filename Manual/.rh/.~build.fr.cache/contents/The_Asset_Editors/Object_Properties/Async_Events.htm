Асинхронные события Асинхронное событие - это событие, которое запускается, когда GameMaker Studio 2 получает обратный вызов из какого-либо внешнего источника, который может быть из Интернета или с устройства, на котором запущена ваша игра. По сути, вы говорите GameMaker Studio 2 сделать что-то - например, загрузить изображение - и он начинает это делать, но продолжает делать все остальное, что он должен делать в это время. Затем, когда запрос будет завершен, в GameMaker Studio 2 и все асинхронные события, определенные для этого типа обратного вызова, будут запущены. Обратите внимание, что асинхронные события запускаются для всех экземпляров, у которых они есть, так же как и события клавиатуры, поэтому вы можете, например, сделать вызов http_get() в одном экземпляре, а асинхронное событие HTTP в другом для обработки обратного вызова. ПРИМЕЧАНИЕ: Вы должны знать, что из-за защиты от XSS в браузерах, запросы и попытки загрузить ресурсы из других доменов блокируются и могут возвращать пустые результаты при использовании любого из следующих событий.
