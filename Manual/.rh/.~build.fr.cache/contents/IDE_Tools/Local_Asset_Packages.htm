Пакеты местных активов В меню Инструменты есть две опции для создания и импорта локальных пакетов активов. Пакет локальных активов - это группа ресурсов, сохраненных с помощью формата файла *.yymp, которые можно легко установить в существующие проекты. Это означает, что вы можете создать собственную библиотеку пакетов для часто используемых ресурсов, таких как библиотеки скриптов, заставки, системы меню и т.д... без необходимости использовать YoYo Games Торговая площадка. Создание пакета активов Чтобы создать пакет активов, необходимо сначала создать ресурсы, которые вы хотите сохранить в проекте, а затем в меню Инструменты выбрать опцию Создать локальный пакет. Откроется следующее окно: В этом окне вы можете указать необязательное Имя издателя и обязательные Отображаемое имя и Идентификатор пакета. Идентификатор пакета будет использоваться для создания имени файла, под которым будет сохранен пакет, и к нему будет автоматически добавлено расширение файла *.yymp при создании. Вы также можете задать номер версии пакета, а также выбрать, подписывать его или нет (по умолчанию эта функция отключена). Для подписания пакета необходимо создать сертификат подписи (вы можете сделать это в ПредпочтенияхMarketplace), и это означает, что любой актив, который был подписан, не сможет быть импортирован, если он был подделан после создания. создания. После того, как вы заполнили детали пакета, вы можете перейти к выбору активов, которые вы хотите включить из раздела "Проект" справа (используя левую кнопку мыши или комбинацию / и вместе с левой кнопкой мыши). Выбранные ресурсы можно переместить в представление "Пакет активов" с помощью кнопок кнопки Добавить или Добавить все и могут быть снова удалены с помощью кнопок Удалить и Удалить все. Когда вы будете удовлетворены выбором ресурсов и заполните необходимые данные, при нажатии кнопки Ok откроется проводник файлов, где вы можете выбрать место для сохранения пакета. После выбора пакет будет создан и появится окно подтверждения с возможностью открыть местоположение пакета в проводнике файлов ОС. Импорт пакета активов Существует два способа импорта локального пакета активов: первый - просто перетащить файл *.yymp из окна проводника в IDE GameMaker Studio 2. второй - выбрать опцию "Импорт локального пакета" в меню Инструменты . При использовании меню "Инструменты" откроется окно проводника, в котором можно перейти к расположению пакета и выбрать его для импорта. В любом случае в GameMaker Studio 2 откроется следующее окно: Слева находится содержимое пакета, а справа - его части, которые вы хотите импортировать. Вы можете выбрать один или несколько ресурсов из пакета (используя левую кнопку мыши или комбинацию / и вместе с левой кнопкой мыши), а затем использовать кнопку Add (Добавить), чтобы добавить их или использовать кнопку Add All (Добавить все). ) и затем использовать кнопку Добавить , чтобы добавить их для импорта, или использовать кнопку Добавить все , чтобы добавить все для импорта. Когда все готово, вы можете нажать кнопку Импорт , чтобы добавить выбранные ресурсы в проект. Обратите внимание, что у вас также есть возможность Импортировать все ресурсы в новый проект, что приведет к созданию нового проекта (и попросит вас сохранить его в нужном месте). проект (и предложит указать место для его сохранения), а затем добавит в него ресурсы пакета. Обратите внимание, что пакеты активов, созданные с помощью версии, предшествующей GameMaker Studio 2.3, потребуют конвертации, поскольку формат файла проекта изменился с обновлением 2.3. Это означает, что предпочтительнее будет импортировать весь проект и преобразовать все файлы. затем экспортировать его как новый пакет в новое место, если активы часто импортируются в другие проекты.
